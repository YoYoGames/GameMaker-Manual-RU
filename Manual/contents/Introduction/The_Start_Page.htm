Начальная страница

Когда вы впервые откроете GameMaker, вам будет предложено войти в свою учетную запись YoYo или Opera (или зарегистрироваться, если вы еще не зарегистрировались - см. здесь). После регистрации и входа в систему на стартовой странице откроется основная среда GameMaker IDE:

Когда вы впервые запустите GameMaker, вы сможете увидеть описание каждого элемента начального экрана, наведя на него курсор.

Здесь вы можете нажать кнопку "SKIP to Setup Wizard", чтобы начать создавать свой первый проект с помощью мастера Setup Wizard, который проведет вас через все шаги, необходимые для создания вашей первой игры в GameMaker.

Вы также можете нажать кнопку "Закрыть", чтобы закрыть мастер и продолжить работу с GameMaker вручную, а также установить флажок "Не показывать мне снова", если вы не хотите видеть мастер при повторном запуске GameMaker.

Элементы стартовой страницы

Стартовая страница содержит следующие элементы:

  • Строка меню IDE: Здесь показаны меню, которые вы можете использовать в IDE; более подробная информация приведена в разделе ниже.
  • Параметры проекта: В разделе "Проекты" можно создавать, открывать или импортировать проекты. Более подробно об этом рассказывается далее на этой странице.
  • Последние проекты: Здесь вы можете увидеть список предыдущих проектов, которые вы можете открыть. Вы также можете увидеть полное название проекта и путь к нему, а нажатие левой кнопки мыши приведет к открытию проекта. Вы также можете переключать вид этого раздела между режимом плитки и режимом списка, нажимая кнопки в правом верхнем углу.
  • Сведения о версии: В этом разделе содержится подробная информация о текущей используемой версии IDE, а также о текущей версии Runtime. В этом разделе вы также будете получать уведомления о любых изменениях, доступных для IDE или Runtime.
  • Сведения о пользователе: Здесь отображается имя пользователя вашей учетной записи YoYo и время последнего входа в систему.
  • Полезные ресурсы: Здесь показаны различные плитки, на которые можно нажать, чтобы получить доступ к официальным обучающим ресурсам по GameMaker.

Меню IDE

В верхней части находятся общие меню IDE, которые описаны в следующих разделах руководства:

  • Меню "Файл
  • Меню редактирования
  • Меню сборки
  • Меню Windows
  • Меню инструментов
  • Меню Marketplace
  • Меню макетов
  • Меню справки

Обратите внимание, что в верхней строке меню также появятся контекстно-специфические пункты меню, в зависимости от того, на каком окне вы сфокусировались в данный момент. Например, если в фокусе находится окно редактора изображений, то здесь появятся дополнительные пункты меню "Изображения", "Просмотр" и "Эффекты". Эти дополнительные пункты меню описаны в соответствующих разделах руководства для рабочей области или окна, которое их создало.

Начало нового проекта

Вы можете нажать кнопку New , чтобы создать новый проект, кнопку Open , чтобы открыть существующий проект, или кнопку Import , чтобы открыть сжатый файл проекта GameMaker YYZ или старый файл проекта GameMaker: Studio 1.4 GMX. Как при открытии, так и при импорте откроется проводник файлов, в котором вы сможете найти нужный вам файл проекта.

ПРИМЕЧАНИЕ: GameMaker не полностью обратно совместим с GameMaker: Studio 1.4, но импортированные проекты 1.4 должны работать, так как устаревшая функциональность была воссоздана автоматически с помощью скриптов совместимости. Подробную информацию о возможных проблемах и изменениях, внесенных в GML, можно найти в статье Справочного центра " Перенос проекта GMS 1.4 на GameMaker", а также в разделе руководства " Устаревшие функции".

Start Page Elements

The Start Page contains the following elements:

  • Menu Bar: This shows you the menus that you can use throughout the IDE; more information is given in a section below.
  • Project Options: The Projects section is where you can create, open or import projects. This is explained in more detail further down this page.
  • Recent Projects: Here you can see a list of previous projects that you can open. You can also see the full project name and path, and clicking the left mouse button  will open the project. You can also switch the view of this section between tile mode and list mode by pressing the buttons in the top-right corner.
  • Version & Log In: This section provides details on the current IDE version being used, as well as the current Runtime version. You will also get notified of any changes available to either the IDE or the Runtime in this section. Here you are also able to log in and access your account. See: Version & Account Details.
  • Useful Resources: This shows you various tiles that you can click on to access official tutorial resources for GameMaker.

IDE Menus

At the top you can find the general IDE menus which are explained in the following sections of the manual:

  • The File Menu
  • The Edit Menu
  • The Build Menu
  • The Windows Menu
  • The Tools Menu
  • The Marketplace Menu
  • The Layouts Menu
  • The Help Menu

Note that there will also appear context specific menu options in the top menu bar, depending on the window that you have focused on currently. For example, if you have the Image Editor window in focus then you will have extra menu items here for "Images", "View" and "Effects". These extra menu items are explained in the relevant sections of the manual for the workspace or window that generated them.

 

Starting A New Project

You can click the New button to create a new project, the Open button to open an existing project or the Import button to open a compressed YYZ GameMaker project file or a legacy GameMaker: Studio 1.4 GMX project file. Both Open and Import will open the file explorer for you to browse to the project file you require.

 GameMaker is not completely backwards compatible with GameMaker: Studio 1.4 projects but imported 1.4 projects should still run, as obsolete functionality has been recreated for you automatically using compatibility scripts. For full details of the possible issues and the changes made to GML, please see the Help Center article Porting A GMS 1.4 Project To GameMaker, as well as the section of the manual on Obsolete Functions.

To create a new project simply click the button labelled New, which will open the Project Type and Template menu:

First select a project type, which should be Game unless you are making a Live Wallpaper or Game Strip for Opera GX.

After choosing your project type, select Blank or choose a template.

Then choose a name and save location for your project. If you selected a non-blank template, you may need to choose between GML Code and GML Visual versions.

Finally, click on Let's Go! to enter your new project.

 You can create your own template by opening a project, and exporting as a template.
