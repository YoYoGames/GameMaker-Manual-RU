Наша база знаний

В руководстве по GameMaker, которое вы сейчас читаете, содержится вся информация, необходимая для использования продукта и начала создания игр. Однако оно не содержит информации о настройке для различных платформ, для которых GameMaker может создавать игры, а также никаких руководств или помощи по различным расширениям, которые доступны для расширения функциональности для некоторых платформ. В нем также не содержится никакой последней информации, которая может быть актуальна для продукта или конкретных платформ и которую не удалось включить в это руководство.
