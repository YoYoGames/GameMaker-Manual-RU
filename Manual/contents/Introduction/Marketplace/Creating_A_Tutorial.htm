Создание учебника Marketplace

Создание учебника похоже на создание обычного пакета активов Marketplace , только теперь вам нужно добавить определенные включаемые файлы, а также включить начальную страницу учебника (и дополнительную страницу видео). Рабочая область учебника будет выглядеть примерно так:Как объяснялось выше для общих Marketplace активов, вам нужно выбрать издателя, затем дать учебнику имя и уникальный ID (обратный формат URL), а затем в выпадающем списке "тип" выбрать учебник. Это даст вам дополнительные возможности добавить базовый HTML-файл, с которого будет начинаться учебник, а также (по желанию) видеофайл и URL. Видеофайл должен использовать шаблон, предоставленный с сайта Marketplace, как и остальная часть учебника, который вы можете получить здесь. Когда пользователь загрузит учебник, видео будет загружено в отдельный фрейм на сайте IDE. Обратите внимание, что страницы, которые вы указываете для начальной страницы и страницы видео, являются относительными путями в Included Files, и вы можете открыть проводник, чтобы выбрать что-то другое, кроме тех, которые GameMaker заполняет поля по умолчанию.

На самом деле для создания учебника вам потребуется добавить в проект Included Files, и они должны быть добавлены очень специфическим образом, чтобы IDE правильно определил, какие файлы будут использоваться в учебнике. Основная структура папок показана на рисунке ниже:

Итак, у нас есть две основные папки:

  • TutorialContent: Все ваши файлы HTML, CSS для их оформления, изображения, необходимые для них (не изображения, необходимые для ресурсов, а те, которые будут отображаться в учебнике), а также шаблон видео, если требуется, помещаются сюда. У вас могут быть и подпапки, если вы хотите разделить учебник, например, на главы, но у вас должен быть как минимум один базовый файл HTML, который будет использоваться в качестве начальной страницы учебника.
  • TutorialResources: В этой папке есть еще две вложенные папки для изображений и звуковых ресурсов. Если ваш учебник требует от пользователя добавить sprite (например), и вы хотите включить его в учебник, то вы поместите ресурс изображения в папку Images. Эти папки импортируются в проект учебника, и когда пользователь открывает редактор sprite и хочет добавить изображение, путь по умолчанию для окна проводника будет указан в этом месте, что облегчает пользователю поиск необходимых ресурсов.

Чтобы ваш учебник работал в рамках IDE , вы должны иметь такую структуру папок, а также базовую стартовую страницу HTML с файлом CSS для стилизации. Существует ряд специальных ссылок, которые можно использовать в ваших обучающих проектах, чтобы перейти к части видео или открыть руководство по определенному разделу или функции, и все они приведены в официальных шаблонах учебников Marketplace.

ВАЖНО! Мы настоятельно рекомендуем вам загрузить шаблоны учебников с Marketplace, прежде чем пытаться создавать свои собственные учебники. Они содержат примеры всех важных частей HTML, а также CSS и изображения для воссоздания кнопок и меню в учебнике, что позволит вам беспокоиться о содержании, а не о том, как его отобразить.

После того, как вы установите включенные файлы и предоставите необходимую информацию, вы можете нажать кнопку Окей, и учебник будет загружен на сайт Marketplace, после чего откроется рабочее пространство в листинге актива, и вы сможете редактировать его и добавлять значки, текст, скриншоты и т.д.. После публикации актива учебника пользователи смогут найти его в разделе " Учебники" на Стартовой странице, где его можно скачать и установить.
