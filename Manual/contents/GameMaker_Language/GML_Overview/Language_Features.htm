Особенности языка

Язык GameMaker Language (GML) имеет ряд особенностей, которые являются частью основной структуры всех программ, написанных на этом языке. Каждая из них будет использоваться в любом написанном вами коде многократно, поэтому вы должны знать, что это такое и как они работают, прежде чем приступать к программированию проекта.

  • repeat
  • while
  • do / until
  • for
  • switch
  • break
  • continue
  • exit
  • with
  • return
  • begin / end
  • try / catch / finally
  • throw
  • new
  • delete
