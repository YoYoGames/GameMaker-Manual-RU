счет

Эта переменная является глобальной и используется для хранения числового значения, которое обычно используется для оценки игрока. Эта переменная предназначена только для поддержки старых проектов предыдущих версий GameMaker, и ее следует не использовать в новых проектах поскольку в будущем она может быть устаревшей.

Синтаксис:

score;

Возвращается:

Real (single precision floating point value)

Пример

switch (object_index)
{
    case obj_Enemy_Fighter:
        score += 10;
    break;

    case obj_Enemy_Mage:
        score += 25;
    break;

    case obj_Enemy_Boss:
        score += 100;
    break;
}
instance_destroy();

Приведенный выше код проверяет индекс объекта экземпляра, выполняющего код, с помощью a switchи затем добавляет различные суммы в переменную score в зависимости от того, какой это объект.
