аргумент

Эта переменная содержит массив, который используется вместе с переменной только для чтения argument_count в функциях или методах сценария. Каждая позиция массива содержит входное значение для функции и специально предназначена для использования в функциях с переменным аргументом.

Обратите внимание, что существует также ряд независимых переменных глобальной области видимости, которые также могут быть использованы в пользовательских функциях для ссылки на различные входные аргументы: argument0, argument1, argument2, и т.д... вплоть до argument15.

ПРИМЕЧАНИЕ: Это больше не является строго обязательным и предоставляется больше для поддержки наследия, чем для чего-либо еще. Все определяемые пользователем функции будут принимать переменные аргументы, и любой аргумент, передаваемый в функцию, который не является частью именованных входов, по умолчанию инициализируется как undefined, что можно проверить с помощью функции is_undefined().

Синтаксис:

argument[n]
argument1
argument2
...
argument15

Возвращается:

Variable (can be of any data type supplied to the function)

Пример:

function my_array()
{
    var i, arg;
    for (i = 0; i < 5; i += 1;)
    {
        if argument_count > i
        {
            arg[i] = argument[i]
        }
        else
        {
            arg[i] = -1;
        }
    }
}

Приведенный выше код использует переменную argument_count вместе с массивом argument для инициализации переменных и является примером того, как разрешить пользовательской функции принимать переменное количество аргументов.
