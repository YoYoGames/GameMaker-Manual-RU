Обзор GML Code

Этот раздел руководства содержит всю информацию, необходимую для понимания и использования GML Code. Функции выполнения на этом языке могут быть использованы для создания ваших игр и добавляются в объекты из программы Редактор объектовхотя он также может быть использован вместе с Визуальный GML. Ниже показано типичное изображение объекта с открытым редактором кода на событии:

Каждое событие имеет свою собственную вкладку в редакторе, и вы можете добавлять, редактировать или удалять код из них в любое время (для получения дополнительной информации о событиях см. События объекта). Сам код должен иметь базовую структуру и может содержать индексы ресурсов, переменные, функции, выражения, ключевые слова и т.д., о которых рассказывается в следующих разделах. Если вы новичок в программировании или переходите с Визуальный GML, рекомендуется начать со страницы Базовая структура кода а затем прочитать все остальные страницы этого раздела, проверяя код каждой из них в рамках GameMaker в самом GameMaker.

Вы можете использовать Feather для интеллектуального завершения кода и сообщения об ошибках в ваших сценариях GML Code.
