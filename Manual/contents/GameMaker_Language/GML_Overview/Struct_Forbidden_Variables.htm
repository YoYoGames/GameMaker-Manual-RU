Структура Запрещенные переменные

При создании struct можно использовать определенные встроенные переменные в качестве имен переменных-членов, например:

mystruct =
{
    object_index : obj_Player,
    speed : 3,
    image_blend : c_red
}

Однако таким образом можно использовать только локальные встроенные переменные и встроенные переменные в области видимости экземпляра, которым можно присваивать значения, как обычным переменным-членам struct. Использование встроенных переменных глобальной области видимости запрещено и приведет к проблемам в вашей игре. Ниже приведен полный список этих переменных, чтобы вы знали, каких из них следует избегать.
