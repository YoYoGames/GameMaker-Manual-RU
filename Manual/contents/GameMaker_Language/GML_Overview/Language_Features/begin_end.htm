GameMaker Language позволяет использовать ключевые слова begin и end вместо более привычных фигурных скобок {} при создании блоков кода. Пример кода ниже показывает, как это работает:

if (!visible)
    begin
    exit;
    end
while (place_meeting(x, y))
    begin
    x -= lengthdir_x(1, direction - 180);
    y -= lengthdir_y(1, direction - 180);
    end

Обратите внимание, что использование этих ключевых слов не является типичным и предоставляется как часть языка больше для поддержки наследия, чем для чего-либо еще, и в любой момент в будущем они могут быть устаревшими.
