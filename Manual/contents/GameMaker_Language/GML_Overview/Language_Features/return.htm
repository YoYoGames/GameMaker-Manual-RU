Оператор return имеет следующий синтаксис:

return (<expression>)

Оператор return используется только в функциях и методах сценария, он используется для возврата значения из функции, которое будет использоваться в дальнейшем коде или вызовах функций. Следует отметить, что выполнение функции заканчивается на операторе return , а это значит, что любой код, который появится после вызова return, не будет выполнен. Вот небольшой пример скрипта под названием sqr_calc, который вычисляет квадрат любого переданного ему значения и включает в себя отлов ошибок в случае, если переданный ему параметр не является действительным числом:

/// @function            sqr_calc(val);
/// @param {real}  val   The value to calculate the square of
/// @description         Calculate the square of the given value

function sqr_calc(value)
{
    if (!is_real(value))
    {
        return 0;
    }
    else
    {
        return (value * value);
    }
}

Чтобы вызвать функцию из кода, просто используйте ее так же, как и при вызове функций времени выполнения - то есть напишите имя функции со значениями параметров в круглых скобках. Таким образом, приведенная выше функция будет вызвана следующим образом:

if keyboard_check_pressed(vk_enter)
{
    val = scr_sqr(amount);
}

Здесь переменная val будет либо равна 0 (поскольку переменная amount не является действительным числом), либо суммарному значению для value * value.
