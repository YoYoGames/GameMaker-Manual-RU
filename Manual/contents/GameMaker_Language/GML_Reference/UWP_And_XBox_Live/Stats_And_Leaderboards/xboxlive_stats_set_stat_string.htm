xboxlive_stats_set_stat_string

Эта функция может быть использована для установки stat в строку для заданного ID пользователя. Вы вводите идентификатор пользователя, возвращаемый (например) функцией xboxlive_get_user(), затем строку stat для установки(если строка stat еще не существует, то будет создана новая stat и установлена в заданное значение) и затем строку, в которую нужно установить stat. Обратите внимание, что имя stat должно быть только буквенно-цифровым, без символов и пробелов, а задаваемое значение строки может содержать пробелы и специальные символы.

При установке строки статистики любая предыдущая строка будет отменена, поэтому вы сами должны определить, следует ли обновить строку статистики или нет, сравнив текущую строку статистики с новой перед ее установкой.

Функция вернет -1, если произошла ошибка или идентификатор пользователя недействителен, или любое другое значение, если функция была вызвана успешно.

Синтаксис:

xboxlive_stats_set_stat_string(user_id, stat, value);

Аргумент Тип Описание
user_id Идентификатор пользователя Xbox The user ID pointer to set the stat for
stat Строка The statistic to set (a string)
value Строка The value to set the stat to (a string)

Возвращается:

Real

Пример:

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

Приведенный выше код устанавливает статистику "LevelName" в строку, хранящуюся в данной глобальной переменной.
