xboxlive_достижение_загрузки_друзей

Эта функция отправит запрос на сервер для получения информации обо всех друзьях вошедших в систему пользователей и вызовет обратный вызов Social Asynchronous Event, который содержит async_load карту, заполненную соответствующими парами ключ/значение. Ключ id этого DS Map используется для идентификации правильного обратного вызова (для любого асинхронного события может быть более одной функции запуска) и будет сопряжен с константой xboxlive_achievement_friends_info, а также рядом других пар ключ/значение для каждого друга. Точное содержимое карты показано ниже:

  • "id" - Для этой функции это должно быть xboxlive_achievement_friends_info
  • "FriendN" - Имя друга, где "N" - целочисленное значение, соответствующее его положению в списке друзей.
  • "FriendidN" - Уникальный идентификатор пользователя друга, "N".

Синтаксис:

xboxlive_achievement_load_friends();

Расширенный пример:

Следующий код, вероятно, будет вызываться после того, как игрок войдет в свой игровой аккаунт, чтобы получить список всех друзей этого пользователя:

xboxlive_achievement_load_friends();

Это отправит запрос на получение информации о друзьях пользователя и сгенерирует асинхронный обратный вызов со специальной картой async_load DS, содержащей следующие данные:

var ident = ds_map_find_value(async_load, "id");
if (ident == xboxlive_achievement_friends_info)
{
    var numfriends = ds_map_find_value(async_load, "numfriends");
    global.numfriends = numfriends;
    for(var i=0; i < numfriends; i++;)
    {
        global.friendname[i] = ds_map_find_value(async_load, "Friend" + string(i));
        global.friendid[i] = ds_map_find_value(async_load, "Friendid" + string(i));
    }
}

Приведенный выше код проверяет возвращаемый DS Map в Social Asynchronous Event и, если его "id" совпадает с проверяемой константой, он затем циклически просматривает карту, сохраняя все различные значения в нескольких массивах.
