xboxlive_stats_get_stat_names

Эта функция может быть использована для получения всех определенных статистик из менеджера статистики для данного пользователя. Вы вводите идентификатор пользователя, который возвращается (например) функцией xboxlive_get_user(), и функция вернет массив строк, содержащих статистику для пользователя. Если произошла ошибка или у пользователя нет статистики, массив будет возвращен, но количество элементов будет равно 0.

Синтаксис:

xboxlive_stats_get_stat_names(user_id);

АргументТип Описание
user_id The user ID pointer to get the stat names of

Возвращается:

Пример:

var _stat_str = xboxlive_stats_get_stat_names(user_id);
for (var i = 0; i < array_length(_stat_str); ++i;)
{
    xboxlive_stats_delete_stat(user_id, _stat_str[i]);
}

Приведенный выше код извлекает все определенные статистики для указателя ID пользователя, хранящегося в данной переменной, а затем проходит по массиву и удаляет указанную статистику из менеджера статистик.
