xboxlive_stats_set_stat_real

Эта функция может быть использована для установки значения stat для заданного ID пользователя. Вы вводите ID пользователя, возвращенный (например) функцией xboxlive_get_user(), затем строку stat для установки(если строка stat еще не существует, то будет создана новая stat и установлена на заданное значение) и значение (вещественное) для установки stat. Обратите внимание, что имя stat должно быть только буквенно-цифровым, без символов и пробелов.

При установке значения стата любое предыдущее значение будет отменено, поэтому вы сами должны определить, нужно ли обновлять значение стата или нет (т.е. проверить, что высокий результат действительно является самым высоким), сравнив текущее значение стата с новым, прежде чем устанавливать его.

Функция вернет -1, если произошла ошибка или идентификатор пользователя недействителен, или любое другое значение, если функция была вызвана успешно.

Синтаксис:

xboxlive_stats_set_stat_real(user_id, stat, value);

АргументТип Описание
user_id The user ID pointer to set the stat for
stat The statistic to set (a string)
value The value to set the stat to (a real)

Возвращается:

Пример:

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

Приведенный выше код проверяет переменную, и если это true, то проверяется значение статистики "PercentDone". Если это значение меньше 100, то генерируется значение и статистика устанавливается на это значение.
