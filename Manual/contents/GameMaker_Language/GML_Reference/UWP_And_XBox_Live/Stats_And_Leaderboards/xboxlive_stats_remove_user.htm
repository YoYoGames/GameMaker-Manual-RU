xboxlive_stats_remove_user

Эта функция может быть использована для удаления (снятия с регистрации) данного пользователя из менеджера статистики, выполняя сброс данных статистики на живой сервер. Согласно документации XBox, игра не обязана удалять пользователя из менеджера статистики, так как XBox OS все равно периодически очищает статистику. Вы передаете идентификатор пользователя, который возвращается (например) функцией xboxlive_get_user(), и функция вернет -1, если произошла ошибка или если ID пользователя недействителен, или любое другое значение, если функция была вызвана успешно.

ПРИМЕЧАНИЕ: Удаление пользователя может привести к ошибке, если статистика, установленная для пользователя, недействительна (например, имена статистики содержат неалфавитные цифровые символы).

Функция создаст обратный вызов, который вызовет системное асинхронное событие. Это событие будет иметь специальную карту DS async_load которая затем должна быть разобрана на следующие ключи:

  • "id" - Будет поддерживать постоянство achievement_stat_event
  • "eventname" - Будет содержать строку "LocalUserRemoved".
  • "userid" - Идентификатор пользователя, связанный с запросом
  • "error" - 0 в случае успеха, другое значение, если произошла ошибка
  • "errormessage" - Строка с сообщением об ошибке, если таковое имеется

Обратите внимание, что если вы хотите сбросить данные статистики на живой сервер в любое время без удаления пользователя, вы можете использовать функцию xboxlive_stats_flush_user().

Синтаксис:

xboxlive_stats_remove_user(user_id);

АргументТип Описание
user_id The user ID (a pointer) to remove

Возвращается:

Пример:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_remove_user(user_id[i]);
}

Приведенный выше код просматривает сохраненные идентификаторы учетных записей пользователей и удаляет (снимает с регистрации) каждого пользователя из менеджера статистики.
