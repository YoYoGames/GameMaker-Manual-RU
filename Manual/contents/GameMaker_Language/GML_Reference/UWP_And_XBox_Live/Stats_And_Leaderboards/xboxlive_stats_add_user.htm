xboxlive_stats_add_user

Эта функция может быть использована для добавления заданного указателя ID пользователя в менеджер статистики. Это должно быть сделано до использования любой другой функции статистики для автоматической синхронизации игры с сервером XBox Live и получения последних значений. Вы указываете ID пользователя, который возвращается (например) функцией xboxlive_get_user(), и функция вернет -1, если произошла ошибка или если ID пользователя недействителен, или любое другое значение, если функция была вызвана успешно.

Функция создаст обратный вызов, который вызовет системное асинхронное событие. Это событие будет иметь специальную карту DS async_load которая затем должна быть разобрана на следующие ключи:

  • "id" - Будет поддерживать постоянство achievement_stat_event
  • "eventname" - Будет содержать строку "LocalUserAdded".
  • "userid" - Идентификатор пользователя, связанный с запросом
  • "error" - 0 в случае успеха, другое значение, если произошла ошибка
  • "errormessage" - Строка с сообщением об ошибке, если таковое имеется

Синтаксис:

xboxlive_stats_add_user(user_id);

АргументТип Описание
user_id The user ID (a pointer) to add

Возвращается:

Пример:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_add_user(user_id[i]);
}

Приведенный выше код получит количество доступных учетных записей пользователей, затем пройдется по ним и присвоит идентификатор учетной записи массиву, а также зарегистрирует пользователя в менеджере статистики.
