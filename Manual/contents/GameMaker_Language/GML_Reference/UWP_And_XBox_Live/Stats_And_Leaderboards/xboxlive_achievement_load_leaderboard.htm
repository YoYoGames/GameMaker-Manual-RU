xboxlive_достижение_загрузки_лидерборд

Эта функция отправляет запрос на сервер для получения информации о заданной таблице лидеров. Таблица лидеров должна быть создана ранее на панели разработчика для вашей игры, а заданный ID - это тот, который вы должны указать в качестве первого аргумента функции. Это вызовет обратный вызов Social Asynchronous Event, который содержит async_load карту, заполненную соответствующими парами ключ/значение. Ключ id этого DS Map используется для идентификации правильного обратного вызова (для любого асинхронного события может быть более одной функции запуска), и будет сопряжен с константой xboxlive_achievement_leaderboard_info, а также рядом других пар ключ/значение для каждого игрока. Точное содержимое карты показано ниже:

  • "id" - Для этой функции это должно быть xboxlive_achievement_leaderboard_info
  • "leaderboardid" - Уникальный ID для таблицы лидеров, определенный на приборной панели провайдера.
  • "numentries" - Количество записей в таблице лидеров, которые вы получили.
  • "PlayerN" - Имя игрока, где "N" - целочисленное значение, соответствующее его позиции в списке лидеров.
  • "PlayeridN" - Уникальный идентификатор пользователя игрока, "N".
  • "RankN" - Ранг игрока "N" в таблице лидеров.
  • "ScoreN" - счет игрока "N".

Вы можете указать точную часть таблицы лидеров для показа, задав минимальное и максимальное значение индекса, а также запросить фильтрацию результатов, чтобы получить только результаты ваших друзей или результаты всех игроков, используя следующие константы фильтра:

Постоянный фильтр достижений Xbox Live
Постоянная Описание
xboxlive_achievement_filter_friends_only  Получайте только оценки игроков-друзей.
xboxlive_achievement_filter_all_players  Получите все результаты для всех игроков.

Синтаксис:

xboxlive_achievement_load_leaderboard(ident, minindex, maxindex, filter)

Аргумент Тип Описание
ident Строка The unique ID of the leaderboard as shown on the developer dashboard.
minindex Реально The starting index value to get the leaderboard data from.
maxindex Реально The maximum index value to get the leaderboard data from.
filter Постоянный фильтр достижений Xbox Live Whether to filter the results to those that are on your friends list or to all players (see the Description above)

Расширенный пример:

Следующий код, вероятно, будет вызываться после того, как игрок войдет в свой игровой аккаунт, чтобы получить список всех друзей, занимающих лидирующие позиции в таблице лидеров в игре:

xboxlive_achievement_load_leaderboard("CgkIs9_51u0PEAIQBw", 1, 100, xboxlive_achievement_filter_friends_only);

Это отправит запрос на получение информации о данной таблице лидеров и сгенерирует асинхронный обратный вызов со специальной картой async_load DS, содержащей следующие данные:

var ident = ds_map_find_value(async_load, "id");
if (ident == xboxlive_achievement_leaderboard_info)
{
    var lbid = ds_map_find_value(async_load,"leaderboardid");
    if lbid == "CgkIs9_51u0PEAIQBw" || lbid == "leaderboard1id"
    {
        global.numentries = ds_map_find_value(async_load,"numentries");
        for(var i = 0; i < numentries; i++;)
        {
            global.playername[i] = ds_map_find_value(async_load, "Player" + string(i));
            global.playerid[i] = ds_map_find_value(async_load, "Playerid" + string(i));
            global.playerrank[i] = ds_map_find_value(async_load, "Rank" + string(i));
            global.playerscore[i] = ds_map_find_value(async_load, "Score" + string(i));
        }
    }
}

Приведенный выше код проверяет возвращаемый DS Map в Social Asynchronous Event, и если его "id" совпадает с проверяемой константой, он затем проверяет, что отправляемая таблица лидеров соответствует исходному вызову функции, после чего проходит по карте и сохраняет все различные значения в нескольких массивах.
