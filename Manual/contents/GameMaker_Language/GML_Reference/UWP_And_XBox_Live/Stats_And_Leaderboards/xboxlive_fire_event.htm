xboxlive_fire_event

Эта функция может быть использована для запуска события stat. Аргумент "имя_события" - это имя события, которое должно быть запущено, как определено в консоли XDP для вашей игры, а следующие дополнительные параметры также будут зависеть от того, что у вас настроено для stat. Функция вернет 0, если событие было отправлено (и должно быть получено/обработано сервером) или -1, если произошла ошибка (т.е. ваше событие не было настроено, т.к. в файле манифеста событий, включенном в проект, указано другое число).

Синтаксис:

xboxlive_fire_event(event_name, [stat_parameters...]);

Аргумент Тип Описание
event_name Строка The name of the event to fire.
stat_parameters Переменная OPTIONAL Дополнительные параметры для добавления к событию.

Возвращается:

Real

Пример:

var _user = xboxlive_get_user(0);
var _uid = xboxlive_user_id_for_user(_user);
result = xboxlive_fire_event("PlayerSessionStart", _uid, global.guid_str, 0, 42, 42);

Приведенный выше код получает некоторую информацию о пользователе, затем запускает событие stat вместе с этими (и дополнительными) данными.
