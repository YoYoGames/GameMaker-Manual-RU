xboxlive_achievement_show_achievements

Эта функция откроет страницу достижений для выбранной платформы. Обратите внимание, что это асинхронная функция, то есть ваша игра будет продолжать работать в фоновом режиме, пока отображается страница достижений. Поэтому следует быть осторожным при ее использовании и обязательно ставить игру на паузу или разрешать ее показ только в тех местах вашей игры, где она не будет мешать игровому процессу.

Синтаксис:

xboxlive_achievement_show_achievements()

Возвращается:

Пример:

if achievements_enabled
{
    global.Pause = true;
    instance_create_layer(0, 0, "Controllers", obj_Pause);
    xboxlive_achievement_show_achievements();
}

Приведенный выше код проверяет пользовательскую переменную, чтобы узнать, включены ли достижения, и если да, то устанавливает глобальную переменную и создает экземпляр контроллера "pause" перед вызовом функции для показа достижений на экране.
