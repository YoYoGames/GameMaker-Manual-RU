xboxlive_stats_get_stat

Эта функция может быть использована для получения одного значения статистики из менеджера статистики для данного пользователя. Вы вводите идентификатор пользователя, возвращаемый (например) функцией xboxlive_get_user(), а затем строку stat, определенную при ее создании с помощью одной из функций xboxlive_stats_set_stat_*. Возвращаемое значение может быть либо строкой, либо вещественным числом (в зависимости от проверяемого stat), либо константой GML undefined , если stat не существует или произошла ошибка.

Синтаксис:

xboxlive_stats_get_stat(user_id, stat);

АргументТип Описание
user_id The user ID pointer to get the stat names of
stat The statistic to set (a string)

Возвращается:

Real, String or undefined

Пример:

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

Приведенный выше код проверяет переменную, и если это true, то проверяется значение статистики "PercentDone". Если это значение меньше 100, то генерируется значение и статистика устанавливается на это значение.
