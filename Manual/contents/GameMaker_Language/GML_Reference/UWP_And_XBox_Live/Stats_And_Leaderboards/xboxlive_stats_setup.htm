xboxlive_stats_setup

Эта функция должна быть вызвана до того, как вы сможете использовать любые другие функции Xbox Live stat, и просто инициализирует необходимые библиотеки в системе. Аргумент "user_id" - это необработанный ID пользователя, возвращаемый функцией xboxlive_get_user()а "service_config" и "title_id" - уникальный ID вашей игры в Xbox Live Dev Center.

Эта функция используется только для статистики, основанной на событиях (2013).

Синтаксис:

xboxlive_stats_setup(user_id, service_config_id, title_id);

Аргумент Тип Описание
user_id Идентификатор пользователя Xbox The ID pointer of the user to check.
service_config_id Строка This is the config_id string for the game.
title_id Строка The unique ID for your game on the Xbox Dev Center.

Возвращается:

N/A

Пример:

var user = xboxlive_get_user(0);
xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6);

Приведенный выше код инициализирует систему статистики для заданного идентификатора пользователя.
