xboxlive_read_player_leaderboard

Функция возвращает информацию о таблице лидеров для заданного имени пользователя. Таблица лидеров должна быть создана ранее на приборной панели XDP для вашей игры, а вызов функции вызовет обратный вызов Social Asynchronous Event, который содержит async_load карту, заполненную соответствующими парами ключ/значение. Ключ "id" этой карты DS используется для идентификации правильного обратного вызова (для любого асинхронного события может быть более одной запускающей функции), и IT будет сопряжен с константой achievement_leaderboard_info, а также рядом других пар ключ/значение. Точное содержимое карты показано ниже:

  • "id" - Для этой функции это должно быть achievement_leaderboard_info
  • "leaderboardid" - Уникальный ID для доски лидеров, определенный на приборной панели провайдера.
  • "numentries" - Количество записей в таблице лидеров, которые вы получили.
  • "PlayerN" - Имя игрока, где "N" - целочисленное значение, соответствующее его позиции в списке участников таблицы лидеров.
  • "PlayeridN" - Уникальный идентификатор пользователя игрока, "N".
  • "RankN" - Ранг игрока "N" в таблице лидеров.
  • "ScoreN" - Счет игрока "N".


Функция требует, чтобы вы указали одну из следующих констант для установки свойств фильтра:

Постоянная Описание
xboxlive_achievement_filter_all_players Получите все баллы.
xboxlive_achievement_filter_friends_only Получить только оценки друзей в порядке возрастания.
xboxlive_achievement_filter_favorites_only Получение только избранных оценок в порядке возрастания
xboxlive_achievement_filter_friends_alt Получить оценки только друзей в порядке убывания.
xboxlive_achievement_filter_favorites_alt Получение только избранных оценок в порядке убывания.

Синтаксис:

xboxlive_read_player_leaderboard(leaderboard_name, user_name, num_items, friend_filter);

Аргумент Тип Описание
leaderboard_name Строка The name of the leaderboard to read, as set up on XDP.
user_name Идентификатор пользователя Xbox The ID of the user to read from.
num_items Реально The number of items to retrieve.
friend_filter Постоянный фильтр достижений Xbox Live One of the filter constants (see the description, above).

Возвращается:

Real

Пример:

var _user = xboxlive_user_for_pad(0);
var _uid = xboxlive_user_id_for_user(_user);
xboxlive_read_player_leaderboard("MyLeaderboard", _uid, 10, achievement_filter_all_players);

Приведенный выше код получает идентификатор пользователя и затем использует его для запроса всех данных игрока для первых 10 мест заданной таблицы лидеров.
