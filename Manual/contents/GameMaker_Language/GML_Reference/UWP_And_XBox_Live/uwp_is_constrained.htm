uwp_is_constrained

Эта функция возвращает true, если игра работает в режиме ограниченных ресурсов, или false, если игра работает нормально. Режим ограниченных ресурсов может быть активирован, когда игра не имеет фокуса, и в этом режиме игра сокращает ресурсы GPU. Обычно реакцией на это должна быть пауза в игре (поскольку пользователь больше не имеет контроля). Кроме того, если в этом режиме наблюдаются замедления, стоит отключить все особо нагружающие игру эффекты. Обратите внимание, что в отличие от uwp_is_suspending() эта функция возвращает значение true постоянно, пока игра находится в режиме ограничения (а не только один раз, когда это происходит впервые).

Синтаксис:

uwp_is_constrained();

Возвращается:

Пример:

if xboxliveis_constrained()
{
    if !instance_exists(obj_PauseMenu)
    {
        instance_create(0,0,obj_PauseMenu);
    }
}

Приведенный выше код проверяет, является ли приложение ограниченным, и если да, то создает экземпляр объекта (используемого для приостановки игры).
