xboxlive_matchmaking_join_invite

Эта функция может быть вызвана для обработки приглашения на присоединение после того, как вы успешно выполнили вызов xboxlive_matchmaking_start(). Вы передаете локальный UserID и значения приглашения (ID и ID хоста) которые были возвращены при получении приглашения, а также имя шаблона сессии, который вы создали (строка). Затем пользователь присоединится к сессии, в которую он был приглашен, и вы получите социальные асинхронные события как если бы вы вызвали xboxlive_matchmaking_find().

Синтаксис:

xboxlive_matchmaking_join_invite(user_who_received_invite, invitation_id, invitation_host, sda_template_name);

АргументТип Описание
user_who_received_invite The local user ID pointer.
invitation_id The invitation ID.
invitation_host The invitation host ID.
sda_template_name The session template name.

Возвращается:

Пример:

var userID = xboxlive_user_for_pad(global.PadIndex);
xboxlive_matchmaking_join_invite(userID, global.InviteID, global.InviteHost, global.SessionTemplate);

Приведенное выше действие получает идентификатор пользователя для данного геймпада, а затем пытается присоединиться к сессии, в которую он был приглашен.
