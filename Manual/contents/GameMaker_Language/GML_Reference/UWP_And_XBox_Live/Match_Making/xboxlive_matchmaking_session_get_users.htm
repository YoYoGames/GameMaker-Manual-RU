xboxlive_matchmaking_session_get_users

Эта функция создаст и заполнит карту DS деталями пользователей в указанной сессии, или -1, если произошла ошибка. Вы можете получить указатель идентификатора сессии из async_load DS map, который возвращается в асинхронном социальном событии, когда вы создали или нашли сессию (см. xboxlive_matchmaking_create() для более подробной информации). Возвращаемая карта DS будет содержать следующие пары ключ/значение:

  • "num_results" - содержит количество пользователей в сессии
  • "userId<index>" - содержит ID пользователя (<index>- это значение от 0 до "num_results" - 1)
  • "userIsOwner<index>" - содержит 1, если пользователь является хозяином, иначе 0 (<index>- значение от 0 до "num_results" - 1)

Синтаксис:

xboxlive_matchmaking_session_get_users(session_id);

АргументТип Описание
session_id The session ID pointer to use

Возвращается:

Async Request ID

Пример:

var session_map = xboxlive_matchmaking_session_get_users(global.SessionID);
if (session_map != -1)
{
    for (var i = 0; i < session_map[? "num_results"]; i++)
    {
        if (session_map[? "userIsOwner" + string(i)] == 1)
        {
            global.OwnerID = session_map[? "userId" + string(i)];
        }
    }
}

Вышеприведенные действия позволят получить пользовательские данные для всех пользователей, участвующих в сеансе подбора, и установить глобальную переменную на ID владельца сеанса.
