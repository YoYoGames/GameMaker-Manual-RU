xboxlive_matchmaking_send_invites

Эта функция вызывает системный диалог Xbox Live, позволяющий пользователю отправлять приглашения в свою игру. Вы указываете локального пользователя, который должен управлять диалогом (и список друзей которого будет использоваться диалогом), уникальное значение ID сессии для сессии, в которую они хотят пригласить людей, и текст по умолчанию, который будет отображаться в диалоге. Идентификатор сессии можно получить из async_load DS-карты, созданной функцией xboxlive_matchmaking_create().

Функция вернет -1, если аргументы недействительны или пользователь не зарегистрирован в Xbox Live, или значение больше 0 в противном случае.

Синтаксис:

xboxlive_matchmaking_send_invites(from_user, session_to_invite_into, text_to_add_to_invite);

АргументТип Описание
from_user The local user ID for controlling the dialogue.
session_to_invite_into The unique session ID for the session to join.
text_to_add_to_invite A short text to add to the invitation.

Возвращается:

Пример:

if gamepad_button_check_pressed(0, gp_face1)
{
    var _uid = xboxlive_user_for_pad(0);
    xboxlive_matchmaking_send_invites(_uid, global.session_ID, "Come and join my game!");
}

Приведенный выше код проверяет нажатие кнопки геймпада, и если оно обнаружено, то открывается диалог Xbox Live для приглашения в сессию.
