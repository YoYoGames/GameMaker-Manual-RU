xboxlive_gamedisplayname_for_user

С помощью этой функции вы можете получить отображаемое имя - в виде строки - из указателя ID пользователя. Обратите внимание, что если пользователь локальный, то возвращаемое значение будет просто строкой его отображаемого имени, но если это удаленный пользователь, то строка будет его геймертагом, который можно использовать, например, для отключения/включения звука в голосовом чате.

Синтаксис:

xboxlive_gamedisplayname_for_user(user_id);

АргументТип Описание
user_id The ID pointer of the user to check.

Возвращается:

Пример:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    user_name[i] = xboxlive_gamedisplayname_for_user(user_id[i]);
}

Приведенный выше код получает указатель идентификатора пользователя для каждой доступной учетной записи пользователя, а затем сохраняет указатель вместе с отображаемым именем в двух массивах.
