xboxlive_спонсор_для_пользователя

Эта функция извлекает ID спонсора данного пользователя.

Синтаксис:

xboxlive_sponsor_for_user(user_id);

Аргумент Тип Описание
user_id Идентификатор пользователя Xbox The ID pointer of the user to check.

Возвращается:

Xbox Sponsor ID

Пример:

var _uid = xboxlive_user_for_pad(0);
sponsor = xboxlive_sponsor_for_user(_uid);

Приведенный выше код получает указатель ID пользователя для пользователя, назначенного на слот геймпада [0], а затем использует этот ID для получения ID спонсора пользователя.
