xboxlive_generate_player_session_id

Эта функция генерирует уникальную строку (содержащую GUID) для текущей сессии игрока. Затем она может быть использована с помощью функции xboxlive_fire_event().

Синтаксис:

xboxlive_generate_player_session_id();

Возвращается:

Пример:

if !xboxlive_user_is_signed_in()
{
    if !xboxlive_user_is_signing_in()
    {
        xboxlive_show_account_picker();
    }
}
else
{
    global.GamerTag = xboxlive_gamertag_for_user();
}

Приведенный выше код проверяет, зарегистрирован ли пользователь в XBox Live, и если он зарегистрирован, то код получит его геймертаг и сохранит его в глобальной переменной для дальнейшего использования. Если пользователь не зарегистрирован, то код проверяет, находится ли он в процессе регистрации, в этом случае ничего не происходит, а если нет, то открывается окно выбора учетной записи, предлагая пользователю зарегистрироваться.
