xboxlive_user_is_remote

Эта функция проверяет заданный идентификатор пользователя и возвращает true, если проигрыватель является удаленным, или false в противном случае.

Синтаксис:

xboxlive_user_is_remote(user_id);

АргументТип Описание
user_id The ID pointer of the user to check.

Возвращается:

Пример:

global.PlayerNum = 0;
global.Player_ID = array_create();
for(var i = 0; i < xboxlive_get_user_count(); ++i)
{
    var _uid = xboxlive_get_user(i);
    if (xboxlive_user_is_active(_uid))
    {
        if (!xboxlive_user_is_remote(_uid))
        {
            global.Remote_Player_ID[global.PlayerNum++] = _uid;
        }
    }
}

Приведенный выше код перебирает все учетные записи пользователей, а затем проверяет, активны ли они. Если они активны, их идентификатор пользователя добавляется в глобальный массив, только если они не являются удаленными пользователями.
