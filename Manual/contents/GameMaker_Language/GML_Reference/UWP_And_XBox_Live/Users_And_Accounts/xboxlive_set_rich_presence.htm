xboxlive_set_rich_presence

Эта функция устанавливает строку богатого присутствия для данного пользователя. Строка Rich Presence отображает игровую активность пользователя после названия игры, в которую он играет, через дефис. Эта строка отображается под Gamertag игрока в списке "Друзья и клубы", а также в профиле пользователя Xbox Live.

При использовании этой функции вам необходимо предоставить указатель User ID для пользователя, а затем вы можете отметить пользователя как активного в игре в данный момент или нет (используя true/false). Если установлено значение false, то к строке присутствия будет добавлено "/afk" или что-то подобное. Следующим аргументом является ID строки богатого присутствия, который нужно показать, и, наконец, вы можете (опционально) предоставить строку service_config_id. Этот аргумент необязателен, и если он не указан, то используется SCID, указанный в опциях игры Xbox.

Синтаксис:

xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id]);

Аргумент Тип Описание
user_id Идентификатор пользователя Xbox The ID pointer of the user to check.
is_user_active Булево Flag the user as active or not.
rich_presence_string Строка The rich presence string ID to use (as defined in the Partner Center - max 50 characters)
service_config_id Строка OPTIONAL Это строка config_id для игры.

Возвращается:

N/A

Пример:

var pad_uid = xboxlive_user_for_pad(0);
xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge");

Приведенный выше код получает указатель идентификатора пользователя, назначенного на слот [0] геймпада, а затем устанавливает строку присутствия для этого пользователя.
