xboxlive_user_is_active

С помощью этой функции вы можете проверить, есть ли заданный идентификатор пользователя в списке пользователей, использующих консоль в данный момент, и функция вернет true, если есть, или false в противном случае. Вы можете получить указатель идентификатора пользователя с помощью функции xboxlive_get_user().

Синтаксис:

xboxlive_user_is_active(user_id);

АргументТип Описание
user_id The ID pointer of the user to check.

Возвращается:

Пример:

global.PlayerNum = 0;
global.Player_ID = array_create();
for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    var _uid = xboxlive_get_user(i);
    if (xboxlive_user_is_active(_uid))
    {
        global.Player_ID[global.PlayerNum++] = _uid;
    }
}

Приведенный выше код перебирает все учетные записи пользователей, а затем проверяет, активны ли они. Если они активны, их идентификатор пользователя добавляется в глобальный массив.
