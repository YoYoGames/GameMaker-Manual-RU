xboxlive_репутация_для_пользователя

С помощью этой функции вы можете получить оценку репутации Xbox Live для заданного указателя ID пользователя.

Синтаксис:

xboxlive_reputation_for_user(requesting_user_id);

АргументТип Описание
requesting_user_id The user ID (a pointer) of the user to check

Возвращается:

Real

Пример:

var _a = 0;
var _num = xboxlive_get_user_count();
for (var i = 0; i < _num; ++i)
{
    var _uid = xboxlive_get_user(i);
    if (_uid != pointer_null)
    {
        global.UserName[_a] = xboxlive_gamedisplayname_for_user(_uid);
        global.UserScore[_a] = xboxlive_gamerscore_for_user(_uid);
        global.UserRep[_a] = xboxlive_reputation_for_user(_uid);
        global.UserAvatar[_a] = xboxlive_sprite_add_from_gamerpicture(_uid, 256, 0, 0);
        ++a;
    }
}

Приведенный выше код перебирает всех зарегистрированных пользователей и сохраняет их данные о геймерах в различных глобальных массивах.
