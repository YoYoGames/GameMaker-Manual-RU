xboxlive_get_user_count

С помощью этой функции вы можете узнать общее количество пользователей, которые в настоящее время зарегистрированы в системе XBox. Возвращаемое значение будет целым числом, от 0 до N, где N - количество зарегистрированных пользователей -1.

Синтаксис:

xboxlive_get_user_count();

Возвращается:

Пример:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

Приведенный выше цикл перебирает всех зарегистрированных пользователей и сохраняет их уникальный идентификационный указатель в массиве.
