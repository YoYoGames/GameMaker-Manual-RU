xboxlive_show_profile_card_for_user

С помощью этой функции вы можете запросить, чтобы XBox показал карточку профиля для целевого указателя ID пользователя. Функция требует как ID-указатель пользователя, который запрашивает информацию, так и ID-указатель пользователя, для которого необходимо получить карточку профиля.

Синтаксис:

xboxlive_show_profile_card_for_user(requesting_user_id, target_user_id);

АргументТип Описание
requesting_user_id The user ID (a pointer) of the requesting user
target_user_id The user ID (a pointer) of the user to get the profile card of

Возвращается:

Пример:

if (gamepad(0, gp_start))
{
    xboxlive_show_profile_card_for_user(user[0], user[1]);
}

Приведенный выше код проверяет нажатие кнопки геймпада и, если таковое обнаружено, показывает карточку профиля для данного пользователя.
