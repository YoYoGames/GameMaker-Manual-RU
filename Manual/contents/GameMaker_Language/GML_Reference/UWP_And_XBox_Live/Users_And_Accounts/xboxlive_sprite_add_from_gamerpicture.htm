xboxlive_sprite_add_from_gamerpicture

С помощью этой функции вы можете получить изображение геймера для заданного указателя ID пользователя. Функция работает асинхронно и будет вызывать асинхронное событие Image Loaded, чтобы сообщить вам о завершении работы функции (как в случае с HTML5/URL-вызовов к sprite_add()). Асинхронное событие Image Loaded будет иметь карту DS, хранящуюся во встроенной переменной async_load. Карта содержит следующую информацию:

  • "filename": Полный путь к запрашиваемому файлу.
  • "id": Идентификатор загруженного ресурса. Он будет совпадать с переменной, которой вы присвоили ресурс.
  • "status": Возвращает значение меньше 0 в случае ошибки.

ПРИМЕЧАНИЕ: Когда вы динамически загружаете sprite в вашу игру во время выполнения, вы должны не забыть удалить его снова с помощью функции sprite_delete() когда он больше не нужен, иначе есть риск утечки памяти, что приведет к замедлению и, в конечном итоге, к краху вашей игры.

Синтаксис:

xboxlive_sprite_add_from_gamerpicture(user_id, imagesize, xorig, yorig);

АргументТип Описание
user_id The user ID (a pointer) to get the gamer picture for
imagesize Size in pixels of the sprite to be returned
xorig Indicate the x position of the origin in the sprite
yorig Indicate the y position of the origin in the sprite

Возвращается:

Async Request ID

Пример:

var _a = 0;
var _num = xboxlive_get_user_count();
for (var i = 0; i < _num; ++i)
{
    var _uid = xboxlive_get_user(i);
    if (_uid != pointer_null)
    {
        global.UserName[_a] = xboxlive_gamedisplayname_for_user(_uid);
        global.UserScore[_a] = xboxlive_gamerscore_for_user(_uid);
        global.UserRep[_a] = xboxlive_reputation_for_user(_uid);
        global.UserAvatar[_a] = xboxlive_sprite_add_from_gamerpicture(_uid, 256, 0, 0);
        ++a;
    }
}

Приведенный выше код перебирает всех зарегистрированных пользователей и сохраняет их данные о геймерах в различных глобальных массивах.
