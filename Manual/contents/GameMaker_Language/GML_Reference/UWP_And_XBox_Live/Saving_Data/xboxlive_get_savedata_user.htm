xboxlive_get_savedata_user

Эта функция возвращает указатель идентификатора пользователя, связанный в данный момент с сохранением файла (или константу pointer_null, если в данный момент идентификатор пользователя не используется). См. xboxlive_set_savedata_user() для более подробной информации.

Синтаксис:

xboxlive_get_savedata_user();

Возвращается:

Xbox User ID or pointer_null

Пример:

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

Приведенный выше код проверяет, назначен ли пользователь в качестве цели сохранения, и если нет, то он назначается.
