Сохранение данных

Следующие функции связаны с сохранением данных с помощью экспорта UWP для Xbox One и использованием функций XBox Live.

Хотя данные сохранения, связанные с прогрессом игрока, всегда должны быть связаны с конкретным пользователем, вы можете сохранять общие данные в области "машинного хранилища", указав в качестве аргумента пользователя значение -1. Эта область зависит от названия, но не связана с конкретным пользователем.

ПРИМЕЧАНИЕ: Размер сохраненного файла не может превышать 16 мегабайт!

Следующие функции предназначены для работы с сохраненными данными Xbox Live:

  • xboxlive_set_savedata_user
  • xboxlive_get_savedata_user
  • xboxlive_get_file_error
