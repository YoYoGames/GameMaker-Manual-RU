xboxlive_set_savedata_user

Эта функция определяет, что будущие файловые операции, которые работают в области сохранения игры (т.е. все записи в файлы и чтения из файлов, которые не находятся в области связки), будут ассоциироваться с указанным указателем ID пользователя. Эта функция может вызываться так часто, как это необходимо для перенаправления данных сохранения соответствующему пользователю, или вы можете использовать константу pointer_null для сохранения в общую область хранения машины.

Синтаксис:

xboxlive_set_savedata_user(user_id);

Аргумент Тип Описание
user_id Идентификатор пользователя Xbox или указатель_null The user ID (a pointer) to set for saving, or pointer_null

Возвращается:

N/A

Пример:

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

Приведенный выше код проверяет, назначен ли пользователь в качестве цели сохранения, и если нет, то он назначается.
