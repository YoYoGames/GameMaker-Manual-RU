xboxlive_get_file_error

Эта функция возвращает константу (показана ниже), которая поможет отладить причину сбоя файловой функции, чтобы вы могли предпринять соответствующие действия в коде игры. Когда одна из файловых функций не работает, вы можете вызвать эту функцию, чтобы узнать, в чем именно заключалась ошибка.

ПРИМЕЧАНИЕ: При использовании функций асинхронного сохранения/загрузки, карта async_load map будет иметь дополнительный ключ "error", который также будет содержать одну из констант (показано ниже).

Синтаксис:

xboxlive_get_file_error();

Возвращается:

Xbox File Error Constant

Xbox File Error Constant
Постоянная Описание
xboxlive_fileerror_outoflocalstorage Если это происходит во время сохранения, это означает, что все доступное место для сохранения игры было использовано, и пользователь отклонил предложение (представленное ОС) освободить место. Игра должна подтвердить пользователю, что он намерен продолжить игру без сохранения. Если пользователь все еще хочет иметь возможность сохраниться, игра должна повторить операцию сохранения.
xboxlive_fileerror_quotaexceeded В этом случае игра должна уведомить пользователя, что он не может больше сохранять данные, пока не освободит место, а затем предложить ему способ сделать это.
xboxlive_fileerror_noerror Функция сохранения или загрузки файлов выполнялась без проблем.

Пример:

buffer_save(buff, "SaveGame.bin");
if (xboxlive_get_file_error() != xboxlive_fileerror_noerror)
{
    show_debug_message("Save failed. Error code = " + string(xboxlive_get_file_error()));
}

Приведенный выше код проверяет, не произошла ли ошибка при сохранении, и если да, то код ошибки выводится на консоль.
