uwp_was_closed_by_user

С помощью этой функции можно выяснить, была ли игра ранее выключена пользователем во время игры (т.е. была ли она завершена правильно, а не системой). Если это так, то функция вернет true, в противном случае она вернет false.

Синтаксис:

uwp_was_closed_by_user();

Возвращается:

Пример:

if uwp_was_closed_by_user()
{
    global.GameLoaded = false;
}

Приведенный выше код проверяет, было ли приложение ранее завершено пользователем, и если да, то устанавливает глобальную переменную.
