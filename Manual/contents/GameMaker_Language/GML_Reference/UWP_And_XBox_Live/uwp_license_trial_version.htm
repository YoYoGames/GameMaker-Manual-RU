uwp_license_trial_version

Эта функция может быть использована для проверки того, является ли игра пробной лицензией или нет. Если да, то функция вернет true, или false в противном случае.

Синтаксис:

uwp_license_trial_version();

Возвращается:

Пример:

if uwp_license_trial_version()
{
    global.LevelCap = 10;
}

Приведенный выше код проверяет, не является ли приложение пробным, и если да, то устанавливает глобальную переменную.
