uwp_is_suspending

Эта функция возвращает true для одного кадра (и false в противном случае) аналогично функции os_is_paused(). Затем у игры есть одна секунда, чтобы сохранить состояние игры, прежде чем она будет потенциально завершена системой. После завершения сохранения данных игра должна вызвать функцию uwp_suspend() чтобы приостановить игру. Возможно, игра также должна войти в состояние паузы, чтобы в случае ее возобновления системой (если она не была полностью завершена) игрок не оказался в игровой ситуации.

Синтаксис:

uwp_is_suspending();

Возвращается:

Пример:

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

Приведенный выше код проверяет, переходит ли приложение в режим приостановки, и если да, то вызывает сценарий для сохранения игровых данных перед приостановкой игры.
