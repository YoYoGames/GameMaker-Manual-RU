uwp_was_terminated

С помощью этой функции можно узнать, была ли игра ранее выключена системой (т.е. это не первый запуск игры). Если это было так, то функция вернет true, в противном случае она вернет false. Примером использования может быть проверка, не была ли игра ранее завершена системой, а затем (если возвращается true) загрузка предыдущего этапа, на котором находился игрок в игре. Если проверка направлена на XBox One, то требуемое поведение игр при обнаружении этого факта будет отличаться в разных версиях документации по сертификации, поэтому стоит проверить последнюю версию, чтобы понять, что должна делать игра.

Синтаксис:

uwp_was_terminated();

Возвращается:

Пример:

if xboxlivewas_terminated()
{
    buffer_load("Player_Save.sav");
    global.GameLoaded = true;
}

Приведенный выше код проверяет, не было ли приложение ранее завершено системой, и если да, то загружает буфер и устанавливает глобальную переменную.
