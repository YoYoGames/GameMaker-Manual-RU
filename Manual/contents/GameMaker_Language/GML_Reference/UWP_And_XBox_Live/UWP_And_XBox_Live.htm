UWP и Xbox Live

В этом разделе описаны функции, специально предназначенные для использования при работе с целевой платформой UWP. Начальный набор функций предназначен для управления общим поведением игры при создании UWP-приложения:

  • uwp_suspend
  • uwp_is_suspending
  • uwp_is_constrained
  • uwp_was_terminated
  • uwp_was_closed_by_user
  • uwp_show_help
  • uwp_license_trial_version
  • uwp_license_trial_user
  • uwp_license_trial_time_remaining
  • uwp_check_privilege

UWP также позволяет создавать игры для платформы XBox One с использованием XBox Live API с помощью следующих функций. Эти функции требуют, чтобы вы отметили опцию Enable XBox Live в опциях игры UWP. Некоторые из этих функций требуют, чтобы пользователь был зарегистрирован в XBox Live, что GameMaker попытается сделать автоматически при запуске проекта. Такая "тихая" регистрация создаст системное асинхронное событие, которое можно проверить, чтобы убедиться в успехе или нет, прежде чем продолжить работу. Обратите внимание, что в отличие от экспорта для Xbox One Console, экспорт UWP для XBox Live позволяет войти в игру только одному пользователю одновременно.

ВАЖНО! Цель UWP имеет минимальную поддержку XBox Live только при настройке в рамках программы Creator Program, которая ограничена некоторыми базовыми функциями пользователя и учетной записи (см. здесь для получения дополнительной информации о настройке). Чтобы иметь возможность использовать все перечисленные здесь функции, вам необходимо иметь учетную запись ID@XBox.
