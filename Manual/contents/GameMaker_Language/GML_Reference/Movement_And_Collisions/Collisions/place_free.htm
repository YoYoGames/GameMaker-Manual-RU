место_свободно

С помощью этой функции можно проверить, не столкнется ли вызывающий экземпляр с любым экземпляром, отмеченным как твердый в вашей игре. Следует отметить, что для работы этой функции у экземпляра, выполняющего код, должна быть действительная маска столкновений (либо для самого sprite, либо с помощью функции mask_index), и он будет регистрировать столкновения только с теми экземплярами, помеченными как твердые, у которых также есть действительная маска.

Сама функция работает следующим образом: берется экземпляр и проверяется на столкновения только со сплошными отмеченными экземплярами при размещении в позиции, заданной аргументами x/y. Проверка столкновений может быть как точной, так и основанной на граничной области экземпляра, в зависимости от того, какая маска столкновений была выбрана.

Обратите внимание, что перед выполнением проверки заданные координаты x/y будут приведены к ближайшему целому числу.

Синтаксис:

place_free(x, y);

АргументТип Описание
x The x position to check.
y The y position to check.

Возвращается:

Пример:

if place_free(mouse_x, mouse_y)
{
    x = mouse_x;
    y = mouse_y;
}

Приведенный выше код проверит столкновение с любым экземпляром с твердым флагом, если вызывающий экземпляр будет помещен в то же положение, что и мышь. Если столкновение не обнаружено, то координаты x/y экземпляра устанавливаются равными координатам мыши.
