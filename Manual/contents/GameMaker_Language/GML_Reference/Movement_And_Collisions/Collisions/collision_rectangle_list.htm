список_прямоугольников_коллизий

Эта функция аналогична функции collision_rectangle() только вместо того, чтобы обнаруживать только один столкнувшийся объект за раз, она будет обнаруживать несколько объектов. Вы задаете x/y позиции верхнего левого и нижнего правого угла области для проверки, а также объект для проверки, и можете задать точность проверки (в этом случае все проверяемые экземпляры должны иметь точные маски столкновений), а также указать, должна ли проверка включать вызывающий экземпляр или нет.

Вы также предоставляете список DS, поэтому id значения всех экземпляров, которые сталкиваются с вызывающим экземпляром, будут добавлены в конец данного списка. Вы можете очистить список перед вызовом этой функции, чтобы он содержал только результаты этого вызова функции. Также можно упорядочить экземпляры по расстоянию от центра прямоугольной области до их начала. Функция возвращает количество найденных экземпляров или 0, если не найдено ни одного.

Обратите внимание, что вместо индекса объекта можно использовать ключевое слово instance all, чтобы проверить все экземпляры в текущей комнате, что может включать экземпляр, выполняющий код (в зависимости от аргумента notme ).

Синтаксис:

collision_rectangle_list(x1, y1, x2, y2, obj, prec, notme, list, ordered);

Аргумент Тип Описание
x1 Реально The x coordinate of the left side of the rectangle to check.
y1 Реально The y coordinate of the top side of the rectangle to check.
x2 Реально The x coordinate of the right side of the rectangle to check.
y2 Реально The y coordinate of the bottom side of the rectangle to check.
obj Идентификаторактива или экземпляраобъекта The object to check for instance collisions.
prec Булево Whether the check is based on precise collisions (true, что медленнее) или его граничная область в целом (false, быстрее).
notme Булево Whether the calling instance, if relevant, should be excluded (true) или нет (false).
list Идентификатор списка DS The DS list to use to store the IDs of colliding instances.
ordered Булево Whether the list should be ordered by distance (true) или нет (false).

Возвращается:

Real (The number of instances found to be in collision)

Пример:

var _list = ds_list_create();
var _num = collision_rectangle_list(x - 100, y - 100, x + 100, y + 100, obj_Enemy, false, true, _list, false);
if (_num > 0)
{
    for (var i = 0; i < _num; ++i)
    {
        instance_destroy(_list[| i]);
    }
}
ds_list_destroy(_list);

Код выше проверяет прямоугольную область в 100 пикселей вокруг позиции вызывающего экземпляра на наличие столкновений с экземплярами "obj_Enemy". Если столкновения есть, то предварительно созданный список просматривается, и каждый экземпляр, который был в столкновении, уничтожается.
