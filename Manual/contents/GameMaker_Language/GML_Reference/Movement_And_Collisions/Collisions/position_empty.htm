позиция_пустая

Эта функция проверяет, входит ли данная позиция в столкновение с любым экземпляром с действительной маской столкновения в данной позиции.

Синтаксис:

position_empty(x, y);

АргументТип Описание
x The x position to check.
y The y position to check.

Возвращается:

Пример:

var xx, yy;
xx = random(room_width);
yy = random(room_height);
if position_empty(xx, yy)
{
    instance_create_layer(xx, yy, "Bullets", obj_Bomb);
}

Он проверит случайную позицию в комнате на наличие столкновения и, если его нет, создаст экземпляр obj_Bomb.
