позиция_совещания

С помощью этой функции можно проверить позицию на столкновение с другим экземпляром или всеми экземплярами объекта. При использовании этой функции вы проверяете одну точку в комнате на наличие экземпляра или объекта. Проверка будет выполнена по ограничивающей рамке экземпляра или по маске экземпляра, если у этого экземпляра проверены точные столкновения. Если вам нужно получить уникальный экземпляр id объекта, с которым происходит столкновение, вам следует использовать instance_position(). Эта функция также принимает специальные ключевые слова all и other.

Синтаксис:

position_meeting(x, y, obj);

Аргумент Тип Описание
x Реально The x position to check.
y Реально The y position to check.
obj Идентификаторактива или экземпляраобъекта The object (or instance id, or the keywords "all" or "other") to check for a collision with.

Возвращается:

Boolean

Пример:

if mouse_check_button(mb_left)
{
    if (!position_meeting(mouse_x, mouse_y, all))
    {
        instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wall);
    }
}

Приведенный выше код проверяет наличие левой кнопки мыши, и если она нажата, то проверяет положение мыши x/y на предмет столкновения с каким-либо экземпляром. Если таковых нет, то создается экземпляр "obj_Wall".
