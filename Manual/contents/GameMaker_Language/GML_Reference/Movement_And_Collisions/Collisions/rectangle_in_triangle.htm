rectangle_in_triangle

Эта функция проверит заданную вами прямоугольную область на предмет отсутствия столкновений, полного попадания в границы места назначения или простого соприкосновения с заданной треугольной областью. Если они вообще не соприкасаются, то функция вернет 0, если источник полностью находится внутри места назначения, то вернет 1, а если они просто перекрываются, то вернет 2. Изображение ниже иллюстрирует это:

Синтаксис:

rectangle_in_triangle(sx1, sy1, sx2, sy2, dx1, dy1, dx2, dy2, dx3, dy3);

АргументТип Описание
sx1 The x coordinate of the left side of the source rectangle.
sy1 The y coordinate of the top side of the source rectangle.
sx2 The x coordinate of the right side of the source rectangle.
sy2 The y coordinate of the bottom side of the source rectangle.
x1 The x coordinate of the first side of the triangle to check.
y1 The y coordinate of the first side of the triangle to check.
x2 The x coordinate of the second side of the triangle to check.
y2 The y coordinate of the second side of the triangle to check.
x3 The x coordinate of the third side of the triangle to check.
y3 The y coordinate of the third side of the triangle to check.

Возвращается:

Пример:

var inst = instance_nearest(x, y, obj_Player);
if (instance_exists(inst))
{
    var _x1 = x + lengthdir_x(100, image_angle - 45);
    var _y1 = y + lengthdir_y(100, image_angle - 45);
    var _x2 = x + lengthdir_x(100, image_angle + 45);
    var _y2 = y + lengthdir_y(100, image_angle + 45);
    if (rectangle_in_triangle(_inst.bbox_left, _inst.bbox_top, _inst.bbox_right, _inst.bbox_bottom, x, y, _x1, _y1, _x2, _y2) == 1)
    {
        can_see = true;
    }
}

Приведенный выше код использует функцию rectangle_in_triangle в качестве "конуса зрения" для проверки наличия экземпляра "obj_player". Если полная граница находится в заданной треугольной области, то переменная будет установлена в значение true.
