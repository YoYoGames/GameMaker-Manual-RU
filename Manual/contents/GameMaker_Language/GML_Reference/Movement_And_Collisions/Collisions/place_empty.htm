Вы можете использовать эту функцию, чтобы проверить, столкнется ли вызывающий экземпляр с любым другим экземпляром объекта или со всеми экземплярами в вашей игре. Чтобы столкновение разрешилось правильно, у экземпляра, выполняющего код, должна быть действительная маска столкновения (либо для самого sprite, либо через mask_index), и он будет регистрировать столкновения только с теми экземплярами, которые также имеют действительную маску.

Функция проверяет, нет ли столкновений, если вызываемый экземпляр будет помещен в определенную позицию, и вы можете указать дополнительный аргумент для уточнения проверки, чтобы проверить, свободна ли позиция от столкновений с экземплярами данного типа. Обратите внимание, что если не указан дополнительный идентификатор объекта, то проверка будет выполнена для всех экземпляров в пределах комнаты. Проверка столкновений будет либо точной, либо основанной на граничной области экземпляра, в зависимости от того, какая маска столкновений была выбрана, но для корректной работы точных столкновений все экземпляры в проверке должны иметь точные маски столкновений.

Обратите внимание, что перед выполнением проверки заданные координаты x/y будут приведены к ближайшему целому числу.

Синтаксис:

place_empty(x, y, [object_id]);

Аргумент Тип Описание
x Реально The x position to check.
y Реально The y position to check.
[object_id] Объект Активы OPTIONAL Объект для проверки.

Возвращается:

Boolean

Пример:

if place_empty(mouse_x, mouse_y, obj_Enemy)
{
    x = mouse_x;
    y = mouse_y;
}

Приведенный выше код проверяет наличие столкновения с любым другим экземпляром объекта "obj_Enemy", как если бы вызывающий экземпляр был помещен в ту же позицию, что и мышь. Если столкновения не обнаружено, то координаты x/y экземпляра устанавливаются равными координатам мыши.
