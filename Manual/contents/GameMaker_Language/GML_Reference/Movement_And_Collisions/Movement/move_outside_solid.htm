перемещение_вне_твердого

С помощью этой функции вы можете указать экземпляру выйти из столкновения с любым экземпляром, отмеченным как solid в любом направлении и на любое количество пикселей за шаг, при этом значение -1 или 0 для maxdist по умолчанию равно 1000px, т.е. GameMaker будет постоянно перемещать экземпляр на 1000 пикселей вверх, пока он не выйдет из столкновения.

Синтаксис:

move_outside_solid(dir, maxdist);

АргументТип Описание
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Возвращается:

Пример:

if other.solid
{
    var pdir;
    pdir = point_direction(other.x, other.y, x, y);
    move_outside_solid(pdir, -1);
}

Приведенный выше код будет входить в событие столкновения и проверять, отмечен ли "другой" экземпляр в столкновении как "твердый". Если это так, то он переместит экземпляр за пределы столкновения.
