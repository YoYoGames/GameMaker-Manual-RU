motion_set

Эта функция задает новое направление движения и новую скорость для экземпляра, выполняющего код. Обратите внимание, что эта функция не добавляет к текущей скорости и направлению движения экземпляра (для этого используется функция motion_add()), а скорее заставляет его принять новые настройки.

Синтаксис:

motion_set(dir, speed);

АргументТип Описание
dir The new direction.
speed The new speed.`

Возвращается:

Пример:

if irandom(9) = 1
{
    motion_set(random(360), 1 + random(3));
}

Приведенный выше код заставит экземпляр менять скорость и направление движения через случайные промежутки времени.
