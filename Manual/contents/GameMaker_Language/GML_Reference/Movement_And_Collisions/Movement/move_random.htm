move_random

С помощью этой функции вы можете установить экземпляр в любое место в комнате, но выровненное по "невидимой" сетке. Так, значение 32 для hsnap и vsnap установит экземпляр в случайную позицию, выровненную по сетке из 32x32 квадратов. Вы можете установить эти значения в 1, чтобы получить позицию в любом месте комнаты.

ПРИМЕЧАНИЕ: Эта функция будет привязываться к одним и тем же позициям при каждом новом запуске игры, поскольку GameMaker каждый раз генерирует одно и то же начальное случайное зерно, чтобы облегчить отладку кода. Чтобы избежать такого поведения, используйте randomise() в начале игры. Это справедливо только при тестировании и отладке игры, так как конечный исполняемый пакет не будет демонстрировать такое поведение и будет случайным при каждом запуске.

Синтаксис:

move_random(hsnap, vsnap);

АргументТип Описание
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

Возвращается:

Пример:

if keyboard_check(vk_space)
{
    move_random(1, 1);
}

При нажатии клавиши пробела экземпляр перемещается в произвольную позицию в любом месте комнаты.
