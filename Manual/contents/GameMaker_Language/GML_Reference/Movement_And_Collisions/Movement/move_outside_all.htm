move_outside_all

С помощью этой функции вы можете указать экземпляру выйти из столкновения в любом направлении и на любое количество пикселей за шаг, при этом значение -1 или 0 для maxdist по умолчанию равно 1000px, т.е. GameMaker будет постоянно перемещать экземпляр на 1000 пикселей вверх, пока он не выйдет из столкновения.

Синтаксис:

move_outside_all(dir, maxdist);

АргументТип Описание
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Возвращается:

Пример:

if place_meeting(x, y, all)
{
    move_outside_all(90, 1);
}

Приведенный выше код будет перемещать экземпляр на 1 пиксель вверх на каждом шаге, когда он сталкивается с любым другим экземпляром.
