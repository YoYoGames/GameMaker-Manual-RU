перемещение_контакта_твердого

Эта функция перемещает экземпляр, выполняющий код, на заданное количество пикселей в указанном направлении до тех пор, пока он не встретит экземпляр с флажком solid отмеченным как true. Вы можете использовать -1 или 0 для максимального расстояния, которое по умолчанию равно 1000px, т.е. GameMaker будет постоянно перемещать экземпляр на 1000 пикселей, пока он не выйдет из столкновения.

Синтаксис:

move_contact_solid( dir, maxdist );

АргументТип Описание
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Возвращается:

Пример:

if !place_meeting(x, y + 1, obj_Floor)
{
    move_contact_solid(270, -1);
}

Приведенный выше код будет проверять под экземпляром на столкновение с "obj_Floor", и если такового нет, то будет перемещать его вниз, пока не произойдет столкновение или не будет пройдено 1000 пикселей.
