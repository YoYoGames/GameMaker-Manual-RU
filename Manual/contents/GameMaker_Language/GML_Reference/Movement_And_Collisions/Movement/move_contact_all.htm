move_contact_all

Эта функция будет перемещать экземпляр, выполняющий код, на заданное количество пикселей в указанном направлении, пока он не встретит любой другой экземпляр с действующей маской. Вы можете использовать -1 или 0 для maxdist, по умолчанию это 1000px, т.е. GameMaker будет постоянно перемещать экземпляр на 1000 пикселей, пока он не выйдет из столкновения.

Синтаксис:

move_contact_all(dir, maxdist);

АргументТип Описание
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Возвращается:

Пример:

if !place_meeting(x, y + 1, all)
{
    move_contact_all(270, -1);
}

Приведенный выше код будет проверять экземпляр на наличие столкновения, и если его нет, то он будет перемещать его вниз, пока не появится или пока не будет пройдено 1000 пикселей.
