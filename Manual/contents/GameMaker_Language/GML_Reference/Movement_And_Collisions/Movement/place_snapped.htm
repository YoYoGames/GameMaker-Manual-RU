место_захвата

С помощью этой функции можно проверить, выровнено ли начало координат экземпляра (его положение x и y) по сетке с указанными вами значениями hsnap и vsnap.

Синтаксис:

place_snapped(hsnap, vsnap);

АргументТип Описание
hsnap The horizontal snapping to check.
vsnap The vertical snapping to check.

Возвращается:

Пример:

with (obj_Pieces)
{
    if !place_snapped(32, 32)
    {
        move_snap(32, 32);
    }
}

Приведенный выше код проверяет все экземпляры "obj_Pieces" на то, привязаны ли они к сетке 32x32 пикселей, и если нет, то привязывает их к ближайшей позиции в этой сетке.
