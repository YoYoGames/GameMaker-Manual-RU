mp_potential_step_object

Эта функция позволяет экземпляру сделать шаг к определенной позиции, заданной xgoal/ygoal, стараясь при этом избежать препятствий. Когда экземпляр столкнется с экземпляром объекта, указанного в аргументе "obj", он изменит направление движения, чтобы попытаться избежать этого объекта и двигаться вокруг него. Не гарантируется, что этот подход сработает, но в большинстве простых случаев он будет эффективно перемещать экземпляр к цели. Функция возвращает, была ли цель достигнута или нет.

Синтаксис:

mp_potential_step_object(xgoal, ygoal, stepsize, obj)

Аргумент Тип Описание
xgoal Реально The target x position.
ygoal Реально The target y position.
stepsize Реально The speed the instance moves in pixels per step.
obj Идентификаторактива или экземпляраобъекта The object that is to block the path of the instance running the function. Can be an object index, an instance id or the special keyword ,all.

Возвращается:

Boolean

Пример:

if instance_exists(obj_Enemy)
{
    var inst;
    inst = instance_nearest(x, y, obj_Enemy);
    mp_potential_step_object(inst.x, inst.y, 5, obj_Wall);
}

Приведенный выше код сначала проверяет, есть ли в комнате экземпляры "obj_Enemy". Если таковые имеются, он находит ближайший и сохраняет его id в переменной. Затем эта переменная используется для того, чтобы указать коду mp_potential_step_object перемещать экземпляр в направлении x/y позиции объекта, который был найден, со скоростью 5 пикселей за шаг, избегая при этом только экземпляры объекта "obj_Wall".
