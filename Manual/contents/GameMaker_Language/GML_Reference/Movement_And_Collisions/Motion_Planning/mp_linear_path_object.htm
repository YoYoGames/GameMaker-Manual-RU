mp_linear_path_object

Эта функция рассчитывает прямолинейный путь от текущей позиции экземпляра до точки, заданной значениями "xgoal" и "ygoal". Она использует указанный размер шага, такой же, как в функции mp_linear_step(). Указанный путь должен уже существовать и будет перезаписан новым путем, а функция вернет, был ли найден полный путь (true) или нет (false). Полный путь будет найден только в том случае, если не было столкновения с указанным объектом или экземпляром, а если возвращается false, то путь все равно будет сгенерирован, но он будет проходить только до того места, где путь был заблокирован.

ПРИМЕЧАНИЕ: Эта функция не перемещает экземпляр. Она только задает путь, а для перемещения необходимо использовать функции Path.

Синтаксис:

mp_linear_path_object(path, xgoal, ygoal, stepsize, obj);

Аргумент Тип Описание
path Путь Актива The index of the path to be used
xgoal Реально The target x position.
ygoal Реально The target y position.
stepsize Реально The speed the instance moves in pixels per step.
obj Идентификаторактива или экземпляраобъекта The object that is to block the path. Can be an object index, an instance id or the special keyword ,all

Возвращается:

Boolean

Пример:

if mp_linear_path_object(path, mouse_x, mouse_y, 4, obj_Wall)
{
    path_start(path, 4, 0, 0);
}

Приведенный выше код проверяет наличие столкновения с "obj_Wall" на пути между объектом, запускающим код, и позицией x/y мыши. Если столкновений нет и сгенерирован полный путь, то запускается объект по сгенерированному пути.
