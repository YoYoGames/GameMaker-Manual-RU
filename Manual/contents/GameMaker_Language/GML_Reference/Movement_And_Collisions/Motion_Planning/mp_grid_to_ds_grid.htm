mp_grid_to_ds_grid

С помощью этой функции вы можете скопировать заданную MP-сетку в DS-сетку. DS-сетка должна быть того же размера, что и MP-сетка, хотя это не обязательно (данные будут потеряны, если она будет меньше, а если больше, то все дополнительные ячейки сетки будут равны 0). Ячейка в сетке DS будет содержать значение -1, если она была отмечена как занятая в сетке MP, или 0, если не была.

Синтаксис:

mp_grid_to_ds_grid(source, destination);

Аргумент Тип Описание
source MP Grid ID Index of the mp_grid that is to be used
destination DS Grid ID Index of the ds_grid that is to be used to copy the grid data to.

Возвращается:

Boolean

Пример:

motion_grid = ds_grid_create(room_width / 32, room_height / 32);
mp_grid_to_ds_grid(mp_grid, motion_grid);

Приведенный выше код создаст новую сетку DS, а затем скопирует данные сетки MP, содержащиеся в переменной "mp_grid", в новую сетку DS.
