mp_potential_settings

Функции mp_potential_ выполняют свою работу, используя ряд параметров, которые можно изменить с помощью этой функции. Глобально метод работает следующим образом: сначала он пытается двигаться прямо к цели, смотря на несколько шагов вперед, которые можно задать с помощью параметра "ahead" (значение по умолчанию - 3). Уменьшение этого значения означает, что экземпляр начнет менять направление позже, так как он не "видит" так далеко вперед, а увеличение означает, что он начнет менять направление раньше. Если эта проверка приведет к столкновению, то он начнет рассматривать направления слева и справа от идеального прямолинейного направления. Он делает это с шагом размером "rotstep" (значение по умолчанию - 10), и уменьшение этого значения дает экземпляру больше возможностей для движения, но будет медленнее, поскольку ему требуется больше вычислительной мощности.

Параметр "maxrot" объяснить немного сложнее! У экземпляра есть текущее направление движения, а maxrot (значение по умолчанию - 30) указывает, насколько ему разрешено изменять направление в любую сторону от текущего направления за один шаг. Поэтому даже если он может двигаться, например, прямо к цели, он сделает это только в том случае, если не нарушит это максимальное изменение направления. Если сделать maxrot большим значением, то экземпляр может сильно менять направление на каждом шаге, и это облегчит поиск короткого пути, но путь будет более уродливым и менее естественным. Если сделать значение меньше, то путь будет более гладким, но может потребоваться больше времени на обход (а иногда даже не удастся найти цель). Наконец, если не удается сделать ни одного шага, поведение зависит от значения параметра "onspot". Если onspot равен true (значение по умолчанию), то экземпляр будет вращаться на своем месте на величину, указанную в параметре "maxrot", а если false, то он вообще не будет двигаться. Установка значения false полезна, например, для автомобилей, но уменьшает вероятность того, что экземпляр найдет путь.

Синтаксис:

mp_potential_settings(maxrot, rotstep, ahead, onspot)

Аргумент Тип Описание
maxrot Реально The number of degrees either side of the current direction that the instance can rotate in a step
rotstep Реально The number of degrees either side of the current direction that the instance can check for a collision
ahead Реально The number of steps ahead that the instance can check for a collision. Larger values are slower than lower ones.
onspot Булево Use this to allow the instance to rotate on the spot when no path is found (true) or not (false)

Возвращается:

N/A

Пример:

mp_potential_settings(45, 5, 5, 0);

Приведенный выше код настроит функции mp_potential_ на использование следующих параметров: экземпляр может смотреть вперед на пять шагов, менять направление на 45 градусов каждый шаг, смотреть на 5 градусов в любую сторону от текущего направления при наличии препятствия и не поворачиваться на месте столкновения.
