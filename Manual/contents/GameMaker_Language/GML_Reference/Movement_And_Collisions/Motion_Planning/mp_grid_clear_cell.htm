mp_grid_clear_cell

Эта функция может быть использована для очистки определенной "ячейки" сетки MP. Ячейки рассчитываются не как координаты комнаты, а как координаты сетки, где (0,0) - верхний левый угол сетки. Это означает, что для очистки ячейки в определенной позиции в комнате, мы должны изменить координаты x и y на координаты ячейки, разделив их на разрешение сетки MP. Пример кода ниже показывает, как это работает.

Синтаксис:

mp_grid_clear_cell(id, h, v);

Аргумент Тип Описание
id MP Grid ID Index of the mp_grid that is to be used
h Реально Horizontal position of the cell to clear
v Реально vertical position of the cell to clear

Возвращается:

Boolean

Пример:

with (obj_Box)
{
    mp_grid_clear_cell(grid, floor(x / 32), floor(y /32));
    instance_destroy();
}

Приведенный выше код заставит все "obj_Box" уничтожить себя и пометить ячейки, которые они занимали в mp_grid, проиндексированные в переменной "grid", как свободные. В этом примере мы находим соответствующую ячейку, взяв координаты x/y объекта и разделив их на разрешение сетки (используя floor, чтобы сохранить значения в виде целых чисел).
