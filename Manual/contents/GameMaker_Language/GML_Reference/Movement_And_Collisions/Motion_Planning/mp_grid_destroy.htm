mp_grid_destroy

Вы можете использовать эту функцию, чтобы уничтожить указанную mp_grid (как определено функцией mp_grid_create()) и освободить выделенную ей память. Важно, чтобы вы вызывали эту функцию всякий раз, когда MP-сетка завершена, иначе вы можете получить утечку памяти, что означает, что ваша игра будет замедляться с течением времени и в конечном итоге завершится крахом.

ПРИМЕЧАНИЕ: Использование функций mp_grid_* на сетке после ее уничтожения приведет к ошибке.

Синтаксис:

mp_grid_destroy(id);

Аргумент Тип Описание
id MP Grid ID Index of the mp_grid that is to be destroyed

Возвращается:

N/A

Пример:

if timer = 0
{
    mp_grid_destroy(grid);
    room_goto(rm_Menu);
}

Приведенный выше код уничтожит mp_grid, проиндексированную в переменной "grid", если переменная "timer" равна 0, а затем перейдет в другую комнату.
