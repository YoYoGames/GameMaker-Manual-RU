mp_grid_get_cell

С помощью этой функции вы можете проверить любую заданную ячейку mp_grid на предмет того, была ли она помечена как занятая или нет, задав индекс (ранее созданной) mp_grid и координаты x и y ячейки для проверки.

Если она была занята или проверяемая позиция находится вне границ сетки, функция вернет -1, в противном случае - 0.

Синтаксис:

mp_grid_get_cell(id, x , y);

Аргумент Тип Описание
id MP Grid ID Index of the mp_grid that is to be used
x1 Реально The x coordinate of the grid to check.
y1 Реально The y coordinate of the grid to check.

Возвращается:

Real

Пример:

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
{
    image_blend = c_red;
}
else
{
    image_blend = c_lime;
}

Приведенный выше код проверит ячейку mp_grid, соответствующую положению мыши, и если она занята, то установит переменную image_blend в красный цвет, а если не занята, то в зеленый.
