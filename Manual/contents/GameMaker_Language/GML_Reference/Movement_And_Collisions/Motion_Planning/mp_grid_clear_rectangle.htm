mp_grid_clear_rectangle

С помощью этой функции вы можете определить область в комнатных координатах, которая затем очистит соответствующие ячейки в указанной MP-сетке. Даже если ячейка частично попадает в заданную прямоугольную область, она будет очищена.

Синтаксис:

mp_grid_clear_rectangle(id, x1, y1, x2, y2);

Аргумент Тип Описание
id MP Grid ID Index of the mp_grid that is to be used
x1 Реально The x coordinate of the left side of the rectangle to check.
y1 Реально The y coordinate of the top side of the rectangle to check.
x2 Реально The x coordinate of the right side of the rectangle to check.
y2 Реально The x coordinate of the bottom side of the rectangle to check.

Возвращается:

Boolean

Пример:

mp_grid_clear_rectangle(grid, 0, 0, 100, 200);

Приведенный выше код пометит как свободные все ячейки mp_grid, индексированные в переменной "grid", которые попадают в область от 0,0 до 100,200.
