Общее управление игрой

Эти функции и переменные связаны с функциональностью, специфичной для игры, например, загрузка/сохранение, перезапуск и так далее.

Следующие встроенные переменные глобальной области видимости полезны для отладки вашей игры - они могут быть использованы в качестве вывода в файл журнала в случае возникновения проблем, например - а также для отображения текущего ID сборки и т.д... пользователю:

  • game_id
  • game_save_id
  • game_display_name
  • game_project_name

Следующие функции используются для общего управления игрой:

  • game_end
  • game_restart
  • game_load
  • game_load_buffer
  • game_save
  • game_save_buffer
  • game_get_speed
  • game_set_speed

Чтобы помочь вам создавать и поддерживать высокие баллы для ваших игр, GameMaker создает глобальный массив высоких баллов из 10 мест, к которому вы можете обращаться, добавлять и изменять, чтобы без лишних хлопот создавать свои собственные списки высоких баллов. Эта функциональность является локальной для игры и работает на всех платформах, поэтому вы можете легко хранить и поддерживать базовую таблицу высоких результатов, используя эти функции вместе с функциями обработки файлов.

Существуют следующие функции, работающие с внутренним списком высоких результатов.

  • highscore_add
  • highscore_name
  • highscore_value
  • highscore_clear

GameMaker также включает функцию для вывода результатов на экран в виде списка, однако она очень проста, и рекомендуется использовать ее для отладки, а не для чего-либо еще, поскольку создать свою собственную таблицу результатов очень просто, и это даст вам гораздо больше контроля над тем, как она отображается:

  • draw_highscore

Наконец, у нас есть функция, позволяющая назначить курсору мыши адрес sprite:

  • cursor_sprite
