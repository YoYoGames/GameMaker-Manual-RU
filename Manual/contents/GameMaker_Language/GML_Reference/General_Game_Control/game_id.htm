game_id

Эта переменная, доступная только для чтения, возвращает уникальный идентификатор для созданной вами игры. Вы можете использовать его, если вам нужно уникальное имя файла или что-то еще, что необходимо для идентификации только вашей игры. Этот параметр можно задать в Опциях игры.

Синтаксис:

game_id

Возвращается:

Real

Пример:

ini_open("Score.ini");
ini_write_real("Scores", "0", score + game_id);
ini_close();

Приведенный выше код выполняет очень базовое шифрование счета, добавляя к нему game_id перед сохранением. При повторном чтении в игре вы вычтете game_id, чтобы снова получить правильное значение.
