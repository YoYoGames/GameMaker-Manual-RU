значение_высокой_оценки

С помощью этой функции вы можете очистить внутренний глобальный список высоких результатов, сбросив все имена на значение по умолчанию "Unknown", а все оценки - на значение по умолчанию 0.

Синтаксис:

highscore_clear();

Возвращается:

Пример:

highscore_clear();

Это очистит список высоких результатов.
