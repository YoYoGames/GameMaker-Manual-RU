highscore_add

С помощью этой функции вы можете добавить имя и результат во внутренний глобальный список высоких результатов. Нет необходимости проверять значение, достаточно ли оно высоко для внесения в список, поскольку GameMaker будет сохранять только те значения, которые больше десятой позиции.

Синтаксис:

highscore_add(str, numb);

Аргумент Тип Описание
str Строка The string (name) to attribute the score to.
numb Реально The number (score) to add.

Возвращается:

N/A

Пример:

if (lives == 0)
{
    highscore_add(global.Name, score);
    score = 0;
    room_goto(rm_MainMenu);
}

Приведенный выше код проверит жизни и, если они равны 0, добавит текущий счет и строку, хранящуюся в глобальной переменной "Name", в список высоких результатов, после чего установит счет на 0 и отправит игрока в комнату, индексированную в "rm_MainMenu".
