game_set_speed

С помощью этой функции можно задать скорость игры. Ее можно задать одним из двух способов - либо как количество игровых кадров в секунду (FPS), либо как количество микросекунд на игровой кадр (MPF) - используя одну из следующих двух констант:

Постоянная Описание
gamespeed_fps Устанавливает скорость игры с использованием кадров в секунду.
gamespeed_microseconds Устанавливает скорость игры, используя микросекунды на кадр.

Так, например, при скорости игры 30 кадров в секунду на один игровой кадр будет приходиться 33333 микросекунды, которые затем будут выражены этой функцией как либо:

game_set_speed(30, gamespeed_fps);

или:

game_set_speed(33333, gamespeed_microseconds);

Синтаксис:

game_set_speed(speed, type);

Аргумент Тип Описание
speed Реально The new game speed (as either FPS or MPF).
type Скорость игры Постоянная The type of method used to set the game speed (see the constants above).

Возвращается:

N/A

Пример:

if os_browser == browser_not_a_browser
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

Приведенный выше код проверяет, запущена ли игра в браузере, и соответствующим образом устанавливает скорость игры в виде значения FPS.
