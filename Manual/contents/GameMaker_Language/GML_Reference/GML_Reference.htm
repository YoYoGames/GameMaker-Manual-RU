Справочник покоду GML

Этот раздел руководства представляет собой справочное руководство для GameMaker Language (GML). Здесь вы найдете все доступные функции, документированные вместе с необходимыми аргументами и примерами кода, показывающими, как их можно использовать. Для получения информации о том, как использовать GML, пожалуйста, обратитесь к разделу Обзор GML раздел руководства.

ПРИМЕЧАНИЕ Везде, где в руководстве в качестве возвращаемого значения функции указано "N/A", это означает, что функция не предназначена для возврата какого-либо значения в рамках своей работы. Если вы попытаетесь проверить возвращаемое значение из этих функций, то получите значение undefined.
