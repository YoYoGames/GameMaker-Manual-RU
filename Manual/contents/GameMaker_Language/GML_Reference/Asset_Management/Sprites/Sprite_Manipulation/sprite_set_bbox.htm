sprite_set_bbox

Эта функция может быть использована для установки значений граничной области для sprite. Вы указываете индекс sprite, который нужно использовать, а затем значения левой, верхней, правой и нижней частей для позиций граничной области. Позиции являются абсолютными значениями, где позиция (0, 0) соответствует левому верхнему углу sprite, независимо от смещения для sprite, любых "пустых" пикселей, которые могут быть у sprite, или места рисования в комнате.

ПРИМЕЧАНИЕ Эта функция влияет на актив sprite таким образом, что все последующие экземпляры с этим sprite будут иметь одинаковые граничные рамки.

Синтаксис:

sprite_set_bbox(ind, left, top, right, bottom);

АргументТип Описание
ind The index of the sprite to set the bounding box on.
left The left side of the bounding box
top The top of the bounding box.
right The right side of the bounding box
bottom The bottom of the bounding box.

Возвращает

Пример:

if sprite_get_bbox_mode(sprite_index) == 2
{
    left = irandom(sprite_width / 2);
    right = irandom((sprite_width / 2) + irandom(sprite_width / 2));
    top = irandom(sprite_height / 2);
    bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2));
    sprite_set_bbox(sprite_index, left, top, right, bottom);
}

Приведенный выше код проверит режим граничного поля sprite, назначенного на sprite_index, и если он установлен на manual, то его граничное поле будет изменено.
