sprite_flush

С помощью этой функции вы можете удалить из текстурной памяти данную текстурную страницу для данного sprite. Вы указываете индекс sprite (как определено при создании ресурса sprite ), и текстурная страница, которой она назначена, будет удалена из текстурной памяти. Обратите внимание, что если страница текстуры используется в другом месте комнаты (другим экземпляром sprite или фоном, и т.д.), вы можете получить небольшое снижение производительности, так как страница будет немедленно перезагружена, поэтому при использовании этой функции следует соблюдать осторожность. Обратите внимание, что вернет -1, если flush не поддерживается для выбранного ресурса, или вернет 0, если все работает правильно.

Синтаксис:

sprite_flush(ind)

=======

sprite_flush

This function removes the texture page for the given sprite from texture memory (VRAM). The texture will stay in RAM after being flushed.

You supply the sprite (as defined when creating the sprite) and the texture page it is assigned to will be removed from texture memory. The function will return -1 if flush is not supported for the chosen asset, or it will return 0 if all worked correctly.

 If the texture page is used elsewhere in the room (by another instance sprite, a background, etc.) you may get a minor performance hit as the page is immediately reloaded, so care should be taken when using this function.
