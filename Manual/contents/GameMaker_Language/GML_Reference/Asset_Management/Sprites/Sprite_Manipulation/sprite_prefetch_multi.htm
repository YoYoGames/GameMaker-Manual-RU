sprite_prefetch_multi

Эта функция может быть использована для предварительной выборки (размещения в памяти текстуры) ряда текстурных страниц, содержащих заданный sprites. Вы передаете массив, заполненный индексами sprite (как определено при создании актива sprite ), и текстурные страницы, на которых они находятся, будут загружены в память. Обратите внимание, что функция вернет -1, если prefetch не поддерживается для выбранного ресурса или целевая платформа - HTML5, или вернет 0, если все работает правильно.

ПРИМЕЧАНИЕ: При загрузке текстуры в память текстуры на большинстве устройств происходит снижение производительности, поэтому рекомендуется вызывать sprite_prefetch_multi() все необходимые графические элементы в начале уровня, чтобы избежать задержек.

Синтаксис:

sprite_prefetch_multi

Аргумент Тип Описание
array Массив активов спрайтов Array with the sprite indices to fetch

Возвращается:

Real (-1 or 0)

Пример:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_prefetch_multi(spr_a);

Приведенный выше код создает массив, каждый элемент которого содержит индекс sprite. Этот массив затем используется для размещения этих sprite текстур в памяти.
