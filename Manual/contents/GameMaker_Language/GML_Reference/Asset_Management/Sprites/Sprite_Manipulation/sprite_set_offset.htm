sprite_set_offset

Эта функция может быть использована для установки начала координат x и y sprite, и принимает относительные значения, основываясь на позиции (0,0), являющейся левым верхним углом sprite. Следующее изображение иллюстрирует это:

ПРИМЕЧАНИЕ Эта функция влияет на актив спрайта так, что все последующие экземпляры с этим sprite будут иметь такое же смещение.

Синтаксис:

sprite_set_offset(ind, xoff, yoff);

АргументТип Описание
ind The index of the sprite to change the offset of.
xoff The x position of the origin.
yoff The y position of the origin.

Возвращает

Пример:

sprite_assign(spr_Custom, spr_Base);
sprite_set_offset(spr_Custom, sprite_get_xoffset(spr_Base), sprite_get_yoffset(spr_Base));

Приведенный выше код назначает sprite с индексом "spr_Base" на sprite с индексом "spr_Custom", а затем использует значения x и y смещения "spr_Base" для установки начала нового спрайта.
