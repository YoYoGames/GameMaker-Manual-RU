sprite_save_strip

Эта функция создает изображение полосы из всех подизображений в файле sprite, сохраняя его на диске с указанным именем. Файл sprite должен быть добавлен во время выполнения программы (нельзя сохранить sprites, добавленный через IDE), а файл должен быть сохранен с расширением *.png.

ПРИМЕЧАНИЕ: В зависимости от выбранной целевой платформы вы ограничены в том, откуда вы можете сохранять и загружать файлы. Дополнительную информацию см. в разделе Работа с файлами.

Синтаксис:

sprite_save_strip(ind, filename);

Аргумент Тип Описание
ind Актив спрайта The index of the sprite to save.
filename Строка The filename for the saved sprite strip.

Возвращает

N/A

Пример:

var spr, i;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16);

for (i = 1; i < 8; i +=1)
{
    sprite_add_from_surface(spr, i, 0, 32, 32, true, true, 16, 16);
}

sprite_save_strip(spr, "Custom_Player_Sprite.png");
sprite_delete(spr);

Приведенный выше код создает sprite из поверхности, индексированной в переменной "surf", присваивая ее индекс локальной переменной "spr_Custom", а затем использует цикл for для перемещения по поверхности и захвата различных участков, которые добавляются в sprite в качестве суб-изображений. этот новый sprite затем сохраняется как полоса png перед удалением из памяти.
