sprite_nineslice_create

Девятикратная нарезка - это техника, используемая для масштабирования прямоугольных изображений с целью сохранения деталей, благодаря чему они сохраняют свою первоначальную форму после масштабирования. Информацию о Девятикратном срезе см. на этой странице.

Эта функция используется для создания структуры Nine Slice, которая может быть изменена и затем применена к sprite. Содержимое этой структуры подробно описано на этой странице.

Вы можете хранить эту структуру в переменной, изменять ее свойства и применять ее к любому sprites с помощью sprite_set_nineslice().

Синтаксис:

sprite_nineslice_create();

Возвращается:

Nine Slice Struct

Пример:

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

Приведенный выше код создает новую структуру Nine Slice, включает ее и устанавливает смещения для направляющих. Затем эта структура применяется к объекту sprite, изменяя его свойства Nine Slice.
