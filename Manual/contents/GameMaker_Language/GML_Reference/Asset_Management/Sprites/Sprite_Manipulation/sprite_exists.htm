sprite_exists

Эта функция возвращает, существует или нет sprite с указанным индексом в выполняемом проекте.

Синтаксис:

sprite_exists(index);

АргументТип Описание
index The index of the sprite to check.

Возвращает

Пример:

if sprite_exists(spr_array[0])
{
    sprite_index = spr_array[0];
}
else
{
    sprite_index = spr_BaseSprite;
}

Приведенный выше код проверяет массив на наличие в нем допустимого индекса sprite, и если да, то присваивает экземпляру sprite, а если нет, то присваивает sprite из включенных ресурсов.
