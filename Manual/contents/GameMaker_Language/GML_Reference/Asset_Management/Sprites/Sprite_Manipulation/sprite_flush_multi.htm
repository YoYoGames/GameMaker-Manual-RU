sprite_flush_multi

С помощью этой функции вы можете удалить из текстурной памяти любое количество текстурных страниц для данного sprites. Вы передаете индексы sprite в виде массива, и текстурные страницы, которым они присвоены, будут удалены из текстурной памяти. Обратите внимание, что если одна из текстурных страниц используется где-либо в комнате (другим экземпляром sprite или фоном, и т.д.), вы можете получить небольшое снижение производительности, так как страница будет немедленно перезагружена обратно в память, поэтому при использовании этой функции следует соблюдать осторожность. Обратите внимание, что функция вернет -1, если flush не поддерживается для выбранных ресурсов, или вернет 0, если все работает правильно.

Синтаксис:

sprite_flush_multi(array);

АргументТип Описание
array Array with the sprite indices to flush

Возвращается:

(-1 or 0)

Пример:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_flush_multi(spr_a);

Приведенный выше код создает массив, каждый элемент которого содержит индекс sprite. Этот массив затем используется для очистки текстур sprite из памяти.
