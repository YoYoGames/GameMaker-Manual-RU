sprite_set_bbox_mode

Эта функция может быть использована для установки режима ограничивающей рамки для sprite. Вы предоставляете индекс sprite и режим, который должен быть одной из следующих констант:

Режим граничного поля Постоянная
Постоянная Описание
bboxmode_automatic Автоматически - Ограничительная рамка будет рассчитана автоматически, на основе настроек допуска для спрайта.
bboxmode_fullimage Полное изображение - Ограничительное поле будет установлено так, чтобы использовать всю ширину и высоту sprite, независимо от допусков и "пустых" пикселей.
bboxmode_manual Вручную - Граничная область устанавливается вручную на значения, заданные пользователем, что требует использования функции sprite_set_bbox()

ПРИМЕЧАНИЕ Эта функция влияет на актив спрайта так, что все последующие экземпляры с этим sprite будут иметь тот же режим ограничивающей рамки.

Синтаксис:

sprite_set_bbox_mode(ind, mode);

Аргумент Тип Описание
ind Актив спрайта The index of the sprite to change the mode of.
mode Режим граничного поля Постоянная The mode to set (a constant).

Возвращает

N/A

Пример:

if sprite_get_bbox_mode(sprite_index) != bboxmode_automatic
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

Приведенный выше код проверяет режим bbox для текущего sprite и, если это не automatic, то он устанавливается в это значение.
