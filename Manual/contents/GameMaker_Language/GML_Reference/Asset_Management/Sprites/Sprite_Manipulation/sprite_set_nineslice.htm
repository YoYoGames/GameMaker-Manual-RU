sprite_set_nineslice

Эта функция используется для применения структуры Nine Slice к sprite. Вы указываете индекс sprite и структуру Nine Slice для применения, которую можно получить с помощью функции sprite_nineslice_create() или sprite_get_nineslice().

ПРИМЕЧАНИЕ Эта функция влияет на актив спрайта, поэтому любые изменения, сделанные с помощью этой функции, повлияют на все экземпляры, использующие этот sprite, а также на все последующие экземпляры.

Синтаксис:

sprite_set_nineslice(ind, nineslice);

Аргумент Тип Описание
ind Актив спрайта The index of the sprite to modify
nineslice Структура из девяти ломтиков The Nine Slice struct to apply

Возвращается:

N/A

Пример:

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

Приведенный выше код создает новую структуру Nine Slice, включает ее и устанавливает смещения для направляющих. Затем эта структура применяется к объекту sprite, изменяя его свойства Nine Slice.
