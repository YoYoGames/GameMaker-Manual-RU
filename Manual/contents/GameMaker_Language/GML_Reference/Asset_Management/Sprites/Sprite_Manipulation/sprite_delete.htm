спрайт_удалить

Эта функция удаляет sprite из игры, освобождая всю память, которая была зарезервирована для него. Это очень важная функция для тех моментов, когда вам нужно создавать и изменять sprites из внешних источников (например, загружать sprite из файла с помощью функции sprite_add(), или дублирование sprite с помощью sprite_duplicate()) и всегда должна использоваться для удаления тех активов, которые больше не нужны. или для удаления индексированного актива из переменной перед повторным назначением другого актива в эту переменную. Это нельзя использовать для удаления sprites, которые включены в игру как часть активов в браузере активов.

Синтаксис:

sprite_delete(index);

АргументТип Описание
index The index of the sprite to be deleted.

Возвращает

Пример:

var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16);
sprite_merge(spr_Player, spr);
sprite_delete(spr);

Приведенный выше код создает локальную переменную и затем сохраняет индекс sprite, созданного на поверхности приложения. Затем этот sprite объединяется с активом, проиндексированным в переменной "spr_Player", после чего снова удаляется из памяти.
