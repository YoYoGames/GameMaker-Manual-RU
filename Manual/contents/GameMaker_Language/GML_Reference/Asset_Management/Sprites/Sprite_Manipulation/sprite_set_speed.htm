sprite_set_speed

Эта функция используется для установки базовой скорости и типа анимации для ресурса sprite. Обычно это задается в редакторе спрайтов, но бывают моменты, когда требуется задать это вручную для данного ресурса sprite. Вам необходимо указать индекс sprite, скорость анимации и тип скорости анимации. Тип скорости анимации может быть любой из следующих констант:

Постоянная Описание
spritespeed_framespersecond На сайте sprite была определена анимация в кадрах в секунду.
spritespeed_framespergameframe На сайте sprite была определена анимация в кадрах на один игровой кадр.


Выбор типа скорости, который будет использоваться для расчета анимации, очень важен, поскольку между этими двумя вариантами существует огромная разница. Следующие два примера иллюстрируют эту разницу:

  • Если у вас есть sprite, который рисует 1 кадр в секунду, а скорость изображения установлена на 0,5, он будет рисовать со скоростью 0,5 кадра в секунду. Если частота кадров в игре составляет 60 кадров в секунду, то sprite будет рисовать 1 кадр на каждые 120 игровых кадров.
  • Если у вас есть sprite, который рисует 1 кадр в игровом кадре, а скорость изображения установлена на 0,5, то он будет рисовать 0,5 кадра в игровом кадре. Если частота кадров в игре составляет 60 кадров в секунду, то sprite будет рисовать 30 кадров на каждые 60 игровых кадров.

Узнать, какое значение скорости использовалось для анимации sprite, можно с помощью функции sprite_get_speed(), а тип скорости анимации можно получить с помощью функции sprite_get_speed_type().

ВАЖНО! Эта функция влияет на ресурс sprite, поэтому любые изменения, которые вы сделаете с помощью этой функции, повлияют на все экземпляры, использующие sprite, а также на все будущие экземпляры.

Синтаксис:

sprite_set_speed(index, speed, type)

Аргумент Тип Описание
index Актив спрайта The index of the sprite to set the animation speed of
speed Реально The speed value to set
type Постоянная скорости спрайта The speed type to use

Возвращается:

Constant (see above)

Пример:

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

Приведенный выше код устанавливает базовую скорость анимации для sprite, назначенного экземпляру, на 10 кадров в секунду.
