Девять функций нарезки

В дополнение к разделу Nine Slice в редакторе спрайтов, GameMaker предоставляет функции для установки свойств Nine Slice для sprites во время выполнения. Прежде чем читать об этих функциях, рекомендуется прочитать эту страницу, чтобы получить представление о технике Nine Slice.

Следующие функции можно использовать для получения и изменения свойств Nine Slice для актива sprite во время выполнения:

  • sprite_nineslice_create
  • sprite_get_nineslice
  • sprite_set_nineslice

Эти функции используют структуру Nine Slice, о содержимом которой вы можете прочитать на этой странице.

Рисование спрайтов девяти ломтиков

Отдельных функций для рисования Nine Slice sprites не существует, так как они работают с существующими функциями рисования спрайтов (за исключением функций, которые рисуют только часть sprite или искажают его, например, draw_sprite_part() или draw_sprite_pos()).

Рекомендуется использовать draw_sprite_stretched() для рисования Nine Slice sprite, так как она позволяет указать абсолютные размеры sprite в пикселях и будет удобна в использовании вместе с Nine Slice. Вы также можете использовать draw_sprite_ext() которая вместо этого принимает значения масштаба.
