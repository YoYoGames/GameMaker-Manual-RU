skeleton_bone_data_get

Ваша скелетная анимация состоит из множества "костей", которые вы определили и дали им имена в своей программе анимации. Эта функция может быть использована для получения информации о названной кости в любое время и сохранения данных в пустой карте DS, которую вы должны были создать ранее. Обратите внимание, что эти данные относятся к позе по умолчанию для скелета, а не к текущей позе, которая рисуется. Если вам нужны данные о текущей позе, воспользуйтесь командой skeleton_bone_state_get().

Когда вы вызываете эту функцию, карта будет заполнена следующими ключами:

  • "x": Локальная x-позиция кости относительно родительской кости.
  • "y": Локальная позиция y кости относительно родительской кости.
  • "угол": Локальный поворот кости относительно родительской кости.
  • "xscale": Локальный горизонтальный масштаб кости по отношению к родительской кости.
  • "yscale": Локальный вертикальный масштаб кости по отношению к родительской кости.
  • "parent": Имя (строка) родительской кости.

Синтаксис:

skeleton_bone_data_get(bone, map);

Аргумент Тип Описание
bone Строка The name (as a string) of the bone.
map Идентификатор карты DS The (previously created) DS map that will get the bone data.

Возвращается:

N/A

Пример:

bone_map = ds_map_create();
skeleton_bone_data_get("leftarm", bone_map);

Приведенный выше код создает карту DS, а затем заполняет ее данными из кости с именем "leftarm".
