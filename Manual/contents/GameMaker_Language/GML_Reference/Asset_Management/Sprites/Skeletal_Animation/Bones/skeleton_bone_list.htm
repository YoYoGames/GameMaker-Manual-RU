скелетный_костный_список

С помощью этой функции вы можете заполнить (предварительно созданный) список DS всеми именами костей, используемых как часть скелетной анимации sprite. Имена будут строками и затем могут быть использованы в других функциях костей скелетной анимации для этих типов sprite.

Синтаксис:

skeleton_bone_list(sprite, list);

Аргумент Тип Описание
sprite Актив спрайта The sprite index of the Spine skeletal animation to get the list from.
list Идентификатор списка DS The ID of the DS list to populate with the bone names.

Возвращается:

N/A

Пример:

bone_list = ds_list_create();
skeleton_bone_list(sprite_index, bone_list);

Приведенный выше код создает список DS, затем заполняет его именами костей для последующего использования.
