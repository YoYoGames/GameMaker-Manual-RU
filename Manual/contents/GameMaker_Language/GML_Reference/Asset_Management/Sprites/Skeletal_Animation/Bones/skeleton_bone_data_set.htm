skeleton_bone_data_set

Ваша скелетная анимация состоит из ряда "костей", которые вы определили и дали им имена в вашей программе анимации, и эта функция может быть использована для установки определенных данных для названной кости в любое время. Обратите внимание, что эти данные относятся к позе по умолчанию для скелета, а не к текущей позе, которая рисуется (для этого используйте функцию skeleton_bone_state_set()), и должны быть установлены из ранее созданной карты DS, которая должна иметь следующие ключи и их эквивалентные значения:

  • "x": Локальная x-позиция кости относительно родительской кости.
  • "y": Локальная позиция y кости относительно родительской кости.
  • "угол": Локальный поворот кости относительно родительской кости.
  • "xscale": Локальный горизонтальный масштаб кости.
  • "yscale": Локальный вертикальный масштаб кости.
  • "parent": Имя (строка) родительской кости.

Синтаксис:

skeleton_bone_data_set(bone, map);

Аргумент Тип Описание
bone Строка The name (as a string) of the bone.
map Идентификатор карты DS The (previously created) DS map that stores the bone data.

Возвращается:

N/A

Пример:

var bone_map = ds_map_create();
skeleton_bone_data_get("head", bone_map);
ds_map_replace(bone_map, "parent", "body");
skeleton_bone_data_set("head", bone_map);
ds_map_destroy(bone_map);

Приведенный выше код создает карту DS, а затем заполняет ее данными из кости с именем "голова". Затем он заменяет ключ кости "parent" в карте новым значением и снова устанавливает кость "head" с новым набором данных.
