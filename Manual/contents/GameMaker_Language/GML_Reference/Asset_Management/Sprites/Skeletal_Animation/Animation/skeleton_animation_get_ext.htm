skeleton_animation_get_ext

Одна скелетная анимация sprite может иметь различные наборы анимации, и эти наборы могут быть назначены различным дорожкам, так что вы можете "смешивать и сопоставлять" наборы анимации. Эта функция возвращает имя набора анимации, используемого в данный момент на заданном номере дорожки (заданном функцией skeleton_animation_set_ext).

Синтаксис:

skeleton_animation_get_ext(track);

Аргумент Тип Описание
track Реально The track number to get the animation set name of.

Возвращается:

String

Пример:

if skeleton_animation_get_ext(1) != "bodyfight"
{
    skeleton_animation_set_ext("bodyfight", 1);
}

Приведенный выше код изменит набор анимации, используемый дорожкой 1, на "bodyfight", если это не так.
