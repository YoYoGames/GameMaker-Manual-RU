skeleton_animation_set_frame

Эта функция установит анимацию, назначенную на данный трек, на указанный вами кадр. Индекс кадра должен быть вещественным значением от 0 до количества кадров анимации (которое можно получить с помощью функции skeleton_animation_get_frames). Если вы зададите значение вне этого диапазона, анимация будет зациклена, чтобы компенсировать разницу, например, если анимация имеет 5 кадров, а вы зададите кадр 8, то реальный кадр будет 3.

Синтаксис:

skeleton_animation_set_frame(track, index);

Аргумент Тип Описание
track Реально The animation track to set the frame of.
index Реально The new index for the animation frame.

Возвращается:

N/A

Пример:

if global.pause
{
    skeleton_animation_set_frame(0, -image_speed);
}

Приведенный выше код установит индекс кадра для анимации на дорожке 0 на отрицательную скорость изображения, когда глобальная переменная "pause" будет true (по сути, приостанавливая анимацию).
