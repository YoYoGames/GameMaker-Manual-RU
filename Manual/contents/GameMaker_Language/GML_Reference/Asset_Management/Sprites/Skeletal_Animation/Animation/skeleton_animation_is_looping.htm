skeleton_animation_is_looping

С помощью этой функции можно проверить, установлена ли текущая анимация на заданном треке в режим цикла.

Синтаксис:

skeleton_animation_is_looping(track);

Аргумент Тип Описание
трек Реально Индекс дорожки анимации

Возвращается:

Boolean

Пример:

if skeleton_animation_is_looping(0)
{
    show_debug_message("Track 0 of the skeleton animation is looping");
}

Приведенный выше код проверяет, установлена ли анимация на дорожке 0 в режим цикла. Если это так, то будет показано отладочное сообщение.
