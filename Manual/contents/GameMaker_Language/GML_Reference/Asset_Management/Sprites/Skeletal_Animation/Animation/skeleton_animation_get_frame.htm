skeleton_animation_get_frame

Эта функция возвращает номер кадра анимации на указанной дорожке анимации. Функция вернет -1, если на указанную дорожку не назначена анимация.

Синтаксис:

skeleton_animation_get_frame(track);

Аргумент Тип Описание
track Реально The animation track to get the frame of.

Возвращается:

Real

Пример:

var frame = skeleton_animation_get_frame(0);
if frame > 0 && frame < 2
{
    var box = skeleton_get_minmax();
    var ww = (box[2] - box[0]) / 2;
    var hh = (box[3] - box[1]) / 2;
    part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);
}

Приведенный выше код проверит текущий кадр анимации, назначенный дорожке 0, а затем вырвет несколько частиц из точки в середине общей граничной области для sprite.
