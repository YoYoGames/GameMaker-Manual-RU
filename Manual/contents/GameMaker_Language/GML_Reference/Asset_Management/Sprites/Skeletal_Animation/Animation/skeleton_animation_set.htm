skeleton_animation_set

После того как вы назначили скелетную анимацию sprite своему экземпляру, вы можете определить, какой набор анимаций он должен использовать в любой момент времени с помощью этой функции. При создании своего sprite (в Spine) вы определили несколько наборов анимации для различных состояний и присвоили каждому из них соответствующее имя. Эти имена вы будете использовать для того, чтобы указать программе GameMaker, какой набор анимации использовать в любой момент времени в вашей игре. Например, если ваш sprite предназначен для платформенной игры, то вы можете иметь набор "прыжок", набор "бег" и набор "бездействие", содержащиеся в одном sprite, и вы можете переключаться между ними в вашей игре с помощью этой функции.
Анимация воспроизводится по кругу, что является стандартом.

Обратите внимание, что установка новой анимации таким образом приведет к сбросу параметра image_index на 0, запуская новый набор анимации с начала цикла анимации. Если такое поведение нежелательно, можно задать значение "mix" с помощью функции skeleton_animation_mix() и при установке новой анимации она будет плавно интерполироваться с предыдущим набором. Также следует убедиться, что в начале каждой анимации стоит ключевой кадр, чтобы данные не задерживались в предыдущих анимациях, если вы не планируете использовать отдельные анимационные дорожки.

Синтаксис:

skeleton_animation_set(animname, [loop]);

Аргумент Тип Описание
animname Строка The name (a string) of the animation set to use.
loop Булево Должна ли анимация зацикливаться или нет (по умолчанию true)

Возвращается:

N/A

Пример:

if keyboard_check_pressed(vk_space)
{
    if skeleton_animation_get() != "jump"
    {
        skeleton_animation_set("jump");
    }
}

Приведенный выше код изменит используемый набор анимации на набор "прыжок" при нажатии клавиши пробела, но только если текущий используемый набор еще не "прыжок".
