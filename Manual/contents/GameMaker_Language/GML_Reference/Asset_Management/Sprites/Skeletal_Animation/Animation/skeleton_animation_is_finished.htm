skeleton_animation_is_finished

С помощью этой функции можно проверить, закончилось ли воспроизведение текущей не зацикленной скелетной анимации на заданной дорожке.

Синтаксис:

skeleton_animation_is_finished(track);

Аргумент Тип данных Описание
трек Реально Индекс дорожки анимации

Возвращается:

Boolean

Пример:

var _finished = skeleton_animation_is_finished(1);
if _finished
{
    show_debug_message("The animation has finished playing");
}

Приведенный выше код сначала проверяет, закончилась ли анимация на дорожке 1, и сохраняет результат в локальной переменной _finished. Затем он выводит отладочное сообщение в случае, если анимация действительно завершила воспроизведение.
