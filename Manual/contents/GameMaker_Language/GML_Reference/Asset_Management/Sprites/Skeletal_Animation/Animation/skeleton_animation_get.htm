skeleton_animation_get

С помощью этой функции вы можете получить текущий набор анимации, используемый вашей программой скелетной анимации sprite. Возвращаемое значение - строка, которая будет именем набора, как вы определили его в вашей программе скелетной анимации.

Синтаксис:

skeleton_animation_get();

Возвращается:

Пример:

if keyboard_check_pressed(vk_space)
{
    if (skeleton_animation_get() != "jump")
    {
        skeleton_animation_set("jump");
    }
}

Приведенный выше код изменит используемый набор анимации на набор "прыжок" при нажатии клавиши пробела, но только если текущий используемый набор еще не "прыжок".
