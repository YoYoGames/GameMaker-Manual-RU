список_анимации_скелета

С помощью этой функции вы можете заполнить (предварительно созданный) список DS всеми именами анимаций, включенных как часть скелетной анимации sprite. Эти имена будут строками и затем могут быть использованы в других функциях анимации для этих типов sprite.

Синтаксис:

skeleton_animation_list(sprite, list);

Аргумент Тип Описание
sprite Актив спрайта The sprite index of the Spine skeletal animation to get the list from.
list Идентификатор списка DS The ID of the DS list to populate with the animation names.

Возвращается:

N/A

Пример:

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);

for (var i = 1; i < ds_list_size(list); i++)
{
    skeleton_animation_mix(list[| 0], list[| i], 0.5);
}

ds_list_destroy(list);

Приведенный выше код создает список DS всех имен анимаций для sprite, используемых данным экземпляром. Затем он перебирает их и устанавливает значение mix для всех анимаций с первой анимацией равным 0,5.
