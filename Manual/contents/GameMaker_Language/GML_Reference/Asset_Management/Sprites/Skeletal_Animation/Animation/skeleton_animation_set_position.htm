skeleton_animation_set_position

Эта функция установит анимацию, назначенную данной дорожке, в указанное вами положение. Позиция должна быть реальным значением между 0 и 1. Если вы предоставите значение вне этого диапазона, то позиция дорожки будет установлена на дробную часть этого значения. Предположим, вы используете значение 0,4, значение 1,4, значение 2,4, ... Все они установят положение на 0,4.

Синтаксис:

skeleton_animation_set_position(track, position);

Аргумент Тип Описание
track Реально The animation track to set the frame of.
position Реально The new position for the animation frame.

Возвращается:

N/A

Пример:

skeleton_animation_set_position(1, 0.25);

Приведенный выше код установит позицию для дорожки 1 анимации скелета на 0,25 или, говоря иначе, на 25% от начала анимации.
