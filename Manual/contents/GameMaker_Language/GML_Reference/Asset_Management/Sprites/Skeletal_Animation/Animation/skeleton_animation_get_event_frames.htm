skeleton_animation_get_event_frames

С помощью этой функции можно получить все кадры для заданного события в заданной анимации. Вы передаете имя скелетной анимации (в виде строки, как определено в программе, используемой для создания анимации, или как возвращено с помощью функции skeleton_animation_get()) и имя события. Функция возвращает массив с кадрами для данного события.

Если указанное имя события не существует, функция вернет массив с -1 в качестве первого (и единственного) элемента.

Синтаксис:

skeleton_animation_get_event_frames(anim_name, event_name);

Аргумент Тип Описание
anim_name Строка The animation name to get the frames of.
event_name Строка The event name to get the frames for.

Возвращается:

Array of Reals

Пример:

var _frames = skeleton_animation_get_event_frames(skeleton_animation_get(), "Footstep");

if (_frames[0] != -1)
{
    var _count = array_length(_frames);
    var _current_frame = skeleton_animation_get_frame(0);

    for (var i = 0; i < _count; i ++)
    {
        if (_frames[i] == _current_frame)
        {
            audio_play_sound(snd_footstep, 1, false);
            break;
        }
    }
}

Приведенный выше код получает кадры для события "Footstep". Если он вернул массив с действительными кадрами, то он проходит по нему и проверяет, равен ли текущий кадр любому из кадров в массиве. В этом случае он воспроизводит звуковой эффект шагов и прерывает цикл.
