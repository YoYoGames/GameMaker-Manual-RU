skeleton_animation_set_ext

Когда вы устанавливаете набор анимации для скелетной анимации sprite, ему назначается дорожка 0. Однако вы можете назначить другие наборы на другие дорожки (1 и выше) и таким образом заставить ваш sprite воспроизводить две (или более) анимации одновременно. Обратите внимание, что sprite должен быть разработан в вашей программе анимации с учетом этого, так, чтобы "базовая" анимация на дорожке 0 имела ключевые кадры только для определенных костей, в то время как следующие дорожки имели ключевые кадры для остальных. Например, если у вас есть анимация "ходьбы", и ее ключевые кадры воздействуют только на ноги sprite, то вы можете иметь набор анимаций головы и рук, которые вы можете установить на дорожках 1 и 2 соответственно, что позволит вам иметь базовое движение и адаптироваться к нему в реальном времени во время игры.
Анимация воспроизводится в циклическом режиме, который используется по умолчанию.

Синтаксис:

skeleton_animation_set_ext(animname, track, [loop]);

Аргумент Тип Описание
animname Строка The name (a string) of the animation set to use.
track Реально The track for the animation set to use.
loop Булево Должна ли анимация зацикливаться или нет (по умолчанию true)

Возвращается:

N/A

Пример:

if mouse_check_button(mb_left)
{
    skeleton_animation_set_ext("bodyfight", 1);
}
else
{
    skeleton_animation_set_ext("bodywalk", 1);
}

Приведенный выше код изменит набор анимации, используемый дорожкой 1, на набор "bodyfight" или "bodywalk" при нажатии левой кнопки мыши.
