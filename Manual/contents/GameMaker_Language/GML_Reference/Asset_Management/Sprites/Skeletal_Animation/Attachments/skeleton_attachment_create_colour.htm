skeleton_attachment_create_colour

Когда вы создаете скелетную анимацию sprite, вы можете назначить слоты для вложений и вложения в них. Это просто изображения (sprites), которые отделены от анимации, но при прикреплении будут двигаться вместе с костью, к которой они прикреплены. Обычно вы назначаете эти вложения в вашей программе анимации (Spine), но с помощью этой функции вы можете создавать свои собственные вложения во время выполнения, используя актив sprite из вашей игры. Функция требует, чтобы вы дали вложению имя (в виде строки), а затем установили параметры sprite_index и image_index для использования, а также начало координат x и y (которое может отличаться от того, которое определено для sprite в свойствах sprite ), и вы также можете задать любые преобразования, которые вы хотите применить к изображению при вложении, а также цвет, который будет смешиваться с изображением, и его альфа-значение (прозрачность). Если вложение было успешно создано, функция вернет 1, а если нет (вы указали неверный индекс sprite или базовый sprite не является Spine sprite), то вернется -1.

Стоит отметить, что экземпляр sprite может иметь цвет наложения и настройки альфа (image_blend и image_alpha), как и слот вложения (см. функцию skeleton_slot_colour_set()), и поэтому окончательный цвет и альфа, которые будет иметь назначенное вложение sprite для слота, будет представлять собой комбинацию всех этих значений.

Синтаксис:

skeleton_attachment_create_colour(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot, colour, alpha);

Аргумент Тип Описание
name Строка The name (as a string) of the attachment to create.
sprite Актив спрайта The sprite_index to get the attachment image from.
ind Реально The image_index to get the attachment image from.
xorigin Реально The x origin for the image being used.
yorigin Реально The y origin for the image being used.
xscale Реально The horizontal scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Реально The vertical scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Реально The rotation of the image: 0 = normal, 90 = turned 90° counter-clockwise etc.
Colour Цвет The colour value to use (A constant, integer or hex value).
Alpha Реально The alpha value to use (from 0 to 1).

Возвращается:

Real (1 if successful, -1 if not)

Пример:

skeleton_attachment_create_colour("sword", spr_Weapons, 0, 0, 80, 1, 1, 90, c_red, 1);
skeleton_attachment_create_colour("knife", spr_Weapons, 1, 0, 45, 1, 1, 90, c_yellow, 1);
skeleton_attachment_create_colour("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0, c_white, 0.5);
skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow"));

Приведенный выше код проверит текущее назначенное имя вложения для слота с именем "slot_leftHand", и если будет возвращена пустая строка, то будет прикреплен новый sprite.
