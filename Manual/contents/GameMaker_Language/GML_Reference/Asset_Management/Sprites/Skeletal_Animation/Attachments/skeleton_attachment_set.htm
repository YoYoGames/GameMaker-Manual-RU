skeleton_attachment_set

Скелетная анимация sprite может содержать другие изображения, добавленные в качестве вложений, причем эти изображения добавляются в названный слот (имя задается при создании слота вложения в вашей программе анимации), и они будут рисоваться вместе с анимацией текущего sprite. С помощью этой функции можно установить вложение в заданный слот, при этом необходимо указать имена (в виде строк) слота и вложения. Эти имена определяются используемой программой анимации или (в случае вложения) при вызове функции skeleton_attachment_create().

Обратите внимание, что вы также можете передать sprite_index в качестве вложения, и тогда будет использован sprite, или вы можете использовать -1, чтобы удалить вложение из слота. Когда вы передаете в качестве аргумента индекс sprite, будет создан слот вложения, используя имя sprite в качестве строки для названия слота (например, при использовании spr_sword будет создан слот вложения "spr_sword"), и слот будет использовать индекс первого изображения (0) из sprite, а также смещение начала координат x и y, с масштабом (1,1) и вращением 0.

Синтаксис:

skeleton_attachment_set(slot, attachment);

Аргумент Тип Описание
slot Строка The slot name (a string) to get the attachment of.
attachment Строка или ассет спрайта The name (as a string or a sprite_index) of the attachment image.

Возвращается:

String

Пример:

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

Приведенный выше код проверяет текущее назначенное имя вложения для слота с именем "slot_leftHand", и если возвращается пустая строка, то прикрепляется новый sprite.
