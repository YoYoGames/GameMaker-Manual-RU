skeleton_attachment_get

Скелетная анимация sprite может иметь другие sprites, добавленные в качестве вложений, причем эти sprites добавляются в названный слот (имя задается при создании слота вложения в вашей анимационной программе), и они будут рисоваться вместе с анимацией текущего sprite. С помощью этой функции вы можете получить имя (в виде строки) вложения для данного слота текущего назначенного sprite. Обратите внимание, что ссылка на прикрепленные sprites осуществляется через их строку имени, заданную в Spine, или когда вы вызвали функцию skeleton_attachment_create().

Синтаксис:

skeleton_attachment_get(slot);

Аргумент Тип Описание
slot Строка The slot name (a string) to get the attachment of.

Возвращается:

String

Пример:

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

Приведенный выше код проверяет текущее назначенное имя вложения для слота с именем "slot_leftHand", и если возвращается пустая строка, то прикрепляется новый sprite.
