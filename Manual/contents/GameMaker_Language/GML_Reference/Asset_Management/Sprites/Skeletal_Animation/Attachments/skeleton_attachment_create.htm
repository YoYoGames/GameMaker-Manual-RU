skeleton_attachment_create

Когда вы создаете скелетную анимацию sprite, вы можете назначить слоты для вложений и вложения в них. Это просто изображения, которые находятся отдельно от анимации, но при прикреплении будут двигаться вместе с костью, к которой они прикреплены. Обычно вы назначаете эти вложения в вашей программе анимации (Spine), но с помощью этой функции вы можете создавать свои собственные вложения во время выполнения, используя актив sprite из вашей игры. Функция требует, чтобы вы дали вложению имя (в виде строки), затем установили sprite_index и image_index для использования, а также x и y origin (которые могут отличаться от тех, что определены в sprite в свойствах sprite ), и вы также можете установить любые преобразования, которые вы хотите применить к изображению при вложении. Если вложение было успешно создано, функция вернет 1, а если нет (вы указали неверный индекс sprite, или основание sprite не является Spine sprite), то вернется -1.

Синтаксис:

skeleton_attachment_create(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot);

Аргумент Тип Описание
name Строка The name (as a string) of the attachment to create.
sprite Актив спрайта The sprite_index to get the attachment image from.
ind Реально The image_index to get the attachment image from.
xorigin Реально The x origin for the image being used.
yorigin Реально The y origin for the image being used.
xscale Реально The horizontal scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Реально The vertical scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Реально The rotation of the image. 0=normal, 90=turned 90 degrees counter-clockwise etc.

Возвращается:

Real (1 if successful, -1 if not)

Пример:

skeleton_attachment_create("sword", spr_Weapons, 0, 0, 80, 1, 1, 90);
skeleton_attachment_create("knife", spr_Weapons, 1, 0, 45, 1, 1, 90);
skeleton_attachment_create("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0);
skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow"));

Приведенный выше код проверит текущее назначенное имя вложения для слота с именем "slot_leftHand", и если будет возвращена пустая строка, то будет прикреплен новый sprite.
