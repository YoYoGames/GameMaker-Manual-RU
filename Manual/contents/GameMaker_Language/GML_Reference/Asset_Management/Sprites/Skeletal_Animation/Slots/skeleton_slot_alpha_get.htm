skeleton_slot_alpha_get

С помощью этой функции вы можете получить значение альфа слота вложения. Вы указываете имя слота (строка), а функция возвращает реальное значение альфа в диапазоне от 0 до 1.

Синтаксис:

skeleton_slot_alpha_get(slot);

Аргумент Тип Описание
slot Строка The slot name to check, a string

Возвращается:

Real

Пример:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_alpha_get(slot_list[| i]) != 1
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

Приведенный выше код создает список DS, затем заполняет его именами слотов, найденных по положению мыши в комнате. Затем он проходит по списку слотов и сбрасывает цвет для найденных слотов на белый с альфой 1, если альфа еще не равна 1.
