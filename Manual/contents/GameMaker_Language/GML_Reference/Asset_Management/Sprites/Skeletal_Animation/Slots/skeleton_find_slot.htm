скелет_найти_слот

С помощью этой функции вы можете найти, какие слоты находятся в заданной позиции пространства комнаты в Spine sprite, связанном с текущим экземпляром. Вы создаете список DS и предоставляете его ID вместе с позицией x/y для проверки, и список будет заполнен строкой имен для каждого из доступных слотов крепления для sprite (включая любые модификации крепления). Обратите внимание, что список всегда сортируется в порядке убывания, начиная с самого верхнего слота.

Синтаксис:

skeleton_find_slot(x, y, list);

Аргумент Тип Описание
x Реально The x position in the room to check.
y Реально The y position in the room to check.
list Идентификатор списка DS The ID of the DS list to populate with the DS maps.

Возвращается:

N/A

Пример:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

Приведенный выше код создает список DS, а затем заполняет его именами слотов, найденных при положении мыши в комнате.
