skeleton_slot_colour_set

С помощью этой функции можно задать цвет и альфа слота вложения, так что все sprites, присоединенные к нему, будут рисоваться с этими значениями наложения. Следует помнить, что экземпляр sprite может иметь настройки цвета наложения и альфы (image_blend и image_alpha), как и вложение (см. функцию skeleton_attachment_create_colour()), и поэтому окончательный цвет и альфа, которые будут у назначенного вложения sprite для этого слота, будут представлять собой комбинацию всех этих значений.

Синтаксис:

skeleton_slot_colour_set(slot, colour, alpha);

Аргумент Тип Описание
slot Строка The slot name to set, a string
colour Цвет The colour to set, either an integer, a constant, or a hex value.
alpha Реально The alpha to set from 0 to 1.

Возвращается:

N/A

Пример:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if (!ds_list_empty(slot_list))
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_colour_get(slot_list[| i]) != c_white
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

Приведенный выше код создает список DS, затем заполняет его названиями слотов, найденных по положению мыши в комнате. Затем он проходит по списку слотов и сбрасывает цвет для найденных слотов на белый с альфой 1, если цвет еще не белый.
