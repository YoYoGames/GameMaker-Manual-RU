skeleton_slot_data

С помощью этой функции вы можете заполнить (предварительно созданный) список DS данными для каждого из доступных слотов крепления, которые содержит анимация. Эти данные возвращаются в виде карты DS, которая содержит следующие пары ключ/значение:

  • "имя": Название слота.
  • "кость": Название кости.
  • "attachment": Имя вложения, если оно используется, или "(none)", если оно не назначено.

Значения для каждого ключа будут строками и затем могут быть использованы в других скелетных функциях вложения для этих типов sprite. Обратите внимание, что созданные карты DS не уничтожаются, поэтому вам придется пройтись по списку DS и уничтожить каждую из созданных карт самостоятельно.

Синтаксис:

skeleton_slot_data(sprite, list);

Аргумент Тип Описание
sprite Актив спрайта The sprite index of the Spine skeletal animation to get the data from.
list Идентификатор списка DS The ID of the DS list to populate with the DS maps.

Возвращается:

N/A

Пример:

var list = ds_list_create();
var open = true;
slot_name = "";
skeleton_slot_data(sprite_index, list);

for (var i = 0; i < ds_list_size(list); i++)
{
    var map = list[| i];
    if open
    {
        if map[? "attachment"] == "(none)"
        {
            open = false;
            slot_name = map[? "name"];
        }
    }

    ds_map_destroy(map);
}

ds_list_destroy(list);

Приведенный выше код создает список DS, а затем заполняет его данными слота для экземпляра sprite. Затем эти данные анализируются для извлечения отдельных карт DS с данными слота. Затем проверяется, есть ли пустой слот, и если да, то переменной "имя_слота" присваивается имя пустого слота перед уничтожением карты DS. Наконец, мы уничтожаем список DS, поскольку он больше не нужен.
