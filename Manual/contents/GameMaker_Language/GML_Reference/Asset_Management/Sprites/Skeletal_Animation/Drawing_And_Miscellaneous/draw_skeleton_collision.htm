draw_skeleton_collision

Эта функция рисует маски столкновений, связанные с заданной скелетной анимацией. Вы предоставляете базовый sprite, набор анимации для использования и кадр, из которого нужно получить информацию, а также можете установить свойства трансформации.

Синтаксис:

draw_skeleton_collision(sprite, animname, frame, x, y, xscale, yscale, rot, colour)

Аргумент Тип Описание
sprite Актив спрайта The index of the sprite to draw.
animname Строка The name of the animation to get the frame from (a string).
frame Реально The animation frame to draw (from 0 to image_number - 1).
x Реально The x coordinate of where to draw the sprite.
y Реально The y coordinate of where to draw the sprite.
xscale Реально The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Реально The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Реально The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour Цвет The colour with which to blend the sprite.

Возвращается:

N/A

Пример:

draw_skeleton_collision(sprite_index, "jump", image_index, x, y, image_xscale, image_yscale, image_angle, c_white);

Приведенный выше код нарисует данные маски столкновений для текущего sprite, используя текущие преобразования, для анимации с набором "прыжок".
