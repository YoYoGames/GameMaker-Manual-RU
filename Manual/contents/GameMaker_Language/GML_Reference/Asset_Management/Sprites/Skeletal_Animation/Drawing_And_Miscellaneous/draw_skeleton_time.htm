рисовать_скелет_время

Эта функция отрисует заданную анимацию с использованием заданного скина в определенное время анимации. Значение времени должно находиться между 0 (начало) и конечной продолжительностью анимации, которую можно найти с помощью функции skeleton_animation_get_duration(). Вы можете установить значение времени больше, чем общая продолжительность анимации, и анимация вернется к началу, но при этом вы рискуете потерять точность с плавающей запятой при увеличении накопленного времени.

Синтаксис:

draw_skeleton_time(sprite, animname, skinname, time, x, y, xscale, yscale, rot, colour)

Аргумент Тип Описание
sprite Актив спрайта The index of the sprite to draw.
animname Строка The name of the animation to get the frame from (a string).
skinname Строка The name of the skin to use for drawing (a string).
time Реально The time-frame to draw the animation at (from 0 to the end duration, in seconds).
x Реально The x coordinate of where to draw the sprite.
y Реально The y coordinate of where to draw the sprite.
xscale Реально The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Реально The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Реально The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour Цвет The colour with which to blend the sprite.

Возвращается:

N/A

Пример:

var time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

Приведенный выше код отрисует заданную скелетную анимацию sprite, используя дельта-время для установки рисуемого кадра.
