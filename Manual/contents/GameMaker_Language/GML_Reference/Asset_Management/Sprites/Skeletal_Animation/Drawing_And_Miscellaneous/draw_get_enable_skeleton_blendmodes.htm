draw_get_enable_skeleton_blendmodes

Эта функция возвращает, включены(true) или отключены(false) режимы наложения на каждый слот для скелета sprites.

Синтаксис:

draw_get_enable_skeleton_blendmodes()

Возвращается:

Boolean

Пример:

if (!draw_get_enable_skeleton_blendmodes())
{
    draw_enable_skeleton_blendmodes(true);
}

Приведенный выше код включает режимы наложения на каждый слот для скелетного sprites, если они отключены.
