skeleton_get_bounds

Эта функция возвращает массив значений, связанных с любой заданной граничной рамкой для текущей назначенной скелетной анимации sprite. Вы вводите индексный номер ограничивающей рамки для получения подробной информации (вы можете получить общее количество ограничивающих рамок для sprite с помощью функции skeleton_get_num_bounds()), и функция вернет массив со следующими элементами:

  • [0] - количество точек на ограничивающей рамке (вещественное)
  • [1] - имя ограничивающей рамки (строка)
  • [2] - положение x первой точки
  • [3] - положение y первой точки
  • [и т.д.] - дальнейшие данные о положении x/y до количества точек на ограничительной рамке

Синтаксис:

skeleton_get_bounds(index);

Аргумент Тип Описание
index Реально The bounding box index to get the values of.

Возвращается:

Array (minimum 2 elements: numPoints, name [, xPos, yPos, etc...])

Пример:

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++)
{
    var b_info = skeleton_get_bounds(i);
    if b_info[0] > 0
    {
        var data = b_info[1] + ":";
        for(var j = 0; j < b_info[0]; j++)
        {
            data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
        }
        draw_text(20, yy, data);
        yy += 20;
    }
}

Приведенный выше код будет циклически просматривать все ограничивающие рамки, связанные с текущим назначенным sprite, а затем выводить эти данные в виде строки на экран.
