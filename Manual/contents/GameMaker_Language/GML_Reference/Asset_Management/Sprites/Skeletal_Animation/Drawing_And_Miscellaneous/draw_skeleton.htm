нарисовать_скелет

Эта функция предназначена только для использования с sprites, которые были созданы с помощью программы скелетной анимации, например Spine. Хотя вы можете рисовать эти sprites с помощью обычных функций рисования (например, таких как draw_self()), бывают случаи, когда необходимо нарисовать один кадр или показать изменение текстуры кожи, не изменяя свойств sprite. В таких случаях вы можете нарисовать sprite с помощью этой функции, где вы указываете индекс sprite, затем имя анимации, из которой нужно взять кадр (строка, заданная в программе, используемой для создания sprite). Затем вы указываете имя скина, который нужно использовать (опять же, заданное при создании sprite ), и значение кадра. Значение кадра указывает программе GameMaker, какую часть анимации нужно показать, и представляет собой значение от 0 до image_number. Остальные аргументы аналогичны тем, которые используются при обычном рисовании sprite, и изменяют масштаб, угол, цвет наложения и альфа-коэффициент конечного рисуемого кадра анимации.

ВНИМАНИЕ! Из-за того, что скелетная анимация интерполируется между различными "ключевыми" кадрами, существует дополнительная нагрузка, связанная с этим вызовом, поэтому рекомендуется избегать рисования скелетной анимации sprites с помощью этого метода, если это не является абсолютно необходимым.

Синтаксис:

draw_skeleton(sprite, animname, skinname, frame, x, y, xscale, yscale, rot, colour, alpha);

Аргумент Тип Описание
sprite Актив спрайта The index of the sprite to draw.
animname Строка The name of the animation to get the frame from (a string).
skinname Строка The name of the skin to use (a string).
frame Реально The animation frame to draw (from 0 to image_number - 1).
x Реально The x coordinate of where to draw the sprite.
y Реально The y coordinate of where to draw the sprite.
xscale Реально The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Реально The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Реально The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour Цвет The colour with which to blend the sprite.
alpha Реально The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Возвращается:

N/A

Пример:

draw_skeleton(spr_Jump, "jump", "skin1", 0, x, y, image_xscale, image_yscale, image_angle, c_white, 0.5);

Приведенный выше код нарисует первый кадр заданной скелетной анимации sprite, используя набор анимации "jump", скин, определенный как "skin1", в том же масштабе и положении, как определено в свойствах экземпляра, но с альфой 0,5.
