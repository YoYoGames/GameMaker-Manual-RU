skeleton_collision_draw_set

С помощью этой функции можно включить (true) или выключить (false) отрисовку данных столкновений для текущей скелетной анимации sprite, используемой экземпляром. Если эта функция включена, ограничивающая рамка и точная маска столкновений будут нарисованы в виде контуров вокруг sprite.

Эта функция работает только в том случае, если рисование скелета sprite выполняется объектом. Чтобы нарисовать ограничивающую рамку столкновения при ручном рисовании скелета sprite, используйте draw_skeleton_collision().

Синтаксис:

skeleton_collision_draw_set(flag);

Аргумент Тип Описание
flag Булево Set to true чтобы включить рисование, и false, чтобы выключить его.

Возвращается:

N/A

Пример:

if debug_mode == true
{
    skeleton_collision_draw_set(true);
}

Приведенный выше код проверяет, запущена ли игра в режиме отладки, и если да, то переключает данные скелетных столкновений для экземпляра, который будет показан.
