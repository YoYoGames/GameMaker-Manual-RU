skeleton_get_num_bounds

Эта функция возвращает количество граничных боксов, связанных со скелетной анимацией sprite, назначенной экземпляру, выполняющему код. Затем она может быть использована вместе с функцией skeleton_get_bounds() для получения данных о каждом из ограничивающих боксов.

Синтаксис:

skeleton_get_num_bounds();

Возвращается:

Real

Пример:

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++)
{
    var b_info = skeleton_get_bounds(i);
    if b_info[0] > 0
    {
        var data = b_info[1] + ":";
        for(var j = 0; j < b_info[0]; j++)
        {
            data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
        }
        draw_text(20, yy, data);
        yy += 20;
    }
}

Приведенный выше код будет циклически просматривать все ограничивающие рамки, связанные с текущим назначенным sprite, а затем выводить эти данные в виде строки на экран.
