skeleton_skin_get

С помощью скелетной анимации sprites можно назначить анимации отдельные текстуры (называемые "скинами"), тем самым используя одну анимацию для нескольких различных действий. Эта функция возвращает имя скина (в виде строки), который в настоящее время назначен скелетной анимации sprite, используемой вашим экземпляром. Возвращаемое имя - это имя, которое вы задали при создании sprite в вашей программе анимации.

Синтаксис:

skeleton_skin_get();

Возвращается:

Пример:

if skeleton_skin_get() == "skin_Enemy1"
{
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

Приведенный выше код проверяет текущий назначенный скин для анимации, и если это "skin_Enemy1", он выбирает и устанавливает новый скин из одного из трех вариантов.
