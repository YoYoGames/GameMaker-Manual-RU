список_кожи_скелета

С помощью этой функции вы можете заполнить (предварительно созданный) список DS всеми именами скинов, входящих в состав скелетной анимации sprite. Эти имена будут строками и затем могут быть использованы в других функциях скинов скелетной анимации для этих типов sprite.

Синтаксис:

skeleton_skin_list(sprite, list);

Аргумент Тип Описание
sprite Актив спрайта The sprite index of the Spine skeletal animation to get the list from.
list Идентификатор списка DS The ID of the DS list to populate with the animation names.

Возвращается:

N/A

Пример:

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1));
ds_list_destroy(list);

Приведенный выше код создает список DS, затем заполняет его именами скинов. Затем выбирается случайный скин и применяется к экземпляру перед уничтожением списка.
