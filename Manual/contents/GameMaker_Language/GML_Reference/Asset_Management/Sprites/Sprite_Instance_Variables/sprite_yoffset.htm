sprite_yoffset

Эта переменная, доступная только для чтения, возвращает локальное смещение y (компонент y начала координат, определенный в редакторе sprite ) объекта sprite, который был назначен данному экземпляру. Смещение по оси y возвращается в пикселях и зависит от размера объекта. image_yscaleтак что (например) если квадрат 32x32 sprite определен как имеющий начало координат y на 16, то при масштабировании этого sprite на 2 эта переменная вернет 32. Если вам нужно немасштабированное смещение y, используйте sprite_get_yoffset().

Синтаксис:

sprite_yoffset;

Возвращается:

Real

Пример:

if y - sprite_yoffset < 0
{
    y = sprite_yoffset;
}

Приведенный выше код проверяет, находится ли верхний край sprite экземпляра за пределами комнаты, и если да, то перемещает экземпляр так, чтобы был виден весь sprite.
