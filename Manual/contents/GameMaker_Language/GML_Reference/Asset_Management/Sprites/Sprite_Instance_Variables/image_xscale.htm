масштаб_изображения

Это значение задает горизонтальное масштабирование, применяемое к sprite, назначенному текущему экземпляру. Значение 1 означает отсутствие масштабирования (1:1), меньшие значения уменьшают масштаб (0.5, например, уменьшает ширину sprite в два раза), большие значения увеличивают масштаб, а отрицательные значения переворачивают sprite и масштабируют его, если только используемое значение не равно -1 (в этом случае sprite просто переворачивается относительно своего начала без масштабирования).

Синтаксис:

image_xscale;

Возвращается:

(single precision floating point value)

Пример:

if image_xscale < 5
{
    image_xscale += 0.2;
    image_yscale = image_xscale;
}
else
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

Приведенный выше код масштабирует объект sprite, а затем, когда он увеличивается в 5 раз, создается новый экземпляр другого объекта и уничтожается.
