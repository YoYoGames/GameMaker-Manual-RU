bbox_top

Эта переменная, доступная только для чтения, возвращает положение в пределах комнаты (вдоль оси y) вершины ограничительной рамки для данного экземпляра, где ограничительная рамка определяется максимальной шириной и высотой маски для данного экземпляра (как установлено sprite_index или mask_index). Даже когда sprite имеет точную маску столкновений, граничная область существует и используется для определенных вещей, поэтому вы можете использовать эту переменную для ее поиска. Обратите внимание, что если экземпляру не назначен sprite, возвращаемое значение будет равно позиции Y экземпляра.

Синтаксис:

bbox_top;

Возвращается:

(integer)

Пример:

if bbox_top < 0
{
    y = sprite_yoffset;
}

Приведенный выше код проверяет, находится ли ограничивающая рамка экземпляра за пределами комнаты, и если да, то устанавливает экземпляр в новое положение.
