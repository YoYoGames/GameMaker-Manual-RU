скорость изображения

Эта переменная определяет скорость, с которой GameMaker будет перебирать суб-изображения для текущего экземпляра sprite. Значение скорости является множителем, по умолчанию используется значение 1, при установке значения 0.5 скорость анимации уменьшится вдвое - как установлено в редакторе спрайтов или редакторе изображений, а при установке значения 2 она увеличится вдвое. Если используемый sprite не имеет вложенных изображений, эта переменная не будет иметь никакого эффекта.

Синтаксис:

image_speed;

Возвращается:

(single precision floating point value)

Пример:

with (instance_create_layer(x, y, "Effects", obj_Explosion))
{
    image_speed = 0.5;
}

Приведенный выше код создает экземпляр объекта "obj_Explosion" и устанавливает его image_speed равным 0,5 (фактически уменьшая скорость бега вдвое).
