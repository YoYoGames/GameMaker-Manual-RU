Переменные экземпляра спрайта

Эти переменные включены по умолчанию в объекты GameMaker, и все они связаны с sprite, назначенным экземпляру в комнате. Эти переменные могут быть доступны в любое время в коде или действиях для экземпляра и использоваться для изменения того, что и как рисуется.

  • sprite_index
  • sprite_width
  • sprite_height
  • sprite_xoffset
  • sprite_yoffset
  • image_alpha
  • image_angle
  • image_blend
  • image_index
  • image_number
  • image_speed
  • image_xscale
  • image_yscale
  • mask_index
  • bbox_bottom
  • bbox_left
  • bbox_right
  • bbox_top
