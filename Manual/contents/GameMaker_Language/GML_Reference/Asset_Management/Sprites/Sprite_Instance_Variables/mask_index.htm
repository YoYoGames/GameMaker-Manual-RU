индекс_маски

Когда вы определяете объект в GameMaker, вы можете назначить ему маску, которая будет использоваться для столкновений, а не ту, которая соответствует определенному sprite. Эта переменная может быть использована для поиска sprite_index этой маски (или она вернет -1, если не было назначено никакого sprite ) или для установки маски для экземпляра на выбранный sprite. Установка индекса маски означает, что вы можете иметь, например, sprite для экземпляра с неправильной формой, но дать ему круговую маску столкновений, полученную из другого sprite.

Синтаксис:

mask_index;

Возвращается:

Sprite Asset

Пример:

mask_index = spr_Round;

Приведенный выше код устанавливает маску экземпляра на маску sprite "spr_Round".
