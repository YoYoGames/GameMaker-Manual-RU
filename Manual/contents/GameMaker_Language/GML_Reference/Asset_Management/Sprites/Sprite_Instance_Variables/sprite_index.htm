sprite_index

Эта переменная возвращает индекс текущего sprite для экземпляра или -1, если экземпляр не имеет связанного с ним sprite. Вы можете изменить ее, чтобы дать экземпляру другой sprite, задав ему имя sprite из дерева ресурсов или используя переменную, в которой проиндексирован загруженный извне sprite. Изменение sprite не изменяет индекс текущего видимого кадра, поэтому, если вы измените sprite на кадре номер 3, новый sprite будет нарисован с видимым кадром (при условии, что у него такое же количество кадров).

Синтаксис:

sprite_index;

Возвращается:

Sprite Asset

Пример:

with (obj_Check)
{
    if !collision_line(x, y, other.x, other.y, obj_wall, false, true)
    {
        sprite_index = spr_spotted;
    }
    else
    {
        sprite_index = spr_clear;
    }
}

Приведенный выше код будет циклически просматривать все экземпляры "obj_Check", проверяя наличие линии столкновения между ними и экземпляром, выполняющим этот код. В зависимости от возвращаемого значения (true или false) для линии столкновения будет изменен адрес sprite этих экземпляров.
