номер_изображения

Эта переменная, доступная только для чтения, может быть использована для получения количества подобразов в sprite, который был назначен экземпляру (если вам нужно количество подобразов для sprite, отличного от того, который назначен экземпляру, вы должны использовать sprite_get_number()). Обратите внимание, что при наличии (например) 1 субизображения эта переменная вернет "1", но image_index этого субизображения будет равен 0.

Синтаксис:

image_number;

Возвращается:

(single precision floating point value)

Пример:

if image_speed > 0
{
    if image_index > image_number - 1 instance_destroy();
}

Приведенный выше код проверяет, анимируется ли sprite, и если да, то проверяет, превышает ли текущий image_index количество вложенных изображений, и если да, то уничтожает экземпляр.
