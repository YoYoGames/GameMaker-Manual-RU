высота спрайта

Эта переменная, доступная только для чтения, возвращает высоту sprite, назначенную данному экземпляру. Эта высота возвращается в пикселях и зависит от значения параметра image_yscale. Если вам нужна немасштабированная высота, используйте sprite_get_height().

Синтаксис:

sprite_height;

Возвращается:

Real

Пример:

if sprite_height != sprite_get_height(sprite_index)
{
    image_yscale = 1;
}

Приведенный выше код сверяет высоту sprite, назначенную экземпляру, выполняющему код, с высотой ресурса sprite, и если они не совпадают, то сбрасывает значение image_yscale на 1,
