bbox_left

Эта переменная, доступная только для чтения, возвращает положение (вдоль оси x) в пределах комнаты левой граничной области для данного экземпляра, где граничная область определяется максимальной шириной и высотой маски для данного экземпляра (как установлено sprite_index или mask_index). Даже когда sprite имеет точную маску столкновений, граничная область существует и используется для определенных вещей, поэтому вы можете использовать эту переменную для ее поиска. Обратите внимание, что если экземпляру не назначен sprite, возвращаемое значение будет равно X-позиции экземпляра.

Синтаксис:

bbox_left;

Возвращается:

(integer)

Пример:

if bbox_left < 0
{
    x = sprite_xoffset;
}

Приведенный выше код проверяет, находится ли ограничивающая рамка экземпляра за пределами комнаты, и если да, то устанавливает экземпляр в новое положение.
