ширина спрайта

Эта переменная, доступная только для чтения, возвращает ширину sprite, назначенную данному экземпляру. Эта ширина возвращается в пикселях и зависит от значения параметра image_xscale. Если вам нужна немасштабированная ширина, следует использовать sprite_get_width().

Синтаксис:

sprite_width;

Возвращается:

Real

Пример:

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

Приведенный выше код сверяет ширину sprite, назначенную экземпляру, выполняющему код, с шириной ресурса sprite, и если они не совпадают, то сбрасывает значение image_xscale на 1,
