image_angle

Это значение задает угол (поворот) sprite и измеряется в градусах: вправо - 0º, вверх - 90º, влево - 180º и вниз - 270º. Установите эту переменную в 0, чтобы вернуть sprite к тому виду, который был задан в редакторе sprite. Обратите внимание, что для того, чтобы изменения в этой переменной были заметны, экземпляр должен либо не иметь события рисования (и тогда GameMaker по умолчанию нарисует sprite), либо быть нарисованным с помощью одной из расширенных функций рисования, таких как draw_self() или draw_sprite_ext().

Синтаксис:

image_angle

Возвращается:

Real

Пример:

image_angle = point_direction(x, y, mouse_x, mouse_y);

Приведенный выше код повернет sprite экземпляра так, чтобы он всегда указывал на позицию мыши.
