sprite_xoffset

Эта переменная, доступная только для чтения, возвращает локальное смещение x (x-компонент начала координат, определенный в редакторе sprite ) sprite, назначенного данному экземпляру. Это xoffset возвращается в пикселях и будет зависеть от размера объекта. image_xscaleтак что (например) если квадрат 32x32 sprite определен как имеющий начало координат x на 16, то при масштабировании sprite на 2 эта переменная вернет 32. Если вам нужно немасштабированное смещение x, используйте sprite_get_xoffset().

Синтаксис:

sprite_xoffset;

Возвращается:

Real

Пример:

if x - sprite_xoffset < 0
{
    x = sprite_xoffset;
}

Приведенный выше код проверяет, находится ли левый край sprite экземпляра за пределами комнаты, и если да, то перемещает экземпляр так, чтобы был виден весь sprite.
