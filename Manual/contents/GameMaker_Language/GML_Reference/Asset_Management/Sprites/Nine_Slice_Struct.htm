Структура девяти ломтиков

Структуру Nine Slice можно создать с помощью функции sprite_nineslice_create(), или получить из sprite с помощью sprite_get_nineslice(). Эта структура будет содержать свойства, которые могут быть изменены и применены к sprite с помощью sprite_set_nineslice().

При извлечении из sprite, изменение значений в этой структуре повлияет на исходный sprite.

В этой структуре будут доступны следующие свойства:

Структура из девяти ломтиков
Переменная Тип Описание
enabled Булево Это булево значение будет истинным , если Nine Slicing включен для sprite, и ложным , если он отключен. Вы можете изменить это значение, чтобы включить или отключить Nine Slicing для sprite.
left Реально Это смещение левой направляющей по оси x относительно левого края sprite.
right Реально Это смещение правой направляющей по оси x относительно правого края sprite.
top Реально Это смещение верхней направляющей по оси y относительно верхнего края sprite.
bottom Реально Это смещение нижней направляющей по оси y относительно нижнего края sprite.
tilemode Массив из Режим плитки Постояннаяs Возвращает массив, содержащий режимы плитки для краевого и центрального срезов.

Режимы плитки

Режим Tile Mode каждого фрагмента может быть прочитан или изменен с использованием этих констант в качестве индекса массива:

Девять ломтиков кромки Постоянная
Постоянная Описание
nineslice_left Левый краевой срез
nineslice_top Верхний краевой срез
nineslice_right Правый краевой срез
nineslice_bottom Нижний краевой срез
nineslice_centre Центральный ломтик

Режим плитки для среза будет (или может быть установлен) одной из следующих констант:

Режим плитки Постоянная
Постоянная Описание
nineslice_stretch Ломтик будет растянут
nineslice_repeat Нарезка будет повторяться
nineslice_mirror Срез будет повторяться путем зеркального отображения
nineslice_blank Ломтик не будет растягиваться или повторяться, в результате чего после него останется пустая область
nineslice_hide Ломтик не появится вообще

Поскольку tilemode является массивом, для изменения режима плитки среза необходимо использовать синтаксис массива. Приведенный ниже пример кода устанавливает режим черепицы верхнего фрагмента на Repeat:

Пример:

var _nineslice = sprite_get_nineslice(spr_box);

_nineslice.tilemode[nineslice_top] = nineslice_repeat;
