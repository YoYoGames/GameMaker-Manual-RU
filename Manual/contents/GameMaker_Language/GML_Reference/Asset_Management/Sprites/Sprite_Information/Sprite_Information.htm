Информация о спрайте

Следующие функции можно использовать для получения информации об активе sprite. Обратите внимание, что эти функции получают информацию из базового sprite (т.е. актива sprite, созданного в браузере активов), а не из sprite, как он используется в экземпляре. Например, вы можете использовать спрайт переменная экземпляра sprite_width для получения ширины sprite, которую использует экземпляр, и эта ширина будет учитывать любое масштабирование, которое могло быть применено. Однако если вам нужно узнать ширину sprite в обычном виде, без применения преобразований, вам нужно использовать функцию sprite_get_width(), приведенную ниже.

Более подробную информацию о переменных экземпляра sprite см. в разделе " Переменные экземпляра".

  • sprite_get_number
  • sprite_get_speed
  • sprite_get_speed_type
  • sprite_get_width
  • sprite_get_height
  • sprite_get_xoffset
  • sprite_get_yoffset
  • sprite_get_bbox_bottom
  • sprite_get_bbox_left
