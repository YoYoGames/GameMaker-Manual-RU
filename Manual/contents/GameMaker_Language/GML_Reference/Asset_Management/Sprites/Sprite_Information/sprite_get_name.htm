sprite_get_name

Эта функция возвращает имя в виде строки указанного sprite. Это имя - то, которое было указано для sprite в дереве ресурсов главного окна GameMaker. Обратите внимание, что это только строка, и ее нельзя использовать для прямой ссылки на sprite - для этого вам понадобится индекс спрайта. Однако вы можете использовать эту строку для получения индекса спрайта, используя возвращаемую строку вместе с функцией asset_get_index().

Синтаксис:

sprite_get_name(index);

Аргумент Тип Описание
index Активы The index of the sprite to get the name of.

Возвращает

String

Пример:

str = sprite_get_name(sprite_index);

Приведенный выше код получит имя индекса sprite для экземпляра, выполняющего код, и сохранит возвращаемое значение в переменной "str".
