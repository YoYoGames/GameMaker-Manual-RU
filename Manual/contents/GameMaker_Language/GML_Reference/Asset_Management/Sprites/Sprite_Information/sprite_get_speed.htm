sprite_get_speed

Эта функция может быть использована для получения скорости sprite, определенной для ресурса sprite в редакторе спрайтов. Возвращаемое значение может быть использовано, например, для расчета того, сколько сколько кадров может быть нарисовано для различных image_speed. Обратите внимание, что возвращаемое значение будет сильно отличаться в зависимости от типа скорости которая была применена в редакторе Sprite, либо Frames Per Second, либо Frames Per Game Frame. Следующие два примера иллюстрируют это:

  • Если у вас есть sprite, который рисует 1 кадр в секунду, а скорость изображения установлена на 0,5, он будет рисовать со скоростью 0,5 кадра в секунду. Если частота кадров вашей игры составляет 60 кадров в секунду, то sprite будет рисовать 1 кадр на каждые 120 игровых кадров.
  • Если у вас есть sprite, который рисует 1 кадр на игровой кадр, и вы установили скорость изображения на 0,5, он будет рисовать 0,5 кадра на игровой кадр. Если частота кадров в игре составляет 60 кадров в секунду, то sprite будет рисовать 30 кадров на каждые 60 игровых кадров.

Узнать, какой тип скорости анимации использовался при определении sprite, можно с помощью функции sprite_get_speed_type(), и можно установить скорость и тип анимации с помощью функции sprite_set_speed().

Синтаксис:

sprite_get_speed(index)

АргументТип Описание
index The index of the sprite to get the speed of

Возвращается:

Пример:

s_speed = sprite_get_speed(sprite_index);
s_type = sprite_get_speed_type(sprite_index);

Приведенный выше код получает скорость sprite и тип анимации sprite и сохраняет их в переменных экземпляра для дальнейшего использования.
