sprite_get_xoffset

Когда вы определяете sprite в редакторе sprite, вам необходимо задать начало координат для этого sprite. Это точка, в которой sprite будет "прикреплен" или "нарисован" при использовании с экземпляром. Эта функция возвращает относительное смещение по оси x относительно начала координат, исходя из того, что левый верхний угол является точкой 0,0, при этом +x означает право, а -x - лево от нее. Следующее изображение иллюстрирует это:

Синтаксис:

sprite_get_xoffset(index);

АргументТип Описание
index The index of the sprite to find the xoffset of.

Возвращает

Пример:

if x - sprite_get_xoffset(sprite_index) < 0
{
    x = _xoffset;
}

Приведенный выше код обеспечит сохранение экземпляра в комнате на основе sprite (поэтому sprite всегда виден).
