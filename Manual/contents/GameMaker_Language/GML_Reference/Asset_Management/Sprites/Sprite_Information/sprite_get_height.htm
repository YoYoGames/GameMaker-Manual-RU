sprite_get_height

С помощью этой функции можно найти высоту базового актива sprite без преобразований в пикселях.

Синтаксис:

sprite_get_height(index);

АргументТип Описание
index The index of the sprite to find the height of.

Возвращает

Пример:

if sprite_height != sprite_get_height(sprite_index)
{
    image_yscale = 1;
}

Приведенный выше код проверяет высоту sprite в текущем экземпляре и, если есть разница между ней и исходной базой sprite, сбрасывает масштаб оси y.
