sprite_get_bbox_mode

Эта функция возвращает текущий "режим" для расчетов ограничивающей рамки. Вы передаете индекс sprite для проверки sprite, и функция возвращает одну из констант, показанных ниже.

Синтаксис:

sprite_get_bbox_mode(ind);

АргументТип Описание
ind The index of the sprite to check.

Возвращает

Bounding Box Mode Constant

Постоянная Описание
bboxmode_automatic Автоматически - Ограничительная рамка будет рассчитана автоматически, на основе настройки допуска для спрайта.
bboxmode_fullimage Полное изображение - Ограничительное поле будет установлено так, чтобы использовать всю ширину и высоту sprite, независимо от допусков и "пустых" пикселей.
bboxmode_manual Вручную - Ограничительная рамка была установлена вручную на заданные пользователем значения (либо в редакторе sprite, либо с помощью функции sprite_set_bbox())

Пример:

if sprite_get_bbox_mode(sprite_index) != 0
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

Приведенное выше действие проверяет режим bbox для текущего sprite и если он не automatic, то устанавливает его в это значение.
