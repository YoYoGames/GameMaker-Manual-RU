sprite_get_number

У sprite должно быть хотя бы одно подизображение, и эта функция вернет, сколько их у него. Помните, что субизображение sprite начинает нумероваться с 0, поэтому если эта функция вернет значение 3 (проверяемый sprite имеет 3 субизображения), то они будут пронумерованы как 0, 1 и 2.

Синтаксис:

sprite_get_number(index);

АргументТип Описание
index The index of the sprite.

Возвращает

Пример:

image_index = floor(random(sprite_get_number(sprite_index)));

Приведенный выше код установит индекс изображения на случайный кадр, основанный на количестве вложенных изображений, которые имеет сайт sprite.
