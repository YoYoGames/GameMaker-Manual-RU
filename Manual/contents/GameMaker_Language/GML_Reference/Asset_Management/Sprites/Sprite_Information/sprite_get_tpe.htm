sprite_get_tpe

Эта функция возвращает специальное значение, относящееся к позиции заданного sprite и суб-изображения на странице текстуры, используемой вашей игрой. Таким образом, вы можете передавать значение расширениям для использования вне скомпилированной игры, эффективно обходя функции рисования GameMaker и позволяя использовать sprite в содержимом DOM, которое затем может быть нарисовано в любом месте окна, содержащего игровую канву. Эта функция представляет особый интерес для тех, кто хочет создать кнопки и другие интерактивные средства вне элемента холста GameMaker на странице, использующей функцию clickable_add().

ПРИМЕЧАНИЕ: Эта функция предназначена только для HTML5.

Синтаксис:

sprite_get_tpe(sprite, index);

Аргумент Тип Описание
sprite Актив спрайта The index of the sprite to find the texture page entry of.
index Реально The image index of the sprite.

Возвращает

Texture Page Entry

Пример:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

Приведенный выше код создает DOM-кнопку в позиции (32, 32) страницы, на которой запущен игровой холст. Кнопка использует sprite, на которую ссылается страница текстуры как "spr_MS_Home", и при нажатии на кнопку откроется новое окно для указанного URL и с определенными свойствами для этого окна.
