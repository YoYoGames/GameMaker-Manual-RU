sprite_get_nineslice

Эта функция возвращает структуру Nine Slice для sprite, содержащую все свойства Nine Slice, установленные в редакторе спрайтов или с помощью функции sprite_set_nineslice(). Содержимое этой структуры подробно описано на этой странице.

Если предоставленный sprite не имеет назначенной ему структуры Nine Slice, для sprite будет создана и возвращена новая структура со свойствами Nine Slice по умолчанию.

Изменение любых значений в этой структуре изменит свойства Nine Slice исходного sprite, влияя на все будущие вызовы рисования, сделанные с этим sprite.

Синтаксис:

sprite_get_nineslice(ind);

АргументТип Описание
ind The index of the sprite from which the Nine Slice struct will be retrieved

Возвращается:

Nine Slice Struct (or -1 if the sprite doesn't exist)

Пример:

var _box_nineslice = sprite_get_nineslice(spr_box_0);

_box_nineslice.enabled = true;
_box_nineslice.left = 10;
_box_nineslice.right = 10;
_box_nineslice.top = 10;
_box_nineslice.bottom = 10;

Приведенный выше код извлекает структуру Nine Slice из sprite, включает для нее Nine Slicing и устанавливает смещения направляющих.
