sprite_get_yoffset

Когда вы определяете sprite в редакторе sprite, вам необходимо задать начало координат для этого sprite. Это точка, в которой sprite будет "прикреплен" или "нарисован" при использовании с экземпляром. Эта функция возвращает относительное смещение оси y относительно начала координат, исходя из того, что левый верхний угол является точкой 0,0, при этом +y означает "вниз", а -y - "вверх". Следующее изображение иллюстрирует это:

Синтаксис:

sprite_get_yoffset(index);

АргументТип Описание
index The index of the sprite to find the yoffset of.

Возвращает

Пример:

if y - sprite_get_yoffset(sprite_index) < 0
{
    y = sprite_get_yoffset(sprite_index);
}

Приведенный выше код обеспечит сохранение экземпляра в комнате на основе sprite (поэтому sprite всегда виден).
