sprite_get_speed_type

Эта функция может быть использована для получения типа анимации sprite, определенного для ресурса sprite в редакторе спрайтов. Возвращаемое значение будет одной из следующих констант:

Постоянная скорости спрайта
Постоянная Описание
spritespeed_framespersecond На сайте sprite была определена анимация в кадрах в секунду.
spritespeed_framespergameframe На сайте sprite была определена анимация в кадрах на один игровой кадр.

Следующие два примера иллюстрируют разницу, которую это может внести при расчете скорости анимации для спрайта:

  • Если у вас есть sprite, который рисует 1 кадр в секунду, а скорость изображения установлена на 0,5, он будет рисовать со скоростью 0,5 кадра в секунду. Если частота кадров в игре составляет 60 кадров в секунду, то sprite будет рисовать 1 кадр на каждые 120 игровых кадров.
  • Если у вас есть sprite, который рисует 1 кадр в игровом кадре, а скорость изображения установлена на 0,5, то он будет рисовать 0,5 кадра в игровом кадре. Если частота кадров в игре составляет 60 кадров в секунду, то sprite будет рисовать 30 кадров на каждые 60 игровых кадров.

Вы можете узнать, какое значение скорости использовалось для анимации sprite, с помощью функции sprite_get_speed(), а установить скорость и тип анимации можно с помощью функции sprite_set_speed().

Синтаксис:

sprite_get_speed_type(index)

Аргумент Тип Описание
index Актив спрайта The index of the sprite to get the animation type of

Возвращается:

Sprite Speed Constant (see above)

Пример:

s_speed = sprite_get_speed(sprite_index);
s_type = sprite_get_speed_type(sprite_index);

Приведенный выше код получает скорость sprite и тип анимации sprite и сохраняет их в переменных экземпляра для дальнейшего использования.
