sprite_get_uvs

Эта функция возвращает массив с UV-координатами и другими данными для текстуры суб-изображения sprite на странице текстуры. Функция возвращает массив со следующими 8 элементами:

  • [0] = левый
  • [1] = верх
  • [2] = право
  • [3] = низ
  • [4] = количество пикселей, которые компилятор активов обрезал с левой стороны sprites.
  • [5] = количество пикселей, которые компилятор активов обрезал с верхней стороны sprites.
  • [6] = нормализованный процент пиксельных данных от исходной ширины sprites, которые были сохранены на странице текстуры.
  • [7] = нормализованный процент пиксельных данных от исходной высоты sprites, который был сохранен на странице текстуры.

Затем этот массив можно использовать в других функциях рисования, особенно при работе в 3D или при использовании функций 2D-примитивов, а также при работе с функциями шейдеров.

ПРИМЕЧАНИЕ: Эта функция не работает с вектором sprites или скелетной анимацией sprites.

Синтаксис:

sprite_get_uvs(sprite, subimage);

АргументТип Описание
sprite The index of the sprite to use.
subimage The sub-image of the sprite to use.

Возвращает

(1D, 8 elements)

Пример:

var tex = sprite_get_uvs(sprite, 0);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

Приведенный выше код сохранит УФ-координаты для заданного sprite в локальном массиве, а затем присвоит значения переменным экземпляра.
