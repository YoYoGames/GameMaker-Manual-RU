script_execute_ext

Эта функция работает аналогично функции script_execute(), только вы можете предоставить массив, содержащий аргументы, необходимые для вызываемой функции/скрипта. Вы также можете передать функции два необязательных аргумента, чтобы указать смещение в массиве для получения аргументов, а также количество аргументов из массива (оно должно быть максимум равно array_length - offset).

Синтаксис:

script_execute_ext(scr, array_args, [offset], [num_args]);

Аргумент Тип Описание
scr Ассет сценария или функция сценария The function/script that you want to call
array_args Массив The array containing the arguments for the function/script
[offset] Реально OPTIONAL Смещение в массиве аргументов
[num_args] Реально OPTIONAL Количество аргументов для использования (начиная со смещения и далее)

Возвращается:

Variable (Will depend on the return value from the function/script being called)

Пример:

script_execute_ext(move_inst, move_array, floor(random(4)), 1);

Приведенный выше код примера будет использовать script_execute_ext для вызова заданной функции, предоставляя массив из четырех аргументов, но используя только один из них случайным образом.

ПРИМЕЧАНИЕ Эту функцию нельзя использовать со встроенными функциями.
