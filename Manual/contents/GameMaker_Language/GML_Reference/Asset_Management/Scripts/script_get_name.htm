имя_скрипта

Эта функция возвращает имя в виде строки указанного скрипта. Это имя - то, которое было указано для скрипта в Asset Browser главного окна GameMaker. Для получения дополнительной информации о скриптах см. раздел Редактор скриптов.

Синтаксис:

script_get_name(scr);

Аргумент Тип Описание
scr Актив сценария The index of the script that you want to get the name of.

Возвращается:

String

Пример:

scr_name = script_get_name(Help_File);

Приведенный выше код примера будет хранить имя указанного индекса скрипта в переменной "scr_name".
