последовательность_get

С помощью этой функции вы можете получить структуру объекта последовательности из индекса актива последовательности. Вы указываете индекс актива для последовательности, как определено в браузере активов, и функция возвращает структуру объекта последовательности, к которой можно получить доступ, или -1, если последовательность не существует или указанный индекс актива не является последовательностью.

Синтаксис:

sequence_get(sequence_index);

Аргумент Тип Описание
sequence_index Активы последовательности The sequence asset as defined in the Asset Browser.

Возвращается:

Sequence Object Struct or -1

Пример:

var _seq = sequence_get(seq_Logo);
seq.loopmode = seqplay_pingpong;

Приведенный выше код извлекает структуру объекта последовательности из объекта последовательности "seq_Loop", а затем устанавливает режим цикла для последовательности (все последующие созданные экземпляры последовательности теперь будут использовать этот режим цикла).
