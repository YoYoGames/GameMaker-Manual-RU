sequence_track_new

С помощью этой функции вы можете создать новую структуру дорожки последовательности, указав тип дорожки, которую вы хотите создать. Аргумент type может быть любым из констант типа дорожки последовательности, перечисленных на этой странице.

Функция возвращает структуру трека последовательности, в которую могут быть добавлены данные, прежде чем она будет присвоена объекту последовательности.

Синтаксис:

sequence_track_new(type);

Аргумент Тип Описание
type Тип дорожки последовательности Постоянная The type of track to create.

Возвращается:

Sequence Track Struct

Пример:

myseq = sequence_create();
var mytracks = array_create(1);
mytracks[0] = sequence_track_new(seqtracktype_graphic);
var graphickeys = array_create(1);
graphickeys[0] = sequence_keyframe_new(seqtracktype_graphic);
graphickeys[0].frame = 0;
graphickeys[0].length = 1;
graphickeys[0].stretch = true;
graphickeys[0].disabled = false;
var graphickeydata = array_create(1);
graphickeydata[0] = sequence_keyframedata_new(seqtracktype_graphic);
graphickeydata[0].spriteIndex = spr_Platform;
graphickeydata[0].channel = 0;
graphickeys[0].channels = graphickeydata;
mytracks[0].name = "TestGraphicTrack";
mytracks[0].keyframes = graphickeys;
myseq.tracks = mytracks;

Приведенный выше код создает новую последовательность, затем создает дорожку графического актива и устанавливает на ней данные ключевых кадров. Затем эта дорожка назначается экземпляру.
