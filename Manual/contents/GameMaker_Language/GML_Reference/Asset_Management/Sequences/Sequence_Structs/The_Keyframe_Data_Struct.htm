Структура данных ключевого кадра

Когда вы определяете структуру ключевых кадров для дорожки, вам также необходимо определить данные, которые будут с ними связаны. Эти данные состоят из различных каналов, где каждый канал - это просто Keyframe Data Struct. Каналу может быть присвоен определенный тип данных в зависимости от того, для какого типа дорожки вы задаете данные ключевого кадра.

Структура данных ключевого кадра содержит следующие переменные:

Структура данных ключевого кадра последовательности
Переменная Тип Описание
channel Реально Это канал, к которому должны быть применены данные ключевого кадра. Это целое положительное значение, начинающееся с 0. Стоит отметить, что при создании дорожек параметров для ключевых кадров "положение" или "масштаб" необходимо использовать очень специфические значения каналов. Они следующие: канал 0 - это положение X или масштаб X, канал 1 - положение Y или масштаб Y.
spriteIndex Актив спрайта Индекс актива sprite, который следует использовать для дорожки. Это свойство доступно только для дорожек типа seqtracktype_graphic, и вы можете получить или установить его.
soundIndex Звуковой актив Индекс аудиоактива, используемого для дорожки. Это свойство доступно только для дорожек типа seqtracktype_audio, и вы можете получить или установить его.
playbackMode Последовательность Аудио Ключ Константа Режим воспроизведения звука. Это свойство доступно только для дорожек типа seqtracktype_audio, и вы можете получить или установить его. Свойство должно быть одной из констант, приведенных в таблице ниже этого свойства.
curve Структура анимационной кривой Это свойство требует наличия структуры кривой анимации (см. здесь для получения дополнительной информации) и доступно только для дорожек типа seqtracktype_real. Если структура кривой не используется, то значение этого свойства будет равно -1.
value Реально Это свойство представляет собой просто значение, связанное с каналом данных ключевых кадров, и доступно только для дорожек типа seqtracktype_real, если не предоставлена структура curve. Это может быть, например, позиция X или Y дорожки, если она размещена внутри дорожки с параметром "position".
colour Массив из Реальныйs Это свойство возвращает (или требует, если установлено) массив для значения цвета ключевого кадра с форматом [A, R, G, B]. Это свойство доступно только для дорожек типа seqtracktype_colour. Обратите внимание, что значения для каждого компонента должны быть выражены между 0 и 1, где 0 соответствует HEX-значению #00, а 1 - HEX-значению #FF (0 - 255, как показано в окне выбора цвета для цветных дорожек в редакторе последовательностей).
sequence Структура объекта последовательности Это свойство возвращает (или требует, если установлено) объект последовательности struct и доступно только для дорожек типа seqtracktype_sequence.
objectIndex Объект Активы Это свойство возвращает (или требует, если установлено) индекс объекта и доступно только для дорожек типа seqtracktype_instance.
events Массив из Строкаs Это свойство позволяет получить доступ к событиям и широковещательным сообщениям, связанным со структурой данных ключевого кадра. Вы можете получить или установить это свойство, при получении возвращается массив строк, а при установке должен быть указан массив строк. Более подробную информацию о событиях можно найти в разделе События и моменты последовательности. Это свойство доступно только для дорожек типа seqtracktype_message.
event Метод Это свойство возвращает (или может быть установлено на) method связанный со структурой данных ключевого кадра. Если метод не был указан или вы хотите удалить метод, то свойство должно быть равно -1. Это свойство доступно только для дорожек типа seqtracktype_moment.
Любые дополнительные переменные, перечисленные в таблице (таблицах) ниже

Переменная playbackMode может быть одной из следующих констант:

Последовательность Аудио Ключ Константа
Постоянная Описание Значение
seqaudiokey_loop При воспроизведении звук будет зацикливаться. 0
seqaudiokey_oneshot Звук будет воспроизводиться только один раз, а затем прекратится. 1

Данные текстовой дорожки

Структура будет содержать следующие дополнительные переменные, если она назначена на текстовую дорожку (seqtracktype_text):

Структура данных ключевого кадра последовательности
Переменная Тип Описание
text Строка Это текстовая строка, которая рисуется на дорожке.
wrap Булево Это булево значение, которое указывает, должен ли текст быть обернут (true) или нет (false).
alignmentH Выравнивание текста последовательности Константа Это горизонтальное выравнивание текста, и оно будет одним из Константа выравнивания текста последовательностиприведенных ниже.
alignmentV Выравнивание текста последовательности Константа Это вертикальное выравнивание текста, и оно будет одним из Константа выравнивания текста последовательностиприведенных ниже.
fontIndex Шрифтовой актив Это Шрифтовой актив используемый текстовой дорожкой.

alignmentH может быть любой из следующих констант:

Константа выравнивания текста последовательности (alignmentH)
Постоянная Описание
seqtextkey_left Текст будет выровнен по левому краю рамки
seqtextkey_right Текст будет выровнен по правому краю рамки
seqtextkey_center Текст будет центрирован по горизонтали в рамке
seqtextkey_justify Текст будет выровнен в рамке

alignmentV может быть любой из следующих констант:

Константа выравнивания текста последовательности (alignmentV)
Постоянная Описание
seqtextkey_top Текст будет выровнен по верхней части рамки
seqtextkey_bottom Текст будет выровнен по нижней части рамки
seqtextkey_middle Текст будет центрирован по вертикали в рамке
