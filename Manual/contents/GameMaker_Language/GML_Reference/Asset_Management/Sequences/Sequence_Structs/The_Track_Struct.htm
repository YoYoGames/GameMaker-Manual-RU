Структура трека

Track struct может использоваться либо длятрековактивовверхнего уровня , либо для трековпараметров нижнего уровня , а поведение трека определяется двумя вещами, его именем и типом. Структуры дорожек создаются с помощью функции sequence_track_new() и могут быть получены из активов последовательности с помощью свойства activeTracks из Sequence Instance Struct или свойства tracks из Sequence Object Struct.

В структуре трека доступны следующие свойства:

Структура дорожки последовательности
Переменная Тип Описание
type Тип дорожки последовательности Постоянная Содержит Константа типа дорожки последовательности которая описывает тип дорожки. Это может быть любая из констант, приведенных ниже в разделе "Тип".
name Строка При создании дорожки активов "верхнего уровня" имя, которое вы здесь даете, может быть любой строкой, необходимой для идентификации дорожки. Однако для дорожек параметров необходимо указать специальные строки, чтобы сообщить GameMaker, какую дорожку параметров вы создаете. Подробнее об этом говорится в разделе "Имя" ниже.
tracks Массив из Структура дорожки последовательностиs Это свойство позволяет получить доступ к списку дорожек, которые являются дочерними для данной дорожки. При получении этого свойства возвращается массив Структура трека последовательностиs возвращается, а при установке этого свойства массив Структура дорожки последовательностиs должен быть указан.
visible Булево Указывает, является ли эта дорожка видимой (значение true) или нет (значение false). Вы можете получить или установить это значение, и если дорожка не видна, то ни одна из ее дочерних дорожек не будет отрисована.
keyframes Массив из Структура ключевого кадра последовательностиs Это свойство позволяет получить доступ к списку структур ключевых кадров для трека. При получении этого свойства возвращается массив структур ключевых кадров, а при установке этого свойства должен быть указан массив структур ключевых кадров.
enabled Boolean This indicates whether this track is enabled. If the track is an asset track it isn't drawn when disabled, if it's a parameter track the track's values aren't applied.

Имя

Свойство name может быть любой из следующих строк:

  • "position" - при установке на дорожку типа seqtracktype_real указывает, что содержимое этой дорожки должно использоваться как позиционные данные.
  • "scale" - при установке на дорожку типа seqtracktype_real указывает, что содержимое этой дорожки должно использоваться в качестве данных масштабирования.
  • "rotation" - при установке на дорожке типа seqtracktype_real указывает, что содержимое этой дорожки должно использоваться в качестве данных вращения.
  • "blend_multiply" или "image_blend" - при установке на дорожку типа seqtracktype_colour указывает, что содержимое этой дорожки должно использоваться как данные умножения цвета. Данные ключевых кадров для такой дорожки должны представлять собой массив из четырех значений в формате [A, R, G, B]. Обратите внимание, что значения для каждого компонента выражаются между 0 и 1, где 0 соответствует HEX-значению #00, а 1 - HEX-значению #FF (0 - 255, как показано в окне выбора цвета для дорожек смешивания изображений в редакторе последовательностей).
  • "image_speed" - при установке на дорожку типа seqtracktype_real указывает, что содержимое этой дорожки должно использоваться в качестве значения скорости изображения.
  • "image_index" - при установке на дорожку, имеющую тип seqtracktype_real, указывает, что содержимое этой дорожки должно использоваться в качестве значения индекса изображения.
  • "image_angle" - при установке на дорожку типа seqtracktype_real указывает, что содержимое этой дорожки должно использоваться в качестве значения угла изображения.


    Имена параметров текстовых дорожек (используются только для текстовых дорожек):
  • "frameSize" - при установке на дорожке seqtracktype_real указывает, что содержимое этой дорожки должно использоваться в качестве размера кадра для текста.Каждый ключевой кадр для этой дорожки должен иметь два канала, 0 и 1, где 0 хранит размер X, а 1 - размер Y кадра.
  • "characterSpacing" - также используется с дорожками seqtracktype_real, используется в качестве интервала между символами текста. Каждый ключевой кадр для этой дорожки должен иметь один канал, содержащий значение интервала.
  • "lineSpacing" - также используется с дорожками seqtracktype_real, используется в качестве межстрочного интервала для текста. Каждый ключевой кадр для этой дорожки должен иметь один канал, содержащий значение интервала.
  • "paragraphSpacing" - также используется с дорожками seqtracktype_real, используется в качестве интервала между абзацами текста. Каждый ключевой кадр для этой дорожки должен иметь один канал, содержащий значение интервала.
