Структура экземпляра последовательности

Последовательность, когда она помещается в комнату, размещается как элемент на слое, и этот элемент содержит экземпляр последовательности. Этот экземпляр последовательности будет иметь следующие свойства в своей структуре:

Структура экземпляра последовательности
Переменная Тип Описание
sequence Структура объекта последовательности Это идентификатор структуры объекта последовательности, на основе которой был создан экземпляр последовательности (подробнее см. здесь). Вы можете получить или установить эту структуру и таким образом изменить базовую последовательность, на которую ссылается экземпляр.
headPosition Реально Это текущее положение головки воспроизведения (в кадрах) для данного экземпляра последовательности. Вы можете получить или установить это значение, но учтите, что установка его в значение больше или меньше, чем текущая определенная область воспроизведения, будет иметь различные эффекты в зависимости от типа воспроизведения, установленного для последовательности. См. layer_sequence_headpos() для получения дополнительной информации.
headDirection Направление последовательности Постоянная Это текущее направление воспроизведения для экземпляра последовательности. Вы можете получить или установить это значение с помощью констант, приведенных в следующей таблице (Обратите внимание, что вы можете использовать функции слоя layer_sequence_get_headdir() и layer_sequence_headdir() для получения или установки этого значения без прямого доступа к struct).
speedScale Реально Это свойство можно использовать для получения или установки шкалы скорости воспроизведения. Шкала скорости представляет собой множитель, где 1 - это скорость воспроизведения по умолчанию, значения меньше 1 замедляют воспроизведение, а значения больше 1 ускоряют его, например: значение 0,5 - это половина скорости воспроизведения, а значение 2 - удвоенная скорость воспроизведения. Обратите внимание, что вы можете использовать функции слоя layer_sequence_get_speedscale() и layer_sequence_speedscale() для получения или установки этого значения без прямого доступа к структуре.
paused Булево Вы можете проверить это свойство, чтобы узнать, была ли последовательность приостановлена или нет, и она будет true, если была, или false в противном случае. Это свойство доступно только для чтения, но вы можете использовать функцию слоя layer_sequence_pause() чтобы приостановить воспроизведение, и layer_sequence_play() возобновить его при необходимости. Вы также можете проверить это свойство с помощью функции layer layer_sequence_is_paused() а не проверять свойство непосредственно в структуре.
finished Булево Вы можете проверить это свойство, чтобы узнать, закончила ли последовательность воспроизведение или нет, возвращая true, если она закончила воспроизведение, и false в противном случае. Это свойство доступно только для чтения и возвращает значение true только для тех дорожек, которые не установлены на цикл или пинг-понг. Вы также можете проверить это свойство с помощью функции layer layer_sequence_is_finished() а не проверять свойство непосредственно в структуре.
elementID Идентификатор элемента последовательности Это свойство содержит идентификатор элемента последовательности. Этот идентификатор - простое идентифицирующее значение, которое ассоциируется со слоем в редакторе помещений, и его можно использовать с соответствующими функциями слоя, чтобы найти слой, которому назначена последовательность, или изменить определенные свойства последовательности без необходимости работать со структурой.
activeTracks Массив из Последовательность Активная дорожка Структураs Это свойство будет содержать массив структур "evaluation", содержащих информацию о текущем состоянии каждого трека актива в последовательности (графика, последовательность, аудио и т.д...). Это дорожки верхнего уровня, и возвращаемый массив доступен только для чтения. О содержимом структуры дорожки актива, возвращаемой в каждом элементе массива, см. раздел ниже.

Направление последовательности Постоянная
Постоянная Описание Значение
seqdir_right Последовательность будет воспроизводить кадры в инкрементном порядке слева направо 1
seqdir_left Последовательность будет воспроизводить кадры в убывающем порядке справа налево -1

activeTracks Struct

Хотя это свойство уже существует в структуре экземпляра последовательности в событии Sequence Create, оно заполняется дорожками только после первого события Sequence Begin Step. Поэтому первый раз вы найдете значения в этой структуре во время первой итерации события Sequence Step.

Следующий список содержит свойства, которые могут быть доступны вам при обращении к активной структуре дорожки (как включено в массив activeTracks структуры экземпляра последовательности, описанного выше). Каждый актив трека struct может иметь любое из следующих свойств, в зависимости от типа актива, используемого треком (обратите внимание, что вы можете изменять все эти свойства, кроме тех, которые указаны как доступные только для чтения), и все возвращаемые значения относятся к текущей позиции головки воспроизведения:

Последовательность Активный трек Структура
Переменная Тип Описание
matrix Матричный массив Это матрица преобразования, используемая для рисования дорожки (sprite, дорожки экземпляра и дорожки последовательности).
posx Реально Положение актива в последовательности вдоль оси X для дорожки в текущем положении головки воспроизведения (все активы).
posy Реально Положение актива в последовательности вдоль оси Y для дорожки в текущем положении головки воспроизведения (все активы).
scalex Реально Масштаб актива в последовательности по оси X для дорожки в текущей позиции головки воспроизведения (sprite, дорожки экземпляра и последовательности).
scaley Реально Масштаб актива в последовательности по оси Y для дорожки в текущей позиции головки воспроизведения (sprite, дорожки экземпляра и последовательности).
xorigin Реально Начало координат X актива для трека (все активы).
yorigin Реально Y начало координат актива для трека (все активы).
width Реально Ширина (в пикселях) актива для дорожки (sprite, экземпляра и дорожки последовательности).
height Реально Высота (в пикселях) актива для дорожки (sprite, экземпляра и дорожки последовательности).
imageindex Реально Индекс изображения для актива на дорожке в последовательности в текущей позиции головки воспроизведения (sprite и дорожки экземпляра).
imagespeed Реально Скорость изображения для актива на дорожке в последовательности в текущей позиции головки воспроизведения (sprite и дорожки экземпляра).
colourMultiply Цвет Значение умножения цвета для актива на дорожке в последовательности в текущей позиции головки воспроизведения (sprite, экземпляра и дорожки последовательности). Это значение будет представлять собой массив из четырех значений ARGB с форматом [A, R, G, B]. Обратите внимание, что значения для каждого компонента выражаются между 0 и 1, где 0 соответствует HEX-значению #00, а 1 - HEX-значению #FF (0 - 255, как показано в окне выбора цвета для дорожек с цветовым умножением в редакторе последовательностей).
soundIndex Sound Instance ID The ID of the sound instance that's playing on this track's emitter (audio tracks).
emitterIndex Идентификатор аудиоизлучателя Индекс аудиоизлучателя, принадлежащего данной дорожке (аудиодорожке).
gain Реально Усиление (громкость) дорожки в текущем положении головки воспроизведения (аудиодорожки).
pitch Реально Высота тона дорожки в текущем положении головки воспроизведения (аудиодорожки).
frameSizeX Реально Горизонтальный размер текстовой рамки (текстовые дорожки).
frameSizeY Реально Вертикальный размер текстовой рамки (текстовые дорожки).
characterSpacing Реально Значение расстояния между символами (текстовые дорожки).
lineSpacing Реально Значение межстрочного интервала (текстовые дорожки).
paragraphSpacing Реально Значение интервала между абзацами (текстовые дорожки).
track Структура дорожки последовательности Структура дорожки, на которой основана эта дорожка (все активы, значение только для чтения ).
parent Структура экземпляра последовательности Идентификатор экземпляра родительской последовательности для дорожки (все активы, значение только для чтения ).
activeTracks Массив из Структура дорожки последовательностиs Это массив структур оценки для каждой дорожки параметров, которую содержит дорожка активов (все активы, значение только для чтения ). Содержимое каждой структуры в массиве перечислено в разделе " Структура дорожки параметров".
