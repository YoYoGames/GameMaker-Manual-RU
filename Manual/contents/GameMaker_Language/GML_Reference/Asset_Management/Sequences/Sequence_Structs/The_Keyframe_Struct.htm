Структура ключевого кадра

Каждая дорожка (как определено на странице, посвященной структурам дорожек) будет иметь одну или несколько структур ключевых кадров, назначенных ей (которые можно получить из свойства keyframes структуры дорожки), и каждая из них будет иметь следующие свойства:

Структура ключевого кадра последовательности
Переменная Тип Описание
frame Реально Позиция (в кадрах) на временной шкале для ключевого кадра. Значение по умолчанию равно 0.
length Реально Длина ключевого кадра. Значение по умолчанию равно 1, при установке больших значений свойство дорожки, на которое ссылается ключевой кадр, будет поддерживаться в начальном значении в течение заданной длины. Обратите внимание, что свойство stretch отменяет это значение, если оно установлено в true.
stretch Булево Если это свойство установлено в true, то ключевой кадр растягивается либо до следующего ключевого кадра дорожки, либо до конца дорожки, если это последний ключевой кадр. Вы можете получить или установить это значение, по умолчанию оно равно false.
channels Массив из Структура данных ключевого кадра последовательностиs Это свойство позволяет получить доступ к списку структур данных ключевых кадров для каналов дорожки. При получении этого свойства возвращается массив структур данных ключевых кадров, а при установке этого свойства должен быть указан массив структур данных ключевых кадров.
