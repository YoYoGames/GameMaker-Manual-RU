экземпляр_последовательности

Это встроенная переменная, которая является частью переменных экземпляра, созданных для каждого экземпляра объекта в вашей игре. Если объект управляется последовательностью, эта переменная будет содержать структуру экземпляра последовательности для последовательности, управляющей объектом, в противном случае это будет undefined. Эта переменная доступна только для чтения и не может быть изменена.

Обратите внимание, что эта переменная станет undefined после завершения управляющей Последовательности, даже если ее элемент Последовательности все еще существует, и снова получит структуру, если этот элемент Последовательности будет воспроизведен.

Синтаксис:

sequence_instance

Возвращается:

Sequence Instance Struct

Пример:

if (in_sequence)
{
    sequence_instance.speedScale = 2;
}

Приведенный выше код проверяет in_sequence и если он равен true (то есть экземпляр управляется последовательностью), то он изменит шкалу скорости этой последовательности на 2.
