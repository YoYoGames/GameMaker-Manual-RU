sequence_instance_override_object

С помощью этой функции вы можете переопределить (заменить) все экземпляры объекта, используемые в последовательности, на другой. Вы предоставляете ID структуры экземпляра последовательности (полученный при создании экземпляра последовательности в комнате или с помощью одной из функций слоя комнаты - см. здесь), а также индекс объекта (определенный в браузере активов) для объекта, который вы хотите переопределить. Наконец, вы указываете индекс объекта или ID экземпляра для использования в качестве объекта, который будет переопределять последовательность (при указании ID экземпляра в качестве переопределяемого будет использоваться объект, из которого был создан экземпляр). Обратите внимание, что это можно сделать только для экземпляров последовательности (не для объектов последовательности) и должно быть сделано до начала воспроизведения последовательности, иначе она не будет работать.

Синтаксис:

sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id);

Аргумент Тип Описание
sequence_instance_struct Структура экземпляра последовательности The sequence instance struct to modify.
object_id Объект Активы The object index of the object within the sequence to override.
instance_or_object_id Идентификаторактива или экземпляраобъекта The object index or instance ID to use to override the sequence objects.

Возвращается:

N/A

Пример:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _seq_inst = layer_sequence_get_instance(_seq);
sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer);

Приведенный выше код создает новый экземпляр последовательности на данном слое, а затем изменяет его таким образом, что все экземпляры объекта "obj_Trees_Winter" заменяются экземплярами объекта "obj_Trees_Summer".
