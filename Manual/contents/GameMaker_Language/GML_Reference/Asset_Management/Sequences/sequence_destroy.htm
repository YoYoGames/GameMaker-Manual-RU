sequence_destroy

С помощью этой функции вы можете уничтожить объект последовательности, который был создан динамически. Вы передаете либо struct объекта последовательности (возвращаемый функцией sequence_create()) или идентификатор последовательности (возвращаемый функцией layer_sequence_get_sequence() или из свойства структуры экземпляра последовательности sequence). Эту функцию следует использовать всякий раз, когда динамически созданная последовательность больше не нужна, чтобы освободить связанную с ней память.

Синтаксис:

sequence_destroy(sequence_struct_or_id);

Аргумент Тип Описание
sequence_struct_or_id Sequence Asset или Sequence Object Struct The sequence object struct or ID to destroy

Возвращается:

N/A

Пример:

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

Приведенный выше код проверяет, существует ли данный объект последовательности, и если да, то он уничтожается.
