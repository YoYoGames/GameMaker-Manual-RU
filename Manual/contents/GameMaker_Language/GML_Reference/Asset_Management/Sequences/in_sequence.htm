в_последовательности

Это встроенная переменная, которая является частью переменных экземпляра, созданных для каждого экземпляра объекта в вашей игре. Если объект управляется последовательностью, эта переменная будет возвращать true, в противном случае она будет возвращать false. Эта переменная доступна только для чтения и не может быть изменена.

Обратите внимание, что эта переменная станет false после завершения управляющей последовательности, даже если ее элемент последовательности все еще существует, и станет true, если этот элемент последовательности будет воспроизведен снова.

Вы можете использовать эту переменную в объектах, управляемых игроком (или CPU), чтобы убедиться, что они перемещаются кодом только тогда, когда они не находятся в Последовательности. Это может оказаться полезным для cutscenes, когда вам нужно использовать одни и те же экземпляры в последовательности (с помощью функции функцию sequence_instance_override_object() ) и вам нужно убедиться, что они не могут быть перемещены обычным кодом, пока активна эта сцена.

Синтаксис:

in_sequence

Возвращается:

Пример:

if (!in_sequence)
{
    x += move_x;
    y += move_y;
}

Приведенный выше код проверяет переменную in_sequence, и если она равна false (то есть экземпляр не управляется последовательностью), то он добавляет move_x и move_y к позиции экземпляра. чтобы убедиться, что он перемещается только тогда, когда он не находится в последовательности.
