экземпляр_самый дальний

Эта функция проверяет все экземпляры заданного объекта, чтобы определить, какой из них находится дальше всего от заданной точки x/y. Все проверки будут проводиться от заданной точки x/y до начала координат (точки x/y) экземпляров указанного объекта. Если экземпляров объекта не существует, функция вернет ключевое слово noone, но если экземпляры существуют, то функция вернет значение id найденного экземпляра. Обратите внимание, что если экземпляр, выполняющий код, имеет тот же индекс объекта, что и проверяемый объект, то он будет включен в проверку (это включает проверку родительских объектов, если вызывающий экземпляр также является дочерним по отношению к родительскому).

Синтаксис:

instance_furthest(x, y, obj);

АргументТип Описание
x The x position to check for instances far from.
y The y position to check for instances far from.
obj The object to check for instances of.

Возвращается:

Instance ID or noone

Пример:

var inst;
inst = instance_furthest(x, y, object_index);
if inst != id
{
    draw_line(x, y, inst.x, inst.y);
}

Приведенный выше код найдет самый удаленный экземпляр, который имеет тот же индекс объекта, что и экземпляр, выполняющий код, и сохранит его id в переменной. Затем эта переменная проверяется, совпадает ли она с id вызывающего экземпляра, и если нет, то между двумя экземплярами проводится линия.
