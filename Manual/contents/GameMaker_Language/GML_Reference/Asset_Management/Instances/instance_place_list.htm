список_мест_экземпляров

С помощью этой функции вы можете проверить позицию на столкновение со всеми экземплярами объекта, используя маску столкновения того экземпляра, который выполняет код проверки. При использовании этой функции вы фактически просите GameMaker переместить экземпляр в новую позицию, проверить наличие столкновения, затем вернуться назад и сообщить вам, было ли обнаружено столкновение или нет. Это будет работать для точных столкновений, но только если и у экземпляра, и у экземпляров проверяемого объекта выбраны маски точных столкновений, в противном случае применяются только столкновения с ограничивающей рамкой.

Вы также предоставляете список DS, поэтому id значения всех экземпляров, которые сталкиваются с вызывающим экземпляром, будут добавлены в конец данного списка. Вы можете очистить список перед вызовом этой функции, чтобы он содержал только результаты этого вызова функции. Также есть возможность упорядочить список по расстоянию от начала координат экземпляра, выполняющего проверку, до начала координат экземпляров, с которыми произошло столкновение.

Обратите внимание, что функция также принимает специальное ключевое слово allв этом случае будут перечислены все найденные экземпляры, с которыми произошла коллизия. Функция возвращает количество найденных экземпляров или 0, если не найдено ни одного.

Синтаксис:

instance_place_list(x, y, obj, list, ordered);

Аргумент Тип Описание
x Реально The x position to check for instances.
y Реально The y position to check for instances.
obj Идентификаторактива или экземпляраобъекта The object to check for instances of.
list Идентификатор списка DS The DS list to use to store the IDs of colliding instances.
ordered Булево Whether the list should be ordered by distance (true) или нет (false).

Возвращается:

Real (The number of instances found to be in collision)

Пример:

var _list = ds_list_create();
var _num = instance_place_list(x, y, obj_Enemy, _list, false);

if (_num > 0)
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}

ds_list_destroy(_list);

Приведенный выше код проверит на столкновение все экземпляры, найденные в позиции вызывающего экземпляра. Они будут добавлены в список DS, который затем будет пройден по циклу для уничтожения каждого из столкнувшихся экземпляров.
