Экземпляры

Экземпляры создаются из объектов, и после создания их можно изменять и манипулировать ими с помощью кода GML и визуальных действий GML. Основными доступными и наиболее часто используемыми функциями, связанными с экземплярами, являются:

  • instance_create_layer
  • instance_create_depth
  • instance_destroy
  • instance_exists
  • instance_change
  • instance_copy
  • instance_find
  • instance_furthest
  • instance_nearest
  • instance_number
  • instance_place
  • instance_place_list
  • instance_position
  • instance_position_list

Обратите внимание, что есть две дополнительные встроенные переменные (это переменные глобальной области видимости ) и одна функция, связанная с использованием экземпляров, которая может быть полезна для получения определенных деталей:

  • instance_id_get
  • instance_id
  • instance_count

Существует ряд встроенных переменных, доступных в каждом экземпляре объекта, которые можно использовать для управления различными аспектами отображения объекта, использования других активов, реакции на столкновения и т.д., которые перечислены здесь:

  • Переменные экземпляра

Экземпляры имеют переменную массива, которая может быть использована для получения или установки значений тревожного события, но есть также две функции, которые могут быть использованы для того же самого:

  • alarm_get
  • alarm_set

Наконец, у вас также есть возможность деактивировать экземпляры, что означает, что они не будут обрабатывать никаких своих событий, хотя они по-прежнему будут существовать в игре. Функции, связанные с этим, доступны на следующей странице:

  • Деактивация экземпляров
