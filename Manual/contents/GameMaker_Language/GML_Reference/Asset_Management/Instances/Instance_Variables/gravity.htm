гравитация

gravity является одной из "встроенных" переменных, которые есть у всех экземпляров и, когда они установлены, прикладывают постоянную силу в gravity_direction экземпляра, влияя как на экземпляр speed , так и на direction. Обратите внимание, что гравитация является кумулятивной силой и будет ускорять объект, если вы решите не ограничивать конечную скорость, и обычно вы устанавливаете эту переменную на небольшие десятичные значения, например 0.01. Если вы установите gravity в 0, то гравитация не будет применяться к экземпляру (это значение по умолчанию).

Синтаксис:

gravity;

Возвращается:

(single precision floating point value)

Пример:

if !place_meeting(x, y + 1, obj_Ground)
{
    gravity = 0.01;
}
else
{
    gravity = 0;
}

Приведенный выше код будет применять гравитацию, только если экземпляр не найдет под собой ни одного экземпляра "obj_Ground".
