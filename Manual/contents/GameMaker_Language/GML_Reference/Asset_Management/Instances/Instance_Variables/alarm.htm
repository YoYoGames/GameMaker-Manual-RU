сигнализация

Этот одномерный массив используется для получения текущего значения для всех сигналов тревоги, которые могут быть у экземпляра, или для установки этих сигналов. В каждый экземпляр объекта встроено двенадцать сигналов тревоги, и у каждого из них есть свой собственный event которая сработает, когда эта переменная достигнет 0.

Сигналы тревоги отсчитываются в начале каждого шага, после события Begin Step, но до события Main Step, и до того, как что-либо будет отрисовано для данного кадра. Это означает, что если вы установите сигнал тревоги на 1 в событии Begin Step, его событие будет запущено в том же кадре (поскольку сигнал тревоги будет отсчитан до 0 после завершения Begin Step), но если вы установите сигнал тревоги на 1 в событии Step, он будет запущен в следующем кадре.

Следует отметить, что тревога не завершена, когда она достигает 0 (хотя событие сработало), так как на следующем шаге она снизится до -1, поэтому если вам нужно остановить тревогу по какой-либо причине, вы должны установить значение массива на -1 , а не на 0. Время тревоги рассчитывается в игровых шагах, при этом значение 30 - это 30 шагов, 60 - 60 шагов и т.д..

ПРИМЕЧАНИЕ Сигнал тревоги, в котором нет никаких действий или кода, не будет запущен. Однако даже при наличии только комментария и отсутствии кода сигнал тревоги будет отсчитывать время.

Вы можете установить массив сигналов тревоги напрямую (и получить текущее значение сигнала тревоги также напрямую, используя массив), но при некоторых обстоятельствах это не самый подходящий метод, поэтому вы также можете использовать функцию alarm_set() для установки сигнала тревоги и функцию alarm_get() для получения значения сигнала тревоги.

Синтаксис:

alarm[val];

Возвращается:

(-1 if the alarm isn't running)

Пример:

if canshoot
{
    if (keyboard_check_pressed(vk_space))
    {
        canshoot = false;
        alarm[0] = room_speed;
        instance_create_layer(x, y, "Bullets", obj_Bullet);
    }
}

Приведенный выше код проверяет переменную, затем, если она истинна, проверяет нажатие клавиши, и если она истинна, то в конце устанавливает переменную в false, устанавливает тревогу (где она снова будет установлена в true) и создает экземпляр.
