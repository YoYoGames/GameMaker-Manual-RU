vspeed

vspeed - это одно из "встроенных" свойств, которым обладают все экземпляры, и определяет скорость вертикального перемещения (вдоль оси y) экземпляра в пикселях за шаг. Так, значение vspeed, равное 3, означает движение на 3 пикселя вниз (+y) каждый шаг, а vspeed -3 означает движение на 3 пикселя вверх (-y) каждый шаг.

Обратите внимание, что если вы установите переменные speed и/или direction, то значение vspeed будет обновлено автоматически, чтобы и аналогично, изменение значения vspeed также повлияет на значения speed и direction. Также обратите внимание, что gravity, gravity_direction и friction могут изменять значение этой переменной, когда они используются в ваших играх.

Синтаксис:

vspeed;

Возвращается:

(single precision floating point value)

Пример:

if keyboard_check(vk_up) vspeed = -5;
if keyboard_check(vk_down) vspeed = 5;

Приведенный выше код будет изменять вертикальную скорость в зависимости от того, какие клавиши нажаты.
