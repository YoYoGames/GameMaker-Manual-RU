трение

Friction является одним из "встроенных" свойств, которые могут иметь экземпляры, и может использоваться для замедления работы экземпляра с течением времени, если параметр speed отлична от нуля. Оно работает просто, вычитая определенную величину из скорости на каждом шаге, пока скорость объекта не станет равной 0. Так, если трение установлено, например, на 0,1, а скорость экземпляра равна 1 (1 пиксель за шаг), он замедлится и остановится после 10 шагов. Обратите также внимание, что трение применяется к положительным и отрицательным скоростям одинаково, и в итоге через определенное время скорость объекта всегда будет равна 0.

Синтаксис:

friction;

Возвращается:

(single precision floating point value)

Пример:

if abs(speed) > 0
{
    friction = 0.05;
}
else
{
    friction = 0;
}

Приведенный выше код будет применять трение, только если абсолютная скорость экземпляра больше 0.
