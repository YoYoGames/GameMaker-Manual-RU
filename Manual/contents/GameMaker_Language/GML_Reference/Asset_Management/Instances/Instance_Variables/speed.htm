speed

Все экземпляры в GameMaker имеют определенные "встроенные" свойства, которые можно использовать и устанавливать для управления их внешним видом и поведением. speed является одним из таких свойств и определяет, на сколько пикселей экземпляр будет перемещаться каждый шаг. В отличие от hspeed и vspeed, speed не имеет связанного с ним направления, так как это определяется direction значение экземпляра, но оно может иметь отрицательное значение, в этом случае экземпляр будет перемещаться в направлении, противоположном тому, которое задано функцией direction (т.е.: direction - 180°). Обратите внимание, что установка speed и/или direction, также изменит значения переменных hspeed и vspeed, и что gravity, gravity_direction и friction могут изменять значение этой переменной, когда они используются в ваших играх.

Синтаксис:

speed

Возвращается:

(single precision floating point value)

Пример:

if keyboard_check(vk_up) speed = 2;
if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

Приведенный выше код будет использовать клавиши со стрелками для установки скорости и направления движения экземпляра.
