direction

Все экземпляры в GameMaker имеют определенные "встроенные" свойства, которые можно использовать и устанавливать для управления их внешним видом и поведением. Direction является одним из таких свойств и может быть использовано для установки направления движения экземпляра. когда экземпляр имеет значение speed Обратите внимание, что направления в GameMaker обычно рассчитываются как 0° - вправо, 90° - вверх, 180° - влево и 270° - вниз. gravity и gravity_direction могут изменять значение direction, когда они используются в ваших играх.

Синтаксис:

direction

Возвращается:

(single precision floating point value)

Пример:

if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

Приведенный выше код будет изменять направление движения экземпляра в зависимости от того, какая клавиша (левая или правая) нажата.
