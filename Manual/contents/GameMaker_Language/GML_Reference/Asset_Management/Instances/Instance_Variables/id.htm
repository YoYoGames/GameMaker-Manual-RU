id

Эта переменная, доступная только для чтения, содержит уникальный идентификационный номер экземпляра. Каждому экземпляру, который вы создаете - будь то с помощью кода или путем добавления в комнату в редакторе комнат - присваивается номер, который используется для внутренней идентификации этого экземпляра, а переменная id - это то, что вы можете использовать для ссылки на него. Номер id также возвращается (и может быть сохранен в переменной) при создании экземпляра с помощью instance_create_layer() или instance_create_depth(), а также других функций экземпляра.

Теперь эта переменная является ссылкой на экземпляр, а не просто вещественным числом. Вы можете получить идентификационный номер экземпляра, вызвав real() или int64() на этой переменной.

ПРИМЕЧАНИЕ Значение этой переменной не совпадает с идентификатором, который присваивается экземпляру в редакторе помещений, а также отличается от массива instance_id массив, который содержит все идентификаторы всех активных в данный момент экземпляров.

Синтаксис:

id;

Возвращается:

Instance ID

Пример:

for (var i = 0; i < instance_count; i++;)
{
    if instance_id[i] != id
    {
        instance_id[i].scr += 5;
    }
}

Приведенный выше код добавляет 5 к переменной "scr" для каждого активного экземпляра в комнате, кроме того, который выполняет код. Это происходит путем перебора ВСЕХ активных экземпляров (используя массив "instance_id" для возврата ID каждого активного экземпляра) и сравнения их с встроенной переменной "id", которая является ID исходного экземпляра, выполняющего код.
