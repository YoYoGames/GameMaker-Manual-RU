Переменные экземпляра

Когда вы создаете новый объект, он будет поставляться с определенными переменными, уже инициализированными значениями по умолчанию. Эти переменные мы называем переменными экземпляра, поскольку они будут создаваться для каждого экземпляра объекта, который вы размещаете в игровых комнатах, и после создания экземпляра значения этих переменных уникальны для этого и только этого экземпляра. Некоторые из этих переменных вы будете использовать часто, например, x и y, в то время как другие нужны меньше, в зависимости от того, что вы хотите сделать, но в целом они очень полезны, и по возможности вы должны использовать их, а не создавать свои собственные переменные.

Большинство переменных экземпляра можно как устанавливать, так и считывать, что позволяет изменять свойства и поведение экземпляра, просто изменяя значение определенной переменной. Например, вы можете запретить экземпляру рисовать, просто установив встроенную переменную visible в значение false. Ниже приведены различные переменные, которые инициализируются для всех экземпляров всех объектов в вашей игре.

ВАЖНО Если экземпляр находится в sequence то некоторые из этих переменных - например, image_xscale / image_yscale / image_angle / x / y - будут перезаписаны при обновлении последовательности на каждом шаге после начала воспроизведения.

Эти переменные относятся к общим свойствам экземпляра:

  • id
  • visible
  • solid
  • persistent
  • depth
  • layer
  • alarm

Эти переменные связаны с положением и перемещением экземпляра:

  • direction
  • friction
  • gravity
  • gravity_direction
  • hspeed
  • vspeed
  • speed
  • xstart
  • ystart
  • x
  • y
  • xprevious
  • yprevious

Свойства объекта

Следующая переменная содержит уникальный индексный идентификатор объекта, на основе которого был создан экземпляр:

  • object_index

Все эти переменные связаны с sprite, назначенным экземпляру, и могут быть использованы для изменения того, что и как рисуется:

  • sprite_index
  • sprite_width
  • sprite_height
  • sprite_xoffset
  • sprite_yoffset
  • image_alpha
  • image_angle
  • image_blend
  • image_index
  • image_number
  • image_speed
  • image_xscale
  • image_yscale

Эти переменные относятся к маске столкновений:

  • mask_index
  • bbox_bottom
  • bbox_left
  • bbox_right
  • bbox_top

Эти переменные связаны с путями и тем, как экземпляр взаимодействует с одним из них, если он назначен:

  • path_index
  • path_position
  • path_positionprevious
  • path_speed
  • path_scale
  • path_orientation
  • path_endaction

Эти переменные предназначены для настройки экземпляра на использование временной шкалы:

  • timeline_index
  • timeline_running
  • timeline_speed
  • timeline_position
  • timeline_loop

Последовательности

Следующие переменные хранят информацию о Последовательности, которая может управлять экземпляром в любой момент времени:

  • in_sequence
  • sequence_instance

Существует огромное количество встроенных переменных для использования с физическими функциями GameMaker, и чтобы было понятнее, их можно найти в разделе руководства, где рассматривается все, что связано с физическим моделированием:

  • Физика
