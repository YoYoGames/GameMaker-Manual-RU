направление силы тяжести

gravity_direction является одним из "встроенных" свойств всех экземпляров и может использоваться для задания направления движения, если значение экземпляра gravity Обратите внимание, что в GameMaker направления обычно рассчитываются как 0° - вправо, 90° - вверх, 180° - влево и 270° - вниз.

Синтаксис:

gravity_direction;

Возвращается:

(single precision floating point value)

Пример:

if place_meeting(x, y, obj_switch)
{
    gravity_direction += 180;
}

Приведенный выше код изменит направление гравитации, если объект игрока встретит экземпляр "obj_switch".
