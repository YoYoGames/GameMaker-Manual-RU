видимый

Экземпляр можно пометить как visible или нет, установив эту переменную в значение true (видимый) или false (невидимый). При этом GameMaker пропускает событие рисования для данного экземпляра, поэтому при использовании этого параметра следует соблюдать осторожность, так как это означает, что никакой код, помещенный в событие рисования, не будет выполнен, пока экземпляр не виден. Также обратите внимание, что если layer которому назначен экземпляр, помечен как невидимый, установка этой переменной в true не будет иметь никакого эффекта, пока сам слой не будет помечен как видимый.

Синтаксис:

visible;

Возвращается:

Пример:

if other.visible = true
{
    x = xprevious;
    y = yprevious;
}

Приведенный выше код проверит флаг видимости экземпляра "другого" в событии столкновения и, если он установлен на true, переместит этот экземпляр на прежнее место.
