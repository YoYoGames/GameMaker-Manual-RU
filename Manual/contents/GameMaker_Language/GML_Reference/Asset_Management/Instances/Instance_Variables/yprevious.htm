yprevious

Эта встроенная переменная возвращает предыдущую позицию y для данного экземпляра. Эта переменная будет установлена в текущее положение x непосредственно перед началом события begin step, но она также может быть установлена через код в любое время, что означает, что вы можете дать ей собственное значение, если это необходимо.

Синтаксис:

yprevious;

Возвращается:

(single precision floating point value)

Пример:

if x != xprevious || y != yprevious
{
    moved = true;
}

Приведенный выше код проверяет, есть ли разница между значениями xprevious и yprevious и текущими значениями x и y. Если есть, то устанавливается переменная true. Если есть, то переменная устанавливается в true.
