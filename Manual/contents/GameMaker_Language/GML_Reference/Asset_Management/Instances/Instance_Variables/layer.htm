слой

Эта встроенная переменная создается для каждого экземпляра в комнате и содержит значение ID слоя, к которому приписан данный экземпляр. Это значение затем может быть использовано в других функциях, таких как layer_get_depth() или его можно изменить, чтобы переместить экземпляр на другой слой, но учтите, что если назначаемый слой не существует в текущей комнате, то вы получите ошибку, которая заставит вашу игру закрыться. При назначении слоя вы должны указать уникальный идентификатор слоя, возвращаемый функцией layer_get_id() (при использовании именованных слоев комнаты), или как возвращается функцией layer_create() (когда вы создаете свои собственные слои во время выполнения).

ВАЖНО Если вы создали экземпляр с помощью функции instance_create_depth() или вручную изменили depth переменную, слой, назначенный данному экземпляру, становится "управляемым" слоем, то есть таким, который GameMaker контролирует и управляет автоматически. В этих случаях переменная layer вернет значение -1.

Синтаксис:

layer;

Возвращается:

Layer ID

Пример:

layer = layer_create(-1000);

Приведенный выше код создаст новый слой с глубиной -1000, а затем установит переменную экземпляра layer в возвращаемый ID слоя, перемещая экземпляр со слоя, на котором он сейчас находится, на новый создаваемый слой.
