позиция экземпляра

С помощью этой функции можно проверить позицию на столкновение с другим экземпляром или всеми экземплярами объекта. При использовании этой функции вы проверяете одну точку в комнате на наличие экземпляра или объекта. Проверка будет выполнена по граничной области экземпляра или по маске экземпляра, если у этого экземпляра есть точная проверка столкновений, и вернет уникальный экземпляр. id. Если вам не нужен идентификатор столкнувшегося экземпляра, то вместо этого следует использовать position_meeting() вместо этого. Эта функция также принимает специальные ключевые слова all и other и возвращает ключевое слово noone, если столкновения не произошло, или уникальное значение ID найденного экземпляра, если столкновение произошло.

Синтаксис:

instance_position( x, y, obj );

Аргумент Тип Описание
x Реально The x position to check for instances.
y Реально The y position to check for instances.
obj Идентификаторактива или экземпляраобъекта The object to check for instances of.

Возвращается:

Instance ID or noone

Пример:

var inst;
inst = instance_position(mouse_x, mouse_y, obj_Pause_Button);
if (inst != noone)
{
    with (inst) image_index=1;
    instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu);
}

Приведенный выше код проверит наличие столкновения с экземпляром "obj_Pause_Button" в позиции мыши, и если оно есть, то будет использовать возвращенный id для установки его image_index на новое значение перед созданием нового экземпляра объекта "obj_Menu".
