экземпляр_активировать_слой

С помощью этой функции можно активировать слой, который ранее был деактивирован. Вам необходимо указать ID слоя, который может быть либо именем слоя, записанным в редакторе кода (в виде строки), либо фактическим значением ID слоя, возвращаемым функцией layer_create() и все деактивированные экземпляры на этом слое снова активируются. Обратите внимание, что если вы деактивировали слой, в котором есть постоянные экземпляры, то перед сменой комнаты вам нужно будет снова активировать слой с помощью этой функции, иначе все постоянные экземпляры не будут перенесены и будут отброшены. Обратите также внимание, что активация не является мгновенной, и экземпляр, активированный таким образом, не будет считаться активным до окончания события, в котором была вызвана функция.

Синтаксис:

instance_activate_layer(layer_id);

Аргумент Тип Описание
layer_id Идентификатор слоя или строка The layer name string (or ID value) to be used

Возвращается:

N/A

Пример:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
instance_deactivate_region(view_xview[0] - 64, view_yview - 64, _vx + 128, _vy + 128, false, false);
instance_activate_layer("Player_Layer");

Приведенный выше код активирует все экземпляры в пределах комнаты, а затем деактивирует те, которые находятся за пределами текущего обзора камеры, за исключением экземпляров на слое "Player_Layer", которые снова активируются в конце.
