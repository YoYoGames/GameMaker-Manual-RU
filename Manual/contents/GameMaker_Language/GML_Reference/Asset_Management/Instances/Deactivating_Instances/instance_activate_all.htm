экземпляр_активировать_все

С помощью этой функции вы можете указать программе GameMaker активировать все экземпляры, которые ранее были деактивированы в комнате. Обратите внимание, что активация не является мгновенной, и экземпляр, активированный таким образом, не будет считаться активным до окончания события, в котором была вызвана функция.

Синтаксис:

instance_activate_all();

Возвращается:

Пример:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);

Приведенный выше код активирует все экземпляры в пределах комнаты, а затем деактивирует те, которые на 64px выходят за пределы текущего обзора камеры.
