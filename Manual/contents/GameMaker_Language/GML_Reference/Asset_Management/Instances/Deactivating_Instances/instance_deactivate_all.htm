экземпляр_деактивировать_все

Эта функция деактивирует все экземпляры, находящиеся в комнате в момент выполнения кода. Это может включать экземпляр, выполняющий код, если флаг "notme" установлен в false, но обычно вы хотите, чтобы этот экземпляр был активен, в этом случае флаг "notme" должен быть установлен в true. Обратите внимание, что деактивация не является мгновенной, и экземпляр, деактивированный таким образом, не будет считаться неактивным до окончания события, в котором была вызвана функция.

ПРИМЕЧАНИЕ: Если вы деактивируете экземпляр при запуске комнаты (т.е. из кода создания комнаты или из события создания экземпляра в комнате), то все экземпляры, размещенные в комнате из редактора комнат, будут выполнять свое событие создания до деактивации.

ПРЕДУПРЕЖДЕНИЕ: Деактивация экземпляров, у которых включена физика, НЕ остановит взаимодействие их светильников в рамках физической симуляции. Для этого необходимо установить их phy_active true или false при активации/деактивации экземпляров.

Синтаксис:

instance_deactivate_all(notme);

Аргумент Тип Описание
notme Булево Whether to keep the calling instance activated (true) or not (false).

Возвращается:

N/A

Пример:

instance_deactivate_all(true);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

Приведенный выше код деактивирует все экземпляры, кроме того, который выполняет код, а затем активирует регион в комнате.
