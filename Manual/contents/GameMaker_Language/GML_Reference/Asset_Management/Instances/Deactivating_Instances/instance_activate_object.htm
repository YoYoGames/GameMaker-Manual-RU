экземпляр_активировать_объект

С помощью этой функции вы можете активировать один экземпляр или все экземпляры определенного объекта из тех, которые были деактивированы ранее. Обратите внимание, что если вы деактивировали экземпляр или объект, который был помечен как постоянный, то вам нужно будет снова активировать его с помощью этой функции перед сменой комнаты, иначе он не будет перенесен и будет отброшен. Обратите внимание, что активация не является мгновенной, и экземпляр, который был активирован таким образом, не будет считаться активным. не будет считаться активным до окончания события, в котором была вызвана эта функция.

Синтаксис:

instance_activate_object(obj);

АргументТип Описание
obj The object or instance to activate (the keyword all can also be used).

Возвращается:

Пример:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);
instance_activate_object(obj_Lights);

Приведенный выше код активирует все объекты в пределах комнаты, а затем деактивирует те, которые находятся за пределами текущего обзора камеры, за исключением объекта "obj_Lights", который снова активируется в конце.
