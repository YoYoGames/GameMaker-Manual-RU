alarm_set

Эта функция может использоваться для установки сигнала тревоги. Вы вводите номер сигнала тревоги от 0 до 11, а затем значение, на которое нужно установить сигнал тревоги. Значение должно быть целым, и вы можете установить его на -1, чтобы остановить сигнал тревоги (нецелые значения будут округлены до ближайшего целого числа). Это альтернативный метод непосредственной установки массива сигналов тревоги.

Синтаксис:

alarm_set(index, value);

Аргумент Тип Описание
index Реально The alarm index to set, from 0 to 11.
value Реально The value (an integer) to set the alarm to.

Возвращается:

N/A

Пример:

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

Приведенный выше код проверяет все сигналы тревоги в вызывающем экземпляре и, если они больше 0, устанавливает их в -1, останавливая дальнейший отсчет.
