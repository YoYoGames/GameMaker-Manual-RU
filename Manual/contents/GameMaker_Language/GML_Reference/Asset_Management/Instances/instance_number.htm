номер экземпляра

С помощью этой функции можно узнать, сколько активных экземпляров указанного объекта существует в комнате. При проверке с помощью этой функции, если объект является родительским, то все дочерние объекты также будут включены в возвращаемое значение, а также обратите внимание, что те экземпляры, которые были деактивированы с помощью функций instance deactivate, не будут включены в эту проверку.

Синтаксис:

instance_number(obj);

АргументТип Описание
obj The object to total the number of instances of.

Возвращается:

Пример:

if instance_number(object_index) < 50
{
    instance_create_layer(random(room_width), random(room_height), "Instances", object_index);
}

Приведенный выше код проверит количество экземпляров, созданных в форме того же объекта, что и текущий экземпляр, а затем, если их меньше 50, создаст еще один экземпляр в случайном месте в пределах комнаты.
