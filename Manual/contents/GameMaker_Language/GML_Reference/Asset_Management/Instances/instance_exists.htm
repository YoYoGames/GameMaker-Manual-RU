instance_exists

Эта функция может быть использована двумя способами в зависимости от того, что вы хотите проверить. Вы можете передать ей object_index для проверки, в этом случае функция вернет true, если в текущей комнате есть активные экземпляры указанного объекта, или вы можете передать ей id экземпляра, в этом случае функция вернет true, если этот конкретный экземпляр существует и активен в текущей комнате.

Обратите внимание, что эта функция не учитывает те экземпляры, которые были деактивированы с помощью функций деактивации экземпляров.

Синтаксис:

instance_exists(obj);

Аргумент Тип Описание
obj Идентификатор экземпляра или актив объекта The object or instance to check for the existence of.

Возвращается:

Boolean

Пример:

if !instance_exists(obj_Enemy)
{
    score += 200;
    room_goto(rm_hiscores);
}

Приведенный выше код проверяет, существуют ли экземпляры объекта "obj_Enemy", и если нет, то добавляет к переменной "score" и меняет комнату.
