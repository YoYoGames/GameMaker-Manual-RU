экземпляр_найти

Все экземпляры имеют уникальный идентификатор (id), который можно использовать для изменения и манипулирования ими во время игры, но вы не всегда можете знать идентификатор конкретного экземпляра, поэтому эта функция может помочь, так как с ее помощью вы можете перебрать все экземпляры, чтобы найти то, что вам нужно. Вы указываете объект, экземпляр которого хотите найти, и число, и если в списке экземпляров есть экземпляр на этой позиции, то функция возвращает id этого экземпляра, а если нет, то возвращает специальное ключевое слово noone. Вы также можете использовать ключевое слово all для перебора всех экземпляров в комнате, а также родительский объект для перебора всех экземпляров, которые являются частью иерархии родитель/ребенок, и вы даже можете указать экземпляр (если у вас есть его id) в качестве проверки, существует ли он в текущей комнате. Обратите внимание, что поскольку экземпляры сортируются произвольным образом, нет определенного порядка проверки экземпляров этой функцией, и любой экземпляр может находиться в любой позиции.

Максимальное значение для "n" в этой функции будет равно

  • Для ключевого слова all: instance_count - 1
  • Для индекса объекта: instance_number(OBJ) - 1

Синтаксис:

instance_find(obj, n);

Аргумент Тип Описание
obj Объект Активы The object to find the nth instance of
n Реально The number of the instance to find.

Возвращается:

Instance ID or noone

Пример:

for (var i = 0; i < instance_number(obj_Enemy); ++i;)
{
    enemy[i] = instance_find(obj_Enemy,i);
}

Приведенный выше код будет использовать цикл for для перебора всех экземпляров "obj_Enemy" и сохранения их id в массиве "enemy[]".
