экземпляр_ближайший

Эта функция проверяет все экземпляры заданного объекта, чтобы узнать, какой из них находится ближе всего к заданной точке x/y. Все проверки будут проводиться от заданной точки x/y до начала координат (точки x/y) экземпляров указанного объекта. Если экземпляров объекта не существует, функция вернет ключевое слово noone, но если экземпляры существуют, то функция вернет значение id найденного экземпляра. Обратите внимание, что если экземпляр, выполняющий код, имеет тот же индекс объекта, что и проверяемый объект, то он будет включен в проверку (это включает проверку родительских объектов, если вызывающий экземпляр также является дочерним по отношению к родительскому).

Синтаксис:

instance_nearest(x, y, obj);

АргументТип Описание
x The x position to check from.
y The y position to check from.
obj The object to check for instances of.

Возвращается:

Instance ID or noone

Пример:

var inst, xx;
xx = x;
x -= 10000;
inst = instance_nearest(xx, y, object_index);
if (inst != id)
{
    draw_line(x, y, inst.x, inst.y);
}
x += 10000;

Приведенный выше код перемещает текущий экземпляр на 10000 пикселей, затем проверяет его предыдущее положение, чтобы найти ближайший экземпляр объекта того же типа. Если этот экземпляр окажется тем же самым, он не будет делать ничего другого, кроме как переместится в исходное положение, но если найденный экземпляр будет другим, он проведет линию между ними.
