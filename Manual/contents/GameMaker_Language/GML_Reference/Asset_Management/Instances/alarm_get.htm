alarm_get

Эта функция может быть использована для получения текущего значения заданного сигнала тревоги. Вы вводите номер сигнала тревоги от 0 до 11, и эта функция возвращает значение, на котором находится сигнал тревоги в данный момент. Это альтернативный метод прямого получения значения массива сигналов тревоги.

Синтаксис:

alarm_get(index);

Аргумент Тип Описание
index Реально The alarm index to get, from 0 to 11.

Возвращается:

Real (integer)

Пример:

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

Приведенный выше код проверяет все сигналы тревоги в вызывающем экземпляре и, если они больше 0, устанавливает их в -1, останавливая дальнейший отсчет.
