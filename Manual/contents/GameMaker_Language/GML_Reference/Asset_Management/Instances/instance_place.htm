место_объекта

С помощью этой функции вы можете проверить позицию на столкновение с другим экземпляром или со всеми экземплярами объекта, используя маску столкновения того экземпляра, который выполняет код проверки. При использовании этой функции вы фактически просите GameMaker переместить экземпляр в новую позицию, проверить наличие столкновения, вернуться назад и сообщить вам, было ли обнаружено столкновение или нет.

Это будет работать для точных столкновений, но только если и у экземпляра, и у проверяемого объекта выбраны маски точных столкновений, в противном случае применяются только столкновения с ограничивающей рамкой.

Эта функция вернет уникальный экземпляр id сталкиваемого объекта, но если в этом нет необходимости, то немного быстрее использовать функцию place_meeting(). Эта функция также принимает специальные ключевые слова all и other и вернет ключевое слово noone, если столкновения не произошло, или уникальное значение ID найденного экземпляра, если столкновение все же произошло.

Синтаксис:

instance_place(x, y, obj);

Аргумент Тип Описание
x Реально The x position to check for instances.
y Реально The y position to check for instances.
obj Идентификаторактива или экземпляраобъекта The object to check for instances of.

Возвращается:

Instance ID or noone

Пример:

var _inst = instance_place(x, y, obj_Enemy);
if _inst != noone
{
    hp -= _inst.dmg;
    instance_destroy(_inst);
}

Приведенный выше код проверит наличие столкновения с экземплярами "obj_Enemy" и, если таковое произойдет, уменьшит переменную "hp" на величину, хранящуюся в переменной "dmg" столкнувшегося экземпляра, а затем уничтожит столкнувшийся экземпляр.
