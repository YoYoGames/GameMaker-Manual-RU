object_set_solid

С помощью этой функции вы можете отметить конкретный объект как твердый или нет (более подробную информацию о флаге "твердый" см. в разделе " Редактор объектов"). Это означает, что все экземпляры этого объекта, созданные после изменения параметра solid, будут созданы с этим новым состоянием, в то время как экземпляры, уже находящиеся в комнате, могут быть не затронуты.

Обратите внимание, что это не функция экземпляра! Вы можете установить флаг solid для отдельных экземпляров с помощью переменной solid и таким образом получить десять экземпляров, помеченных как solid, даже если для данного объекта установлен флаг false в IDE, и даже если вы измените флаг solid объекта на true с помощью этой функции, все экземпляры, которые в данный момент находятся в комнате, останутся такими, какими они были, и только экземпляры, созданные после вызова функции, будут начинаться с флагом solid, помеченным как true.

Синтаксис:

object_set_solid( index, solid );

Аргумент Тип Описание
index Объект Активы The index of the object to change.
solid Булево The new solidity of the object (true= твердый, false= не твердый).

Возвращается:

N/A

Пример:

if !object_get_solid(obj_Block)
{
    object_set_solid(obj_Block, true);
}

Приведенный выше код проверяет флаг solid объекта "obj_Block", и если он равен false, то он изменяется на true.
