object_set_visible

С помощью этой функции вы можете отметить определенный объект как видимый или нет (более подробную информацию о флаге "видимый" см. в разделе " Редактор объектов"). Это означает, что все экземпляры этого объекта, созданные после изменения параметра visible, будут созданы с этим новым состоянием, в то время как экземпляры, уже находящиеся в комнате, могут быть не затронуты.

Обратите внимание, что это не функция экземпляра! Вы можете установить флаг видимости отдельных экземпляров с помощью переменной visible и таким образом иметь десять экземпляров, помеченных как видимые, даже если у данного объекта флаг установлен в false на сайте IDE, и даже если вы измените флаг видимости объекта на true с помощью этой функции, все экземпляры, которые в данный момент находятся в комнате, останутся такими, какими они были, и только экземпляры, созданные после вызова функции, будут начинаться с флагом видимости, помеченным как true.

Синтаксис:

object_set_visible( index, vis );

Аргумент Тип Описание
index Объект Активы The index of the object to change.
vis Булево The new visibility of the object (true=видимый, false=невидимый).

Возвращается:

N/A

Пример:

if !object_get_visible(obj_Block)
{
    object_set_visible(obj_Block, true);
}

Приведенный выше код проверяет флаг видимости объекта "obj_Block", и если он ложный, то он изменяется на истинный.
