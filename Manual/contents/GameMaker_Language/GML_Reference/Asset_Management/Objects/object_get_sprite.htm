объект_get_sprite

Эта функция сообщит вам, есть ли у проверяемого объекта sprite или нет, и если есть, то вернет индекс этого sprite, или -1, если нет. Обратите внимание, что это не функция экземпляра! Вы можете иметь объект без sprite, в то время как экземпляр этого же объекта может иметь его и наоборот, или даже иметь разные sprites. Вы можете установить индивидуальный экземпляр sprite с помощью sprite_index переменную instance.

Синтаксис:

object_get_sprite(obj);

АргументТип Описание
obj The index of the object to check

Возвращается:

Sprite Asset

Пример:

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
    sprite_index = _spr;
}

В приведенном выше примере sprite_index экземпляра будет сверен с sprite экземпляра object_index. Если они не совпадают, то экземпляру будет присвоен тот же sprite, что и индексу объекта.
