object_set_sprite

С помощью этой функции вы можете установить индекс sprite определенного объекта. Это означает, что все экземпляры этого объекта, которые будут созданы после изменения sprite_index, будут созданы с этим новым sprite_index, в то время как экземпляры, которые уже находятся в комнате, могут быть не затронуты.

Обратите внимание, что это не функция экземпляра! Вы можете установить индекс sprite отдельных экземпляров с помощью переменной sprite_index и таким образом иметь десять экземпляров с индексом sprite, отличным от индекса объекта, из которого они созданы, и даже если вы измените индекс sprite объекта с помощью этой функции, все экземпляры, находящиеся в комнате в данный момент, останутся такими, какими они были, и только экземпляры, созданные после вызова функции, будут начинаться с нового sprite.

Синтаксис:

object_set_sprite( index, spr );

Аргумент Тип Описание
index Объект Активы The index of the object to change.
spr Актив спрайта The sprite to assign to the object.

Возвращается:

N/A

Пример:

switch (room)
{
    case rm_start: object_set_sprite(obj_Player, spr_uniform); break;
    case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;
    case rm_end: object_set_sprite(obj_Player, spr_casual); break;
}
instance_create_layer(32, 32, "Instances", obj_Player);

Приведенный выше код установит индекс объекта "obj_Player" sprite на разные значения в зависимости от комнаты, в которой в данный момент находится экземпляр, выполняющий код. Затем создается экземпляр "obj_Player".
