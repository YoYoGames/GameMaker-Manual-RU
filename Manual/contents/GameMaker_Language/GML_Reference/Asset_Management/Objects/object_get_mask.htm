object_get_mask

Эта функция сообщит вам, имеет ли проверяемый объект индекс маски или нет, и если имеет, то вернет индекс этой маски (который является активом sprite ), или -1, если не имеет. Обратите внимание, что это не функция экземпляра! Вы можете иметь объект без маски, а экземпляр этого же объекта может иметь маску, и наоборот, или даже иметь разные маски. Вы можете установить индекс маски отдельного экземпляра с помощью переменной mask_index переменной instance.

Синтаксис:

object_get_mask(obj);

АргументТип Описание
obj The index of the object to check

Возвращается:

Sprite Asset

Пример:

if (mask_index != object_get_mask(object_index))
{
    mask_index = object_get_mask(object_index);
}

В приведенном выше примере индекс маски экземпляра сверяется с маской индекса object_index экземпляра. Если они не совпадают, то экземпляру будет присвоена та же маска, что и у индекса объекта.
