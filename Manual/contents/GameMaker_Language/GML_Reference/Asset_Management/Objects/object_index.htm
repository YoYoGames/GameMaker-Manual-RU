объектный_индекс

Эта переменная, доступная только для чтения, возвращает индекс объекта, на основе которого был создан данный экземпляр. Это не то же самое, что имя объекта, которое является строкой и может быть найдено с помощью функции object_get_name(), поскольку эта функция возвращает номер индекса, который является уникальным значением, присваиваемым GameMaker каждому объекту в момент создания.

Синтаксис:

object_index;

Возвращается:

Object Asset

Пример:

obj_name = object_get_name(object_index);

Приведенный выше код будет использовать object_index для поиска имени объекта, из которого был создан текущий экземпляр.
