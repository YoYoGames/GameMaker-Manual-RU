объект_существует

Эта функция возвращает, является ли указанное значение активом объекта (true) или нет (false). Обратите внимание, что эта функция проверяет, присутствует ли объект в браузере активов, а не в игровой комнате. Для этого следует использовать функцию instance_exists().

Синтаксис:

object_exists(obj);

АргументТип Описание
obj The index of the object to check.

Возвращается:

Пример:

var object_id = buffer_read(net_buffer, buffer_u16);
var pos_x = buffer_read(net_buffer, buffer_u16);
var pos_y = buffer_read(net_buffer, buffer_u16);

if (object_exists(object_id))
{
    instance_create_layer(pos_x, pos_y, "Instances", object_id);
}

Приведенный выше код считывает данные из буфера, полученного по сети. Поскольку он содержит идентификатор объекта (используемый для порождения экземпляров), нам сначала нужно убедиться, что значение не является недействительным и что объект действительно существует. Для этого мы вызываем object_exists() в условии, и если оно истинно, мы создаем экземпляр этого объекта в полученной позиции.
