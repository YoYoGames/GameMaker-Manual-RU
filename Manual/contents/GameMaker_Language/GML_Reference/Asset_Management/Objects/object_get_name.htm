имя_объекта

Эта функция возвращает имя в виде строки указанного объекта. Это имя - то, которое было указано для объекта в Asset Browser главного окна GameMaker. Обратите внимание, что это только строка и не может быть использована для прямой ссылки на объект - для этого вам понадобится индекс объекта. Однако вы можете использовать эту строку для получения индекса объекта, используя возвращаемую строку вместе с функцией asset_get_index().

Синтаксис:

object_get_name(obj);

АргументТип Описание
obj The index of the object to check.

Возвращается:

Пример:

str = object_get_name(object_index);

Приведенный выше код получит имя индекса объекта для экземпляра, выполняющего код, и сохранит возвращаемое значение в переменной "str".
