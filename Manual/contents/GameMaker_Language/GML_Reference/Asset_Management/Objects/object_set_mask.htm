object_set_mask

С помощью этой функции вы можете установить mask_index определенного объекта (более подробную информацию о масках см. в разделе Редактор объектов). Это означает, что все экземпляры этого объекта, созданные после изменения маски, будут созданы с этой новой маской, в то время как экземпляры, уже находящиеся в комнате, могут быть не затронуты. Вы можете установить значение -1, чтобы удалить маску sprite и по умолчанию использовать маску, определенную для sprite объекта (или без маски, если не было выбрано sprite ).

Обратите внимание, что это не функция экземпляра! Вы можете установить индекс маски отдельных экземпляров с помощью переменной mask_index и таким образом иметь десять экземпляров с маской, отличной от маски объекта, из которого они созданы, и даже если вы измените индекс маски объекта с помощью этой функции, все экземпляры, находящиеся в комнате в данный момент, останутся такими, какими они были, и только экземпляры, созданные после вызова функции, начнут работать с новой маской.

Синтаксис:

object_set_mask(index, spr);

Аргумент Тип Описание
index Объект Активы The index of the object to change.
spr Актив спрайта The new sprite to use as the object's mask.

Возвращается:

N/A

Пример:

if (global.level == 10)
{
    object_set_mask(obj_Platform, spr_mask_10);
}

Приведенный выше код проверяет значение глобальной переменной и затем изменяет индекс маски объекта "obj_Platform", если он равен десяти.
