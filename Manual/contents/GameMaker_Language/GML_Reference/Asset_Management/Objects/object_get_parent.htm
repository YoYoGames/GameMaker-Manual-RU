object_get_parent

Эта функция получит индекс объекта любого родителя, который был назначен указанному объекту, или вернет -100, чтобы показать, что у объекта нет родителя, или -1, если проверяемый объект не существует. Более подробную информацию о родителях см. в разделе Редактор объектов.

Синтаксис:

object_get_parent(obj);

АргументТип Описание
obj The index of the object to check

Возвращается:

Object Asset

Пример:

par = object_get_parent(object_index);

Приведенный выше код получит родительский индекс объекта для экземпляра, выполняющего код, и сохранит возвращаемое значение в переменной "par".
