object_get_visible

Эта функция сообщит вам, был ли проверяемый объект отмечен как "видимый" (запускает событие рисования) или нет (не запускает событие рисования). Обратите внимание, что это не функция экземпляра! Таким образом, у вас может быть видимый объект и невидимый экземпляр одного и того же объекта, и наоборот. Вы можете установить видимость отдельного экземпляра с помощью переменной visible переменной instance.

Синтаксис:

object_get_visible(obj);

АргументТип Описание
obj The index of the object to check.

Возвращается:

Пример:

if (!visible) && (object_get_visible(object_index))
{
    visible = true;
}

Приведенный выше код проверит запущенный экземпляр на предмет того, виден он или нет, а также проверит индекс объекта экземпляра на предмет того, отмечен ли он как видимый или нет. Если экземпляр не виден, но индекс объекта отмечен как включенный, то для этого экземпляра будет установлено значение "visible" равное true.
