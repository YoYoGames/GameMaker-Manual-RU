event_perform_object

Эта функция работает так же, как и event_perform() только на этот раз вы можете указать события от другого объекта. Здесь есть множество опций, которые позволяют полностью смоделировать все возможные события, но обратите внимание, что это буквально просто выполняет весь код в событии, и игра не будет ничего изменять, чтобы заставить его сработать вручную, например, если вы выберете событие нажатия на клавиатуру, событие сработает, но соответствующая клавиша не будет распознана как нажатая. Или если вы выполните событие тревоги, счетчик тревоги не будет установлен на -1, а продолжит отсчет. Полный список доступных констант, необходимых этой функции, можно найти на странице функции event_perform().

ПРИМЕЧАНИЕ Действия в событии, вызванном с помощью этой функции, применяются к текущему экземпляру, а не к экземплярам данного объекта.

Синтаксис:

event_perform_object(obj, type, numb);

Аргумент Тип Описание
obj Объект Активы The object that should have its event triggered.
type Тип события Постоянная The type of event to perform.
numb Реальный или событийный номер Константа The specific event, if one is necessary (otherwise, just use 0).

Возвращается:

N/A

Пример:

event_perform_object(obj_Player, ev_keypress, ord("W"));

Это выполнит событие, связанное с Keyboard Check Pressed > W key от объекта "obj_Player" в текущем экземпляре.
