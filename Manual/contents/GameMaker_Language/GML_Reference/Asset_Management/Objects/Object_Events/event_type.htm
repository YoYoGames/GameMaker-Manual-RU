тип события

Эта переменная, доступная только для чтения, возвращает тип события, выполняемого в данный момент, который может быть одной из следующих констант:

  • ev_create
  • ev_destroy
  • ev_step
  • ev_alarm
  • ev_keyboard
  • ev_keypress
  • ev_keyrelease
  • ev_mouse
  • ev_collision
  • ev_other
  • ev_gesture
  • ev_draw


Полный список констант, относящихся к различным событиям, см. event_perform(), а если вам нужен номер события (т.е. под-событие), то вы должны проверить константу event_number.

Синтаксис:

event_type;

Возвращается:

Event Type Constant

Пример:

show_debug_message("Current Event = " + string(event_type));

Приведенный выше код покажет отладочное сообщение с типом события, которое выполняется в данный момент.
