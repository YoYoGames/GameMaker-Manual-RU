event_perform_async

Эта функция используется для выполнения любого из асинхронных событий, предусмотренных в GameMaker. Вы передаете константу события Async (показана в таблице ниже) и карту DS, которая будет доступна в вызываемом событии Async, в переменной async_load.

Тип события Async Постоянная
Постоянная Описание
ev_async_web_image_load Событие "Изображение загружено
ev_async_web HTTP-событие
ev_async_dialog Событие диалога
ev_async_web_iap Событие "Покупка внутри приложения
ev_async_web_cloud Облачное мероприятие
ev_async_web_networking Сетевое мероприятие
ev_async_web_steam Паровое мероприятие
ev_async_social Социальное мероприятие
ev_async_push_notification Событие Push-уведомления
ev_async_save_load Сохранить/загрузить событие
ev_async_audio_recording Событие записи звука
ev_async_audio_playback Событие воспроизведения звука
ev_async_system_event Системное событие

Неасинхронные события могут быть вызваны с помощью event_perform().

Обратите внимание, что карту DS, указанную во втором аргументе, не нужно уничтожать вручную, так как она будет автоматически уничтожена функцией после выполнения события Async.

Синтаксис:

event_perform_async(type, ds_map);

Аргумент Тип Описание
type Тип события Async Постоянная The type of event to perform (see the table above).
ds_map Идентификатор карты DS The DS map to use as async_load в вызываемом событии.

Возвращается:

N/A

Пример:

var _map = ds_map_create();

_map[? "id"] = "custom_async_event";
_map[? "result"] = true;
_map[? "data"] = { a: 13, b: 16 };

event_perform_async(ev_async_social, _map);

Приведенный выше код создает карту DS и заполняет ее пользовательскими записями для чтения в событии Async. Затем выполняется событие Async Social с только что созданной картой, переданной в качестве async_load для вызываемого события.
