номер_события

Эта переменная, доступная только для чтения, возвращает номер вызываемого в данный момент события, где номер фактически относится к "под-событию" события, то есть: для события step номер события может быть любой из констант ev_step_normal, ev_step_begin или ev_step_end. Полный список констант, которые доступны для конкретных подсобытий, см. event_perform(), а если вам нужно узнать само основное событие, вы должны проверить константу event_type.

Синтаксис:

event_number;

Возвращается:

Event Number Constant

Пример:

switch (event_number)
{
    case ev_step_normal: show_debug_message("Step event!"); break;
    case ev_game_start: show_debug_message("Game Start""); break;
    case ev_room_start: show_debug_message("Room Start!"); break;
}

Приведенный выше код можно вызвать из сценария и использовать для отображения отладочных сообщений, информирующих о том, какое событие в данный момент запущено.
