пользователь события

С помощью этой функции вы указываете экземпляру выполнить действия или код, который был помещен в одно из 16 событий, определенных пользователем. Эти события могут быть вызваны только таким образом или с помощью функции event_perform функция.

Синтаксис:

event_user(numb);

Аргумент Тип Описание
numb Реально The number of User Event to call, between 0 and 15.

Возвращается:

N/A

Пример:

event_user(4);

Это вызовет определяемое пользователем событие 4.
