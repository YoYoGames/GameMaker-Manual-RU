tileset_get_name

Эта функция возвращает имя в виде строки указанного актива набора плиток. Это имя - то, которое было указано для набора плиток в браузере активов главного окна GameMaker. Обратите внимание, что это только строка, и ее нельзя использовать для прямой ссылки на набор плиток - для этого вам понадобится индекс набора плиток. Однако вы можете использовать эту строку для получения индекса набора плиток, используя возвращаемую строку вместе с функцией asset_get_index().

Синтаксис:

tileset_get_name(index);

Аргумент Тип Описание
index Активы набора плиток The index of the tile set to get the name of.

Возвращает

String

Пример:

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_get_tileset(_m);
tileset_name = tileset_get_name(_t);

Приведенный выше код получит имя индекса набора плиток для заданного слоя, сохраняя возвращаемую строку в переменной tileset_name.
