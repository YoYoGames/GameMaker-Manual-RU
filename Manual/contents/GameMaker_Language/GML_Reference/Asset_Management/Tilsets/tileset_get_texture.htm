tileset_get_texture

Эта функция возвращает специальный указатель на страницу текстуры набора тайлов. Это значение затем может быть использовано в других функциях рисования, в частности в функциях 2D-примитивов, а также в функциях шейдера. Более подробную информацию о возвращаемой текстурной странице можно получить с помощью различных функций texture_, найденных здесь.

Синтаксис:

tileset_get_texture(tileset);

Аргумент Тип Описание
tileset Активы набора плиток The index of the tile set to use.

Возвращает

Texture

Пример:

var tex;
tex = tileset_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1);
draw_vertex_texture(0, 640, 0, 1);
draw_primitive_end();

Приведенный выше код нарисует полосу треугольника с 4 вершинами, текстурированную текстурой, хранящейся в переменной "tex".
