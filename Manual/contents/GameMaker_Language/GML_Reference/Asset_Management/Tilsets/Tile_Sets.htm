Наборы плиток

Следующие функции доступны при работе с активами набора плиток, созданными в браузере активов. Эти функции предназначены только для работы с самим активом, а не для работы с плитками или картами плиток, когда они присутствуют в вашей игре (для этого см. соответствующие функции слоя в разделе " Комнаты").

Доступны следующие функции:

  • tileset_get_name
  • tileset_get_texture
  • tileset_get_uvs
