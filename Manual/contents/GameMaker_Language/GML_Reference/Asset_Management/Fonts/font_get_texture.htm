font_get_texture

Эта функция возвращает специальный указатель на страницу текстуры шрифта. Это значение может быть использовано в других функциях рисования, в частности, в общем рисовании при использовании примитивов, а также в функциях шейдера. Более подробную информацию о возвращаемой странице текстуры можно получить с помощью различных функций texture_, найденных здесь.

Синтаксис:

font_get_texture(font);

Аргумент Тип Описание
font Шрифтовой актив The index of the font to use.

Возвращается:

Texture

Пример:

tex = font_get_texture(fnt_Main);

Приведенный выше код получит указатель текстуры для шрифта, индексированного как "fnt_Main".
