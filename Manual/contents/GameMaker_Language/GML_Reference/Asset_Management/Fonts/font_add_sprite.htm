font_add_sprite

С помощью этой функции вы можете использовать "спрайтовую полосу" для создания нового актива шрифта, где каждое подизображение будет отдельным символом или буквой. Индекс - это номер карты UTF8 для первого используемого символа, а диапазоны для использования такие же, как и для обычного шрифта, начиная с "!", но помните, что вы должны включать только то, что вам нужно, и что вы должны использовать функцию ord() для получения правильного значения UTF8 для начальной буквы. Созданный шрифт sprite будет иметь суб-изображения, установленные на отдельные глифы шрифта, где "первый" глиф будет иметь image_index равный 0. Вы можете определить "пробел" как любой символ, например, одну строку такого размера, какого вы хотите, чтобы был пробел, и все пробелы в тексте будут отображаться с такой шириной (но выбранное изображение никогда не будет отображаться), или, если вы не указали sprite для пробела, то вместо него будет использоваться ширина самого широкого символа в шрифте sprite.

Вы также можете задать пропорциональный (true) или непропорциональный (false) интервал для шрифта, при этом пропорциональный шрифт использует фактическую ширину букв (например, буква "i" занимает меньше места, чем буква "w"), а непропорциональный - ширину подрисуночного изображения, как моноширинный шрифт. Наконец, вы можете определить пространство, которое нужно оставить между каждой буквой при написании, и это может быть любое целое значение, при этом 0 означает отсутствие пространства (при пропорциональном шрифте буквы будут "соприкасаться").

Функция возвращает значение индекса, которое следует сохранить в переменной, так как оно будет необходимо во всех дальнейших кодах, которые будут ссылаться на этот шрифт, или возвращает -1, если функция не работает по какой-либо причине.

Стоит также отметить, что в карте символов не может быть пробелов, поэтому если вы, например, используете '.' и '0', но не '/', вам нужно либо включить '/' в любом случае, либо оставить пустой подобраз между двумя используемыми символами.

Сам sprite может быть активом sprite из браузера активов или загруженным из файла, используя сначала sprite_add, чтобы получить индекс загруженного sprite (который затем используется в функции). Важно отметить, что для того, чтобы функции выравнивания шрифтов (такие как draw_set_halign()) работают как положено при использовании шрифта sprite, начало шрифта sprite должно быть установлено в левом верхнем углу. Если вы используете другие значения, то вам придется учитывать смещение начала координат при рисовании текста этим шрифтом.

ПРИМЕЧАНИЕ Когда вы загружаете шрифт в GameMaker, вы должны не забыть удалить его снова (с помощью кнопки font_delete()), если он больше не нужен, иначе существует риск утечки памяти, что приведет к замедлению и, в конечном итоге, к краху вашей игры.

Синтаксис:

font_add_sprite(spr, first, prop, sep);

Аргумент Тип Описание
spr Актив спрайта The sprite to add a font based on.
first Реально The index of the first available character.
prop Булево Set as proportional font or not.
sep Реально The space to leave between each letter.

Возвращается:

Font Asset

Пример:

global.Font = font_add_sprite(spr_HUD_Font, ord("!"), true, 2);

Приведенный выше код создаст новый актив шрифта из актива sprite и сохранит индекс нового шрифта в переменной "global.Font" для последующего использования.
