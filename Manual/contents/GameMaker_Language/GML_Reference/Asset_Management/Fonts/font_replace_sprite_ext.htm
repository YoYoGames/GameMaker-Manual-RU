font_replace_sprite_ext

Эта функция заменит шрифт sprite, который вы создали ранее с помощью функции font_add_sprite_ext(). Она использует "спрайтовую полосу", где каждое подизображение является отдельным символом или буквой, а изображения отображаются на буквы на основе аргумента "string_map" функции. Этот аргумент представляет собой строку, с помощью которой вы можете указать GameMaker, в каком порядке расположены суб-изображения шрифта sprite, и он будет автоматически сопоставлять их при написании текста. Так, например, если у вас есть string-map "AaBbCcDdEeFfGgHh", то ваш шрифт sprite должен иметь суб-изображения, упорядоченные таким образом.

Вы также можете задать пропорциональный (true) или непропорциональный (false) интервал для шрифта, при этом пропорциональный шрифт использует фактическую ширину букв (например, буква "i" занимает меньше места, чем буква "w"), а непропорциональный - ширину подрисуночного изображения, как моноширинный шрифт. Наконец, вы можете определить пространство, которое нужно оставить между каждой буквой при написании, и это может быть любое целое значение, при этом 0 означает отсутствие пространства (буквы будут "соприкасаться", если шрифт пропорциональный). Функция возвращает значение индекса, которое следует сохранить в переменной, так как оно понадобится во всех дальнейших кодах, которые будут ссылаться на этот шрифт.

Если функция завершится неудачно, то она завершится молча, не заменив заданный шрифт sprite.

ПРИМЕЧАНИЕ Эта функция требует, чтобы вы предварительно добавили шрифт sprite с помощью функции font_add_sprite(), так как она заменит его и не может быть использована для шрифтов, включенных в ресурсы игры.

Синтаксис:

font_replace_sprite_ext(font, spr, string_map, prop, sep);

Аргумент Тип Описание
font Шрифтовой актив The font to replace.
spr Актив спрайта The sprite to add a font based on.
string_map Строка String from which sprite sub-image order is taken.
prop Булево Set as proportional font or not.
sep Реально The space to leave between each letter.

Возвращается:

N/A

Пример:

font_replace_sprite_ext(global.Font, spr_CalcFont, "0123456789+-*/=", true, 2);

Приведенный выше код заменит актив шрифта sprite, который индексируется в переменной "global.Font", на изображения sprite, индексируемые в переменной "spr_CalcFont".
