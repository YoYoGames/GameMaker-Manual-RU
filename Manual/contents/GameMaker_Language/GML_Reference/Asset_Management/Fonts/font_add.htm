font_add

Эта функция может быть использована для добавления шрифта в вашу игру из файла шрифта, присутствующего в Included Files. Вы можете определить размер шрифта (в пунктах), а также то, должен ли шрифт быть жирным или курсивным, а также определить диапазон символов для включения. Функция возвращает значение индекса, которое следует сохранить в переменной, так как оно будет необходимо во всем дальнейшем коде, который будет ссылаться на этот шрифт, или вернет -1, если функция не выполнится по какой-либо причине.

Обратите внимание, что файл шрифта должен быть либо *.ttf, либо *.otf, что также может быть полезно для добавления нестандартных шрифтов, таких как азиатские или арабские шрифты.

ВНИМАНИЕ!!! Если вы включите такой шрифт в Included Files, он должен быть лицензирован для распространения вместе с игрой.

Для целевого модуля HTML5 эта функция может использоваться только для добавления веб-шрифтов и будет работать только в том случае, если WebGL не включен - если WebGL включен, необходимо использовать общий ресурс шрифта (т.е.: использование draw_set_font() вместе со шрифтом из Asset Browser) или растровые шрифты (см. font_add_sprite()).

ВАЖНО! При использовании HTML5 курсив и полужирный текст поддерживаются только в том случае, если в активе шрифта или в самом веб-шрифте есть эти символы. Они не будут синтезированы из базового шрифта.

При загрузке шрифта из файла размер шрифта указывается в пикселях, а первое и последнее значения игнорируются, то есть все глифы шрифта будут добавлены. Шрифты, добавленные таким образом, назначаются на свою собственную страницу текстуры, поэтому при использовании этой функции следует соблюдать осторожность, так как она увеличивает количество смен текстуры при рисовании. Также стоит отметить, что шрифты могут выглядеть немного больше при отрисовке, так как глифы могут иметь части, которые рисуются за пределами ограничительной рамки, определенной для шрифта. Также следует помнить, что существует ограничение на 200 глифов, которые могут быть отрисованы в одном кадре для определенного шрифта.

По умолчанию в добавленных шрифтах включено сглаживание. Однако вы можете проверить состояние AA в любое время с помощью функции font_add_get_enable_aa() и вы можете включить/выключить AA с помощью функции font_add_enable_aa().

ПРИМЕЧАНИЕ Когда вы загружаете шрифт в GameMaker, вы должны не забыть удалить его снова (с помощью кнопки font_delete()), если он больше не нужен, иначе существует риск утечки памяти, что приведет к замедлению и, в конечном итоге, к краху вашей игры.

Синтаксис:

font_add(name, size, bold, italic, first, last);

Аргумент Тип Описание
name Строка The path of the font file in the Included Files (*.ttf или *.otf).
size Реально The size of the font - points for Web Fonts, pixels for file fonts.
bold Булево Whether the font is bold (true) или нет (false).
italic Булево Whether the font is italic (true) или нет (false).
first Реально The first character to include (if you're unsure, go for 32).
last Реально The last character to include (if you're unsure, go for 128).

Возвращается:

Font Asset

Пример:

new_font = font_add("gamefont.ttf", 24, true, true, 32, 128);

Это загрузит файл "gamefont.ttf" , находящийся в папке Included Files игры, с размером 24pt. Шрифт будет жирным и курсивным, а диапазон шрифта будет включать заглавные и строчные буквы, цифры и всю обычную пунктуацию. После загрузки шрифта его ID будет сохранен в переменной new_font.
