font_get_last

При определении шрифта в GameMaker вы можете задать диапазон символов, которые будут включены в него. Это связано с тем, что сам шрифт не включается в игру (по юридическим причинам), но его изображение шрифта включено в страницу текстуры, и именно его будет использовать ваша игра (как и любой другой графический ресурс). Это означает, что вы захотите свести количество символов, которые вы используете, к минимуму и указать только тот диапазон символов, который чтобы максимально оптимизировать память текстуры. Эта функция может быть использована для поиска последнего символа (в виде значения UTF8), который был использован при добавлении актива шрифта в игру.

Синтаксис:

font_get_last(ind);

АргументТип Описание
ind Index of the font to check.

Возвращается:

Пример:

num = font_get_last(fnt_Main);

Приведенный выше код сохранит значение ASCII последней буквы диапазона шрифта для шрифта, индексированного в "fnt_Main".
