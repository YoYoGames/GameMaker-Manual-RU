font_get_fontname

С помощью этой функции можно получить фактическое системное имя заданного актива шрифта. Эта функция возвращает строку, а не индекс, и возвращаемое имя будет зависеть от используемого шрифта. Например, у вас может быть актив шрифта под названием "fnt_Main" в браузере активов, а сам шрифт может быть системным шрифтом Arial Windows. В этом случае функция вернет "Arial", так как это системное имя шрифта. Если вам нужно имя, которое отображается в браузере активов, используйте функцию font_get_name().

Описанное выше поведение применимо только к шрифтовым активам, добавленным с помощью IDE. При использовании для шрифта, загруженного во время выполнения (с помощью font_add()), функция вернет полный путь к файлу шрифта вместо его имени.

Синтаксис:

font_get_fontname(ind);

АргументТип Описание
ind Index of the font to check.

Возвращается:

String

Пример:

fnt_Name = font_get_fontname(font0);

Приведенный выше код получит системное имя ресурса шрифта и сохранит его в виде строки в переменной "fnt_Name".
