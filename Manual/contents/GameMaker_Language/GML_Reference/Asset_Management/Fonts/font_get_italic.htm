font_get_italic

С помощью этой функции можно проверить любой актив шрифта на наличие или отсутствие в нем флага курсива. Если да, то функция вернет true, в противном случае - false.

Синтаксис:

font_get_italic(ind);

АргументТип Описание
ind Index of the font to check.

Возвращается:

Пример:

if font_get_italic(fnt_Main)
{
    draw_set_font(fnt_Main);
}

Это позволит установить активный шрифт рисунка на fnt_Main, если в свойствах шрифта он установлен как курсив.
