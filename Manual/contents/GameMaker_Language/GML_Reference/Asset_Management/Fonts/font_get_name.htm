font_get_name

С помощью этой функции можно получить имя (в виде строки), которое было присвоено шрифту при его добавлении в качестве актива в GameMaker Asset Browser. Обратите внимание, что это только и не может быть использовано для прямого обращения к шрифту - для этого вам понадобится индекс шрифта. Однако вы можете использовать эту строку для получения индекса шрифта, используя возвращаемую строку вместе с функцией функция asset_get_index().

Синтаксис:

font_get_name(ind);

АргументТип Описание
ind Index of the font to check.

Возвращается:

Пример:

fnt_Name = font_get_name(font0);

Приведенный выше код получит имя ресурса шрифта, как оно отображается в Asset Browser, и сохранит его в виде строки в переменной "fnt_Name".
