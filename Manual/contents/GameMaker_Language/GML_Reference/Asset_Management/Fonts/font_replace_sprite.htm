font_replace_sprite

С помощью этой функции вы можете использовать "спрайтовую полосу" для замены ранее созданного актива шрифта sprite, где каждое подизображение будет отдельным символом или буквой. Индекс - это номер карты UTF8 для первого используемого символа, а диапазоны для использования такие же, как и для обычного шрифта, начиная с "!", но помните, что вы должны включать только то, что вам нужно, и что вы должны использовать функцию ord() для получения правильного значения UTF8 для начальной буквы.

Вы также можете задать пропорциональный (true) или непропорциональный (false) интервал для шрифта, при этом пропорциональный шрифт использует фактическую ширину букв (например, буква "i" занимает меньше места, чем буква "w"), а непропорциональный - ширину подрисуночного изображения, как моноширинный шрифт. Наконец, вы можете определить пространство, которое нужно оставить между каждой буквой при написании, и это может быть любое целое значение, при этом 0 означает отсутствие пространства (при пропорциональном шрифте буквы будут "соприкасаться").

Если функция завершится неудачно, то она завершится молча, не заменив заданный шрифт sprite.

Следует также отметить, что в карте символов не может быть пробелов, поэтому, если вы, например, используете '.' и '0', но не '/', вам необходимо либо включить '/' в любом случае, либо оставить пустое подизображение между двумя используемыми символами. На рисунке ниже показана типичная схема шрифта sprite:

Синтаксис:

font_replace_sprite(ind, spr, first, prop, sep);

Аргумент Тип Описание
ind Шрифтовой актив The index of the font that shall be replaced (ie all references to this resource past this function will refer to the new font, not the original).
spr Актив спрайта The sprite to add a font based on.
first Реально The index of the first available character.
prop Булево Whether white space around each letter should be removed (if false, the created font will be monospaced, otherwise it'll be like most regular fonts).
sep Реально The space between each letter, 0 meaning the right of one letter just touches the left of the next one.

Возвращается:

N/A

Пример:

font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2);

Приведенный выше код заменит актив шрифта, указанный в глобальной переменной "Font", на новый актив шрифта из внешнего файла изображения.
