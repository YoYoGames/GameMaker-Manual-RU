font_get_info

Эта функция используется для получения информации о заданном шрифте. Вы задаете индекс шрифта (это может быть актив, добавленный через Asset Browser, или шрифт, добавленный во время выполнения), и функция возвращает структуру со следующими переменными:

Информационная структура шрифта
Имя переменной Тип данных Описание
ascenderOffset Реально Максимальное смещение от базовой линии к вершине шрифта (в пикселях).
size Реально Приблизительный размер шрифта (в пикселях)
spriteIndex Актив спрайта Индекс sprite для шрифта, если он был создан из спрайта, иначе -1
texture Указатель -1, если шрифт был создан из sprite, иначе - идентификатор текстуры шрифта
name Строка Название шрифта
bold Булево true если шрифт жирный, в противном случае false (см. ПРИМЕЧАНИЕ ниже)
italic Булево true если шрифт курсивный, в противном случае false (см. ПРИМЕЧАНИЕ ниже)
glyphs Структура глифа шрифта Структура, содержащая информацию о каждом глифе в шрифте (более подробная информация приведена ниже)

ПРИМЕЧАНИЕ Переменные bold и italic отражают только настройки пользователя для шрифта, поэтому они могут быть неточными для шрифтов, которые по умолчанию отображаются жирным или курсивным шрифтом.

Если указанный шрифт не существует, функция вернет undefined.

Также обратите внимание, что изменение значений любой из этих переменных или любых переменных, содержащихся в структуре glyphs, не изменит отображение шрифта, поэтому эту информацию следует считать доступной только для чтения.

Структура глифов

Переменная glyphs в возвращаемой структуре будет самостоятельной структурой, содержащей информацию о каждом глифе, включенном в шрифт. Каждая переменная в этой структуре будет именем символа глифа, который будет содержать структуру, содержащую информацию для этого глифа. Вы можете представить эти вложенные структуры в следующем формате (начиная с главной структуры):

{
    glyphs:
    {
        A: {},
        B: {},
        C: {},
        // ...other glyphs
    },
    // ...other font info
}

Вы можете использовать имя символа глифа для получения информации о нем из структуры, как, например, в следующем коде:

var _font_info = font_get_info(Font1);

var _info_A = _font_info.glyphs[$ "A"]; // $ is a struct accessor

show_debug_message(_info_A);

Приведенный выше код получает глиф-структуру для символа "A" и печатает ее в выходной журнал. Вы также можете заменить строку ("A") переменной, содержащей строку, что позволит вам получать информацию о любом глифе через эту переменную во время выполнения.

Структура, возвращаемая для глифа, будет содержать следующие переменные:

ПРИМЕЧАНИЕ В HTML5 структура glyphs будет пустой для любых загруженных файловых шрифтов (таких как *.ttf или *.otf), поскольку среда выполнения не будет иметь информации об этих глифах.

Структура глифа шрифта
Имя переменной Тип данных Описание
char настоящий Если шрифт был создан из sprite, это будет индекс изображения глифа из этого sprite, в противном случае это будет номер символа Unicode.

Примечание: Все переменные ниже этой не будут присутствовать в структуре, если шрифт был создан из спрайта.
x настоящий X позиция глифа на странице текстуры (в текселях).
y настоящий Положение глифа по оси Y на странице текстуры (в текселях)
w настоящий Ширина глифа на странице текстуры (в текселях).
h настоящий Высота глифа на странице текстуры (в текселях)
shift настоящий Количество пикселей для сдвига вправо при переходе к следующему символу (может быть отрицательным для сдвига влево).
offset настоящий Количество пикселей для горизонтального смещения рендеринга этого глифа (может быть положительным или отрицательным).
yoffset Real The number of pixels to vertically offset the rendering of this glyph without affecting the shift position (can be positive or negative)
kerning массив Массив целых чисел, содержащий информацию о кернинге в парах (или группах по 2). Первое целое число в паре - это значение Unicode для символа, а второе целое число - сумма, которую нужно добавить к значению сдвига этого символа (может быть положительным или отрицательным), если он предшествует символу данного глифа.

ПРИМЕЧАНИЕ Для файловых шрифтов, загружаемых во время выполнения, значения x и y для глифа могут быть равны -1, если он еще не был отрисован, поскольку у него не будет позиции на странице текстуры, используемой для кэширования.

Синтаксис:

font_get_info(font);

Аргумент Тип Описание
font Шрифтовой актив The font for which information should be retrieved.

Возвращается:

Font Info Struct (or undefined)

Пример:

var _info = font_get_info(my_font);

if (_info != undefined && _info.bold)
{
    image_xscale *= 2;
    image_yscale *= 2;
}

Этот код извлекает информацию о шрифте, хранящемся в переменной my_font. Если возвращаемое значение не undefined и заданный шрифт является полужирным, масштаб экземпляра удваивается.
