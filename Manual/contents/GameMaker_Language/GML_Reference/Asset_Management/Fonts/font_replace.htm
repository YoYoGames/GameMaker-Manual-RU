font_replace

Эта функция может быть использована для замены шрифта в вашей игре из тех шрифтов, которые установлены на системе, на которой она запущена. Обратите внимание, что заменяемый шрифт не может быть шрифтом из браузера активов, а должен быть шрифтом, который был добавлен ранее с помощью функции font_add().

При замене шрифта таким образом можно задать размер шрифта (в пунктах), а также то, должен ли шрифт быть полужирным или курсивным, а также определить диапазон символов, которые необходимо включить. Если функция не сработает, то она завершится молча, и заданный шрифт не будет заменен.

Для целевого модуля HTML5 эта функция может использоваться только для замены веб-шрифтов и будет работать только в том случае, если WebGL не включен - если WebGL включен, необходимо использовать общий ресурс шрифта (т.е. использовать draw_set_font() вместе со шрифтом из Asset Browser) или растровые шрифты (см. font_add_sprite()).

ВАЖНО При использовании HTML5 курсив и полужирный текст поддерживаются только в том случае, если в шрифтовой актив или сам веб-шрифт включены эти символы. Они не будут синтезированы из базового шрифта.

На всех других (не веб-) объектах вы можете использовать эту функцию для замены шрифта из файла. Файл должен быть включен в пакет игры с помощью функции Included Files программы GameMaker и должен представлять собой файл шрифта формата *.ttf или *.otf, что удобно для добавления нестандартных шрифтов, например, азиатских или арабских.

СОВЕТ Если вы включаете такой файл шрифта в игру, убедитесь, что он должным образом лицензирован для распространения вместе с игрой.

При загрузке шрифта из файла таким способом размер шрифта указывается в пикселях, а первое и последнее значения игнорируются, то есть все глифы шрифта будут добавлены. Шрифты, добавленные таким образом, назначаются на свою собственную страницу текстуры, поэтому при использовании этой функции следует соблюдать осторожность, так как это увеличит количество смен текстуры при рисовании. Также стоит отметить, что шрифты могут выглядеть немного больше при отрисовке, так как глифы могут иметь части, которые рисуются за пределами ограничительной рамки, определенной для шрифта. Также следует помнить, что существует ограничение на 200 глифов, которые могут быть отрисованы в одном кадре для определенного шрифта.

ПРИМЕЧАНИЕ Когда вы загружаете шрифт в GameMaker, вы должны не забыть удалить его снова (с помощью кнопки font_delete()), если он больше не нужен, иначе есть риск утечки памяти, что приведет к замедлению и, в конечном итоге, к краху вашей игры.

Синтаксис:

font_replace(ind, name, size, bold, italic, first, last);

Аргумент Тип Описание
ind Шрифтовой актив The index of the (previously added) font to be replaced.
name Строка The name of the font to be used (eg 'Arial'), or the file path if the font is an included *.ttf или *.otf файл.
size Реально The size of the font - points for Web Fonts, pixels for file fonts.
bold Булево Whether the font is bold (true) or not (false).
italics Булево Whether the font is italic (true) or not (false).
first Реально The first character to include (if you're unsure, go for 32).
last Реально The last character to include (if you're unsure, go for 128).

Возвращается:

N/A

Пример:

font_replace(myfont, "Arial", 24, true, true, 32, 128);

Это заменит шрифт, указанный в переменной "myfont", на новый шрифт размером 24pt, использующий шрифт "Arial", полужирный и курсивный. Шрифт включает заглавные и строчные буквы, цифры и всю обычную пунктуацию.
