font_add_enable_aa

Эту функцию можно использовать для включения или отключения сглаживания (AA) для шрифтов, добавленных с помощью функции font_add(). Эта функция должна быть вызвана до добавления шрифтов, и установка ее в true включит AA, а установка в false отключит его. По умолчанию AA включен. Обратите внимание, что это не влияет на шрифты, которые были добавлены до вызова этой функции, и функция должна вызываться только один раз при добавлении шрифта, а не на каждом отрисовке/шаге, когда шрифт используется.

Синтаксис:

font_add_enable_aa(enable);

АргументТип Описание
enable Whether to enable (true) или отключить (false) AA для добавленных шрифтов.

Возвращается:

Пример:

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

Приведенный выше код проверяет состояние сглаживания для добавленных шрифтов, и если оно не включено, то мы включаем его.
