font_set_cache_size

При выполнении смешивания и раскрашивания изображений HTML5 не может делать это динамически так же, как это может делать исполняемый файл. Поэтому GameMaker должен временно сохранять копию смешивания изображений и загружать ее при необходимости. и загружать их, когда это необходимо. Эта функция устанавливает, сколько смешанных копий данного шрифта можно кэшировать, прежде чем старые копии будут перезаписаны. Значение по умолчанию равно 4.

ПРИМЕЧАНИЕ для шрифтов sprite следует использовать эквивалентную функцию для sprites, sprite_set_cache_size().

Синтаксис:

font_set_cache_size(ind, max);

АргументТип Описание
ind The index of the font to change the cache size of.
max The maximum number of cached copies of the font that can be stored.

Возвращается:

Пример:

font_set_cache_size(fnt_MainMenu, 2);

Это установит кэш шрифта, индексированного в переменной "fnt_MainMenu", в 2 копии.
