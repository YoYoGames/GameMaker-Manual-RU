tilemap_get

С помощью этой функции вы можете получить данные о плитке из ячейки элемента карты плитки. Вы указываете ID элемента карты плитки (который вы получаете при создании элемента карты плитки с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), а также позицию x и y ячейки, и функция вернет "сгусток" данных плитки. Эти данные по сути являются битовой маской, содержащей индекс плитки, булевы flip/rotate/mirror и любые примененные данные маски (см. tilemap_set_mask() подробности), а полученное значение данных можно использовать в функциях tile_* для изменения свойств плитки. Обратите внимание, что мы используем позиции ячеек x/y, основанные на размере сетки карты плитки, а не на положении в комнате, поэтому "cell_x" - это значение от 0 до ширины карты плитки, а "cell_y" - это значение от 0 до высоты карты плитки. Если вам нужно получить данные для плитки в определенном положении в комнате, то вам нужно использовать функцию tilemap_get_at_pixel().

ВАЖНО! Если плитки в карте плиток были неизменны (т.е. они не повернуты, не перевернуты и т.д.), то возвращаемое значение данных набора плиток будет точно равно индексу плитки в наборе плиток. Таким образом, вы можете создавать "карты столкновений" плиток, используя одну плитку с индексом 1 в наборе плиток - например - затем использовать эту функцию для проверки 1 или 0 (пустая плитка) для расчета столкновений.

Синтаксис:

tilemap_get(tilemap_element_id, cell_x, cell_y);

Аргумент Тип Описание
tilemap_element_id Идентификатор элемента карты плитки The unique ID value of the tile map element to get the tile-data of
x_cell Реально The cell along the x-axis to get the tile data from
y_cell Реально The cell along the y-axis to get the tile data from

Возвращается:

Tile Data (-1 if there is an error)

Пример:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

Приведенный выше код получает идентификатор карты плиток на слое "Tiles_Sky", а затем использует его для получения данных плитки в ячейке (0, 0). Эти данные плитки затем переворачиваются, прежде чем снова использоваться для установки плитки на карте плиток.
