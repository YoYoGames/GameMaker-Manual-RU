tilemap_clear

С помощью этой функции вы можете очистить/установить все плитки на заданной карте плиток. Вы указываете ID элемента карты плиток (который вы получаете при создании элемента карты плиток с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), а затем предоставляете данные плитки, которую вы хотите очистить. Значение по умолчанию 0 очистит все плитки из слоя (по сути, делая все плитки "пустыми"), в то время как вы можете использовать специальные функции tile_* для создания собственных данных плиток для очистки карты плиток.

Синтаксис:

tilemap_clear(tilemap_element_id, tiledata)

Аргумент Тип Описание
tilemap_element_id Идентификатор элемента карты плитки The unique ID value of the tile map element to change
tiledata Данные плитки (реальные) The tile data to use to clear the layer

Возвращается:

N/A

Пример:

var layer_id = layer_get_id("Forest");
var tile_id = layer_tilemap_get_id(layer_id);
tilemap_clear(tile_id, 0);

Приведенный выше код получает значение ID карты плиток, созданной в редакторе комнат, а затем очищает ее, используя "пустые" плитки.
