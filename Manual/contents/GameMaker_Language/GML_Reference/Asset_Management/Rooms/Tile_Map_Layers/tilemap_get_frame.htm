tilemap_get_frame

Поскольку плитки могут быть анимированными, иногда бывает полезно знать, какой кадр в данный момент рисуется, и реагировать соответствующим образом, поэтому с помощью этой функции можно получить индекс текущего кадра для данной карты плиток. Вы задаете ID элемента карты плиток (который вы получаете при создании элемента карты плиток с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), и функция вернет индекс кадра.

Синтаксис:

tilemap_get_frame(tilemap_element_id)

Аргумент Тип Описание
tilemap_element_id Идентификатор элемента карты плитки The unique ID value of the tile map element to get the frame index of

Возвращается:

Real (between 0 (inclusive) and the maximum number of frames of animation (exclusive))

Пример:

var lay_id = layer_get_id("Tiles_Traps");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_frame(map_id) >= 2 && tilemap_get_frame(map_id) < 4
{
    global.activate = true;
}
else
{
    global.activate = false;
}

Приведенный выше код проверяет текущий кадр анимации для заданной карты тайлов на заданном слое и устанавливает глобальную переменную на основе возвращаемого значения.
