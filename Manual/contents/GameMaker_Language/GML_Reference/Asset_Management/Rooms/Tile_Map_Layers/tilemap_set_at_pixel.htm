tilemap_set_at_pixel

Эта функция может быть использована для установки ячейки в элементе карты плитки на слое на новую плитку, используя фактическое положение плитки для изменения в помещении. Вы задаете ID элемента карты плитки (который вы получаете при создании элемента карты плитки с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), а затем данные плитки, которые нужно установить, а также позицию в помещении. Вы можете получить данные плитки с помощью функции tilemap_get_at_pixel() и затем с помощью функций tile_ изменить их, прежде чем снова задать их с помощью этой функции. Функция вернет true, если плитка была успешно установлена, и false, если возникла проблема и она не была установлена.

Синтаксис:

tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y)

Аргумент Тип Описание
tilemap_element_id Идентификатор элемента карты плитки The unique ID value of the tile map element to change
tiledata Данные плитки (реальные) The tile set data to set
xcell Реально The x position (in the room)
ycell Реально The y position (in the room)

Возвращается:

Boolean

Пример:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 4, 4);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 4, 4);

Приведенный выше код получает идентификатор карты плиток на слое "Tiles_Sky", а затем использует его для получения данных от плитки в позиции (4, 4). Эти данные плитки затем переворачиваются перед использованием для повторной установки плитки на карте плиток.
