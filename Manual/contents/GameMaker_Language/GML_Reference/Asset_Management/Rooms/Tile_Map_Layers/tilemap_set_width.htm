tilemap_set_width

Эта функция может использоваться для изменения размера элемента карты плиток. Вы передаете идентификатор элемента карты плитки (который вы получаете при создании элемента карты плитки с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), и новую ширину карты плитки в ячейках плитки.

Синтаксис:

tilemap_set_width(tilemap_element_id, width)

Аргумент Тип Описание
tilemap_element_id Идентификатор элемента карты плитки The unique ID value of the tile map element to set the width of
width Реально The width value (in "cells")

Возвращается:

N/A

Пример:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_width(map_id) != room_width div 16)
{
    tilemap_set_width(map_id, room_width div 16);
}

Приведенный выше код проверяет ширину конкретной карты плиток, и если она не соответствует размеру, то ширина устанавливается.
