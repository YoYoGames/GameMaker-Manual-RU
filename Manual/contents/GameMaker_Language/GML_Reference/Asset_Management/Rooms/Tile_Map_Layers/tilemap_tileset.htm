набор плиток

С помощью этой функции можно изменить актив набора плиток, назначенный данному элементу карты плиток на слое. Вы указываете идентификатор элемента карты плитки (который вы получаете при создании элемента карты плитки с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), а затем укажите индекс набора плиток, и карте плиток будет присвоен новый актив sprite.

Синтаксис:

tilemap_tileset(tilemap_element_id, tileset_index)

Аргумент Тип Описание
tilemap_element_id Идентификатор элемента карты плитки The unique ID value of the tile map element to change
tileset_index Активы набора плиток The new tile set index to use

Возвращается:

N/A

Пример:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_tileset(tile_id) != ts_Nighttime)
{
    tilemap_tileset(tile_id, ts_Nighttime);
}

Приведенный выше код проверяет текущий набор плиток, назначенный карте плиток на слое "Tiles_trees", и если он не "ts_Nighttime", то этот набор плиток назначается карте плиток.
