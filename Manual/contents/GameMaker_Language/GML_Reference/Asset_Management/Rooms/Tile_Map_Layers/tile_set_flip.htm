tile_set_flip

Эта функция может быть использована для установки заданного набора данных плитки, чтобы перевернуть плитку или нет. Вы передаете данные плитки, которые можно получить с помощью функции tilemap_get(), а затем устанавливаете аргумент flip либо в true, если вы хотите перевернуть плитку, либо в false, если вы хотите, чтобы плитка находилась в своем стандартном, неперевернутом состоянии. Функция вернет модифицированный набор данных плитки, который затем можно применить с помощью функции tilemap_set() функцию.

Синтаксис:

tile_set_flip(tiledata, flip)

АргументТип Описание
tiledata the tile-data to set
flip Set to true перевернуть и false оставить как есть

Возвращается:

Пример:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_flip(data);
data = tile_set_flip(data, bool);
tilemap_set(map_id, data, mx, my);

Приведенный выше код получает ID карты плитки из заданного слоя, а затем получает позицию x и y ячейки для плитки под мышью. Эта позиция затем используется для получения данных плитки, которые переворачиваются и затем снова используются для установки плитки.
