tilemap_get_mask

Эта функция может использоваться для получения значения битовой маски для заданной карты плиток. Вы передаете идентификатор элемента карты плиток (который вы получаете при создании элемента карты плиток с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), и функция вернет текущее значение маски или -1, если произошла ошибка, или 0, если маска не указана. Для получения дополнительной информации о битовых масках карты плиток см. функцию tilemap_set_mask().

Синтаксис:

tilemap_get_mask(tilemap_element_id);

Аргумент Тип Описание
tilemap_element_id Идентификатор элемента карты плитки The unique ID value of the tile map element to get the mask of

Возвращается:

Real (0 for no mask, -1 for an error)

Пример:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mask = tilemap_get_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if (mask != new_mask)
{
    tilemap_set_mask(map_id, new_mask);
}

Приведенный выше код получает идентификатор карты плитки из данного слоя, а затем проверяет значение маски, связанное с ним. Если оно не совпадает со значением, определенным в переменной "new_mask", то оно устанавливается на это значение.
