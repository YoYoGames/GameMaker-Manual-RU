layer_tilemap_destroy

Эта функция уничтожит заданный элемент карты плиток. Вы передаете ID карты плиток (который вы получаете при создании карты плиток с помощью функции layer_tilemap_create() или когда вы используете идентификатор слоя вместе с layer_get_tilemap_id()), и эта функция удалит его. Обратите внимание, что при этом удаляется не слой, а только карта плиток, и если карта плиток была добавлена в редакторе комнаты, то при следующем выходе из комнаты и последующем возвращении карта плиток будет создана заново. Однако если комната постоянна, карта плиток будет удалена, пока постоянство комнаты не будет снова отключено.

Синтаксис:

layer_tilemap_destroy(tilemap_element_id)

Аргумент Тип Описание
tilemap_element_id Идентификатор элемента карты плитки The unique ID value of the tile map to be destroyed

Возвращается:

N/A

Пример:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if (layer_tilemap_exists(lay_id, tile_id))
{
    layer_tilemap_destroy(tile_id);
}

Приведенный выше код проверяет слой "Tiles_trees" на существование заданного элемента карты плиток, и если он существует, то уничтожает его (но не слой).
