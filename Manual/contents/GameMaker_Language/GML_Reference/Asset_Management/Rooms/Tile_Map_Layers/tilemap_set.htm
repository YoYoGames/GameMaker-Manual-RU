tilemap_set

Эта функция может использоваться для установки любой ячейки (квадрата сетки) в элементе карты плиток на слое на новую плитку. Вы указываете идентификатор элемента карты плитки (который вы получаете при создании элемента карты плитки с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), а затем данные плитки, которые нужно установить, а также позицию в карте плитки. Вы можете получить данные плитки с помощью функции tilemap_get() и затем использовать функции tile_ для их изменения перед установкой ячейки с помощью этой функции. Функция вернет true, если плитка была успешно установлена, и false, если возникла проблема и она не была установлена.

Синтаксис:

tilemap_set(tilemap_element_id, tiledata, xcell, ycell)

Аргумент Тип Описание
tilemap_element_id Идентификатор элемента карты плитки The unique ID value of the tile map element to change
tiledata Данные плитки (реальные) The tile data to set
xcell Реально The cell (grid) position to set along the x-axis
ycell Реально The cell (grid) position to set along the y-axis

Возвращается:

Boolean

Пример:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

Приведенный выше код получает идентификатор карты плиток на слое "Tiles_Sky", а затем использует его для получения данных плитки в ячейке (0, 0). Эти данные плитки затем переворачиваются, прежде чем снова использоваться для установки плитки на карте плиток.
