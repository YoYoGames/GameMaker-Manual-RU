Плитка Слои карты

Редактор комнат GameMaker позволяет добавлять плитки в любую комнату с помощью слоев. Как следует из названия, слой карты плиток - это просто набор плиток, которые находятся на одной и той же глубине в помещении, и путем наложения слоев вы можете сделать так, чтобы одни элементы рисовались поверх или под другими. Вы также можете управлять некоторыми аспектами слоев из кода, добавляя или удаляя элементы, или изменяя определенные свойства слоя или того, что он содержит, во время выполнения. Обратите внимание, что в редакторе комнат вы ограничены одной картой плиток для каждого слоя (кнопка Слой плитки), но при работе с ними в коде вы можете иметь несколько карт плиток, назначенных одному слою, причем каждая карта имеет свой уникальный идентификатор и свойства, например:

  • Набор плиток - это ресурс, который используется для создания слоев плиток и карт плиток.
  • Слой плитки - это то, что вы создаете в редакторе помещений для добавления наборов плиток в
  • Карта плиток - это то, что вы называете коллекцией плиток, которые добавляются в слой, либо в редакторе помещений, либо с помощью кода, как единый элемент на этом слое.

Помимо перечисленных выше функций для установки карт плиток, у вас также есть ряд функций, которые можно использовать для работы с данными отдельных плиток. Сгусток" данных плитки состоит из индекса плитки плюс несколько дополнительных битов, показывающих, была ли плитка повернута, перевернута или зеркально отображена. У вас также есть возможность добавить собственную битовую маску к данным плитки, хотя это только для опытных пользователей, поскольку вы будете маскировать биты, зарезервированные для индекса плитки - по сути, есть 19 битов, зарезервированных для индекса плитки, но если вы используете только 8 из них, то у вас остается 11 битов, которые можно использовать для создания пользовательской маски. Эта маска может быть использована и проверена в коде для создания пользовательских столкновений тайлов или чего-либо еще. Однако обычно вам не нужно создавать свои собственные блобы данных тайлов, а вместо этого вы используете функцию tilemap_get() для получения данных тайла, затем вы будете манипулировать ими с помощью функций ниже, а затем снова установите их с помощью функции tilemap_set().
