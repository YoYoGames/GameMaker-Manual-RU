tilemap_x

Эта функция управляет положением по оси x комнаты элемента карты плитки актива на слое. Вы задаете идентификатор элемента карты плитки (который вы получаете при создании элемента карты плитки с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), а затем задаете значение x, которое необходимо использовать (на основе координат помещения).

Синтаксис:

tilemap_x(tilemap_element_id, x);

Аргумент Тип Описание
tilemap_element_id Идентификатор элемента карты плитки The unique ID value of the tile map element to change
x Реально The x position for the tile map

Возвращается:

N/A

Пример:

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_x(map_id, irandom(room_width));

Приведенный выше код получает значение ID актива карты плитки, назначенного слою "Asset_sky", а затем устанавливает его x-позицию в случайное значение между 0 и шириной комнаты.
