layer_tilemap_create

С помощью этой функции вы можете назначить ресурс набора плиток слою, который будет использоваться в качестве карты плиток в вашем проекте. Вы предоставляете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), затем начальную позицию (x, y) для добавления карты плиток в комнату, ресурс набора плиток, который будет использоваться, а затем ширину и высоту карты плиток в ячейках (т.е.: ширина 20 и высота 10 создадут карту плиток с 200 ячейками, которые будут иметь ширину 20 ячеек и высоту 10 ячеек), при этом размер самих ячеек определяется выбранным набором плиток. Стоит отметить, что вы не можете размещать плитки в отрицательных позициях на карте плиток, поэтому все плитки должны быть размещены в промежутках между ячейками от 0 до ширины - 1, от 0 до высоты - 1.

Синтаксис:

layer_tilemap_create(layer_id, x, y, tileset, width, height)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the layer to target (or the layer name as a string)
x Реально The x position of the tile map in the room
y Реально The y position of the tile map in the room
tileset Активы набора плиток The Tile Set asset to be used
width Реально The width tile map (in cells)
height Реально The height of the tile map (in cells)

Возвращается:

Tile Map Element ID

Пример:

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

Приведенный выше код создает новый слой, а затем добавляет к нему новый элемент карты плиток, устанавливая позицию карты плиток на (0,0), а также набор используемых плиток, ширину и высоту карты плиток.
