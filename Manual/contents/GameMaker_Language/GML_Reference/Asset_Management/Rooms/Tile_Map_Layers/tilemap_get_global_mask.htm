tilemap_get_global_mask

Эта функция может быть использована для получения значения битовой маски для всех карт плиток, возвращая текущее значение маски или -1, если произошла ошибка, или 0, если маска не указана. Для получения дополнительной информации о битовых масках глобальных тайловых карт см. функцию tilemap_set_global_mask().

Синтаксис:

tilemap_get_global_mask();

Возвращается:

(0 for no mask, -1 for an error, mask value otherwise)

Пример:

var mask = tilemap_get_global_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
    tilemap_set_global_mask(new_mask);
}

Приведенный выше код получает глобальное значение маски, связанное со всеми картами плиток. Если оно не совпадает со значением, определенным в переменной "new_mask", то оно устанавливается в это значение.
