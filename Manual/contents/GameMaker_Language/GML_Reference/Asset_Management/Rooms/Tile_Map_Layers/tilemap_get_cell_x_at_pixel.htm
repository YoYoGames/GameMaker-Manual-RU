tilemap_get_cell_x_at_pixel

С помощью этой функции вы можете получить положение оси x отдельной ячейки карты плитки, задав относительное положение оси x в помещении. Вы указываете ID элемента карты плитки (который вы получаете при создании элемента карты плитки с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), а также положение x и y в помещении, и функция вернет положение x ячейки в карте плитки для этой точки. Обратите внимание, что если значение находится за пределами области карты плиток, и ни одна ячейка не доступна, функция вернет -1.

Синтаксис:

tilemap_get_cell_x_at_pixel(tilemap_element_id, x, y);

АргументТип Описание
tilemap_element_id The unique ID value of the tile map element to get the cell x position of
x The x position within the room to use for getting the cell
y The y position within the room to use for getting the cell

Возвращается:

(x-axis cell position or -1 if there is an error)

Пример:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var _y = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
tiledata = tilemap_get(map_id, _x, _y);

Приведенный выше код использует полученный идентификатор карты плитки, чтобы получить положение ячейки из набора координат комнаты, а затем сохранить данные для любой плитки, найденной там, в переменной экземпляра.
