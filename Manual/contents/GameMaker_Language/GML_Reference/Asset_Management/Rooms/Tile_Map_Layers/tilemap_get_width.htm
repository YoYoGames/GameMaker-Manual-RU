tilemap_get_width

С помощью этой функции вы можете получить ширину (в ячейках) элемента карты плитки. Вы передаете идентификатор элемента карты плитки (который вы получаете при создании элемента карты плитки с помощью функции layer_tilemap_create() или когда вы используете функцию layer_tilemap_get_id()), и функция вернет ширину (количество ячеек в ширину).

Синтаксис:

tilemap_get_width(tilemap_element_id);

АргументТип Описание
tilemap_element_id The unique ID value of the tile map element to get the width of

Возвращается:

Пример:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _w = tilemap_get_width(map_id);
var _h = tilemap_get_height(map_id);
total_tiles = (_w * _h);

Приведенный выше код использует полученный идентификатор карты плиток для получения ширины и высоты карты плиток, а затем использует эти значения для установки переменной экземпляра.
