tile_set_mirror

Эта функция может быть использована для установки заданного набора tile-данных для зеркального отображения плитки или нет. Вы передаете данные плитки, которые можно получить с помощью функции tilemap_get(), а затем устанавливаете аргумент mirror либо в true, если хотите, чтобы плитка была зеркальной, либо в false, если хотите, чтобы плитка была в стандартном, незеркальном состоянии. Функция вернет модифицированный набор данных плитки, который затем можно применить с помощью функции tilemap_set() функцию.

Синтаксис:

tile_set_mirror(tiledata, mirror)

АргументТип Описание
tiledata the tile-data to set
mirror Set to true отзеркалить и false оставить как есть

Возвращается:

Tile Data (Real)

Пример:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_mirror(data);
data = tile_set_mirror(data, bool);
tilemap_set(map_id, data, mx, my);

Приведенный выше код получает ID карты плитки из заданного слоя, а затем получает позицию x и y ячейки для плитки под мышью. Эта позиция затем используется для получения данных плитки, которые зеркально отображаются и затем используются для повторной установки плитки.
