room_set_persistent

С помощью этой функции вы можете изменить (или установить) постоянство любой комнаты в вашей игре, кроме текущей. Комната с постоянством, отмеченная как true, будет сохранять состояние всех экземпляров внутри этой комнаты, если игрок покинет ее, а затем вернется, тогда как если постоянство отмечено как false, она будет каждый раз сбрасываться в исходное состояние. Следует отметить, что постоянная комната использует значительно больше памяти, чем обычная комната, поэтому не рекомендуется иметь их слишком много в игре.

ПРИМЕЧАНИЕ: Эта функция НЕ работает для отключения постоянства, если комната уже была посещена! Постоянная комната, после посещения, хранится в памяти, и чтобы отключить постоянство, вы должны зайти в эту комнату и установить переменную room_persistent false , а затем снова выйти из комнаты.

Синтаксис:

room_set_persistent(index, val);

Аргумент Тип Описание
index Комнатный актив The index of the room to set the persistence of.
val Булево Whether the room should be persistent (true) или нет (false).

Возвращается:

N/A

Пример:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, true);

Это создаст новую комнату и сохранит ее индекс в переменной "global.myroom". Затем она установит ее ширину 640 пикселей, высоту 480 пикселей, надпись 'Game Room' и постоянство 'true'.
