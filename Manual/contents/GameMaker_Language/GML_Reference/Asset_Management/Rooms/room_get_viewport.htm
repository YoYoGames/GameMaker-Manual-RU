room_get_viewport

С помощью этой функции вы можете получить информацию о видовом порте в комнате, отличной от текущей. Вы указываете ID комнаты и индекс порта просмотра, который необходимо получить (от 0 до 7), и функция возвращает массив из 5 индексов, где:

  • [0] = видимый (булево: true / false)
  • [1] = положение x (реальное)
  • [2] = положение y (реальное)
  • [3] = ширина (реальная)
  • [4] = высота (реальная)

Синтаксис:

room_get_viewport(rm, vind);

Аргумент Тип Описание
rm Комнатный актив The index of the room to get viewport data from
vind Реально The index of the view port to get

Возвращается:

Array (5 elements: visible, x, y, width, height)

Пример:

v_vals = room_get_viewport(rm_Game, 0);

if v_vals[0] == false
{
    room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]);
}

Приведенный выше код извлекает данные порта просмотра для заданной комнаты, затем проверяет, отмечен ли порт как видимый. Если нет, данные порта вида устанавливаются, чтобы сделать его видимым.
