room_get_viewport

С помощью этой функции вы можете получить информацию о видовом порте в комнате, отличной от текущей. Вы указываете ID комнаты и индекс порта просмотра, который необходимо получить (от 0 до 7), и функция возвращает массив из 5 индексов, где:

  • [0] = видимый (булево: true / false)
  • [1] = положение x (реальное)
  • [2] = положение y (реальное)
  • [3] = ширина (реальная)
  • [4] = высота (реальная)
    • =======

    room_get_viewport

    This function retrieves the details of a view port in a room other than the current one.

    You give the room and the index of the view port to retrieve (from 0 to 7) and the function will return an array of 5 indices, where:

    • [0] = visible (Boolean: true / false)
    • [1] = x position (real)
    • [2] = y position (real)
    • [3] = width (real)
    • [4] = height (real)

    Пример:

    v_vals = room_get_viewport(rm_Game, 0);

    if v_vals[0] == false
    {
        room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]);
    }

    Приведенный выше код извлекает данные порта просмотра для заданной комнаты, затем проверяет, отмечен ли порт как видимый. Если нет, данные порта вида устанавливаются, чтобы сделать его видимым.

    Example:

    var _view_values = room_get_viewport(rm_Game, 0);
    var _visible_text = _view_values[0] ? "visible" : "invisible";
    show_debug_message($"Viewport 0 in rm_Game is {_visible_text} and drawn at the position: {_view_values[1]}, {_view_values[2]}. Its dimensions are {_view_values[3]}x{_view_values[4]}.");

    The above code retrieves the view port data for the given room then checks to see if the port is flagged as visible. If it is not, the view port data is set to make it visible.
