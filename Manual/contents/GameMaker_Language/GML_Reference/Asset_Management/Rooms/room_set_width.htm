room_set_width

С помощью этой функции вы можете изменить (или установить) ширину любой комнаты в вашей игре, кроме текущей.

Синтаксис:

room_set_width(index, w);

Аргумент Тип Описание
index Комнатный актив The index of the room to set the width of.
w Реально The new width of the room in pixels.

Возвращается:

N/A

Пример:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, false);

Это создаст новую комнату и сохранит ее индекс в переменной "global.myroom". Затем она установит ее ширину 640 пикселей, высоту 480 пикселей, надпись 'Game Room' и постоянство 'false'.
