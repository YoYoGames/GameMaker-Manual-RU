layer_get_shader

Эта функция используется для проверки того, назначен ли данному слою шейдер. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), и функция вернет либо индекс назначенного шейдера, либо -1, если шейдер не назначен.

Синтаксис:

layer_get_shader(layer_id)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the layer to target (or the layer name as a string)

Возвращается:

Shader Asset or -1 (if no shader is assigned)

Пример:

if layer_get_shader(layer) == -1
{
    layer_shader(layer, shd_Sepia);
}

Приведенный выше код проверяет, назначен ли шейдер слою, на котором находится экземпляр, выполняющий код, и если нет, то назначает шейдер.
