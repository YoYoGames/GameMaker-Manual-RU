layer_get_id

Эту функцию можно использовать для получения уникального значения ID для данного слоя. В IDE все слои имеют имя и тип, и чтобы иметь возможность редактировать или изменять их с помощью кода, необходимо задать значение ID слоя. Данная функция используется для получения этого ID, используя имя (строку) слоя (как написано в IDE). Если вы создаете новый слой с помощью кода, используя функцию layer_create() то эта функция вернет уникальное значение ID (динамически созданные слои не получают имен). Обратите внимание, что если вы зададите имя слоя, который не существует в текущей комнате, то вы получите ошибку и проект завершится аварийно.

Синтаксис:

layer_get_id(layer_name)

АргументТип Описание
layer_name The name of the layer (a string)

Возвращается:

Layer ID or -1 (if the layer specified doesn't exist)

Пример:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Приведенный выше код сначала получит индекс ближайшего экземпляра к заданной позиции x/y и сохранит его в локальной переменной. Затем он получает уникальный идентификатор слоя экземпляра для слоя с именем "Instances Front" и перемещает найденный экземпляр на этот слой.
