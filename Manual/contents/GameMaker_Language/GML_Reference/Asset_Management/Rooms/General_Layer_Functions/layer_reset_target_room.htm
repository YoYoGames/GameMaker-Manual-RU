layer_reset_target_room

Эта функция используется для сброса цели слоя на текущую комнату. См. функцию layer_set_target_room() для получения дополнительной информации.

Синтаксис:

layer_reset_target_room()

Возвращается:

Пример:

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
{
    layer_sprite_create(l, irandom(1000), irandom(1000), spr_Trees);
}
layer_reset_target_room();

Приведенный выше код устанавливает целевую комнату на комнату "rm_Game", а затем получает идентификатор слоя под названием "SpriteAssets" в этой комнате. Этот идентификатор слоя затем используется для добавления 50 случайных активов sprite в слой, прежде чем цель слоя будет сброшена на текущую комнату.
