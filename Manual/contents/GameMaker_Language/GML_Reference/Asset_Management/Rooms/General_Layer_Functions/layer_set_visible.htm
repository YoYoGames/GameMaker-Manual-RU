layer_set_visible

С помощью этой функции можно переключить видимость слоя. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), а также значение переключения для слоя, где visible - true, а invisible - false.

Когда слой невидим, ничто, назначенное этому слою, не будет рисоваться, а если слою назначены какие-либо экземпляры, то они даже не запустят свое событие рисования (точно так же, как если бы вы установили переменную экземпляра visible в значение false). Обратите внимание, что любые экземпляры, которые уже помечены как невидимые, не будут помечены как видимые, если для слоя, на котором они находятся, установлено значение visible, поскольку это две независимые настройки, хотя их поведение одинаково.

Синтаксис:

layer_set_visible(layer_id, visible)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the layer to target (or the layer name as a string)
visible Булево Whether the layer should be visible (true) или нет (false)

Возвращается:

N/A

Пример:

var lay_id = layer_get_id("Instances");
if (layer_get_visible(lay_id))
{
    layer_set_visible(lay_id, false);
}
else
{
    layer_set_visible(lay_id, true);
}

Приведенный выше код получает значение ID для слоя с именем "Instances" в редакторе помещений, затем использует ID для проверки видимости слоя, переключая видимость слоя в зависимости от возвращаемого значения.
