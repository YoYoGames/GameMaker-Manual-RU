layer_get_target_room

Эта функция возвращает текущее помещение, на которое направлены функции слоя. См. layer_set_target_room() для получения дополнительной информации.

Синтаксис:

layer_get_target_room()

Возвращается:

Room Asset

Пример:

if layer_get_target_room() != room
{
    layer_reset_target_room();
}

Приведенный выше код проверяет текущую целевую комнату, и если она не является текущей, то происходит сброс целевой комнаты слоя.
