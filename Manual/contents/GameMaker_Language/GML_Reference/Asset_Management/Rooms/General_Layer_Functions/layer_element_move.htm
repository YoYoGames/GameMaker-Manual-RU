layer_element_move

Эту функцию можно использовать для перемещения элемента с одного слоя на другой. Вы указываете идентификатор элемента, возвращаемый функцией, используемой для создания элемента, или редактором помещений, или функцией layer_get_all_elements(), а затем указываете ID слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность).

Синтаксис:

layer_element_move(element_id, layer_id)

Аргумент Тип Описание
element_id Идентификатор элемента слоя The unique ID value of the element to move
layer_id Строка или идентификатор слоя The unique ID value of the layer to move the element to (or the layer name as a string)

Возвращается:

N/A

Пример:

var a = layer_get_all_elements(layer);
asset_layer = layer_create(-100);
for (var i = 0; i < array_length(a); i++)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_element_move(a[i], asset_layer)
    }
}

Приведенный выше код получает элементы на слое, к которому назначен экземпляр, выполняющий код, затем проверяет, являются ли они активами sprite, и если да, то они перемещаются на слой с идентификатором, хранящимся в переменной "asset_layer".
