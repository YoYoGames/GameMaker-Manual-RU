layer_get_all

Эта функция возвращает массив, заполненный уникальными значениями ID каждого слоя в комнате.

Синтаксис:

layer_get_all()

Возвращается:

Array (populated with Layer IDs)

Пример:

var a = layer_get_all();
for (var i = 0; i < array_length(a); i++)
{
    layer_destroy(a[i]);
}

Приведенный выше код извлекает все слои в комнате и добавляет их значения ID в массив. Затем этот массив разбирается для уничтожения слоев комнаты.
