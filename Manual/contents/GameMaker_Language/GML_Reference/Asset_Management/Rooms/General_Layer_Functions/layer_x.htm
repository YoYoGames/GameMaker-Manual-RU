layer_x

Эту функцию можно использовать для установки x-позиции слоя в пределах текущего масштабируемого помещения. Вы указываете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), и функция переместит слой на заданное количество пикселей вдоль горизонтальной оси комнаты.

Синтаксис:

layer_x(layer_id, x)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the layer to set the x position of
x Реально The x position in the room to set the layer to

Возвращается:

N/A

Пример:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

Приведенный выше код проверяет заданную позицию слоя, и если она не установлена в (0, 0), то она устанавливается в эту позицию.
