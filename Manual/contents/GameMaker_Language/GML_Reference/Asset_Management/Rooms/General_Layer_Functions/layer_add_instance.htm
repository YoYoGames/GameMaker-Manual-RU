layer_add_instance

Эта функция может использоваться для перемещения данного экземпляра со слоя, на котором он находится в данный момент, на другой слой. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность) и ID экземпляра, который нужно переместить между слоями. Например, допустим, ваш игрок находится на слое, глубина которого меньше, чем у другого слоя, и вы хотите, чтобы он отображался за слоями между ними. Вы можете использовать эту функцию, чтобы переключить игрока с переднего плана на фоновый слой и заставить его появиться за другими рисуемыми слоями.

ПРИМЕЧАНИЕ Эта функция не может быть использована для добавления нового экземпляра в слой. Вы должны были создать экземпляр ранее и сохранить его ID в переменной.

ПРИМЕЧАНИЕ: Эта функция действует только в пределах текущей комнаты и не может быть использована, если область была изменена с помощью функции layer_set_target_room().

Синтаксис:

layer_add_instance(layer_id, instance_id)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the instance layer to target (or the layer name as a string)
instance_id Идентификатор экземпляра The unique instance ID value of the instance to move over to the target layer

Возвращается:

N/A

Пример:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Приведенный выше код сначала получит индекс ближайшего экземпляра к заданной позиции x/y и сохранит его в локальной переменной. Затем он получает уникальный идентификатор слоя экземпляра для слоя с именем "Instances Front" и перемещает найденный экземпляр на этот слой.
