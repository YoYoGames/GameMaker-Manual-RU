layer_get_all_elements

Вы можете использовать эту функцию для получения идентификаторов элементов заданного слоя. Вы передаете ID слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), и функция вернет массив идентификаторов, где каждая запись в массиве является уникальным идентификатором элемента данного слоя. Например, если слой является слоем активов, массив будет заполнен значениями ID для каждого актива sprite, который назначен слою. Обратите внимание, что использование кода для работы со слоями означает, что вы можете назначить различные типы элементов одному и тому же слою - например, у вас могут быть активы sprite наряду с экземплярами - в этом случае вы можете использовать функцию layer_get_element_type() для получения типа элемента, к которому относится идентификатор.

Синтаксис:

layer_get_all_elements(layer_id)

Аргумент Тип Описание
layer_id Строка или Идентификатор слоя The unique ID value of the layer to get the elements from (or the layer name as a string)

Возвращается:

Array of Layer Element IDs

Пример:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++)
{
    if (layer_get_element_type(a[i]) == layerelementtype_sprite)
    {
        layer_sprite_destroy(a[i]);
    }
}

Приведенный выше код получает идентификаторы всех элементов экземпляра, назначенных слою экземпляра, на котором выполняется код. Затем код проверяет, являются ли какие-либо из возвращенных элементов активами sprite, и если да, то они уничтожаются.
