layer_instance_get_instance

Эта функция может быть использована для получения уникального идентификатора экземпляра заданного элемента экземпляра. Вы сообщаете ID элемента экземпляра (как его получить, см. пример кода ниже), и функция возвращает вещественное значение, представляющее уникальный идентификатор экземпляра для элемента. Если элемент не является экземпляром, функция вернет -1.

Синтаксис:

layer_instance_get_instance(element_id)

Аргумент Тип Описание
element_id Идентификатор элемента экземпляра The unique ID value of the instance element to get the ID of

Возвращается:

Instance ID

elements = layer_get_all_elements("Instances");
