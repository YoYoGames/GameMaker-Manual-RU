layer_get_element_type

Вы можете использовать эту функцию для получения типа элемента для заданного элемента. Вы передаете уникальное значение ID элемента (например, возвращаемое функцией, создавшей элемент, или из редактора комнат), а функция возвращает одну из следующих констант (или -1, если элемент не существует или значение ID ошибочно):

=======

layer_get_element_type

You can use this function to get the element type for the given element. You supply the unique element ID value (for example, as returned the function that created the element or from The Room Editor) and the function will return one of the following constants:

layerelementtype_undefined The element does not exist or the ID value is erroneous.

Обратите внимание, что эта функция наиболее полезна, когда у вас есть несколько различных типов элементов, назначенных одному слою, и что вы можете получить список всех элементов на любом данном слое с помощью функции layer_get_all_elements().
