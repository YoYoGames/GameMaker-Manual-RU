layer_is_draw_depth_forced

Эта функция может быть использована для проверки и определения того, включена ли принудительная глубина Z для рендеринга слоев в проекте. См. layer_force_draw_depth() для получения дополнительной информации.

Синтаксис:

layer_is_draw_depth_forced()

Возвращается:

Пример:

if !layer_is_draw_depth_forced()
{
    layer_force_draw_depth(true, 0);
}

Приведенный выше код проверяет, является ли глубина Z слоя принудительной или нет, и если нет, то устанавливает глубину Z равной 0 и включает ее.
