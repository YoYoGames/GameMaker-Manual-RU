layer_get_element_layer

Вы можете использовать эту функцию для получения ID слоя, на котором находится данный элемент. Вы передаете уникальное значение ID элемента (например, возвращенное функцией, создавшей элемент, или из редактора помещений), а функция возвращает уникальный ID слоя, на котором находится элемент. Если заданный ID элемента не является действительным, то функция вернет -1.

Синтаксис:

layer_get_element_layer(element_id)

Аргумент Тип Описание
element_id Идентификатор элемента слоя The unique ID value of the element to get the layer of

Возвращается:

Layer ID or -1 (if the element is invalid)

Пример:

element = layer_get_element_layer(asset_1);

Приведенный выше код получает ID слоя для элемента с ID, хранящимся в переменной "asset_1", и сохраняет его в переменной.
