уничтожение слоя

Эта функция уничтожит заданный слой. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), после чего он будет удален из текущей комнаты. Если слой был создан в редакторе комнат, то в следующий раз, когда вы покинете комнату, а затем вернетесь, слой будет создан заново с оригинальным содержимым, однако если комната постоянна, слой будет удален, пока постоянство комнаты не будет снова отключено. Когда вы уничтожаете слой таким образом, все его содержимое также удаляется, поэтому любые идентификаторы ссылок для фонов, карт плиток и т.д. больше не будут действительны, а все экземпляры, назначенные слою, будут уничтожены (при этом одновременно выполняется событие Destroy, если оно у них есть, а также событие Clean Up).

Синтаксис:

layer_destroy(layer_id)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the layer to destroy (or the layer name as a string)

Возвращается:

N/A

Пример:

if !instance_exists(obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

Приведенный выше код проверит, существуют ли экземпляры объекта "obj_Bullet_Parent", и если нет, то уничтожит слой с идентификатором, сохраненным в глобальной переменной.
