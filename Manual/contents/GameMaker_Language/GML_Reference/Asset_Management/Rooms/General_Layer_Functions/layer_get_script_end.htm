layer_get_script_end

Эта функция возвращает индекс функции сценария, назначенной для запуска в конце рендеринга данного слоя, или возвращает -1, если функция не назначена. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность). Вы можете назначить функции сценария слою с помощью layer_script_begin() и layer_script_end().

Синтаксис:

layer_get_script_end(layer_id);

Аргумент Тип Описание
layer_id Строка или Идентификатор слоя The unique ID value of the layer to target (or the layer name as a string)

Возвращается:

Script Function or -1 (if no function is assigned)

Пример:

if layer_get_script_end(layer) == -1
{
    layer_script_end(layer, scr_ResetShaderValues);
}

Приведенный выше код проверяет, назначена ли слою, на котором находится экземпляр, выполняющий код, функция сценария, и если нет, то назначается.
