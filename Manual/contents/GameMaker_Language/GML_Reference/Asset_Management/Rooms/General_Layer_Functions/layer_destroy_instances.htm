layer_destroy_instances

Эта функция может быть использована для уничтожения всех экземпляров, назначенных данному слою. Вы передаете ID слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), после чего все экземпляры, находящиеся на слое, будут удалены из игры, вызывая события Destroy и Clean Up.

Синтаксис:

layer_destroy_instances(layer_id)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the layer (or the layer name as a string)

Возвращается:

N/A

Пример:

if global.game_over
{
    layer_destroy_instances(layer);
}

Приведенный выше код проверит глобальную переменную, и если она true, то все экземпляры, находящиеся на уровне вызывающего экземпляра, будут уничтожены (включая вызывающий экземпляр).
