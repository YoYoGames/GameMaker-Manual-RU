имя_слоя

Вы можете использовать эту функцию для получения имени заданного слоя. Вы передаете уникальное значение ID слоя, и если слой является одним из именованных слоев, созданных в редакторе помещений, то функция вернет строку с именем слоя. Если слой не является одним из слоев, созданных в редакторе помещений (т.е. он был создан с помощью функции layer_create()), то будет возвращена пустая строка.

Синтаксис:

layer_get_name(layer_id)

Аргумент Тип Описание
layer_id Идентификатор слоя The unique ID value of the layer to get the name of

Возвращается:

String

Пример:

var a = layer_get_all();
var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++;)
{
    if (layer_get_name(_arr_layers[i]) != "")
    {
        array_push(_arr_layers_named, _arr_layers[i]);
    }
}

Приведенный выше код получает идентификаторы всех слоев в комнате и затем циклически проверяет, являются ли они именованными слоями. Если это так, то их ID добавляется в список.
