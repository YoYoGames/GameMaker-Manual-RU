layer_has_instance

Эта функция может быть использована для проверки того, назначен ли данный экземпляр на данный момент данному слою. Вы передаете ID слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность) и ID экземпляра, который нужно проверить. Можно также указать object_index (т.е. имя объекта в Asset Browser), и функция проверит, есть ли экземпляры этого объекта на данном слое. Функция вернет true, если экземпляр находится на слое, и false, если нет.

Синтаксис:

layer_has_instance(layer_id, instance_id)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the layer to target (or the layer name as a string)
instance_id Идентификатор экземпляра The unique instance ID or the object index of the instance to check for

Возвращается:

Boolean

Пример:

if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

Приведенный выше код проверит, содержит ли данный слой какие-либо экземпляры объекта "obj_Bullet_Parent", и если нет, то уничтожит слой.
