layer_get_y

Вы можете использовать эту функцию для получения позиции y слоя в пределах текущего масштабируемого помещения. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), и функция возвращает вещественное число для x-положения слоя относительно (0,0) положения комнаты. По умолчанию - 0.

Синтаксис:

layer_get_y(layer_id)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the layer to get the y position of

Возвращается:

Real

Пример:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

Приведенный выше код проверяет заданную позицию слоя, и если она не установлена в (0, 0), то она устанавливается в эту позицию.
