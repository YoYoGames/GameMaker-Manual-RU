layer_exists

Эта функция может использоваться для проверки существования заданного слоя. Вы указываете ID слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), и функция вернет булево значение true, если он существует, или false, если не существует.

ПРИМЕЧАНИЕ: Эта функция работает в пределах текущей целевой комнаты - по умолчанию это комната, в которой вызывается функция - которую можно установить с помощью функции layer_set_target_room().

Синтаксис:

layer_exists(layer_name)

Аргумент Тип Описание
layer_name Строка или идентификатор слоя The name of the layer (a string or ID value)

Возвращается:

Boolean

Пример:

if !layer_exists(global.tileLayer)
{
    global.tileLayer = layer_create(1000);
}

Приведенный выше код проверит, существует ли слой, хранящийся в глобальной переменной, и если нет, то он будет создан.
