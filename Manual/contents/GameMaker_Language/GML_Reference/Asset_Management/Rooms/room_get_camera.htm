room_get_camera

С помощью этой функции вы можете получить уникальный индекс ID камеры, назначенной на определенный вид в комнате, отличной от текущей. Вы указываете комнату, которую нужно использовать, порт вида, который нужно использовать (от 0 до 7), и функция возвращает индекс камеры.

Синтаксис:

room_get_camera(rm, vind);

=======

room_get_camera

This function gets the unique index ID of the camera assigned to a specific view in a room other than the current one.

You give the room to use, the view port to use (from 0 to 7) and the function will return a camera index.

 

Syntax:

room_get_camera(rm, vind);
