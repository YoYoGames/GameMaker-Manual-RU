следующая комната

С помощью этой функции вы можете получить индекс комнаты после ввода комнаты в функцию. Например, вы можете использовать переменную room для получения индекса текущей комнаты, а затем использовать эту функцию для поиска комнаты, следующей за ней в списке Room Manager. Если после введенной вами комнаты нет комнаты, то возвращается -1. Обратите внимание, что эта функция не распознает и не учитывает комнаты, которые были добавлены динамически с помощью room_add() или room_duplicate().

Синтаксис:

room_next(numb);

АргументТип Описание
numb The index of the room to get the next one after.

Возвращается:

Room Asset

Пример:

if room_next(room) != -1
{
    room_goto_next();
}

Приведенный выше код проверит, существует ли следующая комната, и если да, то перейдет в нее.
