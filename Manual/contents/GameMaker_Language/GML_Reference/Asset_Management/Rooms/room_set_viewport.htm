room_set_viewport

С помощью этой функции вы можете установить свойства порта вида для любой комнаты в вашей игре, кроме текущей. Вы указываете индекс комнаты, в которой нужно установить порт вида, индекс порта вида (от 0 до 7), а затем - должен ли порт вида быть видимым в начале комнаты или нет (установите значение true, чтобы порт был видимым, и false в противном случае). После этого задаются координаты x и y (соответствующие верхнему левому положению порта просмотра), а затем ширина и высота в пикселях.

Синтаксис:

room_set_viewport(rm, vind, vis, xport, yport, wport, hport);

Аргумент Тип Описание
rm Комнатный актив The index of the room to set
vind Реально The index of the view port to set
vis Булево The visibility of the view port (true виден, false - невидим)
xport Реально The x position for the view port in the room
yport Реально The y position of the view port in the room
wport Реально The width (in pixels) of the view port
hport Реально The height (in pixels) of the view port

Возвращается:

N/A

Пример:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480);

Это установит свойства порта просмотра [0] в комнате, проиндексированной в переменной "global.myroom".
