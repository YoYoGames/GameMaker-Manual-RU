layer_sprite_get_speed

Эта функция может быть использована для получения текущего значения множителя скорости элемента sprite. Вы передаете идентификатор элемента sprite (который вы получаете при создании элемента sprite с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), и функция вернет реальное значение, которое представляет собой множитель скорости, используемый для анимации элемента sprite. Значение по умолчанию равно 1.

Синтаксис:

layer_sprite_get_speed(sprite_element_id);

АргументТип Описание
sprite_element_id The unique ID value of the sprite element to get the information from

Возвращается:

Пример:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_speed(spr_id) > 0)
{
    layer_sprite_speed(spr_id, 0);
}

Приведенный выше код получит идентификатор слоя с именем "sprite_sky" и затем использует его для получения идентификатора элемента sprite на этом слое. Этот ID затем используется для проверки скорости анимации для элемента, и если он больше 0, то он устанавливается на 0.
