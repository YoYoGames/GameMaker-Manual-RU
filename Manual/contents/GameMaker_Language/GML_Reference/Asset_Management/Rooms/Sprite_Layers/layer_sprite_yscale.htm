layer_sprite_yscale

С помощью этой функции можно изменить, должен ли данный элемент sprite на слое масштабироваться по оси y или нет. Вы задаете идентификатор элемента sprite (который вы получаете при создании элемента sprite с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), а затем задаете значение масштаба. Значение масштаба 1 означает отсутствие масштабирования (1:1), меньшие значения масштабируют вниз (0,5, например, уменьшает высоту используемого sprite вдвое), большие значения масштабируют вверх, а отрицательные значения зеркально отражают sprite и масштабируют его, за исключением случая, когда используемое значение равно -1 (в этом случае используемый sprite просто зеркально отражается сверху вниз относительно своей позиции (0, 0) без масштабирования).

Синтаксис:

layer_sprite_yscale(sprite_element_id, yscale);

АргументТип Описание
sprite_element_id The unique ID value of the sprite element to change
yscale The yscale value (default is 1)

Возвращается:

Пример:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
    layer_sprite_xscale(asset_sprite, 1);
    layer_sprite_yscale(asset_sprite, 1);
}

Приведенный выше код проверит элемент sprite, назначенный слою, на котором находится экземпляр, выполняющий код, и если он масштабирован в любом направлении, то установит масштаб по оси x и масштаб по оси y равным 1.
