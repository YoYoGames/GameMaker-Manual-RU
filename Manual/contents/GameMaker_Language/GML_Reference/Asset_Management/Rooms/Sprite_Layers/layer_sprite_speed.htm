скорость_слоя

Эта функция может быть использована для установки скорости анимирующего изображения, которое было назначено элементу sprite. Вы задаете идентификатор элемента sprite (который вы получаете при создании элемента sprite с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), а затем задаете скорость анимации. Если IDE задал скорость анимации для используемого sprite элемента как Frames Per Game Frame, то заданное значение скорости является множителем, причем 1 является значением по умолчанию, а установка значения 0,5 уменьшит скорость анимации вдвое, а установка значения 1,5 удвоит ее. Однако если на сайте IDE задана скорость анимации для используемого сайта sprite, рассчитываемая как кадры в секунду, то заданное значение скорости - это буквальное значение кадров в секунду, поэтому, например, установка значения 5 приведет к тому, что анимация будет повторяться по 5 кадров каждую секунду, пока идет игра. Если используемый sprite не имеет вложенных изображений, эта функция не будет иметь никакого эффекта.

Синтаксис:

layer_sprite_speed(sprite_element_id, speed);

Аргумент Тип Описание
sprite_element_id Актив спрайта The unique ID value of the sprite element to change
speed Реально The speed multiplier/value to use for the element

Возвращается:

N/A

Пример:

var lay_id = layer_get_id("Assets_water");
var spr_id = layer_sprite_get_id(lay_id, "gfc_water");
layer_sprite_speed(spr_id, 0.5);

Приведенный выше код получит идентификатор слоя с именем "Assets_water", а затем использует его для получения идентификатора заданного элемента sprite на этом слое. Этот идентификатор затем используется для изменения скорости изображения элемента.
