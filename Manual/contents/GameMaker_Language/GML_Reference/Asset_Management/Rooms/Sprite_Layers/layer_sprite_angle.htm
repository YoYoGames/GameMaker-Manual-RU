угол наклона_слоя

С помощью этой функции можно изменить угол для заданного элемента sprite на слое. Вы задаете идентификатор элемента sprite (который вы получаете при создании элемента sprite с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), а затем задаете значение угла, от 0 до 359 против часовой стрелки, где 0 - справа, 90 - сверху, 180 - слева и 270 - снизу. Если вы зададите значение больше 360, оно будет зациклено, чтобы привести его в диапазон 0 - 359.

Синтаксис:

layer_sprite_angle(sprite_element_id, angle);

Аргумент Тип Описание
sprite_element_id Идентификатор элемента спрайта The unique ID value of the sprite element to change
angle Реально The angle of the sprite (default is 0)

Возвращается:

N/A

Пример:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_angle(asset_sprite) != 0
{
    layer_sprite_angle(asset_sprite, 0);
}

Приведенный выше код проверит элемент sprite, назначенный слою, на котором находится экземпляр, выполняющий код, и если его угол не равен 0, то установит его в 0.
