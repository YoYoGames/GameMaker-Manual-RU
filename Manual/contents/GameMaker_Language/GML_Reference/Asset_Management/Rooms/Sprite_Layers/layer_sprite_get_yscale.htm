layer_sprite_get_yscale

Эта функция может быть использована для получения текущего множителя масштаба по оси y для элемента sprite. Вы указываете идентификатор элемента sprite (который вы получаете при создании элемента sprite с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), и функция вернет реальное значение для масштаба по оси y, где 1 - нет масштабирования.

Синтаксис:

layer_sprite_get_yscale(sprite_element_id);

АргументТип Описание
sprite_element_id The unique ID value of the sprite element to get the information from

Возвращается:

Пример:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_yscale(spr_id) != 1)
{
    layer_sprite_yscale(spr_id, 1);
}

Приведенный выше код получит идентификатор слоя с именем "sprite_sky" и затем использует его для получения идентификатора элемента sprite на этом слое. Этот идентификатор затем используется для проверки значения yscale, и если оно не равно 1, то оно устанавливается равным 1.
