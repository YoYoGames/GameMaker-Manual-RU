слой_спрайта_создать

С помощью этой функции вы можете назначить ресурс sprite слою, который будет использоваться в вашем проекте. Вы предоставляете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create() или когда вы используете имя слоя вместе с layer_get_id()), позицию в помещении и индекс sprite (который будет именем sprite, как показано в браузере активов), и он будет добавлен в слой. Функция возвращает уникальное значение ID для элемента, которое затем может быть использовано в дальнейших функциях слоя для sprites.

Синтаксис:

layer_sprite_create(layer_id, x, y, sprite)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the layer to target
x Реально The x position to use
y Реально The y position to use
sprite Актив спрайта The sprite index to be used

Возвращается:

Sprite Element ID

Пример:

global.asset_layer = layer_create(10000);
for (var i = 0; i< 10; i++;)
{
    var _x = random(room_width);
    var _y = room_height - 100;
    global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, _x, _y, spr_Trees);
}

Приведенный выше код создает новый слой, а затем добавляет к нему 10 новых элементов sprite, сохраняя ID каждого элемента в массив.
