слой_спрайта_y

Эта функция управляет положением по оси y комнаты элемента актива sprite на слое. Вы задаете идентификатор элемента sprite (который вы получаете при создании элемента sprite с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), а затем задаете значение y, которое необходимо использовать (на основе координат помещения).

Синтаксис:

layer_sprite_y(sprite_element_id, y);

АргументТип Описание
sprite_element_id The unique ID value of the sprite element to change
y The y position for the sprite

Возвращается:

Пример:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_y(spr_id, 50 + irandom(room_height - 200));

Приведенный выше код получает значение ID актива sprite "Облака", назначенного слою "Asset_sky", а затем устанавливает его положение по оси y на случайное значение между 50 и высотой комнаты минус 200.
