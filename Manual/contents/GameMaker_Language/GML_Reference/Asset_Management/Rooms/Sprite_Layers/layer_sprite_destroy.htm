слой_спрайта_уничтожить

Эта функция уничтожит заданный элемент sprite. Вы передаете идентификатор sprite (который вы получаете при создании sprite с помощью функции layer_sprite_create() или когда вы используете ID слоя вместе с layer_get_sprite_id()), и эта функция удалит его. Обратите внимание, что при этом не удаляется слой, а только sprite, и если sprite - это элемент, который был добавлен в редакторе комнаты, то в следующий раз, когда вы покинете комнату, а затем вернетесь, sprite будет создан заново. Однако если комната постоянна, то sprite будет удален, пока постоянство комнаты не будет снова отключено.

Синтаксис:

layer_sprite_destroy(sprite_element_id)

АргументТип Описание
sprite_element_id The unique ID value of the sprite to be destroyed

Возвращается:

Пример:

var lay_id = layer_get_id("Asset_Trees");
if (layer_sprite_exists(lay_id, global.Asset_spr_trees))
{
    layer_sprite_destroy(lay_id);
}

Приведенный выше код проверяет слой "Asset_Trees" на предмет существования заданного элемента sprite, и если он существует, то уничтожается (но не слой).
