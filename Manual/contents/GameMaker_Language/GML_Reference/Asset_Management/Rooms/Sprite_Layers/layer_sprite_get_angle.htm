layer_sprite_get_angle

Эта функция может быть использована для получения текущего угла для элемента sprite. Вы задаете идентификатор элемента sprite (который вы получаете при создании элемента sprite с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), и функция вернет реальное значение угла. Это значение будет находиться в диапазоне от 0 до 360, где 0 означает право, 90 - вверх, 180 - влево и 270 - вниз.

Синтаксис:

layer_sprite_get_angle(sprite_element_id);

АргументТип Описание
sprite_element_id The unique ID value of the sprite element to get the information from

Возвращается:

Real

Пример:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_angle(spr_id) != 0)
{
    layer_sprite_yscale(spr_id, 0);
}

Приведенный выше код получит идентификатор слоя с именем "sprite_sky" и затем использует его для получения идентификатора элемента sprite на этом слое. Этот идентификатор затем используется для проверки угла, и если он не равен 0, то он устанавливается равным 0.
