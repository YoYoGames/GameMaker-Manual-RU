layer_sprite_index

Эта функция может быть использована для установки индекса изображения в активе sprite, который имеет несколько суб-изображений на слое. Вы задаете идентификатор элемента sprite (который вы получаете при создании элемента sprite с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), а затем задаете индекс изображения для использования. Если вы зададите значение вне диапазона суб-изображений, то индекс изображения будет зацикливаться.

Синтаксис:

layer_sprite_index(sprite_element_id, image_index);

Аргумент Тип Описание
sprite_element_id Актив спрайта The unique ID value of the sprite element to set
index Реально The image index to use for the sprite

Возвращается:

N/A

Пример:

var lay_id = layer_get_id("sprite_trees");
var spr_id = layer_sprite_get_id(lay_id, "gfc_trees");
layer_sprite_index(spr_id, 1);

Приведенный выше код получит идентификатор слоя с именем "sprite_trees", а затем использует его для получения идентификатора заданного элемента sprite на этом слое. Этот идентификатор затем используется для изменения индекса изображения элемента.
