layer_sprite_alpha

Эта функция управляет альфа (прозрачностью) элемента sprite на слое актива. Вы задаете идентификатор элемента sprite (который вы получаете при создании элемента sprite с помощью функции layer_sprite_create() или когда вы используете функция layer_sprite_get_id()), а затем задаете используемое значение альфа. Альфа может быть от 0 (полностью прозрачный) до 1 (полностью непрозрачный), а значение альфа по умолчанию для элемента sprite равно 1. Обратите внимание что если слой, которому был назначен элемент sprite, не виден, или сам элемент был сделан невидимым, вы не увидите никакой разницы с помощью этой функции, пока слой или элемент не станут видимыми снова.

Синтаксис:

layer_sprite_alpha(sprite_element_id, alpha);

АргументТип Описание
sprite_element_id The unique ID value of the sprite element to change
alpha The alpha for sprite sprite, from 0 to 1 (default is 1)

Возвращается:

Пример:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_alpha(spr_id, random(1));

Приведенный выше код получает значение ID актива sprite под названием "Clouds", назначенного слою "Asset_sky", а затем устанавливает его альфа-фактор на случайное значение между 0 и 1.
