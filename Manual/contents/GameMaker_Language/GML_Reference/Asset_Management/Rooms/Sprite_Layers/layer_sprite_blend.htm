слой_спрайта_смесь

Эта функция управляет смешиванием (или "тонированием") sprite sprite. Значение по умолчанию равно -1 (что представляет константу c_white, которая также может быть использована). Любое другое значение (включая внутренние цветовые константы, такие как c_red, или c_aqua) будет смешивать указанный цвет с исходным sprite. Вы задаете идентификатор элемента sprite (который вы получаете при создании элемента sprite с помощью функции layer_sprite_create() или при использовании функция layer_sprite_get_id()), а затем задаете цвет смешивания. Ниже показан пример элемента sprite, который был смешан с различными цветами:

Обратите внимание, что на платформе HTML5 следует стараться ограничивать смешивание (если не используется WebGL), поскольку каждое смешивание sprite должно кэшироваться отдельно, и поэтому большое количество смешиваний sprites может негативно сказаться на производительности (вы также можете установить размер кэша с помощью функции sprite_set_cache_size()).

Синтаксис:

layer_sprite_blend(sprite_element_id, blend);

АргументТип Описание
sprite_element_id The unique ID value of the sprite element to change
blend The colour to blend with the sprite sprite (default is c_white)

Возвращается:

Пример:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_blend(spr_id, c_gray);

Приведенный выше код получает идентификационное значение sprite под названием "Clouds", назначенное слою "Asset_sky", и затем оттеняет его цветом.
