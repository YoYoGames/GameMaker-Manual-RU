layer_sprite_get_id

Эта функция может использоваться для получения уникального значения ID элемента sprite на слое. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create() или при использовании имени слоя вместе с layer_get_id()) и имя элемента sprite, определенное в редакторе помещений. Функция вернет значение ID, связанное с этим элементом sprite на слое.

Обратите внимание, что эта функция специально предназначена для использования с sprites, которые были добавлены в IDE на слой актива, и если вы добавили sprite на слой через код с помощью функции layer_sprite_create(), то для всех дальнейших ссылок следует использовать идентификатор, возвращаемый этой функцией (поскольку у этого элемента sprite не будет имени, которое можно передать в эту функцию).

Если указанный слой не существует, или заданный элемент sprite не может быть найден, функция вернет -1.

Синтаксис:

layer_sprite_get_id(layer_id, sprite_element_name)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the layer to target
sprite_element_name Строка The unique name of the sprite element on the layer as defined in the Room Editor

Возвращается:

Sprite Element ID

Пример:

var lay_id = layer_get_id("Assets_trees");
var back_id = layer_sprite_get_id(lay_id, "graphic_254367CB");
layer_sprite_change(back_id, spr_Trees_Winter);

Приведенный выше код получит идентификатор слоя "Assets_trees" и затем использует его для получения идентификатора элемента sprite "graphic_254367CB" на этом слое. Полученный идентификатор элемента sprite затем используется для изменения элемента sprite.
