layer_sprite_xscale

С помощью этой функции можно изменить, должен ли данный элемент sprite на слое масштабироваться по оси x или нет. Вы задаете идентификатор элемента sprite (который вы получаете при создании элемента sprite с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), а затем задаете значение масштаба. Значение масштаба 1 означает отсутствие масштабирования (1:1), меньшие значения масштабируют вниз (0,5, например, уменьшает ширину используемого sprite в два раза), большие значения масштабируют вверх, а отрицательные значения переворачивают sprite и масштабируют его, если только используемое значение не равно -1 (в этом случае используемый sprite просто переворачивается вправо-влево относительно исходного положения без масштабирования).

Синтаксис:

layer_sprite_xscale(sprite_element_id, xscale);

АргументТип Описание
sprite_element_id The unique ID value of the sprite element to change
xscale The xscale value (default is 1)

Возвращается:

Пример:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
    layer_sprite_xscale(asset_sprite, 1);
    layer_sprite_yscale(asset_sprite, 1);
}

Приведенный выше код проверит элемент sprite, назначенный слою, на котором находится экземпляр, выполняющий код, и если он масштабирован в любом направлении, то установит масштаб по оси x и масштаб по оси y равным 1.
