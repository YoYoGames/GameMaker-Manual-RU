слой_изменения

С помощью этой функции можно изменить ресурс sprite, назначенный данному элементу sprite на слое. Вы указываете идентификатор элемента sprite (который вы получаете при создании элемента sprite с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), а затем указать индекс sprite элемента, ресурс которого будет изменен. Обратите внимание, что если sprite имеет под-изображения, то они тоже будут анимироваться (это можно контролировать с помощью функций layer_sprite_index() и layer_sprite_speed() функции ). Обратите внимание, что вы можете присвоить значение -1 в качестве нового индекса sprite, и тогда не будет показан sprite, хотя элемент будет существовать и может быть изменен позже.

Синтаксис:

layer_sprite_change(sprite_element_id, sprite_index)

Аргумент Тип Описание
sprite_element_id Идентификатор элемента спрайта The unique ID value of the sprite element to change
sprite_index Актив спрайта The new sprite index to use

Возвращается:

N/A

Пример:

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if layer_sprite_get_sprite(s) != spr_Nighttime
{
    layer_sprite_change(s, spr_nighttime);
}

Приведенный выше код проверяет индекс sprite элемента с идентификатором, хранящимся в глобальной переменной global.Asset_sprite, и если он не "spr_Nighttime", то элементу присваивается индекс sprite.
