слой_спрайта_существует

С помощью этой функции можно проверить, существует ли элемент sprite на данном слое. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность) и идентификатор элемента sprite (который вы получаете при создании элемента sprite с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()) и функция вернет либо true, если элемент существует, либо false, если его нет.

ПРИМЕЧАНИЕ: Эта функция работает в пределах текущей целевой комнаты - по умолчанию это комната, в которой вызывается функция, - которую можно задать с помощью функции layer_set_target_room().

Синтаксис:

layer_sprite_exists(layer_id, sprite_element_id)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the layer to target (or the layer name as a string)
sprite_element_id Идентификатор элемента спрайта The unique ID value of the sprite element to check

Возвращается:

Boolean

Пример:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.TreesSprites)
{
    layer_sprite_destroy(lay_id, global.TreesSprites);
}

Приведенный выше код проверяет слой "Asset_trees" на предмет существования заданного элемента sprite, и если он существует, то уничтожается (но не слой).
