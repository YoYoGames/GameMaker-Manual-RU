layer_get_fx

Эта функция используется для получения структуры FX для слоя. Вы указываете либо ID, либо имя слоя, на который вы хотите нацелиться, и функция возвращает структуру, содержащую информацию о примененном эффекте. Эта структура будет похожа на структуру, которую вы получите из fx_create(), и функции fx_get_parameter/s и fx_set_parameter/s могут быть использованы для чтения и изменения его параметров.

Если к указанному слою не применены фильтры/эффекты, функция вернет -1.

Синтаксис:

layer_get_fx(layer_name_or_id);

Аргумент Тип Описание
layer_name_or_id Строка или идентификатор слоя The name or ID of the layer to read

Возвращается:

FX Struct (or -1 if not found)

Пример:

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

Приведенный выше код запускает for цикл по всем слоям, присутствующим в комнате, и проверяет каждый слой на наличие FX struct. Если слой имеет FX-структуру, проверяется имя этого фильтра/эффекта путем вызова функции fx_get_name() и если оно равно "_filter_tintfilter", что означает, что это фильтр "Colour Tint", то он меняет цвет оттенка на синий.
