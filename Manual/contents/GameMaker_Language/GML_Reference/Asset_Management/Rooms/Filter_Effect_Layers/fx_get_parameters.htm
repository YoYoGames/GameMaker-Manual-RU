fx_get_parameters

Эта функция используется для получения структуры, содержащей все параметры для заданной FX-структуры. Вы указываете структуру FX (как возвращено из fx_create() или layer_get_fx()), и эта функция возвращает структуру, содержащую переменные для всех параметров, применимых к этому фильтру/эффекту. Параметры, использующие векторы (т.е. группы значений), будут представлены в виде массивов.

Вы можете получить названия всех параметров в отдельном массиве, вызвав fx_get_parameter_names(), или найти их на этой странице: Типы и параметры FX.

Изменение значений в этой структуре не приведет к применению этих изменений к фильтру/эффекту; чтобы изменения вступили в силу, необходимо вызвать fx_set_parameters() для применения измененной структуры обратно к фильтру/эффекту.

Синтаксис:

fx_get_parameters(filter_or_effect);

Аргумент Тип Описание
filter_or_effect FX Struct The FX struct for which the parameters will be returned

Возвращается:

Struct (or -1 on failure)

Пример:

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

Приведенный выше код получает структуру FX для слоя, к которому предположительно применен фильтр "Оттенок цвета", и извлекает его структуру параметров, вызывая fx_get_parameters(). После этого он создает колеблющееся значение с помощью sin() и current_time, которое затем используется для значений RGBA для эффекта оттенка, чтобы он продолжал анимироваться. Массив RGBA присваивается переменной g_TintCol в структуре параметров, и эта структура затем применяется обратно к структуре FX вызовом fx_set_parameters().
