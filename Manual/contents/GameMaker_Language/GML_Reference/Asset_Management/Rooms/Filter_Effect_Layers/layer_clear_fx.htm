layer_clear_fx

Эта функция используется для очистки структуры FX, примененной к указанному слою. Вы указываете либо ID, либо имя слоя, который хотите изменить, и эта функция удалит структуру FX, примененную к этому слою, полностью отключив его эффекты.

Если вы используете эту функцию для временного отключения эффекта и хотите восстановить его в слое, то сначала вам следует сохранить структуру FX для слоя, вызвав layer_get_fx(), а затем вызвать эту функцию, чтобы очистить его в слое. Позже вы можете снова применить его к слою, вызвав layer_set_fx(), что снова включит эффект.

Синтаксис:

layer_clear_fx(layer_name_or_id);

Аргумент Тип Описание
layer_name_or_id Строка или Идентификатор слоя The name or ID of the layer to clear the FX for

Возвращается:

N/A

Пример:

// Disable effect
if (keyboard_check_pressed(ord("E")))
{
    if (layer_get_fx("EffectLayer") != -1)
    {
        temp_fx = layer_get_fx("EffectLayer");
        layer_clear_fx("EffectLayer");
    }
}
// Re-enable effect
else if (keyboard_check_pressed(ord("F")))
{
    if (temp_fx != -1)
    {
        layer_set_fx("EffectLayer", temp_fx);
        temp_fx = -1;
    }
}

Приведенный выше код проверяет, нажата ли клавиша "E", а затем проверяет, применен ли к слою "EffectLayer" фильтр/эффект; в этом случае он сохраняет его структуру FX во временной переменной экземпляра и очищает слой от любых фильтров/эффектов. При нажатии клавиши "F" проверяется, не содержит ли временная переменная чего-либо, отличного от -1, и в этом случае она применяет FX обратно к слою и сбрасывает временную переменную на -1. Все это позволяет включать и выключать эффект в любое время без потери его параметров.
