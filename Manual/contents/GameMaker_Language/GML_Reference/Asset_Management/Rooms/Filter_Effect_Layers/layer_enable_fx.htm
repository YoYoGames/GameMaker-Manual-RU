layer_enable_fx

Эта функция включает/выключает фильтр/эффект, назначенный слою помещения. Вы указываете либо ID, либо имя слоя, который хотите изменить, а затем булево значение, указывающее, должен ли FX быть включен(true) или отключен(false).

Если передать false, то FX не удалится из слоя, а просто станет невидимым. Для удаления FX из слоя используйте layer_clear_fx().

Аналогично, передача true ничего не даст, если к слою не применен FX. Используйте layer_set_fx(), чтобы применить FX к слою.

Синтаксис:

layer_enable_fx(layer_name_or_id, enable);

Аргумент Тип Описание
layer_name_or_id Строка или Идентификатор слоя The name or ID of the layer to modify
enable Булево Whether to enable or disable the FX

Возвращается:

N/A

Пример:

if (hp <= 3)
{
    layer_enable_fx("DesaturateLayer", true);
}
else
{
    layer_enable_fx("DesaturateLayer", false);
}

Приведенный выше код включает слой Desaturate FX, если значение HP экземпляра меньше или равно 3, в противном случае он отключается. Это указывает игроку на низкий уровень HP, обесцвечивая экран.
