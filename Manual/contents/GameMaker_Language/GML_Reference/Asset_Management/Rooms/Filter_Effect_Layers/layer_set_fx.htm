layer_set_fx

Эта функция используется для назначения структуры FX слою помещения. Вы указываете либо ID, либо имя слоя, который вы хотите изменить, а затем FX struct (как возвращено из fx_create() или layer_get_fx()), который будет применен к указанному слою, делая эффект видимым в комнате.

Если вы укажете слой, который не является слоем фильтра/эффекта, а относится к другому типу (например, слой экземпляра, слой актива, слой плитки и т. д.), то эффект будет применен к содержимому этого слоя, а также ко всем слоям под ним.

Если для указанного слоя уже назначена структура FX, она будет заменена новой.

ПРИМЕЧАНИЕ Вызов этой функции на корневом уровне сценария ничего не даст, так как сценарии выполняются до загрузки первой комнаты. Вы можете сначала установить целевую комнату, а затем вызвать эту функцию.

Синтаксис:

layer_set_fx(layer_name_or_id, filter_or_effect);

Аргумент Тип Описание
layer_name_or_id Строка или Идентификатор слоя The name or ID of the layer to modify
filter_or_effect FX Struct The FX struct to apply to the layer

Возвращается:

N/A

Пример:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

Приведенный выше код создает новую структуру FX из эффекта "_filter_tintfilter", который является фильтром "Colour Tint", находящимся в Room Editor. Он присваивает значение своему параметру "g_TintCol", который является цветом оттенка, а так как внутри это vec4, он принимает массив, содержащий четыре значения (соответствующие значениям красного, зеленого, синего и альфа). Затем FX-структура для оттенка применяется к FX-слою.
