fx_set_single_layer

Эта функция используется для включения или отключения режима "Single Layer" для фильтра/эффекта. По умолчанию фильтр/эффект применяется к слою, которому он назначен, и ко всем слоям ниже этого слоя, однако вы можете включить режим Single Layer для структуры FX, чтобы убедиться, что он применяется только к слою, которому назначен.

Вы указываете структуру FX (как возвращено из fx_create() или layer_get_fx()) и булево значение, чтобы включить (true) или выключить (false) однослойный режим.

На следующем рисунке показан фильтр, используемый с отключенным режимом Single Layer (это поведение по умолчанию для всех слоев FX) и тот же фильтр с включенным режимом Single Layer:

ПРИМЕЧАНИЕ При включении режима Single Layer для FX убедитесь, что вы применяете его к слою, который не является слоем FX, поскольку слой FX сам по себе ничего не содержит, и поэтому фильтр/эффект не будет виден.

Синтаксис:

fx_set_single_layer(filter_or_effect, enable);

АргументТип Описание
filter_or_effect The FX struct to modify
enable A boolean value to enable (true) или отключить (false) Однослойный режим

Возвращается:

Пример:

shake_fx = fx_create("_filter_screenshake");
fx_set_single_layer(shake_fx, true);
layer_set_fx("ShakeyThings", shake_fx);

Приведенный выше код создает новый фильтр screenshake FX, включает для него режим Single Layer и затем применяет его к слою комнаты. Это означает, что фильтр screenshake будет применен только к слою "ShakeyThings".
