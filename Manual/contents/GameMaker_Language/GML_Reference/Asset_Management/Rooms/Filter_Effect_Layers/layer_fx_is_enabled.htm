layer_fx_is_enabled

Эта функция возвращает, включен ли фильтр/эффект для слоя.

Вы указываете либо ID, либо имя слоя, на который вы хотите нацелиться, и функция возвращает булево значение, указывающее, включен ли FX слоя(true) или отключен(false).

Синтаксис:

layer_fx_is_enabled(layer_name_or_id);

Аргумент Тип Описание
layer_name_or_id Строка или Идентификатор слоя The name or ID of the layer to check

Возвращается:

Boolean

Пример:

x = xstart;
y = ystart;
hp = hp_max;

if (layer_fx_is_enabled("DesaturateLayer"))
{
    layer_enable_fx("DesaturateLayer", false);
}

Приведенный выше код "возрождает" экземпляр, перемещая его в исходное положение и пополняя его HP. Затем он проверяет, включен ли слой Desaturate FX, и если включен, то отключает его.
