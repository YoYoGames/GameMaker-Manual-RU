fx_get_parameter_names

Эта функция используется для получения имен всех параметров, присутствующих в фильтре/эффекте. Вы указываете структуру FX для чтения (как возвращено из fx_create() или layer_get_fx()), и функция возвращает массив, содержащий имена параметров фильтра в виде строк.

Синтаксис:

fx_get_parameter_names(filter_or_effect);

АргументТип Описание
filter_or_effect The FX struct to get the parameter names of

Возвращается:

(of Strings)

Пример:

var _fx_struct = layer_get_fx("EffectLayer");
var _param_names = fx_get_parameter_names(_fx_struct);

for (var i = 0; i < array_length(_param_names); i ++) {
    show_debug_message("Parameter " + string(i) + ": " + _param_names[i]);
}

Приведенный выше код извлекает структуру FX для слоя и получает имена параметров в массив. Затем он запускает for цикл по этому массиву и печатает каждый параметр в выходной журнал.
