fx_get_name

Эта функция используется для получения имени фильтра/эффекта из его структуры. Вы указываете структуру FX для чтения (как возвращено из fx_create() или layer_get_fx()), и функция возвращает ее имя в виде строки. Это имя может быть передано в fx_create() для создания новой структуры FX с тем же типом фильтра/эффекта.

Синтаксис:

fx_get_name(filter_or_effect);

АргументТип Описание
filter_or_effect The FX struct to get the name of

Возвращается:

Пример:

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

Приведенный выше код запускает for цикл по всем слоям, присутствующим в комнате, и проверяет каждый слой на наличие FX-структуры. Если слой имеет FX-структуру, он проверяет имя этого фильтра/эффекта, вызывая fx_get_name(); если оно равно "_filter_tintfilter", что означает, что это фильтр "Colour Tint", он меняет цвет оттенка на синий.
