fx_set_parameter

Эта функция используется для изменения значения параметра в заданной структуре FX. Вы указываете FX struct (как возвращено из fx_create() или layer_get_fx()), имя параметра в виде строки и его новое значение. В результате ваши изменения станут видимыми в слое, которому присвоена данная FX-структура.

Вы можете передать несколько аргументов для параметра, если он является вектором, однако вы также можете просто указать массив в качестве единственного аргумента, который будет содержать несколько значений.

Найдите названия параметров для фильтра/эффекта на этой странице: Типы и параметры FX.

Синтаксис:

fx_set_parameter(filter_or_effect, parameter_name, val, ...);

Аргумент Тип Описание
filter_or_effect FX Struct The FX struct to modify
parameter_name Строка The name of the parameter to change the value of (as a string)
val... Реальный или Массив The new value(s) for that parameter as a real or array

Возвращается:

N/A

Пример:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

Приведенный выше код создает новую структуру FX из эффекта "_filter_tintfilter", который является фильтром "Colour Tint", находящимся в Room Editor. Он присваивает значение своему параметру "g_TintCol", который является цветом оттенка, а так как внутри это vec4, он принимает массив, содержащий четыре значения (соответствующие значениям красного, зеленого, синего и альфа). Затем FX-структура для оттенка применяется к FX-слою.

Обратите внимание, что вы также можете передавать значения в вектор как отдельные аргументы вместо массива:

fx_set_parameter(_fx_tint, "g_TintCol", 1, 0, 0, 1);
