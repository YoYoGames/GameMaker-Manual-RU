fx_set_parameters

Эта функция используется для изменения параметров фильтра/эффекта. Вы указываете структуру FX (как возвращено из fx_create() или layer_get_fx()) и структуру, содержащую его параметры (как возвращено из fx_get_parameters()). В результате ваши изменения будут видны в слое, которому присвоен FX struct.

Ваша структура параметров не обязана содержать все параметры, применимые к фильтру/эффекту, и только переменные, присутствующие в данной структуре, будут обновлены для фильтра/эффекта.

Синтаксис:

fx_set_parameters(filter_or_effect, parameter_struct);

АргументТип Описание
filter_or_effect The FX struct to modify
parameter_struct A struct containing the parameters for the filter/effect

Возвращается:

Пример:

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

Приведенный выше код получает структуру FX для слоя, к которому, как предполагается, применен фильтр "Цветовой оттенок", и извлекает структуру параметров, вызывая команду fx_get_parameters(). После этого он создает осциллирующее значение с помощью sin() и current_time, которое затем используется для значений RGBA для эффекта оттенка, так что он продолжает анимироваться. Массив RGBA присваивается переменной g_TintCol в структуре параметров, а затем эта структура применяется обратно к структуре FX вызовом fx_set_parameters().
