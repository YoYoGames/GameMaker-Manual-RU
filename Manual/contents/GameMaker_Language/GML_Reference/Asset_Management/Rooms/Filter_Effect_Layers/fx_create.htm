fx_create

Эта функция используется для создания новой структуры FX для заданного типа эффекта, который затем может быть применен к слою.

На этой странице перечислены названия фильтров/эффектов, которые можно передать в эту функцию: Все типы фильтров/эффектов.

Использование

Вы можете прочитать параметры для созданного эффекта, вызвав команду fx_get_parameter/s на возвращаемой структуре, и присвоить им новые значения, вызвав fx_set_parameter/s.

Обратите внимание, что структура, возвращаемая этой функцией, не будет содержать видимых членов и может быть изменена только с помощью одной из функций Filter/Effect.

Отказы функций

Функция вернет -1, если по какой-либо причине не удалось создать структуру; это может произойти, если указанное имя FX было неверным или FX не был загружен в пакет игры.

Чтобы эта функция работала правильно, убедитесь, что нужный вам FX загружен в ваш игровой пакет, как описано на странице Функции фильтров и эффектов:

Если эти условия не выполняются, т.е. вы не передаете строковый литерал и фильтр/эффект не присутствует ни в одной комнате, то функция не сможет загрузить этот фильтр/эффект и вернет -1.

Синтаксис:

fx_create(filter_or_effect_name);

Аргумент Тип Описание
filter_or_effect_name Строка The name of the filter/effect to create (get them from All Filter/Effect Types).

Возвращается:

FX Struct or -1 (on failure)

Пример:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

Приведенный выше код создает новую структуру FX из эффекта "_filter_tintfilter", который является фильтром "Colour Tint", находящимся в Room Editor. Он присваивает значение своему параметру "g_TintCol", который является цветом оттенка, а так как внутри это vec4, он принимает массив, содержащий четыре значения (соответствующие значениям красного, зеленого, синего и альфа). Затем FX-структура для оттенка применяется к FX-слою.
