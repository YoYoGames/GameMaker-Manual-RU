fx_get_parameter

Эта функция используется для получения значения параметра из FX-структуры. Вы указываете FX struct для чтения (как возвращено из fx_create() или layer_get_fx()) и имя параметра в виде строки, и функция возвращает его текущее значение. Это может быть реальное значение или массив, если параметр является вектором (это группа значений, например, vec4, который хранит цвет RGBA).

Найдите названия параметров для фильтра/эффекта на этой странице: Типы и параметры FX.

Синтаксис:

fx_get_parameter(filter_or_effect, parameter_name);

Аргумент Тип Описание
filter_or_effect FX Struct The FX struct to get the parameter from
parameter_name Строка The name of the parameter to get the value of (as a string)

Возвращается:

Real or Array

Пример:

var _fx_struct = layer_get_fx("TintEffect");
var _tint_colour = fx_get_parameter(_fx_struct, "g_TintCol");
show_debug_message("The currently active tint colour is: " + string(_tint_colour));

Приведенный выше код извлекает структуру FX из слоя, к которому, как предполагается, применен фильтр "Оттенок цвета", и получает значение параметра "g_TintCol", который возвращает массив, содержащий 4 значения для цвета оттенка (красный, зеленый, синий и альфа). Затем он печатает это значение в выходной журнал.
