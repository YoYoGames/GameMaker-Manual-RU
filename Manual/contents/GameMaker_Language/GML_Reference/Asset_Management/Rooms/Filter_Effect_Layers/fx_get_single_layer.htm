fx_get_single_layer

Эта функция возвращает, использует ли данный фильтр/эффект режим Single Layer (true) или нет (false). См. fx_set_single_layer() информацию о режиме Single Layer.

Вы указываете структуру FX (как возвращено из fx_create() или layer_get_fx()), и функция возвращает булево значение, которое говорит, включен или нет режим Single Layer для FX.

Синтаксис:

fx_get_single_layer(filter_or_effect);

АргументТип Описание
filter_or_effect The FX struct to read

Возвращается:

Boolean

Пример:

var _fx = layer_get_fx("Distort");

if (!fx_get_single_layer(_fx))
{
    fx_set_single_layer(_fx, true);
}

Приведенный выше код получает структуру FX из слоя под названием "Distort" и проверяет, включен ли для него режим Single Layer. Если нет, он включает режим Single Layer для этого FX.
