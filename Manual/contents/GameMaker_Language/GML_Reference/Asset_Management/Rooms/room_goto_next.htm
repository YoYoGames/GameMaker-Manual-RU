номер_следующий

С помощью этой функции вы можете заставить вашу игру перейти в следующую комнату, указанную в Менеджере комнат на момент компиляции игры. Если такой комнаты не существует, будет выдана ошибка, и игра будет принудительно закрыта. Обратите внимание, что комната не изменится до конца события, в котором была вызвана функция, поэтому любой код после ее вызова будет выполняться, если он находится в том же событии.

Синтаксис:

room_goto_next();

Возвращается:

Пример:

if room_exists(room_next(room))
{
    room_goto_next();
}

Приведенный выше код проверит, есть ли еще одна комната после текущей, и если да, то перейдет в эту комнату.
