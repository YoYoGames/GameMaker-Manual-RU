номер

Эта переменная содержит индекс комнаты для текущей комнаты, в которой запущена ваша игра. Это не переменная, доступная только для чтения, однако ее изменение не изменит значение индекса текущей комнаты, а скорее изменит комнату в соответствии с индексом, который вы установили в переменной. При этом следует соблюдать осторожность, так как если индекс, на который вы измените переменную, не соответствует действительности, игра выдаст ошибку и закроется. В целом, гораздо лучше использовать room_goto() для смены комнат.

Примечание: Идентификаторы помещений не основаны на их порядке в браузере активов или менеджере помещений, поэтому вам не следует напрямую указывать числовое значение. Вместо этого используйте константу комнаты для актива, на который вы хотите сослаться (это просто его имя), или получите его через функцию.

Синтаксис:

room;

Возвращается:

Room Asset

Пример:

if (room == rm_level1)
{
    audio_play_sound(snd_level1, 1, 1);
}

Приведенный выше код проверит, является ли текущая комната первым уровнем, и в этом случае воспроизведет музыку этого уровня.
