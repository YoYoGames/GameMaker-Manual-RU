layer_sequence_get_sequence

С помощью этой функции вы предоставляете идентификатор элемента последовательности - как возвращено функцией layer_sequence_create() или одной из функций элементов слоя - и она вернет структуру объекта последовательности. Эта функция позволяет обойти необходимость сначала получить структуру экземпляра последовательности и получить прямой доступ к данным последовательности. Подробнее о формате объекта последовательности struct вы можете узнать на этой странице.

Синтаксис:

layer_sequence_get_sequence(sequence_element_id)

АргументТип Описание
sequence_element_id The unique ID value of the sequence element to target

Возвращается:

Sequence Object Struct

Пример:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_sequence(_seq);
_struct.playbackSpeedType = spritespeed_framespersecond;
_struct.playbackSpeed = 30;

Приведенный выше код создает новый элемент последовательности, а затем извлекает его структуру данных последовательности. Затем он используется для изменения скорости воспроизведения и типа последовательности.
