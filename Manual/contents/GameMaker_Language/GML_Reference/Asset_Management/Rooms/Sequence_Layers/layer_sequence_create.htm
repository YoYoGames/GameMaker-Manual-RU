layer_sequence_create

С помощью этой функции вы можете создать экземпляр актива последовательности на заданном слое. Вы передаете ID слоя, который может быть строкой имени слоя - как определено в редакторе помещения - или уникальным ID слоя - как возвращается функцией layer_get_id(), а также позиции X и Y в комнате, в которых нужно создать последовательность, и, наконец, ID создаваемой последовательности. Идентификатор последовательности - это константа имени, которую вы определили в браузере активов для последовательности. Функция возвращает уникальный ID элемента последовательности, который затем может быть использован во всех дальнейших функциях слоя для последовательностей, или может быть использован для получения структуры экземпляра последовательности с помощью функции layer_sequence_get_instance().

Стоит отметить, что если последовательность содержит какие-либо объектные дорожки, их экземпляры будут созданы сразу после создания самого элемента последовательности, независимо от того, где их ключи активов расположены на Dope Sheet. Контроллер последовательности просто переключает видимость экземпляра, чтобы скрыть и показать его в зависимости от положения и длительности ключа актива дорожки, и не создает и не уничтожает его многократно. В результате экземпляры будут запускать свои события Create, когда создается элемент последовательности, а не когда начинаются их ключи актива.

Синтаксис:

layer_sequence_create(layer_id, x, y, sequence_id)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the instance layer to target
x Реально The x position in the room to create the sequence at
y Реально The y position in the room to create the sequence at
sequence_id Активы последовательности The sequence asset to use, as defined in the Asset Browser

Возвращается:

Sequence Element ID

Пример:

var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(_s);

Приведенный выше код создает новую последовательность на слое "Background", а затем приостанавливает ее.
