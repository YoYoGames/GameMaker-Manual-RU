layer_sequence_get_x

С помощью этой функции вы предоставляете идентификатор элемента последовательности - как возвращается функцией layer_sequence_create() или одной из функций элементов слоя - и она вернет текущую позицию X в игровой комнате для последовательности.

Синтаксис:

layer_sequence_get_x(sequence_element_id)

Аргумент Тип Описание
sequence_element_id Идентификатор элемента последовательности The unique ID value of the sequence element to target

Возвращается:

Real

Пример:

if layer_sequence_get_x(title_sequence) != room_width / 2
{
    layer_sequence_x(title_sequence, room_width / 2);
}

Приведенный выше код проверяет X-позицию идентификатора элемента последовательности, хранящегося в переменной "title_sequence", и если она не установлена на половину ширины комнаты, то она устанавливается на это значение.
