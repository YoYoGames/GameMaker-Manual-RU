layer_sequence_get_length

С помощью этой функции вы предоставляете идентификатор элемента последовательности - как возвращено функцией layer_sequence_create() или одной из функций элементов слоя - и она вернет длину последовательности. Это количество кадров, в течение которых будет выполняться последовательность.

Синтаксис:

layer_sequence_get_length(sequence_element_id)

АргументТип Описание
sequence_element_id The unique ID value of the sequence element to target

Возвращается:

Пример:

var _frames = layer_sequence_get_length(my_seq);
alarm[0] = frames;

Приведенный выше код извлекает количество кадров, которые будет выполнять последовательность, а затем использует это значение для установки сигнала тревоги.
