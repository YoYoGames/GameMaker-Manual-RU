layer_sequence_headpos

С помощью этой функции можно установить положение головки воспроизведения элемента последовательности на определенный кадр. Вы передаете идентификатор элемента последовательности, возвращенный layer_sequence_create() или одной из функций элемента слоя вместе с новой позицией, которую нужно установить. Обратите внимание, что позиция задается в кадрах, и если вы установите значение больше, чем общее количество кадров (или меньше 0), то фактическая конечная позиция головки воспроизведения будет зависеть от типа воспроизведения последовательности, который был выбран, следуя этим правилам:

  • Нет зацикливания: Головка воспроизведения будет прижиматься к концу цикла (или к началу, если заданное значение отрицательно).
  • Зацикливание: Головка воспроизведения обернется назад к началу последовательности плюс дополнительные кадры, например: если вы установите головку воспроизведения в последовательности из 30 кадров на позицию кадра 45, фактическая позиция головки воспроизведения будет равна 15. Если вы зададите отрицательное значение, то головка воспроизведения последовательности будет разворачиваться в обратном направлении.
  • Пинг-понговое зацикливание: Головка воспроизведения продвигается до конца, а затем возвращается на дополнительное количество кадров, превышающее общую длину кадра, меняя направление воспроизведения, например, если последовательность состоит из 30 кадров, а вы установили головку воспроизведения на 40, то конечное положение головки воспроизведения будет равно 20, а направление будет обратным (спуск вниз на 0): Если последовательность состоит из 30 кадров, а вы установили головку воспроизведения на 40, конечная позиция головки воспроизведения будет 20, а направление будет обратным (вниз до 0). Обратите внимание, что никогда не следует задавать отрицательное значение для пинг-понг воспроизведения последовательности, так как это может привести к неопределенным результатам.

Установка положения головки таким образом не остановит воспроизведение, и последовательность просто продолжится с нового положения головки воспроизведения, если она не приостановлена.

Синтаксис:

layer_sequence_headpos(sequence_element_id, position)

Аргумент Тип Описание
sequence_element_id Идентификатор элемента последовательности The unique ID value of the sequence element to target
position Реально The position within the sequence (in frames) to set the playhead position to (can be a decimal value)

Возвращается:

N/A

Пример:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sequence
    {
        layer_sequence_headpos(a[i], 0)
    }
}

Приведенный выше код получает идентификаторы всех элементов, назначенных слою экземпляра, на котором выполняется код. Затем код проверяет, являются ли какие-либо из возвращенных элементов активами последовательности, и если да, то их положение головки воспроизведения устанавливается на 0.
