layer_sequence_get_speedscale

С помощью этой функции вы предоставляете идентификатор элемента последовательности - как возвращается функцией layer_sequence_create() или одной из функций элемента слоя - и она вернет текущую шкалу скорости воспроизведения. Это значение множителя, используемое для замедления или ускорения скорости воспроизведения. Значение 1 является значением по умолчанию, значения меньше 1 означают, что воспроизведение замедляется, а значения больше 1 означают, что воспроизведение ускоряется.

Синтаксис:

layer_sequence_get_speedscale(sequence_element_id)

АргументТип Описание
sequence_element_id The unique ID value of the sequence element to target

Возвращается:

Пример:

if layer_sequence_get_speedscale(title_sequence) != 1
{
    layer_sequence_speedscale(title_sequence, 1);
}

Приведенный выше код проверяет текущую шкалу скорости вращения головки воспроизведения идентификатора элемента последовательности, хранящегося в переменной "title_sequence", и если она не установлена в 1, то она устанавливается в это значение.
