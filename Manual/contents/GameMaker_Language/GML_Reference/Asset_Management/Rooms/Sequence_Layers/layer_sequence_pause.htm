layer_sequence_pause

С помощью этой функции вы можете приостановить воспроизведение заданной последовательности. Вы передаете идентификатор элемента последовательности, возвращенный функцией layer_sequence_create() или одной из функций элементов слоя, и функция приостановит воспроизведение последовательности до тех пор, пока вы не начнете воспроизведение снова с помощью функции layer_sequence_play().

Синтаксис:

layer_sequence_pause(sequence_element_id)

Аргумент Тип Описание
sequence_element_id Идентификатор элемента последовательности The unique ID value of the sequence element to target

Возвращается:

N/A

Пример:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
if (global.Pause)
{
    layer_sequence_pause(_seq);
}

Приведенный выше код создает новую последовательность и сохраняет ее ID в локальной переменной. Затем он проверяет, не поставлена ли игра на паузу, и если поставлена, то приостанавливает воспроизведение последовательности.
