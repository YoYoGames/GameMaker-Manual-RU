layer_sequence_get_instance

С помощью этой функции вы предоставляете идентификатор элемента последовательности - как возвращено функцией layer_sequence_create() или одной из функций элементов слоя - и она вернет структуру экземпляра последовательности. Подробнее о формате struct экземпляра последовательности вы можете узнать на этой странице.

Синтаксис:

layer_sequence_get_instance(sequence_element_id)

Аргумент Тип Описание
sequence_element_id Идентификатор элемента последовательности The unique ID value of the sequence element to target

Возвращается:

Sequence Instance Struct

Пример:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_instance(_seq);
_struct.speedScale = 0.5;

Приведенный выше код создает новый элемент последовательности, а затем извлекает идентификатор экземпляра последовательности. Затем он используется для изменения свойства шкалы скорости воспроизведения экземпляра последовательности.
