layer_sequence_play

С помощью этой функции вы можете запустить воспроизведение заданной последовательности. Вы предоставляете идентификатор элемента последовательности, возвращенный layer_sequence_create() или одной из функций элементов слоя, и функция начнет воспроизведение последовательности, которую при необходимости можно приостановить с помощью функции layer_sequence_pause().

ВАЖНО! Если в вашей последовательности есть экземпляры, эти экземпляры не должны изменять свои переменные image_xscale / image_yscale / image_angle / x / y, поскольку они будут перезаписаны при обновлении последовательности на каждом шаге после начала воспроизведения.

Синтаксис:

layer_sequence_play(sequence_element_id)

Аргумент Тип Описание
sequence_element_id Идентификатор элемента последовательности The unique ID value of the sequence element to target

Возвращается:

N/A

Пример:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    var a = layer_get_all_elements(layer);
    for (var i = 0; i < array_length(a); i++)
    {
        if (layer_get_element_type(a[i]) == layerelementtype_sequence)
        {
            if (global.Pause)
            {
                layer_sequence_pause(a[i]);
            }
            else
            {
                layer_sequence_play(a[i]);
            }
        }
    }
}

Приведенный выше код проверяет, была ли игра поставлена на паузу или нет, когда нажата клавиша. Если игра поставлена на паузу, то выполняется цикл перебора всех элементов последовательности на текущем слое (слой вызывающего экземпляра) и приостанавливается их воспроизведение, а если игра не поставлена на паузу, то цикл снова запускает их воспроизведение.
