layer_sequence_yscale

С помощью этой функции вы можете установить масштаб Y для заданного элемента последовательности. Вы передаете идентификатор элемента последовательности, возвращаемый функцией layer_sequence_create() или одной из функций элемента слоя вместе с новым масштабом, который нужно установить по оси Y, и последовательность будет масштабирована на эту величину. Масштаб 1 означает отсутствие масштабирования (1:1), меньшие значения будут масштабироваться вниз (0,5, например, уменьшит ширину последовательности вдвое), большие значения будут масштабироваться вверх, а отрицательные значения будут переворачивать последовательность относительно ее начала и масштабировать ее, если только используемое значение не равно -1 (в этом случае последовательность просто переворачивается относительно ее начала без масштабирования).

Очень важно отметить, что применение неравномерного масштабирования (например, масштабирование оси X на 3, а оси Y на 2) к элементам последовательности, которые содержат любой экземпляр, использующий вращение, может вызвать проблемы с отрисовкой экземпляра, коллизии, выбраковку и многое другое. В общем, если ваша последовательность зависит от каких-либо свойств экземпляра, мы не рекомендуем использовать неравномерное масштабирование вместе с вращением экземпляра.

Также обратите внимание, что не следует изменять переменные image_xscale \ image_yscale \ image_angle \ x \ y для любых экземпляров в последовательности, использующей эту функцию, так как они будут перезаписаны, как только последовательность обновится после вызова функции.

Синтаксис:

layer_sequence_yscale(sequence_element_id, xscale)

АргументТип Описание
sequence_element_id The unique ID value of the sequence element to target
yscale The new Y axis scale value to apply to the sequence element

Возвращается:

Пример:

if seq_scale < 2
{
    seq_scale += 0.01;
    layer_sequence_xscale(my_seq, seq_scale);
    layer_sequence_yscale(my_seq, seq_scale);
}

Приведенный выше код проверяет значение, хранящееся в переменной seq_scale, и если оно меньше 2, то прибавляет его, а затем использует это значение для установки масштаба X и Y элемента последовательности, на который ссылается переменная my_seq.
