угол_последовательности_слоя

С помощью этой функции вы можете установить угол поворота для заданного элемента последовательности. Вы передаете идентификатор элемента последовательности, возвращенный функцией layer_sequence_create() или одной из функций элемента слоя вместе с новым углом, и последовательность будет повернута вокруг своего начала в новое положение. Углы в GameMaker рассчитываются с 0º вправо и идут против часовой стрелки - так 90º вверх, 180º влево и 270º вниз - и по умолчанию угол для последовательности равен 0º.

Обратите внимание, что не следует изменять переменные image_xscale \ image_yscale \ image_angle \ x \ y для любых экземпляров в последовательности, использующей эту функцию, так как они будут перезаписаны, как только последовательность обновится после вызова функции.

Синтаксис:

layer_sequence_angle(sequence_element_id, angle)

Аргумент Тип Описание
sequence_element_id Идентификатор элемента последовательности The unique ID value of the sequence element to target
angle Реально The new angle to rotate the sequence element to

Возвращается:

N/A

Пример:

if current_angle < 90
{
    current_angle += 1;
    layer_sequence_angle(my_seq, current_angle);
}

Приведенный выше код проверяет значение, хранящееся в переменной current_angle, и если оно меньше 90, то прибавляет к нему значение, а затем использует его для установки угла элемента последовательности, на который ссылается переменная my_seq.
