layer_sequence_get_headpos

С помощью этой функции вы передаете идентификатор элемента последовательности - как возвращается функцией layer_sequence_create() или одной из функций элементов слоя - и она вернет текущую позицию головки воспроизведения (текущий кадр, на котором находится головка воспроизведения).

Синтаксис:

layer_sequence_get_headpos(sequence_element_id)

АргументТип Описание
sequence_element_id The unique ID value of the sequence element to target

Возвращается:

Пример:

if layer_sequence_get_headpos(title_sequence) != 0
{
    layer_sequence_headpos(title_sequence, 0);
}

Приведенный выше код проверяет текущую позицию playhead ID элемента последовательности, хранящегося в переменной "title_sequence", и если она не установлена в 0, то она устанавливается в это значение.
