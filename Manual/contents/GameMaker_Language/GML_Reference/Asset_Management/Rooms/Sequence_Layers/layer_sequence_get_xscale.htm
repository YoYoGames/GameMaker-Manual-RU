layer_sequence_get_xscale

С помощью этой функции вы предоставляете идентификатор элемента последовательности - как возвращается функцией layer_sequence_create() или одной из функций элемента слоя - и она вернет текущий масштаб по оси X элемента последовательности в игровой комнате.

Синтаксис:

layer_sequence_get_xscale(sequence_element_id)

АргументТип Описание
sequence_element_id The unique ID value of the sequence element to target

Возвращается:

Пример:

var _xs = layer_sequence_get_xscale(title_sequence)
if (_xs < 1)
{
    _xs += 0.01;
    layer_sequence_xscale(title_sequence, _xs);
}

Приведенный выше код извлекает текущий масштаб по оси X элемента последовательности с идентификатором, хранящимся в переменной "title_sequence", и если он меньше 1, то к текущему масштабу X добавляется 0,01.
