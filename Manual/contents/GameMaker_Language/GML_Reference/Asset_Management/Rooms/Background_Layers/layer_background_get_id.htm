layer_background_get_id

Эта функция может использоваться для получения уникального значения ID элемента фона на слое. Вы передаете ID слоя (который вы получаете, когда используете имя слоя вместе со словом layer_get_id()), и функция вернет значение ID, связанное с элементом фона на слое. Обратите внимание, что эта функция предназначена для работы с фоновыми элементами, добавленными на сайте IDE, так как если вы добавите фон к слою через код с помощью функции layer_background_create()то она вернет уникальный ID для добавленного элемента фона.

Синтаксис:

layer_background_get_id(layer_id)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the layer to target

Возвращается:

Background Element ID

Пример:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_sprite(back_id, bck_Trees_Winter);

Приведенный выше код получит идентификатор слоя для слоя с именем "Background_trees", а затем использует его для получения идентификатора элемента фона на этом слое. Этот идентификатор затем используется для изменения элемента sprite.
