layer_background_sprite

С помощью этой функции вы можете установить индекс sprite фонового элемента. Вы передаете идентификатор фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а затем задаете индекс sprite, который будет использоваться. Изображение фонового элемента будет заменено новым sprite. Если вы зададите значение -1, то элементу не будет присвоен индекс sprite (но он будет существовать, и позже ему может быть присвоен индекс sprite ).

Эта функция также имеет псевдоним layer_background_change().

Синтаксис:

layer_background_sprite(background_element_id, sprite_index)

Аргумент Тип Описание
background_element_id Идентификатор фонового элемента The unique ID value of the background element to change
sprite_index Актив спрайта The sprite index of the sprite to use for the background element

Возвращается:

N/A

Пример:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
{
    layer_background_sprite(back_id, spr_Clouds);
}

Приведенный выше код получит идентификатор слоя с именем "Background_sky" и затем использует его для получения идентификатора элемента фона на этом слое. Этот ID затем используется для проверки sprite, назначенного элементу, и установки его на sprite "spr_Clouds", если он еще не установлен.
