layer_background_visible

С помощью этой функции можно переключить видимость фонового элемента. Вы предоставляете идентификатор фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а затем устанавливаете аргумент visible в значение true или false. При установке значения true элемент будет отображаться, а при установке значения false элемент не будет отображаться. Обратите внимание, что это зависит от видимости слоя, и даже если фоновый элемент отмечен как видимый, он не будет отрисован, если слой, на котором он находится, отмечен как невидимый.

Синтаксис:

layer_background_visible(background_element_id, visible)

Аргумент Тип Описание
background_element_id Идентификатор фонового элемента The unique ID value of the background element to change
visible Булево The visibility toggle, which can be true или false

Возвращается:

N/A

Пример:

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if layer_background_get_visible(back)
    {
        layer_background_visible(back, false);
    }
    else
    {
        layer_background_visible(back, true);
    }
}

Приведенный выше код проверяет нажатие кнопки мыши и, если оно обнаружено, переключает видимость фонового элемента, назначенного слою, на котором находится экземпляр, выполняющий код.
