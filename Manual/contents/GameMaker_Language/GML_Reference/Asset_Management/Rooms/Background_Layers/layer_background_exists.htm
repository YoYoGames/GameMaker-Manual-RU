layer_background_exists

С помощью этой функции можно проверить, существует ли элемент фона на данном слое. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность) и идентификатор фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()) и функция вернет либо true, если элемент существует, либо false, если его нет.

ПРИМЕЧАНИЕ: Эта функция работает в пределах текущей целевой комнаты - по умолчанию это комната, в которой вызывается функция, - которую можно задать с помощью функции layer_set_target_room().

Синтаксис:

layer_background_exists(layer_id, background_element_id)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the layer to target (or the layer name as a string)
background_element_id Идентификатор фонового элемента The unique ID value of the background element to check

Возвращается:

Boolean

Пример:

var lay_id = layer_get_id("Background_trees");
if (layer_background_exists(lay_id, global.TreesBackground))
{
    layer_background_destroy(lay_id, global.TreesBackground);
}

Приведенный выше код проверяет слой "Background_trees" на предмет существования данного фонового элемента, и если он существует, то он уничтожается (но не слой).
