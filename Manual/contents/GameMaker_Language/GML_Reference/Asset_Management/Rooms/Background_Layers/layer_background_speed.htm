layer_background_speed

Эта функция может быть использована для установки скорости анимированного sprite, назначенного фоновому элементу. Вы задаете идентификатор фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а затем задаете скорость анимации. Если IDE задал скорость анимации для sprite, которая рассчитывается как Frames Per Game Frame, то заданное значение скорости является множителем, причем 1 - это значение по умолчанию, а при установке значения 0,5 скорость анимации уменьшится вдвое, а при установке значения 2 - удвоится. Однако, если на сайте IDE задана скорость анимации для используемого элемента sprite, рассчитываемая как кадры в секунду, то заданное значение скорости является буквальным значением кадров в секунду, поэтому, например, если задать значение 5, то анимация будет повторяться по 5 кадров каждую секунду, пока идет игра. Если элемент фона не имеет вложенного sprite или используемый sprite не имеет вложенных изображений, эта функция не будет иметь эффекта.

Синтаксис:

layer_background_speed(background_element_id, speed);

Аргумент Тип Описание
background_element_id Идентификатор фонового элемента The unique ID value of the background element to change
speed Реально The speed multiplier/value to use for the element

Возвращается:

N/A

Пример:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_speed(back_id, 0.5);

Приведенный выше код получит идентификатор слоя для слоя с именем "Background_water", а затем использует его для получения идентификатора фонового элемента на этом слое. Этот идентификатор затем используется для изменения скорости изображения элемента.
