layer_background_xscale

Эта функция может использоваться для установки масштаба по оси x фонового элемента. Вы задаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а затем задаете значение масштаба. Заданное значение масштаба - это множитель, который будет использоваться для изменения способа отображения фонового элемента, При значении 0,5 элемент будет отображаться в половинном масштабе, а при значении 2 - в двойном. Обратите внимание, что отрицательные значения действительны, и они "переворачивают" элемент вокруг позиции (0,0), поэтому при масштабе x, равном -1, изображение будет отображаться в обратном порядке. изображение будет перевернуто.

Синтаксис:

layer_background_xscale(background_element_id, xscale);

АргументТип Описание
background_element_id The unique ID value of the background element to change
xscale The scale value to use (1 is no scaling)

Возвращается:

Пример:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_xscale(back_id, -1);

Приведенный выше код получит идентификатор слоя для слоя с именем "Background_water" и затем использует его для получения идентификатора элемента фона на этом слое. Этот идентификатор затем используется для изменения xscale элемента.
