layer_background_get_sprite

Эта функция может быть использована для получения текущего значения индекса sprite фонового элемента. Вы передаете идентификатор фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а функция вернет реальное значение, которое представляет собой отображаемый индекс sprite. Если элементу не присвоен индекс sprite, функция вернет -1.

Синтаксис:

layer_background_get_sprite(background_element_id);

АргументТип Описание
background_element_id The unique ID value of the background element to get the information from

Возвращается:

Sprite Asset or -1

Пример:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
{
    layer_background_sprite(back_id, spr_Clouds);
}

Приведенный выше код получит идентификатор слоя с именем "Background_sky" и затем использует его для получения идентификатора элемента фона на этом слое. Этот ID затем используется для проверки sprite, назначенного элементу, и установки его на sprite "spr_Clouds", если он еще не установлен.
