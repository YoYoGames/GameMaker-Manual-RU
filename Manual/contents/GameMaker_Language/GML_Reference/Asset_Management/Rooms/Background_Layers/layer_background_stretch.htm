layer_background_stretch

С помощью этой функции вы можете переключить фоновый элемент sprite на растяжение в соответствии с размерами комнаты или на сохранение разрешения 1:1. Вы предоставляете идентификатор фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а затем устанавливаете аргумент stretch в значение true или false. При установке значения true элемент sprite будет растянут в соответствии с размерами комнаты (либо увеличен, либо уменьшен в зависимости от размеров sprite и комнаты), а при установке значения false элемент sprite будет отображаться в своем родном разрешении.

Синтаксис:

layer_background_stretch(background_element_id, stretch)

Аргумент Тип Описание
background_element_id Идентификатор фонового элемента The unique ID value of the background element to change
stretch Булево The stretch toggle, which can be true или false

Возвращается:

N/A

Пример:

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if (layer_background_get_stretch(back))
    {
        layer_background_stretch(back, false);
    }
    else
    {
        layer_background_stretch(back, true);
    }
}

Приведенный выше код проверяет нажатие кнопки мыши и при обнаружении такового переключает растяжение фонового элемента sprite, назначенного слою, на котором находится экземпляр, выполняющий код.
