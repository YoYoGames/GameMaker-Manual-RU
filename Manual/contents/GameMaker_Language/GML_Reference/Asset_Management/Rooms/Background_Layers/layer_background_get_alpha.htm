layer_background_get_alpha

Эта функция может быть использована для получения значения альфа-фактора фонового элемента. Вы указываете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), и функция вернет значение от 0 (полностью прозрачный) до 1 (полностью непрозрачный).

Синтаксис:

layer_background_get_alpha(background_element_id);

АргументТип Описание
background_element_id The unique ID value of the background element to get the information from

Возвращается:

(from 0 to 1)

Пример:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_alpha(back_id) < 0.1)
{
    layer_background_destroy(back_id);
}

Приведенный выше код получит идентификатор слоя с именем "Background_sky" и затем использует его для получения идентификатора элемента фона на этом слое. Этот идентификатор затем используется для проверки альфа-элемента, и если он меньше 0,1, то элемент слоя уничтожается.
