layer_background_blend

Эта функция управляет смешиванием (или "тонированием") фона sprite. Значение по умолчанию равно -1 (что представляет собой константу c_white, которая также может быть использована). Любое другое значение (включая внутренние цветовые константы, такие как c_red, или c_aqua) будет смешивать указанный цвет с исходным sprite. Вы задаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а затем задаете цвет смешивания. Ниже показан пример sprite, который был смешан с различными цветами:

Обратите внимание, что на платформе HTML5 следует ограничить количество смешиваний (если только не используется WebGL), так как каждый блендинг sprite должен кэшироваться отдельно, поэтому большое количество блендингов sprites может негативно сказаться на производительности (вы также можете установить размер кэша с помощью функции sprite_set_cache_size()).

Синтаксис:

layer_background_blend(background_element_id, blend);

АргументТип Описание
background_element_id The unique ID value of the background element to change
blend The colour to blend with the background sprite (default is c_white)

Возвращается:

Пример:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_blend(back_id, c_aqua);

Приведенный выше код получает значение ID фона, назначенного слою "Background_sky", и затем оттеняет его цветом.
