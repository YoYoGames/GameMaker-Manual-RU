layer_background_get_stretch

Эту функцию можно использовать для получения растянутого состояния фонового элемента sprite. Вы передаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), и функция вернет либо true, если элемент sprite в настоящее время растянут, чтобы соответствовать комнате, либо false, если это не так.

Синтаксис:

layer_background_get_stretch(background_element_id);

АргументТип Описание
background_element_id The unique ID value of the background element to get the information from

Возвращается:

Пример:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_stretch(back_id))
{
    layer_background_stretch(back_id, false);
}

Приведенный выше код получит идентификатор слоя для слоя с именем "Background_sky" и затем использует его для получения идентификатора элемента фона на этом слое. Этот идентификатор затем используется для проверки, будет ли элемент sprite растянут, чтобы соответствовать комнате или нет, и если он будет растянут, то это свойство будет установлено в false.
