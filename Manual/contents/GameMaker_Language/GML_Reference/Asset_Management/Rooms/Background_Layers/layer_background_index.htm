индекс_фона_слоя

Эта функция может быть использована для установки индекса изображения фона sprite, который имеет несколько подизображений. Вы задаете ID элемента фона (который вы получаете при создании элемента фона с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а затем задаете индекс изображения, который будет использоваться. Если вы зададите значение вне диапазона вложенных изображений, то индекс изображения будет зацикливаться.

Синтаксис:

layer_background_index(background_element_id, image_index);

Аргумент Тип Описание
background_element_id Идентификатор фонового элемента The unique ID value of the background element to change
index Реально The image index to use for the background

Возвращается:

N/A

Пример:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_index(back_id, 1);

Приведенный выше код получит идентификатор слоя для слоя с именем "Background_trees" и затем использует его для получения идентификатора элемента фона на этом слое. Этот идентификатор затем используется для изменения индекса изображения элемента.
