layer_background_yscale

Эта функция может использоваться для установки масштаба по оси y фонового элемента. Вы задаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а затем задаете значение масштаба. Заданное значение масштаба - это множитель, который будет использоваться для изменения отображения фонового элемента. При значении 0,5 элемент будет отображаться в половинном масштабе, а при значении 2 - в двойном. Обратите внимание, что отрицательные значения действительны, и они "переворачивают" элемент вокруг позиции (0,0), поэтому при масштабе x, равном -1, изображение будет перевернуто.

Синтаксис:

layer_background_yscale(background_element_id, yscale);

Аргумент Тип Описание
background_element_id Идентификатор фонового элемента The unique ID value of the background element to change
yscale Реально The speed multiplier/value to use for the element

Возвращается:

N/A

Пример:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_yscale(back_id, -1);

Приведенный выше код получит идентификатор слоя для слоя с именем "Background_water" и затем использует его для получения идентификатора элемента фона на этом слое. Этот идентификатор затем используется для изменения шкалы yscale элемента.
