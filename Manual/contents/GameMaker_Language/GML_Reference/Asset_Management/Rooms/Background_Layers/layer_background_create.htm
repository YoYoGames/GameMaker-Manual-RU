layer_background_create

С помощью этой функции вы можете назначить ресурс sprite слою, который будет использоваться в качестве фона в вашем проекте. Вы передаете ID слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность) и индекс sprite (который будет именем sprite, как показано в Asset Browser), и он будет добавлен в слой. Функция возвращает уникальное значение ID для элемента, которое затем может быть использовано в дальнейших функциях слоя для фона.

Синтаксис:

layer_background_create(layer_id, sprite)

Аргумент Тип Описание
layer_id Строка или идентификатор слоя The unique ID value of the layer to target (or the layer name as a string)
sprite Актив спрайта The sprite index to be used

Возвращается:

Background Element ID

Пример:

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

Приведенный выше код создает новый слой, а затем добавляет к нему новый фоновый элемент, устанавливая sprite в качестве используемого фонового изображения.
