layer_background_htiled

С помощью этой функции можно изменить, должен ли данный фоновый элемент на слое быть горизонтально черепичным или нет. Вы задаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а затем задаете значение tiling. Если задать значение true, то элемент будет отображаться горизонтально по всей комнате, а если задать значение false, то элемент не будет отображаться горизонтально. Эта функция предназначена только для 2D-проектов и не будет работать правильно при использовании проекции 3D-камеры.

Синтаксис:

layer_background_htiled(background_element_id, tiled);

Аргумент Тип Описание
background_element_id Идентификатор фонового элемента The unique ID value of the background element to change
tiled Булево The horizontal tiling toggle, which can be true или false

Возвращается:

N/A

Пример:

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
    layer_background_htiled(back, true);
    layer_background_vtiled(back, true);
}

Приведенный выше код проверит элемент фона, назначенный слою, на котором находится экземпляр, выполняющий код, и если он не имеет плитки в любом направлении, то установит его на плитку по вертикали и горизонтали.
