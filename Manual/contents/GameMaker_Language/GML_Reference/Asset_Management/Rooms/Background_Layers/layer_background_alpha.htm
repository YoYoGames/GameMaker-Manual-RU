layer_background_alpha

Эта функция управляет альфа (прозрачностью) фона sprite. Вы задаете ID элемента фона (который вы получаете при создании элемента фона с помощью функции layer_background_create() или когда вы используете функцию layer_background_get_id()), а затем задаете используемое значение альфа. Альфа может быть в диапазоне от 0 (полностью прозрачный) до 1 (полностью непрозрачный), по умолчанию значение альфа для фонового элемента равно 1. элемента по умолчанию равно 1. Обратите внимание, что если слой, которому был назначен фоновый элемент, не виден, или сам элемент был сделан невидимым, вы не увидите никакой разницы с помощью этой функции, пока слой или элемент не станут видимыми. снова.

Синтаксис:

layer_background_alpha(background_element_id, alpha);

АргументТип Описание
background_element_id The unique ID value of the background element to change
alpha The alpha for background sprite, from 0 to 1 (default is 1)

Возвращается:

Пример:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_alpha(back_id, random(1));

Приведенный выше код получает значение ID фона, назначенного слою "Background_sky", а затем устанавливает его альфа-фактор на случайное значение между 0 и 1.
