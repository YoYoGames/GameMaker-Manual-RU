ширина_комнаты

В этой переменной хранится ширина текущей комнаты в пикселях. Вы можете изменить эту переменную, чтобы изменить ширину комнаты в любое время.

Синтаксис:

room_width;

Возвращается:

Пример:

if bbox_right > room_width
{
    x += room_width - bbox_right;
}

Приведенный выше код проверяет, не превышает ли граница текущего экземпляра sprite ширину комнаты, и если да, то перемещает экземпляр вверх.
