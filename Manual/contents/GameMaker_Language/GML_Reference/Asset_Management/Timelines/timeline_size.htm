размер временной шкалы

С помощью этой функции вы можете получить общее количество активных моментов для временной шкалы (активным считается момент, к которому добавлен код или GML Visual). Это может быть удобно при создании динамических временных шкал с помощью функций timeline_moment_add_script() и timeline_moment_clear() так как можно проверить, есть ли в данной временной шкале нужное количество активных моментов или их вообще нет.

Синтаксис:

timeline_size(ind);

АргументТип Описание
ind The index of the timeline get the information from.

Возвращается:

Пример:

if timeline_size(global.tl) == 0
{
    timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
}

Приведенный выше код проверяет заданный размер временной шкалы, и если он возвращает 0 (т.е. временная шкала не имеет активных моментов), то добавляет функцию скрипта, которая будет использоваться в случайный момент временной шкалы, проиндексированный в глобальной переменной "tl".
