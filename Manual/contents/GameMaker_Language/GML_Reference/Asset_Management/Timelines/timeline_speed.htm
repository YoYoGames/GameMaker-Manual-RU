timeline_speed

Обычно на каждом шаге позиция на линии времени увеличивается на 1, однако вы можете изменить это значение, установив переменную в другое значение. Вы можете использовать реальные числа (например, 0,5 или 2,4), и если значение больше единицы, то в течение одного временного шага может произойти несколько моментов (все они будут выполняться в том же порядке, который определен для временной линии, поэтому ни одно действие не будет пропущено).

Синтаксис:

timeline_speed

Возвращается:

(single precision floating point value)

Пример:

if !timeline_running
{
    timeline_running = true;
    timeline_speed = 0.5;
}

Приведенный выше код проверит, запущена ли в экземпляре связанная с ним временная линия, и если нет, то запустит временную линию на половинной скорости.
