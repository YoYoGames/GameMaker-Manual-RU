таймлайн_макс_момент

Эта функция возвращает значение момента, в который временная шкала выполняет свое последнее действие. Так, если у вас есть временная шкала с 8 различными действиями, расположенными на расстоянии 20 мгновений друг от друга, эта функция вернет значение момента последнего действия в этой временной шкале, которое будет равно 160. Эта функция - хороший способ проверить, прошла ли временная шкала последний активный момент, поскольку позиция временной шкалы будет продолжать увеличиваться на каждом шаге игры, независимо от того, есть ли дальнейшие действия или нет.

Синтаксис:

timeline_max_moment(ind);

АргументТип Описание
ind The index of the timeline to get the last moment of.

Возвращается:

Пример:

if timeline_position > timeline_max_moment(timeline_index)
{
    timeline_running = false;
}

Приведенный выше код проверит текущее положение временной шкалы относительно максимального активного момента, и если оно больше, то временная шкала будет остановлена.
