timeline_delete

С помощью этой функции вы можете удалить любую временную линию из вашей игры. Если эта временная линия была создана в GameMaker Asset Browser, обратите внимание, что она полностью удаляется из игры, и попытка снова вызвать индексированную временную линию после использования этой функции приведет к ошибке. Если временная линия была создана динамически с помощью функции timeline_add() то эту функцию следует использовать в тот момент, когда таймлайн больше не нужен, чтобы предотвратить утечку памяти, которая может замедлить (и в конечном итоге привести к краху) вашу игру.

Синтаксис:

timeline_delete(ind);

АргументТип Описание
ind The index of the time line to delete.

Возвращается:

Пример:

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

Приведенный выше код проверяет, существует ли временная линия и индексируется ли она в глобальной переменной "tl", и если да, то удаляет эту временную линию.
