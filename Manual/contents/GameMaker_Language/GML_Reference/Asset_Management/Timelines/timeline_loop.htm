временная_петля

Эта переменная возвращает, зацикливается ли линия времени (true) или нет (false). Вы можете изменить эту переменную, чтобы включить или выключить зацикливание, и она работает с отрицательной скоростью линии времени (если положение линии времени опустится ниже 0, она начнет снова с последнего определенного момента).

Синтаксис:

timeline_loop;

Возвращается:

Пример:

if !timeline_loop
{
    timeline_loop = true;
}

Приведенный выше код проверяет, будет ли временная линия, назначенная в данный момент экземпляру, зацикливаться или нет, и если нет, то устанавливает значение true.
