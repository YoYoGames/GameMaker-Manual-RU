таймлайн_момент_добавить_скрипт

С помощью этой функции вы можете динамически добавлять скрипт-функцию в Timelines в любой заданный "момент" в пределах этой временной линии, где "момент" эквивалентен одному игровому тику (или шагу). Таким образом, вы можете создать новую временную линию с помощью функции timeline_add() и добавить различные модели поведения в любой момент, или просто модифицировать ранее созданный ресурс временной линии с помощью новых моделей поведения. Обратите внимание, что при использовании этой функции не требуется никаких дополнительных аргументов, и если вы используете ее для изменения актива Timeline, присутствующего в Asset Browser, то все экземпляры, использующие эту временную линию, будут затронуты, и изменение будет длиться до конца игры (вызов game_restart() также не сбросит изменение).

Синтаксис:

timeline_moment_add_script(ind, step, script);

Аргумент Тип Описание
ind Временной актив The index of the timeline to add a moment to.
step Реально The moment (step) to add to.
script Функция сценария The index of the script function to add into the moment.

Возвращается:

N/A

Пример:

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
{
    timeline_moment_add_script(global.tl, i, choose(Attack_1, Attack_2, Attack_3);
    i += room_speed * 60;
}

Приведенный выше код создаст новую временную линию и сохранит ее индекс в переменной "global.tl". Затем он добавит на линию времени три сценария, выбранные случайным образом с интервалом в одну минуту.
