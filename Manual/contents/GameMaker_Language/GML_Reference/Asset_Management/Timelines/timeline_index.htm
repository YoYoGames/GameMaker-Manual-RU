timeline_index

Эта переменная содержит индекс временной линии, связанной с данным экземпляром. Вы можете установить его на определенную временную линию, чтобы использовать ее, или установить его на -1, чтобы прекратить использование временной линии для данного экземпляра (если для экземпляра не определена временная линия, возвращается -1). Обратите внимание, что это не запускает временную линию - для этого используйте переменную timeline_running.

Синтаксис:

timeline_index

Возвращается:

Timeline Asset

Пример:

if timeline_exists(global.tl)
{
    timeline_index = global.tl;
}

Приведенный выше код присвоит экземпляру, выполняющему код, временную линию, индексированную в переменной "global.tl", если такая временная линия существует.
