path_get_closed

Эта функция может быть использована для возврата информации о том, отмечен ли путь как закрытый (true) или открытый (false), т.е. зацикливается ли путь или имеет ли он определенное начало и конец.

Синтаксис:

path_get_closed(index);

Аргумент Тип Описание
index Путь Актива The index of the path to check.

Возвращается:

Boolean

Пример:

state = path_get_closed(pth_Patrol);

Это установит "state" либо в true, либо в false в зависимости от закрытого состояния пути, указанного в "pth_Patrol".
