длина_пути

Вы можете использовать эту функцию для получения точной длины пути в пикселях. Это не приблизительная длина от точки до точки, а точная длина вдоль формы пути, даже если путь гладкий с высокой точностью кривизны.

Синтаксис:

path_get_length(index);

АргументТип Описание
index The index of the path to measure.

Возвращается:

Пример:

path_len = path_get_length(pth_AI);

Это установит "path_len" в общую длину пути, индексированного в "pth_AI", в пикселях.
