путь_действия

Эта переменная может быть использована для получения или изменения реакции экземпляра, когда он достигает конца текущего пути. Обычно вы устанавливаете это значение, когда начинаете путь, используя path_start() но вы можете захотеть изменить это поведение в зависимости от любого количества событий в вашей игре. Доступные значения выражаются с помощью следующих констант:

Конец пути Действие Константа
Постоянная Описание
path_action_stop Остановить путь
path_action_restart Продолжить из начальной позиции, прыгая на старт, если путь не закрыт
path_action_continue Начните путь заново с текущей позиции
path_action_reverse Изменить скорость движения (проложить путь в обратном направлении)

Синтаксис:

path_endaction;

Возвращается:

Path End Action Constant

Пример:

if (path_endaction == path_action_stop)
{
    path_endaction = path_action_reverse;
}

Приведенный выше код проверит действие конца пути, и если оно установлено на stop, то действие конца будет изменено на reverse.
