Переменные пути

Следующие переменные встроены во все экземпляры при создании и могут быть использованы для получения информации о пути или для изменения определенных свойств:

  • path_index
  • path_position
  • path_positionprevious
  • path_speed
  • path_scale
  • path_orientation
  • path_endaction

Обратите внимание, что дополнительную информацию о переменных экземпляра вы можете найти здесь: Переменные экземпляра.
