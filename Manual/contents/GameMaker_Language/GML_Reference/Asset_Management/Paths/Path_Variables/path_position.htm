path_position

Эта функция может быть использована для получения или установки положения экземпляра вдоль пути. Значение нормировано от 0 до 1, поэтому если вы установите его, например, на 0,5, экземпляр будет перемещен точно на середину пути.

Синтаксис:

path_position

Возвращается:

Пример:

path_position = random(1);

Приведенный выше код устанавливает экземпляр в случайную позицию в любом месте текущего пути.
