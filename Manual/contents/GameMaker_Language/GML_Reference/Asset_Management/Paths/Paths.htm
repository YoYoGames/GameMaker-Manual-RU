Пути

ВGameMaker имеется обширная коллекция функций и встроенных переменных для использования путей в вашей игре. В этом разделе рассматриваются основные из них, которые используются при работе с путями, ранее созданными в редакторе путей, но существуют и другие функции, позволяющие создавать собственные пути с помощью кода и манипулировать ими различными способами.

Существуют следующие основные функции, работающие с путями:

Paths

GameMaker has an extensive collection of functions and built-in variables for using paths within your game. This section deals with the basic ones that are used when dealing with paths that have previously been created in The Path Editor, but further functions exist that permit you to create your own paths using code, and to manipulate them in various ways.
