конец_пути

Вызов этой функции завершает текущий путь, по которому следует экземпляр, заданный при вызове функции path_start() была вызвана...

Синтаксис:

path_end();

Возвращается:

Пример:

if place_meeting(x, y, obj_Blocker)
{
    path_end();
}

Приведенный выше код завершит текущий путь, если экземпляр обнаружит столкновение с любым экземпляром данного объекта.
