путь_существует

Эта функция возвращает, существует ли путь с заданным индексом или нет. Обратите внимание, что если вы проверяете существование пути через переменную, которая еще не была объявлена, это приведет к ошибке.

Синтаксис:

path_exists(index);

Аргумент Тип Описание
index Путь Актива The index of the path to check for.

Возвращается:

Boolean

Пример:

if (path_exists(my_path))
{
    path_delete(my_path);
}

Этот код проверяет, хранится ли в данной переменной индекс пути, и если да, то путь удаляется из памяти.
