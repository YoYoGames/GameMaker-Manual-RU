обратный путь

С помощью этой функции можно изменить порядок нумерации отдельных точек пути, например, если путь состоит из 5 точек, то точка 0 станет точкой 4, точка 1 станет точкой 3, а точка 2 останется неизменной. Фактическое положение точек остается неизменным, меняется только порядок их обработки. Эта функция изменяет фактический актив пути, поэтому она будет постоянно влиять на то, как этот путь используется всеми экземплярами в игре с момента ее использования и до конца игры. функция используется до конца игры. Если это не то, что вам нужно, то вам следует использовать функцию типа path_duplicate(), чтобы сначала создать копию пути, а затем вызвать эту функцию на дублированном активе (не забудьте вызвать path_delete() на активе, когда он больше не нужен).

Синтаксис:

path_reverse(index);

АргументТип Описание
index The index of the path to change.

Возвращается:

Пример:

path_reverse(pth_AI);

Это изменит порядок обработки всех точек на данном пути.
