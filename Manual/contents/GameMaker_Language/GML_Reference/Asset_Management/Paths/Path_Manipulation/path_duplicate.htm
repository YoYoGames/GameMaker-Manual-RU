path_duplicate

Эта функция принимает путь и копирует его в новый путь. Новый путь создается в процессе, и его индекс возвращается для использования во всех дальнейших вызовах для использования этого нового пути.

Синтаксис:

path_duplicate(index);

АргументТип Описание
index The index of the existing path to duplicate.

Возвращается:

Path Asset

Пример:

mypath = path_duplicate(choose(pth_1, pth_2, pth_3, pth_4));

Приведенный выше код выбирает один из четырех ресурсов пути и дублирует его, сохраняя индекс нового пути в переменной "mypath".
