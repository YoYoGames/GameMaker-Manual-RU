путь_удаления

Вы можете использовать этот код для удаления пути из памяти. Если этот путь был создан динамически с помощью функции path_add()то переменная, содержащая индекс пути, больше не будет действительна для доступа к пути, поскольку он больше не существует, а если путь был создан с помощью редактора путей, то к нему больше нельзя будет обратиться во всей игре, поскольку вы удаляете его навсегда.

Синтаксис:

path_delete(index);

АргументТип Описание
index The index of the path to delete.

Возвращается:

Пример:

var t_path = path_add();
if mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1)
{
    path_assign(mypath, t_path);
}
path_delete(t_path);

Приведенный выше код создаст путь и сохранит его индекс в локальной переменной. Этот путь затем используется для хранения mp_grid_path() сгенерированный путь, который, если ему удастся найти путь к цели, будет присвоен пути, индексированному в "mypath". Наконец, "t_path" удаляется.
