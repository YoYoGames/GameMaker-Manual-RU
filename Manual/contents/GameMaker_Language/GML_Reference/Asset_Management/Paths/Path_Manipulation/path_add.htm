путь_добавления

С помощью этой функции вы можете создать путь в GameMaker без использования редактора путей. Эта функция возвращает индекс пути, который должен быть сохранен в переменной и в дальнейшем использоваться в качестве ссылки для этого пути. Обратите внимание, что созданный путь пуст, то есть в нем нет определенных точек, поэтому вам придется использовать другие доступные функции для добавления точек к пути или использовать сетки MP для генерации пути. После того, как вы закончили использовать путь или хотите создать новый и сохранить его индекс в той же переменной, сначала удалите старый путь с помощью функции path_delete чтобы предотвратить утечку памяти, которая может привести к краху игры.

Синтаксис:

path_add();

Возвращается:

Path Asset

Пример:

global.newpath = path_add();

Это создаст новый путь и присвоит его индекс global.newpath.
