путь_приложения

С помощью этого кода вы можете добавить один путь к другому, эффективно соединив их вместе. Для того чтобы это сработало, оба пути должны существовать ранее. В общем случае вы захотите использовать эту функцию для путей, созданных с помощью path_add(), так как если вы используете ее на активе пути, она будет постоянно влиять на путь для экземпляров в игре с момента использования функции и до конца игры.

ПРИМЕЧАНИЕ Это не удалит добавляемый путь из игры. Он все еще там, только теперь в нем нет точек, и если вы больше не хотите его использовать, вам следует удалить его с помощью функции path_delete().

Синтаксис:

path_append(index, path);

АргументТип Описание
index The new index of the path.
path The index of the path that will be appended to 'index'.

Возвращается:

N/A

Пример:

path_append(mypath, mppath);
path_delete(mppath);

Приведенный выше код добавляет данные пути, указанные в переменной "mppath", к пути, указанному в переменной "mypath", а затем удаляет "mppath" из памяти.
