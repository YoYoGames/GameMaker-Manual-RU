путь_удаления_точки

С помощью этой функции можно удалить точку из указанного пути. Точки пути нумеруются от 0, поэтому первая точка всегда будет 0-й, а последняя точка пути будет path_get_number() - 1. Если вы вызовете эту функцию на активе пути, то удаление точки будет постоянным, и все экземпляры, назначенные этому пути в будущем, будут лишены удаленной точки. Если это не то, что вам нужно, то следует использовать функцию типа path_duplicate() для создания копии пути, а затем вызвать эту функцию на дублированном активе (не забудьте вызвать path_delete() на активе, когда он больше не нужен).

Синтаксис:

path_delete_point(ind, n);

Аргумент Тип Описание
index Путь Актива The index of the path to delete a point from.
n Реально The defining point to delete.

Возвращается:

N/A

Пример:

var _col = true;
while (_col)
{
    var _num = path_get_number(mypath) - 1;
    if (_num > 2)
    {
        var _x1 = path_get_point_x(my_path, _num);
        var _y1 = path_get_point_y(my_path, _num);
        var _x2 = path_get_point_x(my_path, _num -1);
        var _y2 = path_get_point_y(my_path, _num -1);
        if (collision_line(_x1, _y1, _x2, _y2, obj_Wall, true, false))
        {
            path_delete_point(mypath, _num);
            _num -= 1;
        }
        else _col = false;
    }
    else _col = false;
}

Приведенный выше код генерирует цикл, который проверяет точки на обратном пути от конца на предмет столкновения с заданным объектом. Если столкновение найдено, точка пути удаляется до тех пор, пока не произойдет столкновение или путь не станет длиной всего 2 точки.
