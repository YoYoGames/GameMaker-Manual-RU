точка_изменения_пути

С помощью этой функции можно изменить положение и/или коэффициент скорости любой точки, ранее определенной для пути (путь может быть создан в редакторе путей или в коде с помощью функции path_add()). Если функция используется для пути из браузера активов, обратите внимание, что она изменит фактический актив, а значит, навсегда повлияет на то, как этот путь используется всеми экземплярами в игре с момента использования функции и до конца игры. Если это не то, что вам нужно, используйте функцию типа path_duplicate(), чтобы сначала создать копию пути, а затем вызовите эту функцию на дублированном активе (не забудьте вызвать path_delete() на активе, когда он больше не нужен).

Синтаксис:

path_change_point(index, n, x, y, speed);

Аргумент Тип Описание
index Путь Актива The index of the path to change a point of.
n Реально The defining point to change the attributes of.
x Реально The new x coordinate (relative to the path) of the point.
y Реально The new y coordinate (relative to the path) of the point.
speed Реально The new speed factor of the point.

Возвращается:

N/A

Пример:

for (var i = 0; i < path_get_number(path0); ++i;)
{
    var px = path_get_point_x(pth_Patrol, i) + 64 - random(128);
    var py = path_get_point_y(pth_Patrol, i) + 64 - random(128);
    path_change_point(pth_Patrol, i, px, py, 100);
}

Приведенный выше код перебирает все точки пути, индексированные как "path0", и переставляет все точки в случайное положение в области размером 128x128 пикселей.
