path_insert_point

С помощью этой функции можно вставить новую точку в контур (контур может быть создан в редакторе контуров или в коде с помощью функции path_add()). Точка будет добавлена в контур перед точкой "n", указанной в функции.

Синтаксис:

path_insert_point(index, n, x, y, speed);

Аргумент Тип Описание
index Путь Актива The index of the path to insert the point into.
n Реально The defining point to insert the new point BEFORE.
x Реально The x coordinate (relative to the path) of the new point.
y Реально The y coordinate (relative to the path) of the new point.
speed Реально The speed factor of the point.

Возвращается:

N/A

Пример:

path_insert_point(mypath, 0, 50, 50, 100);

Это вставит точку в самом начале пути, индексированного в переменной "mypath", в точке (50,50), с коэффициентом скорости 100%.
