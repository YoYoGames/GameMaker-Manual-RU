путь_флип

Эта функция берет все точки пути и переворачивает их вдоль горизонтальной оси. Если это не то, что вам нужно, то используйте функцию path_duplicate() для создания копии пути, а затем вызовите эту функцию на дублированном активе(не забудьте вызвать path_delete() на активе, когда он больше не нужен).

Синтаксис:

path_flip(index);

АргументТип Описание
index The index of the path to flip.

Возвращается:

Пример:

path_flip(mypath);

Это перевернет путь, указанный в переменной "mypath", вдоль горизонтальной оси.
