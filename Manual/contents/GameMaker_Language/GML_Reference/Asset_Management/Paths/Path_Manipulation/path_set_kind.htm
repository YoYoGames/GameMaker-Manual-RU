вид_пути

С помощью этой функции можно задать вид пути, которым должен быть указанный ресурс. Это может быть либо прямой путь (значение 0), либо сглаженный путь (значение 1), и в этом случае необходимо учитывать точность пути (точность можно установить с помощью функции может быть задана с помощью path_set_precision()). Эта функция изменяет фактический ресурс пути, и поэтому будет постоянно влиять на то, как путь используется всеми экземплярами в игре с момента Если это не то, что вам нужно, то вам следует использовать функцию типа path_duplicate() для создания копии пути, а затем вызвать эту функцию на дублированном активе. на дублированном активе (не забудьте вызвать path_delete() на активе, когда он больше не нужен).

Синтаксис:

path_set_kind(index, val);

АргументТип Описание
index The index of the path to change.
val The kind of the path, 0 for straight or 1 for smooth.

Возвращается:

Пример:

path_set_kind(pth_Patrol, true);

Это установит заданный путь как "гладкий" путь.
