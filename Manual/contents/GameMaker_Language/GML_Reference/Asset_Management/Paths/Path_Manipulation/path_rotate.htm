path_rotate

Вы можете использовать эту функцию для поворота заданного пути вокруг его центра. Помните, что в GameMaker (если вы не используете физику) углы рассчитываются против часовой стрелки, поэтому поворот пути на 90 градусов повернет его влево. Эта функция изменяет фактический ресурс пути, поэтому она будет постоянно влиять на то, как этот путь используется всеми экземплярами в игре с момента использования функции и до конца игры. Если это не то, что вам нужно, то следует использовать функцию типа path_duplicate(), чтобы сначала создать копию пути, а затем вызвать эту функцию на дублированном активе (не забудьте вызвать path_delete() на активе, когда он больше не нужен).

ПРИМЕЧАНИЕ: После назначения пути экземпляру с помощью функции path_start() можно использовать переменную path_orientation для изменения поворота пути.

Синтаксис:

path_rotate(index, angle);

Аргумент Тип Описание
index Путь Актива The index of the path to flip.
angle Реально The angle of rotation degrees.

Возвращается:

N/A

Пример:

path_rotate(pth_Patrol, 90);

Это повернет заданную величину на 90 градусов против часовой стрелки.
