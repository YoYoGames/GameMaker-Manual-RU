путь_очистить_точки

С помощью этой функции можно удалить все определения точек из контура, создавая "пустой" контур. Это не удаляет путь, для этого следует использовать функцию path_delete()Однако следует отметить, что эта функция изменяет фактический актив пути и, следовательно, будет постоянно влиять на то, как этот путь используется всеми экземплярами в игре с момента использования функции и до конца игры. Если это не то, что вам нужно, то вам следует использовать функцию типа path_duplicate() для создания копии пути, а затем вызвать эту функцию на дублированном активе (не забудьте вызвать path_delete() на активе, когда он больше не нужен).

Синтаксис:

path_clear_points(index);

АргументТип Описание
index The index of the path to clear.

Возвращается:

Пример:

if path_get_number(mypath) > 0
{
    path_clear_points(mypath);
}

Приведенный выше код проверяет, есть ли точки на пути, проиндексированном в переменной "mypath", и если есть, то очищает путь.
