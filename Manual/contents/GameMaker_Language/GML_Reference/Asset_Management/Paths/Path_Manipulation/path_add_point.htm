точка_добавления_пути

С помощью этой функции можно добавить точку к заданному пути и установить коэффициент ее скорости. Эта точка добавляется в конец пути, независимо от положения точки, а коэффициент равен проценту от path_speed скорости, с которой движется следующий экземпляр, когда он достигает этой точки пути. Если вы хотите поместить точку пути в какую-либо другую позицию, не являющуюся концом, используйте path_insert_point().

Если функция используется для пути из браузера ассетов, обратите внимание, что она изменит фактический актив и, таким образом, навсегда повлияет на то, как этот путь используется всеми экземплярами в игре с момента использования функции и до конца игры. Если это не то, что вам нужно, используйте функцию типа path_duplicate(), чтобы сначала создать копию пути, а затем вызовите эту функцию на дублированном активе (не забудьте вызвать path_delete() на активе, когда он больше не нужен).

Синтаксис:

path_add_point(index, x, y, speed);

Аргумент Тип Описание
index Путь Актива The index of the path to add the point to.
x Реально The x coordinate of the new point.
y Реально The y coordinate of the new point.
speed Реально The speed factor of the point (default value is 100).

Возвращается:

N/A

Пример:

my_path = path_add();
for (var i = 0; i <= 360; i += 36;)
{
    path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100);
}

Приведенный выше код создаст круговой путь вокруг позиции x/y экземпляра, выполняющего код.
