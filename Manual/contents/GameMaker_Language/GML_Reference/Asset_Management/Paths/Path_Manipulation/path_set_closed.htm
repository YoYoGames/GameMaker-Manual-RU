путь_закрыт

Эта функция может быть использована для обозначения заданного пути как открытого (false) или закрытого (true). У замкнутого пути начальная точка соединена с конечной, образуя петлю, а у открытого пути начало и конец окончательно не соединены. Если это не то, что вам нужно, используйте функцию path_duplicate() для создания копии пути, а затем вызовите эту функцию на дублированном активе(не забудьте вызвать path_delete() на активе, когда он больше не нужен).

Синтаксис:

path_set_closed(index, closed);

Аргумент Тип Описание
index Путь Актива The index of the path to change.
closed Булево Whether the path is closed (true) or not (false).

Возвращается:

N/A

Пример:

path_set_closed(pth_Patrol, true);

Это установит заданный путь как замкнутый.
