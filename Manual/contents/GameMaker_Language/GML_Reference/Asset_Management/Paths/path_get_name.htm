имя_пути

Эта функция возвращает имя пути, на который ссылается, в виде строки. Имя будет тем, которое было использовано для определения пути в редакторе, или (если путь был создан через функцию кода) будет возвращена строка с форматом " _newpathXX", где "XX" - это номер созданного пути, начинающийся с 0 и увеличивающийся на единицу при каждом создании нового пути. Обратите внимание, что это только и не может быть использована для прямой ссылки на путь - для этого вам понадобится идентификатор индекса пути. Однако вы можете использовать эту строку для получения идентификатора индекса пути, используя возвращаемую строку вместе с функцией с функцией asset_get_index().

Синтаксис:

path_get_name(index);

АргументТип Описание
index The index of the path to check.

Возвращается:

Пример:

path_name = path_get_name(path_array[0]);

Это установит "path_name" в имя пути, индексированного в данном массиве в позиции 0.
