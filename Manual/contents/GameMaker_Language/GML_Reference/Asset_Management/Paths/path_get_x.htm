path_get_x

С помощью этой функции вы можете получить координату x позиции на любом заданном пути. 0 - это начало пути, 1 - конец пути, а все, что находится между ними, приравнивается к этому расстоянию на пути. Это не обязательно должна быть определяющая точка пути, она может быть в любом месте пути.

Синтаксис:

path_get_x(ind, pos);

Аргумент Тип Описание
index Путь Актива The index of the path to check.
pos Реально How far through the path to check. Between 0 (start) and 1 (end).

Возвращается:

Real

Пример:

x = path_get_x(pth_WallWalk, 0.5);

Это установит x вызывающего экземпляра в координату x точки, расположенной ровно на половине заданного пути.
