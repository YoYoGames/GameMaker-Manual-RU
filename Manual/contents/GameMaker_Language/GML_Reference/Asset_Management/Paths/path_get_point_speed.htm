path_get_point_speed

С помощью этой функции можно получить скорость точки (как определено в редакторе контура или при динамическом добавлении точки контура с помощью функции path_add_point()), выраженную в процентах. Так, если в редакторе контуров для точки контура установлено значение 50, эта функция при использовании вернет значение 50.

Синтаксис:

path_get_point_speed(index, n);

Аргумент Тип Описание
index Путь Актива The index of the path to check.
n Реально The point number to check.

Возвращается:

Real

Пример:

if (path_get_point_speed(pth_Patrol, 0) != 100)
{
    var _px = path_get_point_x(pth_Patrol, 0);
    var _py = path_get_point_x(pth_Patrol, 0);
    path_change_point(pth_Patrol, 0, _px, _py, 100);
}

Приведенный выше код проверяет точку "0" пути, проиндексированную "pth_Patrol", на предмет того, не равна ли скорость 100. Если это не так, то скорость этой точки устанавливается равной 100.
