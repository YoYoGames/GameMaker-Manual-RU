path_get_number

Эта функция может быть использована для возврата количества точек на пути.

Синтаксис:

path_get_number(index);

АргументТип Описание
index The index of the path to check.

Возвращается:

Пример:

if (path_get_number(pth_AI) > 1)
{
    path_start( pth_AI, 4, 3, 0);
}

Приведенный выше код проверяет, есть ли на пути более одной точки, и если да, то начинает перемещение экземпляра по этому пути.
