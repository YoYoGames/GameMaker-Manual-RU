path_get_y

С помощью этой функции можно получить координату y позиции на любом заданном пути. 0 - это начало пути, 1 - конец пути, а все, что находится между ними, равно нормализованному расстоянию по длине пути. Это значение не обязательно должно быть определяющей точкой пути, оно может находиться в любой точке пути.

Синтаксис:

path_get_y(ind, pos);

Аргумент Тип Описание
index Путь Актива The index of the path to check.
pos Реально How far through the path to check. Between 0 (start) and 1 (end).

Возвращается:

Real

Пример:

y = path_get_y(pth_WallWalk, 0.5);

Это установит y вызывающего экземпляра в координату y точки, расположенной ровно на полпути заданного пути.
