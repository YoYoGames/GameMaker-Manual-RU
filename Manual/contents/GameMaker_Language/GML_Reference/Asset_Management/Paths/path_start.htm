путь_начало

Путь создается из серии определяющих точек, которые соединяются вместе и затем используются для планирования движения экземпляра. Они могут быть созданы с помощью кода или в редакторе путей и назначены экземпляру для использования в игре. С помощью этой функции можно указать экземпляру, по какому пути следовать, с какой скоростью (измеряется в пикселях за шаг), как вести себя, когда он достигнет конца пути, и следовать ли абсолютному или относительному положению пути. Последняя часть означает, что экземпляр либо начинает и следует по пути точно так, как вы спроектировали и разместили его в редакторе путей (абсолютный), либо начинает и следует по пути с позиции, в которой был создан экземпляр (относительный).

Как видно из приведенного выше изображения, два экземпляра следуют по одному и тому же пути, начатому с path_start(), но один из экземпляров следует по абсолютному пути, а другой - по относительному. Следует отметить, что при установке относительного пути на false (абсолютный) экземпляр будет "прыгать" к началу пути независимо от того, в каком месте комнаты он находится.

Константы, используемые для определения поведения экземпляра при достижении им конца пути, приведены на этой странице.

Синтаксис:

path_start(path, speed, endaction, absolute);

Аргумент Тип Описание
path Путь Актива The path index to start.
speed Реально The speed of which to follow the path in pixels per step, negative meaning going backwards.
endaction Конец пути Действие Константа What to do when the end of the path is reached (see the constants listed above).
absolute Булево Whether the calling instance should follow the absolute path as it is in the editor (true) or a relative path to its current position (false).

Возвращается:

N/A

Пример:

path_start(path, 4, path_action_reverse, 0);

Это заставит вызывающий экземпляр следовать по пути, индексированному в переменной "path", со скоростью 4 пикселя за шаг. Он будет следовать по пути относительно своего положения, а когда достигнет конца, изменит направление и вернется назад.
