shader_current

Эта функция возвращает индекс ID шейдера, который в данный момент используется для рендеринга, или возвращает -1, если шейдер не используется.

Синтаксис:

shader_current();

Возвращается:

Shader Asset or -1 (if no shader is assigned)

Пример:

if (shader_current() == -1)
{
    shader_set(sh_warp)
}

Приведенный выше код проверяет текущий шейдер, и если он возвращает -1 (шейдер не используется), то шейдер устанавливается.
