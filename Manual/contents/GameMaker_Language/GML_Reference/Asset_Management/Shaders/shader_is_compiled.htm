shader_is_compiled

Эта функция проверяет шейдер во время выполнения, чтобы убедиться, что он был успешно скомпилирован. Если это так, то функция возвращает true, в противном случае возвращается false. Эту функцию следует использовать в начале игры, чтобы убедиться, что платформа, на которой запущена ваша игра, успешно скомпилировала все используемые шейдеры (особенно на Windows, где некоторые компьютеры могут использовать DX9 с уровнем шейдеров 2.0, а не более позднюю версию с уровнем шейдеров 3.0).

Если ваш шейдер НЕ был скомпилирован, и вы вызовете команду shader_set() игра потерпит крах, поэтому стоит иметь какую-то проверку всякий раз, когда вы используете что-то, кроме простых GLSL ES шейдеров.

Синтаксис:

shader_is_compiled(shader);

АргументТип Описание
shader The shader to check.

Возвращается:

Boolean

Пример:

global.GFX = false;
if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
{
    global.GFX = true;
}

Приведенный выше код установит глобальную переменную в значение false, а затем, если оба проверяемых шейдера скомпилированы правильно, она будет установлена в значение true.
