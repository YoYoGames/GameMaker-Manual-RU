shader_get_sampler_index

Поскольку вы не можете изменить значение шейдерного сэмплера в самом шейдере, вы должны установить его перед вызовом шейдера с помощью одной из доступных функций унифицированного набора. Однако для этого необходимо сначала вызвать эту функцию, чтобы получить "хэндл" шейдерного сэмплера, который вы хотите установить.

ПРИМЕЧАНИЕ Хотя шейдер состоит из двух отдельных программ (вершинной и фрагментной), эта функция не делает различий между ними и возвращает хэндл образца шейдера из любой из них.

Синтаксис:

shader_get_sampler_index(shader, uniform);

Аргумент Тип Описание
shader Шейдерный актив The index of the shader to use.
uniform Строка The shader sampler to get the handle of (a string).

Возвращается:

Shader Sampler Handle

Пример:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

Приведенный выше код получит хэндл для сэмплера в шейдере с индексом "GlassShader", а затем установит константу этого шейдера на заданную текстуру sprite.
