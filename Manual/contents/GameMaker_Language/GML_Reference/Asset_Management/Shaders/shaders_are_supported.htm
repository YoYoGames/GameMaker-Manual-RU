шейдеры_поддерживаются

Эта функция проверяет, поддерживает ли выбранная целевая платформа шейдеры, и возвращает true, если поддерживает, и false, если не поддерживает. Важно отметить, что на Android, если в проекте не определены шейдерные ресурсы, то функция всегда будет возвращать false, независимо от того, поддерживает устройство шейдеры или нет.

Синтаксис:

shaders_are_supported();

Возвращается:

Пример:

global.GFX = false;
if (shaders_are_supported())
{
    if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
    {
        global.GFX = true;
    }
}

Приведенный выше код установит глобальную переменную в значение false, а затем, если платформа поддерживает шейдеры и оба проверяемых шейдера скомпилированы правильно, она будет установлена в значение true.
