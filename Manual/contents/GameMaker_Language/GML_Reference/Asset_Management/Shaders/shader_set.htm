шейдер_set

С помощью этой функции вы можете установить цель рисования на заданный шейдер, и все дальнейшее рисование будет выполняться с его помощью. Завершить использование шейдера можно с помощью функции shader_reset().

Синтаксис:

shader_set(shader);

АргументТип Описание
shader The index of the shader to use.

Возвращается:

Пример:

shader_set(shader_Glass);
draw_self();
shader_reset();

Приведенный выше код установит шейдер, который будет использоваться для отрисовки, а затем отрисует текущий sprite, используемый для экземпляра, который его использует.
