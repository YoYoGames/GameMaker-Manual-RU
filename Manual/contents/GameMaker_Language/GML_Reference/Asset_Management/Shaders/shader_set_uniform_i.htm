shader_set_uniform_i

С помощью этой функции можно установить значение (или значения) константы шейдера. Предварительно вы должны получить "хэндл" константы с помощью функции shader_get_uniform()и вы должны знать тип константы, чтобы передать ей нужное количество целых значений, т.е. если у вас есть vec2, то вам нужно передать в функцию два значения.

ПРИМЕЧАНИЕ: Все формы должны быть установлены после вызова функции shader_set(), и до вызова shader_reset().

Синтаксис:

shader_set_uniform_i(handle, value1 [, value2, value3, value4]);

Аргумент Тип Описание
handle Равномерная обработка шейдера The handle of the shader constant to set.
value1 ... value4 Реально The integer value (or values) to set the shader constant to.

Возвращается:

N/A

Пример:

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_i(shader_params, 0, 65, 255);
draw_self();
shader_reset();

Приведенный выше код получит хэндл шейдерной константы "u_vParams" (a vec3), затем установит эту константу в заданные целочисленные значения.
