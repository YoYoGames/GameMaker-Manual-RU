shader_set_uniform_matrix_array

С помощью этой функции вы можете установить константу шейдера для хранения массива значений матрицы. Вы должны предварительно получить "хэндл" константы с помощью функции shader_get_uniform()и предварительно инициализировать массив как массив значений с плавающей точкой, где каждая последовательная группа из 16 плавающих значений является матрицей 4x4.

ПРИМЕЧАНИЕ: Все формы должны быть установлены после вызова функции shader_set(), и до вызова shader_reset().

Синтаксис:

shader_set_uniform_matrix_array(handle, array);

Аргумент Тип Описание
handle Равномерная обработка шейдера The handle of the shader constant to set.
array Массив A previously initialised array of floating point values.

Возвращается:

N/A

Пример:

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_tint, "cMatrixArray");
shader_set_uniform_matrix_array(shader_params, matrix_array);
draw_self();
shader_reset();

Приведенный выше код получит хэндл константы шейдера "cMatrixArray", а затем установит эту константу в заданный массив.
