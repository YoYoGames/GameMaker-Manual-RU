shader_reset

Эта функция сбрасывает шейдер, используемый для рисования, и должна вызываться, когда вы больше не хотите использовать текущий шейдер (установленный с помощью функции shader_set()).

Синтаксис:

shader_reset();

Возвращается:

Пример:

shader_set(shader_Glass);
draw_self();
shader_reset();

Приведенный выше код установит шейдер, который будет использоваться для отрисовки, затем отрисует текущий sprite, используемый для экземпляра, который его использует. Наконец, он сбросит шейдер, чтобы вернуться к шейдеру по умолчанию GameMaker.
