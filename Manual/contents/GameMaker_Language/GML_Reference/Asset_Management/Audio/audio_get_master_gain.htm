audio_get_master_gain

С помощью этой функции вы можете получить абсолютное значение глобальной громкости всех звуков и музыки для определенного слушателя. Индекс слушателя по умолчанию равен 0, но вы можете использовать функцию audio_get_listener_info() для получения различных индексов, доступных для целевой платформы. Возвращаемое значение усиления основано на линейной шкале от 0 (тишина) до 1 (полная громкость). Обратите внимание, что на некоторых платформах коэффициент усиления может быть больше 1, хотя значение 1 считается "полной громкостью", а большее значение может привести к обрыву звука.

Синтаксис:

audio_get_master_gain(listenerIndex);

Аргумент Тип Описание
listenerIndex Идентификаторреального или аудиослушателя The index of the listener to get the gain of.

Возвращается:

Real

Пример:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var ind = info[? "index"];
    if audio_get_master_gain(ind) != 1
    {
        audio_set_master_gain(info[? "index"], 1);
    }
    ds_map_destroy(info);
}

Приведенный выше код перебирает доступные слушатели, проверяя, равно ли их усиление 1 или нет, и если нет, то устанавливает его в 1 для каждого из них.
