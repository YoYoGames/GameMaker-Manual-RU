audio_get_type

При добавлении звука в GameMaker он может быть либо потоковым, либо находиться в памяти. Если вам нужно узнать, является ли данный индекс звука потоковым или нет, вы можете использовать эту функцию, которая вернет 1 для потокового звука или 0 для звука в памяти. использовать эту функцию, которая вернет 1 для потокового звука, 0 для звука в памяти и -1, если произошла ошибка или индекс не указывает на правильный звуковой ресурс.

Синтаксис:

audio_get_type(index);

АргументТип Описание
index The index of the sound to check.

Возвращается:

Пример:

type = audio_get_type(snd_Music_1);

Приведенный выше код проверяет тип аудио, индексированный в переменной "snd_Music_1", и сохраняет возвращаемое значение в переменной "type".
