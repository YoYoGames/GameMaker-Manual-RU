audio_free_play_queue

Эта функция используется для освобождения памяти, связанной с заданной аудиоочередью. Индекс очереди - это значение, возвращенное при создании очереди с помощью функции audio_create_play_queue()и эта функция должна вызываться, когда очередь больше не нужна, чтобы предотвратить утечку памяти. Освобождение очереди остановит любой воспроизводимый звук, и вы не сможете удалить буфер, из которого передается звук, пока не будет освобождена очередь, которой он назначен. Эта функция вызовет асинхронное событие воспроизведения звука, и в этом событии в переменной будет создана специальная DS карта async_load со следующими парами ключ/значение:

  • "queue_id" - очередь, которая закончила воспроизведение данных
  • "buffer_id" - буфер, воспроизведение из которого прекращено
  • "queue_shutdown" - устанавливается в 0 при обычном воспроизведении и в 1 при получении события, поскольку вы освободили очередь с помощью этой функции. Когда это значение равно 1, вы не хотите больше ставить данные в очередь.

Синтаксис:

audio_free_play_queue(queueIndex);

Аргумент Тип Описание
queueIndex Идентификатор очереди аудио The index of the queue to free.

Возвращается:

N/A

Пример:

var q_id = async_load[? "queue_id"];
if (q_id == audio_queue)
{
    audio_free_play_queue(audio_queue);
    buffer_delete(audio_buffer);
}

Приведенный выше код будет входить в событие асинхронного воспроизведения аудио и проверять, не находится ли аудио, которое закончило воспроизводиться, в "audio_queue", и если да, то освобождает очередь и связанный с ней буфер.
