audio_get_recorder_count

Эта функция возвращает количество источников записи звука (таких как микрофоны и т.д.), доступных в настоящее время для вашей игры. Так, если возвращаемое значение равно, например, четырем, то у вас будет аудио вход на источниках 0, 1, 2 и 3, причем одно из этих значений будет тем, с которого вы начнете запись с помощью функции audio_start_recording(). Это значение может меняться в любое время по мере того, как люди подключают/отключают микрофоны или другие источники входного сигнала к устройству. Обратите внимание, что вы можете использовать функцию audio_get_recorder_info() для получения информации о каждом подключенном устройстве.

ВАЖНО Эта функция всегда будет возвращать 1 на цели Opera GX.

Синтаксис:

audio_get_recorder_count();

Возвращается:

Real

Пример:

if audio_get_recorder_count() > 0
{
    channel_index = audio_start_recording(0);
}

Приведенный выше код проверяет доступные аудиорекордеры и, если их 1 или более, начинает запись с первого из них (источник 0).
