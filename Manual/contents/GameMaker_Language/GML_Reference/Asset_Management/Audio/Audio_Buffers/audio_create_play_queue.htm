audio_create_play_queue

Эта функция подготавливает буферную очередь для аудио. Вы передаете формат данных (в настоящее время поддерживаются только buffer_u8 или buffer_s16 ), частоту дискретизации (которая может быть от 1000гц до 48000гц), и каналы, которые требует аудио из одной из перечисленных ниже констант:

Постоянная Описание
audio_mono Монофонический (одноканальный) звук.
audio_stereo Стерео (двухканальный) звук.
audio_3D 3D (5.1) аудио.


Обратите внимание, что вы должны освободить все очереди аудио, созданные с помощью этой функции, когда они больше не нужны, чтобы предотвратить утечку памяти при использовании функции audio_free_play_queue().

Эта функция вызовет асинхронное событие воспроизведения аудио, когда очередь достигнет конца, и в этом событии будет создана специальная карта DS в переменной async_load со следующими парами ключ/значение:

  • "queue_id" - очередь, которая закончила воспроизведение данных
  • "buffer_id" - буфер, воспроизведение из которого прекращено
  • "queue_shutdown" - устанавливается в 0 при нормальном воспроизведении и в 1 при получении события, потому что audio_free_play_queue() было вызвано. Когда этот параметр установлен в 1, вы не хотите ставить в очередь дальнейшие данные.

Синтаксис:

audio_create_play_queue(queueFormat, queueRate, queueChannels);

Аргумент Тип Описание
queueFormat Тип данных буфера Константа The format of the buffer data to use (buffer_u8 или buffer_s16).
queueRate Реально The sample rate of the data in the buffer queue.
queueChannels Тип аудиоканала The channels to use from one of the constants listed above.

Возвращается:

Audio Queue ID

Пример:

audio_queue = audio_create_play_queue(buffer_s16, 11052, audio_mono);

Приведенный выше код создает новую очередь аудио и присваивает ее ID переменной для дальнейшего использования.
