audio_queue_sound

Эта функция добавит данные из буфера в очередь аудио, которую вы ранее создали с помощью функции audio_create_play_queue(). Вы указываете индекс очереди для добавления, ID буфера для получения данных, а также позицию (смещение) в буфере и количество байт для добавления. После добавления звука из буфера в очередь, буфер не может быть удален до тех пор, пока вы не освободите очередь с помощью функции audio_free_play_queue(), а свойства буфера должны соответствовать свойствам очереди, в которую вы добавляете звук.

Синтаксис:

audio_queue_sound(queueIndex, bufferId, bufferOffset, bufferLength);

Аргумент Тип Описание
queueIndex Идентификатор очереди аудио The index of the queue to add to.
bufferId Идентификатор буфера The buffer to add to the queue.
bufferOffset Реально The offset within the source buffer to start from.
bufferLength Реально The length of the buffer (the number of the bytes in the buffer).

Возвращается:

N/A

Пример:

len = async_load[? "data_len"];
audio_buff = buffer_create(len, buffer_fast, 1);
buffer_copy(async_load[? "buffer_id"], 0, len, buff, 0);
audio_queue_sound(audio_queue, audio_buff, 0, len);
audio_play_sound(audio_queue, 10, 0);

Приведенный выше код будет вызван в асинхронном событии Audio Recording и присвоит некоторые записанные данные буферу, который затем будет добавлен в очередь аудио. Затем происходит воспроизведение.
