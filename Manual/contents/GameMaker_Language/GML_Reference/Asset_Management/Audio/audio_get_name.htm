audio_get_name

Эта функция возвращает имя заданного аудиоресурса в виде строки. Значение "index" может быть значением самого ресурса (как видно в Asset Browser) или уникальным значением ID, которое выдается, например, при воспроизведении звука, audio_play_sound(). Обратите внимание, что возвращаемая строка не совпадает с идентификатором ресурса и не может быть использована для доступа к самому ресурсу, поэтому должна использоваться только для отображения или проверки ошибок.

Синтаксис:

audio_get_name(index);

АргументТип Описание
index The index of the sound to check.

Возвращается:

Пример:

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

Приведенный выше код воспроизводит случайный звук, выбранный из трех различных звуковых ресурсов, а затем показывает отладочное сообщение с его названием.
