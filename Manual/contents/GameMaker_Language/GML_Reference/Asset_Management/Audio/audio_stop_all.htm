audio_stop_all

Эта функция остановит все воспроизводимые в данный момент звуки.

Синтаксис:

audio_stop_all();

Возвращается:

Пример:

if !global.SFX
{
    audio_stop_all();
}

Приведенный выше код проверяет глобальную переменную "SFX" и, если она возвращает значение false, то останавливает все воспроизводимые в данный момент звуки.
