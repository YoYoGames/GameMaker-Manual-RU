audio_resume_sound

С помощью этой функции можно возобновить любой звук, который в данный момент приостановлен (после использования функции audio_pause_sound()). Звук может быть либо одним экземпляром звука (индекс для отдельных воспроизводимых звуков может быть сохранен в переменной при использовании функции audio_play_sound() или audio_play_sound_at() ) или звуковой актив, в этом случае все экземпляры данного звука будут запущены заново.

Синтаксис:

audio_resume_sound(index);

Аргумент Тип Описание
index Идентификаторзвукового актива или звукового экземпляра The index of the sound to resume.

Возвращается:

N/A

Пример:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if (global.Pause)
    {
        audio_pause_sound(snd_Waterfall);
    }
    else
    {
        audio_resume_sound(snd_Waterfall);
    }
}

Приведенный выше код проверяет нажатие клавиши "P" и, если обнаруживает ее, устанавливает глобальную переменную "Pause" в значение true или false, а затем либо приостанавливает звук, индексированный в переменной "snd_Waterfall", либо возобновляет его из состояния паузы.
