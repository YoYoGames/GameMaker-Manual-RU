audio_system_is_available

Эта функция может использоваться для проверки и определения того, была ли инициализирована аудиосистема, или запущен ли аудиоконтекст. На всех платформах эта функция вернет true сразу после запуска игры, когда аудиосистема будет инициализирована, за исключением HTML5. На HTML5 статус аудиоконтекста может измениться в любой момент в зависимости от пользовательского ввода, используемого браузера и его версии, поэтому эту функцию можно использовать для проверки того, может ли воспроизводиться звук или нет. Если функция возвращает false (т.е. статус аудиоконтекста не запущен), то могут воспроизводиться только непотоковые звуки (но это не гарантировано, и вы должны считать, что ни один звук не может быть воспроизведен), в то время как если она возвращает true, то будут воспроизводиться все звуки.

Синтаксис:

audio_system_is_available();

Возвращается:

Пример:

if audio_system_is_available()
{
    if audio_is_paused(global.Music)
    {
        audio_resume_sound(global.Music)
    }
    else
    {
        if !audio_is_playing(global.Music)
        {
            global.Music = audio_play_sound(snd_Music, 0, true);
        }
    }
}
else
{
    if audio_is_playing(global.Music)
    {
        audio_pause_sound(global.Music);
    }
}

Приведенный выше код будет приостанавливать/отключать аудиодорожку в зависимости от того, инициализирована и доступна аудиосистема или нет.
