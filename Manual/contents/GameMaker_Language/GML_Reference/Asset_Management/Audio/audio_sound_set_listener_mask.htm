audio_sound_set_listener_mask

Эта функция может быть использована для установки битовой маски для звука, чтобы он воспроизводился только от указанных слушателей. Вы можете создать битовую маску с помощью функции audio_get_listener_info() а затем с помощью побитового or ("|") создать пользовательскую маску с теми слушателями, от которых вы хотите, чтобы звук воспроизводился, а затем применить эту маску к звуку после его воспроизведения. Эта маска отменяет маску по умолчанию или ту, которую вы задали с помощью функции audio_set_listener_mask().

Эта функция может быть использована только для звука после его установки на воспроизведение и требует идентификатор звука, возвращенный функцией, используемой для воспроизведения звука.

Синтаксис:

audio_sound_set_listener_mask(soundID, mask);

Аргумент Тип Описание
soundID Идентификатор звукового экземпляра The unique ID of the sound to set the mask of
mask Реально The bitmask data to set for the sound

Возвращается:

N/A

Пример:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

Приведенный выше код воспроизводит звук, затем устанавливает слушателя (слушателей) для воспроизведения, используя данные маски, хранящиеся в глобальной переменной.
