audio_resume_all

С помощью этой функции можно возобновить все звуки, которые были приостановлены ранее.

Синтаксис:

audio_resume_all();

Возвращается:

Пример:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if global.Pause
    {
        audio_pause_all();
    }
    else
    {
        audio_resume_all();
    }
}

Приведенный выше код проверяет нажатие клавиши "P" и, если обнаруживает ее, устанавливает глобальную переменную "Pause" в значение true или false, а затем либо приостанавливает все звуки, либо перезапускает все ранее приостановленные звуки.
