audio_sound_is_playable

Эта функция может быть использована для проверки возможности воспроизведения заданного звукового индекса. Это необходимо из-за различий в способах воспроизведения потокового и непотокового звука на целевой платформе HTML5. Функция возвращает true, если звук может быть воспроизведен, и false, если не может. Обратите внимание, что на всех других платформах, кроме HTML5, функция всегда будет возвращать true.

Синтаксис:

audio_sound_is_playable(index);

Аргумент Тип Описание
index Звуковой актив The index of the sound to check.

Возвращается:

Boolean

Пример:

if audio_sound_is_playable(snd_Shot)
{
    audio_play_sound(snd_Shot, 10, false);
}

Приведенный выше код воспроизведет звук "snd_Shot", если он воспроизводим.
