audio_group_set_gain

С помощью этой функции можно приглушить или затушить группу звуков на определенный промежуток времени, а также мгновенно установить усиление группы.

Время измеряется в миллисекундах, и функция требует, чтобы вы ввели конечный уровень усиления, которого группа должна достичь к концу этого времени. Этот уровень усиления может быть от 0 (тишина) до 1 (полная громкость), а шкала линейна, так что значение 0,5 будет равняться половине громкости. Чтобы мгновенно изменить усиление, просто установите аргумент time в 0. Обратите внимание, что на некоторых платформах усиление может быть больше 1, хотя значение 1 считается "полной громкостью", а большее значение может привести к обрыву звука.

Синтаксис:

audio_group_set_gain(groupID, volume, time);

Аргумент Тип Описание
groupID Идентификатор аудиогруппы The index of the audio group to stop (as defined in the Audio Groups Window)
volume Реально The final value for the group volume.
time Реально The length of the change in gain in milliseconds.

Возвращается:

N/A

Пример:

if keyboard_check_pressed(vk_space)
{
    audio_group_set_gain(audiogroup1, 0, 5000);
}

Приведенный выше код проверяет наличие клавиши "пробел" и затем затухает все аудио для "audiogroup1" до 0 в течение 5 секунд.
