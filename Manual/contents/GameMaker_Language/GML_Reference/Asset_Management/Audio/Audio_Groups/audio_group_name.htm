имя_аудиогруппы

Эта функция возвращает строку, содержащую имя заданной аудиогруппы для отображения или проверки. Когда вы определяете аудиогруппу в Опциях игры, вы даете ей уникальное "имя", которое на самом деле является константой, используемой в качестве значенияID для группы. Все, что делает эта функция, это принимает ID и возвращает строку с именем ID, которое вы дали.

Синтаксис:

audio_group_name(groupID);

Аргумент Тип Описание
groupID Идентификатор аудиогруппы The index value constant of the audio group to check (as defined in the Audio Groups Window)

Возвращается:

String

Пример:

var name = audio_group_name(audiogroup_level1);
draw_text(32, 32, "Now Playing Group: " + name);

Приведенный выше код извлекает имя заданной константы аудиогруппы и выводит его на экран.
