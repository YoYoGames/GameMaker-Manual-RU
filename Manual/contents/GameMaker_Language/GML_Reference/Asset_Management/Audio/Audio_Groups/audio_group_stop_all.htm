audio_group_stop_all

Эта функция остановит все воспроизводимые в данный момент звуки из заданной аудиогруппы.

Синтаксис:

audio_group_stop_all(groupID);

Аргумент Тип Описание
groupID Идентификатор аудиогруппы The index of the audio group to stop (as defined in the Audio Groups Window)

Возвращается:

N/A

Пример:

if keyboard_check_pressed(vk_space)
{
    audio_group_stop_all(audiogroup_level1);
}

Приведенный выше код проверяет наличие клавиши "пробел", а затем останавливает воспроизведение всех аудиофайлов из заданной группы.
