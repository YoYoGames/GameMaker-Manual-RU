audio_exists

Эта функция возвращает, существует ли звук (true) или нет (false). Обратите внимание, что если искомый индекс не был инициализирован ранее, эта функция выдаст ошибку, так как она ищет несуществующие индексы активов. Проверяемый звук может быть либо единичным экземпляром звука (индекс для отдельных воспроизводимых звуков может быть сохранен в переменной при использовании функции audio_play_sound() или audio_play_sound_at() ) или звуковой актив.

Синтаксис:

audio_exists(index);

Аргумент Тип Описание
index Активы The index of the sound to check the existence of.

Возвращается:

Boolean

Пример:

if audio_exists(global.Music)
{
    audio_play_sound(global.Music, 0, true);
}

Приведенный выше код проверяет, существует ли звук. Если он существует, он устанавливается для воспроизведения в цикле.
