audio_set_master_gain

С помощью этой функции вы можете установить абсолютное значение глобальной громкости всех звуков и музыки для определенного слушателя. Индекс слушателя по умолчанию равен 0, но вы можете использовать функцию audio_get_listener_info(), чтобы получить различные индексы, доступные для целевой платформы. Значение усиления основано на линейной шкале от 0 (тишина) до 1 (полная громкость) и влияет на относительную громкость всех звуков и музыки, воспроизводимых из вашей игры через данный слушатель.

Синтаксис:

audio_set_master_gain(listenerIndex, gain);

Аргумент Тип Описание
listenerIndex Идентификаторреального или аудиослушателя The index of the listener to set the gain on.
gain Реально Value for the global volume (0 to 1).

Возвращается:

N/A

Пример:

var num = audio_get_listener_count();
for( var i = 0; i < num; i++;)
{
    var info = audio_get_listener_info(i);
    audio_set_master_gain(info[? "index"], 0.75);
    ds_map_destroy(info);
}

Приведенный выше код перебирает доступные слушатели, а затем устанавливает их главный коэффициент усиления на 0,75.
