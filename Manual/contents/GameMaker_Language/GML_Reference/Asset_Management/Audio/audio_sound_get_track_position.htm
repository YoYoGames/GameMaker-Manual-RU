audio_sound_get_track_position

Эта функция получает позицию (в секундах) в звуковом файле для звука, который будет воспроизводиться. Звук может быть только одним экземпляром звука (индекс для отдельных воспроизводимых звуков может быть сохранен в переменной при использовании функции audio_play_sound() или audio_play_sound_at() функции).

Синтаксис:

audio_sound_get_track_position(index);

Аргумент Тип Описание
index Идентификатор звукового экземпляра The index of the sound to get the play position of.

Возвращается:

Real

Пример:

if audio_sound_get_track_position(global.Music) != 0
{
    audio_sound_set_track_position(global.Music, 0);
}

Приведенный выше код проверяет трек, чтобы получить его начальную позицию, и если она не 0 секунд, то устанавливает ее на 0 секунд.
