audio_sound_set_track_position

Эта функция устанавливает позицию (в секундах) для заданного звукового идентификатора или актива.

Указанный "индекс" может быть либо единственным экземпляром звука (как возвращено из функции audio_play_sound()) или звуковым активом (например, добавленным через Asset Browser). Поведение этой функции будет отличаться в зависимости от типа указанного вами индекса:

  • Если это уникальный идентификатор звука, который вы используете и который уже играет, то его позиция немедленно изменится на заданное время.
  • Если это звуковой актив, то все последующие воспроизведения данного звукового актива начнутся в новое время.

Эта функция изменяет позицию дорожки только для воспроизводимого в данный момент звука или для его следующего воспроизведения (в случае, если предоставлен звуковой актив). Если звук воспроизводится с включенным циклом, последующие воспроизведения всегда будут начинаться с начала (0,0 секунды), а не с позиции дорожки, определенной с помощью этой функции. Например, если запустить звуковую петлю на 5,7 секунды, то первый звук будет воспроизводиться с этой точки, однако после этого будет повторяться весь трек, от начала до конца.

Синтаксис:

audio_sound_set_track_position(index, time);

Аргумент Тип Описание
index Идентификаторзвукового актива или звукового экземпляра The index of the sound to change.
time Реально The time (in seconds) to set the start point to. Values longer than the length of the given sound are ignored.

Возвращается:

N/A

Пример:

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

Приведенный выше код воспроизводит звук, а затем использует возвращаемое значение ID для установки начальной позиции звука на 32 секунды.
