номер аудио_канала

С помощью этой функции вы можете установить, сколько аудиоканалов доступно для воспроизведения звука в GameMaker. В основном это означает, что вы задаете количество одновременных звуков, которые могут воспроизводиться в любой момент времени, и если это количество превышает указанное, то звуки с более низким приоритетом останавливаются, чтобы освободить канал для звуков с более высоким приоритетом. Вы можете использовать эту функцию для оптимизации игры для устройств, так как чем меньше каналов для звука, тем лучше производительность, но имейте в виду, что это может привести к заметному обрезанию некоторых звуков, если многие из них воспроизводятся одновременно.

Синтаксис:

audio_channel_num(num);

АргументТип Описание
num Number of available audio channels (default is 128).

Возвращается:

Пример:

switch (os_browser)
{
    case browser_not_a_browser:
        switch (os_type)
        {
            case os_windows:
            case os_macos:
                audio_channel_num(200);
            break;

            default:
                audio_channel_num(64);
            break;
        }
    break;

    default:
        audio_channel_num(16);
    break;
}

Приведенный выше код проверяет платформу, на которой запущена игра, и изменяет количество доступных звуковых каналов для увеличения производительности.
