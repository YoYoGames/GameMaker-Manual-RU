audio_emitter_exists

Эта функция возвращает, существует ли аудиоизлучатель (true) или нет (false). Обратите внимание, что если искомый индекс не был инициализирован ранее, эта функция выдаст ошибку, так как она ищет несуществующие индексы.

Синтаксис:

audio_emitter_exists(index);

Аргумент Тип Описание
index Идентификатор аудиоизлучателя The index of the emitter to check the existence of.

Возвращается:

Boolean

Пример:

if audio_emitter_exists(s_emit)
{
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}
else
{
    s_emit = audio_emitter_create();
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}

Приведенный выше код проверяет, существует ли эмиттер, индексированный в (предварительно инициализированной) переменной "s_emit". Если он существует, то через него воспроизводится звук, если же нет, то он создается, а затем воспроизводится звук.
