audio_emitter_create

Эта функция создает новый звуковой эмиттер и возвращает его индекс. Этот индекс должен храниться в переменной для всех последующих функций, связанных с этим эмиттером, а затем, когда он больше не нужен, его следует удалить из памяти с помощью функции audio_emitter_free() чтобы предотвратить утечку памяти, которая может привести к краху вашей игры.

Аудиоизлучатель создается со следующими настройками по умолчанию:

  • Коэффициент усиления и шаг 1
  • Маска слушателя, установленная только для слушателя по умолчанию (1)
  • Положение (0, 0, 0)
  • Скорость равна 0 (vx = 0, vy = 0, vz = 0)

Синтаксис:

audio_emitter_create();

Возвращается:

Audio Emitter ID

Пример:

s_emit = audio_emitter_create();

Приведенный выше код создает новый аудиоэмиттер и присваивает его индекс переменной "s_emit".
