audio_emitter_velocity

Эта функция может быть использована для придания излучателю эффекта Доплера и имитации движения звука на основе вектора, который определяется из заданных относительных положений x, y и z. Если эмиттер сам по себе никогда не будет двигаться, вам обычно не нужно задавать эти значения, но, например, если вы создаете скроллинговую игру-стрелялку, где враги появляются справа и прокручиваются влево, вы должны установить скорость (hspeed, 0, 0) в событии создания врагов (и обновить эмиттер каждого экземпляра в событии шага с помощью параметра audio_emitter_position()), чтобы придать любым звукам, издаваемым вражескими экземплярами, правильный доплер, когда они проходят мимо экземпляра игрока (который, в свою очередь, будет использовать функцию audio_listener_position() для установки слушателя в правильное положение).

Синтаксис:

audio_emitter_velocity(emitter, vx, vy, vz);

Аргумент Тип Описание
emitter Идентификатор аудиоизлучателя The index of the emitter to change.
vx Реально The x vector value (default 0).
vy Реально The y vector value (default 0).
vz Реально The z vector value (default 0).

Возвращается:

N/A

Пример:

s_emit = audio_emitter_create();
audio_emitter_position(s_emit, room_width, 0, 0);
audio_emitter_velocity(s_emit, -5, 0, 0);

Приведенный выше код создает звуковой излучатель и присваивает его индекс переменной "s_emit". Затем этот излучатель помещается в определенное место в комнате и ему присваивается скорость 5 пикселей за шаг вдоль оси x (он будет правильно доплеровски перемещаться относительно слушателя, как если бы он имел горизонтальную скорость 5 пикселей за шаг).
