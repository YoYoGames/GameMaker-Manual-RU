audio_emitter_free

С помощью этой функции вы можете удалить заданный эмиттер из памяти. Это всегда следует делать, когда эмиттер не будет использоваться далее в комнате или игре, т.е. в событии Destroy Event экземпляра или в событии Room End Event, иначе вы можете получить утечку памяти, которая замедлит и в конечном итоге приведет к краху вашей игры.

Синтаксис:

audio_emitter_free(emitter);

Аргумент Тип Описание
emitter Идентификатор аудиоизлучателя The index of the emitter to free.

Возвращается:

N/A

Пример:

if lives = 0
{
    audio_emitter_free(s_emit);
    room_goto(rm_Menu);
}

Приведенный выше код проверяет значение глобальной переменной "lives" и если возвращает 0, то уничтожает эмиттер, указанный в переменной "s_emit", а затем переходит в комнату, указанную в переменной "rm_Menu".
