audio_emitter_get_y

Эта функция возвращает текущее положение y заданного звукового излучателя.

Синтаксис:

audio_emitter_get_y(emitter);

Аргумент Тип Описание
emitter Идентификатор аудиоизлучателя The index of the emitter to use.

Возвращается:

Real

Пример:

if audio_emitter_get_y(emitter_player) != x
{
    audio_emitter_position(emitter_player, x, y, 0);
}

Приведенный выше код проверяет текущую позицию y данного эмиттера, и если она не равна позиции y экземпляра, то она устанавливается в позицию экземпляра.
