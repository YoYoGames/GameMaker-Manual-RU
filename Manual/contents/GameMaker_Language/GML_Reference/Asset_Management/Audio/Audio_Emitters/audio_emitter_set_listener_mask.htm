audio_emitter_set_listener_mask

Эта функция может быть использована для установки битовой маски для эмиттера, чтобы все звуки, воспроизводимые через эмиттер, воспроизводились только от указанных слушателей. Вы можете создать битовую маску с помощью функции audio_get_listener_info() а затем с помощью побитового or ("|") создать пользовательскую маску с теми слушателями, от которых требуется воспроизведение звука, а затем применить эту маску к эмиттеру. Эта маска отменяет маску по умолчанию или ту, которую вы задали с помощью функции audio_set_listener_mask().

Синтаксис:

audio_emitter_set_listener_mask(emitterID, mask);

Аргумент Тип Описание
emitterID Идентификатор аудиоизлучателя The unique ID of the emitter to set the mask of
mask Реально The bitmask data to set for the sound

Возвращается:

N/A

Пример:

s_emit = audio_emitter_create();
audio_emitter_set_listener_mask(snd, global.PlayerMask);

Приведенный выше код создает эмиттер звука, а затем устанавливает слушателя (слушателей) для воспроизведения, используя данные маски, хранящиеся в глобальной переменной.
