audio_play_sound_on

С помощью этой функции вы можете воспроизводить любой звуковой ресурс через эмиттер, при этом любые изменения усиления, положения, высоты тона или скорости эмиттера влияют на то, как пользователь слышит конечный воспроизводимый звук. Вы указываете индекс эмиттера для использования, индекс звука, который будет воспроизводиться, а затем указываете, должен ли звук зацикливаться или нет. Далее вы можете назначить приоритет, который затем используется для определения того, как звуки будут обработаны, когда количество воспроизводимых звуков превысит предел, установленный функцией audio_channel_num(). Звуки с более низким приоритетом будут остановлены в пользу звуков с более высоким приоритетом, а значение приоритета может быть любым реальным числом (фактическое значение произвольно и может быть от 0 до 1 или от 0 до 100, поскольку GameMaker будет определять их приоритет одинаково). Обратите внимание, что при работе с приоритетами чем больше число, тем выше приоритет, так что звуку с приоритетом 100 будет отдано предпочтение перед звуком с приоритетом 1.

Эта функция также возвращает уникальный индексный номер воспроизводимого звука, который можно сохранить в переменной, чтобы затем поставить его на паузу или остановить с помощью соответствующих функций. Это означает, что если у вас есть несколько экземпляров одного и того же звука, воспроизводимого в любой момент времени, вы можете выбрать только один экземпляр этого звука для работы с ним с помощью аудиофункций.

Синтаксис:

audio_play_sound_on(emitter, sound, loop, priority, [gain], [offset], [pitch], [listener_mask]);

Аргумент Тип Описание
emitter Идентификатор аудиоизлучателя The index of the emitter to use.
sound Звуковой актив The index of the sound to use.
loop Булево Flags the sound as looping or not.
priority Реально Set the channel priority for the sound.
gain Реально OPTIONAL Ценность выигрыша.
offset Реально OPTIONAL Время (в секундах), на которое нужно установить начальную точку. Значения, превышающие длину заданного звука, игнорируются.
pitch Реально OPTIONAL Множитель шага (по умолчанию 1).
listener_mask Реально OPTIONAL Данные битовой маски, которые нужно установить для звука. Для цели HTML5 это не будет иметь никакого эффекта, поскольку цель не поддерживает более одного слушателя.

Возвращается:

Sound Instance ID

Пример 1:

if global.SFX
{
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}

Приведенный выше код проверяет глобальную переменную "SFX", и если она возвращает значение true, то звук, указанный в переменной "snd_Explode", будет воспроизведен через эмиттер, указанный в переменной "s_emit", с низким приоритетом и без зацикливания.

Пример 2:

if hit == true
{
    audio_play_sound_on(s_emit, snd_Hit, false, 1, 1.3);
}

Приведенный выше код проверяет, ударили ли экземпляр, выполняющий код. Если true, он воспроизводит звук "snd_Hit" через эмиттер "s_emit" с немного большим усилением 1.3. Установленное здесь усиление умножается на усиление эмиттера, чтобы получить окончательное усиление, при котором воспроизводится звук.
