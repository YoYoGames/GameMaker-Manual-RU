audio_create_stream

С помощью этой функции вы можете создать новый звуковой индекс, который затем можно использовать в обычных аудиофункциях для потоковой передачи звука непосредственно из внешнего источника файла OGG. Функция требует имя файла (которое может быть, например, включенным файлом) и возвращает новый звуковой индекс для использования. Обратите внимание, что после того, как звук больше не нужен, необходимо вызвать функцию audio_destroy_stream() с индексом звука, чтобы удалить его из памяти, иначе вы можете получить утечку памяти, что приведет к замедлению и, в конечном счете, к краху вашей игры.

ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5.

Синтаксис:

audio_create_stream(filename);

Аргумент Тип Описание
filename Строка Path to the file (OGG only) to stream the audio from.

Возвращается:

Sound Asset

Пример:

snd = audio_create_stream("Music/Track1.ogg");
audio_play_sound(snd, 0, true);

Приведенный выше код создает новый индекс звука в переменной "snd" из заданного файла, затем воспроизводит этот звук.
