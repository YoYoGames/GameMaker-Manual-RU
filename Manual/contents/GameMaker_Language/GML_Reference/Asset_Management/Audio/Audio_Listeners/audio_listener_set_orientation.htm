audio_listener_set_orientation

С помощью этой функции вы можете изменить ориентацию данного слушателя в трехмерном аудиопространстве. По умолчанию индекс слушателя равен 0, но вы можете использовать функцию audio_get_listener_info() для получения различных индексов, доступных для целевой платформы.

Вектор взгляда и вектор вверх основаны на значениях, которые определяются из заданных относительных положений x, y и z, и по умолчанию имеют значения (0, 0, 1) для вектора взгляда и (0, 1, 0) для вектора вверх, как показано на рисунке ниже:

Изменение заданной ориентации слушателя с помощью этой функции изменит восприятие звуков, создаваемых аудиоизлучателями в игровой комнате, игроком вашей игры. В приведенном ниже примере звуки, создаваемые излучателем, когда слушатель находится в положении по умолчанию, будут казаться исходящими снизу и справа от слушателя, но при новом положении и ориентации слушателя они будут восприниматься как исходящие сверху и справа.

Синтаксис:

audio_listener_set_orientation(index, lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

Аргумент Тип Описание
index Реальный или Идентификатор аудиослушателя The listener to set the orientation of.
lookat_x Реально The x look vector (default 0).
lookat_y Реально The y look vector (default 0).
lookat_z Реально The z look vector (default 1).
up_x Реально The x up vector (default 0).
up_y Реально The y up vector (default 1).
up_z Реально The z up vector (default 0).

Возвращается:

N/A

Пример:

var _m = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _m[0], _m[1], _m[2]);
audio_listener_set_orientation(info[? "index"], _m[3], _m[4], _m[5], _m[6], _m[7], _m[8]);

Приведенный выше код извлекает матрицу вида для вида камеры [0], а затем использует ее для установки положения и ориентации аудиослушателя с идентификатором, хранящимся в глобальной переменной "Player_Listener".
