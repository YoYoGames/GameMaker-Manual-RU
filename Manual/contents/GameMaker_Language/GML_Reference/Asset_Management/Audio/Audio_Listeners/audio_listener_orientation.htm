ориентация аудио_слушателя

С помощью этой функции можно изменить ориентацию слушателя в трехмерном аудиопространстве. Направление взгляда и направление вверх основаны на векторах, которые определяются из заданных относительных положений x, y и z, и по умолчанию имеют значения (0, 0, 1) для направления взгляда и (0, 1, 0) для направления вверх, как показано на рисунке ниже:

ПРИМЕЧАНИЕ если у вас несколько слушателей, вы должны использовать функцию audio_listener_set_orientation().

Изменение ориентации слушателя с помощью этой функции изменит восприятие звуков, создаваемых аудиоизлучателями в игровой комнате, игроком вашей игры. В приведенном ниже примере звуки, создаваемые излучателем, когда слушатель находится в положении по умолчанию, будут казаться исходящими снизу и справа от слушателя, но при новом положении и ориентации слушателя они будут восприниматься как исходящие сверху и справа.

Синтаксис:

audio_listener_orientation(lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

Аргумент Тип Описание
lookat_x Реально The x look vector (default 0).
lookat_y Реально The y look vector (default 0).
lookat_z Реально The z look vector (default 1).
up_x Реально The x up vector (default 0).
up_y Реально The y up vector (default 1).
up_z Реально The z up vector (default 0).

Возвращается:

N/A

Пример:

dx = dcos(direction);
dy = - dsin(direction);
dz = - dsin(zdirection);
audio_listener_position(x, y, z);
audio_listener_orientation(dx, dy, dz, 0, 0, 1);

Приведенный выше код устанавливает положение 3D аудиослушателя в переменные x и y экземпляра и дополнительную переменную z. Затем он устанавливает ориентацию аудиослушателя на три переменные, вычисляемые из переменной экземпляра direction и дополнительной переменной zdirection.
