audio_get_listener_info

Эта функция создаст карту DS и заполнит ее информацией для заданного слушателя.

ПРИМЕЧАНИЕ Вы несете ответственность за уничтожение возвращенного DS Map с помощью соответствующей функции.

Карта DS будет содержать следующие ключи:

  • "name" - Имя слушателя в виде строки, причем "default" является стандартным именем слушателя на большинстве целевых платформ.
  • "mask" - Битовая маска для слушателя
  • "index" - Уникальное значение индекса слушателя

Значение маски можно использовать для настройки звука или излучателя на воспроизведение от нескольких слушателей одновременно, просто используя побитовое or "|" для создания маски для звука (см. пример кода ниже), в то время как индекс используется для установки свойств, таких как позиция или скорость для данного слушателя, используя функции типа audio_listener_set_position().

Синтаксис:

audio_get_listener_info(num);

Аргумент Тип Описание
num Идентификаторреального или аудиослушателя The listener number to get the data for.

Возвращается:

DS Map ID

Пример:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

Приведенный выше код проверяет количество доступных слушателей, затем перебирает их и получает биты их масок, которые затем объединяются для создания однобитовой маски, применяемой к глобальному слушателю.
