audio_set_listener_mask

При использовании нескольких слушателей в системе можно установить битовую маску для звука, чтобы он был слышен только отмеченным слушателям. Однако с помощью этой функции можно также установить глобальную маску, и все звуки, воспроизводимые в обычном режиме, будут слышны слушателям, отмеченным этой маской, без необходимости устанавливать маску для каждого звука отдельно.

Синтаксис:

audio_set_listener_mask(mask);

Аргумент Тип Описание
mask Реально The bit-mask data to set for the listeners

Возвращается:

N/A

Пример:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

Приведенный выше код проверяет количество доступных слушателей, затем перебирает их и получает биты их масок, которые затем объединяются для создания однобитовой маски, применяемой к глобальному слушателю.
