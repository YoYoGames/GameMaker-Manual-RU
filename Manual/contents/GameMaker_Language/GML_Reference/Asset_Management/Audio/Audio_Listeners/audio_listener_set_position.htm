audio_listener_set_position

С помощью этой функции вы можете изменить положение данного слушателя в трехмерном аудиопространстве. По умолчанию индекс слушателя равен 0, но вы можете использовать функцию audio_get_listener_info() для получения различных индексов, доступных для целевой платформы. Пример изображения ниже показывает положение слушателя по умолчанию в аудиопространстве:

Как вы можете видеть, положение по умолчанию - (0, 0, 0), но обычно вы используете эту функцию для перемещения слушателя вместе с объектом игрока в вашей игре и, таким образом, для изменения того, как звук, созданный эмиттерами, слышен игроку. Например, на изображении игры сверху вниз, экземпляр игрока устанавливает слушатель, который заставит звук от различных эмиттеров "меняться", когда игрок перемещается по уровню:

Синтаксис:

audio_listener_set_position(index, x, y, z);

Аргумент Тип Описание
index Идентификаторреального или аудиослушателя The listener to set the position of (default 0).
x Реально The x position of the listener (default 0).
y Реально The y position of the listener (default 0).
z Реально The z position of the listener (default 0).

Возвращается:

N/A

Пример:

var _vmat = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _vmat[0], _vmat[1], _vmat[2]);
audio_listener_set_orientation(info[? "index"], _vmat[3], _vmat[4], _vmat[5], _vmat[6], _vmat[7], _vmat[8]);

Приведенный выше код извлекает матрицу вида для вида камеры [0], а затем использует ее для установки положения и ориентации аудиослушателя с идентификатором, хранящимся в глобальной переменной "Player_Listener".
