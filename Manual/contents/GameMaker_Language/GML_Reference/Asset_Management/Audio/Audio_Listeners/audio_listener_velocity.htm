audio_listener_velocity

Эта функция может быть использована для придания слушателю эффекта Доплера и имитации движения звука на основе вектора, который определяется из заданных относительных положений x, y и z. Если сам слушатель никогда не будет двигаться, или движение не будет постоянным, вам обычно не нужно задавать эти значения, но, например, если вы делаете скроллинговую игру, где игрок имеет постоянное движение снизу вверх, а враги - сверху вниз, вы должны установить скорости слушателя и эмиттера (для эмиттеров вы будете использовать audio_emitter_velocity()) на соответствующие векторы, чтобы смоделировать правильный эффект Доплера при движении мимо экземпляра игрока.

ПРИМЕЧАНИЕ если у вас несколько слушателей, вы должны использовать функцию audio_listener_set_velocity().

На рисунке ниже показано, как это можно настроить для приведенного выше примера игры:

Синтаксис:

audio_listener_velocity(vx, vy, vz);

Аргумент Тип Описание
vx Реально The x velocity component of the listener (default 0).
vy Реально The y velocity component of the listener (default 0).
vz Реально The z velocity component of the listener (default 0).

Возвращается:

N/A

Пример:

if speed > 0
{
    audio_listener_velocity(abs(hspeed), abs(vspeed), 0);
}

Приведенный выше код проверяет, не превышает ли скорость экземпляра игрока 0, и если да, то использует соответствующие абсолютные компоненты hspeed и vspeed для установки скорости слушателя.
