audio_sync_group_get_track_pos

Эта функция возвращает текущую позицию воспроизведения заданной группы синхронизации. Индекс группы - это значение, возвращенное при создании группы с помощью функции audio_create_sync_group(), а возвращаемое значение - время в секундах, в течение которого воспроизводятся дорожки.

ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5.

Синтаксис:

audio_sync_group_get_track_pos(group_index);

Аргумент Тип Описание
group_index Идентификатор группы аудиосинхронизации The group index to get the position of.

Возвращается:

Real

Пример:

var real_secs = audio_sync_group_get_track_pos(sg);
var secs = real_secs mod 60;
var mins = string(real_secs div 60);
if (secs > 9)
{
    secs = string(secs);
}
else
{
    secs = "0" + string(secs);
}

draw_text(32, 32, "Time = " + mins + ":" + secs);

Приведенный выше код получает позицию времени для группы синхронизации, проиндексированной в переменной "sg", затем использует это значение для рисования воспроизводимого времени на экране.
