audio_create_sync_group

Создает группу синхронизации и возвращает для нее уникальное значение ID, которое затем должно использоваться во всех дальнейших вызовах аудиофункций для этой группы. Если вы хотите, чтобы группа зацикливалась, передайте true, в противном случае передайте false, но учтите, что если вы хотите, чтобы они зацикливались, все дорожки, добавленные позже, должны быть одинаковой длины. Обратите внимание, что при создании группы синхронизации необходимо освободить память и звуки, связанные с ней, если они не используются, с помощью функции audio_destroy_sync_group() например, в событиях Room End или Destroy.

ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5.

Синтаксис:

audio_create_sync_group(loop);

Аргумент Тип Описание
loop Булево Whether the tracks in the group should loop (true) или нет (false).

Возвращается:

Audio Sync Group ID

Пример:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

Вышеописанное создает новую группу синхронизации и присваивает индекс группы переменной "sg". Затем в группу добавляются четыре звука, причем усиление для трех из них устанавливается на 0. Наконец, группа синхронизации воспроизводится.
