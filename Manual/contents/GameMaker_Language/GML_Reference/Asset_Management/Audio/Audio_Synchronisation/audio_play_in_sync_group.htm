audio_play_in_sync_group

С помощью этой функции можно назначить звук на ранее созданную группу синхронизации. Индекс группы - это значение, возвращенное при создании группы с помощью функции audio_create_sync_group(), а индекс звука - это индекс любого сжатого звука (*.ogg или *.mp3) из браузера активов. Эта функция только добавляет звук в группу, но не воспроизводит его. Возвращаемое значение будет >= 0 в случае успеха и -1 в случае неудачи, где успешное значение указывает порядок звука в группе синхронизации (таким образом, 0 будет первым добавленным звуком, 1 - вторым и т.д.).

ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5.

Синтаксис:

audio_play_in_sync_group(group_index, sound_index);

Аргумент Тип Описание
group_index Идентификатор группы аудиосинхронизации The group index to assign the sound to.
sound_index Звуковой актив The sound index to be assigned to the group.

Возвращается:

Real

Пример:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_play_in_sync_group(sg, sound3);
audio_play_in_sync_group(sg, sound4);
audio_start_sync_group(sg);

Выше приведено создание новой группы синхронизации и присвоение индекса группы переменной "sg". Затем в группу добавляются четыре звука, причем усиление для трех из них устанавливается на 0. Наконец, группа синхронизации воспроизводится.
