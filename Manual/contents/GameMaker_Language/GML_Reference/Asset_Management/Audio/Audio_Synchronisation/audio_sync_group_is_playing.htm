audio_sync_group_is_playing

Эта функция может использоваться для проверки того, воспроизводится ли какой-либо звук в синхронизированной группе. Вам необходимо предоставить идентификатор группы синхронизации, возвращаемый функцией audio_create_sync_group().

Эта функциональность недоступна для целевой платформы HTML5.

Синтаксис:

audio_sync_group_is_playing(group_index);

Аргумент Тип Описание
group_index Идентификатор группы аудиосинхронизации The group index to check.

Возвращается:

Boolean

Пример:

if audio_sync_group_is_playing(group_one)
{
    audio_stop_sync_group(group_one);
}

Приведенный выше код проверяет, воспроизводится ли в данный момент группа синхронизации group_one, и если да, то группа останавливается.
