audio_sync_group_debug

его функцию можно использовать для отображения отладочной информации о любой заданной группе синхронизации (индекс группы - это значение, возвращаемое при создании группы с помощью функции audio_create_sync_group()). Вы можете вызвать функцию со значением -1, чтобы отключить наложение.

При отладке групп синхронизации звука будет отображаться накладка:

  • Какая группа синхронизации отлаживается
  • Сколько каналов в нем
  • Сколько раз группа синхронизации терпела неудачу (это означает, что она пыталась выполнить слишком большой объем работы)
  • Список каналов
    • Красный цвет означает, что данный канал в настоящее время не выполняет никакой работы
    • Желтый означает, что канал находится в начальном состоянии (еще не декодирует и не играет)
    • Зеленый означает, что канал в настоящее время ищет/декодирует
    • Фиолетовый означает, что на канале закончились данные.
  • Полоса профилирования, показывающая, какую работу выполняет поток декодирования
    • синий цвет означает, что файл читается в первый раз
    • красный означает, что поток читает файл, чтобы узнать, откуда читать данные
    • зеленый означает, что поток декодирует данные
    • фиолетовый означает, что поток находится в состоянии сна до тех пор, пока ему не понадобится работа.

Обратите внимание, что для общей отладки аудио у вас также есть функция audio_debug().

ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5.

Синтаксис:

audio_sync_group_debug(group_index);

Аргумент Тип Описание
group_index Идентификатор группы аудиосинхронизации The group index to debug, or -1 to switch off the debugging.

Возвращается:

N/A

Пример:

if (debug_mode)
{
    audio_sync_group_debug(sg);
}
else
{
    audio_sync_group_debug(-1);
}

Приведенный выше код будет включать или выключать отладочное наложение группы синхронизации звука, показывающее группу синхронизации, индексированную в переменной "sg", в зависимости от того, запущена игра в режиме отладки или нет.
