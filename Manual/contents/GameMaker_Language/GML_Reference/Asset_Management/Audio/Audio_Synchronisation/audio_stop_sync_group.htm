audio_stop_sync_group

Эта функция остановит заданную группу синхронизации, если она играет, при этом индекс группы будет значением, возвращенным при создании группы с помощью функции audio_create_sync_group().

ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5.

Синтаксис:

audio_stop_sync_group(group_index);

Аргумент Тип Описание
group_index Идентификатор группы аудиосинхронизации The group index to stop playing.

Возвращается:

N/A

Пример:

if mouse_check_button_pressed(mb_left)
{
    audio_stop_sync_group(sg);
}

Приведенный выше код проверяет наличие щелчка мыши, и если он обнаружен, то останавливает группу синхронизации, проиндексированную в переменной "sg".
