Синхронизация звука

Функции группы синхронизации звука позволяют воспроизводить несколько звуков, которые начинаются в одно и то же время и остаются синхронизированными (на уровне каждого сэмпла). Это позволяет, например, добавлять различные дорожки для одной песни и затем включать и выключать секции, не беспокоясь о том, будет ли между ними задержка. Обратите внимание, что в настоящее время эти функции работают только со сжатым аудио, т.е. с файлами формата *.ogg и *.mp3.

ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5.

Для синхронизации звуковых дорожек существуют следующие функции:

  • audio_create_sync_group
  • audio_play_in_sync_group
  • audio_start_sync_group
  • audio_stop_sync_group
  • audio_pause_sync_group
  • audio_resume_sync_group
  • audio_sync_group_get_track_pos
  • audio_destroy_sync_group
  • audio_sync_group_debug
  • audio_sync_group_is_playing
