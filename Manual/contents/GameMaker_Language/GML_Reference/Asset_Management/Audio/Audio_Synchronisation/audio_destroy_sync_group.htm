audio_destroy_sync_group

Группы синхронизации звука должны быть уничтожены, если они не используются, чтобы освободить память и звуковые ресурсы, связанные с ними, с помощью этой функции. Для этого берется индекс группы, полученный при ее создании с помощью функции audio_create_sync_group()и освобождает все ресурсы, используемые группой.

ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5.

Синтаксис:

audio_destroy_sync_group(group_index);

Аргумент Тип Описание
group_index Идентификатор группы аудиосинхронизации The group index to be destroyed.

Возвращается:

N/A

Пример:

audio_destroy_sync_group(sg);

Приведенный выше код уничтожает группу синхронизации, индексированную в переменной "sg", и, вероятно, будет использоваться в событиях destroy или Room End.
