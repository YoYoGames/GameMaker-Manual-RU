animcurve_exists

Эта функция возвращает true, если заданный Ассистент анимационной кривой или Структура кривой анимации существует и является действительной кривой анимации, или false, если не существует.

Синтаксис:

animcurve_exists(curve_struct_or_id);

Аргумент Тип Описание
curve_struct_or_id Структура анимационной кривой или Ассет анимационной кривой The Animation Curve that will be checked.

Возвращается:

Boolean

Пример:

if (animcurve_exists(spring_curve))
{
    apply_spring_animation(sprite_curve);
}

Приведенный выше код проверяет, существует ли кривая анимации, сохраненная в пользовательской переменной spring_curve. Если существует, то запускается пользовательский метод для использования этой кривой анимации.
