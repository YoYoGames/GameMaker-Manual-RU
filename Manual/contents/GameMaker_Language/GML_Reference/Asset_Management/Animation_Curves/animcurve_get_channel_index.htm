animcurve_get_channel_index

Эта функция возвращает значение индекса для любого заданного канала кривой анимации. Вы передаете ID кривой анимации или struct, где ID кривой - это имя кривой анимации, как она была определена в Asset Browser, или указатель struct, возвращенный функцией animcurve_create(). Затем введите имя канала в виде строки, и функция вернет значение индекса, связанное с этим каналом. Обратите внимание, что если кривая или канал не существуют, то вы получите ошибку.

Синтаксис:

animcurve_get_channel_index(curve_struct_or_id, channel_name);

Аргумент Тип Описание
curve_struct_or_id Структура анимационной кривой The ID or struct pointer of the animation curve to target
channel_name Строка The channel name (a string).

Возвращается:

Real

Пример:

var _channelindex = animcurve_get_channel_index(ac_ButtonTween, "x_pos")
var _channeldata = animcurve_get_channel(ac_ButtonTween, _channelindex);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

Приведенный выше код извлекает структуру канала для канала с именем "x_pos" в активе кривой "ac_ButtonTween", затем перебирает точки на кривой канала и добавляет единицу к их значению.
