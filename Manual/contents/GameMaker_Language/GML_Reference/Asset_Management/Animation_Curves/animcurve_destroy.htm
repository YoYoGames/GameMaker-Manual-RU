animcurve_destroy

Эта функция может быть использована для уничтожения любых анимационных кривых, которые были ранее созданы с помощью функции animcurve_create(). Вызов этой функции удалит анимационную кривую из памяти и очистит все каналы или точки, которые она содержит (их никогда не нужно очищать вручную). Обратите внимание, что нельзя уничтожить анимационные кривые, созданные в браузере активов, только те, которые были созданы во время выполнения.

Синтаксис:

animcurve_destroy(curve_struct);

Аргумент Тип Описание
curve_struct Структура анимационной кривой The pointer to the curve struct to destroy.

Возвращается:

N/A

Пример:

animcurve_destroy(my_curve);

Приведенный выше код уничтожит (ранее созданную) структуру кривой анимации, индексированную в переменной "my_curve".
