animcurve_get_channel

Эта функция возвращает структуру, содержащую данные канала для канала, указанного в активе кривой анимации. Вы передаете идентификатор или структуру анимационной кривой, а также имя или индекс канала, и функция возвращает структуру со следующим форматом:

Имя переменной Тип данных Описание
name строка Это название канала.
type Тип интерполяции кривой анимации Постоянная Это будет одна из констант animcurvetype_linear для линейной интерполяции между точками, animcurvetype_catmullrom для "гладкой" интерполяции между точками с помощью интерполяции катмул-ром или animcurvetype_bezier для интерполяции Безье.
iterations целое число Если канал использует интерполяцию катмул-ром ("гладкую"), это значение показывает, сколько точек было сгенерировано для каждого сегмента кривой (обратите внимание, что эти дополнительные точки являются внутренними для функции и используются только для расчетов во время выполнения). Если канал использует линейную интерполяцию, это значение все равно будет существовать, но его можно игнорировать, так как оно не влияет на то, как интерполируется кривая.
points указатель массива Это массив, где каждый элемент массива является структурой точки.


ID кривой анимации - это имя кривой анимации, определенное в браузере активов, или указатель struct, возвращаемый функцией animcurve_create(). Имя канала - это строка, которая ссылается на канал, как он был определен в активе кривой анимации, или вы можете передать значение индекса, которое составляет от 0 до n, где n - последний канал в активе кривой (например: если кривая анимации имеет 4 канала, они будут проиндексированы от 0 до 3). Обратите внимание, что передача значения индекса требует меньше обработки, чем передача имени канала. Если функция не сработает (т.е. не существует канала с заданным именем или индексом), то вы получите ошибку.

Точки одного канала хранятся в виде структур в массиве, где каждый элемент массива является структурой одной точки. Структура точки имеет следующие переменные:

Имя переменной Тип данных Описание
posx настоящий Положение во времени (нормированное от 0 до 1) точки.
value настоящий Значение точки.

Синтаксис:

animcurve_get_channel(curve_struct_or_id, channel_name_or_index);

Аргумент Тип Описание
curve_struct_or_id Структура анимационной кривой The ID or struct pointer of the animation curve to target
channel_name_or_index Строка или Реальный The channel name (a string) or the channel index (an integer).

Возвращается:

Animation Curve Channel Struct

Пример:

var _channeldata = animcurve_get_channel(ac_ButtonTween, 0);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

Приведенный выше код извлекает структуру канала для канала 0 в активе кривой "ac_ButtonTween", затем проходит по точкам кривой канала и добавляет единицу к их значению.
