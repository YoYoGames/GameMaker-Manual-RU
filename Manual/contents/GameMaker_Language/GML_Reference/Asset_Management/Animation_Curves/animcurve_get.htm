animcurve_get

Эта функция возвращает структуру, содержащую все данные для заданной кривой анимации. Вы передаете идентификатор актива кривой анимации (как определено в браузере активов), и функция возвращает структуру со следующими переменными:

Структура анимационной кривой
Имя переменной Тип данных Описание
name Строка Это имя кривой анимации.
channels Массив из Структура канала кривой анимацииs Это массив, где каждый элемент массива является структурой канала.


Переменная channels в структуре представляет собой массив, где каждый элемент массива является элементом Структура канала анимационной кривой с данными, относящимися к каналу в кривой. Структура канала объясняется на этой странице.

Как и в случае с каналами, точки одного канала хранятся как структуры в массиве, где каждый элемент в массиве является структурой одной точки. Структура точки объясняется на этой странице.

Обратите внимание, что если функция не сработает (например, заданный актив Animation Curve не существует), то функция вернет -1.

Синтаксис:

animcurve_get(curve_id);

Аргумент Тип Описание
curve_id Ассет анимационной кривой The asset browser ID (index) of the animation curve to get.

Возвращается:

Animation Curve Struct or -1

Пример:

var _curve = animcurve_get(ac_ButtonTween);
var _channel = _curve.channels[0];
if (_channel.type != animcurvetype_linear)
{
    _channel.type = animcurvetype_linear;
}

Приведенный выше код извлекает структуру кривой анимации для актива кривой "ac_ButtonTween", затем проверяет тип кривой для первого канала, и если он не является линейным, то устанавливает его на линейный.
