asset_remove_tags

С помощью этой функции вы можете удалить одну или несколько строк тегов для любого актива из браузера активов. Вы передаете либо имя актива (в виде строки), либо его индекс, а также либо одну строку тегов, либо массив, каждый элемент которого является одной строкой тегов. Если вы указываете значение индекса актива, то вам необходимо указать дополнительный аргумент тип актива (константа), так как активы разных типов могут иметь одинаковый индекс, хотя они не могут иметь одинаковое имя. Доступные типы активов перечислены в таблице ниже:

Постоянная Описание
asset_object Данное имя относится к объекту.
asset_sprite Данное имя относится к спрайту.
asset_sound Данное имя обозначает звук.
asset_room Данное имя относится к room.
asset_tiles Данное название относится к набору плиток.
asset_path Данное имя относится к пути.
asset_script Данное имя относится к script.
asset_font Данное название относится к шрифту.
asset_timeline Данное имя относится к временной линии.
asset_shader Данное имя относится к шейдеру.
asset_animationcurve Данное имя относится к кривой анимации.
asset_sequence Данное имя относится к последовательности.

Если функция успешно удаляет метку (метки), она возвращает true, в противном случае возвращает false.

Синтаксис:

asset_remove_tags(name_or_index, tags, [asset_type]);

Аргумент Тип Описание
name_or_index Строка или актив The name of the asset (a string) or its index value (an integer).
tags Строка или массив строк A single asset tag string or an array with various asset tags.
[asset_type] Тип актива Постоянная OPTIONAL! The type of asset to remove the tags from, only used when supplying an index for the first argument.

Возвращается:

Boolean

Пример:

var _a = array_create(3);
_a[0] = "enemy";
_a[1] = "all_levels";
_a[2] = "boss";
asset_remove_tags(obj_Enemy_Boss_Parent, _a, asset_object);

Приведенный выше код создаст массив тегов, а затем удалит их из заданного объекта.
