asset_get_tags

С помощью этой функции вы можете получить все теги, присвоенные активу, из браузера активов. Вы указываете либо имя актива (в виде строки), либо его индекс, и функция возвращает массив тегов для этого актива. Если теги не найдены или произошла ошибка (т.е. указанная строка имени не существует), то возвращаемый массив будет пуст. Если вы указываете значение индекса актива, то вам необходимо указать дополнительный аргумент тип актива (константа), так как активы разных типов могут иметь одинаковый индекс, хотя они не могут иметь одинаковое имя. Доступные типы активов перечислены в таблице ниже:

Постоянная Описание
asset_object Данное имя относится к объекту.
asset_sprite Данное имя относится к спрайту.
asset_sound Данное имя обозначает звук.
asset_room Данное имя относится к room.
asset_tiles Данное название относится к набору плиток.
asset_path Данное имя относится к пути.
asset_script Данное имя относится к script.
asset_font Данное название относится к шрифту.
asset_timeline Данное имя относится к временной линии.
asset_shader Данное имя относится к шейдеру.
asset_animationcurve Данное название относится к кривой анимации.
asset_sequence Данное имя относится к последовательности.

Синтаксис:

asset_get_tags(name_or_index, [asset_type]);

Аргумент Тип Описание
name_or_index Строка или актив The name of the asset (a string) or its index.
[asset_type] Тип актива Постоянная OPTIONAL! The type of asset to get the tags for, only used when supplying an index for the first argument.

Возвращается:

Array

Пример:

my_tags = asset_get_tags(object_get_name(object_index));

Приведенный выше код получит все теги, назначенные объекту, на основе которого был создан экземпляр, выполняющий этот код.
