тип_актива

С помощью этой функции вы можете получить тип актива, на который ссылаются, из его имени (строки). При этом будет возвращена одна из перечисленных ниже констант.

Синтаксис:

asset_get_type(name);

Аргумент Тип Описание
name Строка The name of the game asset to get the type of (a string).

Возвращается:

Asset Type Constant

Постоянная Описание
asset_object Данное имя относится к объекту.
asset_sprite Данное имя относится к спрайту.
asset_sound Данное имя обозначает звук.
asset_room Данное имя относится к room.
asset_tiles Данное название относится к набору плиток.
asset_path Данное имя относится к пути.
asset_script Данное имя относится к script.
asset_font Данное название относится к шрифту.
asset_timeline Данное имя относится к временной линии.
asset_shader Данное имя относится к шейдеру.
asset_animationcurve Данное название относится к кривой анимации.
asset_sequence Данное имя относится к последовательности.
asset_unknown Данное имя относится к активу, который либо не существует, либо не является одним из перечисленных выше.

Пример:

if asset_get_type("pth_Path_" + string(global.Game)) == asset_unknown
{
    show_debug_message("Path doesn't exist!!!");
}
else
{
    path_index = asset_get_index("pth_Path_" + string(global.Game));
}

Приведенный выше код проверяет динамически созданное имя актива, чтобы убедиться, что актив имеет правильный тип. Если это не так, то будет показано отладочное сообщение, в противном случае имя актива используется для назначения актива экземпляру.
