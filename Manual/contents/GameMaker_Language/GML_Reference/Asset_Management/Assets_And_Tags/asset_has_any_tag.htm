актив_имеет_любой_тег

С помощью этой функции вы можете проверить, присвоена ли одна или несколько строк тегов любому активу из браузера активов. Вы передаете либо имя актива (в виде строки), либо его индекс актива, а также либо одну строку тега, либо массив, каждый элемент которого является одной строкой тега. Если вы указываете значение индекса актива, то вам необходимо указать дополнительный аргумент тип актива (константа), так как активы разных типов могут иметь одинаковый индекс, хотя они не могут иметь одинаковое имя. Доступные типы активов перечислены в таблице ниже:

Постоянная Описание
asset_object Данное имя относится к объекту.
asset_sprite Данное имя относится к спрайту.
asset_sound Данное имя обозначает звук.
asset_room Данное имя относится к room.
asset_tiles Данное название относится к набору плиток.
asset_path Данное имя относится к пути.
asset_script Данное имя относится к script.
asset_font Данное название относится к шрифту.
asset_timeline Данное имя относится к временной линии.
asset_shader Данное имя относится к шейдеру.
asset_animationcurve Данное имя относится к кривой анимации.
asset_sequence Данное имя относится к последовательности.

Если функция работает успешно и один или несколько тегов присутствуют для актива, то возвращается true, в противном случае возвращается false. Если вам нужно проверить точное соответствие какому-либо тегу или набору тегов, то используйте функцию asset_has_tags().

Синтаксис:

asset_has_any_tag(name_or_index, tags, [asset_type]);

Аргумент Тип Описание
name_or_index Строка или Актив The name of the asset (a string) or its index value (an integer).
tags Строка или Массив строк A single asset tag string or an array with various asset tags.
[asset_type] Тип актива Постоянная OPTIONAL! The type of asset to check the tags for, only used when supplying an index for the first argument.

Возвращается:

Boolean

Пример:

var _a = array_create(3);
_a[0] = "boss";
_a[1] = "main_boss";
_a[2] = "final_boss";

if asset_has_any_tag(object_index, _a, asset_object)
{
    instance_create_layer(0, 0, "Overlay", obj_Boss_Text);
}

Приведенный выше код создаст массив тегов, затем проверит, применяются ли какие-либо из них к данному объекту, и если да, то создаст экземпляр другого объекта.
