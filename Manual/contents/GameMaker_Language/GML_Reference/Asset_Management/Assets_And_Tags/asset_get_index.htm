asset_get_index

Вы можете использовать эту функцию для получения уникального идентификационного индекса игрового актива из его названия. Если актив не найден, функция вернет значение -1, в противном случае она вернет уникальный индекс id для проверяемого актива. Этот идентификатор можно использовать в других функциях, как и любое другое значение индекса, например, функцию sprite_index или path_indexнапример. Обратите внимание, что хотя эта функция может использоваться для ссылки на активы из строк (см. пример ниже), вы всегда должны убедиться, что актив существует, прежде чем использовать его, иначе вы можете получить ошибки, которые приведут к краху вашей игры.

Синтаксис:

asset_get_index(name);

=======

asset_get_index

This function gets the unique identifying index for a game asset from its name.

If the asset is not found, the function will return a value of -1, otherwise it will return the unique index ID for the asset being checked. This ID can then be used in other functions as you would any other index value, like the sprite_index or the path_index, for example. Please note that although this function can be used to reference assets from strings (see example below), you should always make sure that the asset exists before using it otherwise you may get errors that will crash your game.

 

Syntax:

asset_get_index(name);

Asset (any asset type)

Пример:

var obj = asset_get_index("obj_Enemy_" + string(global.Level));
if obj > -1
{
    instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
}

Приведенный выше код получит индекс объекта из строки, и если этот индекс существует, создаст экземпляр объекта в игре.
