tag_get_asset_ids

С помощью этой функции вы можете получить все активы заданного типа, которым присвоены заданные теги. Вы предоставляете либо один тег (в виде строки), либо массив, где каждый элемент массива является тегом (в виде строки), а также тип актива для проверки. тип актива должен быть одной из следующих констант:

Постоянная Описание
asset_object Данное имя относится к объекту.
asset_sprite Данное имя относится к спрайту.
asset_sound Данное имя обозначает звук.
asset_room Данное имя относится к room.
asset_tiles Данное название относится к набору плиток.
asset_path Данное имя относится к пути.
asset_script Данное имя относится к script.
asset_font Данное название относится к шрифту.
asset_timeline Данное имя относится к временной линии.
asset_shader Данное имя относится к шейдеру.
asset_animationcurve Данное имя относится к кривой анимации.
asset_sequence Данное имя относится к последовательности.

Функция вернет массив, где каждый элемент массива будет одним значением индекса актива. Если не существует активов данного типа, имеющих заданный тег(ы), будет возвращен пустой массив.

Синтаксис:

tag_get_asset_ids(tags, asset_type);

Аргумент Тип Описание
tags Строка или массив строк A single asset tag string or an array with various asset tags.
asset_type Тип актива Постоянная An asset type constant (listed above)

Возвращается:

Array

Пример:

var _paths = tag_get_asset_ids("enemy", asset_path);
var _num = irandom(array_length(_paths) - 1);
path_start(_paths[_num], 1, path_action_reverse, false);

Приведенный выше код использует тег "enemy" для поиска всех активов пути с этим тегом, затем выбирает один случайным образом и назначает его экземпляру, выполняющему код.
