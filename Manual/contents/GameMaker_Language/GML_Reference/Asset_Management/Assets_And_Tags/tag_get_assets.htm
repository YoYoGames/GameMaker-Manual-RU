tag_get_assets

С помощью этой функции можно получить имена всех активов, которым был присвоен заданный тег или теги. Вы передаете либо одну строку тега, либо массив, где каждый элемент массива - это строка тега. Функция вернет массив, каждый элемент которого является именем (в виде строки) актива с заданным тегом. Если вам нужен уникальный индекс актива, то вы можете использовать функцию asset_get_index() вместе с возвращаемым именем. Если активов с заданным тегом (тегами) нет, или если произошла ошибка (например, заданные теги не существуют), то будет возвращен пустой массив.

Синтаксис:

tag_get_assets(tags);

=======

tag_get_assets

This function retrieves the names of all assets that have been assigned the given tag or tags.

You supply either a single tag string or an Array, where each item in the array is a tag string. The function will return an array where each entry is the name (as a string) of the asset with the given tag. If you need the unique index for the asset, then you can use the function asset_get_index along with the returned name. If there are no assets with the given tag(s), or if there is an error (e.g.: the given tags don't exist), then an empty array will be returned.

 

Syntax:

tag_get_assets(tags);

Он циклически просматривает массив возвращенных имен, проверяя, являются ли какие-либо из них активами sprite, и если да, то уникальное значение индекса для актива добавляется в список для дальнейшего использования.
