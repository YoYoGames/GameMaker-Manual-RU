asset_clear_tags

С помощью этой функции вы можете очистить все теги, присутствующие на любом активе, из браузера активов. Вы указываете либо имя актива (в виде строки), либо его индекс, а если вы указываете значение индекса актива, то вам необходимо указать дополнительный аргумент типа актива (константа), так как активы разных типов могут иметь одинаковый индекс, хотя они не могут иметь одинаковое имя. Доступные типы активов перечислены в таблице ниже:

Постоянная Описание
asset_object Данное имя относится к объекту.
asset_sprite Данное имя относится к спрайту.
asset_sound Данное имя обозначает звук.
asset_room Данное имя относится к room.
asset_tiles Данное название относится к набору плиток.
asset_path Данное имя относится к пути.
asset_script Данное имя относится к script.
asset_font Данное название относится к шрифту.
asset_timeline Данное имя относится к временной линии.
asset_shader Данное имя относится к шейдеру.
asset_animationcurve Данное имя относится к кривой анимации.
asset_sequence Данное имя относится к последовательности.

Если функция успешно очистит метки, она вернет true, в противном случае она вернет false.

Синтаксис:

asset_clear_tags(name,[asset_type]);

Аргумент Тип Описание
name_or_index Строка или актив The name of the asset (a string) or its index value (an integer).
[asset_type] Тип актива Постоянная OPTIONAL! The type of asset to clear the tags from, only used when supplying an index for the first argument.

Возвращается:

Boolean

Пример:

asset_clear_tags(obj_Enemy_Parent, asset_object);

Приведенный выше код очистит все теги из заданного объекта-актива.
