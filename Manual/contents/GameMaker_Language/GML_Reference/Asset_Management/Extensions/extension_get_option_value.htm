extension_get_option_value

Эта функция возвращает значение заданной опции в расширении с заданным именем.

Функция вернет неопределенное значение, если предоставленное имя расширения недействительно.

Синтаксис:

extension_get_option_value(extension_name, option_name);

Аргумент Тип Описание
extension_name Строка The name of your extension asset as a string
option_name Строка The name of the option to read

Возвращается:

Any

Пример:

var _enabled = extension_get_option_value("MyExtension", "enabled");

Этот код получает значение опции enabled в расширении MyExtension.
