extension_get_options

Эта функция возвращает структуру, содержащую все опции и их значения для расширения с заданным именем.

Функция вернет неопределенное значение, если предоставленное имя расширения недействительно.

Синтаксис:

extension_get_options(extension_name);

Аргумент Тип Описание
extension_name Строка The name of your extension asset as a string

Возвращается:

Struct

Пример:

var _options = extension_get_options("MyExtension");
var _enabled = _options.enabled;

Этот код получает структуру options для расширения и считывает из нее значение.
