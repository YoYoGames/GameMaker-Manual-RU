extension_get_option_names

Эта функция возвращает массив, содержащий имена всех опций, существующих в расширении с заданным именем.

Функция вернет неопределенное значение, если предоставленное имя расширения недействительно.

Синтаксис:

extension_get_option_names(extension_name);

Аргумент Тип Описание
extension_name Строка The name of your extension asset as a string

Возвращается:

Array of Strings

Пример:

var _names = extension_get_option_names("MyExtension");
var _count = extension_get_option_count("MyExtension");

for (var i = 0; i < _count; i ++)
{
    var _option_name = _names[i];

    show_debug_message(_option_name + ": ");
    show_debug_message(extension_get_option_value("MyExtension", _option_name));
}

Этот код получает имена всех опций и количество опций в расширении MyExtension.

Он перебирает все имена опций и печатает имя и значение каждой опции в журнал вывода.
