физика_частицы_создать

С помощью этой функции вы можете создать одну частицу в любом месте вашей игровой комнаты, установив определенные флаги и свойства. Флаги" являются возвращаемым значением комбинации следующих констант:

Физика Частица Флаг Константа
Постоянная Описание
phy_particle_flag_water Свойства по умолчанию для частицы мягкого тела.
phy_particle_flag_zombie Частица-зомби - это частица, которая будет уничтожена после одного шага, при этом все остальные будут помечены таким образом.
phy_particle_flag_wall Это определяет частицу как статическую, по сути, создавая ее как неподвижный объект в физическом моделировании, поскольку она будет оставаться в фиксированном положении независимо от того, что с ней столкнется. Вы должны использовать этот флаг, а не устанавливать плотность на 0.
phy_particle_flag_spring Пружинные частицы создают эффект прикрепления друг к другу, как будто пружиной. Частицы, созданные с помощью этого флага, "соединяются" попарно, причем каждая частица соединяется с той, которая была ближе всего к ней в момент создания. Попав в пару, частицы не меняют "партнеров", и чем дальше внешняя сила отталкивает их друг от друга, тем с большей силой они столкнутся, когда эта внешняя сила будет снята. Обратите внимание, что как бы далеко ни удалились парные частицы друг от друга, связь между ними не разорвется.
phy_particle_flag_elastic Упругие частицы деформируются и могут также отскакивать при столкновении с другими жесткими телами в физической симуляции.
phy_particle_flag_viscous Вязкая частица - это частица, которая проявляет "цепкость" или "липкость", как масло. Вязкие частицы больше слипаются и склеиваются.
phy_particle_flag_powder Частицы порошка создают эффект рассеивания, подобный тому, который можно наблюдать при использовании песка или пыли.
phy_particle_flag_tensile Растягивающие частицы используются для создания эффекта поверхностного натяжения, или тугой кривизны на поверхности тела жидкости. Они могут использоваться, например, для создания поверхностного натяжения, которое можно наблюдать на капле воды. Когда натяжение нарушается, частицы подпрыгивают, как если бы они были упругими, но при этом продолжают притягиваться друг к другу. В результате частицы, отскакивая, образуют скопления.
phy_particle_flag_colourmixing Частицы, смешивающие цвета, принимают некоторые цвета других частиц, с которыми они сталкиваются. Обратите внимание, что если только одна из двух сталкивающихся частиц является смешивающей цвет, другая частица сохраняет свой цвет до столкновения.

Эти флаги используют битовую маскировку для создания конечного выходного значения, которое затем проверяется для установки различных базовых свойств частицы (при этом базовым свойством всегда является свойство phy_particle_flag_water). Например, если вы хотите смоделировать вязкую жидкость с поверхностным натяжением, вы можете использовать побитовую маску или "|", чтобы отметить соответствующие биты, как показано в примере ниже. Таким образом, вы можете установить различные свойства (кроме глобальных) для каждой отдельной создаваемой частицы.

Кроме флагов, вы также можете задать положение в комнате, в котором будет создана частица, начальную горизонтальную и вертикальную скорость, начальный цвет и альфу, а также категорию пользователя. Значения цвета и альфа используются только при рисовании частицы с помощью функции physics_particle_draw()но значение категории используется во многих других функциях. Это произвольное целочисленное значение, которое вы присваиваете каждой частице, имеющей схожие свойства, и которое может быть использовано для установки флагов в дальнейшем для этой конкретной категории, а также для ряда других вещей. Например, если вы определили частицы "вода" и "масло", вы можете присвоить всем частицам, помеченным как "вода", категорию пользователя 1, а всем частицам, созданным как "масло", категорию пользователя 2, что позволит вам позже выбрать одну категорию, чтобы изменить ее флаги или получить конкретные данные.

ПРИМЕЧАНИЕ: Вы можете использовать любое целочисленное значение для категории, кроме 0, которое зарезервировано симуляцией и используется для выбора всех категорий в других функциях (например, для рисования).

Функция возвращает уникальное значение индекса (или ID) для частицы, которое может быть сохранено в переменной для использования в других функциях, напрямую ссылающихся на частицу.

Синтаксис:

physics_particle_create(flags, x, y, xv, yv, col, alpha, category)

Аргумент Тип Описание
flags Физика Частица Флаг Константа(s) The flags to set on the particle.
x Реально The x position to create the particle at.
y Реально The y position to create the particle at.
xv Реально The initial horizontal velocity.
yv Реально The initial vertical velocity.
col Цвет The base colour to use for the particle.
alpha Реально The base alpha to use for the particle.
category Реально The user defined category to which the particle belongs.

Возвращается:

Physics Particle ID

Пример:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var x_vel = lengthdir_x(5, image_angle);
var y_vel = lengthdir_y(5, image_angle);
var p = physics_particle_create(flags, x, y, x_vel, y_vel, c_white, 1, 1);

Приведенный выше код создаст переменные для установки типа частицы и ее начальной скорости, а затем использует их для создания частицы в позиции экземпляра x/y.
