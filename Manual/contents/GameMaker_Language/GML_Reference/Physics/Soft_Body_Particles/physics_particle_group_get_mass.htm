physics_particle_group_get_mass

С помощью этой функции можно получить массу целой группы частиц. Значение group - это значение, которое было возвращено при создании группы частиц с помощью функции physics_particle_group_end().

Синтаксис:

physics_particle_group_get_mass(group)

Аргумент Тип Описание
group Идентификатор группы физических частиц The particle group to get.

Возвращается:

Real

Пример:

var _mass = physics_particle_group_get_mass(group1);

Приведенный выше код получит массу группы частиц, индексированных в переменной "group1", и сохранит ее в переменной.
