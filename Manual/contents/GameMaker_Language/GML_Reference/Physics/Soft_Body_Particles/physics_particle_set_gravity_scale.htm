physics_particle_set_gravity_scale

С помощью этой функции вы можете установить коэффициент масштаба гравитации для частиц в физическом моделировании. Функция предназначена для предотвращения нестабильности в физическом моделировании, особенно при использовании очень маленьких частиц, которые могут вести себя непредсказуемо (т.е. нарушать сохранение импульса) в таких сценариях, как взрывы. Замедление этих частиц за счет уменьшения масштаба гравитации может стабилизировать их поведение. Эта функция является глобальной, поскольку она изменяет шкалу гравитации не только для новых частиц, созданных после изменения, но и для тех, которые уже присутствуют в моделировании.

ПРИМЕЧАНИЕ: Регулировка количества итераций обновления на шаг (с помощью функции physics_world_update_iterations() может также влиять на воздействие гравитации на частицы. Большее число итераций обеспечивает большую устойчивость к гравитации.

Синтаксис:

physics_particle_set_gravity_scale(scale)

АргументТип Описание
scale The scaling factor to be applied to gravity for all particles.

Возвращается:

Пример:

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

Приведенный выше код установит базовые свойства для всех частиц в симуляции.
