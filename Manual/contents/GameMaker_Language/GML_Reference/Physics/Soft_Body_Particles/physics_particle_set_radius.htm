physics_particle_set_radius

С помощью этой функции можно задать радиус (в пикселях) для частиц в физическом моделировании. Эта функция является глобальной, так как она изменяет радиус не только для новых частиц, созданных после изменения, но и для тех, которые уже присутствующих в симуляции.

Синтаксис:

physics_particle_set_radius(radius)

АргументТип Описание
radius The radius (in pixels) of the particle fixture.

Возвращается:

Пример:

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

Приведенный выше код установит базовые свойства для всех частиц в симуляции.
