physics_particle_get_gravity_scale

С помощью этой функции вы можете узнать, каков текущий коэффициент масштаба гравитации для частиц в физическом моделировании (вы можете установить это значение с помощью функции physics_particle_set_gravity_scale()).

Синтаксис:

physics_particle_get_gravity_scale()

Возвращается:

Пример:

physics_particle_set_gravity_scale(physics_particle_get_gravity_scale() + 0.1);

Приведенный выше код установит шкалу гравитации.
