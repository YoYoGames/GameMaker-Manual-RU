physics_particle_group_polygon

Эта функция задает форму создаваемой группы частиц. Сначала вы должны начать определение группы с помощью функции physics_particle_group_begin(), а затем вы использовать эту функцию, чтобы сообщить GameMaker, что вы собираетесь добавить приблизительные точки, которые будут использоваться для определения формы многоугольника - приблизительные, потому что точная форма будет зависеть от размера базовых частиц, определяемого функцией функция physics_particle_set_radius() поскольку физическая симуляция пытается "вписать" как можно больше частиц в заданную форму.

Затем вы определяете точки многоугольника, вызывая функцию physics_particle_group_add_point()и задаете положение x/y в комнате для каждой из них (минимум 3, максимум 8, и они могут быть определены как вогнутые, но точки внутри фигуры будут проигнорированы). Наконец, вам нужно вызвать physics_particle_group_end() для создания группы частиц в комнате.

Синтаксис:

physics_particle_group_polygon()

Возвращается:

Пример:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_polygon();
physics_particle_group_add_point(200, 200);
physics_particle_group_add_point(300, 300);
physics_particle_group_add_point(100, 300);
mLastGroup = physics_particle_group_end();

Приведенный выше код сохраняет флаги для типа частицы и свойства группы частиц в переменных, а затем использует их для создания группы частиц полигона с тремя сторонами в позиции мыши.
