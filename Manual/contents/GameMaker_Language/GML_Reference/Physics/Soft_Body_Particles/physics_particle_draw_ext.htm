physics_particle_draw_ext

Эта функция нарисует sprite в позиции всех частиц, которые имеют одно и то же значение категории, определенное пользователем, и у которых флаги совпадают с флагами, заданными для маски. Значение маски определяется так же, как и флаги при создании частиц (см. функции physics_particle_create() для примера), т.е. с помощью побитовой комбинации или "|" для маскирования соответствующих битов для каждого флага.

ПРИМЕЧАНИЕ: Эта функция нарисует все частицы в физической симуляции с флагами, совпадающими с флагами маски, будь то отдельные частицы или группы, если категория пользователя одинакова.

Нарисованная частица будет игнорировать значения цвета и альфы, установленные при ее создании, и вместо этого будет нарисована с использованием параметров масштаба, угла, цвета и альфы, заданных в этой функции. Обратите внимание, что эта функция предназначена только для рисования sprite в позиции частицы и никак не влияет на поведение частицы.

Синтаксис:

physics_particle_draw_ext(typemask, category, sprite, subimg, xscale, yscale, ang, col, alpha)

Аргумент Тип Описание
typemask Физика Частица Флаг Константа(ы) The mask to compare the particles flags with.
category Реально The user defined category to draw (use 0 to draw all categories).
sprite Актив спрайта The sprite to use for each particle.
subimg Реально The image_index спрайта для рисования.
xscale Реально The scale factor to apply to the sprite along the x axis.
yscale Реально The scale factor to apply to the sprite along the y axis.
ang Реально The angle (in degrees) to draw the sprite at.
col Цвет The colour to blend with the sprite (default is c_white).
alpha Реально The alpha value to use when drawing (0 - 1).

Возвращается:

N/A

Пример:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw_ext(flags, 2, sprBlob, 0, 2, 2, 0, c_white, 0.5);

Приведенный выше код нарисует sprite для всех частиц в заданной пользователем категории "2", которые имеют те же флаги, что и заданная маска, масштабированные на 2 по обеим осям и с альфой 0,5.
