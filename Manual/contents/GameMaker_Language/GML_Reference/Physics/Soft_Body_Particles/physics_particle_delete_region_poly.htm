physics_particle_delete_region_poly

С помощью этой функции вы можете удалить все частицы, которые попадают в границы определенной многоугольной области, из физической симуляции в текущей комнате. Функция принимает (предварительно созданный) ds_list() содержащий x/y позицию каждой точки полигона, причем четные позиции в списке являются координатами x, а нечетные - координатами y, т.е. для треугольника в списке будет шесть позиций, причем 0, 2 и 4 будут координатами x, а 1, 3 и 5 - координатами y. Затем полигон будет использоваться для удаления всех частиц, которые попадают в определенную область.

Обратите внимание, что заданный многоугольник должен иметь не менее трех точек и не более 8.

Синтаксис:

physics_particle_delete_region_poly(pointList)

АргументТип Описание
pointList A ds_list of points to use to create the polygon.

Возвращается:

Пример:

var list = ds_list_create(p_list);
for (var i = 0; i < 5; i ++;)
{
    ds_list_add(p_list, mx[i]);
    ds_list_add(p_list, my[i]);
}
physics_particle_delete_region_poly(p_list);
ds_list_destroy(p_list);

Приведенный выше код удалит все частицы, находящиеся в многоугольной области, определенной точками, добавленными на DS List.
