physics_particle_group_begin

С помощью этой функции вы можете создать группу частиц в комнате. Группа частиц - это группа, в которой частицы, необходимые для создания определенной формы, создаются все сразу как группа, позволяя вам создавать симулированные мягкие тела с различными свойствами, задаваемыми используемыми флагами. Флаги" - это возвращаемое значение комбинации констант частиц (см. функцию physics_particle_create() а "groupFlags" - это комбинация следующих групповых констант (или 0, если флагов нет):

Физика Группа частиц Флаг Константа
Постоянная Описание
phy_particle_group_flag_solid Группа твердых частиц не позволяет другим элементам закрепиться внутри нее. Если в нее проникает что-либо, группа твердых частиц выталкивает нарушителя обратно на поверхность, благодаря чему группа твердых частиц обладает особенно сильной силой отталкивания.
phy_particle_group_flag_rigid Жесткие группы частиц - это группы, форма которых не меняется, даже когда они сталкиваются с другими приспособлениями.

Эти групповые флаги используют битовую маскировку для создания конечного выходного значения, которое затем проверяется для установки различных основных свойств группы, помимо тех, которые устанавливаются флагами частиц. Вы можете использовать ни одну, одну или обе эти константы, используя побитовую комбинацию или "|" для маскировки соответствующих битов (как показано в примере ниже).

Кроме флагов, вы также можете задать положение в комнате, в котором будет создана группа частиц (заданное положение будет центром масс готовой группы), начальную горизонтальную и вертикальную скорость, а также угловую скорость группы, что позволяет создавать движущиеся, вращающиеся группы.

Вы также можете задать начальный цвет и альфа всех частиц в группе, причем эти значения будут использоваться при рисовании частиц с помощью функции physics_particle_draw(). После этого вы задаете силу сцепления частиц в группе, что является еще одним способом сказать, насколько сильно они "прилипают" друг к другу. Сила может быть значением от 0 до 1, где 0 - отсутствие когезии, а 1 - максимальная когезия.

Наконец, вы должны указать категорию для вашей группы частиц. Это значение является произвольным целым числом, которое вы присваиваете каждой группе частиц, которые вы хотите иметь схожие свойства, и оно может быть использовано для установки флагов в дальнейшем для этой конкретной категории, а также для ряда других вещей.

ПРИМЕЧАНИЕ: Вы можете использовать любое целочисленное значение для категории, кроме 0, которое зарезервировано симуляцией и используется для выбора всех категорий в других функциях (например, для рисования).

Важно отметить, что вызов этой функции не создает группу в физическом кабинете. Эта функция просто начинает процесс и устанавливает базовые свойства частиц для группы. Чтобы действительно создать группу частиц, необходимо задать ей форму (см. пример ниже), а затем вызвать функцию physics_particle_group_end(), после чего она будет создана.

ПРИМЕЧАНИЕ: Флаги типов частиц, которые вы установите, также будут влиять на сплоченность группы частиц, например, если вы отметите частицы как тип воды (например), то при столкновении с другим приспособлением группа распадется на части, в то время как если вы отметите частицы как тип пружины, то они сохранят свою сплоченность при столкновении.

Синтаксис:

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)

Аргумент Тип Описание
flags Физика Частица Флаг Константа(s) The flags to set on the particle.
groupflags Физика Группа частиц Флаг Константа(s) The group flags to set on the particle group (see table at the top).
x Реально The x position to create the particle group at.
y Реально The y position to create the particle group at.
ang Реально The angle (in degrees) to rotate the group to on create.
xv Реально The initial horizontal velocity.
yv Реально The initial vertical velocity.
ang_velocity Реально The initial angular velocity.
col Цвет The base colour to use for the particle group.
alpha Реально The base alpha to use for the particle group.
strength Реально Set the cohesion strength between particles in the group.
category Реально The user defined category to which the group belongs.

Возвращается:

N/A

Пример:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

Приведенный выше код сохраняет флаги для типа частиц и свойства группы частиц в переменных, а затем использует их для создания круглой группы частиц с радиусом 100px в позиции мыши.
