physics_particle_set_damping

С помощью этой функции можно задать линейное демпфирование частиц в симуляции. Демпфирование используется для уменьшения скорости экземпляров в физическом моделировании с течением времени, подобно сопротивлению воздуха в реальном мире. Эта функция является глобальной по области применения, она изменяет демпфирование не только для новых частиц, созданных после изменения, но и для тех, которые уже присутствуют в симуляции.

Синтаксис:

physics_particle_set_damping(damping)

АргументТип Описание
damping The linear damping to be applied to the particle fixture.

Возвращается:

Пример:

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

Приведенный выше код установит базовые свойства для всех частиц в симуляции.
