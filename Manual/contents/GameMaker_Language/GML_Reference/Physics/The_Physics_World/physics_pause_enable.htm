physics_pause_enable

Обычно, чтобы физическая симуляция работала, она должна быть непрерывной, ее нельзя останавливать и запускать или внезапно перемещать объекты из одного места в другое в комнате. Однако бывают моменты, когда необходимо приостановить симуляцию, например, если устройство os остановилось, и тогда используется эта функция. Она приостанавливает моделирование, если флаг установлен в true, и никакие дальнейшие физические расчеты не будут выполняться, пока флаг не будет снова установлен в false.

ПРИМЕЧАНИЕ: Это особенно полезно, если вы хотите деактивировать все экземпляры в комнате, поскольку даже при деактивации физическое тело будет продолжать рассчитываться и моделироваться в физическом мире.

Синтаксис:

physics_pause_enable(flag)

АргументТип Описание
flag This can be set to true чтобы приостановить моделирование, или false, чтобы запустить его снова.

Возвращается:

Пример:

if os_is_paused()
{
    physics_pause_enable(true);
    instance_deactivate_all(true);
    instance_create_layer(x, y, "Controllers", obj_pause_menu);
}

Код выше проверяет, была ли ОС приостановлена, и если да, то приостанавливает физический мир, деактивирует все, кроме себя, а затем создает экземпляр меню паузы.
