physics_world_draw_debug

Эта функция рисует представление системы физического мира комнаты, чтобы помочь в отладке, когда у вас возникают проблемы с правильной работой физической сцены. Иногда проблема может быть в части рендеринга вашей игры, например, sprite может быть нарисован в неправильном положении или вращении, создавая видимость неправильной физики, или может быть, что ваши расчеты не верны. В любом случае, эта функция поможет вам найти и решить проблему, нарисовав то, что происходит в вашей системе физики. Для этого нужно указать, какую часть системы нужно нарисовать, и это делается путем передачи значения, созданного из битов, хранящихся в различных константах "флага":

Флаг отладки физики
Флаг Описание
phy_debug_render_aabb Это показывает абсолютную границу каждого светильника по отношению к оси помещения
phy_debug_render_collision_pairs Это покажет все приспособления, которые в настоящее время находятся в столкновении
phy_debug_render_coms Здесь отмечается центр масс каждого светильника в комнате
phy_debug_render_core_shapes Показывает основные формы, из которых состоят светильники в комнате
phy_debug_render_joints Это позволит нарисовать все стыки всех светильников в комнате
phy_debug_render_obb Здесь показана относительная ограничивающая рамка для светильников в комнате
phy_debug_render_shapes Это показывает фактические формы, из которых состоят светильники в помещении

Таким образом, чтобы нарисовать любые два из этих вариантов, нам нужно сделать побитовое "или" (в GameMaker это обозначается | ) для этих двух вариантов и сохранить полученную переменную для использования функцией. Вот пример того, как можно задать переменную для использования с функцией, которая будет рисовать только центры масс и соединения всех светильников в комнате:

flag = phy_debug_render_coms | phy_debug_render_shapes;

Что даст вам что-то вроде этого:

Как вы можете видеть из приведенного выше кода, на экране рисуются только формы и центры масс (а также цвета, представляющие их состояния - за дополнительной информацией обращайтесь к документации Box 2D on-line). Таким же образом вы можете отобразить больше или даже все константы, которые вы хотите отладить... Просто "или" их все вместе! Следует отметить еще один момент - эта функция должна вызываться только в событии рисования экземпляра, который имеет глубину меньше, чем те, которые вы хотите отладить, иначе вы не сможете увидеть отладочную информацию.

Синтаксис:

physics_world_draw_debug(flag)

Аргумент Тип Описание
flag Флаг отладки физики(s) The flag value for showing different debug features

Возвращается:

N/A

Пример:

flag = phy_debug_render_aabb | phy_debug_render_collision_pairs | phy_debug_render_obb;
physics_world_draw_debug(flag);

Приведенный выше код отрисует все абсолютные ограничительные рамки, относительные ограничительные рамки, а также столкнувшиеся приспособления в рамках текущей системы физики.
