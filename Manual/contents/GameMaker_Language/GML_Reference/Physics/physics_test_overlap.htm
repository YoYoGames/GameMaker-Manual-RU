физика_тест_перекрытие

Эта функция может быть использована для проверки того, перекрывается ли физическое тело (т.е. крепление экземпляра) или будет ли оно перекрываться при повороте и размещении в заданной позиции в комнате. Аргумент "angle" - это угол поворота, который имеет (или будет иметь) вызываемый экземпляр в проверяемой позиции, а аргумент "obj" может быть либо единственным идентификатором экземпляра, либо индексом объекта, либо аргументом "obj". ключевые слова все или другое.

Синтаксис:

physics_test_overlap(xpos, ypos, angle, obj);

Аргумент Тип Описание
xpos Реально The x position in the room to check
ypos Реально The y position in the room to check
angle Реально The angle to check (of the calling instance)
obj Идентификаторактива или экземпляраобъекта The object to check for

Возвращается:

Boolean

Пример:

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
{
    alarm[0] = room_speed;
    ignited = true;
}

Приведенный выше код проверяет позицию на наличие перекрытия физического приспособления, и если оно есть, то устанавливает переменную и сигнал тревоги.
