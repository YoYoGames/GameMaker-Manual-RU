Физика Совместные константы

В GameMaker GML включено большое количество констант, специфичных для соединений между объектами в физическом мире. Их можно использовать в сочетании с различными функциями суставов для установки или получения информации от них в реальном времени во время работы физического моделирования. Однако следует помнить, что при их вызове выполняются сложные вычисления, поэтому их следует использовать с осторожностью и только при необходимости, а также учитывать, что многие из них уникальны для определенного типа соединений. Также обратите внимание, что хотя вы можете получить все эти значения с помощью соответствующей функции, вы можете установить только те, которые помечены как не доступные только для чтения.

В общем случае эти константы используются вместе со следующими функциями:

  • physics_joint_get_value
  • physics_joint_set_value

Следующие константы могут быть применены к любому из имеющихся типов соединений:

Физика Совместная постоянная
Постоянная Описание Только чтение
phy_joint_anchor_1_x Координата x первой опорной точки сустава в помещении Yes
phy_joint_anchor_1_y Координата y первой опорной точки сустава в помещении Yes
phy_joint_anchor_2_x Координата x второй опорной точки сустава в помещении Yes
phy_joint_anchor_2_y Координата y второй опорной точки сустава в помещении Yes
phy_joint_reaction_force_x Это сила реакции, приложенная ко второму экземпляру в суставе в положении привязки x Yes
phy_joint_reaction_force_y Это сила реакции, приложенная ко второму экземпляру в суставе в положении привязки y Yes
phy_joint_reaction_torque Это крутящий момент, прикладываемый ко второму экземпляру в суставе в положении якоря Yes


Эти константы предназначены для тех шарниров, к которым присоединен двигатель (револьверный, призматический, колесный):

Физика Совместная постоянная
Постоянная Описание Только чтение
phy_joint_max_motor_force Значение, указанное при создании сустава для максимальной силы двигателя No
phy_joint_max_motor_torque Значение, указанное при создании соединения для максимального крутящего момента двигателя No
phy_joint_motor_force Текущая сила двигателя Yes
phy_joint_motor_speed Текущая скорость вращения двигателя No
phy_joint_motor_torque Текущий крутящий момент двигателя Yes


Для револьверного шарнира вы можете использовать следующую константу (а также константы двигателя, если он был добавлен):

Физика Совместная постоянная
Постоянная Описание Только чтение
phy_joint_angle Угол, который образует линия между двумя опорными точками сустава. Вычисляется с помощью координат физического мира
(не координаты комнаты GameMaker ) в радианах.		 Да
phy_joint_angle_limits Включите или отключите ограничение угла для сустава. Установите значение true для включения или false для отключения. Нет
phy_joint_upper_angle_limit Верхний предел угла для сустава в градусах. Нет
phy_joint_lower_angle_limit Нижний предел угла для сустава в градусах. Нет


Для призматического соединения можно использовать следующую константу:

Физика Совместная постоянная
Постоянная Описание Только чтение
phy_joint_translation Получает расстояние между координатами x/y якоря и локальными координатами x/y. Yes
phy_joint_speed Текущая скорость движения сустава. Yes


Для расстояния, сварного шва и колесных соединений можно использовать следующие константы (как и для шкивных соединений):

Физика Совместная постоянная
Постоянная Описание Только чтение
phy_joint_damping_ratio Коэффициент демпфирования является безразмерным и определяет "пружинистость" соединения. Значение этой константы обычно находится в диапазоне от 0 до 1, но может быть и больше, а при значении 1 демпфирование является критическим, то есть все колебания должны исчезнуть. No
phy_joint_frequency Это вернет (или установит) частоту колебаний для сустава в герцах, и обычно частота должна быть меньше половины частоты временного шага, как установлено функцией physics_world_update_speed(). No
phy_joint_length_1 Это вернет длину сустава от первой локальной координаты x/y до первой якорной координаты x/y (только для суставов с расстоянием, можно только считать из) Yes
phy_joint_length_2 Это вернет длину сустава от второй локальной координаты x/y до второй якорной координаты x/y (только для суставов с расстоянием, может быть записано только в) No


Для фрикционного соединения можно использовать следующие константы:

Физика Совместная постоянная
Постоянная Описание Только чтение
phy_joint_max_torque Максимальное значение крутящего момента для соединения. No
phy_joint_max_force Максимальное значение силы для сустава. No


Для веревочного соединения можно использовать следующую константу:

Физика Совместная постоянная
Постоянная Описание Только чтение
phy_joint_max_length Максимальное удлинение для соединения между двумя точками крепления. No
