physics_joint_delete

После того как два экземпляра с физическими представлениями были связаны суставом, его можно удалить в любое время. Обычно это происходит автоматически, когда один из двух экземпляров уничтожается, или когда комната заканчивается, но бывают случаи, когда вы хотите сделать это вручную. В этих случаях можно воспользоваться данной функцией.

Следует также отметить, что при уничтожении экземпляра, являющегося частью зубчатого соединения, зубчатое соединение должно быть удалено с помощью этой функции раньше всех экземпляров, участвующих в формировании зубчатого соединения (но остальные соединения будут удалены автоматически), например, в событии destroy экземпляра.

Синтаксис:

physics_joint_delete(joint)

Аргумент Тип Описание
joint Физика Совместный идентификатор The index of the joint that you wish to delete

Возвращается:

N/A

Пример:

var reactionForceX, reactionForceY, reactionForce;
if (shipJoint)
{
    _reaction_force_x = physics_joint_get_value(ship_joint, phy_joint_reaction_force_x);
    _reaction_force_y = physics_joint_get_value(ship_joint, phy_joint_reaction_force_y);
    _reaction_force = sqrt((_reaction_force_x + _reaction_force_x) + (_reaction_force_y + _reaction_force_y));
    if (_reaction_force > 2)
    {
        physics_joint_delete(ship_joint);
        ship_joint = -1;
    }
}

Приведенный выше код проверяет, содержит ли переменная "shipJoint" индекс сустава, и если да, то рассчитывает силу, приложенную к этому суставу, используя две константы. Наконец, если общая сила больше 2, сустав удаляется.
