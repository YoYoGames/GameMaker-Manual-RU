физический_шарнир_получить_значение

Используя ряд предопределенных констант, вы можете попросить GameMaker сообщить вам ряд сведений о состоянии любого сустава. Это очень полезно, поскольку дает вам возможность удалять суставы или изменять поведение экземпляров в зависимости от ваших потребностей на данный момент. Существует ряд констант, которые можно использовать в этой функции, и их можно найти здесь: Physics Joint Constants, но имейте в виду, что при их вызове производятся сложные вычисления, поэтому их следует использовать с осторожностью и только при необходимости, и учтите, что многие из них уникальны для определенного типа суставов.

Если свойство не существует (например, если вы проверяете шарнир шкива на призматический перевод), то возвращаемым значением будет 0.

Синтаксис:

physics_joint_get_value(joint, value)

Аргумент Тип Описание
joint Физика Совместный идентификатор The index of the joint that you wish to test
value Физика Совместная постоянная The constant for the joint property that you wish to test

Возвращается:

Real

Пример:

if shipJoint
{
    var reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    var reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y);
    var reactionForce = point_distance(0, 0, reactionForceX, reactionForceY);
    if (reactionForce > 2)
    {
        physics_joint_delete(shipJoint);
        shipJoint = -1;
    }
}

Приведенный выше код проверяет, содержит ли переменная "shipJoint" индекс сустава, и если да, то рассчитывает силу, приложенную к этому суставу, используя две константы. Наконец, если общая сила больше 2, сустав удаляется.
