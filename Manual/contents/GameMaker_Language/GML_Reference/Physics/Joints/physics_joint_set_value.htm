физическое_соединение_установить_значение

Некоторые свойства сустава могут быть изменены и заданы даже после его создания. Существует ряд констант, которые могут быть использованы в этой функции, и их можно найти здесь: Physics Joint Constants.

Синтаксис:

physics_joint_set_value(joint, field, value)

Аргумент Тип Описание
joint Физика Совместный идентификатор The index of the joint that you wish to change
field Физика Совместная постоянная The constant for the joint property that you wish to change
value Реально The new value for the joint property

Возвращается:

N/A

Пример:

if physics_joint_get_value(revJoint, phy_joint_max_motor_torque) < 2
{
    physics_joint_set_value(revJoint, phy_joint_max_motor_torque, 2);
}

Приведенный выше код проверяет, установлен ли максимальный крутящий момент двигателя суставов меньше 2, и если да, то устанавливает его равным 2.
