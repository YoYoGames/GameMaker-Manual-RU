физика_применить_угловой_импульс

Эта функция дает угловой импульс экземпляру с включенной физикой. Этот импульс установит угловое вращение на заданную величину, игнорируя текущий крутящий момент, по сути, устанавливая количество "вращения", которое имеет приспособление. Если вы хотите применить угловую силу к экземпляру с помощью крутящего момента, то вам следует использовать функцию physics_apply_torque().

Синтаксис:

physics_apply_angular_impulse(impulse)

АргументТип Описание
impulse The impulse (in Newton metres) to be applied

Возвращается:

Пример:

if keyboard_check(vk_up)
{
    physics_apply_angular_impulse(10);
}

Приведенный выше код будет давать приспособлению с включенной физикой угловой импульс при нажатии клавиши.
