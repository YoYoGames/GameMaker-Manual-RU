physics_set_restitution

Когда вы привязываете приспособление к экземпляру с помощью physics_fixture_bind() эта функция возвращает "id" для связанного приспособления. Вы можете использовать этот идентификатор для установки значения реституции связанного приспособления, а не "базового" приспособления, в любое время с помощью этой функции. Реституция обычно устанавливается как значение между 0 и 1, но при необходимости можно использовать и большие значения, хотя результаты могут быть непредсказуемыми.

Синтаксис:

physics_set_restitution(fixture, restitution)

АргументТип Описание
fixture the id of the bound fixture
restitution the new restitution value to apply

Возвращается:

Пример:

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

Приведенный выше код получает текущее значение реституции для связанных физических свойств экземпляра, а затем устанавливает их в другое значение.
