physics_fixture_set_box_shape

Эта функция определяет форму коробки для вашего приспособления. Она принимает половину ширины и высоты, так как в мире физики это значение используется гораздо чаще, чем целые значения ширины/высоты для определения таких вещей, как столкновения.

Синтаксис:

physics_fixture_set_box_shape(fixture, halfWidth, halfHeight)

АргументТип Описание
fixture the index of the fixture
halfWidth the half width of the box
halfHeight the half height of the box

Возвращается:

Пример:

physics_fixture_set_box_shape(fix_Border, room_width/2, 10);

Приведенный выше код применит форму коробки к приспособлению, индексированному в переменной "fix_Border", с шириной комнаты и высотой 20 пикселей.
