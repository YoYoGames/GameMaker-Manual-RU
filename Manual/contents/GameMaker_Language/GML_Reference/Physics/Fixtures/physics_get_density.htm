physics_get_density

Когда вы привязываете приспособление к экземпляру с помощью physics_fixture_bind() это возвращает "id" для связанного приспособления. Вы можете использовать этот идентификатор для получения значения плотности связанного приспособления(не "базового" приспособления) в любое время с помощью этой функции.

Синтаксис:

physics_get_density(fixture)

АргументТип Описание
fixture the id of the bound fixture

Возвращается:

Пример:

var density = physics_get_density(fix_id);
physics_set_density(fix_id, density - 0.1);

Приведенный выше код получает значение плотности тока для связанных физических свойств экземпляра, а затем устанавливает их в другое значение.
