физика_удаление_приспособления

Эта функция удаляет (или "отвязывает") приспособление от экземпляра или экземпляров. Для этого требуется уникальный "id" привязанного приспособления (возвращаемый функцией physics_fixture_bind() и она удалит все определенные в данный момент физические свойства экземпляра, позволяя вам заново определить новое приспособление и привязать его к экземпляру. Таким образом, вы можете изменить физические свойства экземпляра без необходимости его уничтожения и повторного создания.

Синтаксис:

physics_remove_fixture(id, fixture)

Аргумент Тип Описание
id Идентификатор экземпляра The ID of the instance to remove the fixture from
fixture Идентификатор физического приспособления The ID of the fixture that is to be removed from the instance

Возвращается:

N/A

Пример:

physics_remove_fixture(id, my_fix);

Приведенный выше код удалит из экземпляра приспособление с "id", хранящимся в переменной "my_fix".
