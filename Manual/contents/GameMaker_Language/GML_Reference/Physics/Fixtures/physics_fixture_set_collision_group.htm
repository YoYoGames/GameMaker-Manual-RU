physics_fixture_set_collision_group

После создания приспособления можно определить его как часть группы столкновений. Это означает, что вы можете указать экземплярам с этим приспособлением всегда сталкиваться или никогда не сталкиваться с другими экземплярами, которые имеют привязанные к ним приспособления с той же группой столкновений. Значения, которые можно присвоить, основаны на положительных или отрицательных числах, при этом по умолчанию используется значение 0 (это означает, что экземпляр с этим параметром приспособления игнорируется, пока для него не будет создано событие столкновения). Вот изображение для пояснения:

Как видите, настройка групп столкновений может быть очень полезна для создания сложных физических тел, которые взаимодействуют с окружающим физическим миром, но не друг с другом. Однако следует отметить, что из-за сложности расчета столкновений на основе подобных групп рекомендуется стараться свести количество групп, используемых в одной комнате, к абсолютному минимуму.

Значения для группы столкновений могут находиться в диапазоне от -32,768 до 32,767 (оба включительно).

Синтаксис:

physics_fixture_set_collision_group(fixture, group);

Аргумент Тип Описание
fixture Идентификатор физического приспособления The fixture that is to be bound
group Реально The group that this fixture is assigned to (either a positive value, 0, or a negative value)

Возвращается:

N/A

Пример:

physics_fixture_set_collision_group(fix_Wheel, -1);

Приведенный выше код установит приспособление, проиндексированное в переменной "fix_Wheel", в группу "-1".
