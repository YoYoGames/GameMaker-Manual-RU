physics_fixture_set_linear_damping

Демпфирование используется для уменьшения скорости экземпляров в физическом мире и отличается от трения тем, что трение возникает только при контакте двух экземпляров с креплениями. Демпфирование также намного дешевле в моделировании, чем трение, но обратите внимание, что демпфирование не является заменой трения; эти два эффекта можно и нужно использовать вместе. Вот изображение, иллюстрирующее демпфирование:

Параметры демпфирования должны находиться в диапазоне от 0 до бесконечности, при этом 0 означает отсутствие демпфирования, а бесконечность означает полное демпфирование. Обычно используется значение демпфирования между 0 и 1, но при необходимости можно использовать любое неотрицательное значение.

Синтаксис:

physics_fixture_set_linear_damping(fixture, damping)

АргументТип Описание
fixture the index of the fixture
damping the damping of the fixture, usually between 0 and 1

Возвращается:

Пример:

physics_fixture_set_linear_damping(fix_Ball, 0.1);

Приведенный выше код установит линейное демпфирование приспособления, индексируемого в "fix_ball", равным 0,1.
