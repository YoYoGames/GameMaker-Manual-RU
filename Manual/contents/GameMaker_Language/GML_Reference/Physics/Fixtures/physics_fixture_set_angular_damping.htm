physics_fixture_set_angular_damping

Если вспомнить любой вращающийся объект в "реальном мире", если только у него нет двигателя или он не находится в космосе, он со временем замедляется под воздействием внешних сил (например, трения с окружающим воздухом). Мы можем использовать функцию physics_fixture_set_angular_damping() в GameMaker для имитации этого эффекта и уменьшения скорости вращения объектов в физическом мире, поскольку без нее любой вращающийся объект продолжал бы вращаться бесконечно. Параметры демпфирования должны находиться в диапазоне от 0 до бесконечности, при этом 0 означает отсутствие демпфирования, а бесконечность - полное демпфирование. Обычно используется значение демпфирования между 0 и 1, но при необходимости можно использовать любое неотрицательное значение.

Синтаксис:

physics_fixture_set_angular_damping(fixture, damping)

АргументТип Описание
fixture the index of the fixture
damping the angular damping of the fixture, usually between 0 and 1

Возвращается:

Пример:

physics_fixture_set_angular_damping(fix_Ball, 0.1);

Приведенный выше код установит угловое демпфирование приспособления, индексируемого в "fix_ball", равным 0,1.
