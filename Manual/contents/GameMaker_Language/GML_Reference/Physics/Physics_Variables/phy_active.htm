phy_active

Эта переменная контролирует, является ли экземпляр в данный момент "активным". Если установить значение false, то экземпляр не будет участвовать в физическом мире, а если установить значение true, то он снова будет участвовать. Обратите внимание, что это не то же самое, что деактивация экземпляра, так как экземпляр все еще виден на экране и может быть изменен с помощью кода, скорее эта функция просто предотвращает его участие в физическом моделировании.

Синтаксис:

phy_active

Возвращается:

(or undefined if the instance is not physics enabled)

Пример:

if keyboard_check_pressed(ord"P")
{
    global.Pause = !global.Pause;
    with (obj_Parent)
    {
        phy_active = !global.Pause;
    }
}

Приведенный выше код обнаружит нажатие клавиши с буквой "P" и переключит глобальную переменную "Pause" с true на false и обратно. Эта переменная затем используется для установки того, активна или нет физика в дочерних экземплярах объекта, проиндексированных в переменной "obj_Parent".
