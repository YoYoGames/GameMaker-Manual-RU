phy_bullet

Эта переменная определяет, является ли экземпляр чрезвычайно быстро движущимся (например, пуля). Значение по умолчанию - false, но если установить значение true, это сообщит GameMaker, что экземпляр будет двигаться с такой высокой скоростью, что ему потребуется более дорогостоящее обнаружение столкновений, чтобы убедиться, что он не пройдет через другие экземпляры незамеченным.

Синтаксис:

phy_bullet

Возвращается:

(or undefined if the instance is not physics enabled)

Пример:

with (instance_create_layer(x, y, "Bullets", obj_Shoot))
{
    phy_bullet = true;
    physics_apply_local_impulse(0, 10, 0, 200);
}

Приведенный выше код создает новый экземпляр и затем определяет его как "пулю" в мире физики, прежде чем придать ему мощный импульс вдоль оси y.
