фи_ротация

Эта переменная может быть использована для получения (или установки) угла установки экземпляров в градусах, аналогично установке или получению параметра image_angle. Однако обратите внимание, что в физическом мире вращение рассчитывается противоположным образом, чем в обычном игровом мире GameMaker, а это значит, что векторные функции типа point_direction()должны иметь измененные возвращаемые значения (простое преобразование положительного значения в отрицательное должно решить эту проблему).

Синтаксис:

phy_rotation;

Возвращается:

(single precision floating point value, or undefined if the instance is not physics enabled)

Пример:

if (phy_speed_x > 0) || (phy_speed_y > 0)
{
    phy_rotation += sqrt(sqr(phy_speed_x) + sqr(phy_speed_y)) /10;
}

Приведенный выше код проверяет линейную скорость, и если любой из векторов не равен 0, он вычисляет фактическую скорость и использует ее для задания вращения.
