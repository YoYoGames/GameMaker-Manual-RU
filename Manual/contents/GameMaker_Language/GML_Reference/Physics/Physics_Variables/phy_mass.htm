фи_масса

Эта переменная, доступная только для чтения, возвращает массу экземпляра в килограммах. Это значение рассчитывается автоматически на основе площади поверхности назначенных приспособлений и значений их плотности, но его можно изменить с помощью функции physics_mass_properties().

Синтаксис:

phy_mass;

Возвращается:

(single precision floating point value, or undefined if the instance is not physics enabled)

Пример:

if (phy_mass < other.phy_mass)
{
    instance_destroy();
}

Приведенный выше код относится к событию столкновения экземпляра с другим, он сравнивает массу каждого экземпляра и уничтожает тот, у которого масса меньше.
