фи_динамический

Динамический экземпляр - это экземпляр, который полностью смоделирован в физическом мире, и эта переменная, доступная только для чтения, возвращает true, если проверяемый экземпляр полностью смоделирован, или false, если это не так.

Синтаксис:

phy_dynamic;

Возвращается:

(or undefined if the instance is not physics enabled)

Пример:

if other.phy_dynamic
{
    with (other)
    {
        var dir;
        dir = point_direction(x, y, other.x, other.y);
        physics_apply_impulse(x, y, x + lengthdir_x(100, dir), y + lengthdir_y(100, dir));
    }
}

Приведенный выше код создает новый экземпляр и затем определяет его как "пулю" в мире физики, прежде чем придать ему мощный импульс вдоль оси y.
