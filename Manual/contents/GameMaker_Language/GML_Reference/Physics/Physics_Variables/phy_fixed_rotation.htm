phy_fixed_rotation

Эта переменная может быть использована для установки того, могут ли на экземпляр воздействовать вращательные силы (по умолчанию false). Если переменная имеет значение true, никакие внешние силы (ни от кодированных импульсов или сил, ни от столкновений) не будут влиять на значение вращения. на величину вращения экземпляра, и ее придется задавать вручную, используя phy_rotation переменная

Синтаксис:

phy_fixed_rotation

Возвращается:

(or undefined if the instance is not physics enabled)

Пример:

phy_rotation = 0;
phy_fixed_rotation = true;

Приведенный выше код переключит экземпляр на фиксированный поворот, а затем установит угол поворота.
