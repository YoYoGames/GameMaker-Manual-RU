phy_com_y

Эта переменная, доступная только для чтения, возвращает положение y центра масс экземпляра. Она рассчитывается автоматически на основе плотности, инерции и массы экземпляра, как определено соответствующими функциями

Синтаксис:

phy_com_y;

Возвращается:

(single precision floating point value, or undefined if the instance is not physics enabled)

Пример:

xx = phy_com_x;
yy = phy_com_y;

Приведенный выше код устанавливает две переменные в положение x и y центра масс экземпляра.
