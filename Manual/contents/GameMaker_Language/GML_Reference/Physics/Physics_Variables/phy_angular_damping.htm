phy_angular_damping

Эта переменная может быть использована для установки углового демпфирования экземпляра или для получения текущего углового демпфирования. Демпфирование - это величина "сопротивления" угловому вращению, которое оказывает экземпляр с включенной физикой. Меньшее значение позволяет экземпляру вращаться быстрее после столкновения (например), а большее значение требует более сильного толчка и вращается медленнее.

Синтаксис:

phy_angular_damping;

Возвращается:

(single precision floating point value, or undefined if the instance is not physics enabled)

Пример:

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
    phy_angular_damping = 10;
}
else
{
    phy_angular_damping = 3;
}

Приведенный выше код проверит столкновение между вызывающим экземпляром и экземплярами "obj_Water" и соответствующим образом изменит угловое затухание.
