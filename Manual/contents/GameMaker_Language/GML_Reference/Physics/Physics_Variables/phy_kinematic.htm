phy_kinematic

Эта переменная, доступная только для чтения, возвращает значение true, если экземпляр классифицируется как кинематический объект, или false, если это не так. Кинематический объект - это объект, который имеет бесконечную массу (плотность равна 0), но может двигаться. Поэтому, чтобы сделать экземпляр кинематическим, нужно сначала создать статический экземпляр, а затем установить одну или несколько переменных экземпляра, связанных с движением (например: phy_speed_x, phy_speed_y, или phy_angular_velocity)

Синтаксис:

phy_kinematic;

Возвращается:

(or undefined if the instance is not physics enabled)

Пример:

if !phy_kinematic
{
    phy_speed_x = 5;
}

Приведенный выше код проверяет, является ли экземпляр кинематическим, и если нет, то устанавливает горизонтальную скорость равной 5.
