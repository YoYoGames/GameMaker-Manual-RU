фи_спящий

Эта переменная, доступная только для чтения, показывает, "спит" ли данный экземпляр в данный момент (true) или нет (false). "Спящий" экземпляр - это тот, который не принимает активного участия в физической симуляции. GameMaker переводит объекты в спящий режим для экономии циклов моделирования, когда экземпляр находится в состоянии покоя и не сталкивается с другим экземпляром.

Синтаксис:

phy_sleeping;

Возвращается:

(or undefined if the instance is not physics enabled)

Пример:

if phy_sleeping
{
    instance_destroy();
}

Приведенный выше код проверяет, является ли объект активно моделируемым или нет, и если нет, то он уничтожается.
