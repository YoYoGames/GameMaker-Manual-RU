точки_столкновения_фи

Эта переменная только для чтения возвращает количество точек столкновения, обнаруженных между двумя объектами в столкновении

ПРИМЕЧАНИЕ: Эта переменная доступна только в событии столкновения экземпляра с включенной физикой.

Синтаксис:

phy_collision_points;

Возвращается:

(integer value, or undefined if the instance is not physics enabled)

Пример:

for(var i = 0; i < phy_collision_points; i += 1;)
{
    part_particles_create(global.Sname, phy_collision_x[i], phy_collision_x[1], global.Spark, 5);
}

Приведенный выше код создает частицы во всех определенных точках столкновения между двумя экземплярами с поддержкой физики.
