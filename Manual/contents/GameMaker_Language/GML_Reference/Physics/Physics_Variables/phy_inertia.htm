фи_инерция

Инерция - это мера того, насколько трудно заставить что-то начать или прекратить движение, поэтому чем меньше значение этой переменной, доступной только для чтения, тем легче привести экземпляр в движение, в то время как более высокие значения потребуют большей силы для начала движения. движение

Синтаксис:

phy_inertia;

Возвращается:

(single precision floating point value, or undefined if the instance is not physics enabled)

Пример:

physics_mass_properties(70, -10, -10, phy_inertia);

Приведенный выше код изменит массу и центр масс экземпляра при сохранении значения инерции.
