player_avatar_url

В этой переменной хранится URL-адрес аватара игрока, который можно загрузить с помощью sprite_add().

Вы можете использовать player_avatar_sprite, в который уже загружен sprite.

Эта переменная существует только в экземплярах игроков, созданных с помощью rollback_define_player()и для каждого игрока она своя.

Гостевые игроки будут отображать заполнитель sprite.

Синтаксис:

player_avatar_url;

Возвращается:

String
