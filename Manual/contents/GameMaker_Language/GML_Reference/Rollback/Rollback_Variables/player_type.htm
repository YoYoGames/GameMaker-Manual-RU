тип игрока

Эта переменная определяет, является ли экземпляр игрока зарегистрированным пользователем GX.games или гостем. Она может возвращать любую из следующих строк:

Строка Описание
"Guest" Игрок не зарегистрирован в GX.games
"User" Игрок зарегистрирован в GX.games

Гостевые пользователи получают аватары и имена.

Эта переменная существует только в экземплярах игроков, созданных с помощью rollback_define_player()и для каждого игрока она своя.

Синтаксис:

player_type;

Возвращается:

String

Назад: Переменные отката
Далее: player_user_id
