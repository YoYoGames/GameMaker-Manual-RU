управляемый

Эта переменная, доступная только для чтения, возвращает, является ли текущий экземпляр управляемым(true) или нет(false).

Для получения дополнительной информации об управляемых объектах см: Управляемые объекты

Синтаксис:

managed;

Возвращается:

Boolean
