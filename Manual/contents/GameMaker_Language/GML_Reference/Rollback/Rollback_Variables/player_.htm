имя_игрока

В этой переменной хранится имя текущего экземпляра игрока.

Эта переменная существует только в экземплярах игроков, созданных с помощью rollback_define_player()и для каждого игрока она своя.

Гостевым игрокам будет возвращено имя-заполнитель.

Синтаксис:

player_name;

Возвращается:

String

Пример:

// Draw event
draw_set_halign(fa_center);
draw_set_valign(fa_middle);

draw_text(x, y - 100, player_name);

draw_set_valign(fa_left);
draw_set_valign(fa_top);

Это рисует имя игрока на 100px выше его экземпляра, с выравниванием по центру и середине.
