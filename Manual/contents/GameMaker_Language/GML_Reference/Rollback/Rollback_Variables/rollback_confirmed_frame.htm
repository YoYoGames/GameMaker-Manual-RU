откат_подтвержденный_кадр

Эта глобальная переменная хранит номер последнего кадра, в котором входы для всех игроков были синхронизированы и подтверждены.

Вы можете сравнить это с rollback_current_frame, чтобы проверить, есть ли в текущем фрейме подтвержденные входы для всех игроков.

Вы также можете использовать функцию rollback_sync_on_frame(), чтобы проверить, синхронизирован ли текущий кадр.

Синтаксис:

rollback_confirmed_frame

Возвращается:

Real

Пример:

var _is_frame_confirmed = (rollback_current_frame == rollback_confirmed_frame);

if (_is_frame_confirmed)
{
    evaluate_game_over();
}

Этот код проверяет, есть ли в текущем кадре подтвержденные входы от всех игроков, проверяя, равно ли значение текущего кадра значению подтвержденного кадра.

Если это так, то запускается функция для проверки выполнения условия окончания игры, чтобы завершить игру.

Это делается для того, чтобы избежать завершения игры на основе предсказания состояния, и ожидает полностью синхронизированного кадра, прежде чем попытаться проверить условия завершения игры.
