откат_текущего_кадра

Это глобальная переменная, которая хранит количество кадров, прошедших с момента запуска многопользовательской игры.

Его следует использовать вместо current_time и других подобных переменных, как объясняется в разделе " Переменные времени".

Синтаксис:

rollback_current_frame

Возвращается:

Real

Пример:

image_angle = dsin(rollback_current_frame) * 90;

В приведенном выше коде используется rollback_current_frame с dsin() для вычисления колеблющегося значения между 1 и -1, которое умножается на 90 и применяется к углу экземпляра.
