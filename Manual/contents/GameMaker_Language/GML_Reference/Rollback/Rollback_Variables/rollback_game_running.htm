откат_игры_запуска

Эта глобальная переменная указывает, запущена ли игра отката в настоящее время. Если она равна true, это означает, что игра с откатом началась и в настоящее время запущена. В любое другое время она будет иметь значение false.

Синтаксис:

rollback_game_running

Возвращается:

Boolean

Пример:

if (!rollback_game_running) return;

x += move_x;
y += move_y;

Приведенный выше код будет выполняться в событии Step объекта игрока. Когда игра отката не запущена, он выйдет из события, поэтому игроком нельзя будет управлять, пока игра отката не начнется.
