player_user_id

Эта переменная хранит GX.games ID текущего экземпляра игрока для использования при прямых вызовах API на сервер.

Эта переменная существует только в экземплярах игроков, созданных с помощью rollback_define_player()и для каждого игрока она своя.

Уигроков-гостей этой переменной будет присвоена пустая строка.

Синтаксис:

player_user_id;

Возвращается:

String
