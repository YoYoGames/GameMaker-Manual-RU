Переменные отката

В системе Rollback предусмотрены следующие переменные:

  • Переменные экземпляра
    • идентификатор игрока
    • player_local
    • имя_игрока
    • player_avatar_sprite
    • player_avatar_url
    • тип игрока
    • player_user_id
    • параметры игрока
    • управляемый
  • Глобальные переменные
    • откат_игры_запуска
    • откат_текущего_кадра
    • откат_подтвержденный_кадр
    • rollback_api_server
