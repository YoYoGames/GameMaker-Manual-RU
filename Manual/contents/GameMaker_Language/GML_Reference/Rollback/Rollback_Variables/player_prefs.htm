параметры игрока

Эта переменная хранит предпочтения, установленные для текущего игрока. Она присваивается экземплярам игрока, созданным с помощью функции rollback_define_player().

Пожалуйста, прочитайте: Предпочтения игрока

Синтаксис:

player_prefs;

Возвращается:

Any

Пример:

image_index = player_prefs.character;

Приведенный выше код считывает предпочтение и устанавливает переменную экземпляра в проигрывателе.
