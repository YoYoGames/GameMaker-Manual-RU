player_avatar_sprite

В этой переменной хранится sprite текущего экземпляра игрока.

Эта переменная существует только в экземплярах игроков, созданных с помощью rollback_define_player()и для каждого игрока она своя.

Гостевые игроки будут отображать заполнитель sprite.

Синтаксис:

player_avatar_sprite;

Возвращается:

Sprite Asset

Пример:

// Draw event
draw_sprite_stretched(player_avatar_sprite, 0, x, y - 256, 64, 64);

Это отображает аватар экземпляра игрока, выполняющего код, в области 64x64, 256px над экземпляром.
