Rollback Multiplayer

Откат многопользовательской игры в GameMaker позволяет быстро настроить и создать сетевую многопользовательскую игру для цели Opera GX.

Эта функциональность использует серверы GX для подключения игроков и синхронизации ввода, поэтому разработчику не требуется настройка сервера для начала новой сетевой игры.

В то время как функциональность онлайн может использоваться только в мишени Opera GX, вы можете использовать "Sync Test" для локального тестирования многопользовательской игры, при этом случайный или имитирующий ввод будет применяться к другим игрокам. Sync Test функционирует не только в Opera GX, но и в мишенях Windows и macOS.
