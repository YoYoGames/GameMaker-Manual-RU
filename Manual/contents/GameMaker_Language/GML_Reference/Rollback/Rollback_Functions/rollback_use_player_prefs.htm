rollback_use_player_prefs

Эта функция включает предпочтения игроков и позволяет установить предпочтения по умолчанию для всех игроков.

Настройки по умолчанию могут быть любого типа. Если вы не укажете предпочтения по умолчанию, то все игроки должны иметь установленные предпочтения перед началом игры.

Вызов этой функции отключает автозапуск, поэтому вам придется запускать игру вручную. См: Ручной и автоматический запуск

Если используется, эта функция должна выполняться до функции join/create.

Пожалуйста, прочитайте: Предпочтения игрока

Синтаксис:

rollback_use_player_prefs([default]);

Аргумент Тип Описание
default Любой OPTIONAL Предпочтения по умолчанию, устанавливаемые для всех игроков.

Возвращается:

N/A

Пример:

rollback_use_player_prefs({
символ: 0,
цвет: c_белый
});

Приведенный выше код включает предпочтения игрока и устанавливает предпочтения по умолчанию в struct.
