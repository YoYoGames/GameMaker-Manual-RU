откат_вступления_в_игру

Эта функция пытается присоединиться к игре. Вы должны находиться на URL, содержащем ID виртуальной комнаты, к которой вы хотите присоединиться.

Если комната была присоединена, возвращается true, а если ни одна игра не была присоединена, возвращается false.

Когда новый игрок присоединяется к игре, в которой вы уже присутствуете, срабатывает событие Rollback Event. Дополнительную информацию см. в разделе События отката.

Аргумент dry_run является необязательным и при значении true позволяет запустить функцию без фактического присоединения к игре. Это используется для проверки возможности присоединения к игре без фактического присоединения к ней.

Синтаксис:

rollback_join_game([dry_run]);

Аргумент Тип Описание
dry_run Булево Disabled by default (false). Если функция включена(true), она не присоединяется к игре, а просто возвращает true, если к ней можно присоединиться, или false в противном случае.

Таким образом, вы можете использовать эту функцию только для проверки того, был ли игрок приглашен.

Возвращается:

Boolean

Пример:

rollback_define_player(obj_player);
var _joined = rollback_join_game();

if (!_joined)
{
    rollback_create_game(2, false);
}

Приведенный выше код определяет объект игрока, а затем пытается присоединиться к игре. Если он не присоединился, то вместо него создается новая игра, в которой максимум 2 игрока и отключен Sync Test.

Расширенный пример смотрите в разделе Создание многопользовательской игры.
