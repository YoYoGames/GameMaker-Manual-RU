откат_чата

Эта функция используется для отправки строкового сообщения другим игрокам.

Вы указываете строку для отправки, максимальный размер которой составляет 128 байт.

Если второй аргумент не указан, сообщение отправляется всем игрокам, включая отправителя.

Во втором аргументе можно указать ID игрока, который получит сообщение.

Как использовать?

Эта функция предназначена для использования "локально", и поэтому будет являться результатом локального ввода, а не ввода Rollback.

См. пример использования внизу. Прослушивайте полученные сообщения в событии Rollback Event.

Синтаксис:

rollback_chat(message, [to]);

Аргумент Тип Описание
message Строка The message to send, as a string.
to Реально OPTIONAL ID игрока, которому будет отправлено сообщение. Если не указывать этот аргумент или указать -1, сообщение будет отправлено всем игрокам.

Возвращается:

N/A

Пример:

if (keyboard_check_pressed(vk_enter))
{
    rollback_chat(keyboard_string);
    keyboard_string = "";
}

Если клавиша ввода нажата локально, то keyboard_string значение отправляется всем игрокам. Затем эта строка очищается, чтобы можно было набрать новое сообщение.

Это вызовет событие Rollback Event, когда сообщение будет получено всеми клиентами (включая отправителя).

Событие отката

if (rollback_event_id == rollback_chat_message)
{
    // Get received information
    var _message = rollback_event_param.message;
    var _from = rollback_event_param.from;

    // Find sender's name
    var _name = rollback_get_info(_from).player_name;

    // Print to output log
    show_debug_message(_name + " says: " + _message);
}
