rollback_use_random_input

Когда вы играете в игру, используя Sync Test, все "удаленные" игроки получают случайные значения для своих заданных входов в качестве основного способа тестирования. Вы можете отключить это поведение, вызвав эту функцию с false в качестве аргумента, и включить ее снова, указав true.

Синтаксис:

rollback_use_random_input(enabled);

Аргумент Тип Описание
enabled Булево Specifies whether random input should be enabled (true) или отключен(false). По умолчанию он включен.

Возвращается:

N/A

Пример:

rollback_use_random_input(false);

Приведенный выше код отключает случайный ввод во время Sync Test.
