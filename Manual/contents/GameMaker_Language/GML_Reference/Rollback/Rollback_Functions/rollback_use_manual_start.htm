rollback_use_manual_start

По умолчанию многопользовательская игра автоматически начинается, когда все игроки присоединяются к ней.

Вызов этой функции отключает автозапуск, и игра ждет, пока вы запустите ее вручную.

Если используется, эта функция должна выполняться до функции join/create.

Синтаксис:

rollback_use_manual_start();

Возвращается:

N/A

Пример:

rollback_use_manual_start();

if (!rollback_join_game())
{
    rollback_create_game(2);
}

Приведенный выше код включает ручной запуск, а затем присоединяется или создает новую многопользовательскую игру.
