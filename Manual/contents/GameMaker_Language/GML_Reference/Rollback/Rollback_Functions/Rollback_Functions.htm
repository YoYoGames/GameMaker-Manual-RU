Функции отката

Следующие функции используются для начала, присоединения или выхода из многопользовательской игры:

  • откат_вступления_в_игру
  • откат_создания_игры
  • откат_начало_игры
  • откат_покинуть_игру

Следующие функции используются для определения и получения информации для вашей игры:

  • Установите
    • откат_определить_игрока
    • откат_определить_вход
    • rollback_define_mock_input
    • откат_определить_входной_кадр_задержка
    • откат_использование_ручного_старта
    • rollback_use_player_prefs
    • rollback_set_player_prefs
  • Получить
    • rollback_get_input
    • rollback_get_info
    • rollback_get_player_prefs

Следующие функции предусмотрены для использования во время активной игры:

  • откат_чата
  • откат_синхронизации_по_кадру

Следующие функции используются для тестирования или для включения/выключения функций, связанных с тестированием:

  • rollback_use_random_input
  • откат_отображение_событий
