rollback_get_info

Эта функция возвращает структуру, содержащую информацию об игроке. Ее следует использовать только при ручном управлении экземплярами игроков. Если вы используете rollback_define_player()то каждый экземпляр игрока будет автоматически получать эту информацию в виде переменных.

Функция принимает в качестве аргумента ID игрока и возвращает структуру со следующими переменными:

Переменная Тип Описание
player_avatar_url Строка URL, указывающий на изображение аватара игрока
player_avatar_sprite Актив спрайта Спрайт, содержащий аватар игрока
player_name Строка Имя игрока
player_type Строка Говорит о том, зарегистрирован ли игрок в GX.games или нет.

Будет "Гость" для гостевых пользователей и "Пользователь" для зарегистрированных пользователей.
player_user_id Строка ID игрока на GX.games
player_prefs Любой Предпочтения, установленные для игрока

Синтаксис:

rollback_get_info(player_id);

Аргумент Тип Описание
player_id Реально The ID of the player for which information will be returned.
