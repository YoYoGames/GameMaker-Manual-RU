откат_начало_игры

Эта функция заставляет текущую многопользовательскую игру начаться, пока все игроки не присоединились к ней. Вы должны сначала создать многопользовательскую игру, используя rollback_create_game().

Обычно система ждет, пока все игроки присоединятся, прежде чем автоматически начать игру. Эту функцию можно использовать для раннего начала игры, пока все игроки еще не присоединились.

Вы можете вызвать rollback_use_manual_start() чтобы заставить Rollback ждать эту функцию даже после того, как все игроки присоединились.

Вызов этой функции вызовет событие Rollback Start.

Синтаксис:

rollback_start_game();

Возвращается:

N/A

Пример:

// Step event
if (!rollback_game_running && keyboard_check_pressed(vk_enter))
{
    rollback_start_game();
}

Если игра Rollback не запущена, и нажата клавиша Enter, это запустит игру.

Это позволяет игроку начать игру вручную до того, как все игроки присоединятся к ней.
