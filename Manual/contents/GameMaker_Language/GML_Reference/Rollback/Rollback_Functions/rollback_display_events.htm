откат_отображение_событий

По умолчанию при запуске игры в левом верхнем углу появляются отладочные сообщения, отображающие события мультиплеера по мере их возникновения. Вы можете отключить эти сообщения, вызвав эту функцию с аргументом false, и включить их снова, указав true.

Синтаксис:

rollback_display_events(enabled);

Аргумент Тип Описание
enabled Булево Specifies whether event display should be enabled (true) или отключен(false). По умолчанию он включен.

Возвращается:

N/A

Пример:

rollback_display_events(false);

Приведенный выше код отключил отображение событий.
