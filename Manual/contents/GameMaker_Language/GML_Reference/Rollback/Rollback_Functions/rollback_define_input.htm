откат_определить_вход

Эта функция используется для определения пользовательских входов для всех игроков. Она принимает struct, который должен содержать все входы с назначенными им входными константами.

Чтобы узнать, как назначить пользовательские входы, см. раздел Пользовательские элементы управления.

Если используется, эта функция должна выполняться до функции join/create.

Синтаксис:

rollback_define_input(input_struct);

Аргумент Тип Описание
input_struct Структура A struct containing input names as variables, and their assigned inputs as values.

Возвращается:

N/A

Пример:

rollback_define_input({
    fire: mb_left,
    interact: vk_space,
    left: [ord("A"), vk_left],
    right: [ord("D"), vk_right]
});

Приведенный выше код определяет пользовательские входы для всех игроков. Последние два входа используют несколько элементов управления, перечисляя их в массиве.

Расширенный пример см. в разделе Определение входов.
