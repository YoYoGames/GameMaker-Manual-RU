rollback_set_player_prefs

После включения предпочтений эта функция используется для установки предпочтений для локального игрока. Это означает, что функция должна полагаться на локальный ввод, а не на ввод Rollback.

Предпочтения могут быть любого типа. Если заданы предпочтения по умолчанию, убедитесь, что вы используете тот же тип данных и структуру.

Когда любой игрок устанавливает свои предпочтения, все в игре получают событие отката.

Если используется, эта функция должна выполняться до функции join/create.

Пожалуйста, прочитайте: Предпочтения игрока

Синтаксис:

rollback_set_player_prefs(preferences);

Аргумент Тип Описание
preferences Любой The preferences to set for the local player.

Возвращается:

N/A

Пример:

rollback_set_player_prefs({
персонаж: obj_customizer.character_selected,
цвет: obj_customizer.color_selected
});

Приведенный выше код устанавливает настройки для игрока, считывая значения из объекта "customizer", который будет неуправляемым (локальным).
