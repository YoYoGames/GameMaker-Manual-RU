rollback_define_mock_input

Эта функция может быть использована во время Sync Test для назначения временного "имитационного" ввода для указанного плеера. Это отменяет элементы управления, определенные для игрока, и любой случайный ввод из Sync Test.

Информацию о его использовании см. в разделе Макетный ввод.

Синтаксис:

rollback_define_mock_input(player_id, input_struct);

Аргумент Тип Описание
player_id Реально The ID of the player to assign mock input to.
input_struct Структура A struct containing input names as variables, and their assigned inputs as values.

Возвращается:

N/A

Пример:

rollback_define_input({
    fire: mb_left,
    interact: vk_space,
    left: ord("A"),
    right: ord("D")
});

rollback_define_mock_input(1, {
    fire: vk_control,
    interact: vk_shift,
    left: ord("J"),
    right: ord("L")
});

В приведенном выше коде сначала определяются входы для игроков. Затем он определяет макет входа для игрока 1, сохраняя те же имена входов, но назначая разные входы. Таким образом, два человека могут играть в игру локально, используя клавиатуру.
