rollback_get_input

Эта функция возвращает структуру, содержащую входные значения для игрока. Она принимает необязательный аргумент, указывающий ID игрока, для которого будут возвращены входные данные.

При использовании в экземпляре игрока, который был создан с помощью функции rollback_define_player()вам не нужно указывать ID игрока, поэтому вы можете вызвать эту функцию без аргументов.

При использовании в любых других экземплярах или экземплярах игроков, созданных вручную, необходимо указать ID игрока.

Информацию о том, какие входы возвращаются и как их можно изменить, см. в разделе Определение входов.

Синтаксис:

rollback_get_input(player_id);

Аргумент Тип Описание
player_id Реально OPTIONAL Идентификатор игрока, для которого будут возвращены входные данные.

Возвращается:

Struct

Пример:

var _input = rollback_get_input();

if (_input.left) x -= 2;
if (_input.right) x += 2;
if (_input.up) y -= 2;
if (_input.down) y += 2;

Приведенный выше код получает входной struct в экземпляре игрока, который был автоматически создан системой Rollback. На основе возвращенных значений он перемещает экземпляр на 2 пикселя в каждом направлении.

Расширенный пример смотрите в разделе Создание многопользовательской игры.
