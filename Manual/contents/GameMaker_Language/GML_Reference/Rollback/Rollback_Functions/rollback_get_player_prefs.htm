rollback_get_player_prefs

Эта функция возвращает набор предпочтений для данного ID игрока.

Если ID игрока не указан, функция должна быть вызвана внутри экземпляра игрока, в этом случае будут возвращены предпочтения для этого игрока.

Вы также можете использовать переменную экземпляра player_prefs.

Эта функция возвращает последние предпочтения, которые были синхронизированы для данного игрока, поэтому установка предпочтений не приведет к немедленному обновлению предпочтений, которые возвращает эта функция.

Пожалуйста, прочитайте: Предпочтения игрока

Синтаксис:

rollback_get_player_prefs([player_id]);

Аргумент Тип Описание
player_id Реально OPTIONAL Идентификатор игрока, для которого возвращаются предпочтения.

Возвращается:

Any

Пример:

var _my_prefs = rollback_get_player_prefs();

image_index = _my_prefs.character;

Приведенный выше код получает предпочтения, установленные для текущего игрока, и применяет предпочтение к переменной экземпляра.
