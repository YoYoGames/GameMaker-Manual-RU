rollback_leave_game

Эта функция завершает текущую многопользовательскую сессию, позволяя игроку присоединиться или создать новую игру снова.

Когда хозяин комнаты уходит, право собственности на эту комнату переходит к другому игроку, и игра может продолжаться. Игра заканчивается только тогда, когда все игроки покидают комнату.

Избегание ошибок

После выхода игрока из многопользовательской сессии GameMaker продолжает выполнять весь игровой код, однако он возвращается в состояние "перед запуском", в котором нельзя создавать или изменять управляемые экземпляры.

Это означает, что вам нужно вручную уничтожать экземпляры игроков при выходе из игры и приостанавливать любую другую игровую логику, используя rollback_game_running в условии, чтобы избежать ошибок.

См: Состояние перед началом отката

Триггерное событие

Когда игрок покидает игру, в которой вы присутствуете, срабатывает событие Rollback Event. Дополнительную информацию см. в разделе События отката.

Синтаксис:

rollback_leave_game();

Возвращается:

N/A

Пример:

if (keyboard_check_pressed(vk_escape))
{
    rollback_leave_game();
}

Приведенный выше код заставляет игрока выйти из игры при нажатии клавиши Escape.
