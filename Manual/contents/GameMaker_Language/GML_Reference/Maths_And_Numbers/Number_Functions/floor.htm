пол

Возвращает пол числа n, то есть n, округленное до целого числа. Это аналогично функции round() но округляет только в меньшую сторону, независимо от десятичного значения, поэтому floor(5.99999) вернет 5, как и floor(5.2), floor(5.6457) и т.д..

Синтаксис:

floor(n);

АргументТип Описание
n The number to floor.

Возвращается:

Пример:

val = floor( 3.9 );

При этом значение val будет равно 3.
