random_get_seed

Для генерации случайного числа GameMaker начинает со случайного начального числа. С помощью этой функции вы можете получить это число, которое затем можно сохранить для воспроизведения определенной серии случайных событий. Например, допустим, у вас есть процедурно но игрок должен иметь возможность возвращаться в ранее открытые области. В начале каждой области вы можете сохранить случайное зерно с помощью этой функции, и каждый раз, когда вам нужно вернуться туда, вы можете использовать функцию random_set_seed() чтобы установить семя на прежнее значение и тем самым заставить GameMaker воспроизвести тот же набор "случайных" функций, который он использовал ранее.

Синтаксис:

random_get_seed();

Возвращается:

Пример:

ini_open("Save.ini")
ini_write_real("Levels", string(level), random_get_seed());
ini_close();

Приведенный выше код получит случайное зерно и сохранит его в ini-файле.
