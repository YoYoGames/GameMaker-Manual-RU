max

Эта функция возвращает максимальное из входных значений, которых может быть до 16. Например, max(12, 96, 32, 75) вернет 96, так как это максимальное из всех входных значений.

Синтаксис:

max(val1, val2, ... max_val);

АргументТип Описание
val0 ... max_val The values to compare.

Возвращается:

Пример:

x = max(x, 0);

Это не позволит игроку выйти из левой части комнаты. Это работает путем постоянной установки x либо на себя, либо на 0, в зависимости от того, что больше. Если игрок выйдет слева, его x будет меньше 0 (т.е. отрицательным), поэтому он будет установлен прямо назад.
