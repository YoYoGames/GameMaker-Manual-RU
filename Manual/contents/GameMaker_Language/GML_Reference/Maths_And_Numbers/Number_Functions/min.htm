min

Эта функция возвращает минимальное из входных значений, которых может быть столько, сколько вам нужно (обратите внимание, что большее количество аргументов означает, что функция будет разбираться медленнее). Например, min(12, 96, 32, 75) вернет 12, так как это наименьшее значение из всех входных значений. наименьшее из всех входных значений.

Синтаксис:

min(val1, val2, ... max_val);

АргументТип Описание
val0 ... max_val The values to compare.

Возвращается:

Пример:

x = min(x, room_width);

Это не позволит игроку выйти из правой части комнаты. Это работает, постоянно устанавливая x либо на себя, либо на переменную room_width, в зависимости от того, что меньше. Если игрок выйдет справа, его x будет больше, чем room_width, поэтому он будет установлен обратно.
