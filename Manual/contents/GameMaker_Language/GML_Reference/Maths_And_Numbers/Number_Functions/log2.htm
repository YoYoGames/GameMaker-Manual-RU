log2

Эта функция в основном спрашивает: "Сколько двоек нужно перемножить, чтобы получить n?". Обычным примером использования в программировании является вычисление количества битов, необходимых для представления числа. Для этого мы вычисляем log2(x) числа, округляем его вниз и прибавляем 1 - например, log2(100) возвращает 6,643856, что округленное вниз число равно 6, к которому мы прибавляем 1 и получаем 7. Таким образом, для представления 100 нам нужно 7 бит. Она также может быть использована для описания экспоненциального роста или и часто называется двоичным логарифмом.

Синтаксис:

log2(n);

АргументТип Описание
n The input value.

Возвращается:

Пример:

colourbits = floor(log2(colour)) - 1;

Приведенный выше код хранит количество битов, необходимых для представления значения, хранящегося в переменной "color".
