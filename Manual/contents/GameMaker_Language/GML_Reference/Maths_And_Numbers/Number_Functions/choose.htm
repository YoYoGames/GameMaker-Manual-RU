выбрать

Иногда для случайного выбора нужно указать не числа, а что-то другое, или нужные числа не расположены в реальном порядке или в заданном диапазоне. В этих случаях для генерации случайного результата используется choose(). Например, если вы хотите создать объект со случайным sprite в начале, то вы можете использовать эту функцию для установки индекса sprite на один из множества заданных sprites. Обратите внимание, что вы можете иметь столько аргументов, сколько вам нужно (обратите внимание, что большее количество аргументов будет означать, что функция будет медленнее разбираться).

ПРИМЕЧАНИЕ Эта функция будет возвращать одно и то же значение при каждом новом запуске игры, поскольку GameMaker каждый раз генерирует одно и то же начальное случайное зерно, чтобы облегчить отладку кода. Чтобы избежать такого поведения, используйте randomise() в начале вашей игры.

Синтаксис:

choose(val0, val1, val2... max_val);

Аргумент Тип Описание
val0... max_val Переменная Any type of value(s).

Возвращается:

Variable (One of the given arguments)

Пример:

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen");

Приведенный выше код использует select для установки ряда свойств экземпляра.
