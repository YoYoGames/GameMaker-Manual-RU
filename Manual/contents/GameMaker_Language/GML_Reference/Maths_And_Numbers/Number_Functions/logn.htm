logn

Эта функция аналогична функциям log2(n) и log10(n), только вы указываете значение логарифмической базы. Например, logn(5,25) вернет, сколько 5 нужно перемножить. чтобы получить 25 (что равно 2).

Синтаксис:

logn(n, val);

АргументТип Описание
n The log base.
val The input value.

Возвращается:

Пример:

logval = logn(5, num);

Приведенный выше код получает log5 значения, хранящегося в "num".
