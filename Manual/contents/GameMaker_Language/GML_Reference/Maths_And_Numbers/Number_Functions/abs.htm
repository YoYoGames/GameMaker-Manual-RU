абс

Эта функция возвращает абсолютное значение входного аргумента, так что если это положительное значение, то оно останется неизменным, а если отрицательное, то оно будет умножено на -1, чтобы сделать его положительным.

Синтаксис:

abs(val)

АргументТип Описание
val The number to turn absolute.

Возвращается:

Пример:

x += abs (x - mouse_x)

Это добавит количество, равное абсолютному значению текущей позиции x экземпляра и позиции x мыши.
