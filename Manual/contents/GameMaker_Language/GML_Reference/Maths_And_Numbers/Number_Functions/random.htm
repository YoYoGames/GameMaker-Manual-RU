случайный

Эта функция хороша для вероятностей, где возвращение целого числа не является необходимым. Например, random(100) вернет значение от 0 до 99, но это значение может быть 22,56473! Вы также можете использовать в этой функции не целые, а действительные числа, например, random(0.5), которая вернет значение от 0 до 0,4999999.

ПРИМЕЧАНИЕ: Эта функция будет возвращать одно и то же значение при каждом новом запуске игры, поскольку GameMaker каждый раз генерирует одно и то же начальное случайное зерно, чтобы облегчить отладку кода. Чтобы избежать такого поведения, используйте randomise() в начале вашей игры.

Синтаксис:

random(n);

АргументТип Описание
n The upper range from which the random number will be selected.

Возвращается:

Пример:

if (random(10) >= 9)
{
    score += 100;
}

Это даст примерно один шанс из десяти на добавление 100 к счету.
