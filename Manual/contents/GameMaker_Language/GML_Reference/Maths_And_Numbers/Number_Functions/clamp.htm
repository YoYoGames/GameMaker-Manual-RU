С помощью этой функции вы можете поддерживать входное значение в заданном диапазоне.

Синтаксис:

clamp(val, min, max)

АргументТип Описание
val The value to clamp.
min The minimum value to clamp between.
max The maximum value to clamp between.

Возвращается:

Пример:

speed = clamp(speed, 1, 10);

Приведенный выше код будет зажимать скорость так, чтобы она никогда не падала ниже 1 и не превышала 10.
