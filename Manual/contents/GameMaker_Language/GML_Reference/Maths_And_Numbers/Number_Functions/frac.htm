frac

Эта функция возвращает дробную часть n, то есть часть, стоящую за десятичной точкой. Она возвращает только десятичные дроби, стоящие за точкой, поэтому frac(3.125) вернет 0,125, frac(6.921) вернет 0,921 и т.д..

Синтаксис:

frac(n);

АргументТип Описание
n The number to change.

Возвращается:

Пример:

val = frac(3.4);

Это позволит установить значение val на 0,4.
