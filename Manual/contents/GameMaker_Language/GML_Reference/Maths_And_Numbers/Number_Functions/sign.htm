подписать

Эта функция позволяет определить, является ли число положительным, отрицательным или ни тем, ни другим, и возвращает 1, -1, 0 соответственно. Например - sign(458) вернет 1, sign(-5) вернет -1 и sign(0) вернет 0.

Синтаксис:

sign(n);

АргументТип Описание
n The number to get the sign of.

Возвращается:

Пример:

y += sign(y - mouse_y);

Приведенный выше код добавит к y 1, -1 или 0 в зависимости от результата y - mouse_y.
