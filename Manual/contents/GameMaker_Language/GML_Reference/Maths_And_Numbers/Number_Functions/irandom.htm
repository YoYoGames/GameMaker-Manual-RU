irandom

Эта очень полезная функция возвращает только целые числа (целые числа). Так, например, для получения случайного числа от 0 до 9 вы можете использовать irandom(9), и она вернет число от 0 до 9 включительно. Обратите внимание, что можно использовать и плавающие числа, но верхнее значение после точки будет исключено, поэтому irandom(9.7) вернет значение только от 0 до 9. Верхняя граница функции составляет $7fffffffffffffffLL, поэтому при использовании очень больших чисел следует соблюдать осторожность.

ПРИМЕЧАНИЕ Эта функция будет возвращать одно и то же значение при каждом новом запуске игры, поскольку GameMaker каждый раз генерирует одно и то же начальное случайное зерно, чтобы облегчить отладку кода. Чтобы избежать такого поведения, используйте randomise() в начале вашей игры.

Синтаксис:

irandom(n);

АргументТип Описание
n The upper range from which the random number will be selected.

Возвращается:

Пример:

if (irandom(9) == 1)
{
    score += 100;
}

Это даст один шанс из десяти (поскольку 0 включено) добавить 100 к счету.
