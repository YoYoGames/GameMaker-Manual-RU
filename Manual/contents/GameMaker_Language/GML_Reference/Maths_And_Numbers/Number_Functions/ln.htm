ln

Натуральный логарифм ln(n) - это количество времени, необходимое для достижения определенного уровня непрерывного роста, где n - достигнутый уровень. Таким образом, если мы хотим узнать, сколько единиц времени нам нужно, чтобы получить 20 единиц роста, мы используем ln(20), который возвращает 2,99 единиц времени для достижения такого роста.

Синтаксис:

ln(n);

АргументТип Описание
n The input value.

Возвращается:

Пример:

alarm[0] = ln(age) * room_speed;

Приведенный выше код использует натуральный логарифм значения, хранящегося в переменной "возраст", для установки сигнала тревоги.
