мощность

Это вернет значение числа, умноженного на себя "n" раз. Например, power(5,3) умножит 5 на себя 3 раза и вернет 125, что равносильно тому, что сказать 5*5*5=125. Обратите внимание, что значение "x" (число, которое нужно изменить) не может быть отрицательным значением.

Синтаксис:

power(x, n);

АргументТип Описание
x The number to change.
n How many times to multiply x by itself.

Возвращается:

Пример:

score += power(dmg, 3);

Это добавит значение удерживаемого в переменной "dmg" в степени 3 к переменной score.
