dcos

В прямоугольном треугольнике cos(val) = Прилегание / Гипотенуза, где val - один из трех углов. Эта функция всегда возвращает число от 1 до -1.

ПРИМЕЧАНИЕ: Эта функция принимает значение в градусах, а не в радианах.

Синтаксис:

dcos(val);

АргументТип Описание
val The angle (in degrees) to return the cosine of.

Возвращается:

Пример:

val = dcos(45);

Это установит значение "val" равным 0,71.
