darcsin

Возвращает обратный синус x, при этом если dsin(x)=n, darcsin(n)=x, то полученное число будет находиться между -90 и 90.

ПРИМЕЧАНИЕ: Здесь принимается только число от -1 до 1 (все остальное приведет к ошибке).

ПРИМЕЧАНИЕ: Эта функция возвращает значение в градусах, а не в радианах.

Синтаксис:

darcsin(x);

АргументТип Описание
val The value to return the inverse sine of.

Возвращается:

Пример:

val = darcsin(-1);

Это установит значение "val" на -90.
