точка_расстояния_3d

Эта функция принимает заданные компоненты вектора и возвращает длину (расстояние) вектора. Она работает точно так же, как point_distance() но с добавлением факторизации значения z (глубины) для использования в трехмерном пространстве.

Синтаксис:

point_distance_3d(x1, y1, z1, x2, y2, z2);

АргументТип Описание
x1 The x coordinate of the first component of the vector
y1 The y coordinate of the first component of the vector
z1 The z coordinate of the first component of the vector
x2 The x coordinate of the second component of the vector
y2 The y coordinate of the second component of the vector
z2 The z coordinate of the second component of the vector

Возвращается:

Пример:

var inst, ex, ey, ez;
inst = instance_nearest(x, y, enemy);
if inst
{
    ex = inst.x;
    ey = inst.y;
    ez = inst.z;
    if point_distance_3d(x, y, z, ex, ey, ez) < 200
    {
        instance_create_layer(x, y, "Bullets", obj_Missile)
    }
}

Приведенный выше код получит координаты x и y и z ближайшего врага, а затем использует их для проверки расстояния (длины) вектора, образованного им и координатами игрока. Если значение меньше 200, объект игрока создаст экземпляр "obj_Missile".
