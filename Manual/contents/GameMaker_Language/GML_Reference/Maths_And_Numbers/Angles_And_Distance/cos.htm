cos

В прямоугольном треугольнике cos(val) = Прилегание / Гипотенуза, где val - один из трех углов. Эта функция всегда возвращает число от 1 до -1.

ПРИМЕЧАНИЕ: Эта функция принимает значение в радианах, а не в градусах.

Синтаксис:

cos(val);

АргументТип Описание
val The angle (in radians) to return the cosine of.

Возвращается:

Пример:

val = cos( 0 );

При этом значение val будет равно 1.
