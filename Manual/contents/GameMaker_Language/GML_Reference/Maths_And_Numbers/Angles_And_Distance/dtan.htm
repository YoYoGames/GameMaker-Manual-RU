dtan

В прямоугольном треугольнике dtan(val) = Противоположный / Смежный, где "val" - один из трех углов.

ПРИМЕЧАНИЕ: Огромное количество значений (например, 90 или -90) будут ошибочными в этой функции из-за того, что они возвращаются в бесконечность, графическое представление этого даст асимптоты при этих значениях.

ПРИМЕЧАНИЕ: Эта функция принимает значение в градусах, а не в радианах

Синтаксис:

dtan(val);

Аргумент Тип Описание
val Реально The angle (in degrees) to return the tangent of.

Возвращается:

Real

Пример:

val = dtan(45);

Это установит значение "val" равным 1.
