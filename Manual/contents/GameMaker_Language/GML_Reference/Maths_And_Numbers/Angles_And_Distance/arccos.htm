arccos

Возвращает обратный косинус x, при этом если cos(val)=n, arccos(n)=val, то полученное число будет между pi и 0.

ПРИМЕЧАНИЕ:Здесь будет приниматься только число от -1 до 1 (все остальное приведет к ошибке).

ПРИМЕЧАНИЕ: Эта функция принимает значение в радианах, а не в градусах.

Синтаксис:

arccos(x)

АргументТип Описание
x The angle (in radians) to return the inverse cosine of.

Возвращается:

Пример:

val = arccos(0);

Это установит значение val равным pi/2.
