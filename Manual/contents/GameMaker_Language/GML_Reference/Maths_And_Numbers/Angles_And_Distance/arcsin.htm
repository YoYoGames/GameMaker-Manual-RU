arcsin

Возвращает обратный синус x, причем если sin(x)=n, arcsin(n)=x, то полученное число будет находиться между -pi/2 и pi/2.

ПРИМЕЧАНИЕ: Это значение может принимать только значения от -1 до 1 (все остальное приведет к ошибке).

ПРИМЕЧАНИЕ: Эта функция возвращает значение в радианах, а не в градусах.

Синтаксис:

arcsin(x);

АргументТип Описание
x The value to return the inverse sine of.

Возвращается:

Пример:

val = arcsin(0);

Это установит значение val равным 0.
