point_direction

Эта функция возвращает направление вектора, образованного указанными компонентами [x1,y1] и [x2,y2], относительно фиксированных координат x/y комнаты. Например, на изображении ниже, если мы хотим получить направление от позиции корабля игрока к позиции врага, чтобы мы могли выпустить ракету по врагу, мы используем эту функцию (точный код приведен в примере под изображением):

Синтаксис:

point_direction(x1, y1, x2, y2)

АргументТип Описание
x1 The x coordinate of the first component of the vector
y1 The y coordinate of the first component of the vector
x2 The x coordinate of the second component of the vector
y2 The y coordinate of the second component of the vector

Возвращается:

Пример:

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
with (instance_create_layer(x, y, "Bullets", obj_Missile))
{
    direction = point_direction(x, y, _ex, _ey);
}

Приведенный выше код получит координаты x и y ближайшего врага и затем передаст их объекту пули, чтобы использовать их в функции point_direction для правильной установки направления движения.
