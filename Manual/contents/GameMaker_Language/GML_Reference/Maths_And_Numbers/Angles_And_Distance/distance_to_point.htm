расстояние_до_точки

Эта функция вычисляет расстояние от края ограничительной рамки вызывающего объекта до указанной позиции x/y в комнате, при этом возвращаемое значение будет в пикселях. Обратите внимание, что если у вызывающего объекта нет sprite или не определена маска, результаты будут неверными.

Синтаксис:

distance_to_point(x, y);

АргументТип Описание
x The x position to check.
y The y position to check.

Возвращается:

Пример:

if (distance_to_point(obj_Player.x, obj_Player.y) < range)
{
    can_shoot = true;
}

Приведенный выше код проверяет расстояние до позиции x/y объекта игрока, и если оно меньше значения, хранящегося в переменной "range", то переменная "canshoot" устанавливается в true.
