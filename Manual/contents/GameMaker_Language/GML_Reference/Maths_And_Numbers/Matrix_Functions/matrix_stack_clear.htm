matrix_stack_clear

Эта функция очищает стек матрицы.

Синтаксис:

matrix_stack_clear();

Возвращается:

Пример:

if !matrix_stack_is_empty()
{
    matrix_stack_clear();
}

Приведенный выше код проверяет стек матрицы на то, пуст ли он, и если нет, то очищает его.
