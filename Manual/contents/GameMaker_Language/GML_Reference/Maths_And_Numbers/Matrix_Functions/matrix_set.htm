matrix_set

Эта функция принимает массив из 16 значений, соответствующих заданному типу матрицы 4x4, где элементы [0 - 3] будут строкой 1, элементы [4 -7] - строкой 2 и т.д.... (см. рисунок на главной странице). Вы можете создать такую матрицу, используя matrix_build() или matrix_get() или просто создать массив самостоятельно и передать его в функцию. Доступными типами матриц являются вид, проекция и мир, для которых можно использовать одну из следующих констант:

Постоянная Описание
matrix_view Текущая матрица представления
matrix_projection Текущая матрица проекции
matrix_world Текущая мировая матрица

Синтаксис:

matrix_set(type, matrix);

Аргумент Тип Описание
type Тип матрицы Постоянная The type of matrix to get the values of (see the constants listed above)
matrix Матричный массив The matrix data as an array

Возвращается:

N/A

Пример:

matrix_set(matrix_world, m_array);

Приведенный выше код установит значения текущей мировой матрицы на значения, хранящиеся в матрице массива "m_array".
