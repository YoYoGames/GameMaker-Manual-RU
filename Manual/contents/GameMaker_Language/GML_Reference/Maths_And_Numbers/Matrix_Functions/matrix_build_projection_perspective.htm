matrix_build_projection_perspective

Эта функция строит матрицу перспективной проекции на основе размеров ближней плоскости обрезания, используя указанные ниже параметры.

Обратите внимание, что поле зрения камеры будет меняться в зависимости от указанных значений width, height и znear, поскольку ширина и высота проекции размещаются в позиции Z, указанной в аргументе znear.

Например, при постоянном размере 640x480 поле зрения будет шире, если znear находится ближе к камере, но будет более узким, если znear находится далеко от камеры. Это поведение демонстрируется на следующем рисунке:

В связи с этим рекомендуется использовать значение znear, равное ширине проекции, в результате чего поле зрения будет соответствовать эквивалентной ортографической проекции.

Синтаксис:

matrix_build_projection_perspective(width, height, znear, zfar);

АргументТип Описание
w The width of the projection at the near Z position.
h The height of the projection at the near Z position.
znear The near clipping plane.
zfar The far clipping plane.

Возвращается:

Matrix Array

Пример:

var projmat = matrix_build_projection_perspective(640, 480, 640.0, 32000.0);
camera_set_proj_mat(view_camera[0], projmat);

Приведенный выше код создает перспективную проекцию, а затем использует возвращенную матрицу для установки камеры, назначенной на порт просмотра[0].
