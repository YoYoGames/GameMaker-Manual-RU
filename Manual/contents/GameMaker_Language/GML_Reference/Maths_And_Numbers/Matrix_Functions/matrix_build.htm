matrix_build

Эта функция может использоваться для создания вашей собственной матрицы и возвращает матричный массив, который должен быть сохранен в переменной для дальнейшего использования. Она принимает 3-мерные(x, y, z) значения перевода, вращения и масштаба и использует их для построения матричного массива.

Всего матричный массив содержит 16 значений, где начальными 4 элементами являются строка/колонка 1, следующие 4 элемента строка/столбец 2 и так далее, как часть матрицы 4x4. Упорядочен ли массив по строкам или столбцам, зависит от целевой платформы, так как графический рендерер, используемый платформой, может использовать матрицы с упорядочиванием по строкам или по столбцам.

ПРИМЕЧАНИЕ При построении новой матрицы таким образом порядок работы - YXZ.

Синтаксис:

matrix_build(x, y, z, xrotation, yrotation, zrotation, xscale, yscale, zscale);

АргументТип Описание
x The x component of the translation vector.
y The y component of the translation vector.
z The z component of the translation vector.
xrotation The angle to rotate around the x-axis (in degrees °).
yrotation The angle to rotate around the y-axis (in degrees °).
zrotation The angle to rotate around the z-axis (in degrees °).
xscale The x scale amount.
yscale The y scale amount.
zscale The z scale amount.

Возвращается:

Matrix Array

Пример:

t_matrix = matrix_build(x, y, 0, 0, 90, 0, 1, 2, 1);

Приведенный выше код построит новое преобразование матрицы и сохранит индекс результирующей матрицы в переменной "t_matrix".
