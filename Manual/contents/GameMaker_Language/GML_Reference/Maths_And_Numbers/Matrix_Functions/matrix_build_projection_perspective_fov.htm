matrix_build_projection_perspective_fov

Эта функция строит матрицу перспективной проекции на основе поля зрения, используя заданные параметры, перечисленные ниже.

Синтаксис:

matrix_build_projection_perspective_fov(fov_y, aspect, znear, zfar);

АргументТип Описание
fov The angle of the field of view.
aspect The aspect ratio of the field of view.
znear The near clipping plane.
zfar The far clipping plane.

Возвращается:

Matrix Array

Пример:

projmat = matrix_build_projection_perspective_fov(60, 320/240, 1.0, 32000.0);
camera_set_proj_mat(view_camera[0], projmat);

Приведенный выше код создает матрицу проекции поля зрения, которая затем сохраняется в переменной. Эта матрица затем используется для настройки камеры, назначенной на порт просмотра[0].
