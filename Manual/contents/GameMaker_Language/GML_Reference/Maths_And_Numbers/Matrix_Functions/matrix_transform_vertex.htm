matrix_transform_vertex

Эта функция преобразует вектор по матрице. Вы передаете идентификатор матрицы преобразования (возвращаемый функцией matrix build()), а также значения x, y и z для вектора, который нужно преобразовать. Функция вернет массив из 3 элементов, где:

  • массив[0] = x
  • массив[1] = y
  • array[2] = z.

Синтаксис:

matrix_transform_vertex(matrix, x, y, z);

АргументТип Описание
matrix The matrix to use
x The x component of the transform vector
y The y component of the transform vector
z The z component of the transform vector

Возвращается:

(3 elements)

Пример:

t_matrix = matrix_build(0, 0, 0, 0, 90, 0, 1, 2, 1);
verts = matrix_transform_vertex(t_matrix, x, y, z);

Приведенный выше код преобразует заданные значения с помощью матрицы, хранящейся в переменной "t_matrix", и сохраняет их в массиве "verts".
