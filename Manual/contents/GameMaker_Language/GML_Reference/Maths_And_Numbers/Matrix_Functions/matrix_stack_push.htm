matrix_stack_push

Эта функция используется для перемещения новой матрицы на вершину текущего стека матриц. Сначала заданная матрица умножается на матрицу, находящуюся на вершине стека, а затем полученная матрица помещается в стек.

Эта функция полезна для применения нескольких матричных преобразований к визуальным изображениям без необходимости вручную перемножать различные матрицы вместе.

Например, вы можете поместить в стек базовую матрицу, которая рисует графику в определенной прямоугольной области (например, в окне). После рисования графики с помощью этой матрицы вы можете поместить в стек другую матрицу для рисования графики внутри под-области (например, кнопки в окне). После рисования чего-либо в этой под-области, вы можете вызвать matrix_stack_pop(), чтобы удалить матрицу из стека и продолжить рисование графики в основной области окна.

Синтаксис:

matrix_stack_push(matrix);

АргументТип Описание
matrix The matrix to push to the stack

Возвращается:

Пример:

var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2);
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(spr_tyre, 0, 0, 0);
matrix_stack_pop();
matrix_stack_pop();

Приведенный выше код построит две новые матрицы и поместит их в стек матриц. После этого матрица мира устанавливается на вершину стека, рисуется sprite, а затем трансформации выталкиваются из стека.
