matrix_build_projection_ortho

Эта функция строит матрицу ортографической проекции на основе указанных ниже параметров (это метод проекции по умолчанию, используемый при создании комнаты в GameMaker без каких-либо изменений). Иногда требуется переключиться с перспективной проекции на ортографическую, для чего и предназначена эта функция. Обычно она используется для рисования наложения, например, для отображения счета или других аспектов, поскольку это дает "плоский" вид нарисованных элементов (т.е. без перспективы) в 3D-игре. Чтобы получить представление о разнице между ортографическим и перспективным видами, смотрите рисунок ниже.

ПРИМЕЧАНИЕ: Вам также может понадобиться временно отключить удаление скрытых поверхностей, если вы хотите, чтобы информация рисовалась независимо от текущего значения глубины.

Синтаксис:

matrix_build_projection_ortho(width, height, znear, zfar);

АргументТип Описание
w The width of the projection.
h The height of the projection.
znear The near clipping plane.
zfar The far clipping plane.

Возвращается:

Matrix Array

Пример:

viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Приведенный выше код создает новую матрицу взгляда и ортографическую матрицу, сохраняет их идентификаторы в локальных переменных, а затем использует их для установки матриц вида и проекции для камеры, назначенной на порт вида[0].
