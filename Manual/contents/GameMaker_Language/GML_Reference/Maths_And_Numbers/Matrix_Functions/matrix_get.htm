matrix_get

Эта функция возвращает массив из 16 значений, соответствующих заданному типу матрицы 4x4, где строка 1 - элементы [0 - 3], строка 2 - элементы [4 -7] и т.д.. (см. изображение на главной странице). Доступными матрицами являются view, projection и world, для которых можно использовать одну из следующих констант:

Тип матрицы Постоянная
Постоянная Описание
matrix_view Текущая матрица представления
matrix_projection Текущая матрица проекции
matrix_world Текущая матрица мира

Синтаксис:

matrix_get(type);

Аргумент Тип Описание
type Тип матрицы Постоянная The type of matrix to get (see the constants listed above)

Возвращается:

Matrix Array

Пример:

v_array = matrix_get(matrix_view);

Приведенный выше код получит значения текущей матрицы представления и заполнит ими массив "v_array".
