matrix_build_lookat

Эта функция строит матрицу "вида" (view-at) на основе заданных параметров, перечисленных ниже. Поскольку эта функция изменяет матрицу вида, а не матрицу проекции, сначала следует инициализировать матрицу проекции с помощью другой функции матрицы. matrix_build_projection_perspective()а затем использовать эту функцию для перемещения камеры обзора в пределах проекции.

Чтобы задать вид, сначала нужно определить позицию, из которой вы смотрите, которая обозначается параметрами (xfrom, yfrom, zfrom). Затем необходимо указать направление, в котором вы смотрите, для этого нужно указать вторую точку, в которую вы смотрите, с помощью аргументов (xto, yto, zto). Наконец, можно еще повернуть камеру вокруг линии от точки обзора до точки взгляда, и для этого нужно задать вектор "вверх" - направление, в котором находится камера. Это задается тремя последними аргументами (xup, yup, zup).

Синтаксис:

matrix_build_lookat(xfrom, yfrom, zfrom, xto, yto, zto, xup, yup, zup);

АргументТип Описание
xfrom The x coordinate of the position to look from.
yfrom The y coordinate of the position to look from.
zfrom The z coordinate of the position to look from.
xto The x coordinate of the position to look to.
yto The y coordinate of the position to look to.
zto The z coordinate of the position to look to.
xup The x coordinate of the "up" vector.
yup The y coordinate of the "up" vector.
zup The z coordinate of the "up" vector.

Возвращается:

Matrix Array

Пример:

viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Приведенный выше код создает новую матрицу взгляда и ортографическую матрицу, сохраняет их идентификаторы в локальных переменных, а затем использует их для установки матриц вида и проекции для камеры, назначенной на порт вида[0].
