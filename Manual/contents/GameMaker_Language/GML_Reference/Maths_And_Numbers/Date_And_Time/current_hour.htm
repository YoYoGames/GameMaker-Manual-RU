текущий_час

Эта переменная, доступная только для чтения, возвращает час, который соответствует текущему моменту, основываясь на часовом поясе по умолчанию для системы (т.е. местное время). Вы можете изменить используемый базовый часовой пояс с помощью функции date_set_timezone().

Синтаксис:

current_hour;

Возвращается:

Пример:

draw_text(32, 32, "The time is " + string(current_hour) + ":" + string(current_minute) + "." + string(current_second));

Приведенный выше код выводит на экран текущее международное время.
