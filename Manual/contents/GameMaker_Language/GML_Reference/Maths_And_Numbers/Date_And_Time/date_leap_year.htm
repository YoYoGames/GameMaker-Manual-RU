дата_високосного_года

Эта функция возвращает значение true, если компонент года данного значения времени является високосным годом, или false в противном случае. Эта функция может быть удобна для таких вещей, как пасхальные яйца в ваших играх или для разблокировки специального содержимого.

Синтаксис:

date_leap_year(date);

АргументТип Описание
date The datetime to use.

Возвращается:

Пример:

if date_leap_year(date_current_datetime())
{
    if (!global.ExtraContent)
    {
        global.ExtraContent = true;
    }
}

Приведенный выше код проверяет текущую дату, чтобы узнать, является ли год високосным или нет. Если это так, то устанавливается глобальная переменная.
