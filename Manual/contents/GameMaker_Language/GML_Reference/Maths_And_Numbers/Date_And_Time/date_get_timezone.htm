дата_get_timezone

Эта функция получает базовый часовой пояс, используемый для всех остальных функций даты и времени. Этот часовой пояс может быть либо локальным (установленным системой), либо UTC, и функция возвращает одну из следующих констант:

Часовой пояс Постоянный
Постоянная Описание
timezone_local использовать местный часовой пояс, установленный системой
timezone_utc использовать всемирное координированное время

Синтаксис:

date_get_timezone();

Возвращается:

Time Zone Constant

Пример:

if (date_get_timezone() != timezone_utc)
{
    date_set_timezone(timezone_utc);
}

Этот код проверяет базовую настройку часового пояса для игры и, если она не UTC, то изменяет ее.
