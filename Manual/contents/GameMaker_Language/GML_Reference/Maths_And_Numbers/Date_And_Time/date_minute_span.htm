дата_минута_расстояния

С помощью этой функции вы можете получить количество минут между двумя датами. Это значение всегда положительное, а неполные минуты будут возвращены в виде дроби.

Синтаксис:

date_minute_span(date1, date2);

АргументТип Описание
date1 The first datetime.
date2 The datetime to compare it to.

Возвращается:

Пример:

diff = date_minute_span(date_create_datetime(2011, 9, 15, 11, 4, 0 ), date_current_datetime());

Это установит "diff" на количество минут между 15 сентября 2011 года, 11:04.0 и текущей датой и временем.
