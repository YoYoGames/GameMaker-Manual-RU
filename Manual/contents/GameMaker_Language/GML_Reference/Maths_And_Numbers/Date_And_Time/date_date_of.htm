date_date_of

Возвращает значение даты для заданного времени.

Синтаксис:

date_date_of(date);

АргументТип Описание
date The datetime to extract the date from.

Возвращается:

Пример:

today = date_date_of(date_current_datetime());

Это вернет только текущую дату и сохранит значение в переменной "today".
