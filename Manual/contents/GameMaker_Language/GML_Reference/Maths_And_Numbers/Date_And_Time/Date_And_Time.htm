Дата и время

ВGameMaker есть множество функций и переменных, связанных с получением даты и времени и вычислением других значений на их основе. По умолчанию эти функции возвращают местное время, установленное системой, но вы можете использовать следующую функцию для переключения между местным и UTC временем:

  • date_set_timezone
  • date_get_timezone

Следующие встроенные переменные могут быть использованы для получения определенных компонентов даты и времени:

  • current_time
  • current_second
  • current_minute
  • current_hour
  • current_day
  • current_weekday
  • current_month
  • current_year

Следующие функции можно использовать для получения определенных компонентов даты и времени, а также для создания новых или для получения промежутка дней/часов между двумя заданными датами и т.д....

У нас также есть две специальные функции времени, которые обеспечивают более высокий уровень точности и контроля при ведении игры:

  • get_timer
  • delta_time

Если вы ищете функции, связанные с таймингом, см. раздел Источники времени.
