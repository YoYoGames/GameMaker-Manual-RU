дата_дней_в_году

С помощью этой функции вы можете получить количество дней, которое имеет данный год, возвращая 365 для обычного года и 366 для високосного.

Синтаксис:

date_days_in_year(date);

АргументТип Описание
date The date to use.

Возвращается:

Пример:

days = date_days_in_year(date_current_datetime());

В этом случае значение "days" будет равно количеству дней в текущем году.
