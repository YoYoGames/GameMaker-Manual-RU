дата_установить_временную зону

Устанавливает базовый часовой пояс, который будет использоваться для всех остальных функций даты и времени. Этот часовой пояс может быть либо локальным (установленным системой), либо UTC, и вы можете использовать одну из следующих констант, чтобы определить, какой из них используется (по умолчанию это локальное время):

Постоянная Описание
timezone_local использовать местный часовой пояс, установленный системой
timezone_utc использовать всемирное координированное время

Синтаксис:

date_set_timezone(timezone);

Аргумент Тип Описание
timezone Часовой пояс Постоянный The time zone to use for the base time.

Возвращается:

N/A

Пример:

if (date_get_timezone() != timezone_utc)
{
    date_set_timezone(timezone_utc);
}

Этот код проверяет базовую настройку часового пояса для игры и, если она не UTC, то изменяет ее.
