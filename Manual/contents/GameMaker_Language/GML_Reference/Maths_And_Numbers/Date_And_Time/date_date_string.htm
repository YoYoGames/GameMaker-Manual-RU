дата_дата_строка

С помощью этой функции вы можете создать строку, содержащую заданную дату, отформатированную как день/месяц/год.

Синтаксис:

date_date_string(date);

АргументТип Описание
date The date to use.

Возвращается:

Пример:

str = date_date_string(date_current_datetime());
draw_text(32, 32, str);

В этом случае "str" будет содержать форматированную строку текущей даты и времени, отображаемую системой.
