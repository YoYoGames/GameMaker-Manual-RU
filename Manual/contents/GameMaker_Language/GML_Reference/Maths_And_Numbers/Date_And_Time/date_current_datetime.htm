дата_текущего_времени

Возвращает значение даты-времени текущего момента. Возвращаемое время основано на часовом поясе по умолчанию для системы (т.е. местное время). Вы можете изменить используемый базовый часовой пояс с помощью функции date_set_timezone().

Синтаксис:

date_current_datetime();

Возвращается:

Datetime

Пример:

myhour = date_get_hour(date_current_datetime());
myday = date_get_day(date_current_datetime());

Это установит локальную переменную myhour в час текущего времени, а myday - в день текущей даты.
