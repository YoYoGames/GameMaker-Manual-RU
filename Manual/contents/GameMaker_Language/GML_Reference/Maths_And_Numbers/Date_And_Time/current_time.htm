current_time

Эта переменная, доступная только для чтения, возвращает количество миллисекунд, прошедших с момента запуска игры.

Синтаксис:

current_time

Возвращается:

Пример:

if (current_time > 600000)
{
    msg = show_question_async("Would you like to rate?");
}

Приведенный выше код проверяет, не прошло ли более 10 минут, прежде чем задать вопрос пользователю.
