дата_инк_неделя

С помощью этой функции вы можете увеличить заданное значение времени на определенное количество недель, и она вернет новое значение времени.

Синтаксис:

date_inc_week(date, amount);

АргументТип Описание
date The datetime to add to.
amount The number of weeks (must be an integer) to add.

Возвращается:

Пример:

mynewdatetime = date_inc_week(date_current_datetime(), 52);

Это установит mynewdatetime на текущую дату, но с добавлением 52 недель.
