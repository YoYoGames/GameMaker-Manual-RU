дельта_время

Эта переменная возвращает время дельты кадра, которое представляет собой разницу во времени между предыдущим и текущим кадром. Это значение в микросекундах, где 1 микросекунда - это 1 000 000 000-я доля секунды.

Цель дельта-времени - устранить эффект запаздывания или медлительности компьютеров, которые пытаются обрабатывать сложную графику или большое количество кода. Это величина, которую можно прибавить к скорости объектов, чтобы в конечном итоге они двигались с одинаковой скоростью, независимо от запаздывания. В GameMaker это достигается путем использования переменной delta_time на каждом шаге, поскольку она измеряет время, прошедшее между одним шагом и следующим, в микросекундах. Поэтому переменная delta_time может быть использована для расчета того, насколько быстрее (например) должен двигаться игровой персонаж, чтобы компенсировать скачок задержки в игре.

Синтаксис:

delta_time

Возвращается:

Real (integer)

Пример:

var _dt = delta_time / 1000000;
speed = spd * _dt;

Приведенный выше код установит скорость экземпляра, используя дельта-время для коррекции запаздывания. Переменная "_dt" будет использоваться для хранения ранее рассчитанного значения дельта-времени, чтобы это отношение можно было использовать для умножения базового значения скорости (хранящегося в переменной "spd") и таким образом получить согласованную скорость для экземпляра.
