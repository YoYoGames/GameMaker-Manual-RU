Математика и цифры

GameMaker содержит большое количество функций для работы с различными областями математики, числами всех типов, датой и временем. Большинство из этих функций можно найти в разделах, перечисленных ниже:

  • Дата и время
  • Функции чисел
  • Углы и расстояние
  • Матричные функции

Есть несколько встроенных констант, которые также могут быть оценены как вещественные числа (хотя всегда следует проверять константу, а не значение):

Постоянная Значение
true 1 (хотя любое значение, равное или большее 1, будет оцениваться как true)
false 0 (хотя любое значение меньше 1 также будет оцениваться как false)
pi 3.141592653589793280etc... (точное значение будет зависеть от различных факторов, таких как целевая платформа)
