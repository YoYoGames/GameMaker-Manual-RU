строковые цифры

Вы можете использовать эту функцию для разбора заданной строки и получения из нее любых чисел. Например, допустим, у вас есть такой текст - "Мне 81 год". С помощью этой функции вы получите в ответ строку "81".

Синтаксис:

string_digits(string);

АргументТип Описание
string The string to get the digits from.

Возвращается:

Пример:

var t_str = string_digits(input_str);
age = real(t_str);

Приведенный выше код принимает входную строку, очищает ее от всех символов, кроме цифр, а затем устанавливает переменную age для хранения реального числового значения возвращаемой строки (так, если входная строка была, например, "Мне 18", функция функция вернет "18").
