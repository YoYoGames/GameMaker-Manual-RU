chr

Эта функция возвращает строку, содержащую символ, который относится к входному коду Unicode для отображения. Этот символ зависит от кодовой страницы набора символов текущего чертежного шрифта, и если шрифт не задан, то используется кодовая страница по умолчанию для машины.

Синтаксис:

chr(val);

АргументТип Описание
val The Unicode code value to get the string from

Возвращается:

Пример:

mystring = chr(53) + chr(48);

Это установит mystring на "50" (строка, а не целое число), поскольку chr(53) - это "5", а chr(48) - "0".
