string_replace_all

Вы можете использовать эту функцию для разбора строки в поисках определенных частей, которые затем могут быть заменены новой строкой, указанной вами, во всех местах, где они встречаются.

Синтаксис:

string_replace_all(str, substr, newstr);

АргументТип Описание
str The string to be copied.
substr The substring within the string to be replaced in all instances.
newstr The new substring to replace the previous one(s).

Возвращается:

Пример:

str1 = "Hexxo Worxd";
str2 = string_replace_all(str1, "x", "l");

Это приведет к тому, что str2 станет str1, но в нем экземпляры "x" будут заменены на "l", в результате чего str2 станет "Hello World".
