длина_байта_строки

Эта функция возвращает количество байтов в строке. Из-за того, что они хранятся в формате UTF8, это число не будет равно числу символов в строке.

Синтаксис:

string_byte_length(string);

Аргумент Тип Описание
string Строка The string to measure the number of bytes of.

Возвращается:

Real

Пример:

bytesize = string_byte_length("Hello World");

Это установит bytesize на количество байт в "Hello World".
