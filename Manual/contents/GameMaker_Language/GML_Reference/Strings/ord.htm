орден

Эта функция принимает входную строку с одним символом и возвращает значение Unicode (UTF8) для этого символа. Обратите внимание, что при использовании функций keyboard_check* входная строка может быть длиной только в один символ и может быть только числом от 0 до 9 или заглавным римским символом от A до Z.

Синтаксис:

ord(string);

Аргумент Тип Описание
string Строка The string with which to find the Unicode value.

Возвращается:

Real

Пример:

if keyboard_check(ord("W"))
{
    y -= 4;
}

Это переместит вызывающий экземпляр на четыре пикселя вверх, если удерживать клавишу W.
