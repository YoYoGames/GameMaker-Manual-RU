string_width_ext

Эта функция возвращает максимальную ширину (в пикселях) вводимой строки с учетом разделения строк и ширины перевода строки (которая определяется как количество пикселей, которые строка может занять до того, как будет вставлен перевод строки). Это очень удобно для вычисления расстояний между элементами текста на основе максимальной ширины строки, разделенной на несколько строк, как это было бы нарисовано при draw_text_ext() с использованием определенного в данный момент шрифта. Разделение и ширина могут быть установлены в -1 для получения расстояния по умолчанию.

Синтаксис:

string_width_ext(string, sep, w);

АргументТип Описание
string The string to measure the width of.
sep The distance in pixels between lines of text as if the string was being drawn.
w The maximum width (in pixels) of the string before a line break as if the string was bring drawn.

Возвращается:

Пример:

var ww = string_width_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32 + ww, 32, str_Story_Text[2], -1, 100);

Приведенный выше код получит ширину заданной строки с учетом разделения строк и ширины перевода строки, а затем нарисует две строки текста, используя возвращенную общую ширину строки в качестве разделителя.
