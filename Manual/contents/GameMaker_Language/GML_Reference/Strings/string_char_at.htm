string_char_at

Вы можете использовать эту функцию для возврата определенного символа в определенной позиции в строке, при этом индекс для первого символа начинается с 1. Если символ не найден или строка короче заданного значения индекса, то возвращается пустая строка "", однако если заданный индекс равен или меньше 0, то возвращается первый символ строки.

Синтаксис:

string_char_at(str, index);

АргументТип Описание
str The string to check.
index The position to get the character from.

Возвращается:

Пример:

str1 = "Hello World";
str2 = string_char_at(str1, 7);

Это установит str2 на седьмой символ ("H" считается первым) в строке str1, в данном случае "W".
