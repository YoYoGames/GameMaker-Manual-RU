ширина строки

Эта функция возвращает ширину (в пикселях) введенной строки с учетом возможных переносов строк. Это очень удобно для вычисления расстояний между элементами текста на основе общей ширины букв, составляющих строку как она была бы нарисована с draw_text() с использованием определенного в данный момент шрифта.

Синтаксис:

string_width(string);

АргументТип Описание
string The string to measure the width of.

Возвращается:

Пример:

var ww = string_width(str_Name + " ");
draw_text(32, 32, str_Name));
draw_text(32 + ww, 32, "has won the game!");

Приведенный выше код получит ширину заданной строки, а затем нарисует две строки текста, используя возвращенную ширину строки в качестве разделителя.
