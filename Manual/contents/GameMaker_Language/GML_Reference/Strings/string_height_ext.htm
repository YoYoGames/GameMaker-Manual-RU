string_height_ext

Эта функция возвращает высоту (в пикселях) введенной строки с учетом разделения строк и ширины перевода строки (которая определяется как количество пикселей, которые строка может занять до того, как будет вставлен перевод строки). Это очень удобно для расчета расстояний между элементами текста на основе самой высокой из букв, составляющих строку, а также фактической длины самой строки при распределении на несколько строк, как это было бы нарисовано с помощью draw_text_ext() использованием определенного в данный момент шрифта. Разделение и ширина могут быть установлены в -1 для получения расстояния по умолчанию.

Синтаксис:

string_height_ext(string, sep, w);

АргументТип Описание
string The string to measure the height of.
sep The distance in pixels between lines of text as if the string was being drawn.
w The maximum width (in pixels) of the string before a line break as if the string was bring drawn.

Возвращается:

Пример:

var hh = string_height_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32, 32 + hh, str_Story_Text[2], -1, 100);

Приведенный выше код получит высоту заданной строки с учетом разделения строк и ширины перевода строки, а затем нарисует две строки текста, используя возвращенную общую высоту строки в качестве разделителя.
