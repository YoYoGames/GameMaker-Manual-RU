string_pos

Эта функция возвращает позицию символа экземпляра подстроки в строке при поиске с начала строки (для поиска с конца используйте функцию string_last_pos()). Функция вернет 0, если подстрока не найдена, или позицию первого символа подстроки, если она найдена. Имейте в виду, что для старой поддержки строки индексируются с 1, поэтому 1 - это первая позиция в строке, а не 0, как вы могли бы ожидать. Это можно использовать для фильтрации слов, которые могут считаться оскорбительными, или для поиска правильного места для вставки текста в другую строку.

Синтаксис:

string_pos(substr, str);

АргументТип Описание
substr The substring to look for in the string.
str The string.

Возвращается:

Пример:

if (string_pos(",", text) != 0)
{
    string_insert(name, text, string_pos(",", text));
}

Приведенный выше код ищет в строке, хранящейся в переменной "text", запятую, и если находит ее, то вставляет в эту позицию подстроку "name".
