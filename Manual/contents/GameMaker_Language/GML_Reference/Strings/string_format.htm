строковый_формат

Превращает вещественное число в строку, используя ваше собственное форматирование, где вы можете выбрать, сколько "мест" сохраняется в строке и сколько десятичных знаков также сохраняется. Оба варианта могут быть очень удобными, некоторые игры предпочитают отображать счет в виде заданного количества цифр, а контроль над десятичными знаками может быть полезен для высокой точности, которую не могут обеспечить два знака после запятой в программе string() не может обеспечить. Если количество указанных знаков больше, чем отображаемое значение, и/или число плюс указанные десятичные знаки меньше общего количества знаков, то перед значением будут добавлены пробелы, чтобы компенсировать разницу (см. пример ниже). Нули будут добавлены справа от десятичной точки, если указанное значение меньше общего числа и количества десятичных знаков, которые необходимо включить. Формат по умолчанию - без дополнительных пробелов слева и только два десятичных знака справа, например, "265,73".

Синтаксис:

string_format(val, tot, dec);

Аргумент Тип Описание
val Реально The real number to be turned into a string.
tot Реально The total number of places of the main number to be shown. Zeroes or spaces will be inserted to match this.
dec Реально The number of decimal places to be included.

Возвращается:

String

Пример:

str1 = string_format(1234, 8, 0);
str2 = string_format(pi, 1, 10);
str3 = string_format(pi, 5, 5);

В результате str1 примет значение " 1234", str2 - "3.1415926535", а str3 - "3.14159".
