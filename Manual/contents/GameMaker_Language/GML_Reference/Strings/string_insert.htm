string_insert

С помощью этой функции вы можете создать новую строку из двух строк, одна из которых вставлена в другую в заданном месте. Это может быть полезно, например, для добавления имени пользователя в заранее определенный текст, чтобы игрок вашей игры чувствовал себя более вовлеченным в действие. При вычислении позиции для вставки следует помнить, что строки индексируются с 1, поэтому первым символом в строке является 1, а не 0, как можно предположить.

Синтаксис:

string_insert(substr, str, index);

АргументТип Описание
substr The substring to be inserted.
str The string to be copied.
index The position in characters of the string to insert the substring.

Возвращается:

Пример:

str2 = string_insert(username, "Hello, , how are you?", 8);

Это вставит строку в переменной "username" в заданную фразу, и результирующая строка будет выглядеть так: "Hello, NAME, how are you?".
