буфер обмена_получить_текст

Эта функция возвращает строку текста, содержащегося в буфере обмена. если текст не сохранен, то возвращается пустая строка "".

Синтаксис:

clipboard_get_text();

Возвращается:

String

Пример:

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

Приведенный выше код проверяет буфер обмена на наличие текста и, если он есть, считывает его как строку в переменную "str". Наконец, буфер обмена очищается путем установки в него пустой строки.
