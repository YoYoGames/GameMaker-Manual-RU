длина_строки

Эта функция возвращает количество символов в заданной строке. Она может быть полезна для таких вещей, как определение того, когда следует ограничить длину символов пользовательского текста (например, ограничить имя игрока до 10 символов). Помните, что этот параметр отличается от string_width() тем, что измеряет количество символов в строке, а не ее ширину, отображаемую на экране в пикселях.

Синтаксис:

string_length(string);

Аргумент Тип Описание
string Строка The string to measure the number of characters of.

Возвращается:

Real

Пример:

if (string_length(name) > 10)
{
    name = string_copy(name, 1, 10);
}

При этом будет проверено, превышает ли строка имени десять символов, и если да, то будут скопированы и использованы только первые десять символов.
