копирование_строки

С помощью этой функции вы можете легко выбрать несколько символов из строки для копирования в другую строку. Первый символ в строке всегда индексируется как 1, а не 0, как можно было бы ожидать, поэтому для копирования (например) первых пяти символов строки вам потребуется string_copy(str, 1, 5). Еще один пример приведен ниже.

Синтаксис:

string_copy(str, index, count);

АргументТип Описание
str The string to copy from.
index The position of the first character in the string to copy from (numbered from 1)
count The number of characters, starting from the position of the first, to copy

Возвращается:

Пример:

name = keyboard_string;
if (string_length(name) > 15)
{
    keyboard_string = string_copy(name, 1, 15);
}

Приведенный выше код позволяет игроку ввести строку с клавиатуры, которая затем сохраняется в переменной name. Затем эта переменная проверяется, не превышает ли она максимальную длину в 15 символов, и если да, то строка_клавиатуры заменяется 15-символьной копией строки name. Это эффективно ограничивает ввод игрока 15 символами.
