строковые_буквы

С помощью этой функции вы можете удалить все символы, которые не относятся к категории букв. Это удобно для предотвращения ввода нежелательных символов в текст (например, ввод "#" для перехода на новую строку).

ПРИМЕЧАНИЕ Эта функция распознает и возвращает только 26 букв английского алфавита от A до Z (верхний или нижний регистр).

Синтаксис:

string_letters(string);

АргументТип Описание
string The string to convert to letters.

Возвращается:

Пример:

username = string_letters(username);

Приведенный выше код установит переменную "username" только для буквенных символов исходной строки (если исходная строка была "MacSw33ny", то возвращаемая строка будет "MacSwny").
