string_lower

С помощью этой функции вы можете заставить строку содержать только символы нижнего регистра.

ПРИМЕЧАНИЕ Эта функция определяет только 26 букв английского алфавита от A - Z.

Синтаксис:

string_lower(string);

АргументТип Описание
string The string to convert to lowercase.

Возвращается:

Пример:

str1 = "Hello World";
str2 = string_lower(str1);

Приведенный выше код установит str2 на "hello world".
