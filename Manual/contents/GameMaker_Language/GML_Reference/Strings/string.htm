строка

С помощью этой функции вы можете превратить любое вещественное число в строку. Если вещественное число является целым, то оно будет сохранено без десятичных знаков, в противном случае оно будет сохранено с двумя десятичными знаками. Если вам требуется больше десятичных знаков, то используйте функцию string_format(). Также обратите внимание, что использование этой функции для переменной, хранящей массив, структуру данных или struct, преобразует содержимое этих переменных в строку, которую можно вывести на консоль или сохранить в файл для отладки.

Синтаксис:

string(val);

Аргумент Тип Описание
val Реально The real number to be turned into a string.

Возвращается:

String

Пример:

draw_text(100, 100, "Score: " + string(score) + " / Health: " + string(health));

Приведенный выше код использует функцию string для рисования как вещественных чисел, так и строк вместе (draw принимает только строку или вещественное число, но не оба).
