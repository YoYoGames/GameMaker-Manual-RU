string_repeat

Эта функция просто возвращает одну и ту же строку, повторенную заданное количество раз над самой собой.

Синтаксис:

string_repeat(str, count);

АргументТип Описание
str The string to repeat.
count The number of times to repeat the string.

Возвращается:

Пример:

str1 = "Hello World";
str2 = string_repeat(str1, 3);

Приведенный выше код установит str2 на "Hello WorldHello WorldHello World".
