string_ord_at

Вы можете использовать эту функцию для возврата кода определенного символа в определенной позиции в строке, при этом индекс начинается с 1 для первого символа. Если символ не найден или строка короче, чем значение, заданное индексом, возвращается -1.

Синтаксис:

string_ord_at(str, index);

АргументТип Описание
str The string to check.
index The position to get the character code from.

Возвращается:

Пример:

str = "Hello World";
char_code = string_ord_at(str, 7);

Это позволит получить код символа для седьмого символа (где "H" считается первым) в строке "str" и сохранить его в переменной "char_code".
