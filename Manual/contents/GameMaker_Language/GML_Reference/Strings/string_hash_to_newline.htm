string_hash_to_newline

Эта функция возвращает строку, в которой символ "#" был преобразован в новую строку. Таким образом, строка, отформатированная как:

"Hello#World"

Будет выведено на экран в виде:

Hello
World

ВАЖНО! Эта функция предусмотрена для совместимости между GameMaker и предыдущими версиями программы, поэтому ее не следует использовать, за исключением случаев, когда необходимо воспроизвести устаревшие функции. Вместо этого для обозначения новой строки следует использовать управляющие символы. Более подробную информацию о совместимости импорта см. здесь.

Синтаксис:

string_hash_to_newline(string);

АргументТип Описание
string The string to convert over multiple lines

Возвращается:

Пример:

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

Приведенный выше код преобразует строку с символом хэша в строку, разделенную на две строки, и выводит ее на экран.
