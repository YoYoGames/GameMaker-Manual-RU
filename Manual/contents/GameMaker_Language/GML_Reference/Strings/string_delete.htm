string_delete

Эта функция используется для удаления части заданной строки. Вы передаете входную строку, начальную позицию в строке, из которой нужно удалить символы (индекс начинается с 1), и количество символов, которые нужно удалить. Функция вернет новую строку без этой части.

Синтаксис:

string_delete(str, index, count);

АргументТип Описание
str The string to copy and delete from.
index The position of the first character to remove (from 1).
count The number of characters to remove.

Возвращается:

String

Пример:

str1 = "Helloo World";
str2 = string_delete(str1, 5, 1);

Это установит str2 в "Hello World", так как удалит лишнюю букву "o" из "Hello" (которая присутствует в позиции 5).
