высота_строки

Эта функция возвращает высоту (в пикселях) введенной строки с учетом разделения строк и любых разрывов текста. Это очень удобно для расчета расстояния между элементами текста на основе самой высокой из букв. составляющих строку, как она была бы нарисована с помощью draw_text() используя текущий определенный шрифт.

Синтаксис:

string_height(string);

АргументТип Описание
string The string to measure the height of.

Возвращается:

Пример:

var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

Приведенный выше код получит высоту заданной строки, а затем нарисует две строки текста, используя возвращенную высоту строки в качестве разделителя.
