строка_замены

Вы можете использовать эту функцию для разбора строки в поисках определенной части, которая затем может быть заменена новой строкой, которую вы указали.

Синтаксис:

string_replace(str, substr, newstr);

АргументТип Описание
str The string to be copied.
substr The substring within the string to be replaced.
newstr The new substring to replace the previous one.

Возвращается:

Пример:

str1 = "Hello Earth";
str2 = string_replace(str1, "Earth", "World");

Это установит str2 в str1, но с его экземпляром "Earth", замененным на "World", в результате чего str2 будет "Hello World".
