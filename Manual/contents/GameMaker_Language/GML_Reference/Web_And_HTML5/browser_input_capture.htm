browser_input_capture

С помощью этой функции вы можете установить, должно ли окно браузера захватывать весь ввод (установите ее в false) или игра должна захватывать ввод (установите ее в true). Обратите внимание, что эта функция предназначена только для использования с модулем HTML5.

Синтаксис:

browser_input_capture(enable);

Аргумент Тип Описание
enable Булево Enable or disable game input capture

Возвращается:

N/A

Пример:

browser_input_capture(true);

Приведенный выше код устанавливает игру на захват всего ввода браузера на целевой платформе HTML5.
