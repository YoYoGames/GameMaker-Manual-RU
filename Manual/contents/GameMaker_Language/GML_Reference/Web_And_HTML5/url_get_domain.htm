url_get_domain

Вы можете использовать его для получения домена, на котором воспроизводится ваша HTML5-игра.

Синтаксис:

url_get_domain();

Возвращается:

Пример:

dom_name=url_get_domain();

Это сохранит домен вашей игры в переменной "dom_name".
