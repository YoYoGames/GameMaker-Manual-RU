кликабельное_удаление

Эта функция должна быть использована для удаления кликабельного значка, ранее созданного с помощью функции clickable_add() из игрового окна.

Синтаксис:

clickable_delete(index);

Аргумент Тип Описание
index Кликабельный идентификатор Index of the clickable icon to remove.

Возвращается:

N/A

Пример:

if clickable_exists(global.Help_Icon)
{
    clickable_delete(global.Help_Icon);
}

Приведенный выше код удаляет из окна игры кликабельную иконку, индексированную в глобальной переменной "Help_Icon", если она существует.
