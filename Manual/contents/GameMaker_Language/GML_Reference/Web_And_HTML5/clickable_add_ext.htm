clickable_add_ext

С помощью этой функции вы можете создать пользовательскую, кликабельную иконку, которая действует как "кнопка" вне игровой канвы, но в самом окне браузера. Функция возвращает значение ID, которое должно быть сохранено в переменной для использования во всех последующих функциях кнопки, которые ссылаются на новую кликабельную иконку. Эта функция очень полезна для предотвращения создания браузером всплывающего окна при нажатии на кнопку, поскольку все это делается в самом GameMaker.

Эта функция работает точно так же, как и clickable_add() где необходимо задать позицию относительно (0,0) окна браузера, указать страницу текстуры для sprite, которую вы хотите видеть на значке кнопки (см: sprite_get_tpe()), указать URL для открытия или скрипт (из браузера активов GameMaker) для открытия или запуска, затем задать цель и параметры для использования. Однако эта функция также имеет две дополнительные опции, которые позволяют установить альфа-фактор значка от 0 (полностью прозрачный) до 1 (полностью непрозрачный) и масштаб значка, который представляет собой множитель, применяемый к используемому значению sprite. Таким образом, значение 1 - это масштаб по умолчанию, в то время как (например) 0,5 - это половинный масштаб, а 2 - двойной.

Также стоит отметить, что если передаваемый вами аргумент URL является именем функции скрипта (т.е. не начинается с "http://") И скрипт называется gmcallback_*, то вы можете получить прямую передачу в html5-событие. Имена функций скриптов GameMaker не обфусцируются, если они начинаются с gmcallback_.

Синтаксис:

clickable_add_ext(x, y, tpe, url, target, params, alpha, scale);

Аргумент Тип Описание
x Реально The x position within the window.
y Реально The y position within the window.
tpe Ввод страницы текстуры The texture page entry for the sprite to be used.
url Строка The URL (website address) to link to, or the script to be run.
target Строка This is the target area to open the URL in (see description).
params Строка The various parameters for the icon (see description).
alpha Реально The image alpha of the icon (default 1).
scale Реально The scale of the icon (default 1).

Возвращается:

Clickable ID

Пример:

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2);

Приведенный выше код создает кликабельную DOM-иконку в позиции (32, 32) страницы, на которой запущен игровой холст. Иконка использует sprite, указанный на странице текстуры как "spr_MS_Home", и при нажатии на иконку откроется новое окно для указанного URL и с определенными свойствами для этого окна. Кнопка также будет масштабирована до удвоенного размера исходного sprite и альфа изображения равна 1.
