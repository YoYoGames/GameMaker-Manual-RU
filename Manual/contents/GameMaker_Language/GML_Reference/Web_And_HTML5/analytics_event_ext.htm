analytics_event_ext

Эта функция отправит указанный текст поставщику аналитики, которого вы настроили через опции HTML5 Game Options. Эта функция может быть использована для создания пользовательской аналитики для отслеживания чего-либо а также принимать пользовательские пары параметров/значений, где параметр - строка, а значение - действительное число. Для Google Analytics вы можете добавить только одну дополнительную пару, в то время как Flurry принимает до 7.

Синтаксис:

analytics_event_ext(string, string_param, value);

АргументТип Описание
string A string to send to the provider.
string_param[0 -9] The parameter to send (a string).
value[0 - 9] The value of the parameter (a real number).

Возвращается:

Пример:

var num, time;
ini_open("Options.ini");
num = ini_read_real("Data", "Plays", 0);
num += 1;
time = current_time;
analytics_event_ext(GAME_NAME, "Plays", num, "Time", time);
ini_write_real("Data", "Plays", num);
ini_close();

Приведенный выше код получит информацию об игре из ini-файла, а также текущее время и затем отправит эти данные поставщику аналитики.
