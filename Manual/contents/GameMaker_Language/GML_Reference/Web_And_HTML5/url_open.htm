url_open

Это откроет указанный URL в браузере выбранного целевого устройства или, если вы используете модуль HTML5, в текущем открытом браузере.

Синтаксис:

url_open(url);

Аргумент Тип Описание
url Строка The URL (website address) to link to.

Возвращается:

N/A

Пример:

url_open("http://gamemaker.io");

Это откроет домашнюю страницу GameMaker в текущем окне.
