webgl_enabled

Эта переменная, доступная только для чтения, возвращает, включен ли WebGL (true) или нет (false) для вашей игры. Она будет работать только для игр, запускаемых через браузер (т.е. HTML5), а для всех остальных платформ она будет возвращать true.

Синтаксис:

webgl_enabled

Возвращается:

Пример:

if webgl_enabled
{
    global.quality = 1;
}
else global.quality = 0;

Приведенный выше код проверяет флаг WebGL и затем устанавливает глобальную переменную "quality" соответствующим образом.
