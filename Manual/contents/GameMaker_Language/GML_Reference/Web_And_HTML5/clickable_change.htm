кликабельное_изменение

С помощью этой функции вы можете изменить sprite и положение кликабельного значка, созданного ранее с помощью функции clickable_add(). Обратите внимание, что позиция основана на позиции окна, а не холста (0,0), и что ссылка на sprite должна быть сделана непосредственно со страницы текстуры (см: sprite_get_tpe()).

Синтаксис:

clickable_change(index, tpe, x, y)

Аргумент Тип Описание
index Кликабельный идентификатор The index of the clickable icon to change.
tpe Ввод страницы текстуры The texture page entry for the sprite to be used.
x Реально The new x position within the window.
y Реально The new y position within the window.

Возвращается:

N/A

Пример:

switch (room)
{
    case rm_Menu: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32); break;
    case rm_Game: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32); break;
}

Приведенный выше код будет изменять индекс изображения и положение кликабельной иконки, индексируемой в переменной "global.Help", в зависимости от текущей комнаты.
