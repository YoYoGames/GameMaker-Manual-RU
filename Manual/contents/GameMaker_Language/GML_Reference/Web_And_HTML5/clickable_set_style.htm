clickable_set_style

Эта функция позволяет установить свойства стиля CSS для заданной кнопки с помощью пар ключ/значение в предоставленном файле DS Map. Вам необходимо предварительно создать элемент кнопки (с помощью clickable_add()) и DS Map и предоставить сохраненные индексы для каждого из них в качестве аргументов.

Синтаксис:

clickable_set_style(index, map)

Аргумент Тип Описание
index Кликабельный идентификатор The index of the clickable icon to style.
map Идентификатор карты DS The index of the DS Map для установки стиля.

Возвращается:

N/A

Пример:

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy();

Приведенный выше код изменит стиль нажимаемой кнопки с индексом, хранящимся в переменной "button".
