clickable_change_ext

С помощью этой функции вы можете изменить sprite и положение кликабельного значка, созданного ранее с помощью функции clickable_add(). Следует помнить, что позиция основывается на позиции окна, а не холста (0,0), и что ссылка на sprite должна быть сделана непосредственно со страницы текстуры (см: sprite_get_tpe()). Эта функция также позволяет изменить масштаб используемого sprite (в виде множителя, так что 1 - это значение по умолчанию, 0,5 - половина, а 2 - удвоение) и значение альфа от 0 (полностью прозрачный) до 1 (полностью непрозрачный) для конечного значка на экране.

Синтаксис:

clickable_change_ext(index, tpe, x, y, alpha, scale)

Аргумент Тип Описание
index Кликабельный идентификатор The index of the clickable icon to change.
tpe Ввод страницы текстуры The texture page entry for the sprite to be used
x Реально The new x position within the window.
y Реально The new y position within the window.
scale Реально The scale of the icon (default 1).
alpha Реально The image alpha of the icon (default 1).

Возвращается:

N/A

Пример:

switch (room)
{
    case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;
    case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;
}

Приведенный выше код изменит индекс изображения и положение кликабельной иконки, индексируемой в переменной "global.Help", в зависимости от текущей комнаты, изменяя при этом масштаб и альфа иконки.
