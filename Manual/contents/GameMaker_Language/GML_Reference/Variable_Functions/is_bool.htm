is_bool

Эта функция возвращает, является ли данная переменная булевой (true или false) или нет. В некоторых случаях вы хотите проверить, имеет ли переменная в GameMaker булево значение, и тогда вы используете эту функцию.

Синтаксис:

is_bool(n);

Аргумент Тип Описание
n Переменная The argument to check.

Возвращается:

Boolean

Пример:

if is_bool(val)
{
    global.Sound = val;
}
else
{
    global.Sound = true;
}

Приведенный выше код проверяет переменную "val" на то, является ли она действительным числом, и если да, то использует ее для установки глобальной переменной.
