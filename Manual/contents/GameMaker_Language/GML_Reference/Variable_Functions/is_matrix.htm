матрица

Эта функция возвращает, относится ли данная переменная к матрице или нет. В некоторых случаях вы хотите проверить и узнать, к какому типу данных относится переменная в GameMaker, и тогда вы используете эту функцию. Она возвращает true или false в зависимости от того, является ли значение матрицей или нет.

Синтаксис:

is_matrix(n);

Аргумент Тип Описание
n Переменная The argument to check.

Возвращается:

Boolean

Пример:

if !is_matrix(val)
{
    show_debug_message("Not a valid matrix!");
}

Приведенный выше код проверяет переменную "val" на наличие в ней матрицы, и если это не так, то выводит сообщение в консоль отладки.
