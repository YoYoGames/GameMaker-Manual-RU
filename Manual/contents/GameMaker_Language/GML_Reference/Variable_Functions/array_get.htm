array_get

С помощью этой функции вы можете получить значение по индексу в массиве. Функция требует, чтобы вы указали переменную, содержащую массив, и индекс для получения значения из этого массива. Обратите внимание, что если указанный индекс массива выходит за границы, то игра завершится с ошибкой. Эта функция также может быть использована для многомерных массивов, если вы укажете, какое измерение вы хотите получить при указании индекса массива, по следующей схеме:

// 1D array
array_get(my_array, 0);
// 2D array
array_get(my_array[0], 0);
// 3D array
array_get(my_array[0][0], 0);
// etc...

Синтаксис:

array_get(variable, index);

Аргумент Тип Описание
variable Массив The variable that holds the array.
index Реально The index of the array element to get the value from.

Возвращается:

Variable (Any valid data type that an array can hold)

Пример:

for (var i = 0; i < 10; ++i;)
{
    show_debug_message(array_get(my_array, i));
}

Приведенный выше код выведет на консоль первые 10 элементов из заданных.
