variable_struct_set

С помощью этой функции вы можете установить значение заданной переменной в структуре. Вы передаете ссылку на struct, а также имя переменной, значение которой нужно установить, в виде строки (см. пример кода ниже), и, наконец, значение, которое нужно установить (это может быть любой допустимый тип данных). Если переменная еще не существует в структуре, она будет создана, а затем ей будет присвоено значение.

Синтаксис:

variable_struct_set(struct, name, val);

Пример:

{
    variable_struct_set(mystruct, "shields", 0);

Приведенный выше код проверит, существует ли заданная переменная в заданной структуре, и если нет, то она будет создана и установлена в 0.
