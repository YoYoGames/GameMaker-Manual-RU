is_method

Эта функция может быть использована для проверки и определения, является ли переменная переменной метода (в этом случае она вернет true) или нет (в этом случае она вернет false).

Синтаксис:

is_method(n);

Аргумент Тип Описание
n Переменная The variable to check.

Возвращается:

Boolean

Пример:

if is_method(get_vec)
{
    show_debug_message("Method variable!");
}

Приведенный выше код проверяет переменную на предмет того, является ли она методом, и если функция возвращает true, то на консоль выводится отладочное сообщение.
