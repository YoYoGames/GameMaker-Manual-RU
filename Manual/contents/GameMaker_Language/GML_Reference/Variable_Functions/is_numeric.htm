is_numeric

Эта функция возвращает, является ли данная переменная числовым значением (real, int32, int64 или boolean) или нет. В некоторых случаях вы хотите проверить, имеет ли переменная в GameMaker числовое значение, и тогда вы используете эту функцию. Функция вернет true, если заданное значение является числовым, и false в противном случае.

Синтаксис:

is_numeric(n);

Аргумент Тип Описание
n Переменная The input to check.

Возвращается:

Boolean

Пример:

if is_numeric(val)
{
    current_val += val;
}

Приведенный выше код проверяет переменную "val" на то, является ли она числовым значением, и если да, то добавляет его к заданной переменной.
