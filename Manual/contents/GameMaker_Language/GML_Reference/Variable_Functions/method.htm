метод

С помощью этой функции можно привязать существующую функцию (или метод) к данному экземпляру или структуре, создав новую переменную метода, которую можно использовать позже.

Вы указываете ID экземпляра, который нужно использовать (экземпляр, который активен в комнате, а не индекс объекта) или ссылку на структуру, а также ID функции (или ссылку на метод), которую вы хотите связать. Функция вернет новый метод, который может быть вызван из переменной, которой она присвоена (см. пример кода ниже).

Возвращаемый метод будет "привязан" к данному экземпляру или структуре, то есть он всегда будет выполняться в области видимости этого экземпляра/структуры.

Вы можете связывать как встроенные функции, так и определенные пользователем функции/методы, а также указывать undefined в качестве аргумента экземпляра/структуры, что означает, что текущая область видимости будет использована для привязки. self область видимости будет использоваться для связывания.

Синтаксис:

method(struct_ref_or_instance_id, func);

Аргумент Тип Описание
struct_ref_or_instance_id Идентификаторструктуры или экземпляра The unique reference or ID value of the struct or instance to use (can be self или undefined)
func Функция или методсценария The ID of the function (or method reference) to use

Возвращается:

Method

Пример:

var _inst = instance_position(mouse_x, mouse_y, obj_Enemy);
if (instance_exists(_inst))
{
    enemy_func = method(_inst, enemy_ai);
}

Приведенный выше код проверяет, существует ли экземпляр врага в позиции мыши. Если да, то метод enemy_ai привязывается к экземпляру врага и возвращается как новая переменная метода "enemy_func".
