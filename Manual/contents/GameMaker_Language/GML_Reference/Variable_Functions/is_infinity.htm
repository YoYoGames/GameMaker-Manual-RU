бесконечность

Эта функция возвращает, является ли данная переменная infinity (бесконечным числом) или нет, возвращая true, если является, и false, если не является.

Синтаксис:

is_infinity(n);

Аргумент Тип Описание
n Переменная The argument to check.

Возвращается:

Boolean

Пример:

if is_infinity(global.value)
{
    show_debug_message("Value is infinite!");
}

Приведенный выше код проверяет глобальную переменную "value" на то, бесконечна она или нет, и показывает отладочное сообщение, если это так.
