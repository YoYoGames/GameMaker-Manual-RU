int64

Эта функция пытается преобразовать заданное значение в знаковое 64-битное целое число, при этом значение должно быть либо вещественным, либо строкой, либо int64, либо int32, либо ptr. Любой другой вариант приведет к аварийному завершению игры с сообщением об ошибке. Проверить, хранит ли переменная значение int64, можно с помощью функции is_int64().

Синтаксис:

int64(val);

Аргумент Тип Описание
val Реальный, Строка, или Указатель The value to convert.

Возвращается:

int64 (signed 64-bit integer)

Пример:

steam_handle = int64(global.fileReadString);

Приведенный выше код преобразует значение, хранящееся в глобальной переменной, в 64-битное целое число.
