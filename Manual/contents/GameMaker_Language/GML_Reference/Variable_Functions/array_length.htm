длина_массива

С помощью этой функции вы можете получить длину (количество записей) размерности массива. Вы указываете значение индекса массива, и функция возвращает целочисленное значение, представляющее количество записей, которые содержит массив. Эту функцию можно использовать и для многомерных массивов, если при указании индекса массива указать, длину какого измерения вы хотите получить, следуя следующей схеме:

// Returns the first dimension of the array
val = array_length(my_array);

// Returns the second dimension of the array
val = array_length(my_array[0]);

// Returns the third dimension of the array
val = array_length(my_array[0][0]);

// etc...

Синтаксис:

array_length(array_index);

АргументТип Описание
array_index The index of the array to check.

Возвращается:

Пример:

for (var i = 0; i < array_length(a); ++i)
{
    a[i] = -1;
}

Приведенный выше код будет циклически просматривать массив и устанавливать для каждой записи значение -1.
