массив_создать

С помощью этой функции вы можете создать одномерный массив заданного размера. Вы сообщаете функции длину создаваемого массива, и она возвращает "хэндл" для массива, который вы можете присвоить переменной. В массивах, созданных таким образом, каждая запись будет инициализирована 0, если вы не укажете (необязательное) инициализирующее значение. Если вы укажете дополнительное значение для инициализации массива, то все индексы в новом массиве будут установлены в это значение вместо 0, но учтите, что в этом случае функция будет иметь более высокую производительность.

Синтаксис:

array_create(size, [value]);

Аргумент Тип Описание
size Реально The size of the array to create.
value Переменная OPTIONAL Значение, используемое для инициализации всех индексов массива.

Возвращается:

Array

Пример:

instance_array = array_create(100, noone);

Приведенный выше код создаст новый массив из 100 записей, каждая из которых будет установлена на ключевое слово noone, а затем присвоит его переменной "instance_array".
