variable_instance_get_names

С помощью этой функции вы можете получить массив, заполненный именами переменных экземпляра или глобальных переменных игры. Когда вы передаете значение ID экземпляра, каждая запись в массиве будет строкой имени переменной, соответствующей переменной области видимости экземпляра, которая была создана в экземпляре. Однако если вы передадите ключевое слово global, каждая запись в массиве будет строкой имени переменной, соответствующей переменной глобальной области видимости.

Синтаксис:

variable_instance_get_names(instance_id/global);

Аргумент Тип Описание
instance_id/global Идентификатор экземпляра или global The unique ID value of the instance to check or the keyword global

Возвращается:

Array (each entry is a string)

Пример:

var str = "";
var array = variable_instance_get_names(id);
show_debug_message("Variables for " + object_get_name(object_index) + string(id));
for (var i = 0; i < array_length(array); i++;)
{
    str = array[i] + ":" + string(variable_instance_get(id, array[i]));
    show_debug_message(str);
}

Приведенный выше код получит массив всех переменных области видимости экземпляра для экземпляра, выполняющего блок кода, и затем отобразит их вместе с их значениями в отладочном выводе.
