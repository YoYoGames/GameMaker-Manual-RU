instanceof

Эта функция может быть использована для получения имени функции-конструктора, которая была использована для создания структуры. Сама структура должна быть создана с помощью функции new вместе с самим конструктором. Вы передаете переменную со ссылкой на struct для проверки, и функция возвращает либо строку с именем конструктора, либо undefined.

Обратите внимание, что если вы передадите функции литерал struct (т.е. struct, который был создан без использования функции-конструктора), то она просто вернет строку "struct".

Эта функция также может быть использована для проверки того, является ли ссылка на структуру слабая ссылка или нет, в этом случае функция вернет строку "weakref" вместо имени функции, создавшей структуру. Для получения дополнительной информации см. функцию weak_ref_create().

Возвращаемое имя конструктора будет включать дополнительный текст, если функция конструктора была создана не в сценарии. Если она была создана внутри объекта, ее имя будет включать объект и событие, в котором она была создана. Например, конструктор с именем test_constructor, созданный в событии Create объекта Object1, будет возвращен как "test_constructor_gml_Object_Object1_Create_0". Чтобы избежать этого, создавайте свои конструкторы в формате script.

Синтаксис:

instanceof(struct_id);

Аргумент Тип Описание
struct Структура The struct reference to use.

Возвращается:

String or undefined

Расширенный пример:

В этом примере мы должны сначала определить функцию, которая будет использоваться в качестве конструктора для нашей структуры. Следующая функция определена в активе сценария:

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

Затем эту функцию можно использовать вместе с оператором new для создания структуры и заполнения ее переменными, установленными в значения аргументов, используемых в функции:

mystruct = new init_struct(10, 100, "Hello World");

Затем мы можем получить имя функции-конструктора, которая была использована следующим образом:

var _name = instanceof(mystruct);
if is_string(_name)
{
    show_debug_message(_name);
}

Это выведет в выходной журнал строку "init_struct", которая является именем функции-конструктора, определенным в ее сценарии.
