variable_instance_set

С помощью этой функции вы можете установить значение заданной переменной в экземпляре. Вы предоставляете уникальное значение ID экземпляра (которое можно найти в Свойствах экземпляра в редакторе комнаты, или оно возвращается при вызове функции instance_create_layer()), имя переменной для установки значения в виде строки (см. пример кода ниже), и, наконец, значение для установки (может быть любым допустимым типом данных). Если переменная еще не существует в экземпляре, она будет создана, а затем ей будет присвоено значение.

Синтаксис:

variable_instance_set(instance_id, name, val);

Аргумент Тип Описание
instance_id Идентификатор экземпляра The unique ID value of the instance to use
name Строка The name of the variable to set (as a string)
val Переменная The value to set the variable to

Возвращается:

N/A

Пример:

if !variable_instance_exists(id, "shields")
{
    variable_instance_set(id, "shields", 0);
}

Приведенный выше код проверит, существует ли переменная экземпляра в вызывающем экземпляре, и если нет, то она будет создана и установлена в 0.
