struct_exists

С помощью этой функции вы можете проверить, существует ли переменная в заданной структуре или нет. Вы передаете ссылку на структуру, которую нужно использовать, а также имя переменной, которую нужно проверить, в виде строки (см. пример кода ниже). Функция вернет true, если переменная с заданным именем существует в структуре, и false в противном случае.

 

Синтаксис:

struct_exists(struct, name);

Аргумент Тип Описание
struct Struct The struct reference to check
name String The name of the struct variable to check for (as a string)

 

Возвращается:

Boolean

 

Пример:

if !struct_exists(mystruct, "shields")
{
    mystruct.shields = 0;
}

Приведенный выше код проверит, существует ли переменная "shields" в данной структуре, и если нет, то она будет создана и инициализирована в 0.
