is_struct

Эта функция проверяет, является ли предоставленное значение структурой. Если это так, возвращается true, в противном случае возвращается false.

Обратите внимание, что переменные методов также возвращают true, а экземпляры объектов - false.

Синтаксис:

is_struct(val);

Аргумент Тип Описание
val Переменная The value to check.

Возвращается:

Boolean

Пример:

if is_struct(a)
{
    delete(a);
}

Приведенный выше код проверяет переменную на предмет того, является ли она структурой, и если функция возвращает true, то структура удаляется.
