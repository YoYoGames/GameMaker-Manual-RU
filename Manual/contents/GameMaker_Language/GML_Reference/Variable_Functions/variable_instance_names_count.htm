variable_instance_names_count

С помощью этой функции вы можете найти общее количество переменных, определенных для экземпляра. Вы указываете ID экземпляра для проверки, и функция возвращает целочисленное значение количества переменных, или (если экземпляра с заданным ID не существует) -1.

Синтаксис:

variable_instance_names_count(instance_id);

Аргумент Тип Описание
instance_id Идентификатор экземпляра The unique ID value of the instance to check.

Возвращается:

Real

Пример:

var _num = variable_instance_names_count(myinst);
show_debug_message("Instance Variables = " + string(_num));

Приведенный выше код получит количество переменных в данном экземпляре и покажет отладочное сообщение в консольном выводе с этим значением.
