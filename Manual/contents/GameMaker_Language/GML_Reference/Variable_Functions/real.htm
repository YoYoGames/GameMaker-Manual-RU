real

Эта функция может быть использована для преобразования заданной строки в вещественное число. При использовании этой функции учитываются цифры, знаки минус, десятичные точки и экспоненциальные части в строке, в то время как другие символы (например, буквы) вызовут ошибку будет выброшена. Если вы знаете или подозреваете, что в строке могут быть другие символы, вы можете использовать функцию string_digits() для удаления всех нечисловых символов, прежде чем использовать эту функцию для преобразования полученной строки в действительное число.

 

Синтаксис:

real(string)

Аргумент Тип Описание
string String The string to be converted to a real value.

 

Возвращается:

Real

 

Пример:

var t_str = string_digits(input_str);
age = real(t_str);

Приведенный выше код примет входную строку, очистит ее от всех символов, кроме цифр, а затем установит переменную "age" для хранения реального числового значения возвращаемой строки.
