переменная_структура_get

С помощью этой функции вы можете получить значение заданной именованной переменной в структуре. Вы передаете ссылку на struct, а также имя переменной, значение которой нужно получить, в виде строки (см. пример кода ниже). Функция вернет значение переменной или undefined, если именованная переменная не существует.

ВАЖНО! Если переменная, которую вы получаете, не существует, то функция вернет ключевое слово undefined, и вы можете получить ошибки, которые остановят работу игры, поэтому, если вы сомневаетесь, используйте функцию variable_struct_exists() сначала.

Синтаксис:

variable_struct_get(struct, name);

АргументТип Описание
struct The struct reference to use
name The name of the variable to get (as a string)

Возвращается:

Variable (any data type) or undefined (if the named variable does not exist)

Пример:

if variable_struct_exists(mystruct, "shields")
{
    var ss = variable_struct_get(mystruct, "shields");
}
else
{
    var ss = -1;
}

Приведенный выше код проверит, существует ли переменная в заданной структуре, и если существует, то значение, которое она содержит, будет извлечено и сохранено в локальной переменной. Если она не существует, то локальная переменная устанавливается в -1.
