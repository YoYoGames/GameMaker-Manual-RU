переменная_структура_имена_счет

С помощью этой функции вы можете найти общее количество переменных, определенных для структуры. Вы указываете ID структуры, которую нужно проверить, и функция возвращает целочисленное значение количества переменных, или (если структуры с заданным ID не существует) -1.

Синтаксис:

variable_struct_names_count(struct_id);

АргументТип Описание
struct_id The unique ID value of the struct to check.

Возвращается:

Пример:

var _num = variable_struct_names_count(mystruct);
show_debug_message("Struct Variables = " + string(_num));

Приведенный выше код получит количество переменных в заданной структуре и покажет отладочное сообщение в консольном выводе с этим значением.
