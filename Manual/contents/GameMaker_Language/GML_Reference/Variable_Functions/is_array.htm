массив

Эта функция может быть использована для проверки и определения, содержит ли переменная массив (в этом случае она вернет true) или нет (в этом случае она вернет false).

Синтаксис:

is_array(n);

Аргумент Тип Описание
n Переменная The variable to check.

Возвращается:

Boolean

Пример:

if is_array(a)
{
    a = -1;
}

Приведенный выше код проверяет переменную на то, является ли она массивом, и если функция возвращает true, то массив удаляется путем установки переменной в -1.
