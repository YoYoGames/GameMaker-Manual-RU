массив_толчков

С помощью этой функции вы можете поместить значение (или значения) в конец массива, не зная его длины. Функция требует, чтобы вы предоставили переменную, содержащую массив, а также хотя бы одно значение для вставки, хотя по желанию вы можете предоставить дополнительные аргументы, и все они будут вставляться в массив в последовательном порядке.

Синтаксис:

array_push(variable, value, [value], [value], [etc...]);

Аргумент Тип Описание
variable Массив The variable that holds the array.
value Переменная The value to push onto the end of the array
[value], [value], [etc...] Переменная OPTIONAL Дополнительные значения, которые необходимо поместить в массив

Возвращается:

N/A

Пример:

array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr);

Приведенный выше код поместит четыре значения в конец заданного массива.
