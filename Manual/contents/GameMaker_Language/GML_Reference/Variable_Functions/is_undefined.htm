is_undefined

Эта функция возвращает, определена ли данная переменная или нет. В некоторых случаях вы хотите проверить и узнать, какой тип данных имеет переменная в GameMaker, и тогда вы используете эту функцию. Она возвращает true или false в зависимости от того, определено значение или нет.

ПРИМЕЧАНИЕ: Эта функция не может быть использована для проверки существования переменной. Используйте variable_instance_exists() или variable_global_exists() вместо этого.

Синтаксис:

is_undefined(n);

Аргумент Тип Описание
n Переменная The argument to check.

Возвращается:

Boolean

Пример:

var val = ds_map_find_value(map, 13);
if is_undefined(val)
{
    show_debug_message("Map entry does not exist!");
}

Приведенный выше код проверяет переменную "val" на то, является ли она неопределенной или нет, и показывает отладочное сообщение, если это так.
