Переменные функции

Функции на этой странице предназначены для работы с различными переменными и типами переменных при использовании GameMaker Language в ваших играх. Более подробную информацию о переменных можно найти на странице " Переменные и область действия переменных".

Доступны следующие функции переменных:

  • variable_instance_exists
  • variable_instance_get_names
  • variable_instance_names_count
  • variable_instance_get
  • variable_instance_set
  • variable_global_exists
  • variable_global_get
  • variable_global_set
  • variable_struct_exists
  • variable_struct_get
  • variable_struct_set
  • переменная_структура_удаления
  • variable_struct_get_names
  • variable_struct_names_count
  • instanceof

Существует несколько функций, связанных с использованием массивов. Они предназначены для обеспечения гибкости кода и позволяют создавать более функциональные и динамичные массивы при создании игр. К таким функциям относятся:

  • array_create
  • array_copy
  • array_equals
  • array_get
  • array_set
  • array_push
  • array_pop
  • array_insert
  • array_delete
  • array_sort
  • array_length
  • array_resize
  • array_length_1d(deprecated)
  • array_length_2d(deprecated)
  • array_height_2d(устаревшее)

Все эти следующие функции связаны с использованием переменных метода:

  • method
  • method_get_self
  • method_get_index

Наконец, у вас также есть следующие функции для определения типа переменной (более подробную информацию о типах данных можно найти здесь, а таблицы типов- здесь):

  • is_string
  • is_real
  • is_numeric
  • is_bool
  • is_array
  • is_struct
  • is_method
  • is_ptr
  • is_int32
  • is_int64
  • is_vec3
  • is_vec4
  • is_matrix
  • is_undefined
  • is_nan
  • is_infinity
  • typeof
  • bool
  • ptr
  • int64
