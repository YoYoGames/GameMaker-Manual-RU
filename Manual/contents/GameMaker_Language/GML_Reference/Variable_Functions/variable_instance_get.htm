variable_instance_get

С помощью этой функции вы можете получить значение из заданной именованной переменной. Вы предоставляете уникальное значение ID экземпляра (которое можно найти в свойствах экземпляра в редакторе комнаты или оно возвращается при вызове функции), а также имя переменной, значение которой нужно получить в виде строки (см. пример кода ниже). instance_create_layer()), а также имя переменной, значение которой нужно получить в виде строки (см. пример кода ниже). Функция вернет значение переменной или undefined, если переменная не существует.

ВАЖНО! Если переменная, которую вы получаете, не существует, то функция вернет ключевое слово undefined, и вы можете получить ошибки, которые остановят работу игры, поэтому, если вы сомневаетесь, используйте функцию variable_instance_exists сначала.

Синтаксис:

variable_instance_get(instance_id, name);

Аргумент Тип Описание
instance_id Идентификатор экземпляра The unique ID value of the instance to use
name Строка The name of the variable to get (as a string)

Возвращается:

Variable (any data type) or undefined (if the named variable does not exist)

Пример:

if variable_instance_exists(id, "shields")
{
    var ss = variable_instance_get(id, "shields");
}
else
{
    var ss = -1;
}

Приведенный выше код проверяет, существует ли переменная, и если существует, то значение, которое она содержит, извлекается и сохраняется в локальной переменной. Если она не существует, то локальная переменная устанавливается в -1.
