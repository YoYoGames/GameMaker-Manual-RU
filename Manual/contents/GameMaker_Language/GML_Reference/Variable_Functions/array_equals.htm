array_equals

С помощью этой функции вы можете проверить, равны ли два массива (эквивалентны или одинаковы). Вы задаете два массива для проверки, и функция возвращает true, если они равны (эквивалентны или одинаковы), или false, если нет. Обратите внимание, что это не то же самое, что проверка одинаковости двух массивов с помощью функции ==, которая не проверяет, хранят ли два массива эквивалентные значения, а только проверяет, ссылаются ли массивы на один и тот же исходный массив. Например:

var a = [1,2,3,4];
var b = [1,2,3,4];

if (a == b)
{
    show_debug_message( "This will never fire, as a and b do not reference the SAME array" );
}

if (array_equals(a, b))
{
    show_debug_message( "This will fire as both arrays contain similar values" );
}

Обратите внимание, что некоторые константы могут быть не равны самим себе, что может привести к ошибке. Вот пример:

if (array_equals([NaN], [NaN]))
{
    show_debug_message( "This will never fire as NaN cannot be equal to itself" );
}

Более подробную информацию см. в Таблице равенства.

Синтаксис:

array_equals(var1, var2);

Аргумент Тип Описание
var1 Массив The first array.
var2 Массив The second array.

Возвращается:

Boolean

Пример:

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

Приведенный выше код проверит два массива на наличие в них эквивалентных значений, и если это не так, то код скопирует все содержимое массива "inventory_array" в массив "item_array".
