method_get_index

С помощью этой функции вы можете получить индекс сценария, в котором был определен метод. Если метод не был определен в скрипте, то функция вернет -1, в противном случае она вернет значение индекса для скрипта.

Синтаксис:

method_get_index(method);

Аргумент Тип Описание
method Метод The method variable to check

Возвращается:

Script Asset or Real

Пример:

var _index = method_get_index(light_setup);
if (_index != undefined)
{
    show_debug_message(script_get_name(_index));
}

Приведенный выше код проверит индекс скрипта на наличие метода "light_setup", и если он не равен -1, то выведет имя скрипта в консоль.
