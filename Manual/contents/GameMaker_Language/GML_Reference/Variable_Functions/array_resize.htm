array_resize

С помощью этой функции вы можете изменить размер существующего массива на новый. Вы указываете массив, размер которого необходимо изменить, а также новое количество индексов для массива, и функция изменит размер массива. Обратите внимание, что эта функция предназначена для изменения размера массива до меньшей длины, так как вы можете изменить размер массива, просто установив новый индекс в массиве. Тем не менее, если вы используете эту функцию для увеличения размера массива, все новые индексы будут установлены в значение по умолчанию 0.

Эта функция также может быть использована для многомерных массивов, если при указании индекса массива вы укажете, какое измерение вы хотите изменить, следуя этой схеме:

// Resize the first dimension of the array
array_resize(my_array, 10);

// Resize the second dimension of the array (only for the first slot)
array_resize(my_array[0], 10);

// Resize the third dimension of the array (only for the first slots)
array_resize(my_array[0][0], 10);

// ...and so on.

В коде указано, что изменяется размер только первого слота второго измерения, поскольку слоты в любом данном измерении массива не связаны между собой и могут иметь разные размеры; например:

array_resize(my_array[0], 10);
array_resize(my_array[1], 22);

В приведенном выше коде длина второго измерения массива равна 10 в первом слоте, но 22 во втором слоте.

Синтаксис:

array_resize(array_index, new_size);

АргументТип Описание
array_index The index of the array to resize.
new_size The new size for the array (an integer, starting from 0).

Возвращается:

Пример:

if array_length(a) > 10
{
    array_resize(a, 10);
}

Приведенный выше код проверяет длину массива, и если он имеет более 10 индексов, то его размер изменяется.
