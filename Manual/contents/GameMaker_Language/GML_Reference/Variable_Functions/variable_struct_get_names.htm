struct_get_names

С помощью этой функции вы можете получить массив, заполненный именами переменных из структуры. Вы передаете ссылку на struct для проверки, и каждая запись в массиве будет строкой имен переменных, которые содержит struct.

Синтаксис:

struct_get_names(struct);

АргументТип Описание
struct The struct reference to check.

Возвращается:

(each entry is a string)

Пример:

var str = "";
var array = variable_struct_get_names(mystruct);
show_debug_message("Variables for struct: " + string(array));
for (var i = 0; i < array_length(array); i++;)
{
    str = array[i] + ":" + string(variable_struct_get(mystruct, array[i]));
    show_debug_message(str);
}

Приведенный выше код получит массив имен переменных для заданной структуры и затем отобразит их вместе с их значениями в отладочном выводе.
