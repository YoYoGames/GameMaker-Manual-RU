array_pop

Эта функция удаляет последний элемент в заданном массиве и возвращает его значение. Если массив пуст, возвращается undefined.

Синтаксис:

array_pop(array);

АргументТип Описание
array The array to remove the last value from.

Возвращается:

Variable or undefined

Пример:

var _lastscore = array_pop(score_array);
draw_text(32, 32, "Last Score = " + string(_lastscore));

Приведенный выше код извлечет последнее значение из заданного массива, а затем выведет его вместе с текстом на экран. Извлеченная запись также удаляется из массива очков.
