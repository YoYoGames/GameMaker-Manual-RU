array_insert

С помощью этой функции вы можете вставить значение (или значения) в массив в любую заданную позицию. Функция требует, чтобы вы предоставили переменную, содержащую массив, индекс (позицию) в массиве для вставки, а также хотя бы одно значение для вставки, хотя по желанию вы можете предоставить дополнительные аргументы, и все они будут вставлены в массив в последовательном порядке, начиная с заданного индекса.

Обратите внимание, что если заданный индекс находится за пределами длины массива, значения будут добавлены по заданному индексу, а все пустые места в массиве между его последним значением и вставленными значениями будут установлены в значение по умолчанию 0.

Синтаксис:

array_insert(variable, index, value, [value], [value], [etc...]);

Аргумент Тип Описание
variable Массив The variable that holds the array.
index Реально The index (position) in the array to insert the value(s) 
value Переменная The value to insert
[value], [value], [etc...] Переменная OPTIONAL Дополнительные значения для вставки в массив

Возвращается:

N/A

Пример:

array_insert(score_array, score_rank, current_score);

Приведенный выше код вставит текущий результат в массив score по индексу, заданному переменной score_rank.
