is_string

Эта функция возвращает, является ли данная переменная строковым числом или нет. В некоторых случаях вы хотите проверить и убедиться, что переменная в GameMaker содержит строку, а не вещественное число, и тогда вы используете эту функцию. Она возвращает не строку, а true или false, поэтому значение, например, "рыба" для n вернет true, а значение 200 для n вернет false.

Синтаксис:

is_string(n);

Аргумент Тип Описание
n Переменная The argument to check.

Возвращается:

Boolean

Пример:

if is_string(val)
{
    name = "Player: " + val;
}

Приведенный выше код проверяет переменную "val" на наличие в ней строки, и если она содержит строку, то добавляет ее в другую строку, которая затем присваивается переменной "name".
