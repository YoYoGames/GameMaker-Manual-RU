array_set

С помощью этой функции вы можете установить значение индекса в массиве в значение. Функция требует, чтобы вы предоставили переменную, содержащую массив, а также индекс, который нужно установить, и значение, которое нужно установить. Эта функция также может быть использована для многомерных массивов, если при указании индекса массива вы укажете, какую размерность вы хотите установить, следуя следующей схеме:

// 1D array
array_set(my_array, 0, 100);
// 2D array
array_set(my_array[0], 0, 100);
// 3D array
array_set(my_array[0][0], 0, 100);
// etc...

Синтаксис:

array_set(variable, index, value);

Аргумент Тип Описание
variable Массив The variable that holds the array.
index Реально The index of the array element to set the value for.
value Переменная The value to set.

Возвращается:

N/A

Пример:

for (var i = 0; i < 10; ++i;)
{
    array_set(score_array, i, i*100));
}

Приведенный выше код установит первые 10 элементов в заданном массиве в значение.
