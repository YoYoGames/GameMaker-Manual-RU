typeof

Эта функция возвращает тип данных любой заданной переменной в виде строки. Возможные возвращаемые значения перечислены в таблице ниже:

Строка Описание
"number" Переменная содержит вещественное число (с плавающей точкой) - оно может включать NaN и infinity
"string" Переменная содержит строку
"array" Переменная ссылается на массив
"bool" Переменная содержит булево значение (true / false).
"int32" Переменная содержит 32-битное целое число
"int64" Переменная содержит 64-битное целое число
"ptr" Переменная содержит указатель
"undefined" Переменная не определена
"null" Переменная имеет нулевое значение (в норме это не должно наблюдаться)
"vec3" Переменная содержит вектор из 3 значений
"vec4" Переменная содержит вектор из 4 значений
"method" Переменная содержит ссылку на функцию
"struct" Переменная содержит ссылку на структуру
"ref" Переменная содержит ссылку на экземпляр
"unknown" Значение неизвестно. Это никогда не должно встречаться и означает, что что-то пошло не так на самом базовом уровне, например, перезапись памяти.

Обратите внимание, что бывают случаи, когда эта функция может не вернуть правильное значение для метода. Рассмотрим следующие два определения функций:

a = function()
{
    // something
}

function b()
{
    // Something
}

Технически, они оба считаются методами, так как связывают функцию с переменной области видимости экземпляра, однако вызов typeof() на функцию b вернет "число", а не "метод", в то время как вызов на a вернет "метод". Это связано с тем, что методам, созданным подобно методу b, присваиваются индексы сценария (которые являются целочисленными значениями), поскольку именно так компилятор распознает функции сценария, даже если функция не была определена в сценарии.

Синтаксис:

typeof(variable);

Аргумент Тип Описание
variable Любой The variable to get the data type of.

Возвращается:

String (see table above)

Пример:

var _str = typeof(global.ExtensionInput);
show_debug_message(" global.ExtensionInput is a " + _str);

Приведенный выше код получает тип данных, хранящихся в заданной глобальной переменной, и возвращает строку в локальную переменную, которая затем используется для вывода сообщения на консоль.
