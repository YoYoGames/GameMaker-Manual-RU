метод_get_self

С помощью этой функции вы можете получить идентификатор экземпляра или ссылку на структуру, которая является контекстом, используемым при вызове метода. self контекст, используемый при вызове метода. Если переменная не является методом, то функция вернет undefined. Обратите внимание, что функция также может вернуть константу pointer_null, в этом случае во время вызова используется текущий self.

Синтаксис:

method_get_self(method);

АргументТип Описание
method The method variable to check.

Возвращается:

Instance ID, Struct, undefined, or pointer_null

Пример:

var _self = method_get_self(light_properties);
show_debug_message(string(_self));

Приведенный выше код получает контекст self для заданной переменной метода и выводит его на консоль.
