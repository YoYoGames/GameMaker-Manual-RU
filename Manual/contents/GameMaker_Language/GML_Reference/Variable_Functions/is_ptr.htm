is_ptr

Эта функция возвращает, является ли данная переменная указателем или нет. В некоторых случаях вы хотите проверить и узнать, какой тип данных имеет переменная в GameMaker, и тогда вы используете эту функцию. Она возвращает true или false в зависимости от того, является ли значение указателем или нет.

Синтаксис:

is_ptr(n);

Аргумент Тип Описание
n Переменная The argument to check.

Возвращается:

Boolean

Пример:

if !is_ptr(val)
{
    show_debug_message("Not a valid texture!");
}

Приведенный выше код проверяет переменную "val" на наличие в ней указателя, и если это не так, то выводит сообщение в консоль отладки.
