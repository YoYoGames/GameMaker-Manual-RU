array_copy

С помощью этой функции вы можете скопировать весь массив или его часть в другой массив в любой позиции. Вам необходимо указать исходный и конечный массивы (оба должны быть созданы ранее), а также позицию в исходном массиве для копирования и позицию в конечном массиве для копирования. Наконец, необходимо указать длину массива (или длину нужной части) для копирования. Если копируемые данные превышают длину массива назначения, массив будет расширен, чтобы принять эти данные.

Эта функция также может быть использована для многомерных массивов, если при указании индекса массива вы укажете, какое измерение вы хотите скопировать, следуя этой схеме:

// Copy to the first dimension of an array
// from the second dimension of an array
array_copy(item_array, 0, inventory_array[0], 0, len);

// Copy to the third dimension of an array
// from the first dimension of an array
array_copy(item_array[0][0], 0, inventory_array, 0, len);

// etc...

Синтаксис:

array_copy(dest, dest_index, src, src_index, length);

Аргумент Тип Описание
dest Массив The ID of the array to copy to.
dest_index Реально The index within the array to copy to.
src Массив The ID of the array to copy from.
src_index Реально The index with the array to start copying from.
length Реально The length (number of array indices) to copy.

Возвращается:

N/A

Пример:

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

Приведенный выше код проверит два массива на наличие в них эквивалентных значений, и если это не так, то код скопирует все содержимое массива "inventory_array" в массив "item_array".
