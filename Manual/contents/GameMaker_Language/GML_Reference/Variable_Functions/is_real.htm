is_real

Эта функция возвращает, является ли данная переменная действительным числом (единичным, двойным или целым) или нет. В некоторых случаях вы хотите проверить, является ли переменная вещественным числом, и тогда вы используете эту функцию. Она возвращает не действительное число, а true или false, поэтому значение, например, "рыба" вернет false, а значение 200 вернет true.

ПРИМЕЧАНИЕ: Эта функция будет возвращать false для любых enum значения, поскольку они хранятся как int64s. Правильной функцией для проверки этого типа данных является is_int64().

Синтаксис:

is_real(n);

Аргумент Тип Описание
n Переменная The argument to check.

Возвращается:

Boolean

Пример:

if is_real(val)
{
    score += val;
}

Приведенный выше код проверяет переменную "val" на то, является ли она действительным числом, и если да, то добавляет ее к счету.
