переменная_структура_удаления

С помощью этой функции вы можете удалить переменную из структуры. Вы передаете идентификатор структуры, из которой нужно удалить переменную, и строковое имя переменной, которую нужно удалить.

Синтаксис:

variable_struct_remove(struct, name);
