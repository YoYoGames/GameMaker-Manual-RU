struct_remove

С помощью этой функции вы можете удалить переменную из структуры. Вы передаете идентификатор структуры, из которой нужно удалить переменную, и строковое имя переменной, которую нужно удалить.

 

Синтаксис:

struct_remove(struct, name);

Аргумент Тип Описание
struct Struct The struct reference to remove the variable from
name String The name of the variable to remove (as a string)

 

Возвращается:

N/A

 

Пример:

if struct_exists(mystruct, "shields")
{
    struct_remove(mystruct, "shields");
}

Приведенный выше код проверит, существует ли заданная переменная в заданной структуре, и если да, то она будет удалена.
