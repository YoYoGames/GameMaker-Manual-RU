отмена вызова

Эта функция отменяет функцию обратного вызова, которая была запущена ранее с помощью call_later.

Синтаксис:

call_cancel(handle);

Аргумент Тип Описание
handle Идентификатор источника времени The handle to a Time Source возвращаемый call_later.

N/A

Пример:

call_cancel(handle);
