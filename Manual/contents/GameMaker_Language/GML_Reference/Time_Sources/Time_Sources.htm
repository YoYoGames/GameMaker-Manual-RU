Источники времени

Источник времени" - это созданный вами пользовательский таймер. Он работает в течение заданного периода времени и в конце истекает.

Источник времени (TS) имеет возможность вызвать пользовательский метод ("обратный вызов") по истечении срока действия.

Он также способен повторяться определенное количество раз или вечно. Каждое повторение Источника времени известно как "повторение".

Как использовать источники времени?

Создайте и запустите источник времени, который вызывает пользовательский метод:

  • Вызовите time_source_create() / Создать источник времени для создания источника времени
    • Укажите, от какого источника времени он наследуется: глобального, игрового или пользовательского .
    • Укажите период времени, в течение которого он работает, и используемые единицы измерения (секунды или кадры)
    • Укажите пользовательский метод или функцию скрипта с массивом аргументов для передачи в этот метод.
    • По желанию, укажите количество повторений.
      • По умолчанию он запускается только один раз
  • Вызовите time_source_start() / Запустить источник времени, чтобы запустить этот источник времени немедленно, или вызвать его позже, когда вам нужно, чтобы он запустился

Это заставит источник времени работать, и по истечении указанного периода он истечет и вызовет указанный вами метод.

Простые таймеры

Хотя настройка источника времени не занимает много строк кода, может быть удобно быстро настроить таймер и управлять им с помощью одной функции.

Следующие две функции обеспечивают более простой, короткий способ создания источника времени и его остановки:

  • звонок_позже
  • отмена вызова

ПРИМЕЧАНИЕ Вы не можете использовать функции источника времени с идентификаторами источника времени, используемыми этими функциями. Дополнительную информацию см. в разделе call_later.

Список функций

Для создания и изменения источников временииспользуются следующие функции:

  • время_источника_создать
  • время_источника_уничтожения
  • время_источника_начало
  • время_источника_остановки
  • время_источника_паузы
  • время_источника_возобновления
  • время_источника_переконфигурировать
  • время_источника_сброса
  • время_источник_получить_детей
  • время_источник_получить_родителя
  • время_источника_получить_период
  • time_source_get_reps_completed
  • время_источника_получения_остатка
  • время_источника_получить_состояние
  • время_источника_получить_время_остатка
  • время_источник_получить_единицы
  • источник_времени_существует

Каждая функция get_* вернет неопределенное значение, если указанный источник времени не существует.

Для преобразования временных периодов предусмотрены следующие вспомогательные функции:

  • время_секунды_до_bpm
  • время_bpm_до_секунд
