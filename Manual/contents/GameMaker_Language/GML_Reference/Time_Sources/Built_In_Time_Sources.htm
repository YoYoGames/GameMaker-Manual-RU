Встроенные источники времени

Источник времени Постоянная
Постоянная Описание Значение
time_source_global Источникглобального времени 0
time_source_game Источникигрового времени 1

Вы можете использовать глобальный источник времени или игровой источник времени в качестве родительского для вашего пользовательского источника времени. Оба источника временидоступны глобально.

Глобальный источник времени работает вне основного цикла игры, а игровой источник времени работает как часть цикла игры.

Выбор между Global/Game не влияет на то, будут ли источники временизависеть от частоты кадров в вашей игре; это зависит от используемого вами устройства.

Заказ

Источники времени, наследующие от глобального источника времени, обрабатываются раньше, чем источник времени игры.

Государства

Источник игрового времени имеет состояния, то есть он может быть paused. Глобальный источник времени не имеет состояний и не может быть приостановлен.

Вы можете использовать глобальный источник времени для таймеров, которые должны запускаться независимо от состояния игры, и источник игрового времени для таймеров, привязанных к основному циклу игры. Затем вы можете приостановить все таймеры, связанные с игрой, приостановив источник игрового времени.
