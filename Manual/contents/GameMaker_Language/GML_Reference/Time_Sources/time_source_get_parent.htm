время_источник_получить_родителя

Эта функция возвращает Идентификатор источника времени родителя данного источника времени.

Возвращаемое значение может быть идентификатором пользовательского источника времени или одного из встроенных источников времени, в зависимости от того, как был создан данный источник времени.

Синтаксис:

=======

time_source_get_parent

This function returns the Time Source of the given Time Source's parent.

The returned value may be a custom Time Source, or one of the Built-In Time Sources, depending on how the given Time Source was created.
