Единицы источника времени

Источник времени Единица измерения Постоянная
Постоянная Описание Значение
time_source_units_seconds Используйте секунды для периода источника времени (не зависит от кадра) 0
time_source_units_frames Использовать кадры для периода источника времени (зависит от кадров) 1

Эти константы используются в time_source_create() и time_source_reconfigure() для установки единиц, используемых для периода источника времени, и возвращаются функцией time_source_get_units().

Если вы используете секунды, ваш источник времени будет работать независимо от частоты кадров в игре.

Если вы используете кадры, ваш источник времени будет зависеть от частоты кадров в игре.

Вы также можете использовать BPM (удары в минуту) в качестве единицы измерения, вызвав команду time_bpm_to_seconds() для преобразования значения BPM в период в секундах, а затем используя time_source_units_seconds в качестве единицы измерения.
