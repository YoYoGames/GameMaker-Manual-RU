время_источника_возобновления

Эта функция возобновляет работу заданного источника времени и его дочерних элементов. Данный источник времени должен быть ранее приостановлен с помощью функции time_source_pause().

Синтаксис:

time_source_resume(id);

Аргумент Тип Описание
id Идентификатор источника времени The Time Source возобновить

Возвращается:

N/A

Пример:

if (keyboard_check_pressed(vk_space))
{
    var _state = time_source_get_state(time_source);

    if (_state == time_source_state_active)
    {
        time_source_pause(time_source);
    }
    else if (_state == time_source_state_paused)
    {
        time_source_resume(time_source);
    }
}

При нажатии клавиши Space этот код получит состояние источника времени.

Когда состояние активно, оно приостанавливает работу Источника времени, а когда он приостановлен, возобновляет ее.
