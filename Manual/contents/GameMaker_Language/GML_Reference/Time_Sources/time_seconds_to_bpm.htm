время_секунды_до_bpm

Эта функция принимает длительность такта в секундах и возвращает значение частоты ударов в минуту.

Синтаксис:

time_seconds_to_bpm(seconds);

Аргумент Тип Описание
seconds Реально The seconds-per-beat value to convert into beats-per-minute

Возвращается:

Real

Пример:

var _period = time_source_get_period(time_source);
var _bpm = time_seconds_to_bpm(_period);

show_debug_message(_bpm);

Эта функция получает период источника времени, преобразует его в BPM и печатает это значение в выходной журнал.

Этот код следует примеру для time_bpm_to_seconds().
