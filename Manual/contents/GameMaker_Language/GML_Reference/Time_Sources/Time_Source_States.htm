Государства-источники времени

Источник времени Государственная константа
Постоянная Описание Значение
time_source_state_initial Источник времени еще не запущен 0
time_source_state_active Источник времени запущен и ведет обратный отсчет времени 1
time_source_state_paused Источник времени - это paused 2
time_source_state_stopped Источник времени был остановлен или его срок действия полностью истек 3

Источник времени может иметь состояние, которое может быть любой из вышеперечисленных констант. Состояние по умолчанию для вновь созданного источника времени - time_source_state_initial.

Состояние можно изменить с помощью time_source_start(), time_source_stop(), и time_source_pause().

Состояние источника времени возвращается с помощью time_source_get_state().
