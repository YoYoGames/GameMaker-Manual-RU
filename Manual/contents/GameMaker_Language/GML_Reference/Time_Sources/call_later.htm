звонок_позже

Эта функция запускает функцию обратного вызова через указанное время или через определенный интервал. Для этого создается источник времени (см. Источники времени).

Чтобы отменить функцию обратного вызова, см. call_cancel.

ВАЖНО Хотя эта функция возвращает фактический ID источника времени, этот ID нельзя использовать с функциями источника времени, так как источники времени, используемые call_later и call_cancel, являются "невидимыми" - они не появляются как дочерние элементы встроенных источников времени и не могут быть использованы с другими функциями источника времени. Однако они включаются в подсчет источников времени, возвращаемый функцией debug_event("ResourceCounts").

Период

=======

call_later

This function triggers a callback function after the specified time or after a certain interval. It does this by creating a Time Source (see Time Sources).

To cancel the callback function, see call_cancel.

IMPORTANT While this function returns an actual time source, it cannot be used with the time source functions, as the time sources used by call_later and call_cancel are "invisible" - they will not appear as children of any built-in time sources and cannot be used with other time source functions. They are, however, included in the time source count returned by debug_event("ResourceCounts").

Example 1:

var _callback = function()
{
    show_debug_message("Five seconds have passed!");
}
var _handle = call_later(5, time_source_units_seconds, _callback);

В этом примере мы создаем функцию обратного вызова, которая показывает отладочное сообщение по истечении пяти секунд. После этого базовый источник времени уничтожается.

Пример 2:

var _create_enemy = function()
{
    var _x = random(room_width);
    var _y = random(room_height);
    instance_create_depth(_x, _y, depth, obj_enemy);
}

var _handle = call_later(10, time_source_units_seconds, _create_enemy, true);

В этом примере мы создаем функцию обратного вызова, которая создает нового врага в случайной позиции в комнате. Затем эта функция выполняется каждые 10 секунд, передавая ее в call_later. Последний аргумент имеет значение true чтобы функция продолжала вызываться каждые 10 секунд.

Используйте call_cancel с хэндлом, возвращенным call_later, чтобы остановить источник времени и вызовы обратного вызова.
