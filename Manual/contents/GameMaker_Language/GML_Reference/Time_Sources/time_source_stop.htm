время_источника_остановки

Эта функция останавливает данный источник времени и сбрасывает его таймер. Вы не можете остановить источник времени, который либо не запущен, либо находится в состоянии завершения.

Синтаксис:

time_source_stop(id);

Аргумент Тип Описание
id Идентификатор источника времени The Time Source остановиться

Возвращается:

N/A

Пример:

if (time_source_get_state(time_source) != time_source_state_stopped)
{
    time_source_stop(time_source);
}

Приведенный выше код остановит источник времени, если он еще не остановлен.
