время_источника_получить_дочерей

Для заданного источника времени эта функция возвращает массив, содержащий Идентификатор источника времениего дочерних элементов.

Синтаксис:

time_source_get_children(id);

Аргумент Тип Описание
id Идентификатор источника времени The Time Source чтобы получить детей

Возвращается:

Array of Time Source ID

Пример:

var _children = time_source_get_children(time_source);
var _count = array_length(_children);

for (var i = 0; i < _count; i ++)
{
    time_source_destroy(_children[i]);
}

time_source_destroy(time_source);

Этот код проходит через всех детей Источника времени и уничтожает их одного за другим.

В конце он уничтожает родительский Источник времени.
