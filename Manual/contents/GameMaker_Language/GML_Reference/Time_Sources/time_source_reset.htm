время_источника_сброса

Эта функция обнуляет заданный источник времени, сбрасывая его таймер и информацию о повторениях.

После вызова этой функции вам нужно будет снова запустить источник времени, чтобы он начал отсчет.

Синтаксис:

time_source_reset(id);

Аргумент Тип Описание
id Идентификатор источника времени The Time Source сбросить

Возвращается:

N/A

Пример:

// Room Start event
time_source_reset(global.spawn_time_source);
time_source_start(global.spawn_time_source);

Этот код будет сбрасывать данный источник времени всякий раз, когда начинается новая комната, а затем снова запускать источник времени.
