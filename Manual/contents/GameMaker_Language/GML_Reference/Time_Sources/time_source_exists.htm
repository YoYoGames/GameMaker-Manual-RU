источник_времени_существует

Эта функция возвращает, является ли заданный идентификатор действительным источником времени.

Если источник времени был уничтожен, то возвращается false, однако если источник времени был остановлен или просто истек, то он продолжает существовать, а значит, эта функция возвращает true.

Синтаксис:

=======

time_source_exists

This function returns whether the given variable holds a Time Source.

If the Time Source was destroyed, this will return false, however if the Time Source was stopped or it merely expired, it will continue to exist, meaning this function returns true.
