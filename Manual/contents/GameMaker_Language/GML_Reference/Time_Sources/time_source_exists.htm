источник_времени_существует

Эта функция возвращает, является ли заданный идентификатор действительным источником времени.

Если источник времени был уничтожен, то возвращается false, однако если источник времени был остановлен или просто истек, то он продолжает существовать, а значит, эта функция возвращает true.

Синтаксис:

time_source_exists(id);

Аргумент Тип Описание
id Идентификатор источника времени The Time Source проверять

Возвращается:

Boolean

Пример:

if (time_source_exists(global.spawn_time_source))
{
    if (keyboard_check_pressed(vk_space))
    {
        time_source_destroy(global.spawn_time_source);
    }
}

Этот код проверяет, существует ли источник времени. Если он существует, то проверяется, была ли нажата клавиша Space.

В этом случае он уничтожает Источник времени.
