время_источника_получить_время_остатка

Эта функция возвращает время, оставшееся до истечения срока действия заданного источника времени, и выполняет свой метод обратного вызова.

Это значение времени будет выражено в единицах, установленных для источника времени (секунды или кадры).

Синтаксис:

time_source_get_time_remaining(id);

Аргумент Тип Описание
id Идентификатор источника времени The Time Source чтобы получить информацию для

Возвращается:

Real
