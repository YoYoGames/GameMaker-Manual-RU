время_bpm_до_секунд

Эта функция принимает значение частоты ударов в минуту и возвращает длительность каждого удара в секундах.

Это можно использовать при создании источника времени, чтобы использовать значение BPM для источника времени. Важно, чтобы такой источник времени использовал секунды в качестве единицы измерения.

Синтаксис:

time_bpm_to_seconds(bpm);

Аргумент Тип Описание
bpm Реально The beats-per-minute value to convert into seconds-per-beat

Возвращается:

Real

Пример:

var _bpm = 90;
var _seconds = time_bpm_to_seconds(_bpm);

time_source = time_source_create(time_source_game, _seconds, time_source_units_seconds, function()
{
    show_debug_message("BEAT!");
}, [], -1);

Этот код преобразует значение 90 BPM в секунды и использует его для создания источника времени, который работает неограниченное время.

При каждом ударе он печатает сообщение "BEAT!" в выходной журнал.
