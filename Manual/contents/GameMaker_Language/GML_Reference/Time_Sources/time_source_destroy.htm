time_source_destroy

Эта функция уничтожит заданный источник времени.

При желании можно указать, должны ли быть удалены и все дочерние источники времени.

Синтаксис:

time_source_destroy(id, [destroy_tree]);

Аргумент Тип Описание
id Идентификатор источника времени The Time Source уничтожить
destroy_tree Булево OPTIONAL Следует ли уничтожать и дочерние источники времени, по умолчанию false

Возвращается:

N/A

Пример:

// Create event
time_source = time_source_create(time_source_game, 1, time_source_units_seconds, my_method, [], -1);

// Clean Up event
time_source_destroy(time_source);

Код события Create создает новый источник времени, который истекает один раз в секунду и повторяется неограниченное количество раз.

Код события Clean Up уничтожает этот источник времени, поскольку он не понадобится после уничтожения экземпляра.
