Типы истечения срока действия источника времени

Срок действия источника времени Постоянная
Постоянная Описание Значение
time_source_expire_nearest Источник времени истечет на кадре, ближайшем к его времени истечения 0
time_source_expire_after Срок действия источника времени истекает на первом кадре после истечения времени его действия 1

Источники времени, использующие секунды в качестве единицы измерения, не всегда истекают в игровом кадре. Они могут истекать после обработки игрового кадра, но до начала обработки нового кадра.

Однако обратный вызов для источника времени может выполняться только на одном игровом кадре, поэтому тип истечения срока действия источника времени контролирует, когда будет запущен его обратный вызов, в случае если он истекает в середине двух кадров.

При использовании time_source_expire_nearest обратный вызов для источника времени будет выполняться на кадре, ближайшем к моменту истечения его срока действия, это может быть кадр до него или кадр после него.

При использовании time_source_expire_after обратный вызов для источника времени всегда будет выполняться на кадре после истечения срока его действия.
