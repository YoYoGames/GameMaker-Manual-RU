время_источника_начало

Эта функция запускает заданный источник времени, изменяя его состояние на time_source_state_active.

Указанный источник времени может быть новым источником времени, который никогда не запускался, был приостановлен, остановлен или истек без повторений.

Эта функция выполняет "мягкий сброс" данного источника времени, сбрасывая время истечения и оставшиеся повторения до значений, которые были для него настроены.

Синтаксис:

time_source_start(id);

Аргумент Тип Описание
id Идентификатор источника времени The Time Source начинать

Возвращается:

N/A

Пример:

var _my_method = function()
{
    instance_destroy();
}

var _time_source = time_source_create(time_source_game, 300, time_source_units_frames, _my_method);

time_source_start(_time_source);

В данном примере мы хотим, чтобы экземпляр уничтожил себя через 300 кадров.

Сначала код создает метод, который просто вызывает instance_destroy().

Затем создается источник времени, наследующий источник времени игры. Он устанавливает его период в 300 кадров.

Наконец, запускается источник времени.
