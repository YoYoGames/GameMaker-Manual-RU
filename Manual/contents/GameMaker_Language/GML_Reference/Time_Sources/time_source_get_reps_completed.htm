time_source_get_reps_completed

Эта функция возвращает количество повторений, которые данный источник времени выполнил с момента своего последнего сброса. Это количество раз, когда он истек и выполнил свой метод обратного вызова.

Синтаксис:

time_source_get_reps_completed(id);

Аргумент Тип Описание
id Идентификатор источника времени The Time Source чтобы получить информацию для

Возвращается:

Real
