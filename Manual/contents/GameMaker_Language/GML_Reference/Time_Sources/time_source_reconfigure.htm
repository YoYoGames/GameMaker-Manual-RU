время_источника_переконфигурировать

Эта функция используется для изменения основных свойств источника времени без необходимости создания совершенно нового.

Вы указываете существующий Источник времени, а затем устанавливаете свойства, которые также установлены в time_source_create()кроме родительского. Подробную информацию об этих свойствах читайте на этой странице.

Указанный источник времени будет сброшен и деактивирован, и его нужно будет запустить снова.

Синтаксис:

time_source_reconfigure(id, period, units, callback, [args, repetitions, expiry_type]);

Аргумент Тип Описание
id Идентификатор источника времени The Time Source изменить конфигурацию
period Реально The period that the Time Source работает для, в заданных единицах
units Источник времени Единица измерения Постоянная The units that the given period is in
callback Метод The method to call when the Time Source истекает
args Массив OPTIONAL Массив, содержащий аргументы для передачи в метод
repetitions Реально OPTIONAL Количество повторений источника времени или -1 для бесконечного повторения.
expiry_type Срок действия источника времени Постоянная OPTIONAL Истекает ли срок действия источника времени на ближайшем к его истечению кадре или на следующем кадре

Возвращается:

N/A

Пример:

function change_spawn_delay(new_delay)
{
    time_source_reconfigure(obj_game.spawn_time_source, new_delay, time_source_units_frames, obj_game.spawn_method, [], -1);
    time_source_start(obj_game.spawn_time_source);
}

Это создает новую функцию, которая изменяет задержку спавна, используемую для внутриигровых врагов.

Если игра использует источник времени под названием obj_game.spawn_time_source для спавна врагов, то этот источник времени нужно будет обновить, когда изменится задержка спавна.

Эта функция делает именно это, перенастраивая источник времени с новой задержкой, а затем запуская его снова.
