доски достижений_показывать_лидеры

Эта функция откроет страницу таблицы лидеров для выбранной платформы. Обратите внимание, что это асинхронная функция, т.е. ваша игра будет продолжать работать в фоновом режиме, пока отображается страница таблицы лидеров. Поэтому вам следует быть осторожно использовать эту функцию и обязательно ставить игру на паузу или разрешать ее показ только в тех местах вашей игры, где она не будет мешать игровому процессу.

Синтаксис:

achievement_show_leaderboards()

Возвращается:

N/A

Пример:

if achievement_available()
    {
    global.Pause = true;
    instance_create_layer(0, 0, "Controllers", obj_Pause);
    achievement_show_leaderboards();
    }

Приведенный выше код проверяет, доступен ли выбранный сервис таблиц лидеров и достижений, и если да, то устанавливает глобальную переменную и создает экземпляр контроллера "pause" перед вызовом функции для отображения таблиц лидеров на экране.
