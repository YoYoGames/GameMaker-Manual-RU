достижение_доступно

Эта функция вернет true, если пользователь в данный момент подключен к Интернету и выбранная система таблиц лидеров и достижений доступна, в противном случае она вернет false.

Синтаксис:

achievement_available()

Возвращается:

Boolean

Пример:

if achievement_available()
    {
    achievement_post("ach_Level1", 100);
    }

Приведенный выше код проверит, доступен ли сервис таблицы лидеров, и, если да, опубликует на нем сообщение.
