достижение_логин_статус

В настоящее время эта функция полезна только для Google Play на платформе Android. Она проверяет, вошел ли пользователь уже в службы Google, в этом случае возвращается true, если нет - false. Если возвращается true, то нет необходимости вызывать achievement_login() функцию.

Синтаксис:

achievement_login_status();

Возвращается:

Boolean

Пример:

if !achievement_login_status()
    {
    achievement_login();
    }

Приведенный выше код позволит вам войти в выбранный сервис таблиц лидеров и достижений.
