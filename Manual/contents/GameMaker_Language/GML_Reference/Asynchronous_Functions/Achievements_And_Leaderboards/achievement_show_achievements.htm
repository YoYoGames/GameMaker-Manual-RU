достижение_показать_достижения

Эта функция откроет страницу достижений для выбранной платформы. Обратите внимание, что это асинхронная функция, то есть ваша игра будет продолжать работать в фоновом режиме, пока отображается страница достижений. Поэтому вам следует быть осторожно использовать эту функцию и обязательно ставить игру на паузу или разрешать ее показ только в тех местах вашей игры, где она не будет мешать игровому процессу.

Синтаксис:

achievement_show_achievements()

Возвращается:

N/A

Пример:

if achievement_available()
    {
    global.Pause = true;
    instance_create_layer(0, 0, "Controllers", obj_Pause);
    achievement_show_achievements();
    }

Приведенный выше код проверяет, доступен ли выбранный сервис лидербордов и достижений, и если да, то устанавливает глобальную переменную и создает экземпляр контроллера "pause" перед вызовом функции для показа достижений на экране.
