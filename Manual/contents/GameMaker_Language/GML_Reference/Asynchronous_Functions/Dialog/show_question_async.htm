show_question_async

Эта функция открывает окно и показывает пользователю вопрос, заданный в функции. Это асинхронная функция, поэтому GameMaker не блокирует устройство, на котором она выполняется, в ожидании ответа, а продолжает выполнять события в обычном режиме. Функция имеет две кнопки, которые показывают "Да" и "Нет", и как только пользователь нажимает одну из них, запускается асинхронное событие Dialog, которое, только на время этого события, будет иметь карту DS, сохраненную в переменной async_load.

Эта карта будет содержать два ключа, "id" и "status". "id" - это значение, которое было возвращено функцией при вызове, а "status" будет либо true, либо false для "Yes" и "No" соответственно.

ПРИМЕЧАНИЕ Эта функция предназначена только для отладки и тестированияи не должна использоваться в выпущенных играх. Для этих целей вы должны создать пользовательский интерфейс для получения ввода от игроков, чтобы иметь полный контроль над тем, как выглядят и ведут себя диалоги.

Синтаксис:

show_question_async(string);

Аргумент Тип Описание
string Строка The question to ask to the user.

Возвращается:

Async Request ID

Расширенный пример:

Событие нажатия левой кнопки мыши (например) объекта, который показывает сообщение, будет иметь следующий код:

msg = show_question_async("Do you want to buy some armour for " + string(armour[0, 5]) + "coins?");

Вышеприведенная функция покажет вопрос с заданной строкой, требуя, чтобы пользователь нажал либо "Да", либо "Нет". Идентификатор функции хранится в переменной "msg" и будет использован в событии Asynchronous Dialog, как показано ниже:

var i_d, stat;
i_d = ds_map_find_value(async_load, "id");
if (i_d == msg)
{
    if (ds_map_find_value(async_load, "status"))
    {
        coins -= armour[0,5];
        global.protection += armour[0,0];
    }
}

Приведенный выше код сверяет ключ "id" возвращаемого DS Map со значением, хранящимся в переменной "msg". Если они совпадают, то проверяется, была ли нажата одна из двух кнопок, и если возвращается true, то вычитается значение из переменной и добавляется значение в глобальную переменную.
