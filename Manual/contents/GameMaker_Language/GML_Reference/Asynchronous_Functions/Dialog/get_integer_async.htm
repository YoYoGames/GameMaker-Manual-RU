get_integer_async

Эта функция открывает окно и отображает сообщение, а также место для ввода пользователем значения (которое для начала будет содержать значение по умолчанию). Это асинхронная функция, поэтому GameMaker не блокирует устройство, на котором она выполняется, в ожидании ответа, а продолжает выполнять события в обычном режиме. После того как пользователь набрал свою строку и нажал кнопку "Ок", запускается асинхронное событие Диалог срабатывает асинхронное событие, которое только на время этого события будет иметь карту DS, хранящуюся в переменной async_load.

Эта карта будет содержать три ключа: "id", "status" и "value". "id" - это значение, которое было возвращено функцией при вызове, "статус" будет либо true для нажатия кнопки "Ок", либо false, если сообщение было отменено (где это применимо, поскольку не все целевые платформы позволяют отменять сообщения). Наконец, "value" вернет целое число, которое пользователь ввел как строку (пустая строка "" будет возвращена, если ввод не был произведен).

ПРИМЕЧАНИЕ Эта функция предназначена только для отладки и тестированияи не должна использоваться в выпущенных играх. Для этих целей вы должны создать пользовательский интерфейс для получения ввода от игроков, чтобы иметь полный контроль над тем, как выглядят и ведут себя диалоги.

Синтаксис:

get_integer_async(string, default);

Аргумент Тип Описание
string Строка The message to show in the dialog.
default Реально The default value.

Возвращается:

Async Request ID

Расширенный пример:

Событие нажатия левой кнопки мыши (например) объекта, который показывает сообщение, будет иметь следующий код:

msg = get_integer_async("How old are you?", 0);

Выше будет показано сообщение, требующее, чтобы пользователь ввел строку и нажал кнопку "Ок". Идентификатор функции хранится в переменной "msg" и будет использован в асинхронном событии Dialogs, как показано ниже:

var i_d = ds_map_find_value(async_load, "id");
if i_d == msg
{
    if ds_map_find_value(async_load, "status")
    {
        global.Age = ds_map_find_value(async_load, "value");
    }
}

Приведенный выше код сверяет ключ "id" возвращаемого DS Map со значением, хранящимся в переменной "msg". Если они совпадают, то проверяется, было ли нажато "Okay" (а не закрыто/отменено окно), и если возвращается true, то устанавливается глобальная переменная из возвращаемого целого числа.
