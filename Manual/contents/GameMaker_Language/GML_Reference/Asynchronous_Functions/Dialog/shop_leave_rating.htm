магазин_оставить_рейтинг

Эта функция открывает диалог, зависящий от ОС, в котором вы можете попросить пользователя разместить оценку или комментарий к определенной странице. Вы можете определить текст, который будет отображаться в диалоге, а также текст, который вы хотите видеть на двух кнопках и URL, где должен быть размещен комментарий.

ПРИМЕЧАНИЕ: Эта функция предназначена только для целей Android и iOS.

ВАЖНО! На iOS, при использовании TestFlight, в процессе разработки будет отображаться подсказка, но вы не сможете ничего сделать. Как только ваше приложение появится в App Store, эта кнопка станет активной, и пользователю будет предоставлена возможность написать отзыв или оставить оценку. Обратите внимание, что если вы распространяете игру через TestFlight, это окно может даже не отображаться.

Синтаксис:

shop_leave_rating(text, yes_string, no_string, url)

Аргумент Тип Описание
text Строка Text that appears on the dialog.
yes_string Строка Text that appears on the "yes" button.
no_string Строка Text that appears on the "no" button.
url Строка The full URL where the comment has to be sent.

Возвращается:

N/A

Пример:

if timer <= 0
{
    shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://MacSweeney/comments");
    timer = 10000000;
}
else
{
    timer -= 1;
}

Приведенный выше код попросит пользователя оставить комментарий, если переменная "timer" досчитала до 0.
