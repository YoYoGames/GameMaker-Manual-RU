Facebook

Facebook является одной из самых важных социальных сетей в Интернете, и многие игры и разработчики в настоящее время имеют свои собственные страницы для размещения игр и медиа. В GameMaker есть расширение, позволяющее подключаться к Facebook и выполнять определенные действия и задачи, которые позволяет API. Перед началом использования расширения рекомендуется посетить страницу Facebook Developers, так как твердое понимание того, как работает Facebook Graph и доступные вызовы и разрешения, необходимо для того, чтобы получить максимальную отдачу от этих функций.

Вы можете узнать, как настроить и использовать расширение Facebook для всех доступных платформ по следующей ссылке:

  • Android, iOS и HTML5: Интеграция Facebook

Ниже перечислены устаревшие функции, которые были действительны только для устаревшего Facebook API. Поэтому вы больше не можете использовать эти функции в своих проектах, и они перечислены здесь на случай, если вы импортировали старый проект с этими функциями. Для использования текущих функций Facebook необходимо активировать Facebook API в соответствующих опциях игры для целевой платформы, в результате чего вам будет предложено загрузить и установить расширение. Расширение поставляется с "мини-руководством" формата *.pdf, объясняющим, как его использовать, а также показывающим все определения функций и примеры. Все ссылки на старые функции facebook_*() необходимо заменить (и обновить), чтобы использовать эквиваленты расширения fb_*().

Перечисленные ниже функции предназначены для интеграции вашей игры с Facebook:

  • facebook_init
  • facebook_login
  • facebook_request_publish_permissions
  • facebook_request_read_permissions
  • facebook_check_permission
  • facebook_status
  • facebook_accesstoken
  • facebook_user_id
  • facebook_graph_request
  • facebook_dialog
  • facebook_send_invite
  • facebook_post_message
  • facebook_logout
  • facebook_launch_offerwall
