http_get_file

С помощью этой функции вы можете подключиться к указанному URL-адресу, чтобы получить информацию в виде файла. Поскольку это асинхронная функция, GameMaker не будет блокироваться в ожидании ответа, а продолжит работу до тех пор, пока не получит обратную информацию. Эта информация будет в виде строки и вызовет событие Async Event в любом экземпляре, у которого оно определено в свойствах объекта.

ВНИМАНИЕ Вы должны знать, что из-за XSS-защиты в браузерах запросы и попытки загрузить ресурсы из других доменов блокируются и могут возвращать пустые результаты. Более подробную информацию см. в разделе " Междоменные проблемы ".

Это событие генерирует "обратный вызов", который подхватывается любыми HTTP Events, в этом случае генерируется файл DS Map (более известный как "словарь"), который является эксклюзивным для этого события и хранится в специальной переменной async_load. Эта DS карта будет содержать различные значения в зависимости от возвращаемых данных, т.е. событие будет срабатывать несколько раз по мере получения каждого пакета данных, чтобы, например, можно было показать индикатор выполнения. Общая структура карты DS будет выглядеть следующим образом:

  • id: ID, который был возвращен командой. Если вы отправляете серию запросов http_, вам необходимо знать, на какой из них вы получаете ответ, поэтому вы будете использовать это значение для сравнения со значением, которое вы сохранили при первоначальной отправке запроса, чтобы найти нужный запрос.
  • статус: Возвращает значение меньше 0 для ошибки, 0 для завершения и 1 для получения пакетов (подробнее см. ниже).
  • результат: Полученные данные (только строка).
  • url: Полный URL-адрес, который вы запросили.
  • http_status: Необработанный код статуса http (если доступен). Возвращается стандартный код статуса для большинства браузеров, например: 304 для "Not Modified" или 204 для "No Content" и т.д..

Если в вашу игру возвращается несколько пакетов, ключ обратного вызова "status" вернет 1, в этом случае карта DS будет иметь следующие дополнительные ключи:

  • "contentLength": Это размер файла, который, по словам веб-сервера, вы должны ожидать получить (может быть -1, если сервер не возвращает эти данные).
  • "sizeDownloaded": Размер данных, которые уже были загружены.

Синтаксис:

http_get_file(url, local_target);

Аргумент Тип Описание
url Строка The web address of the server that you wish to get file from
local_target Строка The local storage path to save the file to

Возвращается:

Async Request ID

Функция http_get_file() может быть вызвана из любого события, и поскольку она асинхронна, обратный вызов может быть почти мгновенным или занять несколько секунд. Вызов функции прост и выглядит следующим образом:

file = http_get_file("http://www.macsweeneygames.com/downloads/upgrade.zip", "\Downloads\Upgrade.zip");

Приведенный выше код запросит файл с заданного URL и настроит его загрузку в локальное хранилище (как указано в параметрах HTML5-игры), в каталог "Downloads" с заданным именем файла (оно не обязательно должно совпадать с именем исходного файла, но должно использовать то же расширение файла). Индекс карты async_load будет сохранен в переменной "file", чтобы вы могли проверить правильность обратного вызова в асинхронном событии, и если каталог сохранения не существует, он будет создан. Асинхронное событие будет вызвано под-событием HTTP, и в этом событии у вас будет что-то вроде следующего:

if ds_map_find_value(async_load, "id") == file
{
    var _status = ds_map_find_value(async_load, "status");
    if (_status == 0)
    {
        var _path = ds_map_find_value(async_load, "result");
        var _files = zip_unzip(path, "/NewContent/");
        if (_files > 0)
        {
            global.extra_content = true;
        }
    }
}

Приведенный выше код сначала проверит "id" созданной ds_map, затем проверит статус обратного вызова. Если значение равно 0 (сигнализирует об успехе), то результат обратного вызова будет использован вместе с функцией zip_unzip() для разархивирования загруженного файла в указанный каталог. Если распаковка прошла успешно, глобальная переменная устанавливается в true.
