Рисование

В этом разделе собраны все функции, связанные с рисованием в игровой комнате, а также с управлением процессом рисования (смешивание, альфа, обтравка и т.д...). Существует большое количество различных функций для рисования, и они разделены на следующие категории, чтобы облегчить поиск нужных вам функций:

  • Цвет и альфа
  • Управление графическим процессором
  • Mipmapping
  • Основные формы
  • Спрайты и плитки
  • Текст
  • Примитивы и форматы вершин
  • Поверхности
  • Освещение
  • Частицы
  • Текстуры
  • Шейдеры
  • Воспроизведение видео

Также доступны две специальные функции, одна из которых используется для включения или отключения события Draw Event для вашего проекта, а другая - для промывки всего конвейера рисования:

  • draw_enable_drawevent
  • draw_flush
