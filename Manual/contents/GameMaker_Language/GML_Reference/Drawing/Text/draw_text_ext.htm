нарисовать_текст_текста

Эта функция будет рисовать текст аналогично функции draw_text() только теперь вы можете задать пространство между каждой строкой текста - если текст занимает более одной строки - и ограничить ширину (в пикселях) строки в строке, так что если любая строка превысит это значение, GameMaker автоматически разделит текст на следующую строку в ближайшем доступном месте (если в тексте нет свободного места, он превысит это максимальное значение). значение ширины). Обратите внимание, что любое белое пространство в начале строки будет вычеркнуто перед разбором для отрисовки. Также обратите внимание, что при значении -1 для аргумента разделения строк по умолчанию будет установлено разделение, основанное на значении высоты символа "M" в выбранном шрифте.

Синтаксис:

draw_text_ext(x, y, string, sep, w);

АргументТип Описание
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.

Возвращается:

Пример:

draw_text_ext(100, 50, keyboard_string, 3, 300);

Приведенный выше код будет рисовать текст, который пользователь набирает на клавиатуре, разбивая его на новые строки каждый раз, когда длина строки превышает 300 пикселей. Код также будет поддерживать разделение в 3 пикселя между строками, если это произойдет.
