draw_text_ext_transformed

Эта функция представляет собой комбинацию базовой draw_text() функции с draw_text_ext() и draw_text_transformed() позволяет масштабировать и поворачивать текст, сохраняя при этом определенный межстрочный интервал и максимальную ширину строки. Обратите внимание, что аргумент "ширина" основан на масштабе, равном 1, поэтому, если масштаб другой, это значение должно быть изменено пропорционально. Например, если базовая ширина для переноса строки равна 300, а вы установили масштаб 2, то текст будет выглядеть неправильно, выходя за пределы заданной ширины. Вместо этого вам следовало бы установить ширину 150, чтобы компенсировать масштабирование.

Синтаксис:

draw_text_ext_transformed(x, y, string, sep, w, xscale, yscale, angle);

АргументТип Описание
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.
xscale The horizontal scale.
yscale The vertical scale.
angle The angle of the text.

Возвращается:

Пример:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_ext_transformed(room_width / 2, room_height / 2, keyboard_string, 10, 300, 2, 2, image_angle);

Приведенный выше код нарисует заданный текст в центре комнаты, с максимальной длиной строки 300 пикселей, расстоянием между каждой строкой 10 пикселей, вращающийся по кругу и масштабированный до удвоенного исходного размера.
