рисовать_текст_текст_цвет

Эта функция представляет собой комбинацию базовой draw_text() функции с draw_text_ext() и draw_text_colour() позволяет задавать цвета градиента для текста, а также межстрочный интервал и максимальную ширину строки.

ПРИМЕЧАНИЕ Градиентное смешивание недоступно для HTML5, если не включен WebGL, хотя вы все равно можете задать цвета смешивания, и шрифт будет смешиваться с первым заданным цветом. Однако при таком смешивании создается дубликат шрифта, который затем сохраняется в кэше и используется при необходимости, что далеко не оптимально, а если вы используете многократное изменение цвета, это замедлит производительность вашей игры. Вы можете установить размер кэша шрифтов, чтобы попытаться ограничить это, если это необходимо, используя функцию font_set_cache_size().

Синтаксис:

draw_text_ext_colour(x, y, string, sep, w, c1, c2, c3, c4, alpha);

Аргумент Тип Описание
x Реально The x coordinate of the drawn string.
y Реально The y coordinate of the drawn string.
string Строка The string to draw.
sep Реально The distance in pixels between lines of text.
w Реально The maximum width in pixels of the string before a line break.
c1 Цвет The colour for the top left of the drawn text.
c2 Цвет The colour for the top right of the drawn text.
c3 Цвет The colour for the bottom right of the drawn text.
c4 Цвет The colour for the bottom left of the drawn text.
alpha Реально The alpha for the text.

Возвращается:

N/A

Пример:

draw_text_ext_colour(200, 200, keyboard_string, 5, 300, c_blue, c_blue, c_navy, c_navy, 1);

Приведенный выше код будет рисовать текст, который пользователь набирает на клавиатуре, разбивая его на новые строки каждый раз, когда длина строки превышает 300 пикселей. Код также будет поддерживать разделение в 5 пикселей между строками, если это произойдет. Каждая строка текста будет окрашена с помощью синего градиента: светло-синий вверху и темно-синий внизу.
