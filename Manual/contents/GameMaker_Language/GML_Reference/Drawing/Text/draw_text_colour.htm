нарисовать_текст_цвет

Эта функция будет рисовать текст аналогично функции draw_text() только теперь вы можете выбрать цвета для окрашивания текста, а также значение альфа, и эти новые значения будут использоваться вместо базового цвета рисунка и альфа-значения.

ПРИМЕЧАНИЕ: Градиентное смешивание недоступно для HTML5, если не включен WebGL, хотя вы все равно можете задать цвета смешивания, и шрифт будет смешиваться с первым заданным цветом. Однако при таком смешивании создается дубликат шрифта, который затем сохраняется в кэше и используется по мере необходимости, что далеко не оптимально, а если вы используете многократное изменение цвета, это замедлит производительность вашей игры. Вы можете установить размер кэша шрифтов, чтобы попытаться ограничить это в случае необходимости используя функцию font_set_cache_size().

Синтаксис:

draw_text_colour(x, y, string, c1, c2, c3, c4, alpha);

АргументТип Описание
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
c1 The colour for the top left of the drawn text.
c2 The colour for the top right of the drawn text.
c3 The colour for the bottom right of the drawn text.
c4 The colour for the bottom left of the drawn text.
alpha The alpha for the text.

Возвращается:

Пример:

draw_set_colour(c_white);
draw_text(100, 100, "Health");
draw_text_colour(100, 200, string(health), c_lime, c_lime, c_green, c_green, 1);

Приведенный выше код рисует два участка текста на одной линии, причем первый текст рисуется белым (так как это базовый цвет рисунка), а второй текст рисуется с градиентом от лаймово-зеленого к обычному зеленому.
