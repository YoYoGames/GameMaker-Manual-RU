draw_get_halign

Эта функция используется для получения настройки выравнивания текста по горизонтальной оси и возвращает одну из перечисленных ниже констант.

Синтаксис:

draw_get_halign();

Возвращается:

Constant

Горизонтальное выравнивание Постоянная
Постоянная Выравнивание
fa_left
fa_center
fa_right

Пример:

var _cur_halign = draw_get_halign();
var _cur_valign = draw_get_valign();

draw_set_halign(fa_right);
draw_set_valign(fa_bottom);

draw_text(100, 32, "Score: " + string(score));

draw_set_halign(_cur_halign);
draw_set_valign(_cur_valign);

Приведенный выше код сохраняет текущие значения "halign" и "valign" в локальных переменных, а затем изменяет выравнивание, чтобы нарисовать текст. После рисования текста он возвращает выравнивания обратно к значениям, сохраненным в локальных переменных.
