нарисовать_текст_трансформированный

Эта функция рисует текст аналогично функции draw_text() только теперь вы можете выбрать масштаб текста по горизонтальной или вертикальной оси (эффективно растягивая или сжимая его), а также заставить GameMaker рисовать его под углом (где 0 - нормальный, а каждый градус больше 0 поворачивает текст против часовой стрелки).

Синтаксис:

draw_text_transformed(x, y, string, xscale, yscale, angle);

АргументТип Описание
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
xscale The horizontal scale (default 1).
yscale The vertical scale(default 1).
angle The angle of the text.

Возвращается:

Пример:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed(room_width / 2, room_height / 2, "GAME OVER!", 2, 2, image_angle);

Приведенный выше код нарисует заданный текст в центре комнаты, вращаясь по кругу и масштабируя его до удвоенного исходного размера.
