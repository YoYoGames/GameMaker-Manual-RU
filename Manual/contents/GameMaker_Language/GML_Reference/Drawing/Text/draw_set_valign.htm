draw_set_valign

Эта функция используется для выравнивания текста по вертикальной оси, и изменение вертикального выравнивания изменит положение и направление, в котором будет рисоваться весь дальнейший текст. По умолчанию используется значение fa_top. Принимаются следующие константы:

Постоянная Выравнивание
fa_top
fa_middle
fa_bottom

Синтаксис:

draw_set_valign(valign);

Аргумент Тип Описание
valign Выравнивание по вертикали Постоянная Vertical alignment constant (from the table above).

Возвращается:

N/A

Пример:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text(100, 32, "Score: " + string(score));

Приведенный выше код нарисует оценку, центрированную по самому центру текста.
