рисовать_текст

С помощью этой функции вы можете нарисовать любую строку в любой позиции в пределах комнаты (для рисования вещественных чисел следует использовать функцию string() для преобразования их в текст). Для объединения строк можно использовать + (см. пример ниже), а также \n в строке для добавления разрыва строки, чтобы она рисовалась в несколько строк (информацию о том, как правильно форматировать строку и какие управляющие символы можно использовать, смотрите здесь). Цвет текста и альфа определяются текущими базовыми значениями альфа и цвета, заданными в параметрах draw_set_alpha() и draw_set_colour().

ПРИМЕЧАНИЕ: Фактическое положение текста будет зависеть от значений выравнивания, заданных параметрами draw_set_halign() и draw_set_valign().

Синтаксис:

draw_text(x, y, string);

АргументТип Описание
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.

Возвращается:

Пример:

draw_text(x, y, "Hello, " + global.Name + "!\nI hope you are well!");

Приведенный выше код нарисует строку в позиции экземпляра x/y, которая будет использовать строку, хранящуюся в глобальной переменной "Name", и разделит ее на две строки.
