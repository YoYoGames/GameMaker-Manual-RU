draw_text_ext_transformed_colour

Эта функция представляет собой комбинацию базовой draw_text() функции с draw_text_ext(), draw_text_transformed() и draw_text_colour() функциями. Как таковая, это наиболее общая из всех функций для рисования текста и дает большую гибкость при рисовании. Обратите внимание, что аргумент "ширина" основан на масштабе, равном 1, поэтому если масштаб другой, это значение должно быть изменено пропорционально. Например, если базовая ширина для переноса строки равна 300, а масштаб установлен на 2, то текст будет выглядеть неправильно, выходя за заданную ширину. Вместо этого вам следовало бы установить ширину 150, чтобы компенсировать масштабирование.

ПРИМЕЧАНИЕ: Градиентное смешивание недоступно для HTML5, если не включен WebGL, хотя вы все равно можете задать цвета смешивания, и шрифт будет смешиваться с первым заданным цветом. Однако при таком смешивании создается дубликат шрифта, который затем сохраняется в кэше и используется по мере необходимости, что далеко не оптимально, а если вы используете многократное изменение цвета, это замедлит производительность вашей игры. Вы можете установить размер кэша шрифтов, чтобы попытаться ограничить это в случае необходимости используя функцию font_set_cache_size().

Синтаксис:

draw_text_ext_transformed_colour(x, y, string, sep, w, xscale, yscale, angle, c1, c2, c3, c4, alpha);

АргументТип Описание
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.
xscale The horizontal scale.
yscale The vertical scale.
angle The angle of the text.
c1 The colour for the top left of the drawn text.
c2 The colour for the top right of the drawn text.
c3 The colour for the bottom right of the drawn text.
c4 The colour for the bottom left of the drawn text.
alpha The alpha for the text.

Возвращается:

Пример:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text_ext_transformed_colour(200, 200, "My name is" + global.name +".", 20, 200, 0.5, 0.5, 180, c_blue, c_blue, c_navy, c_navy, 1);

В результате будет нарисован заданный текст с нисходящим градиентом от синего к темно-синему, в позиции (200,200) в комнате, повернутый вверх ногами и с размером шрифта вдвое меньше обычного. Между каждой линией должно быть 20 пикселей, если нарисовано более одна линия, а максимальная ширина каждой линии составляет 200 пикселей.
