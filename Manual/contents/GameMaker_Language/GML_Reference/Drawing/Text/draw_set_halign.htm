draw_set_halign

Эта функция используется для выравнивания текста по горизонтальной оси, и изменение горизонтального выравнивания изменит положение и направление, в котором будет рисоваться весь дальнейший текст; значение по умолчанию - fa_left. Принимаются следующие константы:

Постоянная Выравнивание
fa_left
fa_center
fa_right

Синтаксис:

draw_set_halign(halign);

Аргумент Тип Описание
halign Горизонтальное выравнивание Постоянная Horizontal alignment constant (from the table above).

Возвращается:

N/A

Пример:

draw_set_halign(fa_left);
draw_text(100, 32, "Score: " + string(score));
draw_set_halign(fa_right);
draw_text(room_width - 100, 32, "Health: " + string(health));

Приведенный выше код нарисует две строки на одной линии, причем счет будет выровнен по левому краю, а здоровье - по правому.
