color_get_saturation

Эта функция возвращает насыщенность заданного цвета. Это количество цветового тона, которое смешивается в конечный цвет, и является частью метода определения цвета по оттенку, насыщенности и значению (светлоте). На следующем рисунке показано как это значение соответствует цветовой шкале HSV:

Синтаксис:

colour_get_saturation(col);

АргументТип Описание
col The colour to check

Возвращается:

Пример:

col = make_colour_hsv(random(255), colour_get_sat(c_teal), 255);

Приведенный выше код получает насыщенность, используемую для создания цветовой константы "c_teal", а затем использует это значение для установки случайного цвета с той же насыщенностью, сохраняя полученный цвет в переменной "col".
