draw_get_colour

Эта функция возвращает текущий цвет рисования, который используется для рисования форм, текста, примитивов и нетекстурированных 3D-моделей. Его можно установить с помощью функции draw_set_colour() функция.

Синтаксис:

draw_get_colour()

Возвращается:

Colour Constant

Пример:

var _cur_color = draw_get_color();
draw_set_color(text_color);
draw_text(x, y, text);
draw_set_color(_cur_color);

Приведенный выше код сохраняет текущий цвет рисунка в локальной переменной и изменяет цвет рисунка на основе переменной экземпляра. После отрисовки текста цвет возвращается к значению, сохраненному в локальной переменной.
