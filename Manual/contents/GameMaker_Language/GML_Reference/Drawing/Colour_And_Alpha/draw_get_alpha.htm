draw_get_alpha

Эта функция возвращает текущее значение альфы рисунка, которое находится в диапазоне от 0 (полностью прозрачный) до 1 (полностью непрозрачный). Альфа рисунка влияет на прозрачность всех функций рисования и может быть установлена с помощью функции draw_set_alpha().

Синтаксис:

draw_get_alpha()

Возвращается:

Пример:

var _cur_alpha = draw_get_alpha();
draw_set_alpha(text_alpha);
draw_text(x, y, text);
draw_set_alpha(_cur_alpha);

Приведенный выше код сохраняет текущую альфу рисунка в локальной переменной и изменяет альфу рисунка на основе переменной экземпляра. После отрисовки текста он возвращает альфу обратно к значению, сохраненному в локальной переменной.
