colour_get_green

Эта функция возвращает количество зеленого цвета, использованного для создания заданного цвета, причем значение находится в диапазоне от 0 до 255, где 0 - нет зеленого, а 255 - весь зеленый. Следующее изображение иллюстрирует это:

Синтаксис:

colour_get_green(col);

АргументТип Описание
col The colour to check

Возвращается:

Пример:

g_comp = colour_get_green(c_teal);

Приведенный выше код получит зеленый компонент цветовой константы "c_teal" и сохранит его значение в переменной "g_comp".
