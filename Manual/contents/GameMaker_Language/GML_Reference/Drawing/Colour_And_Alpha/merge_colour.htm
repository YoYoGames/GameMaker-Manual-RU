слияние_цветов

С помощью этой функции вы можете взять два цвета и затем объединить их вместе, чтобы получить новый цвет. Количество каждого из составляющих цветов можно определить, изменив аргумент "amount", где значение 0 вернет первый цвет (col1), значение 1 вернет второй цвет (col2), а значение между ними вернет соответствующее смешение. Например, значение 0,5 смешает два цвета поровну. На следующем изображении показано, как это работает на примере слияния красного и синего цветов:

Синтаксис:

merge_colour(col1, col2, amount);

АргументТип Описание
col1 The first colour to merge
col2 The second colour to merge
amount How much of each colour should be merged. For example, 0 will return col1, 1 will return col2, and 0.5 would return a merge of both colours equally

Возвращается:

Colour Constant

Пример:

col = merge_colour(c_lime, c_orange, 0.5);

Приведенный выше код использует функцию для создания цвета путем слияния лайма и оранжевого 50/50, а затем сохраняет его значение в переменной "col" для последующего использования.
