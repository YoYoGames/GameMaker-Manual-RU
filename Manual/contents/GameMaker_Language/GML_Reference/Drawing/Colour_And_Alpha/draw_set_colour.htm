draw_set_colour

С помощью этой функции вы можете установить базовый цвет рисунка для игры.

Это повлияет на рисование шрифтов, форм, примитивов и 3D, однако не повлияет на sprites (нарисованные вручную или экземпляром). Если все затронутые графические объекты рисуются с собственными значениями цвета, это значение будет проигнорировано.

Синтаксис:

draw_set_colour(col);

Аргумент Тип Описание
col Цвет The colour to set for drawing.

Возвращается:

N/A

Пример:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

Приведенный выше код нарисует текст в указанной позиции с эффектом тени, созданным измененными альфа и цветом рисунка.
