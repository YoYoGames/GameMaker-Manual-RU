color_get_blue

Эта функция возвращает количество синего цвета, использованного для создания заданного цвета. Значение находится в диапазоне от 0 до 255, где 0 - нет синего, а 255 - весь синий. Следующее изображение иллюстрирует это:

Синтаксис:

colour_get_blue(col);

АргументТип Описание
col The colour to check

Возвращается:

Пример:

b_comp = colour_get_blue(c_teal);

Приведенный выше код получит синюю составляющую цветовой константы "c_teal" и сохранит ее значение в переменной "b_comp".
