draw_clear_alpha

Эта функция может использоваться для очистки всего экрана заданным цветом, при этом альфа-компонент места назначения устанавливается на заданное значение. могут быть настроены на использование альфа-компонента назначения, который вы установили. Эта функция предназначена только для использования в событии рисования экземпляра (она не будет отображаться при использовании в любом другом событии), и она также может быть очень полезна для очистки поверхностей при их новом создании.

Стоит отметить, что при использовании модуля HTML5 вы можете использовать эту функцию для того, чтобы сделать фон холста прозрачным, что позволит любым изображениям, используемым в хосте HTML, быть видимыми при запуске игры поверх них. Чтобы добиться этого эффекта сначала нужно зайти в редактор комнаты и удалить все фоновые слои из редактора комнаты, затем снять флажок с опции Clear View Background, расположенной в Свойствах вида. Наконец, нужно создать новый объект, поместить его в первую комнату игры и присвоить ему очень большую глубину (например, 10000). Затем в событии Pre Draw используйте эту функцию, чтобы установить альфа-фактор холста на цвет и прозрачность, которые вы хотите. которые вы хотите иметь.

Синтаксис:

draw_clear_alpha(col, alpha);

АргументТип Описание
col The colour with which the screen will be cleared
alpha The transparency of the colour with which the screen will be cleared

Возвращается:

Пример:

draw_clear_alpha(c_red, 0.5);

Это очистит экран с прозрачностью 50% (0,5) и красным цветом.
