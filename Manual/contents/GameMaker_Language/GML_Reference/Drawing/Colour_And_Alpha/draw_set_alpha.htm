draw_set_alpha

С помощью этой функции вы можете установить базовый альфа-фактор рисования для игры. Это значение может быть установлено от 0 до 1, при этом 0 будет полностью прозрачным, а 1 - полностью непрозрачным (значение по умолчанию), и будет влиять на все дальнейшее рисование, включая фон, sprites, шрифты, примитивы и 3D.

Обратите внимание, что если вы рисуете любой из активов и передаете аргумент альфа как часть функции (например, при использовании функции draw_sprite_ext()), то значение альфа, установленное здесь, будет отменяется, поэтому при рисовании sprite с помощью draw_sprite_ext() с аргументом альфа, равным 1, sprite будет нарисован полностью непрозрачным, независимо от альфа, установленного в draw_set_alpha() (это будет работать так же, если использовать draw_self() и установке image_alpha экземпляра). Также обратите внимание, что если экземпляр рисует по умолчанию (т.е. у него есть sprite, но нет события Draw Event), то переменная image_alpha переменная будет привязана к альфе, заданной с помощью draw_set_alpha(), так что при значении image_alpha равном 1 и альфе рисования 0.5 будет рисоваться sprite с альфой 0.5, а при значении image_alpha равном 0.25 и альфе рисования 0.5 будет рисоваться с альфой 0.5. 0,5 нарисует sprite с альфой 0,25.

Синтаксис:

draw_set_alpha(alpha);

АргументТип Описание
alpha The alpha to set (between 0 and 1)

Возвращается:

Пример:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

Приведенный выше код нарисует текст в указанной позиции с эффектом тени, созданным измененными альфой и цветом рисунка.
