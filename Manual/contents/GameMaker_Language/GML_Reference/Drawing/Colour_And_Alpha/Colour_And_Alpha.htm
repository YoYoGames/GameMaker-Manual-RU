Цвет и альфа

ВGameMaker имеется обширный набор функций для работы с цветом и альфой рисуемого на экране изображения. С помощью этих функций вы можете разделять цвета на составные части, создавать новые цвета и манипулировать альфой рисунка, чтобы изменить то, как различные объекты в вашей игре рисуются вместе.

О смешивании изображений с использованием специальных режимов наложения и других функций, специфичных для GPU, см. раздел " Управление GPU".

Шестнадцатеричные цвета

Вы можете создавать цвета из шестнадцатеричных значений, используя предварительно символ $, и эти значения также можно использовать с различными цветовыми функциями для получения значений компонентов по отдельности. Шестнадцатеричное значение цвета делится на три части в следующем формате: Синий/Зеленый/Красный(или $BBGGRR), поэтому, например, для определения светло-голубого/аква цвета можно использовать:

col = $FFCC11;

Здесь FF - синий компонент, CC - зеленый компонент и 11 - красный компонент.

Вы также можете использовать "цвета CSS", которые начинаются со знака хэш/фунт(#). При этом используется формат красный/зеленый/синий(или #RRGGBB), который более широко распространен и совместим с различными другими приложениями.

col = #11CCFF;

Это тот же цвет, что и в предыдущем блоке кода, только значения RR и BB поменяны местами для соответствия формату.

ПРИМЕЧАНИЕ GML также поддерживает формат $AABBGGRR (с шестнадцатеричным значением альфа в начале), однако большинство цветовых функций GML смогут использовать только цветовую часть и игнорировать альфа.

Функции

Следующие функции можно использовать для получения компонентных оттенков, составных оттенков, насыщенности и светлоты выбранного цвета, а также альфа и других свойств:

  • colour_get_blue
  • colour_get_green
  • colour_get_red
  • colour_get_hue
  • colour_get_saturation
  • colour_get_value
  • draw_getpixel
  • draw_getpixel_ext
  • draw_get_colour
  • draw_get_alpha

Эти функции можно использовать для создания цветов из необработанных входных значений цветовых компонентов:

  • make_colour_hsv
  • make_colour_rgb
  • merge_colour

Следующие функции можно использовать для настройки различных параметров рисования на экране, включая альфа, цвет и смешивание:

  • draw_clear
  • draw_clear_alpha
  • draw_set_alpha
  • draw_set_colour
