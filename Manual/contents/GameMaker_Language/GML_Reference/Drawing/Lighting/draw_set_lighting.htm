draw_set_lighting

Эта функция используется для включения всех эффектов освещения. По умолчанию отключена (false).

Синтаксис:

draw_set_lighting(enable);

Аргумент Тип Описание
enable Булево Enable or disable all lighting (true или false)

Возвращается:

N/A

Пример:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

Приведенный выше код включит освещение для всей сцены, затем определит белый направленный свет в пространстве комнаты и, наконец, включит этот свет.
