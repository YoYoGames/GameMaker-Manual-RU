draw_light_define_ambient

Эта функция используется для управления окружающим светом сцены, то есть светом, который есть в сцене даже без определения точечных или направленных источников света. Это общий цвет и яркость (или темнота) сцены. По умолчанию используется цвет c_black.

Синтаксис:

draw_light_define_ambient(col);

АргументТип Описание
col The colour to use (either a constant, a real or a hex value).

Возвращается:

Пример:

draw_light_define_ambient(c_white);

Приведенный выше код определяет окружающее освещение как белое.
