draw_light_define_point

Эта функция предназначена для определения позиционного света, где вы можете определить положение света по x, y и z, диапазон освещенности и его цвет (что также влияет на воспринимаемую интенсивность света, так как некоторые цвета кажутся "темнее", чем другие). другие). Вы также должны присвоить свету индексный номер, который будет использоваться в других функциях для ссылки на него.

ПРИМЕЧАНИЕ: Имеется только 8 аппаратных светильников, поэтому одновременно могут быть включены только 8 определенных светильников (хотя можно определить и больше).

Синтаксис:

draw_light_define_point(ind, x, y, z, range, col);

АргументТип Описание
ind The index number of the light (arbitrary)
x The x position of the light
y The y position of the light
z The z position of the light
range The light range (in pixels)
col The colour to use for the light (either a constant, a real or a hex value).

Возвращается:

Пример:

draw_set_lighting(true);
draw_light_define_point(1, 200, 123, 50, 2000, c_white);
draw_light_enable(1, true);

Приведенный выше код включит освещение для всей сцены, затем определит белый свет в определенной точке пространства комнаты и, наконец, включит этот свет.
