draw_light_get_ambient

Эта функция возвращает текущий цвет, используемый для окружающего освещения.

Синтаксис:

draw_light_get_ambient()

Возвращается:

Colour Constant

Пример:

light_col = draw_light_get_ambient();

Приведенный выше код получит текущий цвет окружающего освещения и сохранит его в переменной.
