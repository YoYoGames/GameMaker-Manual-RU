draw_light_get

Эта функция получит указанные параметры света в виде массива со следующими 6 элементами -...

  • [0] = включен / выключен (true / false)
  • [1] = Тип освещения Постоянная (см. таблицу ниже)
  • [2] = x позиция
  • [3] = положение y
  • [4] = положение z
  • [5] = радиус света (только для точечных светильников, для направленных будет 0)
  • [6] = светлый цвет (настоящий)

Тип света (элемент [1]) может быть одной из следующих двух констант:

Тип освещения Постоянный
Постоянная Описание
lighttype_dir Свет является направленным
lighttype_point Свет точечный

Синтаксис:

draw_light_get(ind);

Аргумент Тип Описание
ind Реально The index number of the light (from 0 to 7)

Возвращается:

Array

Пример:

light_a = draw_light_get(1);
if light_a[5] < 200
{
    light_a[5] += 5;
    draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
}

Приведенный выше код получит значения, используемые для определения света, индексированного как "1", затем проверит радиус, и если он меньше 200, то он будет увеличен, а радиус света установлен на новое значение.
