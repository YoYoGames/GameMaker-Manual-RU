draw_get_lighting

Эта функция возвращает, включено ли освещение (true) или нет (false) для всей сцены.

Синтаксис:

draw_get_lighting()

Возвращается:

Пример:

if !draw_get_lighting()
{
    draw_set_lighting(true);
}

Приведенный выше код проверит, включено ли освещение для сцены, и если нет, то включит его.
