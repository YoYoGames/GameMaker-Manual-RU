draw_light_enable

Эта функция используется для включения определенного освещения. Когда вы определяете позиционный или направленный свет, вы должны присвоить ему индексный номер, который затем используется этой функцией для включения или выключения света. По умолчанию отключен (false).

ПРИМЕЧАНИЕ: Имеется только 8 аппаратных светильников, поэтому одновременно могут быть включены только 8 определенных светильников (хотя можно определить и больше).

Синтаксис:

draw_light_enable(ind, enable);

Аргумент Тип Описание
ind Реально The index number of the light (from 0 to 7)
enable Булево Enable or disable all lighting (true или false)

Возвращается:

N/A

Пример:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

Приведенный выше код включит освещение для всей сцены, затем определит белый направленный свет в пространстве комнаты и, наконец, включит этот свет.
