texture_debug_messages

Эта функция может быть использована для включения или отключения отладочных сообщений текстуры. Если функция включена (установлено значение true), в окно вывода будет отправляться дополнительная информация об использовании страницы текстуры. Установите значение false, чтобы снова отключить этот вывод.

Синтаксис:

texture_debug_messages(enable);

АргументТип Описание
enable Enable or disable the texture debug messages

Возвращается:

Пример:

=======

texture_debug_messages

This function enables or disables texture debug messages.

When enabled (set to true), additional information about texture page use will be sent to the output window. Set to false to disable this output again.

 

Syntax:

texture_debug_messages(enable);

Argument Type Description
enable Boolean Enable or disable the texture debug messages

 

Returns:

N/A
