Текстуры

Следующие функции связаны с "текстурами", т.е. изображениями, которые хранятся в памяти VRAM для использования в качестве sprites или других объектов в вашей игре. Первый набор функций используется для получения информации о конкретных изображениях, их положении или размере на странице текстур, а также для установки определенных характеристик текстур:

  • texture_get_width
  • texture_get_height
  • texture_get_texel_width
  • texture_get_texel_height
  • texture_get_uvs
  • texture_set_stage
  • texture_global_scale

GameMaker также имеет ряд функций, связанных с управлением VRAM, связанным с отдельными текстурами, а также целыми группами текстур:

  • draw_texture_flush
  • texture_prefetch
  • texture_flush
  • sprite_flush
  • sprite_flush_multi
  • sprite_prefetch
  • sprite_prefetch_multi
  • draw_flush

Наконец, у нас есть следующие функции, которые в основном используются для отладки проекта и обеспечения эффективного использования текстурной памяти:

  • texture_debug_messages
  • texture_is_ready
  • texturegroup_get_textures
  • texturegroup_get_sprites
  • texturegroup_get_fonts
  • texturegroup_get_tilesets
