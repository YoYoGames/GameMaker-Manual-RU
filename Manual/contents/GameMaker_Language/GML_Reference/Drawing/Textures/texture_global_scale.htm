текстурный_глобальный_масштаб

Эта функция позволяет вам управлять масштабированием текстурных страниц при загрузке из WAD-файла, который является частью конечного исполняемого файла вашей игры. Вводимое значение должно быть степенью двойки и будет работать так:

  • Чтобы установить текстурные страницы в соотношении 1:1, используйте texture_global_scale(1)
  • Чтобы постоянно использовать половину текстур, используйте texture_global_scale(2)
  • Для их четвертования используйте texture_global_scale(4), и т.д..

Таким образом, вы можете лучше контролировать использование памяти текстурных страниц на платформах с низким уровнем памяти. Обратите внимание, что страницы текстур создаются по запросу из WAD, поэтому вы можете либо вызывать эту функцию в начале игры, чтобы загрузить все текстуры в масштабе, либо можно вызывать ее в определенное время для загрузки определенных текстур, и в этом случае вы, вероятно, захотите вызвать функцию draw_texture_flush() перед вызовом этой функции, а затем использовать функции предварительной выборки, чтобы получить нужные страницы текстур в память.

Синтаксис:

texture_global_scale(pow2integer);

=======

texture_global_scale

This function allows you to control the scaling of the texture pages on load from the WAD file that is part of your final game executable.

The input value must be a power of two value and will work such that:

  • To set the texture pages to be 1:1 use texture_global_scale(1)
  • To half textures all the time use texture_global_scale(2)
  • To quarter them use texture_global_scale(4), etc.

In this way you can better control texture page memory usage on platforms with low memory issues. Note that textures pages are created on demand from the WAD, and so you can either call this at the start of the game to load all textures scaled, or you can call it at specific times to load specific textures, in which case you'd probably want to call draw_texture_flush before calling this function, then use the pre-fetch functions to get the correct texture pages into memory.

This function only works for PNG textures. You can change the texture format using Texture Groups.

 

Syntax:

texture_global_scale(pow2integer);

Возвращается:

Пример:

draw_texture_flush();
texture_global_scale(2);
sprite_prefetch(spr_Trees);

Приведенный выше код очистит все текстуры из памяти, затем установит масштабирование текстур на 2 (таким образом, страницы текстур будут вдвое меньше) и, наконец, предварительно получит страницу текстуры, в которой находится sprite "spr_Trees".

Example:

draw_texture_flush();
texture_global_scale(2);
sprite_prefetch(spr_Trees);

The above code will flush all textures from memory, then set the texture scaling to 2 (so texture pages are half size) and then finally pre-fetch the texture page that the sprite spr_Trees is contained within.
