texture_get_uvs

Эта функция возвращает одномерный массив с 4 элементами, представляющими УФ-координаты для заданного указателя текстуры, заполняя массив следующими значениями:

  • [0] = левый
  • [1] = верх
  • [2] = право
  • [3] = низ
  • [4] = количество пикселей, которые компилятор активов обрезал с левой стороны sprites (только для активовsprite ).
  • [5] = количество пикселей, которые компилятор активов обрезал с верхней стороны sprites (только для активовsprite ).
  • [6] = нормализованный процент пиксельных данных от исходной ширины sprites, сохраненных на странице текстуры (только для активовsprite ).
  • [7] = нормализованный процент пиксельных данных от исходной высоты sprites, сохраненных на странице текстуры (только для активовsprite ).

Затем это значение можно использовать в других функциях рисования, особенно в общих функциях 3D и некоторых функциях 2D примитивов, а также в функциях шейдера. Если вам нужен UVS для sprite, то вы можете использовать sprite_get_uvs() и для шрифта, font_get_uvs().

Синтаксис:

texture_get_uvs(texid)

Аргумент Тип Описание
texid Текстура The texture pointer to get the UVS for

Возвращается:

Array (4 - 8 elements)

Пример:

var _tex = surface_get_texture(surf_back);
var _uvs = texture_get_uvs(tex);
var _uvs_left = _uvs[0];
var _uvs_top = _uvs[1];
var _uvs_right = _uvs[2];
var _uvs_bottom = _uvs[3];

Приведенный выше код сначала извлекает текстуру поверхности, хранящуюся в surf_back, а затем получает UV-координаты для этой текстуры. Затем из полученного массива извлекаются левая, верхняя, правая и нижняя UV-координаты и сохраняются в локальных переменных.
