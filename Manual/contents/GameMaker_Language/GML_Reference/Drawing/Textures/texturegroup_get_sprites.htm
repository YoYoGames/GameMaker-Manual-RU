texturegroup_get_sprites

С помощью этой функции вы можете получить sprite индекс каждой из sprites, назначенных текстурным страницам в заданной текстурной группе. Вы вводите строку ID группы текстур (как определено в редакторе групп текстур), и функция возвращает одномерный массив, каждая запись которого содержит индекс sprite для ресурса sprite. Если функция не работает - т.е. указана неверная группа, или группа не имеет текстуры, назначенной ей - то массив будет пуст (длина 0).

Синтаксис:

texturegroup_get_sprites(tex_id);

Аргумент Тип Описание
tex Строка The name of the texture group to check (a string)

Возвращается:

Array

Пример:

var _tex_array = texturegroup_get_sprites( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    show_debug_message("Sprite " + string(i) + " Index:" + string(tex_array[i]));
}

Приведенный выше код получит индексы sprite для группы текстур "MainMenu", а затем отобразит эти идентификаторы в окне консольного вывода.
