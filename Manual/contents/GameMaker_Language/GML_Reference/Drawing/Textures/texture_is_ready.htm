texture_is_ready

Эта функция может быть использована для проверки того, была ли распакована и готова к использованию определенная текстурная страница или группа текстурных страниц была распакована и готова к использованию. Вы передаете уникальный ID текстурной страницы (как возвращается функциями texturegroup_* ) или строку ID текстурной группы (как определено в Texture Group Editor), и функция возвращает true, если они были распакованы, или false в противном случае.

Синтаксис:

texture_is_ready(tex_id);

Аргумент Тип Описание
tex_id Указатель текстуры или строка The texture page pointer or a texture group name (a string)

Возвращается:

Boolean

Пример:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if !texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

Приведенный выше код получит идентификаторы текстурных страниц для группы текстур "MainMenu", затем проверит, распакованы ли они, и если нет, то они будут префетишированы в VRAM.
