texture_get_height

Возвращает высоту текстуры с заданным id, который всегда является значением в диапазоне 0 - 1. Это значение можно использовать при отображении текстур на модели или примитивы.

=======

texture_get_height

This function returns the height of the texture with the given ID, which is always a value within the range 0 - 1. This can then be used when mapping textures to models or primitives.

 

Syntax:

texture_get_height(tex);
