texture_flush

Эта функция может использоваться для "прошивки" текстурной страницы или группы текстурных страниц, т.е. для удаления их из VRAM, когда они больше не нужны. Вы можете указать уникальный ID текстурной страницы (возвращаемый функциями texturegroup_* ), чтобы прошить одну страницу, или указать имя группы текстур (определенное в редакторе групп текстур), чтобы прошить все текстурные страницы в группе.

Синтаксис:

texture_flush(tex_id);

Аргумент Тип Описание
tex_id Указатель текстуры или строка The texture page pointer or a texture group name (a string)

Возвращается:

N/A

Пример:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i)
{
    texture_flush(_tex_array[i]);
}

Приведенный выше код прошьет все страницы текстур под группой текстур "MainMenu".
