texture_get_width

Возвращает ширину текстуры с заданным id, который всегда является значением в диапазоне 0 - 1. Это значение можно использовать при отображении текстур на модели или примитивы.

Синтаксис:

texture_get_width(tex);

Аргумент Тип Описание
tex Указатель текстуры The texture page asset pointer to use

Возвращается:

Real

Пример:

tex_w = texture_get_width(sprite_get_texture(spr_Model_tex, 0));

Приведенный выше код получит ширину текстуры, взятой из актива sprite.
