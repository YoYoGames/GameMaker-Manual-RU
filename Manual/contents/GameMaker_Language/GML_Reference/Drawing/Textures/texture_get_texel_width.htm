texture_get_texel_width

Тексель, или элемент текстуры, - это фундаментальная единица текстурного пространства, используемая в компьютерной графике. Текстуры представлены массивами текселей, так же как изображения представлены массивами пикселей, и эта функция возвращает ширину одного текселя из текстурной страницы используемого актива изображения.

Синтаксис:

texture_get_texel_width(tex);

Аргумент Тип Описание
tex Указатель текстуры The texture page asset pointer to use

Возвращается:

Real

Пример:

var tex = sprite_get_texture(sprite_index, 0);
tex_w = texture_get_texel_width(tex);
tex_h = texture_get_texel_height(tex);

Приведенный выше код получит ширину и высоту текселя текстуры, взятой из актива sprite.
