texturegroup_get_textures

Эта функция может использоваться для получения идентификаторов текстурных страниц отдельных страниц, составляющих группу текстур. Вы передаете строку ID группы текстур (как определено в редакторе групп текстур), и функция возвращает массив 1D, где каждая запись в массиве - это ID одной страницы текстуры. Если функция не работает - т.е. указана неверная группа, или группа не имеет назначенной ей текстуры - массив будет пуст (длина 0).

Синтаксис:

texturegroup_get_textures(tex_id);

Аргумент Тип Описание
tex_id Строка The name of the texture group to check (a string)

Возвращается:

Array of Texture Pointers

Пример:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

Приведенный выше код получит идентификаторы текстурных страниц для группы текстур "MainMenu", затем проверит, распакованы ли они, и если да, то поместит их в VRAM.
