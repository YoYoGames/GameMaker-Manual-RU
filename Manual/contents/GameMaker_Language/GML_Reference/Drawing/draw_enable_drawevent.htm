draw_enable_drawevent

С помощью этой функции вы можете включить (true) или выключить (false) событие рисования для всех экземпляров в игре, таким образом, вы получаете контроль над тем, как и когда рисуются объекты, что полезно, если вы хотите реализовать технику "пропуска кадров". Обратите внимание, что это не просто предотвращает рисование экземпляров на экране, а полностью подавляет событие draw, что означает, что следует быть осторожным, поскольку любая игровая логика, присутствующая в этом событии, также не будет запущена.

Важно понимать, что если вы вызовете эту функцию в самом начале игры, до того, как будет отрисован начальный кадр (т.е.: событие Create первого объекта в первой комнате игры), то окно игры не будет отрисовано. Это может быть полезно для многопользовательских проектов, требующих выделенного сервера, которому не нужно ничего рендерить, однако имейте в виду, что это не делает игру безголовой, так как GameMaker не поддерживает этого - поэтому вы не сможете запустить ее на сервере, не имеющем графического интерфейса.

Синтаксис:

draw_enable_drawevent(enable);

АргументТип Описание
Enable Set to true или false.

Возвращается:

Пример:

frame_skip ++;
if (frame_skip mod 5) == 0
{
    draw_enable_drawevent(true);
}
else
{
    draw_enable_drawevent(false);
}

Приведенный выше код проверяет переменную и, если она возвращает значение true, включает событие draw, в противном случае событие draw отключается.
