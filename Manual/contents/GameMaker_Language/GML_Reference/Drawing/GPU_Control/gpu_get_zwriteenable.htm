gpu_get_zwriteenable

Эта функция может быть использована для получения информации о том, включена ли z-запись (функция возвращает true) или нет (функция возвращает false). По умолчанию значение z-writing включено, поэтому функция возвращает true.

Синтаксис:

gpu_get_zwriteenable();

Возвращается:

Пример:

if gpu_get_zwriteenable() == false
{
    gpu_set_zwriteenable(true);
}

Приведенный выше код проверяет, включена ли z-запись или нет, и если она отключена, то снова включается.
