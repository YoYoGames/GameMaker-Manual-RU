gpu_set_ztestenable

Эта функция может быть использована для включения или выключения тестирования z-буфера, что влияет на то, как будут отрисовываться объекты (в общем случае это полезно только при работе с 3D-проектами). По сути, по умолчанию, когда z-тестирование выключено и у вас есть два объекта, рисующих в одно и то же пространство, оба объекта будут отрисованы независимо от того, будет ли один из них перерисовывать другой, что приведет к ненужным вызовам рисования. Если включить эту функцию, то z-буфер проверяется на то, будет ли объект "видимым", и не отрисовывается, если это не так. нет. Обратите внимание, что это поведение по умолчанию, но вы можете изменить его, изменив тип сравнения, используемого для тестирования z-буфера (см. функцию gpu_set_zfunc().

По умолчанию тестирование z-буфера выключено (false).

Синтаксис:

gpu_set_ztestenable(enable);

АргументТип Описание
enable Enable or disable z-buffer testing (true или false).

Возвращается:

Пример:

gpu_set_ztestenable(true);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

Приведенный выше код включает проверку z-буфера для рисования фона sprite, а затем снова выключает ее для продолжения рисования.
