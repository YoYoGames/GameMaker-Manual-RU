gpu_set_zfunc

Эта функция может использоваться для установки режима сравнения тестирования z-буфера (см. gpu_set_ztestenable() для получения дополнительной информации). Доступными для использования значениями являются любые из следующих констант (значение по умолчанию - cmpfunc_lessequal):

Постоянная Описание
cmpfunc_never Никогда
cmpfunc_less Меньше
cmpfunc_equal Равный
cmpfunc_lessequal Меньше или равно
cmpfunc_greater Большая
cmpfunc_notequal Не равны
cmpfunc_greaterequal Больше или равно
cmpfunc_always Всегда

Синтаксис:

gpu_set_zfunc(cmp_func);

Аргумент Тип Описание
cmp_func Функция сравнения Постоянная The comparison mode to use (see list above)

Возвращается:

N/A

Пример:

gpu_set_ztestenable(true);
gpu_set_zfunc(cmpfunc_always);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

Приведенный выше код включает проверку z-буфера и устанавливает его режим сравнения перед рисованием фона sprite, а затем снова выключает его для продолжения рисования.
