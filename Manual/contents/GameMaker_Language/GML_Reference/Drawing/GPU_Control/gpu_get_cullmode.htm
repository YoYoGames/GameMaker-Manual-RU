gpu_get_cullmode

Эта функция может быть использована для получения режима выбраковки задней поверхности. Возвращаемое значение будет одной из следующих констант (значение по умолчанию cull_noculling):

Режим выбраковки Постоянный
Постоянная Описание
cull_noculling Выбраковка не будет проводиться
cull_clockwise Все треугольники по часовой стрелке будут удалены
cull_counterclockwise Все треугольники против часовой стрелки будут удалены

Синтаксис:

gpu_get_cullmode();

Возвращается:

Culling Mode Constant (see above for constants)

Пример:

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

Приведенный выше код получает текущий режим cull и, если он не cull_clockwise, то устанавливается на эту константу.
