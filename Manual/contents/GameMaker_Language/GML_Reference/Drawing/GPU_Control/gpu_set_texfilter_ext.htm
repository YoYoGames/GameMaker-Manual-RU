gpu_set_texfilter_ext

Эта функция может использоваться для установки линейной интерполяции для одного "слота" сэмплера при использовании шейдеров в GameMaker. Если эта функция включена (true), текстура сэмплера будет сглажена, а если отключена (false), то изображения будут рисоваться на основе ближайшего пикселя. По умолчанию используется значение, установленное в Глобальных игровых параметрах вашей игры, или значение, установленное с помощью функции gpu_set_texfilter().

ПРИМЕЧАНИЕ: Эта настройка будет отменена значением, установленным при вызове функции gpu_set_texfilter().

ПРИМЕЧАНИЕ: На цели HTML5 эта функция будет работать только при включенном WebGL.

Синтаксис:

gpu_set_texfilter_ext(sampler_id, enable);

АргументТип Описание
sampler_id The sampler id from the shader.
enable Enable or disable texture filtering (true / false)

Возвращается:

Пример:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
    gpu_set_texfilter_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

Приведенный выше код проверяет, включена или выключена фильтрация текстур для определенного ID сэмплера (хранится в локальной переменной), и переключает ее соответствующим образом.
