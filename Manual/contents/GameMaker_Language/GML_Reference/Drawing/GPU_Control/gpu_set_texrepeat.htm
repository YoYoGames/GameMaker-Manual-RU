gpu_set_texrepeat

Эта функция может быть использована для указания GameMaker, должны ли текстуры повторяться (true) или нет (false) при использовании для моделей и примитивов. Обычно ширина и высота текстуры лежат в диапазоне 0 - 1, но если вы укажете значение больше 1, то по умолчанию остаток не отрисовывается (вся текстура будет отрисована с "пустым" участком, соответствующим проценту больше 1, который используется. Таким образом, значение 2 для ширины или высоты текстуры нарисует текстуру на половине пространства, а вторая половина останется пустой). Однако при использовании этой функции для установки параметра repeat в true текстура автоматически повторяется для всех моделей и примитивов. Скорее всего, эти функции используются для повторения текстуры в 3D, но для того, чтобы они работали и не вытягивали изображения с остальной страницы текстуры, используемый sprite должен быть помечен как "отдельная страница текстуры" в редакторе Sprite.

ПРИМЕЧАНИЕ: Sprites и фоны всегда рисуются без повторений, поэтому после рисования sprite или фона это значение сбрасывается на false. Такое поведение по умолчанию не влияет на настройки сцены текстуры (т.е. вызов функции влияет, а внутреннее значение false - нет).

ПРИМЕЧАНИЕ: Эта настройка отменяет любые повторы стадий текстуры, установленные для шейдеров при использовании функции gpu_set_texrepeat_ext().

Синтаксис:

gpu_set_texrepeat(enable);

АргументТип Описание
enable Enable or disable texture filtering (true / false)

Возвращается:

Пример:

if gpu_get_texrepeat()
{
    gpu_set_texrepeat(false);
}
else
{
    gpu_set_texrepeat(true);
}

Приведенный выше код проверяет, включено или выключено повторение текстуры, и переключает его соответствующим образом.
