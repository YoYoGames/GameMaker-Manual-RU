gpu_set_colourwriteenable

С помощью этой функции можно включить или выключить цветовые каналы и альфа-канал для всего дальнейшего рисования. Например, вы можете использовать эту функцию для создания поверхностей только с альфа-каналом (отключите запись RGB перед использованием поверхности) или для создания специальных эффектов при рисовании на экране. Значение по умолчанию для каждого из компонентов равно true, и может быть предоставлено либо как четыре уникальных аргумента, либо как 4-элементный одномерный массив со следующими элементами, которые будут либо true (включен), либо false (отключен):

  • [0] = Красный канал включен/выключен
  • [1] = Зеленый канал включен/выключен
  • [2] = Синий канал включен/выключен
  • [3] = Альфа-канал включен/выключен

Синтаксис:

gpu_set_colorwriteenable(red, green, blue, alpha);

Аргумент Тип Описание
red Булево Enable/disable the red channel
green Булево Enable/disable the green channel
blue Булево Enable/disable the blue channel
alpha Булево Enable/disable the alpha channel

Возвращается:

N/A

Пример:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

Приведенный выше код получает текущие значения записи цвета, а затем устанавливает альфа-компонент на false.
