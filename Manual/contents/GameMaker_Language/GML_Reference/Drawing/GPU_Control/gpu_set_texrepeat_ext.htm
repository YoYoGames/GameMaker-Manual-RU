gpu_set_texrepeat_ext

Эта функция позволяет установить, повторяет ли один "слот" сэмплера заданную текстуру при использовании шейдеров в GameMaker. Если установить значение true, текстура будет повторяться, если uv-координаты находятся вне диапазона 0-1, в то время как значение false означает отсутствие повторения. Скорее всего, эти функции будут использоваться для повторения текстуры в 3D, но для того, чтобы они работали и не вытягивали изображения из остальной страницы текстуры, используемый sprite должен быть помечен как "отдельная страница текстуры" в редакторе Sprite.

ПРИМЕЧАНИЕ Эта настройка будет отменена значением, установленным при вызове функции gpu_set_texrepeat().

Синтаксис:

gpu_set_texrepeat_ext(sampler_id, enable);

АргументТип Описание
sampler_id The sampler id from the shader.
enable Enable or disable texture filtering (true / false)

Возвращается:

Пример:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if (gpu_get_texfilter_ext(s_tex))
{
    gpu_set_texfilter_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

Приведенный выше код проверяет, включена или выключена фильтрация текстур для определенного ID сэмплера (хранится в локальной переменной), и переключает ее соответствующим образом.
