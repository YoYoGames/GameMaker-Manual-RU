gpu_pop_state

Эта функция извлекает предыдущее состояние GPU из стека и применяет его. См. gpu_push_state() для получения дополнительной информации.

Синтаксис:

gpu_pop_state();

Возвращается:

Пример:

gpu_push_state();
gpu_set_blendmode(bm_add);
gpu_set_blendenable(false);
gpu_set_cullmode(true);
with (obj_Effect_Parent)
{
    draw_self();
}
gpu_pop_state();

Приведенный выше код "сохранит" текущее состояние GPU в стеке, затем изменит определенные настройки GPU и отрисует группу экземпляров, после чего сбросит состояние GPU на прежнее.
