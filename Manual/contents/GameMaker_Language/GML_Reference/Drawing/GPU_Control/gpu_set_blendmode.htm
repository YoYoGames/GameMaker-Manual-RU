gpu_set_blendmode

Когда GameMaker рисует пиксель, есть исходный цвет (цвет пикселя, который мы собираемся нарисовать) и цвет назначения (цвет, который уже есть в пикселе, к которому мы рисуем), поэтому при определении нового цвета пикселя исходный и конечный цвета рассчитываются в соответствии с выбранным режимом наложения. Каждый компонент цвета хранится как значение с плавающей точкой между 0 и 1, а новый цвет вычисляется путем умножения каждого компонента исходного цвета на некоторый коэффициент и умножения каждого компонента цвета назначения на другой коэффициент, а затем сложения результатов по компонентам.

Эта функция позволяет установить режим наложения четырьмя основными способами, используя следующие константы:

Постоянная Описание Расширенный режим наложения
bm_normal Обычное смешивание (режим смешивания по умолчанию). (bm_src_alpha, bm_inv_src_alpha)
bm_add Аддитивное смешивание. Добавляются значения светимости светлых областей. (bm_src_alpha, bm_one)
bm_subtract Вычитающее смешивание. Значения светимости светлых областей вычитаются. (bm_zero, bm_inv_src_colour)
bm_max Максимальное смешивание. Аналогично аддитивному смешиванию. (bm_src_alpha, bm_inv_src_colour)

Как видно из приведенной выше таблицы, эти четыре режима наложения на самом деле являются композициями расширенных режимов наложения, которые можно найти на странице с описанием gpu_set_blendmode_ext().

Чтобы максимально использовать режимы наложения и понять, как они работают и как влияют на конечное изображение, выводимое на экран, мы рекомендуем вам ознакомиться со следующим руководством:

  • Руководство по использованию блендмодов

Синтаксис:

gpu_set_blendmode(mode);

Аргумент Тип Описание
mode Режим наложения Постоянный The blend mode to use (see the table above)

Возвращается:

N/A

Пример:

gpu_set_blendmode(bm_add);
draw_circle_colour(100, 100, 50, c_white, c_black, 0);
gpu_set_blendmode(bm_normal);

Это превратит черный цвет в прозрачность, создавая эффект "свечения" за счет того, что белый цвет сильно выражен снаружи и постепенно ослабевает от центра круга. Режим наложения сбрасывается после рисования круга, чтобы аддитивное смешивание не применялось ко всему остальному после него.
