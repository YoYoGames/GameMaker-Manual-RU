gpu_set_alphatestref

Эта функция устанавливает опорное значение для альфа-тестирования (если оно было включено с помощью кнопки gpu_set_alphatestenable()). Это порог "отсечения", при котором пиксели с альфой не будут отрисовываться.

В принципе, если у вас это значение установлено в 0, а линейная интерполяция включена (либо в Опциях игры, либо с помощью функции gpu_set_texfilter()), вы все равно можете получить визуальные ошибки на краях перекрывающихся объектов. Это происходит потому, что у вас могут быть почти полностью прозрачные области, которые блокируют то, что нарисовано за ними, но если вы установите это опорное значение, например, 254 (т.е. на единицу ниже максимального), это гарантирует, что будут нарисованы только 100% непрозрачные пиксели. Вы также можете отключить интерполяцию текстур, что обычно позволяет использовать значения по умолчанию без проблем и без необходимости использовать эту функцию вообще, однако многое зависит от исходного произведения искусства.

ПРИМЕЧАНИЕ: Это расширенная функция рисования, поэтому если вы не уверены, нужна ли она вам или нет, или не знаете точно, что она делает, вам, вероятно, лучше не использовать ее, так как она может серьезно испортить то, как рисуется ваша игра.

Синтаксис:

gpu_set_alphatestref(val);

АргументТип Описание
val Sets the reference value for alpha testing from 0 to 255 (default value is 0)

Возвращается:

Пример:

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

Приведенный выше код проверит, включено ли альфа-тестирование, и если нет, то включит альфа-тестирование и установит порог теста на 128 (будут отрисованы только пиксели с альфой более 0,5).
