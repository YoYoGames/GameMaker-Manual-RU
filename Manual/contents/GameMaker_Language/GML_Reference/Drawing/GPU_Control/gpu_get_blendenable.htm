gpu_get_blendenable

Эта функция может быть использована для получения состояния альфа-клинки. Если она возвращает значение true, то альфа-смешение включено, а если false, то отключено. По умолчанию она включена, поэтому функция возвращает следующее значение true

Синтаксис:

gpu_get_blendenable();

Возвращается:

Пример:

if gpu_get_blendenable() == false
{
    gpu_set_blendenable(true);
}

Приведенный выше код проверяет состояние альфа-смешивания, и если оно отключено, то снова включается.
