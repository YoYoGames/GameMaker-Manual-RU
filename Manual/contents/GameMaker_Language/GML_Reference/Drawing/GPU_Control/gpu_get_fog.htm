gpu_get_fog

Эта функция может быть использована для получения настроек тумана. Функция возвращает 4-элементный массив 1D со следующей информацией:

  • [0] = включение переключения ( булево значение, либо true, либо false), по умолчанию false
  • [1] = Постоянный цвет, по умолчанию c_black
  • [2] = начальное расстояние(Real), по умолчанию 0
  • [3] = конечное расстояние(Real), по умолчанию 1

Обратите внимание, что вы можете изменить эти значения и передать полный массив в функцию gpu_set_fog() функции в качестве единственного аргумента.

Синтаксис:

gpu_get_fog();

Возвращается:

Array (4 elements only; see above for details)

Пример:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

Приведенный выше код получает текущие настройки тумана, затем устанавливает элемент color массива на c_red, после чего снова устанавливает туман, используя измененный массив.
