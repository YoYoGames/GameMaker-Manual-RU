gpu_set_blendenable

Эта функция может использоваться для включения и выключения альфа-смешения. В принципе, если установить значение false, все рисуемые изображения будут на 100% непрозрачными, то есть все прозрачные или полупрозрачные области sprite или фона будут видны как сплошной цвет. будут видны как сплошной цвет. Хорошей идеей является отключение альфа-смешивания при любой возможности (особенно при разработке для мобильных устройств), так как это значительно увеличивает скорость отрисовки.

Синтаксис:

gpu_set_blendenable(enable);

АргументТип Описание
enable Enable or disable alpha blending value (true или false).

Возвращается:

Пример:

gpu_set_blendenable(false);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_blendenable(true);

Приведенный выше код отключает альфа-смешивание для рисования фона sprite, а затем снова включает его для продолжения рисования.
