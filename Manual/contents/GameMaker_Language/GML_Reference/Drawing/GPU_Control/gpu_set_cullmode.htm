gpu_set_cullmode

Эта функция используется для установки режима отбраковки задней стороны. Многоугольник (примитив) имеет переднюю и заднюю стороны. Передней считается та сторона, где вершины определены в порядке против часовой стрелки. Обычно рисуются обе стороны, но если вы создаете замкнутую форму (например, пирамиду), то это пустая трата вычислительной мощности, поскольку обратная сторона треугольника никогда не будет видна. Именно в таких случаях можно включить отбраковку обратной стороны, так как это экономит примерно половину времени рисования, хотя это означает, что вы должны определить многоугольники правильным образом, чтобы избежать проблем.

Используемое значение cull mode должно быть одной из следующих констант (значение по умолчанию - cull_noculling):

Постоянная Описание
cull_noculling Выбраковка не будет проводиться
cull_clockwise Все треугольники по часовой стрелке будут удалены
cull_counterclockwise Все треугольники против часовой стрелки будут удалены

Синтаксис:

gpu_set_cullmode(cullmode);

Аргумент Тип Описание
cullmode Режим выбраковки Постоянный The cull mode to use (see list above)

Возвращается:

N/A

Пример:

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

Приведенный выше код получает текущий режим cull и, если он не cull_clockwise, то устанавливается на эту константу.
