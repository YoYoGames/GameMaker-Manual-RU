gpu_set_fog

Эта функция может использоваться для включения или отключения рисования тумана. Туман можно использовать в 3D-играх, чтобы объекты вдали выглядели блеклыми или даже невидимыми, что помогает создать атмосферу и скрыть тот факт, что вы не рисуете объекты, находящиеся далеко.

Вы задаете, включен ли он (true) или выключен (false), цвет, который туман должен использовать для смешивания, а также начальное и конечное расстояния рисования. Значения расстояния - это, по сути, значения глубины в пикселях, и они относятся к положению камеры.

Например, позиция 0 является ближайшей к камере, и с каждым увеличением она становится все дальше. По умолчанию камера вида располагается на глубине -16000, поэтому если вы хотите, чтобы туман начинался на глубине 0 и заканчивался на глубине 1000, установите начальное расстояние 16000, а конечное - 17000.

Функция может принимать четыре отдельных аргумента (приведенных выше) или один массив со следующей структурой (в примере кода ниже показан этот метод):

  • [0] = включение переключения (булево значение, либо true, либо false), по умолчанию false
  • [1] = цвет (реальный), по умолчанию c_black
  • [2] = начальное расстояние (реальное), по умолчанию 0
  • [3] = конечное расстояние (реальное), по умолчанию 1

Синтаксис:

gpu_set_fog(enable, colour, start, end);

Аргумент Тип Описание
enable Булево Enable or disable fog
colour Цвет The fog colour
start Реально The distance to start applying fog from (relative to the camera)
end Реально The distance to end the fog (relative to the camera)

Возвращается:

N/A

Пример:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

Приведенный выше код получает текущие настройки тумана, затем устанавливает элемент color массива на c_red, после чего снова устанавливает туман, используя измененный массив.
