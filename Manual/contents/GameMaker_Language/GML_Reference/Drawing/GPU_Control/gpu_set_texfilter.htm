gpu_set_texfilter

Эта функция может быть использована для настройки фильтрации текстур (линейной интерполяции) всех изображений, рисуемых на игровом экране. Если эта функция включена (true), то все текстуры будут сглаживаться при отрисовке (сюда входит и sprites, так как они тоже считаются текстурами), что означает, что при масштабировании или перемещении, если соотношение пикселей не 1:1, будет наблюдаться "размазывание" по различным пикселям, из-за чего изображения могут выглядеть размытыми в зависимости от используемого художественного стиля. Если этот параметр отключен (false), то при масштабировании или перемещении изображения будут отрисовываться по ближайшему пикселю, что может привести к появлению "блочных" изображений. Значение по умолчанию - false, его также можно изменить в глобальных опциях игры для отдельных целевых платформ.

ПРИМЕЧАНИЕ:Эта настройка отменяет любую интерполяцию текстурной стадии, установленную для шейдеров с помощью функции gpu_set_texfilter_ext().

ПРИМЕЧАНИЕ: На цели HTML5 эта функция будет работать только при включенном WebGL.

Синтаксис:

gpu_set_texfilter(enable);

АргументТип Описание
enable Enable or disable texture filtering (true / false)

Возвращается:

Пример:

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

Приведенный выше код проверяет, включена или выключена фильтрация текстур, и переключает ее соответствующим образом.
