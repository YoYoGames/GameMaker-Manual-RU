gpu_get_blendmode_dest

Эта функция может быть использована для получения текущего коэффициента расширенного режима наложения назначения, используемого для рисования. Возвращаемое значение будет одной из следующих констант ("s" означает значение, взятое из источника, а "d" означает значение из назначения):

Постоянная Коэффициент смешивания (красный, зеленый, синий, альфа)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (As, As, As, As, As)
bm_inv_src_alpha (1-Ас, 1-Ас, 1-Ас, 1-Ас, 1-Ас)
bm_dest_alpha (Ad, Ad, Ad, Ad, Ad)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1), где f = min(As, 1-Ad)

Синтаксис:

gpu_get_blendmode_dest();

Возвращается:

Blend Mode Factor Constant (see above table)

Пример:

var bm;
bm[0] = gpu_get_blendmode_src();
bm[1] = gpu_get_blendmode_dest();
gpu_set_blendmode_ext_sepalpha(bm[0], bm[1], bm_inv_src_alpha, bm_inv_dest_colour);

Приведенный выше код создает локальный массив и получает текущие коэффициенты смешивания источника и назначения. Этот массив затем используется для манипулирования альфа-компонентом коэффициентов смешивания.
