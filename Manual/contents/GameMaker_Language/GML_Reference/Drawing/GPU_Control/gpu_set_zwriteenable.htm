gpu_set_zwriteenable

Эта функция может использоваться для включения (true) и выключения (false) упорядочивания глубины с помощью z-буфера. При рендеринге экземпляра в GameMaker глубина сгенерированного пикселя (его координата z) сохраняется в z-буфере, затем, если на тот же пиксель должен быть отрендерен другой экземпляр сцены, эти две глубины сравниваются (z-тестирование - см. gpu_set_ztestenable()), и та, что ближе к наблюдателю, сохраняется в z-буфере, заменяя старую. Однако это не всегда то, что вы хотите, и бывают моменты, когда вы хотите, чтобы что-то было нарисовано поверх всего остального, независимо от того, каково его значение z. В таких случаях вы можете использовать эту функцию, чтобы отключить запись в z-буфер и таким образом заставить то, что рисуется, рисоваться поверх всего остального. Обратите внимание, что описание, приведенное здесь, относится к режиму сравнения z-тестирования по умолчанию, но его можно изменить.

По умолчанию z-буферизация включена (true).

Синтаксис:

gpu_set_zwriteenable(enable);

АргументТип Описание
enable The comparison mode to use (see list above)

Возвращается:

Пример:

gpu_set_zwriteenable(false);
with (obj_Menu_Containers)
{
    draw_self();
}
gpu_set_zwriteenable(true);

Приведенный выше код отключает запись в z-буфер, рисует несколько объектов, а затем снова включает его для продолжения рисования.
