gpu_get_texrepeat_ext

С помощью этой функции вы можете проверить, включено ли повторение текстуры (возвращается true) или нет (возвращается false) для данной текстуры шейдерного сэмплера.

Синтаксис:

gpu_get_texrepeat_ext(sampler_id);

АргументТип Описание
sampler_id The sampler id from the shader.

Возвращается:

Пример:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if !gpu_get_texrepeat_ext(s_tex)
{
    gpu_set_texrepeat_ext(s_tex, true);
}

Приведенный выше код проверяет, выключена ли фильтрация текстур для определенного ID сэмплера (хранится в локальной переменной), и включает ее, если нет.
