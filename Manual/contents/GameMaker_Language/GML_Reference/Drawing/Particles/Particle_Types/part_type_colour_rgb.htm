part_type_colour_rgb

С помощью этой функции вы можете задать сочетание красного, зеленого и синего цветов для всех создаваемых частиц данного типа. Вы задаете минимальное значение и максимальное значение для каждого из трех компонентов, и созданные частицы будут иметь случайный цвет, основанный на заданном диапазоне параметров. Все значения должны быть в диапазоне от 0 до 255.

Синтаксис:

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

Аргумент Тип Описание
ind Идентификатор типа частицы The index of the particle type to change.
rmin Реально The minimum the final colour's red component can be.
rmax Реально The maximum the final colour's red component can be.
gmin Реально The minimum the final colour's green component can be.
gmax Реально The maximum the final colour's green component can be.
bmin Реально The minimum the final colour's blue component can be.
bmax Реально The maximum the final colour's blue component can be.

Возвращается:

N/A

Пример:

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

Приведенный выше код устанавливает, что каждая частица, испускаемая типом частицы, проиндексированным в глобальной переменной "Blood_Part", имеет только различные оттенки красного цвета.
