тип_части_направление

Эта функция используется для определения направления частицы при ее создании, а также может использоваться для того, чтобы частица увеличивала или уменьшала свое направление в течение жизни. Минимальное и максимальное значения направления по умолчанию равны 0, что означает, что частица будет двигаться вправо без изменений, но эти значения можно изменить, чтобы ограничить направление движения в соответствии со стандартными направлениями GameMaker: 0 - вправо, 90 - вверх, 180 - влево и 270 - вниз. Если вы установите одинаковые значения, частицы будут двигаться только в одном направлении.

Вы также можете задать значение приращения, которое будет добавлять (если это положительное число) или вычитать (если это отрицательное число) количество градусов к направлению в течение срока службы. Это значение может составлять минимум (+/-) 0,01.

Наконец, вы можете задать коэффициент "вихляния". Это значение, которое будет случайным образом добавляться или вычитаться к направлению на каждом шаге жизни частиц. Очевидно, что большие значения более выражены, чем меньшие, и это значение может быть даже отрицательным с максимальным диапазоном от -20 до 20.

Синтаксис:

part_type_direction(ind, dir_min, dir_max, dir_incr, dir_wiggle);

Аргумент Тип Описание
ind Идентификатор типа частицы The index of the particle type to change.
dir_min Реально The minimum direction the particle can start at.
dir_max Реально The maximum direction the particle can start at.
dir_incr Реально How much the particle direction should increase or decrease per step.
dir_wiggle Реально How much should randomly be added or subtracted from the particle's direction per step.

Возвращается:

N/A

Пример:

part_type_shape(particle1, pt_shape_pixel);
part_type_size(particle1, 0.19, 0.79, 0.01, 0);
part_type_scale(particle1, 5, 5);
part_type_colour3(particle1, 32768, 8454016, 8454016);
part_type_alpha3(particle1, 0, 0.80, 0);
part_type_speed(particle1, 8, 10, -0.50, 1);
part_type_direction(particle1, 0, 359, 1, 20);
part_type_gravity(particle1, 0, 270);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 50);

Приведенный выше код установит различные значения частиц, включая направление, которое будет случайным значением от 0 до 359 градусов. Оно также будет увеличиваться на 1 градус с каждым шагом, и к нему будет добавляться случайная величина от 0 до 20 с каждым шагом.
