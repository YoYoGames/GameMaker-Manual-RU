тип_детали_масштаб

Эта функция устанавливает горизонтальный и вертикальный масштаб частицы до того, как будут произведены другие изменения размера. Эта функция работает на соотношении размеров sprite, поэтому значение 1 (значение по умолчанию) - это соотношение 1:1, то есть 0,5 будет половиной, а 2 - удвоением.

Синтаксис:

part_type_scale(ind, xscale, yscale);

Аргумент Тип Описание
ind Идентификатор типа частицы The index of the particle type to change.
xscale Реально The horizontal scaling of the base particle.
yscale Реально The vertical scaling of the base particle.

Возвращается:

N/A

Пример:

part_type_shape(particle1, pt_shape_flare);
part_type_size(particle1, 2, 2, 0.10, 1);
part_type_scale(particle1, 8, 0.50);
part_type_colour3(particle1, 8388608, 4227327, 8454143);
part_type_alpha3(particle1, 0.10, 0.10, 0);
part_type_speed(particle1, 0, 1, 0.18, 20);
part_type_direction(particle1, 0, 359, 0, 0);
part_type_orientation(particle1, 0, 0, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 40);

Приведенный выше код установит различные значения частиц, включая масштаб, который устанавливает xscale в восемь раз больше размера, а yscale в половину размера.
