тип_детали_чистый

С помощью этой функции вы можете "сбросить" частицу, вернув все значения для каждой из функций, относящихся к частице (жизнь, цвет, альфа, ориентация и т.д.), к значениям по умолчанию. Обратите внимание, что эта функция не удаляет с экрана частицы, видимые в данный момент в комнате, для этого следует использовать функцию part_particles_clear().

Синтаксис:

part_type_clear(ind);

Аргумент Тип Описание
ind Идентификатор типа частицы The index of the particle type to clear.

Возвращается:

N/A

Пример:

part_type_clear(global.explode_part);

Приведенный выше код очистит тип частицы, указанный в глобальной переменной "explode_part", до значений по умолчанию.
