тип_части_уничтожить

С помощью этой функции вы можете удалить из игры частицу указанного типа. При использовании этой функции все частицы указанного типа исчезнут из комнаты, а сама частица будет удалена из памяти, поэтому эту функцию следует использовать только тогда, когда частица больше не нужна.

Синтаксис:

part_type_destroy(ind);

Аргумент Тип Описание
ind Идентификатор типа частицы The index of the particle type to destroy.

Возвращается:

N/A

Пример:

if part_particles_count(smoke_sys) = 0
{
    part_type_destroy(smoke_part);
    part_system_destroy(smoke_sys);
    instance_destroy();
}
else alarm[0] = 1;

Приведенный выше код проверяет систему частиц на предмет того, видны ли в данный момент в комнате какие-либо частицы, и если нет, то частицы, система и экземпляр уничтожаются.
