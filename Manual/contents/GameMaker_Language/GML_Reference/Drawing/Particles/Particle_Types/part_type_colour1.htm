тип_детали_цвет1

Эта функция используется для установки типа частицы, чтобы она была одного цвета в течение всего времени жизни каждой отдельной частицы.

Синтаксис:

part_type_colour1(ind, colour1);

Аргумент Тип Описание
ind Идентификатор типа частицы The index of the particle type to change.
colour1 Цвет The single colour to make the particle type.

Возвращается:

N/A

Пример:

part_type_colour1(global.Snow_Part, c_white);

Приведенный выше код установит, что все частицы, созданные по типу частиц, индексированных в глобальной переменной "Snow_Part", будут только белыми.
