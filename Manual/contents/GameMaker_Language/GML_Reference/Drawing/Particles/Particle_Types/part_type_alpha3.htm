тип_части_альфа3

Эта функция может быть использована для установки градиента с тремя значениями альфа (прозрачности) для каждой созданной частицы данного типа. Первая альфа - это та, с которой будут начинаться все частицы, вторая альфа - это та, которая будет смешиваться до половины времени жизни частицы, а третья альфа - это та, с которой частица будет заканчиваться. В течение жизни частицы будет происходить плавное изменение градиента от одной альфы к другой.

Синтаксис:

part_type_alpha3(ind, alpha1, alpha2, alpha3);

Аргумент Тип Описание
ind Идентификатор типа частицы The index of the particle type to change.
alpha1 Реально The starting alpha of the particle.
alpha2 Реально The halfway point alpha of the particle.
alpha3 Реально The ending alpha of the particle.

Возвращается:

N/A

Пример:

part_type_alpha3( part_Health, 0.5, 1, 0);

Приведенный выше код заставит все созданные частицы типа, указанного в переменной "part_Health", иметь альфа-смесь от 0.5 до 1 до 0 в течение их жизни.
