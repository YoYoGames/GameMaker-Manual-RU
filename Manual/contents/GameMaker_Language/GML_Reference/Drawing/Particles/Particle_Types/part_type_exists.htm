тип_детали_существует

С помощью этой функции вы можете проверить, существует ли в игре данный тип частиц с индексом или нет. Обратите внимание, что если проверяемая переменная является неинициализированной переменной (которой тип частиц в противном случае присвоил бы свой индекс), это приведет к ошибке.

Синтаксис:

part_type_exists(ind);

Аргумент Тип Описание
ind Идентификатор типа частицы The index of the particle type to check for.

Возвращается:

Boolean

Пример:

if !part_type_exists(global.p1)
{
    global.p1 = part_type_create();
}

Приведенный выше код проверяет, хранит ли глобальная переменная "p1" индекс типа частицы, и если нет, то создает его.
