тип_части_альфа1

Эта функция используется для установки типа частиц, чтобы они имели одно значение альфа (прозрачность) в течение всего времени жизни каждой отдельной частицы, и это значение может быть от 0 (прозрачный) до 1 (непрозрачный).

Синтаксис:

part_type_alpha1(ind, alpha1);

Аргумент Тип Описание
ind Идентификатор типа частицы The index of the particle type to change.
alpha1 Реально The alpha of the particle.

Возвращается:

N/A

Пример:

part_type_alpha1(global.Snow_Part, 0.5);

Приведенный выше код установит для всех созданных частиц типа частицы, указанного в глобальной переменной "Snow_Part", значение альфа 0,5 (полупрозрачный).
