тип_детали_скорость

Эта функция используется для установки скорости движения частицы при ее создании, а также может использоваться для увеличения или уменьшения скорости частицы в течение ее жизни. Минимальное и максимальное значения скорости по умолчанию равны 1, что означает, что частица будет перемещаться на 1 пиксель каждый шаг. Если они установлены в одно и то же значение, начальная скорость для всех частиц будет одинаковой, но если они установлены в разные значения, каждая частица будет стартовать со случайной скоростью из заданного диапазона. Например, если вы задали минимальное значение 3 и максимальное 6, все частицы будут созданы со скоростью от трех до шести пикселей за шаг.

Вы также можете задать значение приращения, которое заставит частицу двигаться быстрее (если число положительное) или медленнее (если число отрицательное) в течение ее жизни, причем заданное значение будет добавляться к скорости частицы на каждом шаге. Это значение может составлять минимум (+/-)0,01.

Наконец, вы можете задать коэффициент "вихляния". Это значение, которое будет случайным образом прибавляться или вычитаться к скорости на каждом шаге времени жизни частиц. Очевидно, что большие значения более выражены, чем меньшие, и это значение может быть даже отрицательным с максимальным диапазоном от -20 до 20.

Синтаксис:

part_type_speed(ind, speed_min, speed_max, speed_incr, speed_wiggle);

Аргумент Тип Описание
ind Идентификатор типа частицы The index of the particle type to change.
speed_min Реально The minimum speed the particle can start at.
speed_max Реально The maximum speed the particle can start at.
speed_incr Реально How much the particle speed should increase or decrease per step.
speed_wiggle Реально How much should randomly be added or subtracted from the particle's speed per step.

Возвращается:

N/A

Пример:

part_type_shape(particle2, pt_shape_flare);
part_type_size(particle2, 0.01, 0.50, 0.01, 20);
part_type_scale(particle2, 0.10, 0.10);
part_type_colour3(particle2, 4210816, 4227327, 8240895);
part_type_alpha3(particle2, 0.20, 0.80, 0);
part_type_speed(particle2, 5, 10, -0.20, 20);
part_type_direction(particle2, 0, 359, -1, 20);
part_type_gravity(particle2, 0, 270);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 20);

Приведенный выше код установит различные значения частиц, включая скорость, которая устанавливает минимальную скорость 5 и максимальную скорость 10, при этом скорость замедляется на значение -0,2 каждый шаг. Скорость также будет колебаться на случайное значение от 0 до 20 на каждом шаге.
