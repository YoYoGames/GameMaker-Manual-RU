part_type_colour2

Эта функция может быть использована для установки двухцветного градиента для каждой созданной частицы заданного типа. Первый цвет - это цвет, с которого будут начинаться все частицы, а второй цвет - это цвет, на котором частица будет заканчиваться, и плавное градиентное изменение цвета будет происходить на протяжении всего времени жизни частицы от одного цвета к другому.

Синтаксис:

part_type_colour2(ind, colour1, colour2);

Аргумент Тип Описание
ind Идентификатор типа частицы The index of the particle type to change.
colour1 Цвет The colour the particle will start at.
colour2 Цвет The colour the particle will end at.

Возвращается:

N/A

Пример:

part_type_colour2( part_Health, c_red, c_white);

Приведенный выше код заставит все созданные частицы типа, указанного в переменной "part_Health", менять цвет от красного к белому в течение их жизни.
