тип_детали_цвет3

Эта функция может быть использована для установки трехцветного градиента для каждой созданной частицы заданного типа. Первый цвет - это цвет, с которого все частицы будут начинаться, второй цвет - это цвет, который будет смешиваться до половины времени жизни частицы, а третий цвет - это цвет, с которым частица будет заканчиваться. На протяжении жизни частицы будет происходить плавное градиентное изменение цветов от одного к другому.

Синтаксис:

part_type_colour3( ind, colour1, colour2, colour3 );

Аргумент Тип Описание
ind Идентификатор типа частицы The index of the particle type to change.
colour1 Цвет The colour the particle will start at.
colour2 Цвет The colour the particle will be halfway through its lifespan.
colour3 Цвет The colour the particle will end at.

Возвращается:

N/A

Пример:

part_type_colour3( part_Health, c_red, c_white, c_maroon);

Приведенный выше код заставит все частицы, созданные по типу, указанному в переменной "part_Health", иметь смесь цветов от красного до белого и бордового в течение их жизни.
