тип_части_альфа2

Эта функция может быть использована для установки градиента альфа-значения (прозрачности) для каждой созданной частицы заданного типа. Первая альфа - это та, с которой все частицы будут начинаться, а вторая альфа - та, с которой частица будет заканчиваться, и будет происходить плавное градиентное изменение альфы по времени жизни частиц от одной к другой. Это значение может быть от 0 (прозрачный) до 1 (непрозрачный).

Синтаксис:

part_type_alpha2(ind, alpha1, alpha2);

Аргумент Тип Описание
ind Идентификатор типа частицы The index of the particle type to change.
alpha1 Реально The start alpha of the particle.
alpha2 Реально The end alpha of the particle.

Возвращается:

N/A

Пример:

part_type_alpha2(global.Snow_Part, 0, 1);

Приведенный выше код установит для всех созданных частиц типа частиц, индексированных в глобальной переменной "Snow_Part", альфа-значение 0 (прозрачный), а в конце их жизни альфа-значение станет равным 1 (непрозрачный).
