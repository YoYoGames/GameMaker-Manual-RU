part_type_colour_hsv

С помощью этой функции вы можете задать оттенок, насыщенность и диапазон значений для всех частиц данного типа. Вы задаете минимальное значение и максимальное значение для каждого из трех компонентов, и созданные частицы будут иметь случайный цвет, основанный на заданном диапазоне параметров. Таким образом, вы можете создавать частицы одного оттенка, но разной насыщенности, или разных оттенков, но одинаковой величины (светимость) и т.д.. Все значения должны быть в диапазоне от 0 до 255.

Синтаксис:

part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax);

Аргумент Тип Описание
ind Идентификатор типа частицы The index of the particle type to change.
hmin Реально The minimum the final colour's hue component can be.
hmax Реально The maximum the final colour's hue component can be.
smin Реально The minimum the final colour's saturation component can be.
smax Реально The maximum the final colour's saturation component can be.
vmin Реально The minimum the final colour's value component can be.
vmax Реально The maximum the final colour's value component can be.

Возвращается:

N/A

Пример:

part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 );

Приведенный выше код устанавливает для каждой испускаемой частицы типа частицы, индексируемой в глобальной переменной "Stars", разные цвета и насыщенность, но одинаковое значение (светимость).
