часть_системы_глубина

С помощью этой функции вы можете установить глубину прорисовки для системы частиц, точно так же, как вы можете установить глубину рендеринга различных слоев в комнате, где низкая глубина прорисовки означает, что она будет появляться поверх всего, что нарисовано с более высокой глубиной, а высокая глубина прорисовки помещает ее ниже всего, что имеет более низкую глубину прорисовки.

Синтаксис:

part_system_depth( ind, depth );

Аргумент Тип Описание
ind Идентификатор системы частиц The index of the particle system to change.
depth Реально The depth at which to set the particle system.

Возвращается:

N/A

Пример:

global.Sname = part_system_create();
part_system_depth(global.Sname, -1000 );

Приведенный выше код создаст систему частиц и сохранит ее индекс в глобальной переменной "Sname". Затем этой системе присваивается малая глубина прорисовки -1000, что означает, что она будет отображаться поверх всего, что имеет большую глубину прорисовки.
