part_particles_clear

С помощью этой функции вы можете удалить из комнаты все частицы, созданные системой в данный момент. Она не сбрасывает и не удаляет сами типы частиц, только их визуальное представление, и если у вас есть какой-либо объект, излучающий частицы из эмиттера, эти частицы исчезнут, но начнут появляться снова на следующем шаге после вызова этого кода.

Синтаксис:

part_particles_clear(ind);

Аргумент Тип Описание
ind Идентификатор системы частиц The index of the particle system.

Возвращается:

N/A

Пример:

if (lives <= 0)
{
    part_particles_clear(global.Sname);
    room_goto(rm_intro);
}

Приведенный выше код проверит значение переменной "lives" и, если оно равно 0, очистит все частицы из системы, а затем сменит комнату.
