part_system_get_layer

Эта функция может использоваться для получения уникального значения ID слоя для заданной системы частиц. Вы передаете значение ID системы частиц (возвращаемое функцией part_system_create_layer()), а функция вернет значение ID слоя.

Синтаксис:

part_system_get_layer(ind);

Аргумент Тип Описание
ind Идентификатор системы частиц The particle system ID value to get the layer ID of

Возвращается:

Layer ID or 0 (if the layer is an internally managed one)

Пример:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, _effects_layer);
}

Приведенный выше код проверит систему частиц, чтобы узнать, на каком слое она находится, и если она не находится на заданном слое, то она будет изменена.
