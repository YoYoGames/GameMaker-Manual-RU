часть_системы_автоматическое_обновление

Эта функция управляет тем, должен ли GameMaker автоматически обновлять систему частиц или нет. Обычно вам не нужно использовать эту функцию, поскольку значение по умолчанию true (автоматическое обновление включено) - это то, что вы хотите, чтобы произошло. Однако для специальных эффектов или для приостановки игры вы можете установить значение false, и это предотвратит любое обновление данной системы частиц, если не будет дана явная команда с помощью функции part_system_update() или вы снова используете эту функцию для включения автоматического обновления. Отсутствие обновления означает, что частицы, созданные в этой системе, больше не будут менять положение, цвет или любой другой параметр, и эмиттеры тоже перестанут работать, "заморозив" систему в той самой точке, в которой автоматическое обновление было установлено на false.

Следует отметить, что если вы используете простые эффекты, созданные функциями effect_create_above() или effect_create_below() то вы можете использовать значения 0 (для эффектов ниже) или 1 (для эффектов выше) в качестве индекса системы частиц и таким образом переключить автоматическое обновление для них (это также работает для переключения обновления для эффектов визуальных частиц GML ).

Синтаксис:

part_system_automatic_update(ind, automatic);

Аргумент Тип Описание
ind Идентификатор системы частиц The index of the particle system to change.
automatic Булево Whether automatic updating is on (true) или нет (false).

Возвращается:

N/A

Пример:

if global.Pause
{
    part_system_automatic_update(global.Sname, false);
}

Приведенный выше код отключит обновление частиц, если глобальная переменная "Paused" равна true.
