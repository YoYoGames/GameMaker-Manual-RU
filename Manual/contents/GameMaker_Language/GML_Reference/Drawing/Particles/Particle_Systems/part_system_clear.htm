part_system_clear

С помощью этой функции вы можете очистить индексированную систему до состояния по умолчанию, удалив все эмиттеры и сбросив глубину и положение (если они были изменены). Будьте осторожны при использовании этой функции, так как если у вас есть установка экземпляра, разрыв или любое другое действие, использующее эмиттер, связанный с этой системой, вы получите ошибку, если только вы не используете функцию part_emitter_exists() для проверки. Также нет необходимости вызывать функцию part_emitter_destroy(), так как это происходит автоматически.

ПРИМЕЧАНИЕ: эта функция очистит видимые частицы в комнате, но она не очищает свойства частиц и не удаляет их из памяти. Для этого следует использовать функции part_type_clear() и part_type_destroy().

Синтаксис:

part_system_clear(ind);

Аргумент Тип Описание
ind Идентификатор системы частиц The index of the particle system to clear.

Возвращается:

N/A

Пример:

part_system_clear(global.Sname);

Приведенный выше код очистит систему частиц, индексированную в глобальной переменной "Sname", до состояния по умолчанию.
