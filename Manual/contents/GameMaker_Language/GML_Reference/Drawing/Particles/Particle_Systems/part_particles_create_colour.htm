part_particles_create_colour

Эта функция идеально подходит для тех эффектов, которые не требуют функциональности, предлагаемой эмиттерами частиц (например, для создания дыма от ракеты или простого эффекта взрыва), поскольку она позволяет быстро и легко создавать частицы в любом месте игровой комнаты. Вы также можете окрашивать частицы "на лету", когда они создаются с помощью этой функции, и этот цвет будет перекрывать предопределенный цвет частицы, но он не будет смешиваться с цветом в течение всего времени жизни частицы. Обратите внимание, что для использования этой функции необходимо предварительно создать систему частиц и тип частиц.

Синтаксис:

part_particles_create_colour(ind, x, y, parttype, colour, number);

Аргумент Тип Описание
ind Идентификатор системы частиц The index of the particle system.
x Реально The x coordinate of where to create the particles.
y Реально The y coordinate of where to create the particles.
parttype Идентификатор типа частицы The index (type) of the particles to be created.
colour Цвет The colour blending for the particles.
number Реально The number of particles to create.

Возвращается:

N/A

Пример:

if (speed > 1)
{
    var _c = choose(c_aqua, c_lime, c_fuchsia, c_orange);
    part_particles_create_colour(sname, x, y, p_Smoke, _c, 1);
}

Приведенный выше код будет генерировать одну частицу на каждом шаге, когда экземпляр с кодом имеет скорость больше единицы. Эти частицы будут иметь случайный цвет.
