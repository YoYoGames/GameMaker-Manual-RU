часть_системного_слоя

Эта функция может использоваться для переключения системы частиц с текущего слоя на новый. Вы передаете значение ID системы частиц (возвращаемое функцией part_system_create_layer()), а затем уникальный идентификатор слоя (возвращаемый функцией layer_create() или имя слоя в виде строки - например, "instance_layer" - как определено в редакторе комнат), и система будет перемещена на новый слой.

Синтаксис:

part_system_layer(ps, layer);

Аргумент Тип Описание
ps Идентификатор системы частиц The particle system ID
layer Строка или идентификатор слоя The new layer ID

Возвращается:

N/A

Пример:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, "effects_layer";
}

Приведенный выше код проверит систему частиц, чтобы узнать, на каком слое она находится, и если она не находится на заданном слое, то она будет изменена.
