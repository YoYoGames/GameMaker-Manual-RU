part_system_update

Эта функция продвигает заданную систему частиц на один шаг в игровом времени и может использоваться для имитации поведения системы частиц, когда автоматическое обновление выключено, или для продвижения системы частиц к определенной точке за один шаг. Например, если у вас есть система частиц для снега, и вы хотите, чтобы казалось, что снег падает с того момента, как игрок входит в комнату, то вы можете продвинуть частицы снега на несколько секунд в событии создания экземпляра, чтобы при первом рисовании частицы были по всему экрану (пример кода ниже показывает это). Вы также можете использовать эту функцию, когда автоматическое обновление системы отключено функцией part_system_automatic_update() для перемещения системы.

Синтаксис:

part_system_update(ind);

Аргумент Тип Описание
ind Идентификатор системы частиц The index of the particle system to update.

Возвращается:

N/A

Пример:

repeat (room_speed * 3)
{
    part_system_update(global.RainSys);
}

Приведенный выше код продвинет систему частиц, индексированную в глобальной переменной "RainSys", на 3 секунды.
