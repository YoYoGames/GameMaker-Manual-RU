количество_частиц

С помощью этой функции вы можете проверить, есть ли в системе частиц в данный момент частицы, созданные в комнате, и она также вернет их количество.

Синтаксис:

part_particles_count(ind);

Аргумент Тип Описание
ind Идентификатор системы частиц The index of the particle system.

Возвращается:

Real

Пример:

if (part_particles_count(Sname) == 0)
{
    part_system_destroy(Sname);
    instance_destroy();
}

Приведенный выше код проверит количество частиц в локальной системе частиц, указанной в переменной "Sname", и если их нет, то уничтожит систему, а затем и экземпляр.
