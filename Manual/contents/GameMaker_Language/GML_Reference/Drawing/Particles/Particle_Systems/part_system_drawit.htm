часть_системы_обновление

С помощью этой функции вы можете заставить систему частиц нарисовать все текущие частицы на экране. Если part_system_automatic_draw() выключена, то эта функция покажет частицы при использовании в событии draw экземпляра, или она может быть использована, когда цель рисования установлена на поверхность (с помощью surface_set_target()), чтобы нарисовать частицы в системе на этой поверхности.

Синтаксис:

part_system_drawit(ind);

Аргумент Тип Описание
ind Идентификатор системы частиц The index of the particle system to draw the particles of.

Возвращается:

N/A

Пример:

if surface_exists(surf)
{
    surface_set_target(surf);
    part_system_drawit(global.psys);
    surface_reset_target();
}

Приведенный выше код проверяет, существует ли поверхность, указанная в переменной "surf", и если существует, то устанавливает цель рисования на поверхность, рисует систему частиц с идентификатором, сохраненным в глобальной переменной, а затем сбрасывает цель рисования, чтобы на экране снова появилась обычная отрисовка.
