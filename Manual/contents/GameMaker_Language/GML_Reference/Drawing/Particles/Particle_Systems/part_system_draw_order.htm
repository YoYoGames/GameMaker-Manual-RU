порядок_системы_черчения

С помощью этой функции вы можете задать способ отрисовки частиц при их создании на экране. Система по умолчанию использует старый > новый вид (функция установлена на true), где старые частицы рисуются на большей глубине, чем новые, и поэтому появляются "под" новыми частицами, но, установив эту функцию на false, вы можете изменить этот порядок, и новые частицы будут рисоваться выше, и поэтому появятся "под" старыми. Приведенные ниже изображения иллюстрируют это, причем изображение слева имеет значение по умолчанию true, а изображение справа - false:

ПРИМЕЧАНИЕ: Когда частицы рисуются с аддитивным режимом наложения, эффект от этой функции не всегда очевиден.

Синтаксис:

part_system_draw_order(ind, oldtonew);

Аргумент Тип Описание
ind Идентификатор системы частиц The index of the particle system to change.
oldtonew Булево Whether old particles should be drawn behind newer ones (true) or vice versa (false).

Возвращается:

N/A

Пример:

mysystem = part_system_create();
part_system_draw_order(mysystem, true);

Это создаст новую систему частиц с индексом "mysystem", а затем установит частицы, чтобы новые частицы рисовались поверх старых.
