часть_системы_существует

С помощью этой функции вы можете проверить, существует ли данная система частиц в игре или нет. Обратите внимание, что если проверяемая переменная является неинициализированной переменной (которой система частиц в противном случае присвоила бы свой индекс), это приведет к ошибке.

Синтаксис:

part_system_exists(ind);

Аргумент Тип Описание
ind Идентификатор системы частиц The index of the particle system to check for.

Возвращается:

Boolean

Пример:

if !part_system_exists(global.Sname)
{
    global.Sname = part_system_create();
}

Приведенный выше код проверяет, существует ли система частиц, на которую ссылается глобальная переменная, и если нет, то она создается.
