эффект_создания_нижнего

С помощью этой функции вы можете создать простой эффект под всеми экземплярами вашей комнаты (на самом деле он создается на глубине 100000). Если эффект не ef_rain или ef_snow, то вы можете определить позицию x/y для создания эффекта, а размер может быть равен 0, 1 или 2, где 0 - маленький, 1 - средний и 2 - большой.

Стоит отметить, что эти эффекты можно включать и выключать, а также приостанавливать их отрисовку с помощью функций part_system_automatic_draw() и part_system_automatic_update() с соответствующим значением индекса системы частиц (где 0 - для эффектов ниже, а 1 - для эффектов выше).

Доступными константами для различных видов частиц являются:

Постоянная Пример Описание
ef_cloud An effect that creates random cloud particles of varying sizes
ef_ellipse An effect that creates expanding ellipses
ef_explosion An effect that creates expanding fading explosions
ef_firework An effect that creates multiple small particles to generate a firework explosion
ef_flare An effect that generates a brilliant point that flares up and fades out
ef_rain An effect that generates rain particles coming down from the top of the screen
ef_ring An effect that generates expanding and fading circles
ef_smoke An effect that generates little puffs of smoke
ef_smokeup An effect that creates a smoke plume that rises up the screen
ef_snow An effect that generates multiple snow particles falling down the screen
ef_spark An effect that generates a small spark
ef_star An effect that generates star particles

Синтаксис:

effect_create_below(kind, x, y, size, colour);

Аргумент Тип Описание
kind Тип эффекта Постоянная The kind of effect (use one of the constants listed above).
x Реально The x positioning of the effect if relevant.
y Реально The y positioning of the effect if relevant.
size Реально The size of the effect.
colour Цвет The colour of the effect.

Возвращается:

N/A

Пример:

if speed > 0
{
    effect_create_below(ef_smoke, x, y, 0, c_gray);
}

Приведенный выше код будет создавать небольшую струйку серого дыма на каждом шаге, когда скорость экземпляра больше 0 в координатах x,y экземпляра.
