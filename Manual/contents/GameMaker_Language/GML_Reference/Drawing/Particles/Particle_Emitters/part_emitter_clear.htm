part_emitter_clear

С помощью этой функции вы можете очистить данный эмиттер от указанной системы частиц и вернуть его в состояние по умолчанию. Это также остановит все частицы, которые в данный момент поступают из эмиттера, и если вы хотите использовать эмиттер снова, вам нужно будет установить позицию региона и тип частиц с помощью функции part_emitter_region() функцию.

Синтаксис:

part_emitter_clear(ps, ind);

Аргумент Тип Описание
ps Идентификатор системы частиц The particle system that the emitter is in.
ind Идентификатор излучателя частиц The index of the emitter to clear.

Возвращается:

N/A

Пример:

part_emitter_clear(Sname, p_emit1 );

Приведенный выше код очистит эмиттер частиц, индексированный в переменной "p_emit1".
