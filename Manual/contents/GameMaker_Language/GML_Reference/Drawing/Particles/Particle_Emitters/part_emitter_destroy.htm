part_emitter_destroy

Эта функция удалит указанный эмиттер из данной системы и очистит его из памяти (это также остановит производство любых частиц данным эмиттером, но НЕ удалит их из комнаты). Эту функцию всегда следует вызывать, когда данный эмиттер больше не нужен системе, чтобы предотвратить утечку памяти и ошибки.

Синтаксис:

part_emitter_destroy( ps, ind );

Аргумент Тип Описание
ps Идентификатор системы частиц The particle system to destroy the emitter from.
ind Идентификатор излучателя частиц The index of the emitter to destroy.

Возвращается:

N/A

Пример:

if part_emitter_exists(global.Sname, p_emit)
{
    part_emitter_destroy(global.Sname, p_emit);
}

Приведенный выше код проверит, существует ли эмиттер частиц, указанный в переменной "p_emit", в системе частиц give, и если да, то он будет уничтожен.
