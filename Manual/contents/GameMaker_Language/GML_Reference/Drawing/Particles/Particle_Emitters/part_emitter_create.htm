часть_излучателя_создать

Эта функция используется для создания нового эмиттера частиц в заданной системе частиц. Функция возвращает значение индекса для нового эмиттера, которое должно храниться в переменной и использоваться во всех дальнейших функциях, ссылающихся на эмиттер, а сам эмиттер должен быть уничтожен, если он больше не используется, чтобы предотвратить утечку памяти (это можно сделать с помощью специальных функций уничтожения эмиттера или путем уничтожения всей системы частиц, к которой принадлежит эмиттер).

ПРИМЕЧАНИЕ Идентификаторы эмиттеров, возвращаемые этой функцией, используются повторно, например, если вы сначала создадите три эмиттера частиц, им будут присвоены идентификаторы 0, 1 и 2. Если вы затем уничтожите второй эмиттер (который получил ID 1) с помощью функции part_emitter_destroy и затем создадите другой эмиттер, то этот эмиттер будет повторно использовать ID 1. Поскольку ID эмиттера 2 используется, следующий эмиттер получит ID 3, и так далее.

Синтаксис:

part_emitter_create(ps);

Аргумент Тип Описание
ps Идентификатор системы частиц The particle system to create the emitter in.

Возвращается:

Particle Emitter ID

Пример:

p_emit1 = part_emitter_create(Sname);

Это создаст новый эмиттер частиц и сохранит его индекс в переменной "p_emit".
