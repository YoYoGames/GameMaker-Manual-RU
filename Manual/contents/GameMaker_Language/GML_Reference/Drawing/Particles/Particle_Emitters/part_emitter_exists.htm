часть_излучателя_существует

С помощью этой функции вы можете проверить, существует ли данный эмиттер частиц с заданным индексом в данной системе или нет. Обратите внимание, что если проверяемая переменная является неинициализированной переменной (которой эмиттер частиц в противном случае был бы присвоен его индекс), это приведет к ошибке.

Синтаксис:

part_emitter_exists(ps, ind);

Аргумент Тип Описание
ps Идентификатор системы частиц The particle system to check for an emitter.
ind Идентификатор излучателя частиц The index of the emitter to search for.

Возвращается:

Boolean

Пример:

if part_emitter_exists(sname, p_emit1)
{
    part_emitter_burst(sname, p_emit1, part_1, 30);
}

Приведенный выше код проверит наличие эмиттера, указанного в переменной "permit", и если он существует, то выпустит из него 30 частиц.
