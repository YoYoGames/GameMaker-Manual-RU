часть_излучателя_уничтожить_все

Эта функция удалит все определенные эмиттеры из данной системы и очистит их из памяти (это также остановит производство любых частиц данным эмиттером, но НЕ удалит их из комнаты). Эту функцию всегда следует вызывать, когда эмиттеры больше не нужны системе, чтобы предотвратить утечку памяти и ошибки.

Синтаксис:

part_emitter_destroy_all( ps );

АргументТип Описание
ps The particle system to destroy all emitters from.

Возвращается:

Пример:

if lives == 0
{
    part_emitter_destroy_all(global.Sname);
    room_goto(rm_Menu);
}

Приведенный выше код проверяет встроенную глобальную переменную "lives", и если она равна 0, то уничтожает все излучатели частиц, а затем меняет комнату.
