часть_излучателя_потока

Эта функция позволяет настроить эмиттер на поток частиц определенного типа и обычно используется в событии create, поскольку это одноразовый код, который устанавливает эмиттер на генерацию заданного количества частиц на каждом последующем шаге игры. Частицы испускаются в соответствии с распределением, формой и положением, заданными функцией part_emitter_region(). Вы можете установить количество излучаемых частиц на ноль и вызвать эту функцию снова, чтобы "выключить" поток частиц, также функция будет принимать отрицательные числа для количества, в этом случае эмиттер будет генерировать частицы на основе случайности. Например, если количество частиц установлено на -5, то вероятность того, что частица будет производиться каждый шаг, составляет 1:5. Если вам нужно, чтобы частицы появлялись все сразу, а не на каждом шаге, используйте функцию part_emitter_burst().

Синтаксис:

part_emitter_stream(ps, ind, parttype, number);

Аргумент Тип Описание
ps Идентификатор системы частиц The particle system that the emitter is in.
ind Идентификатор излучателя частиц The index of the emitter to burst from.
parttype Идентификатор типа частицы The index (type) of the particles to be created.
number Реально The number of particles to create per step.

Возвращается:

N/A

Пример:

part_emitter_stream(global. Sname, p_emit1, p1, 1);

Приведенный выше код будет передавать по 1 частице на каждом шаге игры, пока эмиттер не будет уничтожен или поток не установится на 0.
