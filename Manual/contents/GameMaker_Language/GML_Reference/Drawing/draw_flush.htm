draw_flush

С помощью этой функции можно очистить весь конвейер рисования. Эта функция предназначена только для отладки, и в целом ее не следует использовать без указания сотрудника службы поддержки YoYo Games. так как беспорядочное использование приведет к серьезным проблемам с производительностью вашей игры.

Синтаксис:

draw_flush();

Возвращается:

Пример:

draw_flush();

Приведенный выше код промывает конвейер рисования.
