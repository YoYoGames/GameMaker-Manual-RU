нарисовать_часть

С помощью этой функции можно нарисовать часть любого sprite в заданном положении в помещении. Как и в случае с draw_sprite() вы можете указать sprite и под-изображение для рисования, затем вы должны указать относительные координаты в пределах sprite области, которую нужно выбрать для рисования. Это означает, что левая позиция 0 и верхняя позиция 0 будут левым верхним углом sprite, и все дальнейшие координаты должны быть взяты из этой позиции. На рисунке ниже показан пример как это работает:

Следует отметить, что если страница текстуры допускает автоматическое кадрирование, то эта функция может работать не так, как ожидается, поскольку при создании текстурной страницы будет удалено дополнительное "пустое" пространство вокруг sprite. Чтобы решить эту проблему, необходимо в настройках страницы текстуры (в редакторе группы текстур) отключить опцию Автоматическое кадрирование.

ПРИМЕЧАНИЕ: При рисовании с помощью этой функции смещение sprite по x и y игнорируется, и деталь sprite будет нарисована с левым верхним углом в заданном положении x / y в помещении.

ПРИМЕЧАНИЕ: Эта функция полезна только для растровых изображений sprites и не работает с SWF или JSON (Spine) sprites.

Синтаксис:

draw_sprite_part(sprite, subimg, left, top, width, height, x, y);

АргументТип Описание
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
left The x position on the sprite of the top left corner of the area to draw.
top The y position on the sprite of the top left corner of the area to draw.
width The width of the area to draw.
height The height of the area to draw.
x The x coordinate of where to draw the sprite.
y The y coordinate of where to draw the sprite.

Возвращается:

Пример:

draw_sprite_part(sprite_index, image_index, 4, 0, sprite_width-16, sprite_height-16, x, y );

Это отрисует назначенный экземпляр sprite (sprite_index) и его текущий кадр анимации (image_index), однако при этом будет сбрито 4px поля по ширине с обеих сторон и 8 pixel поля по высоте от нижней части оригинального 24x24 пикселей sprite.
