draw_sprite_pos

С помощью этой функции можно нарисовать sprite, искаженное в области, определенной четырьмя угловыми координатами. Первыми двумя аргументами являются sprite для рисования и субизображение sprite (такое же, как в функции draw_sprite()) , а следующие - те, которые определяют положение каждого из четырех углов ограничивающего поля данного sprite. Их следует указывать по часовой стрелке, поэтому первой координатой будет верхняя левая, затем верхняя правая, затем нижняя правая и, наконец, нижняя левая. правая и, наконец, нижняя левая. Вы также можете задать значение альфа для sprite, чтобы нарисовать его прозрачным. На рисунке ниже показано, как работает эта функция:

ВНИМАНИЕ! Изображение выше приведено только для примера, и если вы используете эту функцию на sprite, вы получите другие результаты и можете столкнуться со "сдвигом" текстуры из-за того, что sprite строится из квадрокоптера примитивов.

ПРИМЕЧАНИЕ: Эта функция полезна только для растровых изображений sprites и не работает с SWF или JSON (Spine) sprites.

ПРИМЕЧАНИЕ: При использовании этой функции параметр Автоматически обрезать для текстурных страниц должен быть отключен.

Синтаксис:

draw_sprite_pos(sprite, subimg, x1, y1, x2, y2, x3, y3, x4, y4, alpha);

АргументТип Описание
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
x1 The first x coordinate.
y1 The first y coordinate.
x2 The second x coordinate.
y2 The second y coordinate.
x3 The third x coordinate.
y3 The third y coordinate.
x4 The fourth x coordinate.
y4 The fourth y coordinate.
alpha The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Возвращается:

Пример:

draw_sprite_pos(sprite_index, image_index, x - 100, y - 50, x - 50, y +150, x + 100, y + 200, x + 100, y, 1);

Приведенный выше код нарисует sprite, связанный с экземпляром, выполняющим код, с искажением вокруг x / y позиции экземпляра и с полностью непрозрачной альфой.
