draw_tilemap

Эта функция может быть использована для рисования заданного tilemap в любом месте комнаты. Вы передаете ID элемента tilemap (который вы получаете при создании элемента tilemap с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), затем указываете позицию x/y для рисования (в комнате). Обратите внимание, что это просто нарисует тайлмап в указанной точке, используя глубину слоя экземпляра, вызывающего функцию. Это не перемещает тайлмап, не изменяет его каким-либо образом, и не имеет значения, отмечен ли тайлмап как видимый или нет.

Синтаксис:

draw_tilemap(tilemap_element_id, x, y);

Аргумент Тип Описание
tilemap_element_id Идентификатор элемента карты плитки The unique ID value of the tilemap element to draw
x Реально The x position within the room to draw at
y Реально The y position within the room to draw at

Возвращается:

N/A

Пример:

var lay_id = layer_get_id("Tiles_Effects");
var map_id = layer_tilemap_get_id(lay_id);
draw_tilemap(map_id, mouse_x, mouse_y);

Приведенный выше код получает идентификатор слоя, затем использует его для получения определенного идентификатора тайлмапа, который, в свою очередь, используется для рисования тайлмапа в позиции мыши.
