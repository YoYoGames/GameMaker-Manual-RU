draw_enable_swf_aa

С помощью этой функции вы можете включить или отключить сглаживание (AA) для векторного формата SWF sprites. AA просто сглаживает края векторных изображений, чтобы придать им более приятный вид. Количество используемого сглаживания зависит от значения, установленного с помощью функции draw_set_swf_aa_level(). По умолчанию эта функция отключена.

Синтаксис:

draw_enable_swf_aa(enable);

АргументТип Описание
enable Enable (true) или отключить (false) AA для всех SWF sprites.

Возвращается:

Пример:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

Приведенный выше код проверит значение AA для формата SWF sprites, и если оно равно 0, то включит AA и установит значение 0,5.
