нарисовать_прямую_плитку

Эта функция принимает sprite и затем многократно чередует его по всему виду (или комнате, если вид не определен), начиная с координат, которые вы задаете в функции. Плитка основывается на ширине и высоте sprite, как определено переменными sprite переменных экземпляра, выполняющего код. Эта функция предназначена только для 2D (ортографических) проекций и не будет работать корректно, если используется 3D проекции камеры.

Синтаксис:

draw_sprite_tiled(sprite, subimg, x, y);

АргументТип Описание
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
x The x coordinate of where to draw the sprite.
y The y coordinate of where to draw the sprite.

Возвращается:

Пример:

draw_sprite_tiled(sprite_index, image_index, x, y);

Это нарисует назначенный экземпляр sprite (sprite_index) и его текущий кадр анимации (image_index) в собственном положении экземпляра по x и y и в виде плитки по горизонтали и вертикали по всему представлению.
