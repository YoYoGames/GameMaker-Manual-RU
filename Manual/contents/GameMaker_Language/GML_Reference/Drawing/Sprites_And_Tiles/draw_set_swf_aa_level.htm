draw_set_swf_aa_level

Эта функция может использоваться для установки уровня сглаживания (AA) для векторного формата SWF sprites. Это может быть вещественное значение от 0 до 1 и будет "сглаживать" края этих sprites. Обратите внимание, что для того, чтобы увидеть этот эффект, вы должны сначала включить AA с помощью функции draw_enable_swf_aa().

Синтаксис:

draw_set_swf_aa_level(AA);

АргументТип Описание
AA The anti-aliasing value to use from 0 to 1.

Возвращается:

Пример:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

Приведенный выше код проверит значение AA для формата SWF sprites, и если оно равно 0, то включит AA и установит значение 0,5.
