video_get_status

Эта функция возвращает статус текущего загруженного видео. Статус может быть любой из следующих констант:

Состояние видео Постоянная
Постоянная Описание
video_status_closed В настоящее время видео не загружено, или видео было закрыто с помощью функции video_close()
video_status_preparing Видео находится в стадии подготовки и еще не начало воспроизводиться
video_status_playing Видео в настоящее время воспроизводится
video_status_paused Видео приостанавливается (см. video_pause()).

Синтаксис:

video_get_status();

Возвращается:

Video Status Constant

Пример:

var _status = video_get_status();

if (keyboard_check_pressed(vk_space))
{
    if (_status == video_status_playing)
    {
        video_pause();
    }
    else if (status == video_status_paused)
    {
        video_resume();
    }
}

Приведенный выше код получает статус видео, а затем проверяет, нажал ли игрок кнопку Space. В этом случае он приостанавливает видео, если оно воспроизводится, и возобновляет его, если оно приостановлено.
