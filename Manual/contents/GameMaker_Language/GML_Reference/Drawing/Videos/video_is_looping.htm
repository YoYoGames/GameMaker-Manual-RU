video_is_looping

Эта функция возвращает, настроен ли загруженный в данный момент видеоролик на зацикливание(true) или нет(false). Вы можете установить видео на цикл, вызвав video_enable_loop().

Синтаксис:

video_is_looping();

Возвращается:

Boolean

Пример:

if (video_is_looping()) {
    obj_player.control_enabled = false;
}

Приведенный выше код проверяет, не зацикливается ли загруженное видео, и в этом случае отключает элементы управления плеера, устанавливая пользовательскую переменную экземпляра в false.
