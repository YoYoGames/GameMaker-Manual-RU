video_get_volume

Эта функция возвращает громкость текущего загруженного видео, которая представляет собой значение между 0 и 1. Вы можете изменить громкость загруженного видео с помощью video_set_volume().

Синтаксис:

video_get_volume();

Возвращается:

Real

Пример:

/// Step event
var _video_volume = video_get_volume();

if (_video_volume < 1) {
    _video_volume += 0.02;
    video_set_volume(_video_volume);
}

Приведенный выше код будет выполняться в событии Step. Он получает значение громкости воспроизводимого видео и, если оно меньше 1, увеличивает его на 0,02, постепенно повышая громкость до максимального уровня.

Назад: Видео
