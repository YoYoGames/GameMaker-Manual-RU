video_close

Эта функция закрывает загруженный в данный момент видеофайл. Убедитесь, что эта функция вызывается только после вызова video_open(), иначе она ничего не сделает.

Синтаксис:

video_close();

Возвращается:

N/A

Пример:

if (keyboard_check_pressed(vk_escape))
{
    video_close();
}

Приведенный выше код закрывает видео при нажатии клавиши Escape.
