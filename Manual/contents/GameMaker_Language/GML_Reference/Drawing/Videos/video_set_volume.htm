video_set_volume

Эта функция изменяет громкость звука видео, которое загружено в данный момент. Она принимает значение от 0 до 1, где 0 - тишина, а 1 - максимальная громкость.

Убедитесь, что этот вызов происходит только после вызова video_open(), но до вызова video_close(), иначе он ничего не сделает (так как не будет загружено видео).

На некоторых платформах эта функция будет работать только после полной загрузки видео, поэтому используйте Async Callbacks, чтобы узнать, когда видео будет готово, и затем изменить громкость.

Синтаксис:

video_set_volume(value);

Аргумент Тип Описание
value Реально Volume value between 0 and 1

Возвращается:

N/A

Пример:

my_video = video_open("splash.mp4");
video_set_volume(0.5);

Приведенный выше код загружает файл splash.mp4 из папки Included Files игры и изменяет его громкость на 0,5 (50% от максимальной громкости).
