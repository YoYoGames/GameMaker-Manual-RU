video_open

Эта функция открывает видеофайл по указанному пути. Путь может ссылаться на файл в Included Files вашего проекта, файл, локально присутствующий на устройстве пользователя, или URL для загрузки файла из интернета (что может работать не на всех платформах).

Поскольку за один раз можно загрузить только одно видео, эта функция загружает указанное видео в единственный доступный "видеослот", которым затем можно управлять с помощью других функций Video.

С воспроизведением видео связаны асинхронные обратные вызовы. Дополнительные сведения см. в разделе Асинхронные обратные вызовы.

Синтаксис:

video_open(path);

Аргумент Тип Описание
path Строка Path to the video file

Возвращается:

N/A

Пример:

my_video = video_open("splash.mp4");

Приведенный выше код загружает файл splash.mp4 из папки Included Files игры.

Назад: Воспроизведение видео
