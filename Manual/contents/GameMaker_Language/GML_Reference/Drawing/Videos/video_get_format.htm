video_get_format

Эта функция возвращает формат цвета текущего загруженного видео. Это может быть любая из следующих констант:

Формат видео Постоянный
Постоянная Описание
video_format_rgba Видеоповерхность использует цветовую модель RGBA
video_format_yuv Видеоповерхность использует цветовую модель YUV

Windows, macOS, Opera GX, Android, iOS и HTML5 используют модель RGBA для видео. Все остальные платформы используют модель YUV.

Поскольку методы рисования для RGBA и YUV видео отличаются, возвращаемое значение этой функции может быть использовано для запуска различного кода в зависимости от формата воспроизводимого видео. Пример приведен в разделе Draw Video.

Синтаксис:

video_get_format();

Возвращается:

Video Format Constant

Назад: Воспроизведение видео
