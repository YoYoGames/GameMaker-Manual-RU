vertex_submit

Вы можете использовать эту функцию для передачи содержимого вершинного буфера в графический конвейер для использования в шейдере. Вы указываете индекс буфера, базовый тип примитива (см. константы ниже) и текстуру, которая будет использоваться. Базовый тип примитива используется только для назначения порядка, в котором вершины, сохраненные в буфере, будут рисоваться и соединяться, но фактические данные, используемые для каждой из вершин, будут теми, которые вы определили при создании буфера вершин.

ПРИМЕЧАНИЕ: Эта функция может быть использована только в событии Draw Event.

В качестве наглядного примера различных доступных базовых примитивов см. рисунок ниже:

Постоянная Описание
pr_pointlist Примитив, состоящий из списка точек.
pr_linelist Примитив, состоящий из отдельных строк в списке.
pr_linestrip Примитив, состоящий из последовательной полосы линий.
pr_trianglelist Примитив, состоящий из отдельных треугольников в списке.
pr_trianglestrip Примитив, состоящий из последовательной полосы треугольников.

Синтаксис:

vertex_submit(buffer, primitive, texture);

Аргумент Тип Описание
buffer Идентификатор буфера вершин The buffer to use.
primitive Примитивный тип Константа The primitive base type.
texture Указатель текстуры The texture to use (or -1 for no texture).

Возвращается:

N/A

Пример:

shader_set(shader_prim);
vertex_submit(buff, pr_trianglelist, sprite_get_texture(sprite_index));
shader_reset();

Приведенный выше код передаст буфер вершин, индексированный в переменной "buff", для отрисовки шейдером "shader_prim", используя список треугольников в качестве базового примитива и текстуру sprite для экземпляра, выполняющего код.
