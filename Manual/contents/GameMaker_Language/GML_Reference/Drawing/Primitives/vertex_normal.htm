вершинная_нормаль

Эта функция добавит данные о нормали поверхности к вершине, определяемой в данный момент для пользовательского примитива. Вы предоставляете буфер для записи данных, а также x, y и z компоненты нормали.

Синтаксис:

vertex_normal(buffer, nx, ny, nz);

vertex_normal

This function adds normal data to the vertex currently being defined for the custom primitive.

You supply the vertex buffer to write the data into as well as the x, y and z components of the normal.

 

Syntax:

vertex_normal(buffer, nx, ny, nz);
