вершина_плоская3

Эта функция добавит три значения с плавающей точкой к данным вершины. Вершина должна быть правильно отформатирована, чтобы принять ее с помощью функции vertex_format_add_custom() функция.

Синтаксис:

vertex_float3(buffer, float, float, float);

Аргумент Тип Описание
buffer Идентификатор буфера вершин The buffer to write the information to.
float Реально The first input value.
float Реально The second input value.
float Реально The third input value.

Возвращается:

N/A

Пример:

vertex_float3(buff, 0.05, 0.01, room_width / x);

Приведенный выше код добавит три значения с плавающей точкой к определяемым данным вершин.
