Примитивы и форматы вершин

Примитив - это тип рисунка, созданный с помощью точек для определения того, что отображается на экране. Они могут состоять из отдельных точек, линий или треугольников, могут быть цветными, иметь альфа-смешение и даже текстуру, что означает, что они могут быть использованы для создания некоторых очень интересных эффектов в 2D-игре! Например, вы можете создать прямоугольник, используя несколько треугольных примитивов, затем текстурировать его флагом sprite и в событии рисования экземпляра заставить этот флаг "развеваться" на ветру и исказить используемый вами sprite, просто изменив точки, образующие примитив.

ВНИМАНИЕ! Эти функции не работают с модулем HTML5, если вы не включили OpenGL в Опциях игры.

    Vertex Formats

    GameMaker permits you to define your own vertex formats from which you can create your own custom primitives. This can greatly speed up shader operations or can be used to extend their capabilities and create surprising effects.

    A vertex format lists the attributes that are stored for a vertex in order. You start defining one with vertex_format_begin, then add the attributes using the vertex_format_add_* functions and finally end it using vertex_format_end.

    ВАЖНО! Примитивы, которые вы строите, должны соответствовать формату, который вы задали с помощью функций Vertex Format, поэтому если вы определили формат вершин только с позиционными данными, нет смысла строить примитив с цветовыми данными. Следует отметить, что порядок добавления свойств в создаваемый примитив определяется порядком добавления этих свойств при создании вершинного формата, поэтому если вы определили векторный формат с порядком позиция, цвет и координата текстуры, то вы должны добавить эти свойства в создаваемый примитив в том же порядке, иначе вы получите ошибки.

    Обзор того, как строить примитивы с помощью функций вершин, см. в статье:
