draw_vertex

Эта функция определяет положение вершины для примитива. Окончательный вид примитива будет зависеть от типа примитива, выбранного для рисования, и порядка добавления вершин к нему. См. draw_primitive_begin() для дополнительную информацию. Чтобы завершить и нарисовать примитив, необходимо вызвать команду draw_primitive_end().

Синтаксис:

draw_vertex(x, y)

АргументТип Описание
x The x coordinate of the vertex.
y The y coordinate of the vertex.

Возвращается:

Пример:

draw_primitive_begin(pr_trianglelist);
draw_vertex(100, 100);
draw_vertex(100, 200);
draw_vertex(150, 150);
draw_primitive_end();

Приведенный выше код нарисует простой примитив треугольника.
