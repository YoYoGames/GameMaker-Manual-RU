vertex_delete_buffer

Эта функция может быть использована для удаления ранее созданного буфера вершин (см. vertex_create_buffer()) из системной памяти. При этом также удаляются все вершинные данные для всех пользовательских примитивов, которые в нем содержались.

Синтаксис:

vertex_delete_buffer(buffer);

Аргумент Тип Описание
buffer Идентификатор буфера вершин The vertex buffer to be removed.

Возвращается:

N/A

Пример:

vertex_delete_buffer(v_buff);

Приведенный выше код удалит из памяти буфер, индексированный в переменной "v_buff".
