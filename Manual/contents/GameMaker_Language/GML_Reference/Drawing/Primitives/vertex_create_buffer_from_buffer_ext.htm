vertex_create_buffer_from_buffer_ext

Как и в случае с функцией vertex_create_buffer()эта функция создаст новый буфер вершин, только теперь данные вершин, которые в нем хранятся, копируются из обычного буфера, указанного в качестве источника. Буфер предварительно отформатирован в соответствии с форматом вершин для построения примитивов для использования (например) с шейдерами, также можно указать смещение в исходном буфере для копирования и количество вершин, которое должно быть в конечном буфере. Обратите внимание, что если количество вершин не совпадает с копируемыми, вы можете получить поврежденные вершинные данные.

Синтаксис:

vertex_create_buffer_from_buffer_ext(buffer, format, src_offset, vert_num);

Аргумент Тип Описание
buffer Идентификатор буфера The buffer to create the vertex buffer from.
format Примитивный тип Константа The primitive vertex format to use.
src_offset Реально The offset within the the source buffer to copy from.
vert_num Реально The number of vertices the buffer should have.

Возвращается:

Vertex Buffer ID

Пример:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat, 0, 512);

Приведенный выше код создаст новый вершинный формат, затем создаст новый вершинный буфер из ранее созданного обычного буфера, применив к нему пользовательский вершинный формат со смещением 0. Функция сообщает новому вершинному буферу, что он должен ожидать 512 вершин.
