цвет_вершины

Если определенный вами формат вершины принимает значение цвета, вы можете использовать эту функцию для добавления этих данных к вершине, определяемой для текущего примитива. Функция требует буфера для хранения данных и принимает либо цветовую константу, либо шестнадцатеричное значение (используя стандартный формат GameMaker BGR, например: $FF0000 для синего цвета), а также значение альфа от 0 (прозрачный) до 1 (полностью непрозрачный).

Синтаксис:

vertex_colour(buffer, colour, alpha);

Аргумент Тип Описание
buffer Идентификатор буфера вершин The buffer to write the information to.
colour Цвет The colour for this vertex (can be a constant or a hex value).
alpha Реально The alpha value for the vertex (from 0 to 1).

Возвращается:

N/A

Пример:

vertex_colour(b, c_white, 1);

Приведенный выше код установит для определяемой вершины белый цвет с альфа-значением 1.
