vertex_ubyte4

Эта функция добавит к данным вершины четыре беззнаковых байтовых значения (0 - 255). Вершина должна быть правильно отформатирована, чтобы принять ее с помощью функции vertex_format_add_custom() функцию.

Синтаксис:

vertex_ubyte4(buffer, byte, byte, byte, byte);

Аргумент Тип Описание
buffer Идентификатор буфера вершин The buffer to write the information to.
byte Реально The first input value.
byte Реально The second input value.
byte Реально The third input value.
byte Реально The fourth input value.

Возвращается:

N/A

Пример:

vertex_ubyte4(buff, irandom(255), irandom(255), irandom(255), 127);

Приведенный выше код добавит четыре значения к определяемым данным вершин.
