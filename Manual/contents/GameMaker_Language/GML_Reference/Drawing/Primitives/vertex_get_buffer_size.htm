размер вершины_get_buffer_size

С помощью этой функции вы можете получить размер заданного вершинного буфера в байтах.

Синтаксис:

vertex_get_buffer_size(buffer);

Аргумент Тип Описание
buffer Идентификатор буфера вершин The buffer to get the size of.

Возвращается:

Real

Пример:

bufferBytes = vertex_get_buffer_size(buff);

Приведенный выше код получит количество байт, используемых данным вершинным буфером, и сохранит это значение в переменной "bufferBytes".
