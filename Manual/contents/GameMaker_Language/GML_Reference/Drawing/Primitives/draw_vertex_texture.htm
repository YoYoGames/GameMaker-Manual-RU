рисовать_вершинную_текстуру

Эта функция определяет положение текстурированной вершины для примитива. Окончательный вид примитива будет зависеть от типа примитива, выбранного для рисования (см. draw_primitive_begin() более подробную информацию), порядка добавления вершин и положения начальной точки, заданной для текстуры. Чтобы завершить и нарисовать примитив, необходимо вызвать команду draw_primitive_end().

Вы также можете выбрать рисование только части текстуры на поверхности примитива, установив соответствующую xtex/ytex позицию для выборки (эти координаты также известны как UV-точки ). Эта точка определяется значением от 0 до 1, причем левый верхний угол используемого изображения является точкой (0,0), а правый нижний угол - точкой (1,1). Таким образом, можно текстурировать примитив, используя только часть исходного изображения.

При рисовании текстурированного примитива текстура будет смешиваться с цветом, заданным параметром draw_set_colour. Это означает, что для рисования текстуры без смешивания следует установить значение c_white, как показано в примере ниже.

ПРИМЕЧАНИЕ Для того чтобы текстура повторялась, ее размер должен быть равен степени двойки, т.е.: 32x32, 128x128 и т.д..

Синтаксис:

draw_vertex_texture(x, y, xtex, ytex)

Аргумент Тип Описание
x Реально The x coordinate of the vertex.
y Реально The y coordinate of the vertex.
xtex Реально The width of the texture sample to use.
ytex Реально The height of the texture sample to use.

Возвращается:

N/A

Пример:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

Приведенный выше код нарисует полосу треугольника с 4 вершинами (образуя прямоугольник), текстурированную текстурой, хранящейся в переменной "tex", и вся текстура будет использована для покрытия готового примитива.
