vertex_create_buffer_from_buffer

Как и в случае с функцией vertex_create_buffer()эта функция создаст новый вершинный буфер, только теперь вершинные данные, хранящиеся в нем, копируются из обычного буфера, указанного в качестве источника. Создаваемый буфер - это специальный растущий буфер, который предварительно отформатирован в вершинном формате для построения примитивов для использования (например) в шейдерах. Эта функция требует, чтобы вы указали указатель на ранее созданный обычный буфер и формат вершин, который должен быть применен к скопированным данным.

Синтаксис:

vertex_create_buffer_from_buffer(buffer, format);

Аргумент Тип Описание
buffer Идентификатор буфера The buffer to create the vertex buffer from.
format Примитивный тип Константа The primitive vertex format to use.

Возвращается:

Vertex Buffer ID

Пример:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat);

Приведенный выше код создаст новый формат вершин, затем создаст новый буфер вершин из ранее созданного обычного буфера, применив к нему пользовательский формат вершин.
