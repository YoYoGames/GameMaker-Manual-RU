вершинный_формат_конец

Эта функция должна быть вызвана после определения любого нового формата вершин. Она возвращает "handle" (индекс) нового формата, который должен использоваться во всех последующих вершинных функциях, обращающихся к этому новому формату.

Синтаксис:

vertex_format_end();

Возвращается:

Vertex Format ID

Пример:

=======

vertex_format_end

This function returns the vertex format that was started with vertex_format_begin.

 

Syntax:

vertex_format_end();

 

Returns:

Vertex Format
