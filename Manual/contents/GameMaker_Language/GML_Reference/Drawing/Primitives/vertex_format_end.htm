вершинный_формат_конец

Эта функция должна быть вызвана после определения любого нового формата вершин. Она возвращает "handle" (индекс) нового формата, который должен использоваться во всех последующих вершинных функциях, обращающихся к этому новому формату.

Синтаксис:

vertex_format_end();

Возвращается:

Vertex Format ID

Пример:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

Приведенный выше код создаст новый формат вершин, содержащий только значения цвета и позиции, а затем сохранит идентификатор формата в переменной "my_format".
