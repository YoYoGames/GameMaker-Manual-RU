вершинный_коорд

Эта функция устанавливает координаты текстуры для вершины, которая в данный момент определена для пользовательского примитива. Вы предоставляете буфер для записи данных, а также позицию UV в текстуре, которую необходимо использовать.

Синтаксис:

vertex_texcoord(buffer, u, v);

=======

vertex_texcoord

This function sets the texture coordinates to use for the vertex currently being defined for the custom primitive.

You supply the vertex buffer to write the data into as well as the UV position within the texture to use.

 

Syntax:

vertex_texcoord(buffer, u, v);
