вершинный_коорд

Эта функция устанавливает координаты текстуры для вершины, которая в данный момент определена для пользовательского примитива. Вы предоставляете буфер для записи данных, а также позицию UV в текстуре, которую необходимо использовать.

Синтаксис:

vertex_texcoord(buffer, u, v);

Аргумент Тип Описание
buffer Идентификатор буфера вершин The buffer to write the information to.
u Реально The first texture coordinate to use (0 - 1).
v Реально The second texture coordinate to use (0 - 1).

Возвращается:

N/A

Пример:

vertex_texcoord(buff, 0, 0);

Приведенный выше код установит значения UV для определяемой вершины.
