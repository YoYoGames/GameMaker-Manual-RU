vertex_create_buffer_ext

Как и в случае с функцией vertex_create_buffer()эта функция создаст новый буфер вершин. Это специальный растущий буфер, созданный GameMaker, который предварительно отформатирован для использования при создании примитивов для использования с шейдерами. Вы можете указать начальный размер буфера (в байтах), и функция вернет индекс буфера, который будет использоваться во всех последующих обращениях к буферу.

Синтаксис:

vertex_create_buffer_ext(size);

АргументТип Описание
size The initial size of the buffer (in bytes).

Возвращается:

Vertex Buffer ID

Пример:

v_buff = vertex_create_buffer_ext(1024 * 1024);

Приведенный выше код создаст новый буфер вершин, первоначально размером 1 МБ, и сохранит его хэндл в переменной "v_buff".

=======

vertex_create_buffer_ext

This function creates a new vertex buffer and sets its initial size in bytes.

This is a special grow buffer created by GameMaker which is pre-formatted for use when building primitives for use with shaders. You can specify an initial starting size for the buffer (in bytes) and it will return an index for the buffer which should then be used in all further calls to the buffer.

 

Syntax:

vertex_create_buffer_ext(size);

Argument Type Description
size Real The initial size of the buffer (in bytes).

 

Returns:

Vertex Buffer

 

Example:

v_buff = vertex_create_buffer_ext(1024 * 1024);

The above code creates a new vertex buffer, initially 1MB in size, and store it in the variable v_buff.
