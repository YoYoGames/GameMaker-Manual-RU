конец_вершины

С помощью этой функции вы завершаете создание пользовательского примитива. После вызова этой команды примитив может быть использован в функции vertex_submit() для использования в шейдере или вы можете заморозить буфер (делая вершинный буфер доступным только для чтения и намного быстрее) с помощью функции vertex_freeze().

Синтаксис:

vertex_end(buffer);

=======

vertex_end

This function ends the building of a custom primitive that was started with vertex_begin.

Once you call this command the primitive can be used in the vertex_submit function for use in a shader or you can freeze the buffer (making the vertex buffer used read-only and much faster) using vertex_freeze.

 

Syntax:

vertex_end(buffer);
