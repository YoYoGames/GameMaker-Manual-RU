draw_primitive_begin

Эта функция должна быть вызвана до того, как вы сможете определить какие-либо примитивы. Существует 6 типов примитивов, которые можно определить как следующие константы:

Примитивный тип Константа
Постоянная Описание
pr_pointlist Список пунктов
pr_linelist Список строк
pr_linestrip Линейная полоса
pr_trianglelist Список треугольников
pr_trianglestrip Треугольная полоса
pr_trianglefan Треугольный веер

На следующем изображении показано, как они должны выглядеть, а также порядок, в котором следует определять вершины:

Обратите внимание, что на некоторых платформах (Windows, UWP, XBox) тип pr_trianglefan не поддерживается изначально, поэтому GameMaker выполняет преобразование при компиляции игры, чтобы заставить их работать. Это означает, что на этих платформах тип pr_trianglefan будет использоваться гораздо медленнее, чем на других. Также обратите внимание, что для использования этой функции на HTML5 необходимо включить WebGL в опциях игры.

Синтаксис:

draw_primitive_begin(kind)

Аргумент Тип Описание
kind Примитивный тип Константа The kind of primitive you are going to draw.

Возвращается:

N/A

Пример:

var _steps = 20;
var _xx = 50;
var _yy = 50;
var _radius = 30;
draw_primitive_begin(pr_trianglefan);
draw_vertex(_xx, _yy);
for(var i = 0; i <= _steps; ++i)
{
    draw_vertex(_xx + lengthdir_x(_radius, 270 * i / _steps), _yy + lengthdir_y(_radius, 270 * i / _steps));
}
draw_primitive_end();

Приведенный выше код нарисует три четверти окружности, созданной из примитивов.
