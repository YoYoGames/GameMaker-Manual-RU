вершина_начало

С помощью этой функции вы начинаете определение пользовательского примитива. Вы назначаете буфер для записи примитива и формат вершин, который будет использоваться (предварительно определенный с помощью функций формата вершин). Затем вы определяете необходимые точки для каждой вершины примитива перед вызовом функции vertex_end() для завершения создания примитива.

Синтаксис:

vertex_begin(buffer, format);

Аргумент Тип Описание
buffer Идентификатор буфера вершин The buffer to be written to.
format Идентификатор формата вершины The vertex format to use.

Возвращается:

N/A

Пример:

vertex_format_begin();
vertex_format_add_position();
vertex_format_add_colour();
vertex_format_add_textcoord();
v_format = vertex_format_end();
v_buff = vertex_create_buffer();
vertex_begin(v_buff, v_format);

Приведенный выше код определит новый формат вершин, создаст новый буфер и запустит процесс определения нового примитива.
