вершинная_позиция_3d

Эта функция добавит данные о трехмерной позиции к вершине, которая в данный момент определяется для пользовательского примитива. Вы предоставляете буфер для записи данных, а также позиции x, y и z для рисования.

Синтаксис:

vertex_position_3d(buffer, x, y, z);

Аргумент Тип Описание
buffer Идентификатор буфера вершин The buffer to write the information to.
x Реально The x position.
y Реально The y position.
z Реально The z position.

Возвращается:

N/A

Пример:

vertex_position_3d(buff, x - 100, y - 125, 0);

Приведенный выше код установит позицию определяемой вершины.
