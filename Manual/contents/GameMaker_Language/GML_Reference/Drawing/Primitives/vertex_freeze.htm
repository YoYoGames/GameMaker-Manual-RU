вершинная_заморозка

Эта функция может быть использована для "замораживания" буфера вершин. Этот буфер становится доступным только для чтения, то есть, если вам понадобится изменить его, вам придется удалить весь буфер и создать его заново. Замороженный буфер может быть передан в шейдер быстрее, чем обычный динамический буфер, и рекомендуется для тех эффектов, которые требуют неизменного пользовательского примитива на протяжении всего уровня или игры.

Функция вернет 0 при успешном замораживании указанного вершинного буфера, однако если по какой-либо причине это не удалось, она вернет -1.

Синтаксис:

vertex_freeze(buffer);

=======

vertex_freeze

This function can be used to "freeze" a vertex buffer.

This vertex buffer becomes read-only, meaning that should you need to change it, you would have to delete the whole buffer and re-create it. A frozen vertex buffer can be submitted to the shader faster than a normal, dynamic buffer and is recommended for those effects that require an unchanging custom primitive for the duration of a level or the game.

The function will return 0 on successfully freezing the specified vertex buffer, however if it failed for any reason, it will return -1.

 

Syntax:

vertex_freeze(buffer);
