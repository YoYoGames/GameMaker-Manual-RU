вершинная_заморозка

Эта функция может быть использована для "замораживания" буфера вершин. Этот буфер становится доступным только для чтения, то есть, если вам понадобится изменить его, вам придется удалить весь буфер и создать его заново. Замороженный буфер может быть передан в шейдер быстрее, чем обычный динамический буфер, и рекомендуется для тех эффектов, которые требуют неизменного пользовательского примитива на протяжении всего уровня или игры.

Функция вернет 0 при успешном замораживании указанного вершинного буфера, однако если по какой-либо причине это не удалось, она вернет -1.

Синтаксис:

vertex_freeze(buffer);

Аргумент Тип Описание
buffer Идентификатор буфера вершин The vertex buffer to freeze.

Возвращается:

Real (0 for success, -1 for failure)

Пример:

// Create event
vertex_freeze(vbuff);

Приведенный выше код заморозит заданный буфер вершин в событии Create, чтобы его можно было использовать как "статический" буфер в других событиях для более быстрого доступа.
