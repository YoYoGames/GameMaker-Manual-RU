вершинная_позиция

Эта функция добавит данные о двумерной позиции к вершине, которая в данный момент определена для пользовательского примитива. Вы предоставляете буфер для записи данных, а также позиции x и y для рисования.

Синтаксис:

vertex_position(buffer, x, y);

=======

vertex_position

This function adds 2D position data to the vertex currently being defined for the custom primitive.

You supply the vertex buffer to write the data into as well as the x and y components of the vertex position.

 

Syntax:

vertex_position(buffer, x, y);
