вершинная_позиция

Эта функция добавит данные о двумерной позиции к вершине, которая в данный момент определена для пользовательского примитива. Вы предоставляете буфер для записи данных, а также позиции x и y для рисования.

Синтаксис:

vertex_position(buffer, x, y);

Аргумент Тип Описание
buffer Идентификатор буфера вершин The buffer to write the information to.
x Реально The x position.
y Реально The y position.

Возвращается:

N/A

Пример:

vertex_position(buff, x - 100, y - 125);

Приведенный выше код установит позицию определяемой вершины.
