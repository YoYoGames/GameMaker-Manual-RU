vertex_format_add_texcoord

Сообщите GameMaker, чтобы он принимал данные о положении текстуры (u и v) как часть создаваемого нового формата вершин.

Синтаксис:

vertex_format_add_texcoord();

Возвращается:

Пример:

vertex_format_begin();
vertex_format_add_texcoord();
vertex_format_add_position_3d();
my_format = vertex_format_end();

Приведенный выше код создаст новый формат вершин только со значениями текстуры и позиции, а затем сохранит идентификатор формата в переменной "my_format".
