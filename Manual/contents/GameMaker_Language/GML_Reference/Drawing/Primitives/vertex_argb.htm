вершина_argb

Эта функция устанавливает значения ARGB для вершины, определяемой в данный момент для пользовательского примитива. Вы предоставляете буфер для записи данных, а также значения красного, зеленого, синего и альфа, которые вы хотите использовать, в виде одного 32-битного целого без знака - альфа в старших 8 битах, затем красный, зеленый и, наконец, синий в младших 8 битах. Вы можете использовать шестнадцатеричную нотацию ($AARRGGBB) как вещественное число или использовать любую из функций make_colour_*() для определения значения цвета.

Синтаксис:

vertex_argb(buffer, argb);

Аргумент Тип Описание
buffer Идентификатор буфера вершин The buffer to write the information to.
argb Цвет The ARGB value to set.

Возвращается:

N/A

Пример:

vertex_argb(buff, $FFFFFFFF);

Приведенный выше код установит значения ARGB определяемой вершины в белый цвет.
