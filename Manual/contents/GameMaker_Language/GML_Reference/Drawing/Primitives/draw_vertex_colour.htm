draw_vertex_colour

Эта функция определяет положение вершины для примитива, с ее собственным цветом и настройкой альфа. Окончательный вид примитива будет зависеть от типа примитива, выбранного для рисования, и порядка добавления вершин к нему (см. draw_primitive_begin() более подробную информацию), и вершины с разными цветами и альфа будут плавно переходить одна в другую. Чтобы завершить и нарисовать примитив, необходимо вызвать команду draw_primitive_end().

Синтаксис:

draw_vertex_colour(x, y, col, alpha)

АргументТип Описание
x The x coordinate of the vertex.
y The y coordinate of the vertex.
col The colour to draw this vertex with.
alpha The alpha to draw this vertex with (0-1).

Возвращается:

Пример:

draw_primitive_begin(pr_trianglelist);
draw_vertex_colour(100, 100, c_blue, 0.1);
draw_vertex_colour(100, 200, c_red, 0.1);
draw_vertex_colour(150, 150, c_green, 1);
draw_primitive_end();

Приведенный выше код нарисует полупрозрачный треугольник, каждая вершина которого окрашена в свой цвет.
