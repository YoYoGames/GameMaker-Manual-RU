вершинный_создать_буфер

С помощью этой функции вы можете создать новый вершинный буфер. Это специальный растущий буфер, созданный GameMaker, который предварительно отформатирован для использования при построении примитивов (например, для использования с шейдерами). Функция вернет индекс буфера, который будет использоваться во всех последующих обращениях к нему.

При использовании буфера вершин, созданного с помощью этой функции, вы просто вызываете функцию vertex_begin() чтобы начать присваивать ему вершинные данные для определения вашего пользовательского примитива, который затем будет храниться в буфере, готовый к передаче в шейдер. Буфер может быть повторно использован при необходимости (если вы не использовали функцию vertex_freeze() ), при этом каждый вызов vertex_begin() стирает предыдущие данные буфера, готовые к приему новых данных.

Синтаксис:

vertex_create_buffer();

Возвращается:

Vertex Buffer ID

Пример:

v_buff = vertex_create_buffer();

Приведенный выше код создаст новый буфер вершин и сохранит его хэндл в переменной "v_buff".
