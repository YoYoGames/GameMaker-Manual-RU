нарисовать_линию_ширины_цвета

С помощью этой функции можно нарисовать линию определенной ширины со смешением цвета между цветом 1 в первой точке и цветом 2 во второй точке. Настройки цвета будут перекрывать базовый цвет, заданный с помощью функции draw_set_colour(). Обратите внимание, что для построения линии с нужными размерами могут потребоваться разные значения (+/-1 по x, y) из-за различий в различных поддерживаемых платформах.

ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого.

Синтаксис:

draw_line_width_colour(x1, y1, x2, y2, w, col1, col2);

АргументТип Описание
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line.
y2 The y coordinate of the end of the line.
w The width in pixels of the line.
col1 The colour of the start of the line.
col2 The colour of the end of the line.

Возвращается:

Пример:

draw_line_width_colour(50, 50, 300, 50, 4, c_red, c_blue);

Это позволит нарисовать горизонтальную линию от (50,50) до (300,50), шириной четыре пикселя, с плавным градиентом от красного к синему.
