draw_healthbar

С помощью этой функции можно нарисовать цветную полосу для отображения постоянной величины. Хотя в функции используется слово "healthbar", вы можете использовать ее для отображения чего угодно, если отображаемая величина представляет собой процентное значение от 0 до 100 (процент от любого значения можно рассчитать по формуле (CURRENT_Value / MAXIMUM_value) * 100), поэтому, например, вы можете использовать ее для отображения силы, здоровья, маны, времени или чего-либо еще, что может быть полезно при отображении в виде столбика.

Вы можете установить различные параметры, которые изменяют визуальный аспект полосы, и они показаны на рисунке ниже (обратите внимание, что здесь используются цвета c_green для максимального значения и c_red для минимального значения, но вы можете использовать цвета по своему усмотрению):

ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого.

Синтаксис:

draw_healthbar(x1, y1, x2, y2, amount, backcol, mincol, maxcol, direction, showback, showborder);

Аргумент Тип Описание
x1 Реально The x coordinate of the left of the healthbar
y1 Реально The y coordinate of the top of the healthbar
x2 Реально The x coordinate of the right of the healthbar
y2 Реально The y coordinate of the bottom of the healthbar
amount Реально The variable which defines total health (between 0 and 100)
backcol Цвет The colour of the background for the bar
mincol Цвет The colour of the bar when at no health (0)
maxcol Цвет The colour of the bar when at full health (100)
direction Реально Where the bar is "anchored" (0 = left, 1 = right, 2 = top, 3 = bottom)
showback Булево Whether the bar's background is drawn (true) или нет (false). Если false, то аргумент backcol не принимается во внимание.
showborder Булево Whether the bar's elements have a 1px wide black border (true) или нет (false).

Возвращается:

N/A

Пример:

var pc;
pc = (hp / max_hp) * 100;
draw_healthbar(100, 100, 500, 200, pc, c_black, c_red, c_lime, 0, true, true)

Приведенный выше код использует процентное значение, полученное из переменных "hp" и "hp_max", для рисования стандартной красно-зеленой панели здоровья.
