draw_roundrect_ext

С помощью этой функции можно нарисовать либо контур закругленного прямоугольника, либо заполненный закругленный прямоугольник, где позиция (x1,y1) - левый верхний угол, а позиция (x2,y2) - правый нижний угол. Вы также должны указать значения радиуса для осей x и y (в пикселях), и углы будут скруглены на эти величины. Вы можете определить, насколько точной будет отрисовка углов, с помощью функции draw_set_circle_precision(). Обратите внимание, что для рисования прямоугольника нужных размеров могут потребоваться другие значения (+/-1 по осям x, y, ширины или высоты) из-за различий в разных поддерживаемых платформах.

ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого.

Синтаксис:

draw_roundrect_ext(x1, y1, x2, y2, xrad, yrad, outline);

Аргумент Тип Описание
x1 Реально The x coordinate of the top left corner of the rounded rectangle (as if it were a normal rectangle).
y1 Реально The y coordinate of the top left corner of the rounded rectangle (as if it were a normal rectangle).
x2 Реально The x coordinate of the bottom right corner of the rounded rectangle (as if it were a normal rectangle).
y2 Реально The y coordinate of the bottom right corner of the rounded rectangle (as if it were a normal rectangle).
xrad Реально The radius of the curve along the x axis from the rectangle corners.
yrad Реально The radius of the curve along the y axis from the rectangle corners.
outline Булево Whether the rounded rectangle is drawn filled (false) or as a one pixel wide outline (true).

Возвращается:

N/A

Пример:

var dist = point_distance(x, y, mouse_x, mouse_y) / 10;
draw_set_colour(c_lime);
draw_roundrect_ext(x - 50, y - 50, x + 50, y + 50, dist, dist, 0);

Это нарисует заполненный закругленный прямоугольник, углы которого будут закруглены на величину, основанную на положении мыши относительно экземпляра, выполняющего рисование.
