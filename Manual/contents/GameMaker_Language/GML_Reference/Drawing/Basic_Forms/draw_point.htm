точка_рисунка

С помощью этой функции вы можете нарисовать один пиксель в любом месте экрана, используя установленные в данный момент цвет рисунка и альфу.

Синтаксис:

draw_point(x, y);

АргументТип Описание
x The x coordinate of the point to be drawn.
y The y coordinate of the point to be drawn.

Возвращается:

Пример:

draw_set_colour(c_yellow);
draw_point(100,100);

Это нарисует желтый пиксель в позиции (100,100).
