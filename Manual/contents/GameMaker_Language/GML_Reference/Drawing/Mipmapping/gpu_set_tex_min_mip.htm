gpu_set_tex_min_mip

С помощью этой функции вы можете установить минимальный уровень mipmap, который используется в настоящее время, где значение 0 - это самое высокое разрешение, 1 - использование первого mipmap, 2 - второго и т.д....

Синтаксис:

gpu_set_tex_min_mip(minmip);

АргументТип Описание
minmip The minimum mipmap level to use

Возвращается:

Пример:

if gpu_get_tex_min_mip() != 0
{
    gpu_set_tex_min_mip(0);
}

Приведенный выше код проверит текущий минимальный уровень mipmap и, если он не равен 0, то будет установлен на 0.
