gpu_set_tex_max_aniso

С помощью этой функции можно установить текущий максимальный уровень анизотропии при использовании режима фильтра tf_anisotropic (см. gpu_get_tex_mip_filter() для более подробной информации). Вводимое значение должно находиться в диапазоне от 1 до 16.

Синтаксис:

gpu_set_tex_max_aniso(maxaniso);

АргументТип Описание
maxaniso The maximum level for anisotropic filtering (default: 16)

Возвращается:

Пример:

if gpu_get_tex_max_aniso() != 8
{
    gpu_set_tex_max_aniso(8);
}

Приведенный выше код проверит текущий максимальный уровень анизотропной фильтрации, и если он не равен 8, то будет установлен на 8.
