gpu_set_tex_mip_bias_ext

С помощью этой функции вы можете установить значение смещения mipmap для данного шейдерного сэмплера. Вы вводите значение индекса шейдерного сэмплера (возвращаемое функцией shader_get_sampler_index(), а затем значение смещения, где 0 - без смещения, 1 - для первого мипмапа, 2 - для второго мипмапа и т.д.. Эта функция контролирует скорость смены карт мипов и обычно делает текстуры шейдера более размытыми, чем выше значение и чем больше "расстояние", на котором происходит просмотр. Обратите внимание, что эта функция может принимать и отрицательные значения, в этом случае шейдерные текстуры будут более четкими на большем расстоянии, чем меньше значение.

Синтаксис:

gpu_set_tex_mip_bias_ext(sampler_index, bias);

АргументТип Описание
sampler_index The index of the shader sampler to get
bias The mipmap bias value to use (default: 0)

Возвращается:

Пример:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_bias_ext(_sampleIndex) != 0)
{
    gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Приведенный выше код устанавливает смещение фильтра mip в 0 для данного шейдерного текстурного сэмплера, если оно еще не установлено в 0.
