gpu_get_tex_mip_enable_ext

С помощью этой функции вы можете узнать, выключен ли mipmapping, включен ли он для всего или включен только для групп текстур, выбранных в менеджере групп текстур на шейдерном сэмплере. Вы передаете значение индекса шейдерного сэмплера (как возвращается функцией shader_get_sampler_index(), а функция вернет одну из перечисленных ниже констант.

Синтаксис:

gpu_get_tex_mip_enable_ext(sampler_index);

Аргумент Тип Описание
sampler_index Ручка шейдерного пробоотборника The index of the shader sampler to get

Возвращается:

Constant

Постоянная Описание
mip_off Mipmapping отключен.
mip_on Mipmapping для всех текстур включен.
mip_markedonly Mipmapping включен для текстур, у которых он включен в параметрах группы текстур.

Пример:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on)
{
    gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Приведенный выше код включает mipmapping для данного шейдерного текстурного сэмплера, если он еще не включен.
