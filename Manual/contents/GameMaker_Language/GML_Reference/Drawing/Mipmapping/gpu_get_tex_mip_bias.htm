gpu_get_tex_mip_bias

С помощью этой функции можно получить значение смещения mipmap, где значение 0 равно отсутствию смещения, 1 равно первой mipmap, 2 равно второй mipmap и т.д.. Это контролирует скорость смены мип-карт и обычно делает рендеринг текстур более размытым, чем выше значение и чем больше "расстояние". Обратите внимание, что этот параметр может возвращать и отрицательные значения, в этом случае отрисованные текстуры будут более четкими на большем расстоянии, чем меньше значение.

Синтаксис:

gpu_get_tex_mip_bias();

Возвращается:

(default: 0)

Пример:

if gpu_get_tex_mip_bias() != 0
{
    gpu_set_tex_mip_bias(0);
}

Приведенный выше код проверяет текущее смещение mipmap и, если оно не равно 0, то устанавливается в 0.
