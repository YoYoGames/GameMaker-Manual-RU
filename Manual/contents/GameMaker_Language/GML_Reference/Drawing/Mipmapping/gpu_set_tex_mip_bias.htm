gpu_set_tex_mip_bias

С помощью этой функции вы можете установить значение смещения мипмапы, где значение 0 равно отсутствию смещения, 1 равно первой мипмапе, 2 равно второй мипмапе и т.д.. Это контролирует скорость смены мип-карт и обычно делает рендеринг текстур более размытым, чем выше значение и чем больше "расстояние". Обратите внимание, что эта функция может принимать и отрицательные значения, в этом случае отрисованные текстуры будут более четкими на большем расстоянии, чем меньше значение.

Синтаксис:

gpu_set_tex_mip_bias(bias);

АргументТип Описание
bias The mipmap bias value to use (default: 0)

Возвращается:

Пример:

if gpu_get_tex_mip_bias() != 0
{
    gpu_set_tex_mip_bias(0);
}

Приведенный выше код проверяет текущее смещение mipmap и если оно не равно 0, то устанавливается на 0.
