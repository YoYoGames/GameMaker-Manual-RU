gpu_get_tex_max_mip

С помощью этой функции вы можете получить установленный на данный момент максимальный уровень mipmap, который должен быть использован, где значение 0 - самое высокое разрешение, 1 - использование первого mipmap, 2 - второго и т.д.. Обратите внимание, что это может быть полезно для предотвращения артефактов растекания при рендеринге текстур, например, установив границу страницы текстуры на 8px и затем установив максимальный уровень mipmap на 3, вы не получите никаких проблем с растеканием на больших расстояниях рендеринга.

Синтаксис:

gpu_get_tex_max_mip();

Возвращается:

(default: 16)

Пример:

if gpu_get_tex_max_mip() != 4
{
    gpu_set_tex_max_mip(4);
}

Приведенный выше код проверит текущий максимальный уровень mipmap, и если он не равен 4, то будет установлен на 4.
