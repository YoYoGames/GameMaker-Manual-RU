gpu_get_tex_max_mip_ext

С помощью этой функции вы можете получить установленный на данный момент максимальный уровень mipmap, который будет использоваться для данного шейдерного сэмплера. Вы вводите значение индекса шейдерного сэмплера (как возвращается функцией shader_get_sampler_index(), и функция вернет текущий максимальный уровень mipmap, где значение 0 - самое высокое разрешение, 1 - использование первого mipmap, 2 - второго и т.д.. Обратите внимание, что это может быть полезно для предотвращения артефактов растекания при рендеринге текстур, например, установив границу страницы текстуры на 8px и затем установив максимальный уровень mipmap на 3, вы не получите никаких проблем с растеканием на больших расстояниях рендеринга.

Синтаксис:

gpu_get_tex_max_mip_ext(sampler_index);

АргументТип Описание
sampler_index The index of the shader sampler to get

Возвращается:

(default: 16)

Пример:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_max_mip_ext(_sampleIndex) != 4)
{
    gpu_set_tex_max_mip_ext(_sampleIndex, 4);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Приведенный выше код устанавливает максимальный уровень mipmap на 4 для данного шейдерного текстурного сэмплера, если он еще не установлен на 4.
