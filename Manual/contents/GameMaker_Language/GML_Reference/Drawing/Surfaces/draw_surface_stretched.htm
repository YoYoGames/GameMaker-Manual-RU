нарисовать_поверхность_растянутой

Эта функция просто берет поверхность и растягивает ее по заданной ширине и высоте так, чтобы она заняла заданную область. Как и в случае с draw_surface() вы можете указать поверхность, затем позицию x / y в комнате, в которой будет нарисована поверхность, и, наконец, ширину и высоту (которые должны быть пиксельными значениями).

Синтаксис:

draw_surface_stretched(id, x, y, w, h);

Аргумент Тип Описание
id Идентификатор поверхности The unique ID value of the surface to draw.
x Реально The x position of where to draw the surface.
y Реально The y position of where to draw the surface.
w Реально The width at which to draw the surface.
h Реально The height at which to draw the surface.

Возвращается:

N/A

Пример:

draw_surface_stretched(surf, 10, 10, 100, 100);

Это нарисует поверхность, индексированную в переменной "surf", с левым углом в точке (10,10). Ее ширина и высота установлены в 100, то есть столько места она будет занимать независимо от фактической ширины и высоты поверхности.
