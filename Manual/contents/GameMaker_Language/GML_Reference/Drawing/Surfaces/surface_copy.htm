surface_copy

Эта функция просто берет изображение с одной поверхности и копирует его на другую поверхность в указанной локальной позиции внутри этой поверхности (где позиция (0,0) - левый верхний угол поверхности назначения). Если на поверхности назначения уже есть информация, она будет перезаписана копией, при этом функция никак не изменяет исходную поверхность.

ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстуры. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней.

Синтаксис:

surface_copy(destination, x, y, source);

Аргумент Тип Описание
destination Идентификатор поверхности The ID of the surface to copy the other surface to.
x Реально The x position to copy to.
y Реально The y position to copy to.
source Идентификатор поверхности The ID of surface to be copied.

Возвращается:

N/A

Пример:

if view_current == 0
{
    surface_copy(surf, 0, 0, temp_surf);
}
else
{
    draw_surface(surf, 0, 0);
}

Приведенный выше код проверяет текущий вид и, если это вид[0], копирует поверхность, индексированную в переменной "temp_surf", на поверхность, индексированную в переменной "surf". Если текущий вид не является видом[0], то на экран выводится поверхность "surf".
