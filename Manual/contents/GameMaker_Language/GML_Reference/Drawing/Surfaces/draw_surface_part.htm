нарисовать_поверхность_части

С помощью этой функции можно нарисовать часть любой поверхности в заданном положении в помещении. Как и в случае с draw_surface(), вы можете указать поверхность, но затем вам нужно указать относительные координаты внутри поверхности области, которую нужно выбрать для рисования. Это означает, что левая позиция 0 и верхняя позиция 0 будут левым верхним углом поверхности, и все дальнейшие координаты должны быть взяты из этой позиции.

Синтаксис:

draw_surface_part(id, left, top, w, h, x, y);

=======

draw_surface_part

This function draws part of any surface at a given position within the room.

As with draw_surface you can specify a surface, but you then need to specify the relative coordinates within the surface of an area to select for drawing. This means that a left position of 0 and a top position of 0 would be the top left corner of the surface and all further coordinates should be taken from that position.

 

Syntax:

draw_surface_part(surface, left, top, w, h, x, y);
