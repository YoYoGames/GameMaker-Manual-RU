surface_get_target

С помощью этой функции можно получить идентификатор текущей поверхности, на которую производится отрисовка. Если вы создали эту поверхность с помощью функции surface_create()то это будет целое положительное число, которое является уникальным идентификатором этой поверхности.

Если пользовательская поверхность не установлена, но при этом включено значение application_surface по умолчанию, то функция обычно возвращает 0. Однако, если вы ранее манипулировали с application_surface, вы можете проверить, что полученный вами ID не совпадает с текущим значением переменной application_surface, в случае если это значение больше не 0.

Если пользовательская поверхность не задана, и поверхность по умолчанию application_surface также была отключена, то функция вернет -1.

Синтаксис:

surface_get_target();

Возвращается:

Surface ID or -1 (if no target surface is set)

Пример:

if surface_get_target() != -1
=======

surface_get_target

This function gets the surface that's currently set as the drawing target.

A couple of values can be returned: 

  • If you created that surface using surface_create, then that surface will be returned.
  • If no custom surface is set, but the default application surface is still enabled, then the function will return the application surface. However, if you've been manipulating the application surface previously you might wish to if this still matches the current value of the application_surface variable.
  • If no custom surface is set and the default application surface has also been disabled, then the function will return -1.

 

Syntax:

surface_get_target();

 

Returns:

Surface or -1 (if no target surface is set)

 

Example:

if surface_get_target() != -1
