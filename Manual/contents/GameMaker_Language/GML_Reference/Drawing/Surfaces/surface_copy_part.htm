поверхность_копирования_части

Эта функция просто берет изображение с одной поверхности и копирует его на другую поверхность в указанной локальной позиции внутри этой поверхности (где позиция (0,0) - это левый верхний угол целевой поверхности). Вы можете указать локальную позицию x и y для копирования, а также ширину и высоту участка. Обратите внимание, что это координаты, основанные на размере поверхности, а не на положении, в котором поверхность рисуется в комнате. Если на поверхности назначения уже есть информация, она будет перезаписана копией, и функция никак не изменит исходную поверхность.

ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстуры. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней.

Синтаксис:

surface_copy_part(destination, x, y, source, xs, ys, ws, hs);

Аргумент Тип Описание
destination Идентификатор поверхности The ID of the surface to copy the other surface to.
x Реально The x position to copy to.
y Реально The y position to copy to.
source Идентификатор поверхности The ID of the surface to be copied.
xs Реально The x position in the source surface to copy from.
ys Реально The y position in the source surface to copy from.
ws Реально The width of the area in the source surface to copy from.
hs Реально The height of the area in the source surface to copy from.

Возвращается:

N/A

Пример:

if view_current == 0
{
    surface_copy_part(surf, 0, 0, temp_surf, 0, 0, view_xview[1] - mouse_x, view_yview[1] - mouse_y);
}
else
{
    draw_surface(surf, 0, 0);
}

Приведенный выше код проверяет текущий вид, и если это вид[0], то копирует поверхность, индексированную в переменной "temp_surf", на поверхность, индексированную в переменной "surf". Скопированная область соответствует прямоугольнику, образованному относительным положением мыши в пределах поверхности, как это было бы нарисовано в представлении[1]. Если текущий вид не является видом[0], то поверхность "surf" рисуется на экране.
