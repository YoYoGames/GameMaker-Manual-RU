поверхность_getpixel

Эта функция может использоваться для получения цвета определенного пикселя на поверхности, используя локальные координаты поверхности, где (0,0) - левый верхний угол. Эту функцию не следует использовать очень часто, так как она работает очень медленно и может вызвать паузу в игре.

ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстуры. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней.

Синтаксис:

surface_getpixel(surface_id, x, y);

=======

surface_getpixel

This function can be used to get the colour of a specific pixel from a surface, using the local coordinates of the surface where (0, 0) is the top left corner. This function should not be used very often as it is extremely slow and may cause a pause in your game.

The data type returned by this function will depend on the format of the given surface:

Formats Function Return
surface_rgba8unorm (default)
surface_rgba4unorm
surface_r8unorm
surface_rg8unorm 		For these formats, the function will return a regular colour value. Any unused channels are set to 0.
surface_rgba16float
surface_r16float
surface_rgba32float
surface_r32float		 For these formats, the function will return an array with 3 values (R, G, B) with each being a 32-bit float. Any unused channels are set to 0.

To get the pixel colour with the alpha channel, use surface_getpixel_ext.

 

Syntax:

surface_getpixel(surface_id, x, y);

АргументТип Описание
surface_id The ID of the surface.
x The x position on the surface from which to get the pixel.
y The y position on the surface from which to get the pixel.

Возвращается:

Colour Constant

Пример:

col = surface_getpixel(surf, 56, 78 );

Это вернет цвет пикселя в координатах (56,78) поверхности, индексированной в переменной "surf".

======= surface_id Surface The surface. x Real The x position on the surface to get the pixel from. y Real The y position on the surface to get the pixel from.

 

Returns:

Colour or Array

 

Example:

col = surface_getpixel(surf, 56, 78);

This will return the colour of the pixel at coordinates (56, 78) of the surface stored in the variable surf.
