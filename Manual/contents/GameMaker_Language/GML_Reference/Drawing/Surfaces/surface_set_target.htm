поверхность_задать_цель

С помощью этой функции вы устанавливаете все дальнейшее рисование на целевую поверхность, а не на экран, и таким образом вы можете указать GameMaker рисовать только определенные вещи на указанной поверхности. Обратите внимание, что если вы не вызовете функцию surface_reset_target() после завершения работы, на экране ничего не будет нарисовано, так как все дальнейшее рисование (даже в других случаях) будет происходить на поверхности. Вы также должны понимать, что ничего не будет видно, если сама поверхность не будет нарисована на экране в событии draw экземпляра. Вы также можете проверить возвращаемое значение этой функции в качестве инструмента отладки, чтобы проверить, была ли установлена цель поверхности или нет. Возвращаемое значение 0 означает неудачу в установке цели, а любое другое положительное значение - успех.

Следует отметить, что поверхности складываются в стек, поэтому вы не можете переходить от цели к цели, а затем вернуться к обычной цели рисования в конце, но скорее вы должны открывать и закрывать цели рендеринга. Например, это не будет работать правильно:

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();

Вместо этого вы должны сбросить цель для каждой из заданных поверхностей, точно так же, как вы должны использовать открывающие и закрывающие скобки {} для блоков кода. Таким образом, вышеприведенное должно быть написано либо так, либо так:

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_reset_target();
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();

или вот так:

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();
surface_reset_target();

ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстур. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней.

Синтаксис:

surface_set_target(surface_id);

АргументТип Описание
surface_id The ID of the surface to set as the drawing target.

Возвращается:

Пример:

if view_current = 0
{
    surface_set_target(surf);
    with (obj_effect)
    {
        draw_self();
    }
    surface_reset_target();
}
else
{
    draw_surface(surf, 0, 0);
}

Приведенный выше код проверяет, какой вид в данный момент рисуется, и если это вид[0], то устанавливает цель рисования на поверхность и рисует все экземпляры объекта "obj_Effect", после чего снова сбрасывает цель рисования. Если вид не является видом[0], то поверхность рисуется на экране.
