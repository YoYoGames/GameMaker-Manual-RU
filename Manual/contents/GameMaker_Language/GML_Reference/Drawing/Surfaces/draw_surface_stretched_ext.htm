draw_surface_stretched_ext

Эта функция делает то же самое, что и draw_surface_stretched() с добавлением возможности установить смешивание цветов и значение альфа для поверхности при ее рисовании (аналогично функции draw_surface_ext()).

Синтаксис:

draw_surface_stretched_ext(id, x, y, w, h, col, alpha);

Аргумент Тип Описание
id Идентификатор поверхности The unique ID value of the surface to draw.
x Реально The x position of where to draw the surface.
y Реально The y position of where to draw the surface.
w Реально The width at which to draw the surface.
h Реально The height at which to draw the surface.
colour Цвет The colour with which to colour the surface.
alpha Реально The alpha with which to blend the surface.

Возвращается:

N/A

Пример:

draw_surface_stretched_ext(surf, x, y, 200, 200, c_white, 0.5);

Это нарисует заданную поверхность с левым углом в позиции экземпляра x/y, и она будет растянута на область 200x200 пикселей без наложения, но с частичной прозрачностью.
