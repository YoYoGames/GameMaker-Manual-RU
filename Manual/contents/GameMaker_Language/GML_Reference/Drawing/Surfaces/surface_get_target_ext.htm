surface_get_target_ext

Эта функция извлекает идентификатор поверхности, назначенной одной из 4 целей рендеринга, доступных для поверхностей. Вы указываете индекс цели рендеринга для проверки, и функция вернет -1, если поверхность не назначена, или целочисленное значение >= 0, представляющее ID назначенной поверхности (как возвращается функцией surface_create()).

Синтаксис:

surface_get_target_ext(index);

АргументТип Описание
index The render target index to check (from 0 to 3).

Возвращается:

Surface ID or -1 (if no target surface is set)

Пример:

if surface_get_target_ext(0) == -1
{
    surface_set_target_ext(0, global.Surf);
}

Приведенный выше код сначала проверит, была ли шейдер render target 0 установлен на поверхность, и если нет, то назначит ее.
