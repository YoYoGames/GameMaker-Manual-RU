поверхность_свободная

Когда вы работаете с поверхностями, вы всегда должны использовать эту функцию, когда вы закончили работу с ними. Поверхности занимают место в памяти, поэтому их необходимо удалять, обычно в конце комнаты, но это можно сделать в любое время, в зависимости от того, для чего они используются. Невыполнение этого требования может привести к утечке памяти, что в конечном итоге приведет к замедлению и краху вашей игры.

ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстуры. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней.

Синтаксис:

surface_free(surface_id);

АргументТип Описание
surface_id The ID of the surface to be freed.

Возвращается:

Пример:

=======

surface_free

This function frees a surface from memory.

When you are working with surfaces, you should always use this function whenever you are finished using them. Surfaces take up space in memory and so need to be removed, normally at the end of a room, but it can be done at any time depending on the use you put them to. Failure to do so can cause memory leaks which will eventually slow down and crash your game.

 

Syntax:

surface_free(surface);

Argument Type Description
surface Surface The surface to be freed.

 

Returns:

N/A
