draw_surface_tiled

Эта функция возьмет поверхность и многократно проложит плитку по всей комнате, начиная с координат, которые вы укажете в функции.

Синтаксис:

draw_surface_tiled(id, x, y);

Аргумент Тип Описание
id Идентификатор поверхности The unique ID value of the surface to draw.
x Реально The x position of where to draw the surface.
y Реально The y position of where to draw the surface.

Возвращается:

N/A

Пример:

draw_surface_tiled(surf, x, y);

Это нарисует поверхность, индексированную в "surf", в собственном положении экземпляра по x и y, а также плитку во всех направлениях в комнате.
