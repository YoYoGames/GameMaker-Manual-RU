application_surface

Эта глобальная встроенная переменная может быть использована для доступа к поверхности приложения для использования в любой из функций поверхности. Эта поверхность постоянно доступна и является местом, где игра рисуется GameMaker.

Обычные события Draw(Draw Begin, Draw и Draw End) по умолчанию рисуют на этой поверхности. Другие события Draw (например, Pre/Post Draw и Draw GUI) не рисуют на поверхности приложения. Дополнительные сведения о работе этих событий см. в разделе События рисования.

Синтаксис:

application_surface

Возвращается:

Surface ID

Пример:

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

Приведенный выше код изменит размер поверхности приложения так, чтобы соотношение с слоем GUI было 1:1.
