поверхность_приложения

Эта глобальная встроенная переменная может быть использована для доступа к поверхности приложения для использования в любой из функций поверхности. Эта поверхность постоянно доступна и является местом, где игра рисуется GameMaker.

Обычные события Draw(Draw Begin, Draw и Draw End) по умолчанию рисуют на этой поверхности. Другие события Draw (например, Pre/Post Draw и Draw GUI) не рисуют на поверхности приложения. Дополнительные сведения о работе этих событий см. в разделе События рисования.

Синтаксис:

application_surface;

Возвращается:

Surface ID

Пример:

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

Приведенный выше код изменит размер поверхности приложения так, чтобы соотношение с слоем GUI было 1:1.

=======

application_surface

This global, built-in variable can be used to access the application surface, for use in any of the surface functions. This surface is permanently available and is where the game is drawn by GameMaker.

The regular Draw events (Draw Begin, Draw and Draw End) draw to this surface by default. Other Draw events (like Pre/Post Draw and Draw GUI) do not draw on the application surface. See Draw Events for more information on how those events work.

 

Syntax:

application_surface;

 

Returns:

Surface

 

Example:

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

The above code will resize the application surface to have a 1:1 ratio with the GUI layer.
