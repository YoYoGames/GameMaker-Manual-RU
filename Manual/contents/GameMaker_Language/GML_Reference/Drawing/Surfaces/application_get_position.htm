application_get_position

Если в опциях игры для целевой платформы отмечена опция "поддерживать соотношение сторон", GameMaker автоматически установит позицию рисования для поверхности приложения так, чтобы она отображалась правильно центрированной и масштабированной на заданном дисплее. Однако если вы работаете с этой поверхностью и хотите нарисовать ее самостоятельно, то эта функция дает вам простой способ найти точное место в дисплее или окне, где была нарисована поверхность, чтобы затем вы могли нарисовать ее там самостоятельно, или выровнять по ней изображения GUI или изображения post draw.

Функция вернет массив с четырьмя ключами, где ключи 0 и 1 - это x и y верхнего левого угла поверхности, а ключи 2 и 3 - x и y нижнего правого угла поверхности. поверхности, все относительно размера дисплея или окна.

Синтаксис:

application_get_position();

Возвращается:

Пример:

var a = application_get_position();
xx = a[0];
yy = a[1];
ww = a[2] - a[0];
hh = a[3] - a[1];

Приведенный выше код получит положение поверхности приложения, а также абсолютную ширину и высоту по отношению к окну дисплея, и сохранит их в четырех переменных для дальнейшего использования.

=======

application_get_position

When you have "maintain aspect ratio" ticked in the Game Options for a target platform, GameMaker will automatically set the draw position for the application surface so that it is displayed correctly centered and scaled on the given display. However, if you are manipulating this surface and wish to draw it yourself, then this function gives you an easy way to find exactly where within the display or window that the surface was being drawn so that you can then draw it there yourself, or align GUI images or post draw images to it.

The function returns an array with four keys, where key 0 and 1 are the (x, y) position of the top left-hand corner of the surface, and keys 2 and 3 are the x and y of the bottom right-hand corner of the surface, all relative to the size of the display or window.

 

Syntax:

application_get_position();

 

Returns:

Array

 

Example:

var a = application_get_position();
xx = a[0];
yy = a[1];
ww = a[2] - a[0];
hh = a[3] - a[1];

The above code will get the position of the application surface, as well as the absolute width and height in relation to the display window, and store them in four variables for future use.
