размер_поверхности

Эта функция изменит размер поверхности до заданных размеров (в пикселях). В качестве "surface_id" указывается ранее созданная поверхность, или номер поверхности application_suface, и функция изменит размер этой поверхности. Обратите внимание, что функция не обрезает и не растягивает содержимое поверхности, а уничтожает текущую поверхность и воссоздает ее заново с той же ручкой (surface_id) с новыми размерами, что означает, что ее нужно будет очистить и нарисовать заново (если только это не application_surface, в этом случае GameMaker сделает это автоматически).

ПРИМЕЧАНИЕ: Если вы изменяете размер поверхности приложения, эти изменения не будут зарегистрированы до начала следующего кадра рисования, что означает, что вызов функции surface_get_width() или surface_get_height() в том же событии или шаге вернет предыдущие значения.

Синтаксис:

surface_resize(surface_id, w, h);

АргументТип Описание
surface_id The ID of the surface to change.
w The width of the new surface.
h The height of the new surface.

Пример:

=======

surface_resize

This function resizes a surface to the given dimensions (in pixels).

The "surface_id" is that of a surface you have created previously, or the application_surface, and the function will resize that surface. Note that this will neither crop nor stretch the contents of the surface, but rather it destroys the current surface and recreates it with the same handle (surface_id) with the new dimensions, meaning that it will need to be cleared and drawn to again (unless it's the application_surface in which case GameMaker will do this automatically).

 If you are resizing the application surface, these changes will not be registered until the start of the next draw frame, meaning that calling the surface_get_width or surface_get_height functions in the same event or step will return the previous values.

 

Syntax:

surface_resize(surface_id, w, h);

Argument Type Description
surface_id Surface The surface to resize.
w Real The new width of the surface.
h Real The new height of the surface.

 

Returns:

N/A
