surface_set_target_ext

Эта функция предназначена для использования с функциями шейдеров и устанавливает MRT (0 - 3) для шейдеров родного уровня (DX9, DX11, OpenGL).

ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстур. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней.

Синтаксис:

surface_set_target_ext(index, surface_id);

=======

surface_set_target_ext

This function is for use with the Shader Functions and sets the MRT (0 - 3) for native level shaders (OpenGL ES, OpenGL, DX9, DX11).

 MRT's are not supported on HTML5.

 

Syntax:

surface_set_target_ext(index, surface_id);
