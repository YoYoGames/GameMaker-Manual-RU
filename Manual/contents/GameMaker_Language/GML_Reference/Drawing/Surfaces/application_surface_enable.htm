приложение_поверхность_включено

С помощью этой функции вы можете включить или выключить поверхность приложения. По умолчанию поверхность приложения включена, и все рисование для событий Draw будет выполняться на ней, а поверхность будет рисоваться на экране. Однако на некоторых старых устройствах и определенных чипсетах это может привести к снижению производительности. В таких случаях можно отключить отрисовку на поверхность приложения, и тогда все отрисовки по событиям Draw будут выполняться непосредственно в буфер экрана.

Вы можете использовать эту функцию для включения или отключения поверхности приложения в любое время в вашей игре, но следует учитывать, что для вступления изменений в силу требуется не менее одного кадра (шага). Выключение отключит всю коррекцию соотношения сторонкак установлено в Опциях игры для целевой платформы. Это означает, что у вас должен быть установлен метод, который масштабирует игру до размеров требуемого устройства, окна или экрана. Если вы снова включите его, то на следующем шаге снова вступят в силу параметры игры.

Рекомендуется использовать эту функцию осторожно, поскольку она влияет на то, как все рисуется на экране. В общем случае ее лучше использовать в качестве опции в меню игры или вызывать только один раз в начале игры и писать код в зависимости от того, включена она или выключена.

Синтаксис:

application_surface_enable(enable);

АргументТип Описание
enable Set to enable (true) или отключить (false) поверхность приложения.

Возвращается:

Пример:

if keyboard_check_pressed(vk_space)
{
    if (application_surface_is_enabled())
    {
        application_surface_enable(false);
    }
    else
    {
        application_surface_enable(true);
    }
}

Приведенный выше код проверяет нажатие клавиши и включает или выключает поверхность приложения в зависимости от ее состояния (как в меню опций).
