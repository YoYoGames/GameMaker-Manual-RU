приложение_поверхность_включено

С помощью этой функции вы можете включить или выключить поверхность приложения. По умолчанию поверхность приложения включена, и все рисование для событий Draw будет выполняться на ней, а поверхность будет рисоваться на экране. Однако на некоторых старых устройствах и определенных чипсетах это может привести к снижению производительности. В таких случаях можно отключить отрисовку на поверхность приложения, и тогда все отрисовки по событиям Draw будут выполняться непосредственно в буфер экрана.

Вы можете использовать эту функцию для включения или отключения поверхности приложения в любое время в вашей игре, но следует учитывать, что для вступления изменений в силу требуется не менее одного кадра (шага). Выключение отключит всю коррекцию соотношения сторонкак установлено в Опциях игры для целевой платформы. Это означает, что у вас должен быть установлен метод, который масштабирует игру до размеров требуемого устройства, окна или экрана. Если вы снова включите его, то на следующем шаге снова вступят в силу параметры игры.

Рекомендуется использовать эту функцию осторожно, поскольку она влияет на то, как все рисуется на экране. В общем случае ее лучше использовать в качестве опции в меню игры или вызывать только один раз в начале игры и писать код в зависимости от того, включена она или выключена.

Синтаксис:

application_surface_enable(enable);

=======

application_surface_enable

This function enables or disables the application surface.

By default the application surface is enabled and all drawing for the Draw Events will be done to that and the surface drawn to the screen. However on some older devices and specific chipsets this may result in poorer performance. In those cases you can switch off drawing to the application surface and all Draw event drawing will then be done directly to the display buffer.

You can use this function to enable or disable the application surface at any time in your game, but you should note that it requires at least one frame (step) for the changes to take effect. Switching it off will switch off all aspect ratio correction, as set in the Game Options for the target platform. This means that you should have some method in place that scales the game to the required device, window or screen dimensions. If you switch it back on again, then the Game Options will take effect again the next step.

It is recommended that you use this function carefully as it will impact on how everything is drawn to the screen. In general you would want to have it as an option in a menu of your game, or call it only once at the start of your game and write your code around whether you have it enabled or disabled.

 Disabling the application surface will disable all Filters and Effects.

 

Syntax:

application_surface_enable(enable);
