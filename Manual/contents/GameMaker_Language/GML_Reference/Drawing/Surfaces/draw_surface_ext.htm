draw_surface_ext

Эта функция рисует заданную поверхность, как в функции draw_surface() но с дополнительными опциями для изменения масштаба, смешивания, вращения и альфа-функции рисуемой поверхности. Изменение этих значений никак не изменяет ресурс (только способ его рисования).

Синтаксис:

draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha);

Аргумент Тип Описание
id Идентификатор поверхности The unique ID value of the surface to draw.
x Реально The x position of where to draw the surface.
y Реально The y position of where to draw the surface.
xscale Реально The horizontal scale.
yscale Реально The vertical scale.
rot Реально The rotation or angle to draw the surface.
col Цвет The colour with which to blend the surface.
alpha Реально The alpha transparency for drawing the surface.

Возвращается:

N/A

Пример:

draw_surface_ext(surf, 0, 0, 2, 2, 0, c_red, 0.5);

Приведенный выше код нарисует поверхность, индексированную в переменной "surf" в позиции (0,0) в комнате, с удвоенным исходным масштабом, с красной и полупрозрачной смесью.
