application_surface_draw_enable

С помощью этой функции можно включить или отключить автоматическое рисование поверхности приложения. По умолчанию эта функция включена, но во многих случаях вы захотите взять на себя ответственность за то, когда и как будет рисоваться поверхность (например, при использовании шейдеров), и в этом случае вы можете использовать эту функцию для установки значения false, чтобы вы могли рисовать поверхность самостоятельно, когда и как вам нужно. Важно отметить, что когда вы отключаете автоматическое рисование и рисуете поверхность приложения самостоятельно, могут возникнуть определенные проблемы с альфа-компонентом sprites и самой поверхностью. Это происходит потому, что GameMaker рисует поверхность приложения без альфа-смешения, когда автоматическое рисование включено. Если вы отключите автоматическое рисование, то вам нужно сделать это самостоятельно, используя следующий код (для примера):

gpu_set_blendenable(false);
draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1);
gpu_set_blendenable(true);

Синтаксис:

application_surface_draw_enable(flag);

АргументТип Описание
flag The flag will be either true (включено, значение по умолчанию) или false (отключено).

Возвращается:

Пример:

application_surface_draw_enable(false);

Приведенный выше код отключит автоматическое рисование поверхности приложения.

=======

application_surface_draw_enable

This function enables or disables the automatic drawing of the application surface.

By default this is enabled, but in many cases you will want to take over when and how the surface is drawn (when using shaders for example), in which case you would use this function to set it to false so that you can draw it yourself when and how you require. It's important to note that when you switch off automatic drawing and draw the application surface yourself, that you may see certain issues with the alpha component of sprites and the surface itself. This is because GameMaker will draw the application surface without alpha blending when the automatic drawing is on. If you switch off automatic drawing then you need to handle this yourself, using the following code (for example):

gpu_set_blendenable(false);
draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1);
gpu_set_blendenable(true);

 

Syntax:

application_surface_draw_enable(flag);

Argument Type Description
flag Boolean The flag will be either true (enabled, the default value) or false (disabled).

 

Returns:

N/A

 

Example:

application_surface_draw_enable(false);

The above code will switch off the automatic drawing of the application surface.
