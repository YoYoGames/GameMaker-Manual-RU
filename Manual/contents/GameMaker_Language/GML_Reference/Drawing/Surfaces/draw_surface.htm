нарисовать_поверхность

С помощью этой функции можно нарисовать поверхность в заданном положении в помещении, при этом левый верхний угол поверхности будет нарисован в заданном положении x/y.

Синтаксис:

draw_surface(id, x, y);

Аргумент Тип Описание
id Идентификатор поверхности The unique ID value of the surface to draw.
x Реально The x position of where to draw the surface.
y Реально The y position of where to draw the surface.

Возвращается:

N/A

Пример:

var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
draw_surface(surf, _vx, _vy);

Приведенный выше код рисует поверхность, индексированную в "surf", в той же позиции, что и камера view[0].
