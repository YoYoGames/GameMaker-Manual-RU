surface_get_depth_disable

Эта функция проверяет, включена ли автоматическая генерация буферов глубины для поверхностей. Обычно все поверхности имеют буферы глубины, поэтому если вы рисуете 3D объекты на них, то они будут правильно сортироваться по глубине, однако выделение буферов глубины, по сути, удваивает размер поверхностей, что может быть чрезмерным и ненужным накладным расходом, особенно если ваша игра очень требовательна к памяти или преимущественно 2D. В таких случаях вы можете проверить это с помощью данной функции и при необходимости отключить буфер глубины для поверхностей с помощью функции surface_depth_disable().

Синтаксис:

surface_get_depth_disable();

Возвращается:

Пример:

if (!surface_exists(surf))
{
    if surface_get_depth_disable() == false
    {
        surface_depth_disable(true);
    }
    surf = surface_create(room_width, room_height);
}

Приведенный выше код проверит, существует ли заданная поверхность, и если нет, то проверит текущее состояние буфера глубины поверхности, и если он включен, то отключит его, после чего окончательно создаст поверхность.
