fps

В GameMaker есть два основных способа, которые можно использовать для определения скорости, с которой работает ваша игра. Это game speed (как указано в Опциях игры) и fps (кадров в секунду). Эти значения часто путают, но в основном одно из них - это количество шагов игры, которые GameMaker должен выполнить за секунду, а другое (fps) - это количество шагов процессора, которые GameMaker фактически выполняет за секунду, вплоть до максимального значения скорости самой комнаты. Чтобы получить истинный fps, то есть фактическое количество шагов процессора на шаг игры, используйте переменную fps_real переменную.

Эта переменная, доступная только для чтения, возвращает текущий fps в виде целочисленного значения. Обратите внимание, что функция будет обновляться только один раз на каждом шаге вашей игры, поэтому может показаться, что значение "прыгает" от одного к другому, но это вполне нормально.

Синтаксис:

fps

Возвращается:

Пример:

if debug_mode
{
    draw_text(32, 32, "FPS = " + string(fps));
}

Приведенный выше код проверит, находится ли игра в режиме отладки, и если да, то отобразит текущий fps на экране.
