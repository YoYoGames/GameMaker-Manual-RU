show_debug_message

Эта функция создает пользовательское отладочное сообщение, которое отображается в окне компилятора во время выполнения программы. Обратите внимание, что сообщение может быть как строкой, так и вещественным числом, но если вам нужно и то, и другое, то число необходимо сначала преобразовать в строку с помощью функции string() (см. пример ниже) и что если число имеет более двух десятичных знаков, то следует использовать string_format() чтобы показать их, так как по умолчанию десятичные числа округляются до ближайших двух знаков после запятой (поэтому "1.2468" будет отображаться как "1.25" в окне вывода).

Отладочные сообщения, показанные с помощью этой функции, будут показаны в окне вывода компилятора в нижней части сайта IDE, а также в графическом окне отладчика при запуске игры в режиме отладки. Если вы хотите видеть сообщения только в режиме отладки, то вам, вероятно, следует использовать функцию debug_event() вместо этого.

Синтаксис:

show_debug_message(string);

АргументТип Описание
string The custom debug message to show.

Возвращается:

Пример:

if !instance_exists(obj_Exit)
{
    show_debug_message("Exit not created!");
    show_debug_message("Error Num: " + string(global.error));
    game_end();
}

Приведенный выше код проверяет, существует ли экземпляр, и если нет, то в окне компиляции появляется отладочное сообщение и игра завершается.
