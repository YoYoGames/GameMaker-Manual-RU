show_debug_overlay

Эта функция может использоваться для включения и выключения стандартной отладочной накладки при тестировании вашей игры и по умолчанию отключена. Отладочная накладка показывает графическую полосу использования CPU/GPU в самом игровом окне, а также текущее реальное значение fps, количество смен текстур и количество вершинных батчей (обратите внимание, что смены текстур и вершинных батчей никогда не будут равны нулю и обычно показывают значения 2 или 3, поскольку даже в пустой комнате без объектов GameMaker все равно должен рисовать и батчить).

Эта полоса разделена на секции, каждая из которых составляет 1/60 секунды. Как видно из изображения выше, полоса состоит из различных цветов:

  • Зеленый - обработка ввода/вывода (т.е.: клавиатура, мышь, геймпад, сеть и т.д...)
  • Красный - Скорость обновления события шага
  • Желтый - Время, необходимое для проведения розыгрыша
  • Оранжевый -Время обновления отладки, которое обычно видно только при использовании модуля отладки
  • Белый - оставшееся время GPU, то есть время, потраченное на ожидание, пока GPU закончит рендеринг кадра, прежде чем начнется следующий.
  • Cyan - Время отрисовки текста
  • Grey - Время, необходимое для очистки экрана на каждом этапе рисования
  • Blue - Время, необходимое для работы сборщика мусора
  • Темно-красный цвет - промывка GPU, т.е. сколько времени требуется GPU для очистки изображений из памяти.

Используя эту функцию, вы можете добавить оверлей в режиме отладки или нет, и таким образом вы сможете увидеть, насколько эффективно работает ваша игра, и получить визуальную подсказку о том, как она использует доступные ресурсы, без накладных расходов, связанных с запуском отладчика.

ПРИМЕЧАНИЕ: Эта функция не будет работать на цели HTML5 из-за того, что она работает иначе, чем все остальные цели.

Синтаксис:

show_debug_overlay(enable);

=======

show_debug_overlay

This function toggles The Debug Overlay. This overlay is disabled by default.

The debug overlay enables a menu with three options: FPS (open by default), Log, and Audio. It displays the GameMaker version in the top-right corner, along with the version and name of the project: 

If the optional minimised parameter is set to false, the overlay is opened with the FPS window minimised, only showing basic info in the header.

To create custom debug views that will appear under the Views menu, see Creating Debug Views.

 This function will not work on the HTML5 target due to the different way it works compared to all the other targets.

 To bring up The Debug Overlay with the Log window open, use show_debug_log instead.

 

Syntax:

show_debug_overlay(enable[, minimised, scale, alpha]);
