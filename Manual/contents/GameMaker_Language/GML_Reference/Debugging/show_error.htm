show_error

Эта функция выводит пользовательскую строку в качестве сообщения об ошибке.

ВАЖНО Второй аргумент существует только для обратной совместимости со старыми проектами и не влияет на отображаемый диалог ошибки.

ПРИМЕЧАНИЕ Эта функция предназначена только для отладки.

Синтаксис:

show_error(str, abort);

Аргумент Тип Описание
str Строка The string to show in the pop-up message.
abort Булево UNUSED Должна ли ошибка прервать игру (true) или позволить игроку игнорировать ее (false).

Возвращается:

N/A

Пример:

if (room != rm_Dungeon)
{
    show_error("Error: Went to wrong area. Aborting game.", true);
}

Приведенный выше код проверит, является ли текущая комната rm_Dungeon, и если нет, то выдаст сообщение об ошибке.
