code_is_compiled

Эта функция вернет true, если платформа компилируется вне виртуальной машины, как, например, для платформ YYC и JS.

Синтаксис:

code_is_compiled();

Возвращается:

Пример:

if code_is_compiled()
{
    show_debug_message("Compiler okay!");
}

Приведенный выше код проверяет, была ли игра скомпилирована с помощью компилятора YoYo, и показывает отладочное сообщение, если это так.
