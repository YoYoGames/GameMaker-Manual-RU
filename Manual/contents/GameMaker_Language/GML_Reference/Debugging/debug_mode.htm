режим отладки

Эта переменная, доступная только для чтения, возвращает значение true, если игра ведется в режиме отладки, и false, если игра ведется в обычном режиме.

Синтаксис:

debug_mode

Возвращается:

Пример:

if debug_mode
{
    ini_open("Cheats.ini");
}
else
{
    ini_open("Game.ini");
}

Приведенный выше код открывает разный ini файл в зависимости от значения переменной только для чтения debug_mode.
