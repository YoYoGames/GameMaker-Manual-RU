buffer_md5

В криптографии MD5 (Message-Digest algorithm 5) - это широко используемая криптографическая функция хэширования со 128-битным хэш-значением, которая применяется в самых разных приложениях безопасности. Она также часто используется для проверки целостности данных. Эта функция принимает входные данные из заданного буфера (или части буфера) и возвращает 32-символьный шестнадцатеричный хэш MD5, уникальный для этих данных. Таким образом, вы можете сгенерировать безопасный ключ, который можно хранить и использовать для проверки целостности информации, отправляемой на внешний сервер (или получаемой от него) (например). При применении этой функции к буферам вы должны указать идентификатор буфера, который нужно использовать, затем значение смещения (в байтах) для начала, а затем размер (опять же в байтах) для области, которая будет хэшироваться.

ПРИМЕЧАНИЕ: MD5 не является полностью безопасным и может быть взломан. См. эту страницу для получения дополнительной информации.

Синтаксис:

buffer_md5(buffer, offset, size);

АргументТип Описание
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Возвращается:

Пример:

check_string = buffer_md5(buff, 0, buffer_get_size(buff));

Приведенный выше код создаст md5-хэш для полных данных, хранящихся в буфере, индексируемом переменной "buff", и сохранит возвращенный хэш в переменной "check_string".
