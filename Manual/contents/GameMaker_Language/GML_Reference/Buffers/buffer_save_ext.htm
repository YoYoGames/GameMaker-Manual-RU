buffer_save_ext

С помощью этой функции вы можете сохранить часть содержимого буфера в файл, готовый к считыванию обратно в память с помощью функции buffer_load() функцию. Смещение" определяет начальную позицию в буфере для сохранения (в байтах), а "размер" - размер сохраняемой области буфера, начиная с этого смещения и далее (также в байтах).

Синтаксис:

buffer_save_ext(buffer, filename, offset, size);

АргументТип Описание
buffer The index of the buffer to save.
filename The name of the file to save as.
offset The offset within the buffer to save from (in bytes).
size The size of the buffer area to save (in bytes).

Возвращается:

Пример:

buffer_save_ext(buff, "Player_Save.sav", 0, 16384);

Сохраняет часть текущего содержимого буфера с идентификатором, хранящимся в переменной "buff", в файл.
