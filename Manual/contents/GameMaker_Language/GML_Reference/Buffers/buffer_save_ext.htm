buffer_save_ext

С помощью этой функции вы можете сохранить часть содержимого буфера в файл, готовый к считыванию обратно в память с помощью функции buffer_load() функцию. Смещение" определяет начальную позицию в буфере для сохранения (в байтах), а "размер" - размер сохраняемой области буфера, начиная с этого смещения и далее (также в байтах).

Синтаксис:

buffer_save_ext(buffer, filename, offset, size);

АргументТип Описание
buffer The index of the buffer to save.
filename The name of the file to save as.
offset The offset within the buffer to save from (in bytes).
size The size of the buffer area to save (in bytes).

Возвращается:

Пример:

buffer_save_ext(buff, "Player_Save.sav", 0, 16384);

Сохраняет часть текущего содержимого буфера с идентификатором, хранящимся в переменной "buff", в файл.

=======

buffer_save_ext

This function saves part of the contents of a buffer to a file, ready to be read back into memory using the buffer_load function.

The "offset" defines the start position within the buffer for saving (in bytes), and the "size" is the size of the buffer area to be saved from that offset onwards (also in bytes).

 

Syntax:

buffer_save_ext(buffer, filename, offset, size);

Argument Type Description
buffer Buffer The buffer to save.
filename String The name of the file to save as.
offset Real The offset within the buffer to save from (in bytes).
size Real The size of the buffer area to save (in bytes).

 

Returns:

N/A

 

Example:

buffer_save_ext(buff, "Player_Save.sav", 0, 16384);

This function saves part of the current contents of the buffer stored in the variable buff to a file.
