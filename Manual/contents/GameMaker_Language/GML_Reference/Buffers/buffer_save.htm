буфер_сохранения

С помощью этой функции вы можете сохранить содержимое буфера в файл, готовый к повторному считыванию в память с помощью функции buffer_load() функцию.

ПРИМЕЧАНИЕ: На HTML5 при использовании этой функции содержимое буфера будет сохранено в виде строк в кодировке base64.

Синтаксис:

buffer_save(buffer, filename);

=======

buffer_save

This function saves the contents of a buffer to a file, ready to be read back into memory using the buffer_load function.

 On HTML5 the contents of the buffer will be saved as base64 encoded strings when using this function.

 

Syntax:

buffer_save(buffer, filename);
