buffer_fill

Эта функция может быть использована для заполнения ранее созданного буфера данными заданного типа и значения. Данные, которыми вы заполняете буфер, должны соответствовать аргументу "тип" этой функции, то есть вы не можете пытаться заполнить буфер строкой, а использовать тип unsigned 16bit integer, например. строкой и использовать тип unsigned 16bit integer, например. Константы типов такие же, как и те, которые используются функциями buffer_read() и buffer_write() функции. "Размер" это размер буфера (в байтах), который вы хотите заполнить, а смещение - это значение смещения (также в байтах) от начала буфера для начала заполнения.

Синтаксис:

buffer_fill(buffer, offset, type, value, size);

АргументТип Описание
buffer The index of the buffer to fill.
offset The data offset value (in bytes).
type The type of data that is to be written to the buffer (see the list of constants here).
value The data to write.
size The size of the buffer (in bytes) that you wish to fill.

Возвращается:

Пример:

map_buffer = buffer_create(16384, buffer_fixed, 0);
buffer_fill(map_buffer, 0, buffer_u16, 0, 16384);

Приведенный выше код находит начало буфера с идентификатором, хранящимся в переменной "buff", и записывает в него серию знаковых 16-битных целочисленных значений.
