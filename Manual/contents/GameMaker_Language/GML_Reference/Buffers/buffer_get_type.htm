тип_буфера

С помощью этой функции вы получаете тип буфера, на который ссылается значение идентификатора буфера. Возвращаемое значение будет одной из следующих констант:

Постоянная Описание
buffer_fixed Буфер фиксированного размера.
buffer_grow Буфер, который будет динамически "расти" по мере добавления данных
buffer_wrap Буфер, в который будут "заворачиваться" данные. Когда добавляемые данные достигнут предела размера буфера, перезапись будет помещена обратно в начало буфера, и дальнейшая запись будет продолжаться с этого момента.
buffer_fast Специальный "полосатый" буфер, который очень быстро читается/записывается. Может использоваться только с типами данных buffer_u8 и должен быть выровнен по 1 байту.
buffer_vbuffer Буфер вершин

Синтаксис:

buffer_get_type(buffer);

АргументТип Описание
buffer The index of the buffer to check.

Возвращается:

Buffer Type Constant

Пример:

type = buffer_get_type(buff);

Приведенный выше код получит тип буфера из значения, проиндексированного в переменной "buff", и сохранит его в переменной.

=======

buffer_get_type

This function returns the type of a given buffer.

The return value will be one of the following constants:

 

Syntax:

buffer_get_type(buffer);

Argument Type Description
buffer Buffer The buffer to check.

 

Returns:

Buffer Type Constant

 

Example:

type = buffer_get_type(buff);

The above code gets the type of the buffer stored in the variable buff and stores it in a variable.
