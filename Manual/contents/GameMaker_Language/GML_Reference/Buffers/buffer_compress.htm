сжатие_буфера

С помощью этой функции вы можете сжать часть (или весь) буфера, используя сжатие zlib. Вы передаете идентификатор буфера, который нужно сжать (как возвращается функцией buffer_create()), смещение внутри буфера в байтах и размер данных буфера для сжатия (также в байтах). Функция вернет новое значение идентификатора буфера для сжатого буфера или значение меньше 0, если по какой-либо причине она не выполнилась. Эта функция не изменяет исходный буфер.

Синтаксис:

buffer_compress(buffer, offset, size);

АргументТип Описание
buffer The index of the buffer to compress.
offset The offset within the buffer to compress (in bytes).
size The size of the buffer area to compress (in bytes).

Возвращается:

Buffer ID

Пример:

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff);

Приведенный выше код создает буфер и заполняет его данными из строки. Затем этот буфер сжимается и сохраняется, а исходный и сжатый буферы удаляются.
