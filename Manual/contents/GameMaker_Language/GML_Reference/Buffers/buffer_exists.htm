буфер_существует

Эта функция может быть использована для проверки переменной на предмет того, содержит ли она действительное значение идентификатора буфера или нет. Если да, то функция вернет true, в противном случае она вернет false.

Синтаксис:

buffer_exists(buffer)

АргументТип Описание
buffer The index of the buffer to check.

Возвращается:

Пример:

if buffer_exists(buff)
{
    buffer_delete(buff);
}

Приведенный выше код проверяет, содержит ли переменная "buff" идентификатор буфера, и если да, то буфер удаляется.
