буфер_копирование_из_вершинного_буфера

Эта функция может использоваться для копирования некоторых (или всех) вершинных данных, хранящихся в одном вершинном буфере, в ранее созданный обычный буфер. При копировании из вершинного буфера в обычный буфер с помощью этой функции оба буфера должны быть предварительно созданы (с помощью функций vertex_create_buffer() и buffer_create() например). Вы можете указать диапазон вершинных данных, которые вы хотите скопировать в буфер, где начальная вершина может быть в любом месте от 0 до количества вершин -1, и вы можете указать количество вершин с этой точки для копирования. Вы можете использовать функцию vertex_get_number() для буфера вершин, чтобы получить общее количество хранящихся вершин. Наконец, вы указываете индекс буфера для копирования данных вершины, а также смещение данных, чтобы определить позицию для копирования данных вершины в буфере назначения.

Синтаксис:

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

=======

buffer_copy_from_vertex_buffer

This function copies some (or all) of the vertex data stored in one vertex buffer into a previously created regular buffer.

When copying from a vertex buffer into a regular buffer with this function, both buffers must have previously been created (using the vertex_create_buffer and buffer_create functions, for example). You can specify the range of vertex data that you wish to copy into the buffer, where the start vertex can be anywhere between 0 and the number of vertices -1, and you can give the number of vertices from that point on to copy. You can use the function vertex_get_number on the vertex buffer to get the total number of vertices stored. Finally you give the buffer index to copy the vertex data into, as well as a data offset to define the position to copy the vertex data to in the destination buffer.

 

Syntax:

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

Пример:

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

Приведенный выше код скопирует данные вершин, хранящиеся в буфере вершин, индексированном в переменной "model_buffer", а затем вставит их в буфер, индексированный в переменной "player_buffer".
