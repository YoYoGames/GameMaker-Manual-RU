buffer_sizeof

Эта функция возвращает размер (в байтах) заданной константы типа данных буфера (перечислены здесь).

Синтаксис:

buffer_sizeof(type);

Аргумент Тип Описание
type Тип данных буфера Константа The type of data that is to be checked (see the list of constants here).

Возвращается:

Real

Пример:

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

Приведенный выше код сначала вычисляет размер создаваемого буфера, умножая тип данных unsigned 8bit на 12 (поскольку мы будем использовать буфер для хранения 12 частей данных), а затем использует это значение для установки фиксированного буфера нужного размера.
