буфер_get_address

С помощью этой функции вы можете получить указатель на необработанный, выровненный адрес буфера. Это в первую очередь предназначено для использования расширениями, так как вы можете передать им это значение, позволяя им получить доступ к данным буфера. Обратите внимание, что есть несколько вещей которые нельзя делать с буфером при доступе из расширения:

  • Вы не можете освободить память.
  • Изменить размер памяти невозможно.
  • Вы не можете писать дальше адреса буфера плюс размер буфера (вы можете использовать buffer_get_size() для этого), иначе вы получите ошибку выхода за границы.

Синтаксис:

buffer_get_address(buffer);

Возвращается:

Пример:

var b_address = buffer_get_address(buff_model);
var b_end_address = b_address + buffer_get_size(buff_model);

Приведенный выше код получит адрес памяти буфера, индексированного в переменной "buff_model", а затем получит адрес памяти для конца буфера с помощью функции buffer_get_size(), сохранив оба значения в локальных переменных для дальнейшего использования. дальнейшего использования.

======= buffer Buffer The buffer to use.

 

Returns:

Pointer

 

Example:

var _address = buffer_get_address(buff_model);
var _end_address = _address + buffer_get_size(buff_model);

The above code gets the memory address of the buffer stored in the variable buff_model and then gets the memory address for the end of the buffer by adding the size of the buffer, returned by buffer_get_size, to the address. Both values are stored in local variables for further use.
