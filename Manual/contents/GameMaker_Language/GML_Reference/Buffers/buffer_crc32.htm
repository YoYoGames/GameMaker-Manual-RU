buffer_crc32

Эта функция принимает входные данные из буфера и возвращает хэш контрольной суммы crc32. Вы указываете ID буфера, который нужно использовать, затем значение смещения (в байтах) для начала, а затем размер (опять же в байтах) для области, которую нужно хэшировать, и функция вернет 32-битное целочисленное значение для этого региона.

Синтаксис:

buffer_crc32(buffer, offset, size);

АргументТип Описание
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Возвращается:

Пример:

check_val = buffer_crc32(buff, 0, buffer_get_size(buff));

Приведенный выше код создаст хэш-значение контрольной суммы crc32 для полных данных, хранящихся в буфере, индексируемом переменной "buff", и сохранит возвращаемое целочисленное хэш-значение в переменной "check_val".
