буферная спица

С помощью функции buffer_write() вы можете записывать данные в заданный буфер в текущей позиции "seek", при этом каждая порция данных продвигает эту позицию на записываемые или считываемые байты. Однако может возникнуть необходимость изменить заданный фрагмент данных без изменения текущей позиции поиска, и тогда вы можете использовать эту функцию. Вы просто сообщаете функции индекс буфера, затем позицию смещения от начала буфера (в байтах) в пределах этого буфера для записи, а также тип данных и записываемое значение.

Синтаксис:

buffer_poke(buffer, offset, type, value);

Аргумент Тип Описание
buffer Идентификатор буфера The index of the buffer to use.
offset Реально The offset position (in bytes) within the buffer to write the given data to.
type Тип данных буфера Константа The type of data that is to be written to the buffer (see the list of constants here).
value Переменная The data to add to the buffer, in accordance with the type specified.

Возвращается:

N/A

Пример:

buffer_poke(buff, 3, buffer_u8, colour_get_blue(image_blend));

Приведенный выше код добавит значение синей составляющей цвета, используемого для наложения изображения, в буфер, индексированный в переменной "buff", на третью позицию в буфере и как беззнаковое 8-битное значение.
