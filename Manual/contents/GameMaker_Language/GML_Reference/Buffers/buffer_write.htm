буферная_запись

Эта функция может быть использована для записи данных в ранее созданный буфер. Записываемые данные должны соответствовать аргументу "тип" этой функции, то есть нельзя, например, пытаться записать строку как беззнаковое 16-битное целое число. Для определения типа данных можно использовать следующие константы:
