buffer_get_alignment

С помощью этой функции можно получить выравнивание по байтам для заданного идентификатора буфера.

Синтаксис:

buffer_get_alignment(buffer);

АргументТип Описание
buffer The index of the buffer to check.

Возвращается:

Пример:

alignment = buffer_get_alignment(buff);

Приведенный выше код получит выравнивание буфера из значения, проиндексированного в переменной "buff", и сохранит его в переменной.
