С помощью функции buffer_read() вы можете читать данные из заданного буфера в текущей позиции "seek", при этом каждая часть данных читается, продвигаясь вперед по этой позиции на считанные или записанные байты. Однако может возникнуть необходимость прочитать определенный фрагмент данных без изменения текущей позиции поиска, и тогда вы можете использовать эту функцию. Вы просто сообщаете функции идентификатор буфера, затем позицию смещения (от начала буфера) в этом буфере для чтения, а также тип данных, которые вы хотите прочитать.

ПРИМЕЧАНИЕ: Использование неправильного типа данных для считываемых данных приведет к получению ошибочных значений!

При успешном выполнении функции будет возвращено значение (Real, string, boolean и т.д...) в зависимости от типа данных, в то время как при неудаче будет возвращен просто 0.

Синтаксис:

buffer_peek(buffer, offset, type);

Аргумент Тип Описание
buffer Идентификатор буфера The index of the buffer to use.
offset Реально The offset position (in bytes) within the buffer to read the given data from.
type Тип данных буфера Константа The type of data that is to be read from the buffer (see the list of constants here).

Возвращается:

Real, Boolean, String or 0 (if it fails)

Пример:

var red = buffer_peek(buff, 1, buffer_u8);
var green = buffer_peek(buff, 2, buffer_u8);
var blue = buffer_peek(buff, 3, buffer_u8);
image_blend = make_colour_rgb(red, green, blue);

Приведенный выше код получит три значения из трех различных позиций в буфере, индексированных в переменной "buff", а затем использует эти значения для установки смешивания изображения экземпляра.
