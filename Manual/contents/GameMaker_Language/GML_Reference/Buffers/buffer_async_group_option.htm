buffer_async_group_option

С помощью этой функции вы можете установить некоторые специфические для платформы параметры для сохраняемой/загружаемой группы буферов. Доступны следующие опции:

  • "subtitle" или "slottitle" - Значением для этой опции будет строка, которая будет показана пользователю при управлении его сохраненными данными в ОС. Это важно только при сохранении данных, а не при повторной загрузке.
  • "showdialog" - Значение этой опции - булево true или false. Если установлено значение true, то при сохранении или загрузке GameMaker будет показывать системный пользовательский интерфейс, в противном случае он будет выполнять квиксейв/квиклоад без показа пользовательского интерфейса. Обычно это нужно только в том случае, если вы поддерживаете несколько слотов сохранения, чтобы пользователь мог выбрать слот, но если вы поддерживаете только один слот для пользователя, установите значение false.
  • "savepadindex" - Для этой опции вы должны указать индекс пада игрока, который сохраняется или загружается, и система будет записывать данные в и читать данные из папки сохранения этого игрока.
  • "saveslotsize" - Эта опция требует, чтобы вы указали фактический размер в байтах, который вы хотите сохранить (так что вы можете сделать buffer_seek() и buffer_tell() чтобы получить это, например). Обратите внимание, что указывать это значение не обязательно, так как для всех сохранений предварительно назначается минимальное пространство, которое обычно зависит от целевой платформы.
  • "ps_create_backup" - (только для PlayStation) - Установка этого параметра на true гарантирует, что резервная копия данных сохранения будет создана, когда каталог сохранения будет размонтирован (т.е. когда будет вызвана функция buffer_async_group_end() ). Дополнительную информацию см. в документации по консоли.

ВАЖНО! В настоящее время эта функция действует только для целевых модулей PSVita, PS4 и XBox One. На всех остальных целевых модулях она ничего не даст.

Синтаксис:

buffer_async_group_option(option, value);

Аргумент Тип Описание
option Строка The option to set.
value Реальный или Строка The value to set (can be a real or string, depending on the option).

Возвращается:

N/A

Пример:

buffer_async_group_begin("save_folder_name");
buffer_async_group_option("showdialog", false);
buffer_async_group_option("slottitle", "Catch The Haggis Save");
buffer_async_group_option("subtitle", "All your haggis are saved here!");
save = buffer_save_async(buff, "Player_Save.sav", 0, 16384);
buffer_async_group_end();

Приведенный выше код начинает группу буферов, затем устанавливает параметры группы, прежде чем установить 4 файла для асинхронного сохранения. Затем определение группы завершается (в этот момент начнется сохранение).
