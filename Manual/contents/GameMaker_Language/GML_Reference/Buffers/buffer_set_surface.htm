buffer_set_surface

С помощью этой функции можно записать информацию из буфера на заданную поверхность. Буфер, из которого производится чтение, и поверхность, на которую производится запись, должны быть созданы ранее, и вы можете указать смещение в буфере для начала чтения. Обратите внимание, что чтение всегда будет начинаться с начала буфера плюс значение смещения, а не с текущей позиции поиска плюс значение смещения.

ПРИМЕЧАНИЕ: Эта функция изменилась начиная с обновления GameMaker 2.3.1 и далее. В предыдущих версиях использовалась константа buffer_surface_copy, которая теперь устарела.

Синтаксис:

buffer_set_surface(buffer, surface, offset);

АргументТип Описание
buffer The index of the buffer to use.
surface The index of the surface to use.
offset The data offset value.

Возвращается:

Пример:

buffer_set_surface(buff, application_surface, 0);

Приведенный выше код скопирует все данные, хранящиеся в буфере, проиндексированном в переменной "buff", на поверхность приложения без смещения.
