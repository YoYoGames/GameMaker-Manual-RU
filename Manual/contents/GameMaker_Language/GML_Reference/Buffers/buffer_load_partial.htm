буферная_загрузка_частичная

Эта функция загрузит некоторые данные из буфера, которые были ранее сохранены с помощью buffer_save() функции в существующий буфер. Вы указываете id ранее созданного буфера для загрузки, затем сохраненный файл буфера для загрузки, а затем смещение от начала буфера (в байтах), из которого вы хотите загрузить данные. Следующие аргументы предназначены для задания длины данных буфера (в байтах) от начальной точки смещения, с которой вы хотите загрузить данные, и точки смещения для загрузки данных в буфер (опять же, в байтах).

Пожалуйста, ознакомьтесь со страницей buffer_load() для получения информации о конкретной платформе.

Синтаксис:

buffer_load_partial(buffer, filename, offset, src_len, dest_offset);

АргументТип Описание
buffer The index of the buffer to load into (destination).
filename The name of the file to load from (source).
offset The offset within the destination buffer to load to (in bytes).
src_len The length of the part of the source buffer to load (in bytes).
dest_offset The offset where to start putting the partial data in the destination buffer (in bytes).

Возвращается:

N/A

Пример:

buff = buffer_create(256, buffer_grow, 1);
var _file = "save.dat";
var _so = 6;
var _sl = 5;
var _do= 0;
buffer_load_partial(buff, _file, _so, _sl, _do);

Приведенный выше код создаст новый буфер "grow", а затем загрузит в него часть данных, сохраненных в файле "save.dat".
