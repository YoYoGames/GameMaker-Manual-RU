buffer_decompress

С помощью этой функции вы можете распаковать ранее сжатый буфер, используя сжатие zlib. Вы указываете идентификатор буфера для распаковки, и функция возвращает новый идентификатор буфера, содержащий несжатые данные. Если декомпрессия не удалась (например, вы предоставляете буфер, который не был сжат), то функция вернет значение меньше 0.

Синтаксис:

buffer_decompress(buffer);

АргументТип Описание
buffer The index of the buffer to decompress.

Возвращается:

Buffer ID

Пример:

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

Приведенный выше код сначала загрузит сохраненный буфер, затем распакует его и, наконец, прочитает строковые данные из распакованного буфера в глобальную переменную.
