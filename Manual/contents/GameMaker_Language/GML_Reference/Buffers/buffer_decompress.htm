буферная_декомпрессия

С помощью этой функции вы можете распаковать ранее сжатый буфер, используя сжатие zlib. Вы указываете идентификатор буфера для распаковки, и функция возвращает новый идентификатор буфера, содержащий несжатые данные. Если декомпрессия не удалась (например, вы предоставляете буфер, который не был сжат), то функция вернет значение меньше 0.

Синтаксис:

buffer_decompress(buffer);

АргументТип Описание
buffer The index of the buffer to decompress.

Возвращается:

Buffer ID

Пример:

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

Приведенный выше код сначала загрузит сохраненный буфер, затем распакует его и, наконец, прочитает строковые данные из распакованного буфера в глобальную переменную.

=======

buffer_decompress

This function returns a new buffer with the decompressed data of a buffer that stores data using zlib compression.

You supply the buffer to decompress, and the function will return a new buffer that contains the uncompressed data. If the decompression failed (for example, you supplied a buffer that wasn't compressed) then the function will instead return -1.

 

Syntax:

buffer_decompress(buffer);

Argument Type Description
buffer Buffer The index of the buffer to decompress.

 

Returns:

Buffer or -1 in case the buffer couldn't be decompressed

 

Example:

var _cmpBuff = buffer_load("Player_Save.sav");
var _srcBuff = buffer_decompress(_cmpBuff);
global.DataString = buffer_read(_srcBuff, buffer_string);

The above code first loads a saved buffer, then decompresses it and finally reads the string data from the decompressed buffer into a global variable.
