buffer_delete

С помощью этой функции вы можете удалить буфер, ранее созданный с помощью функции buffer_create() из памяти, освобождая ресурсы, использованные для его создания, и удаляя любые данные, которые он может содержать в данный момент.

ПРИМЕЧАНИЕ: Важно всегда удалять любые динамически создаваемые ресурсы из памяти, когда они больше не нужны, чтобы предотвратить утечку памяти.

ВАЖНО! При создании буфера значение индекса, используемое для его идентификации, является целым числом, начинающимся с 0. Эти индексы повторно используются GameMaker, поэтому значение индекса уничтоженного буфера может быть использовано впоследствии вновь созданным, поэтому мы рекомендуем после уничтожения буфера установить любую переменную, содержащую индекс буфера, в значение -1.

Синтаксис:

buffer_delete(buffer)

АргументТип Описание
buffer The index of the buffer to delete.

Возвращается:

Пример:

buffer_delete(player_buffer);
player_buffer = -1;

Приведенный выше код удалит ранее созданный буфер с id, хранящимся в переменной "player_buffer", а затем установит переменную в -1.
