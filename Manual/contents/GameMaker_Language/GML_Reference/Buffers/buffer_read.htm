буферное_чтение

Эта функция может быть использована для чтения данных из ранее созданного буфера. Возвращаемое значение будет зависеть от типа считываемых данных, который сам по себе определяется следующими константами:

Постоянная Описание
buffer_u8 Беззнаковое 8-разрядное целое число. Это положительное значение от 0 до 255.
buffer_s8 Знаковое 8-разрядное целое число. Это может быть положительное или отрицательное значение от -128 до 127 (0 считается положительным).
buffer_u16 Беззнаковое 16-разрядное целое число. Это положительное значение от 0 до 65 535.
buffer_s16 Подписанное 16-разрядное целое число. Это может быть положительное или отрицательное значение от -32,768 до 32,767 (0 считается положительным).
buffer_u32 Беззнаковое 32-разрядное целое число. Это положительное значение от 0 до 4 294 967 295.
buffer_s32 Подписанное 32-разрядное целое число. Это может быть положительное или отрицательное значение от -2,147,483,648 до 2,147,483,647 (0 считается положительным).
buffer_u64 Беззнаковое 64-разрядное целое число.
buffer_f16 16-разрядное плавающее значение. Это может быть положительное или отрицательное значение в диапазоне +/- 65504. (В настоящее время не поддерживается!)
buffer_f32 32-разрядное плавающее значение. Это может быть положительное или отрицательное значение в диапазоне +/-16777216.
buffer_f64 64-разрядное плавающее число.
buffer_bool Булево значение. Может быть только 1 или 0 (true или false).
buffer_string Струна любого размера.
buffer_text Строка любого размера без завершающего нулевого символа.

Если функция выполнится успешно, то она вернет значение заданного типа, однако если она не выполнится, то это приведет к ошибке runner.

ПРИМЕЧАНИЕ Использование неправильного типа данных для считываемых данных приведет к возврату ошибочных значений.

Синтаксис:

buffer_read(buffer, type)

Аргумент Тип Описание
buffer Идентификатор буфера The index of the buffer to read from.
type Тип данных буфера Константа The type of data that is to be read from the buffer (see the list of constants above).

Возвращается:

Real, Boolean or String

Пример:

var cmd = buffer_read(buff, buffer_s16);

Приведенный выше код считывает из буфера с id, хранящегося в переменной "buff", знаковое 16-битное значение в локальную переменную "cmd".
