размер_буфера

С помощью этой функции вы можете изменить размер буфера до указанного вами размера (в байтах).

Синтаксис:

buffer_resize(buffer, newsize);

АргументТип Описание
buffer The index of the buffer to change the size of.
newsize The new size of the buffer (in bytes).

Возвращается:

Пример:

if (buffer_get_size(buff) < 16384)
{
    buffer_resize(buff, 16384);
}

Приведенный выше код проверит размер буфера, индексированного в переменной "buff", и если он меньше заданного значения, размер буфера будет изменен.
