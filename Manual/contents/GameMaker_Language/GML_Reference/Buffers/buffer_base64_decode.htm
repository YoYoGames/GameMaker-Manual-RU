buffer_base64_decode

С помощью этой функции вы можете декодировать строку в кодировке base64 (созданную с помощью функции buffer_base64_encode() ) в буфер. Эта функция создаст буфер (как выровненный по 1 байту "растущий" буфер") и вернет уникальный индекс для буфера, который будет использоваться во всех дальнейших вызовах функции.

ПРИМЕЧАНИЕ: Важно, чтобы вы удаляли любые динамически созданные ресурсы, подобные этому, из памяти, когда они вам больше не нужны, чтобы предотвратить утечку памяти, поэтому, когда вы закончите работу с созданным буфером, вы должны освободить его снова, используя buffer_delete().

Синтаксис:

buffer_base64_decode(string);

=======

buffer_base64_decode

This function decodes a base64 encoded string (created using the buffer_base64_encode function) into a new buffer.

The buffer is created as a 1 byte aligned "grow" buffer.

 

Syntax:

buffer_base64_decode(string);
