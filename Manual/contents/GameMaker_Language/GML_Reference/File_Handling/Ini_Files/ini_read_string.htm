ini_read_string

Вы можете использовать эту функцию для чтения строки (текста) из файла данных ini. Ini файлы разбиваются на секции, а затем каждая секция разбивается на пары ключ - значение. Таким образом, типичный ini файл будет выглядеть примерно так:

Синтаксис:

ini_read_string(section, key, default);

Аргумент Тип Описание
section Строка The section of the .ini to read from.
key Строка The key within the relevant section of the .ini to read from.
default Строка The string to return if a string is not found in the defined place (or the .ini file does not exist). Must be a string.

Возвращается:

String

Пример:

ini_open("savedata.ini");
global.name = ini_read_string("player", "name', "Player1");
ini_close();

Это откроет файл "savedata.ini" и установит global.name в строку "save1" > "Name" в нем, затем снова закройте .ini. Если в строке "save1" > "Name" нет строки, или отсутствует файл "savedata.ini", global.name будет установлен на "Player1".
