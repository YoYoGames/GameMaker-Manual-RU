ini_section_delete

С помощью этой функции вы можете удалить целый раздел ini-файла, при этом будут удалены все пары ключ-значение, которые с ним связаны.

Синтаксис:

ini_section_delete(section);

АргументТип Описание
section The section to delete.

Возвращается:

Пример:

ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
ini_section_delete("save1");
ini_close();

Этот пример откроет файл "savedata.ini" и запишет значение в "save1" > "Score". Затем он удалит раздел "save1" и закроет файл .ini.
