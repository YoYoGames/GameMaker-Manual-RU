ini_close

Эта функция должна быть вызвана в тот момент, когда вы закончили чтение или запись в любой открытый ini-файл. Если вы не используете эту функцию после использования любой из функций записи ini, то на диск ничего не будет записано, так как информация о файле хранится в памяти до вызова этой функции, которая принудительно выполняет запись. Если вы попытаетесь открыть ini, не закрыв предварительно другой (или тот же самый), вы также получите ошибку.

Функция также возвращает строку с закодированным в ней ini-файлом. Эту строку можно сохранить на сервере и/или использовать снова вместе с функцией ini_open_from_string() для повторного создания ini.

Синтаксис:

ini_close();

Возвращается:

String

Пример:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Это откроет файл "savedata.ini" и прочитает значение из секции "save1" с ключом "score", затем закроет ini-файл. Если в разделе "save1" нет значения "score" или отсутствует файл "savedata.ini", будет возвращен 0 и создан новый ini-файл.
