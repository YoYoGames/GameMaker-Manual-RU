ini_section_exists

Эта функция проверяет, существует ли раздел в текущем открытом ini, и возвращает true, если существует, или false в противном случае. Это не является необходимой проверкой для предотвращения ошибок, поскольку, если раздел не существует, чтение из несуществующего раздела просто вернет значение по умолчанию, однако это может быть полезно, чтобы узнать, сохранил ли ini файл определенные данные.

Синтаксис:

ini_section_exists(section);

АргументТип Описание
section The section in the open .ini file to check for.

Возвращается:

Пример:

ini_open("savedata.ini");
if (!ini_section_exists("save1"))
{
    global.savegame = "1";
}
ini_close();

Это установит переменную global.savegame в значение "1", если "save1" НЕ найден. С помощью цикла for это можно использовать как основной способ определения количества сохранений для игры (т.е. создать систему для нескольких одновременных сохранений для одной игры).
