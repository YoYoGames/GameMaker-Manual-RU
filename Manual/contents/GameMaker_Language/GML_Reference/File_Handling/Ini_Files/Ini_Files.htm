Файлы Ини

Файлы INI - это небольшие, легкие файлы, совместимые с большинством платформ. Они идеально подходят для хранения небольших фрагментов информации, например, предпочтений интерфейса, локальных высоких результатов, данных об уровнях и т.д... и как таковые являются отличным и простым способом начать сохранять информацию из ваших игр. Если вы ищете более продвинутые способы сохранения информации в файлы, вам стоит обратить внимание на некоторые другие функции файлов, такие как текстовые файлы, двоичные файлы и буферы.

ВАЖНО! Все проекты GameMaker имеют файл с именем "options.ini", поэтому вы не можете назвать какой-либо файл таким именем (вы получите ошибку uid при попытке скомпилировать ваш игровой проект).

Обратите внимание, что для игр с локализацией очень важно, чтобы акцентированные буквы могли быть прочитаны из внешних файлов. Это означает, что вам следует сначала создать ini-файл в формате UTF8, а затем добавить его в GameMaker как включенный файл, чтобы он экспортировался при запуске игры и использовался вместо стандартного ini-файла в формате ANSI, который создается функциями ini по умолчанию, если он не был предоставлен ранее. Таким образом, вы сможете читать и писать в нем правильно, с сохранением всех ударений и нелатинских букв.

Для файлов ini существуют следующие функции:

  • ini_open
  • ini_close
  • ini_write_real
  • ini_write_string
  • ini_read_real
  • ini_read_string
  • ini_key_exists
  • ini_section_exists
  • ini_key_delete
  • ini_section_delete
  • ini_open_from_string
