ini_key_delete

С помощью этой функции вы можете удалить выбранный ключ (и соответствующее ему значение) из ini-файла.

Синтаксис:

ini_key_delete(section, key);

АргументТип Описание
section The section to delete a key from.
key The key to delete.

Возвращается:

Пример:

ini_open("savedata.ini");
ini_write_real("save1","Score",score);
ini_key_delete("save1","Score");
ini_close();

Этот пример откроет файл "savedata.ini" и запишет значение в "save1" > "Score". Затем он удалит ключ "Score" и закроет файл .ini.
