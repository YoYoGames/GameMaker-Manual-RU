ini_key_exists

Эта функция проверяет, существует ли ключ в текущем открытом ini, и возвращает true, если существует, или false в противном случае. Эта проверка не является необходимой для предотвращения ошибок, поскольку, если ключ не существует, чтение из несуществующего ключа просто вернет значение по умолчанию. Однако это может быть полезно, чтобы узнать, сохранил ли ini файл определенные данные и некоторые другие вещи.

Синтаксис:

ini_key_exists(section, key);

АргументТип Описание
section The section in the open .ini file to check a key in.
key The key to check for.

Возвращается:

Пример:

ini_open("savedata.ini");
if !ini_key_exists("save1", "name")
{
    global.name = "Player1";
}
ini_close();

Это установит переменную global.name в значение "Player1", если не существует такого ключа "name".
