ini_open_from_string

Эта функция создаст временный ini-файл из строки и откроет его для чтения/записи. Строка должна быть правильно отформатирована как ini файл (т.е. с секциями, разрывами строк ключами и значениями), иначе ini файл не будет создан правильно. Обратите внимание, что этот ini файл является временным и будет удален из памяти в момент закрытия, потеряв всю информацию, которая была в нем сохранена, однако функция ini_close() возвращает строку полного ini файла, который затем может быть сохранен на сервере или на диске.

Синтаксис:

ini_open_from_string(string);

АргументТип Описание
string The string containing all the ini information.

Возвращается:

N/A

Пример:

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

Приведенный выше код создает временный ini для хранения информации из строки "str", затем читает из ini, прежде чем снова закрыть его.
