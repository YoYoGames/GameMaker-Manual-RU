ini_read_real

Вы можете использовать эту функцию для чтения числа из файла данных ini. Ini файлы разбиваются на секции, а затем каждая секция разбивается на пары ключ - значение. Таким образом, типичный ini файл выглядит следующим образом:

Синтаксис:

ini_read_real(section, key, default);

АргументТип Описание
section The section of the .ini to read from.
key The key within the relevant section of the .ini to read from.
default The value to return if a value is not found in the defined place (or the .ini file does not exist). Must be a real number.

Возвращается:

Пример:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0 );
ini_close();

Это откроет файл "savedata.ini" и установит значение score в значение в разделе "save1" > "score" в нем, затем снова закроет .ini. Если в разделе "save1" > "score" нет значения, или файл "savedata.ini" отсутствует. файла, счет будет установлен на 0.
