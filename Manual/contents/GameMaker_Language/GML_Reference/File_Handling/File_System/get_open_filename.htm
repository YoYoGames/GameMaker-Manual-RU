get_open_filename

Эта функция открывает диалог и запрашивает у игрока имя файла, который нужно открыть с помощью заданного фильтра. Фильтр имеет вид "name1|mask1|name2|mask2|... ", где маска может содержать различные варианты с точкой с запятой между ними, а также вы можете использовать "*" для обозначения любой строки. Например: "bitmaps|*.bmp;*.wmf", попросит пользователя выбрать имя растрового файла из системы, но только тех, которые имеют два указанных расширения (bmp и png). эта функция не открывает сам файл! Она только возвращает строку с полным именем и путем к файлу. Если пользователь нажимает кнопку "Отмена", возвращается пустая строка "".

При использовании этой функции важно отметить, что она предоставляет вам определенные права доступа к этому файлу на время игры. Таким образом, получив путь к файлу от пользователя, вы сможете обращаться к нему снова и снова без необходимости спрашивать. Однако, поскольку эта функция предназначена для чтения файла, предоставленные разрешения предназначены только для чтения, а сохранение не будет разрешено ОС. Если вам потребуется чтение и запись в файл, вы можете получить разрешение с помощью функции get_save_filename().

ПРИМЕЧАНИЕ: Эта функция действительна только для целей Windows и macOS.

Синтаксис:

get_open_filename(filter, fname);

АргументТип Описание
filter The file type filter and/or name.
fname The suggested filename to use.

Возвращается:

Пример:

var file;
file = get_open_filename("text file|*.txt", "");
if file != ""
{
    file_text_open_read(file);
}

Приведенный выше код попросит пользователя выбрать текстовый файл для открытия, затем проверит имя и путь к файлу, и если он существует, то откроет файл для чтения.
