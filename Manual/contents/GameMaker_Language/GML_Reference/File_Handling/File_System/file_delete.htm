file_delete

Эта функция удалит указанный файл из системы. Следует отметить, что эта функция удалит только те файлы, которые GameMaker может создать и разобрать: ini файлы, текстовые файлы и бинарные файлы, или файлы, созданные для хранения ресурсов, созданных игрой, например, sprites или поверхностей. Однако она не будет удалять никакие другие файлы. Функция также вернет true, если файл был успешно удален, или false при любых других обстоятельствах.

ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker является "песочницей"! Пожалуйста, обратитесь к разделу " Ограничения файловой системы " для получения дополнительной информации.

Синтаксис:

file_delete(fname);

АргументТип Описание
fname The name of the file to delete.

Возвращается:

Пример:

if file_exists("level.txt")
{
    file_delete("level.txt");
}

Это позволит проверить наличие файла и, если он существует, удалить его.
