переименование файла

Эта функция переименует указанный файл с указанным именем. Функция возвращает true, если файл был успешно переименован, или false при любых других обстоятельствах.

ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации.

Синтаксис:

file_rename(oldname, newname);

АргументТип Описание
oldname The name of the file to change.
newname The new name to give the file.

Возвращается:

Пример:

if file_exists("level1.txt")
{
    file_rename("level1.txt", "level.txt");
}

Это позволит проверить наличие файла и, если он существует, переименовать его.
