Файловая система

В GameMaker существует ряд функций, которые позволяют манипулировать файлами всех типов различными способами:

  • file_exists
  • file_delete
  • file_rename
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • file_attributes

ПРИМЕЧАНИЕ Эти функции не работают с реальными файлами! Они работают только со строками, связанными с различными элементами пути, и должны использоваться вместе с другими файловыми функциями.

  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

Наконец, есть несколько специальных функций для получения имен файлов, которые можно использовать для загрузки и сохранения файлов с фильтрами типа файла. Эти функции откроют окно проводника и, следовательно, заставят игру потерять фокус. Это означает, что любой ввод, от которого зависит игра, может быть неправильно зарегистрирован, и поэтому мы рекомендуем (при использовании этих функций в событии клавиатуры) использовать событие " вверх", а не "вниз" или "нажато".

ПРИМЕЧАНИЕ Эти функции доступны только для Windows и macOS.

ПРЕДУПРЕЖДЕНИЕ Эти функции могут работать не так, как вы ожидаете, из-за того, что GameMaker находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации.

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
