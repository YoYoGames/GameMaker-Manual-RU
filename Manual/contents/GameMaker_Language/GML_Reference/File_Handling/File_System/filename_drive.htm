имя_диска

Эта функция возвращает информацию о приводе имени файла.

Синтаксис:

filename_drive(fname);

АргументТип Описание
fname The file to use.

Возвращается:

Пример:

drive = filename_drive(file_find_first(working_directory + "*.doc", 0));

Приведенный выше код получает информацию о диске (в виде строки) указанного файла.
