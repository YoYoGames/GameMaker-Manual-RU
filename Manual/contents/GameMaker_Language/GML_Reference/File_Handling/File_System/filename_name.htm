имя_файла

Эта функция возвращает часть имени указанного файла с расширением, но без пути.

Синтаксис:

filename_name(fname);

АргументТип Описание
fname The file to use.

Возвращается:

String

Пример:

name = filename_name(file_find_first("C:/Games/*.doc", 0));

Приведенный выше код получает имя (в виде строки) первого файла типа "doc", найденного в указанном каталоге.
