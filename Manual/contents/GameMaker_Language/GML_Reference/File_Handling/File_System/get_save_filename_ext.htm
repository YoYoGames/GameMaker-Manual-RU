get_save_filename_ext

Эта функция открывает диалог и запрашивает у игрока имя файла для сохранения с заданным фильтром. Фильтр имеет вид "name1|mask1|name2|mask2|... ", где маска может содержать различные варианты с точкой с запятой между ними, а также вы можете использовать "*" для обозначения любой строки. Например: "bitmaps|*.bmp;*.wmf", попросит пользователя выбрать имя растрового файла из системы, но только тех, которые имеют два указанных расширения (bmp и png). эта функция не сохраняет сам файл! Она только возвращает строку с полным именем и путем к файлу. Если пользователь нажимает кнопку "Отмена", возвращается пустая строка "". Вы также можете добавить каталог, с которого будет начинаться поиск, и надпись, которая будет отображаться в верхней части окна вместо стандартной.

При использовании этой функции важно отметить, что она предоставляет вам определенные разрешения на этот файл на все время игры. Таким образом, получив от пользователя путь к файлу, вы можете обращаться к нему снова и снова без необходимости спрашивать, причем разрешение распространяется в равной степени на чтение и запись.

ПРИМЕЧАНИЕ: Эта функция действительна только для целей Windows и macOS.

Синтаксис:

get_save_filename_ext(filter, fname, directory, caption);

АргументТип Описание
filter The file type filter and/or name.
fname The suggested filename to use.
directory The directory to start the search from.
caption The caption for the window.

Возвращается:

Пример:

var file;
file = get_save_filename_ext("screenshot|*.png", "", working_directory, "Save a level here");
if (file != "")
{
    screen_save(file);
}

Приведенный выше код попросит пользователя выбрать имя файла изображения для сохранения в рабочем каталоге, а затем проверит возвращаемое имя файла и путь Если это не пустая строка, то сделает снимок экрана и сохранит его, используя заданное имя.
