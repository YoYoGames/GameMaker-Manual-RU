файл_найти_первым

Эта функция возвращает имя первого файла, который удовлетворяет маске и атрибутам. Если такого файла не существует, то возвращается пустая строка.

Маска может содержать путь с подстановочными знаками, например, C:\temp\*.doc.

Атрибуты задают дополнительные файлы, которые вы хотите увидеть, поэтому обычные файлы всегда возвращаются, если они удовлетворяют маске. Вы можете сложить следующие константы, чтобы увидеть нужный вам тип файлов (с помощью |). Если вы не хотите добавлять никаких атрибутов, используйте 0.

Этот код проверяет, существует ли указанный каталог, и если да, то идет туда и возвращает первый найденный файл "только для чтения" .doc. Затем он ищет еще два файла и закрывает программу поиска файлов.

Вы можете искать любое количество файлов с помощью while цикл:

var files = [];
var file_name = file_find_first("/User Content/*.doc", fa_readonly);

while (file_name != "")
{
    array_push(files, file_name);

    file_name = file_find_next();
}

file_find_close();

Приведенный выше код создает пустой массив для хранения всех найденных имен файлов и начинает поиск файлов .doc, доступных только для чтения. Если имя файла не является пустой строкой, он помещает его в массив files, а затем пытается найти следующий файл. Цикл будет повторяться до тех пор, пока не будет найдена пустая строка (это означает, что больше нет подходящих файлов), после чего цикл завершается и закрывает программу поиска файлов.
