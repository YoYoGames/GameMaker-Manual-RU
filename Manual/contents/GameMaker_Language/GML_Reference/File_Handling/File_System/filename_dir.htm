имя_дира

Эта функция возвращает часть каталога указанного имени файла, которая обычно совпадает с путем, за исключением последней обратной косой черты.

Синтаксис:

filename_dir(fname);

АргументТип Описание
fname The file to use.

Возвращается:

Пример:

dir = filename_dir("Test.ini");

Приведенный выше код получает каталог (в виде строки) указанного файла.
