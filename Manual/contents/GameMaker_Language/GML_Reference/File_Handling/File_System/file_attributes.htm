атрибуты файла

Вы можете использовать эту функцию для проверки атрибутов заданного файла. Вы можете сложить следующие константы, чтобы узнать тип нужных вам файлов:

Постоянная Описание
fa_readonly Файлы только для чтения
fa_hidden Скрытые файлы
fa_sysfile Системные файлы
fa_volumeid Файлы с идентификатором тома
fa_directory Справочники
fa_archive Архивные файлы

ПРИМЕЧАНИЕ: Это функция только для сайта Windows.

Синтаксис:

file_attributes(fname, attr);

Аргумент Тип Описание
fname Строка The name of the file to check.
attr Атрибут файла Постоянная The attributes to check for.

Возвращается:

Boolean

Пример:

if !file_attributes(file, fa_hidden)
{
    file_delete(file);
}

Это позволит проверить файл на предмет того, скрыт он или нет, и если нет, то он будет удален.
