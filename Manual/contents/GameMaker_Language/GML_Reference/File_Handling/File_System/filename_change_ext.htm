filename_change_ext

Эта функция возвращает указанное имя файла с расширением (включая точку), измененным на новое расширение. Используя пустую строку в качестве нового расширения, вы можете полностью удалить часть расширения.

Синтаксис:

filename_change_ext(fname, newext);

АргументТип Описание
fname The file to use.
newext The new extension to use.

Возвращается:

Пример:

ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), "");

Приведенный выше код получает имя файла (в виде строки) с удаленной частью расширения.
