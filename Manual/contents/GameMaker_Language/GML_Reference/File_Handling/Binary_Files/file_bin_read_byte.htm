файл_bin_read_byte

Эта функция возвращает байт данных из текущей позиции в файле с заданным идентификатором файла. Вы передаете значение ID файла, возвращаемое функцией file_bin_open().

ПРИМЕЧАНИЕ: Эти функции не работают, если целевым модулем является HTML5.

Синтаксис:

file_bin_read_byte(binfile);

АргументТип Описание
binfile The ID of the file to get the data from.

Возвращается:

Пример:

file = file_bin_open("myfile.bin", 2);
data = file_bin_read_byte(file);
file_bin_close(file);

Это позволит открыть файл из того же каталога, что и игра, и получить из него байт данных, после чего снова закрыть его.
