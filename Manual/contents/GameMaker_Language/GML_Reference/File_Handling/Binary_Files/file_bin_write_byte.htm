файл_bin_запись_байт

Эта функция записывает байт данных в файл, идентифицированный идентификатором файла, в текущей позиции записи. Вы передаете значение идентификатора файла, возвращаемое функцией file_bin_open() и байт данных для записи.

ПРИМЕЧАНИЕ: Эти функции не работают, если целевым модулем является HTML5.

Синтаксис:

file_bin_write_byte(binfile, byte);

АргументТип Описание
binfile The ID of the file to write to.
byte The data to write.

Возвращается:

Пример:

file_bin_write_byte(file, data);

Это запишет байт в выбранный файл.
