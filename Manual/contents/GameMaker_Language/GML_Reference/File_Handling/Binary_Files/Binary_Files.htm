Двоичные файлы

В редких ситуациях вам может понадобиться прочитать данные из двоичных файлов. Эти функции могут быть использованы для этой цели, хотя в целом мы рекомендуем вам использовать специальные функции буфера для создания, сохранения и загрузки двоичных данных, а не эти функции.

ПРИМЕЧАНИЕ: Эти функции не работают, если целевым модулем является HTML5.

Для этого существуют следующие низкоуровневые процедуры:

  • file_bin_open
  • file_bin_rewrite
  • file_bin_close
  • file_bin_size
  • file_bin_position
  • file_bin_seek
  • file_bin_write_byte
  • file_bin_read_byte
