load_csv

Эта функция загрузит файл формата CSV и преобразует его в сетку DS, возвращая уникальное значение ID для созданной сетки.

Ваш CSV-файл должен соответствовать спецификации CSV. Если синтаксис вашего CSV не соответствует требованиям, вы получите неожиданные результаты при загрузке файла.

Синтаксис:

load_csv(filename)

Аргумент Тип Описание
filename Строка The name of the file to open (as a string)

Возвращается:

DS Grid ID

Пример:

file_grid = load_csv("spreadsheet.csv");
var ww = ds_grid_width(file_grid);
var hh = ds_grid_height(file_grid);
var xx = 32;
var yy = 32;
for (var i = 0; i < ww; i++)
{
    for (var j = 0; j < hh; j++)
    {
        draw_text(xx, yy, file_grid[# i, j]);
        yy += 32;
    }
    yy = 32;
    xx += 32;
}

Приведенный выше код открывает заданный CSV-файл и сохраняет полученную сетку DS в переменной "file_grid". Затем эта сетка будет разобрана в нескольких for и содержимое выведено на экран.
