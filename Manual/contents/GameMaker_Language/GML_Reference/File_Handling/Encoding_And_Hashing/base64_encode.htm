base64_encode

Эта функция преобразует строку в строку, закодированную в формате base64. Это широко используемая схема кодирования, которая часто применяется для любых носителей, которые должны храниться или передаваться через Интернет в виде текста, и делает вывод нечитаемым для человеческим глазом.

Синтаксис:

base64_encode(string)

АргументТип Описание
string The string to encode.

Возвращается:

Пример:

var str, file;
str = base64_encode(game_data);
file = file_text_open_write("save.txt");
file_text_write_string(file, str);
file_text_close(file);

Приведенный выше код преобразует строку, хранящуюся в "game_data", в строку в кодировке base64, которая затем сохраняется во внешнем текстовом файле.
