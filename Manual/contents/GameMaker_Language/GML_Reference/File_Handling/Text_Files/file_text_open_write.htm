file_text_open_write

Эта функция открывает текстовый файл с указанным именем только для записи (если файл не существует, он создается), возвращая уникальный идентификатор файла, который следует сохранить в переменной, так как он будет использоваться для всех дальнейших действий с этим файлом.

Обратите внимание, что если файл не может быть создан (например, из-за недопустимого имени файла), функция вернет -1.

Если файл уже существует, использование этой функции приведет к его перезаписи при сохранении данных. Если вы хотите добавить данные в файл, сохранив его существующее содержимое нетронутым, используйте функцию file_text_open_append().

ПРИМЕЧАНИЕ: Одновременно может быть открыто не более 32 файлов. Вы также должны всегда закрывать файлы после завершения работы, так как при этом записывается информация и освобождается память, связанная с файлом.

ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации.

Синтаксис:

file_text_open_write(fname);

Аргумент Тип Описание
fname Строка The name of the file to write to.

Возвращается:

Text File ID or -1

Пример:

var file;
file = file_text_open_write(working_directory + "level.txt");
file_text_write_string(file, level_data);
file_text_close(file);

Приведенный выше код откроет файл "level.txt" для записи, а затем запишет строку, хранящуюся в переменной "level_data", после чего снова закроет файл.
