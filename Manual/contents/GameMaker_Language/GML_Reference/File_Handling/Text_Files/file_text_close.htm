файл_текст_закрыть

После завершения работы с файлом (чтения из него или записи в него) вы должны закрыть его снова, иначе вы рискуете потерять содержащуюся в нем информацию. Это также предотвращает утечку памяти и гарантирует, что вы никогда не превысите лимит файлов, имея более 32 открытых файлов.

Синтаксис:

file_text_close(fileid);

АргументТип Описание
fileid The id of the file to close.

Возвращается:

Пример:

var file = file_text_open_write(working_directory + "Game_Data.txt");
while (!file_text_eof(file))
{
    file_text_readln(file);
}
file_text_write_string(file, level_data);
file_text_close(file);

Приведенный выше код открывает файл для записи, затем циклически просматривает строки текста, уже записанные в файл, пока не дойдет до конца. В этот момент он записывает строку, а затем снова закрывает файл.
