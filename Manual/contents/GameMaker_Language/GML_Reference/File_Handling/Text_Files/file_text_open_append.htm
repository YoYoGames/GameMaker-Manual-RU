file_text_open_append

Эта функция открывает текстовый файл с указанным именем для записи (если файл не существует, он создается), возвращая уникальный идентификатор файла, который следует сохранить в переменной, так как он будет использоваться для всех дальнейших действий с этим файлом. Позиция в файле для записи устанавливается на последнюю строку текста, которую содержит файл, поэтому все существующие данные не перезаписываются.

Обратите внимание, что если файл не может быть создан (например, из-за недопустимого имени файла), функция вернет -1.

ПРИМЕЧАНИЕ: Одновременно может быть открыто не более 32 файлов. Вы также должны всегда закрывать файлы после завершения работы, так как при этом записывается информация и освобождается память, связанная с файлом.

ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации.

Синтаксис:

file_text_open_append(fname);

Аргумент Тип Описание
fname Строка The name of the file to append to.

Возвращается:

Text File ID or -1

Пример:

file = file_text_open_append(working_directory + "save.txt");

Это откроет файл "save.txt" из той же директории, что и игра, и сохранит идентификатор файла в переменной "file".
