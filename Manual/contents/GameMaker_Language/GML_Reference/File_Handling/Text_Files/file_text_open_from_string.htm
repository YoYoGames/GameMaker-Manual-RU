file_text_open_from_string

<<<<<<< HEAD

Эта функция создает текстовый файл из строки и открывает его для чтения, возвращая "handle" файла, который должен использоваться во всех последующих вызовах функции file для чтения из этого файла. Обратите внимание, что этот файл является временным и предназначен только для чтения, и как таковой он будет удален из памяти в момент его закрытия.

ПРИМЕЧАНИЕ: Одновременно может быть открыто не более 32 файлов. Вы также должны всегда закрывать файлы после завершения работы, так как при этом записывается информация и освобождается память, связанная с файлом.

ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации.

Синтаксис:

=======

This function will create a text file from a string and open it for reading, returning the file "handle" that should be used in all further file function calls to read from this file. Note that this file is temporary and read only, and as such it will be removed from memory the moment it is closed.

 You can only have a maximum of 32 files open at any one time. You should also always close files when finished as this writes the information and frees the memory associated with the file.
