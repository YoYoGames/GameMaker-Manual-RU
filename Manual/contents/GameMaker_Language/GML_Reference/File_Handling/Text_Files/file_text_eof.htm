file_text_eof

Эта функция возвращает true, если достигнут конец заданного открытого текстового файла, или false, если нет.

Синтаксис:

file_text_eof(fileid);

АргументТип Описание
fileid The id of the file to check.

Возвращается:

Пример:

var num = 0;
var file = file_text_open_read(working_directory + "Game_Data.txt");
while (!file_text_eof(file))
{
    str[num++] = file_text_readln(file);
}
file_text_close(file);

Приведенный выше код открывает файл для записи, затем циклически просматривает строки текста, уже записанные в файл, пока не достигнет конца, сохраняя каждую строку в массиве "str".
