file_text_open_read

<<<<<<< HEAD

Эта функция открывает текстовый файл с указанным именем только для чтения, возвращая уникальный идентификатор файла, который следует сохранить в переменной, так как он будет использоваться для всех дальнейших действий с этим файлом. Если файл не существует, то функция вернет значение -1.

ПРИМЕЧАНИЕ: Одновременно может быть открыто не более 32 файлов. Вы также должны всегда закрывать файлы после завершения работы, так как при этом записывается информация и освобождается память, связанная с файлом.

ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации.

Синтаксис:

=======

This function opens the text file with the indicated filename for reading only, returning the unique id of the file that which should be stored in a variable as it will be used for all further actions to do with that file. If the file does not exists then the function will return the value -1.

 You can only have a maximum of 32 files open at any one time. You should also always close files when finished as this writes the information and frees the memory associated with the file.
