строка_текста_записи_файла

С помощью этой функции вы можете записать строку в ранее открытый текстовый файл. Если файл уже содержит информацию, эта информация будет стерта, а строка будет записана в начало файла, если только вы не открыли файл с помощью функции file_text_open_append(). Вы также можете избежать этого, используя file_text_readln() вместе с функцией file_text_eof() чтобы просмотреть содержимое файла до конца, а затем начать запись.

Синтаксис:

file_text_write_string(fileid, str);

АргументТип Описание
fileid The id of the file to edit.
str The string to write to the file.

Возвращается:

Пример:

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i)
{
    file_text_write_real(file, scr[i]);
    file_text_writeln(file);
    file_text_write_string(file, scr_name[i]);
    file_text_writeln(file);
}
file_text_close(file);

Приведенный выше код открывает файл для записи, а затем в цикле просматривает два массива, записывая каждое значение массива в новую строку файла. После завершения цикла файл закрывается.
