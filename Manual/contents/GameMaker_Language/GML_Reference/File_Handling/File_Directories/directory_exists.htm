directory_exists

Эта функция возвращает true, если указанный каталог существует, или false, если не существует. Указанное имя должно включать полный путь, а не относительный, и по умолчанию вы не можете получить доступ к каталогам, находящимся вне пакета игры, так как все игры находятся в песочнице (см. раздел о файловой системе для получения дополнительной информации).

Синтаксис:

directory_exists(dname)

АргументТип Описание
dname The name of the directory to look for.

Возвращается:

Пример:

if directory_exists(working_directory + "Saves/")
{
    file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);
}

Эта программа проверит, существует ли указанный каталог, затем, если существует, перейдет туда и вернет первый найденный файл "только для чтения" doc.
