temp_directory

Это можно использовать для возврата временного каталога, создаваемого для вашей игры при каждом запуске (корень не содержит конечного "\"). В этом каталоге будут храниться файлы, к которым можно получить доступ во время работы игры, но он будет удален (вместе со всеми файлами, которые в нем содержатся), когда игра будет закрыта. вместе со всеми файлами, которые он содержит), когда игра будет закрыта.

ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации.

Синтаксис:

temp_directory

Возвращается:

Пример:

ini_open(temp_directory + "\temp_ini.ini");

Это откроет ini-файл во временном каталоге игры (создав его, если он еще не существует).
