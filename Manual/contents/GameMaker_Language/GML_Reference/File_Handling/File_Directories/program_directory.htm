Это вернет каталог, в котором хранится исполняемый файл игры. Однако это не всегда может быть полезно, особенно потому, что некоторые устройства запускают exe из *.zip-файла, так что это возвращает одно и то же, независимо от того, откуда на самом деле запущена игра.

ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации.

ПРИМЕЧАНИЕ: Если вы отключите песочницу, вы все равно не сможете записывать в этот каталог, чтобы предотвратить возможные проблемы с удалением необходимых файлов игры.

Синтаксис:

program_directory

Возвращается:

Пример:

dir = program_directory;

Это сохранит каталог, в котором находится исполняемый файл, в переменной.
