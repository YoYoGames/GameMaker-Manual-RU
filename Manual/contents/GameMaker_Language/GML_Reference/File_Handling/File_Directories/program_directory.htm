директория программы

Это вернет каталог, в котором хранится исполняемый файл игры. Однако это не всегда может быть полезно, особенно потому, что некоторые устройства запускают exe из *.zip-файла, так что это возвращает одно и то же, независимо от того, откуда на самом деле запущена игра.

ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации.

ПРИМЕЧАНИЕ: Если вы отключите песочницу, вы все равно не сможете записывать в этот каталог, чтобы предотвратить возможные проблемы с удалением необходимых файлов игры.

Синтаксис:

program_directory

Возвращается:

Пример:

dir = program_directory;

Это сохранит каталог, в котором находится исполняемый файл, в переменной.

program_directory

This function returns the directory where the game executable is stored. However, this may not always be useful, particularly as some devices run the exe from a *.zip file, so this would return the same no matter where the game is actually running from.

 This function may not work as you expect due to GameMaker being sandboxed! Please see the section on The File System for more information.

 GameMaker doesn't limit writing to this directory so whether this is possible depends on OS permissions. It is however discouraged that you write to program_directory, as it is possible to damage the game installation this way.

 

Syntax:

program_directory

 

Returns:

String

 

Example:

dir = program_directory;

This will store the directory where the executable is stored in a variable.
