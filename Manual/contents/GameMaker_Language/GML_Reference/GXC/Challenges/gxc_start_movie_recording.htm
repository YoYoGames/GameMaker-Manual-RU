gxc_start_movie_recording

Эта функция используется для начала записи вашего испытания на GXC. Вы указываете ширину, высоту и частоту кадров записи, а также булево значение для того, нужно ли записывать звук или нет.

Синтаксис:

gxc_start_movie_recording(width, height, fps, record_audio);

Аргумент Тип Описание
width Реально The width of the recording
height Реально The height of the recording
fps Реально The frame rate (frames-per-second) of the recording
record_audio Булево Whether audio should be recorded (true) или нет (false)

Возвращается:

N/A

Пример:

gxc_start_movie_recording(320, 240, 4, false);

При этом начнется запись видео с разрешением 320x240 при частоте 4 кадра в секунду, без записи звука.

Назад: Opera GX и GXC
Далее: gxc_stop_movie_recording
