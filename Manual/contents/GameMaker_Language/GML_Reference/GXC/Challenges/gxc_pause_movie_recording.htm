gxc_pause_movie_recording

Эта функция используется для приостановки записи GXC, которая ранее была начата с помощью gxc_start_movie_recording().

Синтаксис:

gxc_pause_movie_recording();

Возвращается:

Пример:

if (keyboard_check_pressed(vk_escape))
{
    gxc_pause_movie_recording();
}

Это приостановит запись видеоролика GXC, если нажата клавиша escape.
