gxc_stop_movie_recording

Эта функция используется для остановки записи GXC, ранее начатой с помощью gxc_start_movie_recording(). Она также отправляет запись на URL, указанный в аргументах, однако поведение отличается в зависимости от типа указанной строки:

  • Если вы укажете имя файла, запись будет сохранена с этим именем в папке "Загрузки".
  • Если вы укажете URL, запись будет загружена на этот URL.

Метод URL следует использовать, когда запись необходимо загрузить в GXC, а файл следует сохранить локально только для проверки успешности записи.

Синтаксис:

gxc_stop_movie_recording(filename);

Аргумент Тип Описание
filename Строка The file name or URL where the recording is to be saved or uploaded

Возвращается:

N/A

Пример:

gxc_submit_challenge_score(_score, function(_status, _result) {
    var _uploadUrl = _result.data.gameplayVideoUploadUrl;
    gxc_stop_movie_recording(_uploadUrl);
});

Приведенный выше код использует библиотеку API GXC для загрузки оценки в активный в данный момент вызов, а когда в его методе обратного вызова получен ответ, он извлекает URL-адрес загрузки видео и загружает запись в него.
