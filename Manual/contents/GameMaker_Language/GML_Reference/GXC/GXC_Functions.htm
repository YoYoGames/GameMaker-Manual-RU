Opera GX и GXC

В этом разделе описаны функции, предоставляемые для целевой платформы Opera GX. Функции, перечисленные на этой странице, позволяют записывать экран и загружать его в GXC вместе с результатом испытания, однако следует учитывать, что для этого необходимо использовать библиотеку API GXC. Пожалуйста, обратитесь к разделу Opera GX на сайте YoYo Games Helpdesk за инструкциями по внедрению библиотеки GXC API и работе с задачами.

Для загрузки видеозаписей с вызовами предусмотрены следующие функции:

  • gxc_start_movie_recording
  • gxc_stop_movie_recording
  • gxc_pause_movie_recording
  • gxc_resume_movie_recording
