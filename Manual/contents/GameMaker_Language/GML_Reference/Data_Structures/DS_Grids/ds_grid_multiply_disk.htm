ds_grid_multiply_disk

Эта функция берет все значения в заданной дискообразной области сетки DS и умножает каждое из них на заданную величину.

ПРИМЕЧАНИЕ Эта функция работает только с вещественными числами, а не со строками.

Синтаксис:

ds_grid_multiply_disk(index, xm, ym, r, val);

Аргумент Тип Описание
index DS Grid ID The index of the grid.
xm Реально The x position of the disk on the grid.
ym Реально The y position of the disk on the grid.
r Реально The radius of the disk on the grid.
val Реально The value to multiply the cells within the disk with.

Возвращается:

N/A

Пример:

ds_grid_multiply_disk(mygrid, 5, 5, 5, 2)

Приведенный выше код возьмет все значения, найденные в области круговой сетки, и умножит каждое из них на 2.
