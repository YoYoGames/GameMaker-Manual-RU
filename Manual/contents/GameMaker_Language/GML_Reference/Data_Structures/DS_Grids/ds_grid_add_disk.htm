ds_grid_add_disk

Эта функция может быть использована для добавления заданного значения (вещественного или строкового) ко всем значениям ячеек, находящихся в пределах определенной области диска сетки. Добавляемое значение должно быть того же типа, что и ячейки сетки, т.е. нельзя добавить строку к вещественному значению или наоборот. к вещественному или наоборот, а для строк это соответствует конкатенации.

Синтаксис:

ds_grid_add_disk(index, xm, ym, r, val);

АргументТип Описание
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.
val The value to add to the cells within the disk.

Возвращается:

Пример:

ds_grid_add_disk(grid, 7, 6, 5, 2)

Это добавит 2 ко всем значениям, хранящимся в ячейках в пределах определенной области диска сетки DS, на которую ссылается переменная "grid".
