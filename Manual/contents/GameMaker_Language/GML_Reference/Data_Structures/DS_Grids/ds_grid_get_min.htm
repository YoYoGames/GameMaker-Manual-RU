ds_grid_get_min

Эта функция может быть использована для нахождения минимального значения для всех ячеек, находящихся в заданной области сетки, как показано на рисунке ниже:

Синтаксис:

ds_grid_get_min(index, x1, y1, x2, y2);

АргументТип Описание
index The index of the grid.
x1 The left cell column of the region.
y1 The top cell row of the region.
x2 The right cell column of the region.
y2 The bottom cell row of the region.

Возвращается:

Real or String

Пример:

val = ds_grid_get_min(grid, 0, 0, 5, 5)

Приведенный выше код установит переменную "val" в минимальное значение, содержащееся в заданной области сетки DS, индексированной в переменной "grid".
