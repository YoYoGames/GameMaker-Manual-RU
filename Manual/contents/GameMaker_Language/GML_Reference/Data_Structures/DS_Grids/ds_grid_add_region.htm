ds_grid_add_region

Эта функция может быть использована для добавления заданного значения (вещественного или строкового) ко всем значениям ячеек, находящихся в определенной области сетки. Добавляемое значение должно быть того же типа, что и значения в ячейках сетки, т.е. нельзя добавить строку к вещественному или наоборот, а для строк это соответствует конкатенации.

Синтаксис:

ds_grid_add_region(index, x1, y1, x2, y2, val);

АргументТип Описание
index The index of the grid.
x1 The x position of the left of the region in the grid.
y1 The y position of the top of the region in the grid.
x2 The x position of the right of the region in the grid.
y2 The y position of the bottom of the region in the grid.
val The value to add to the region cells.

Возвращается:

Пример:

ds_grid_add_region(grid, 2, 4, 5, 5, ".")

Это добавит "." ко всем строкам, хранящимся в ячейках внутри определенной области сетки DS, на которую ссылается переменная "grid".
