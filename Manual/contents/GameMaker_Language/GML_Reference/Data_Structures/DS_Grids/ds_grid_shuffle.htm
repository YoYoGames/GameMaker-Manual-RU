ds_grid_shuffle

Эта функция может быть использована для рандомизации положения всех значений во всех ячейках сетки.

ПРИМЕЧАНИЕ Эта функция будет перетасовывать ячейки сетки на одни и те же позиции каждый раз, когда игра запускается заново, поскольку GameMaker каждый раз генерирует одно и то же начальное случайное зерно, чтобы облегчить отладку кода. Чтобы избежать такого поведения, используйте randomise() в начале игры. Это справедливо только при тестировании и отладке игры, так как конечный исполняемый пакет не будет демонстрировать такое поведение и будет случайным в каждой игре.

Синтаксис:

ds_grid_shuffle(index);

АргументТип Описание
index The index of the grid to shuffle.

Возвращается:

Пример:

ds_grid_shuffle(grid)

Это позволит взять все значения в сетке DS, проиндексированные в переменной "grid", и перетасовать их, чтобы получить другой порядок.
