ds_grid_width

Эта функция возвращает ширину заданной сетки. Это значение представляет собой количество ячеек сетки вдоль оси x и всегда является целым числом, как показано на рисунке ниже:

Синтаксис:

ds_grid_width(index);

АргументТип Описание
index This index of the grid to find the width of.

Возвращается:

Пример:

for (var i = 0; i < ds_grid_width(grid); ++i)
{
    for (var j = 0; j < ds_grid_height(grid); ++j)
    {
        if (ds_grid_get(grid, i, j) == 1)
        {
            instance_create_layer(i * 32, j * 32, "Walls", obj_Wall);
        }
    }
}

Приведенный выше код пройдется по сетке DS, индексированной в переменной "grid", и если значение, найденное в любой конкретной ячейке, равно 1, он создаст экземпляр "obj_Wall" в соответствующей позиции в комнате.
