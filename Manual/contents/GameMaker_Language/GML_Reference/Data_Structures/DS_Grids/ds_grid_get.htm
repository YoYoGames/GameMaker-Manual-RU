ds_grid_get

Эта функция может быть использована для получения значения из любой ячейки в заданной сетке DS. Если вы передадите в функцию недопустимые координаты сетки, то возвращаемое значение будет undefined, а в окне вывода будет показана ошибка.

Синтаксис:

ds_grid_get(index, x, y);

АргументТип Описание
index The index of the grid.
x The x position of the cell you want to find the value of.
y The y position of the cell you want to find the value of.

Возвращается:

Variable

Пример:

var xx = irandom(ds_grid_width(grid) - 1);
var yy = irandom(ds_grid_height(grid) - 1);
val = ds_grid_get(grid, xx, yy)

Приведенный выше код выбирает случайную ячейку из сетки DS, индексированную в переменной "grid", и сохраняет ее значение в переменной "val".
