ds_grid_get_sum

Эта функция может быть использована для суммирования всех значений всех ячеек, находящихся в определенной области сетки, как показано на рисунке ниже:

Синтаксис:

ds_grid_get_sum(index, x1, y1, x2, y2);

АргументТип Описание
index The index of the grid.
x1 The left cell column of the region.
y1 The top cell row of the region.
x2 The right cell column of the region.
y2 The bottom cell row of the region.

Возвращается:

Real or String

Пример:

val = ds_grid_get_sum(grid, 0, 0, 5, 5)

Приведенный выше код установит переменную "val" в сумму всех значений, содержащихся в данной области сетки DS, индексированной в переменной "grid".
