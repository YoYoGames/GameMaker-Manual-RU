ds_grid_read

Эта функция может быть использована для преобразования строки, которая была создана ранее функцией ds_grid_write() обратно в сетку DS. Сетка DS должна быть создана ранее (см. пример ниже).

Обратите внимание, что если указанная строка DS была написана в среде GameMaker: Studio 1.2.x (или старше), следует указать дополнительный аргумент "legacy", установив его в значение true, поскольку после этого формат строки изменился.

Синтаксис:

ds_grid_read(index, string [, legacy]);

Аргумент Тип Описание
index DS Grid ID The index of the grid to read.
string Строка The string to read into the DS grid.
legacy Булево OPTIONAL Может быть либо true, либо false, либо полностью опущен.

Возвращается:

N/A

Пример:

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close();

Приведенный выше код создает сетку DS на основе размера комнаты (каждый квадрат 32x32 пикселей представляет собой одну ячейку сетки), а затем считывает ранее сохраненный набор данных сетки из ini-файла в новую сетку DS.
