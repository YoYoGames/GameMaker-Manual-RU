ds_grid_destroy

Эта функция удалит заданную структуру данных сетки из памяти, освободив используемые ресурсы и удалив все содержащиеся в ней значения. Эту функцию всегда следует использовать, когда вы закончили работу с сеткой DS, чтобы предотвратить утечку памяти, которая может замедлить работу и привести к краху вашей игры.

ВАЖНО! Когда вы создаете структуру данных, значение индекса для ее идентификации является целым числом, начинающимся с 0. Это означает, что структуры данных разных типов могут иметь одинаковое значение индекса, поэтому, если вы сомневаетесь, следует использовать функцию ds_exists() функцию перед обращением к ним. Также обратите внимание, что индексы используются повторно, поэтому значение индекса уничтоженной структуры данных может быть использовано вновь созданной, поэтому мы рекомендуем после уничтожения всегда устанавливать переменную, в которой хранился индекс DS, в -1.

Синтаксис:

ds_grid_destroy(index);

АргументТип Описание
index This index of the grid to destroy.

Возвращается:

Пример:

if (lives == 0)
{
    ds_grid_destroy(Wall_Grid);
    Wall_Grid = -1;
    room_goto(rm_Menu);
}

Приведенный выше код проверит значение встроенной глобальной переменной "lives" и, если оно равно 0, уничтожит сетку DS, индексированную в переменной "Wall_Grid", а затем поменяет комнаты.
