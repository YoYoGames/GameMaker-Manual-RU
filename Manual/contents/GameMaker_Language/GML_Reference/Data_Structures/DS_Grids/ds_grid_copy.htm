ds_grid_copy

С помощью этой функции можно скопировать содержимое одной сетки в другую. Обе сетки должны быть созданы ранее с помощью функции ds_grid_create() функция.

Синтаксис:

ds_grid_copy(destination, source);

Аргумент Тип Описание
destination DS Grid ID This index of the grid to copy to.
source DS Grid ID This index of the grid to copy from.

Возвращается:

N/A

Пример:

n_grid = ds_grid_create(ds_grid_width(a_grid), ds_grid_height(a_grid));
ds_grid_copy(n_grid, a_grid);
ds_grid_clear(a_grid, -1)

Приведенный выше код создает новую сетку DS, основываясь на ширине и высоте ранее созданной сетки, затем копирует информацию из предыдущей сетки в новую. Наконец, он очищает старую сетку, чтобы все ячейки имели значение -1.
