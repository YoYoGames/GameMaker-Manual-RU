ds_grid_resize

С помощью этой функции вы можете изменить размер заданной сетки DS на другую ширину и/или высоту. Если размер сетки больше, чем текущая сетка, новые ячейки будут иметь базовое значение 0, а если размер меньше, то значения, хранящиеся в ячейках, которые больше не входят в новый размер, будут потеряны. Все остальные ячейки останутся нетронутыми.

Синтаксис:

ds_grid_resize(index, w, h);

АргументТип Описание
index This index of the grid to resize.
w The new width of the grid.
h The new height of the grid.

Возвращается:

Пример:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

Приведенный выше код изменит размер сетки DS, указанной в глобальной переменной "Grid", а затем очистит ее так, чтобы каждая ячейка имела значение -1.
