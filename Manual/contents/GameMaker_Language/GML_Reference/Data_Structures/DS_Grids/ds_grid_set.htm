ds_grid_set

Эта функция может быть использована для установки заданной ячейки в заданной сетке DS на любое значение, которое может быть вещественным числом или строкой. Изображение ниже иллюстрирует это:

Синтаксис:

ds_grid_set(index, x, y, value);

АргументТип Описание
index This index of the grid.
x The x position of the cell to set.
y The y position of the cell to set.
value The value with which to set the cell.

Возвращается:

Пример:

grid = ds_grid_create(5, 5);
var i = 0;
var j = 0;

repeat (ds_grid_width(grid))
{
    repeat (ds_grid_height(grid))
    {
        ds_grid_set(grid, i, j, irandom(9));
        j += 1;
    }

    j = 0;
    i += 1;
}

Приведенный выше код создает сетку и сохраняет ее индекс в переменной "grid". Затем он заполняет эту сетку случайными целыми числами от 0 до 9.
