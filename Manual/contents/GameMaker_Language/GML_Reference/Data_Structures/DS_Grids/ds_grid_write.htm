ds_grid_write

Эта функция может быть использована для преобразования заданной ds_grid в строку, которая затем может быть сохранена во внешнем файле (например). Вы можете прочитать возвращаемую строку из этой функции обратно в ds_grid с помощью функции ds_grid_read().

ПРИМЕЧАНИЕ Возвращаемая строка не является человекочитаемой строкой, а скорее дампом содержимого структуры данных.

Синтаксис:

ds_grid_write(index);

АргументТип Описание
index The index of the grid to write.

Возвращается:

Пример:

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close()

Приведенный выше код откроет файл ini (создаст его, если он еще не существует), а затем запишет заданную ds_grid в виде строки в этот файл.
