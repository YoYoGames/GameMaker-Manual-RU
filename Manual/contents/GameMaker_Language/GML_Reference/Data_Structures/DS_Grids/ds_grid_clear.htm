ds_grid_clear

Эта функция может быть использована для очистки заданной сетки DS до определенного значения. После этого все ячейки сетки будут содержать это значение, которое может быть вещественным числом или строкой. На рисунке ниже показано, как это работает:

Синтаксис:

ds_grid_clear(index, val);

АргументТип Описание
index This index of the grid to clear.
val The new value for all grid cells.

Возвращается:

Пример:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

Приведенный выше код изменит размер сетки DS, указанной в глобальной переменной "Grid", а затем очистит ее так, чтобы каждая ячейка имела значение -1.
