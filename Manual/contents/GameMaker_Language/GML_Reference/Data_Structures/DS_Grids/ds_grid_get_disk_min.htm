ds_grid_get_disk_min

Эта функция может быть использована для нахождения минимального значения для всех ячеек, находящихся в пределах определенной области диска сетки, как показано на рисунке ниже:

Синтаксис:

ds_grid_get_disk_min(index, xm, ym, r);

АргументТип Описание
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.

Возвращается:

Real or String

Пример:

val = ds_grid_get_disk_min(grid, 5, 5, 2)

Приведенный выше код установит переменную "val" в минимальное значение, содержащееся в данном диске сетки DS, индексируемой в переменной "grid".
