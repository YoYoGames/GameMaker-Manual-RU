ds_grid_value_exists

С помощью этой функции можно проверить, присутствует ли определенное значение (вещественное или строковое) в прямоугольной области заданной сетки DS. Если оно присутствует, функция вернет true, в противном случае - false.

Синтаксис:

ds_grid_value_exists(index, x1, y1, x2, y2, val);

АргументТип Описание
index The index of the grid.
x1 The x position of the left of the region in the grid.
y1 The y position of the top of the region in the grid.
x2 The x position of the right of the region in the grid.
y2 The y position of the bottom of the region in the grid.
val The value to find.

Возвращается:

Пример:

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
{
    xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
    ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
}

Приведенный выше код проверяет сетку DS на наличие определенного значения в прямоугольной области. Если оно найдено, то сохраняет x и y позицию значения в двух переменных для последующего использования.
