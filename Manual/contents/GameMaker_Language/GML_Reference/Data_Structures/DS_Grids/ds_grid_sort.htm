ds_grid_sort

Эта функция может быть использована для сортировки сетки DS на основе значений из заданного столбца (подобно тому, как вы сортируете файлы по дате, размеру и т.д. в проводнике файлов ОС). На следующем изображении показан пример:

Синтаксис:

ds_grid_sort(index, column, ascending);

АргументТип Описание
index The index of the grid to sort.
column The column to use for sorting the rows
ascending Whether to sort lowest to highest (true), или от высшего к низшему (false).

Возвращается:

Пример:

ds_grid_sort(grid, 3, false)

Это позволит взять все значения в сетке DS, индексированные в переменной "grid", и отсортировать их в соответствии со значениями, находящимися в 3-м столбце сетки (как показано на изображении выше).
