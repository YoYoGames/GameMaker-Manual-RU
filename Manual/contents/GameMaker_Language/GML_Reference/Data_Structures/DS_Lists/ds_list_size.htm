ds_list_size

Эта функция возвращает "размер" списка, т.е. количество элементов, которые были в него добавлены.

Синтаксис:

ds_list_size(id);

АргументТип Описание
id The id of the data structure to check.

Возвращается:

Пример:

if !ds_list_empty(control_list)
{
    num = ds_list_size(control_list);
}

Приведенный выше код проверяет список DS на то, пуст он или нет, и если нет, то получает количество элементов, которые он содержит, и сохраняет это значение в переменной.
