ds_list_find_value

С помощью этой функции вы можете проверить заданную позицию списка и вернуть значение, хранящееся в списке для этой позиции. Обратите внимание, что если вы зададите позицию, которая находится за пределами заданного размера списка (например, позиция 11 в списке из 10 значений), то функция может вернуть undefined или 0. Это связано с тем, что при создании списка первые несколько записей в списке устанавливаются в 0, чтобы минимизировать проблемы с производительностью при первоначальном добавлении элементов в список (хотя функция ds_list_size() функция все равно вернет 0 для только что созданного списка). Если вы хотите убедиться, что список "действительно" пуст при создании, то вам следует вызвать функцию ds_list_clear() после создания списка, что будет означать, что все значения, возвращаемые для незаполненных слотов списка, будут undefined, которые затем можно последовательно проверить с помощью функции is_undefined() функцию.

Синтаксис:

ds_list_find_value(id, pos);

Аргумент Тип Описание
id Идентификатор списка DS The id of the list to use.
pos Реально The position to look at, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.

Возвращается:

Variable or undefined

Пример:

val = ds_list_find_value(list, ds_list_size(list) - 1);

Приведенный выше код проверяет список, проиндексированный в переменной "list", на последней позиции в списке и сохраняет возвращаемое значение в переменной "val".
