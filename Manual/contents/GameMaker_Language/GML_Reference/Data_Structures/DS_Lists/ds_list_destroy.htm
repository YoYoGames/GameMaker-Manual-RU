ds_list_destroy

Эта функция удаляет из памяти заданную структуру данных списка, освобождая используемые ею ресурсы и удаляя все содержащиеся в ней значения. Эту функцию всегда следует использовать, когда вы закончили использовать ds_list, чтобы предотвратить утечку памяти, которая может замедлить работу и привести к краху вашей игры. Обратите внимание, что при уничтожении списка все хранящиеся в нем структуры данных будут удалены, что приведет к утечке памяти, поэтому перед уничтожением списка вам нужно будет пройтись по списку и уничтожить все элементы структуры данных вручную, чтобы предотвратить это. Единственный случай, когда это не требуется, это когда вы отметили элементы в списке как другой список DS или как карту DS, в этом случае эти элементы будут уничтожены и их память также будет очищена автоматически.

ВАЖНО! Когда вы создаете структуру данных, значение индекса для ее идентификации является целочисленным значением, начинающимся с 0. Это означает, что структуры данных разных типов могут иметь одинаковое значение индекса, поэтому, если вы сомневаетесь, следует использовать функцию ds_exists() функцию перед обращением к ним. Также обратите внимание, что индексы используются повторно, поэтому значение индекса уничтоженной структуры данных может быть использовано вновь созданной, поэтому мы рекомендуем после уничтожения всегда устанавливать переменную, в которой хранился индекс DS, в -1.

Синтаксис:

ds_list_destroy(id);

АргументТип Описание
id The id of the data structure to remove.

Возвращается:

Пример:

if (lives == 0)
{
    ds_list_destroy(AI_list);
    AI_list = -1;
    room_goto(rm_Menu);
}

Приведенный выше код проверит значение встроенной глобальной переменной "lives" и, если оно равно 0, уничтожит список DS, проиндексированный в переменной "AI_list", а затем сменит комнаты.
