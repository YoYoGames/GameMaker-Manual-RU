ds_list_insert

Эта функция добавит заданное значение в список в заданной позиции. если список содержит больше значений после заданной позиции, их позиции будут сдвинуты на единицу вверх, чтобы освободить место, увеличивая список на единицу.

Синтаксис:

ds_list_insert(id, pos, val);

Аргумент Тип Описание
id Идентификатор списка DS The id of the list to add to.
pos Реально The position to add the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val Переменная The value to add to the list.

Возвращается:

N/A

Пример:

ds_list_insert(list, 9, score);

Приведенный выше код добавит значение, хранящееся в переменной "score", в 10-ю позицию списка, индексируемого переменной "list" (списки отсчитываются от 0, поэтому список из десяти значений нумеруется от 0 до 9).
