ds_list_create

Эта функция создает новую структуру данных списка и возвращает значение индекса. Это значение должно храниться в переменной и использоваться во всех последующих вызовах функций, связанных со списком. Обратите внимание, что если вы попытаетесь получить доступ к значению в списке до того, как он будет заполнен - т.е. список будет "пустым" - то возвращаемое значение может быть равно 0, поскольку при создании списка устанавливаются первые несколько записей, чтобы минимизировать проблемы производительности при первоначальном добавлении элементов в список (хотя функция ds_list_size() функция все равно вернет 0 для только что созданного списка). Если вы хотите убедиться, что список "действительно" пуст при создании, то вам следует вызвать функцию ds_list_clear() после создания списка, тогда все значения, возвращаемые для незаполненных слотов списка, будут undefined.

ВАЖНО! Когда вы создаете структуру данных, значение индекса для ее идентификации является целочисленным значением, начинающимся с 0. Это означает, что структуры данных разных типов могут иметь одинаковое значение индекса, поэтому, если вы сомневаетесь, следует использовать функцию ds_exists() функцию перед обращением к ним. Также обратите внимание, что индексы используются повторно, поэтому значение индекса уничтоженной структуры данных может быть использовано вновь созданной впоследствии.

Синтаксис:

ds_list_create();

Возвращается:

DS List ID

Пример:

list = ds_list_create();

Это создаст новый список и присвоит его индекс id переменной экземпляра "list".
