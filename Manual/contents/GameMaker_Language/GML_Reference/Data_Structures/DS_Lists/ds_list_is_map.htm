ds_list_is_map

С помощью этой функции можно проверить, хранится ли карта DS в заданной позиции в списке DS. Если заданная позиция содержит ID карты DS, то функция вернет true, в противном случае она вернет false.

Обратите внимание, что при этом будут обнаружены только те карты, которые были помечены вручную с помощью функции ds_list_mark_as_map() .

Синтаксис:

ds_list_is_map(id, pos);

АргументТип Описание
id The id of the list to check.
pos The position within the list to check.

Возвращается:

Пример:

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if (ds_list_is_map(ships, i))
    {
        ds_map_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

Приведенный выше код проходит по списку DS и проверяет, содержат ли какие-либо записи идентификаторы карт. Если да, то эти карты уничтожаются, а после завершения цикла уничтожается основной список.
