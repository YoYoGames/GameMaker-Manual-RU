ds_list_delete

С помощью этой функции можно удалить значение, хранящееся в определенной позиции в списке.

Синтаксис:

ds_list_delete(id, pos);

АргументТип Описание
id The id of the list to change.
pos Where in the list to delete the value.

Возвращается:

Пример:

if (ds_list_size(sc_list) > 10)
{
    while (ds_list_size(sc_list) > 10)
    {
        ds_list_delete(sc_list, 0);
    }
}

Приведенный выше код проверяет размер списка DS, и если он больше десяти, он проходит по списку, удаляя верхнее значение (позиция 0), пока в списке не останется только 10 записей.
