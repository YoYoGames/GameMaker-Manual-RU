ds_list_read

С помощью этой функции можно воссоздать сохраненный список DS (тот, который ранее был записан в виде строки с помощью функции ds_list_write()). Сначала необходимо создать новый список DS для чтения строки, и если список DS уже существует и в нем сохранена информация, то она будет очищена перед чтением. Эта функция имеет жизненно важное значение при создании механизмов сохранения/загрузки для вашей игры.

Обратите внимание, что если указанная строка DS была написана в среде GameMaker: Studio 1.2.x (или старше), необходимо указать дополнительный аргумент "legacy", установив его в значение true, поскольку после этого формат строки изменился.

Синтаксис:

ds_list_read(id, str [, legacy]);

Аргумент Тип Описание
id Идентификатор списка DS The id of the data structure to read into.
str Строка The string to read from.
legacy Булево OPTIONAL Может быть либо true, либо false, либо полностью опущен.

Возвращается:

N/A

Пример:

list = ds_list_create();
ini_open("save.ini");
var str = ini_read_string("Lists", "0", "");
if str != ""
{
    ds_list_read(list, str);
}
ini_close();

Приведенный выше код создает список и сохраняет индекс в переменной "list". Затем он открывает файл ini и считывает из него строку, проверяя, что сначала строка не возвращается пустой. Затем эта строка считывается во вновь созданный список ds_list.
