ds_list_set

Эта функция может использоваться для установки ранее добавленного элемента списка. Вы указываете ID списка (полученный при создании списка) и позицию в списке, которую нужно установить, а также значение, которое нужно установить. Обратите внимание, что если устанавливаемая запись находится вне границ списка (т.е. вы установили запись списка 20, но текущий список содержит только 10 записей), то список будет заполнен до заданной позиции, а каждая запись будет установлена в 0. Эта функция аналогична использованию аксессора DS list.

Синтаксис:

ds_list_set(id, pos, val);

Аргумент Тип Описание
id Идентификатор списка DS The id of the list to add to.
pos Переменная The position within the list to set.
[val2, ... max_val] Переменная The value to set in the list.

Возвращается:

N/A

Пример:

for (var i = 0; i < ds_list_size(list); i++;)
{
    ds_list_set(list, i, -1);
}

Приведенный выше код добавит значение, хранящееся в переменной "score", в список, проиндексированный в переменной "sc_list".
