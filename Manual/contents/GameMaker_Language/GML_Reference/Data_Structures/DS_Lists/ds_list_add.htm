ds_list_add

Эта функция может использоваться для добавления нового значения (вещественного или строкового) в список, который будет добавлен в конец. Функция может принимать дополнительные необязательные аргументы (столько, сколько вам нужно), что позволяет добавлять в список несколько значений последовательно за один вызов.

Синтаксис:

ds_list_add(id, val1 [, val2, ... max_val]);

Аргумент Тип Описание
id Идентификатор списка DS The id of the list to add to.
val Переменная The value to add to the list.
[val2, ... max_val] Переменная Optional values to be added to the list.

Возвращается:

N/A

Пример:

ds_list_add(sc_list, score);

Приведенный выше код добавит значение, хранящееся в переменной "score", в список, проиндексированный в переменной "sc_list".
