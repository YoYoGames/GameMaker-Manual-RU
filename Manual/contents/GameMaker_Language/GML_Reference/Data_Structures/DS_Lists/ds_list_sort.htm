ds_list_sort

С помощью этой функции можно отсортировать все значения в списке по возрастанию или убыванию. Если список содержит строки, они будут отсортированы в алфавитном порядке на основе английского алфавита из 26 букв.

Синтаксис:

ds_list_sort(id, ascend);

АргументТип Описание
id The id of the list to sort.
ascend Whether the values should be ascending (true) или по убыванию (false).

Возвращается:

Пример:

if newgame
{
    ds_list_sort(name_list, true);
}

Приведенный выше код отсортирует список, индексированный в переменной "name_list", если переменная "newgame" помечена как true.
