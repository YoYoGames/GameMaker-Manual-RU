ds_list_empty

С помощью этой функции вы можете проверить заданный список DS на предмет того, пуст он (возвращается true) или нет (возвращается false).

Синтаксис:

ds_list_empty(id);

Аргумент Тип Описание
id Идентификатор списка DS The id of the data structure to check.

Возвращается:

Пример:

if (count == 15( && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    count = 0;
}

Приведенный выше код проверяет переменную, достигла ли она определенного значения, и если достигла, то очищает список DS, индексированный в переменной "command_list", устанавливает сигнал тревоги и сбрасывает переменную в 0.
