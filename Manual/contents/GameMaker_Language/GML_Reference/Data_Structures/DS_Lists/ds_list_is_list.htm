ds_list_is_list

С помощью этой функции можно проверить, хранится ли другой список DS в заданной позиции внутри списка DS. Если заданная позиция содержит идентификатор списка DS, то функция возвращает true, в противном случае она возвращает false.

Обратите внимание, что при этом будут обнаружены только те списки, которые были помечены вручную с помощью функции ds_list_mark_as_list() .

Синтаксис:

ds_list_is_list(id, pos);

АргументТип Описание
id The id of the list to check.
pos The position within the list to check.

Возвращается:

Пример:

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if (ds_list_is_list(ships, i))
    {
        ds_list_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

Приведенный выше код проходит по списку DS и проверяет, не содержат ли какие-либо записи идентификаторы других списков. Если да, то эти списки уничтожаются, а после завершения цикла уничтожается основной список.
