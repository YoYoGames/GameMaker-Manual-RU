ds_list_write

Эта функция возвращает строку, которая затем может быть сохранена или передана в другую структуру данных с помощью функции ds_list_read() функцию.

ПРИМЕЧАНИЕ: Возвращаемая строка не является человекочитаемой строкой, а скорее дампом содержимого структуры данных.

Синтаксис:

ds_list_write(id);

АргументТип Описание
id The id of the data structure to write.

Возвращается:

Пример:

ini_open("save.ini");
var str = ds_list_write(list);
ini_write_string("Lists", "0", str);
ds_list_clear(list);
ini_close();

Приведенный выше код открывает ini-файл, а затем записывает строку, содержащую информацию, хранящуюся в списке DS, индексированном в переменной "list". Затем список очищается, а ini-файл закрывается.
