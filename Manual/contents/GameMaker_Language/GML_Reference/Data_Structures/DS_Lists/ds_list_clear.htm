ds_list_clear

С помощью этой функции вы можете очистить все данные из заданной структуры данных списка. Это НЕ уничтожает структуру данных (для этого вы должны использовать функцию ds_list_destroy()), а только стирает все данные из нее и делает список пустым (нулевого размера). Обратите внимание, что очистка списка приведет к тому, что все структуры данных, хранящиеся в нем, перестанут ссылаться, что приведет к утечке памяти, поэтому перед очисткой списка вам нужно будет пройтись по списку и уничтожить все элементы структуры данных вручную, чтобы предотвратить это. Единственный случай, когда это не требуется, это когда вы пометили элементы в списке как другой список DS или как карту DS, в этом случае эти элементы будут уничтожены (не очищены!) и их память очистится автоматически при очистке родителя.

Синтаксис:

ds_list_clear(id);

АргументТип Описание
id The id of the data structure to clear.

Возвращается:

Пример:

if (count == 15) && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    ai_count = 0;
}

Приведенный выше код проверяет переменную, достигла ли она определенного значения, и если достигла, то очищает список DS, индексированный в переменной "command_list", устанавливает сигнал тревоги и сбрасывает переменную в 0.
