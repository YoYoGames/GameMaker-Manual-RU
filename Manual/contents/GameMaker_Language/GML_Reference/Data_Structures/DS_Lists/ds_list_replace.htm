ds_list_replace

Эта функция заменит значение в заданной позиции на другое.

Синтаксис:

ds_list_replace(id, pos, val);

Аргумент Тип Описание
id Идентификатор списка DS The id of the list to change.
pos Реально The position to replace the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val Переменная The new value to replace the given value with.

Возвращается:

N/A

Пример:

ds_list_replace(n_list, 3, name);

Приведенный выше код заменит значение, хранящееся на позиции 3 в списке, на значение, хранящееся в переменной "name".
