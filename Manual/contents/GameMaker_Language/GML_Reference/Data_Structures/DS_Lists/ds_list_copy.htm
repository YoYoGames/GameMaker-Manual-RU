ds_list_copy

С помощью этой функции можно скопировать содержимое одного списка в другой. Оба списка должны быть созданы ранее, и если список, в который копируется информация, уже содержит ее, то этот список будет очищен первым. В результате вы получите два независимых списка, содержащих одинаковую информацию.

Синтаксис:

ds_list_copy( id, source );

АргументТип Описание
id The id of the list being copied to.
source The id of the list to be copied from.

Возвращается:

Пример:

if !ds_list_empty(main_list)
{
old_list = ds_list_create();
ds_list_copy(old_list, main_list);
ds_list_clear(main_list);
}

Приведенный выше код проверяет список DS на предмет его пустоты. Если он не пуст, он копируется в другой список DS (который был создан ранее), а затем исходный список очищается.
