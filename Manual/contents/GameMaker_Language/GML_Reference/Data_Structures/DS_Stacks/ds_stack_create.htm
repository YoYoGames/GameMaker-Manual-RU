ds_stack_create

Эта функция создает новую структуру данных стека и возвращает значение индекса. Это значение должно храниться в переменной и использоваться во всех дальнейших вызовах функций, связанных со стеком.

ВАЖНО! Когда вы создаете структуру данных, значение индекса для ее идентификации является целочисленным значением, начинающимся с 0. Это означает, что структуры данных разных типов могут иметь одинаковое значение индекса, поэтому, если вы сомневаетесь, следует использовать функцию ds_exists() функцию перед обращением к ним. Также обратите внимание, что индексы используются повторно, поэтому значение индекса уничтоженной структуры данных может быть использовано впоследствии вновь созданной.

Синтаксис:

ds_stack_create();

Возвращается:

DS Stack ID

Пример:

stack = ds_stack_create();

Это создаст новый стек и присвоит его индекс id переменной экземпляра "stack".
