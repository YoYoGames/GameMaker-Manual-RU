ds_stack_push

Эта функция заталкивает (добавляет) значение, которое может быть любого типа данных, на вершину стека. Функция может принимать еще 14 необязательных аргументов (всего 15 возможных добавлений), позволяя вам последовательно заталкивать несколько значений в стек за один вызов.

Синтаксис:

ds_stack_push(id, val [, val2, ... val15]);

АргументТип Описание
id The id of the data structure to push onto.
val The value to push onto the stack.
[val2, ... val13] Optional values to be added to the stack.

Возвращается:

Пример:

move_stack = ds_stack_create();
ds_stack_push(move_stack, x, y, x, y + 200, x + 200, y + 200, x +200, y);

Приведенный выше код создает новый стек DS и сохраняет его индекс в переменной "move_stack". Затем он помещает несколько значений в стек для дальнейшего использования.
