ds_stack_top

Эта функция считывает только первое значение стека (то, которое находится "сверху"). Она не вытаскивает значение, то есть оно может быть прочитано в будущем этой функцией или функцией ds_stack_pop(). Если стек пуст, то функция вернет константу undefined, в противном случае она вернет вещественное или строковое значение, содержащееся в стеке.

Синтаксис:

ds_stack_top(id);

АргументТип Описание
id The id of the data structure to read from.

Возвращается:

Variable (Data type value that is stored in the stack) or undefined

Пример:

num = ds_stack_top(control_stack);

Приведенный выше код будет считывать значение из стека, индексированного в переменной "control_stack", и сохранять возвращаемое значение в переменной "num".
