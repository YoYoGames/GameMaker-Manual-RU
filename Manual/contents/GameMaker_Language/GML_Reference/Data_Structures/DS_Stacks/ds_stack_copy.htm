ds_stack_copy

Эта функция может использоваться для копирования содержимого одного стека в другой. Обратите внимание, что это НЕ удаляет содержимое из исходного стека и не уничтожает его. При использовании этой функции стек, в который копируется содержимое, должен быть создан ранее, и если он содержал какие-либо элементы до копирования, то они будут очищены первыми (это означает, что информация будет потеряна).

Синтаксис:

ds_stack_copy(id, source);

Аргумент Тип Описание
id Идентификатор стека DS The id of the new stack.
source Идентификатор стека DS The original stack to copy from.

Возвращается:

N/A

Пример:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    stack = ds_stack_create();
    ds_stack_copy(stack, other.stack);
}

Приведенная выше функция создает новый экземпляр, а затем в этом экземпляре создает новый стек DS и копирует содержимое стека в экземпляре, выполняющем блок кода, во вновь созданный стек экземпляра.
