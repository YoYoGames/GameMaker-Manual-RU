ds_stack_clear

С помощью этой функции вы можете очистить все данные из заданной структуры данных стека. Это НЕ уничтожает структуру данных (для этого вы должны использовать ds_stack_destroy()), а только стирает все данные из нее и возвращает пустой стек.

Синтаксис:

ds_stack_clear(id);

АргументТип Описание
id The id of the data structure to clear.

Возвращается:

Пример:

if (ai_count = 15 && !ds_stack_empty(AI_stack))
{
    ds_stack_clear(AI_stack);
    alarm[0] = room_speed;
    ai_count = 0;
}

Приведенный выше код проверяет переменную, достигла ли она определенного значения, и если достигла, то очищает стек DS, указанный в переменной "AI_stack", устанавливает сигнал тревоги и сбрасывает переменную в 0.
