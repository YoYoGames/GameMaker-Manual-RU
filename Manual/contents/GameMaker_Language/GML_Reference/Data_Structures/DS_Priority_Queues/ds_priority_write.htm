ds_priority_write

Эта функция возвращает строку, которая затем может быть сохранена или передана в другую структуру данных с помощью функции ds_priority_read() функцию.

ПРИМЕЧАНИЕ: Возвращаемая строка не является человекочитаемой строкой, а скорее дампом содержимого структуры данных.

Синтаксис:

ds_priority_write(id);

АргументТип Описание
id The id of the data structure to check.

Возвращается:

Пример:

var str;
ini_open("save.ini");
str = ds_priority_write(p_queue);
ini_write_string("P_Queues", "0", str);
ds_priority_clear(p_queue);
ini_close();

Приведенный выше код открывает ini-файл, а затем записывает строку, содержащую информацию, хранящуюся в очереди приоритетов DS, проиндексированной в переменной "p_queue". Затем очередь приоритетов очищается, а ini-файл закрывается.
