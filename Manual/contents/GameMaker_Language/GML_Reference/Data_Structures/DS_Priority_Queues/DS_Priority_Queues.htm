Очереди приоритетов DS

Очередь приоритетов DS - это специальная структура данных, похожая на очередь приоритетов. DS queueтолько здесь очередь значений упорядочена по приоритетам (всегда вещественные числа), которые были присвоены значениям. Эта структура может быть очень полезна для построения таблиц лидеров или информационных списков, где важность каждой записи может быть взвешена.

ПРИМЕЧАНИЕ: Как и все динамические ресурсы, структуры данных занимают память, поэтому их всегда следует уничтожать, когда они больше не нужны, чтобы предотвратить утечку памяти, которая замедлит и в конечном итоге приведет к краху вашей игры.

ПРИМЕЧАНИЕ: Если вам нужно проверить, существует ли структура данных, вы можете использовать функцию ds_exists() функцию.

Существуют следующие функции, работающие с очередями приоритетов DS:

  • ds_priority_create
  • ds_priority_destroy
  • ds_priority_clear
  • ds_priority_empty
  • ds_priority_size
  • ds_priority_add
  • ds_priority_change_priority
  • ds_priority_delete_max
  • ds_priority_delete_min
  • ds_priority_delete_value
  • ds_priority_find_max
  • ds_priority_find_min
  • ds_priority_find_priority
  • ds_priority_copy
  • ds_priority_read
  • ds_priority_write
