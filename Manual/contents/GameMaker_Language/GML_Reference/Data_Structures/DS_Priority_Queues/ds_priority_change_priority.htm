ds_priority_change_priority

Эта функция принимает заданное значение и изменяет его приоритет в указанной очереди приоритетов. Заданное значение должно уже существовать в очереди приоритетов.

Синтаксис:

ds_priority_change_priority(id, val, priority);

Аргумент Тип Описание
id Идентификатор приоритета DS The id of the priority queue to change.
val Переменная The value to change the priority of.
priority Реально The new priority of the value.

Возвращается:

N/A

Пример:

if (global.Game_Time < 1000)
{
    ds_priority_change(ai_priority, AI_Search, 1);
}

Приведенный выше код проверяет глобальную переменную, и если она ниже определенного значения, то изменяет приоритет индекса функции сценария, находящегося в очереди приоритетов.
