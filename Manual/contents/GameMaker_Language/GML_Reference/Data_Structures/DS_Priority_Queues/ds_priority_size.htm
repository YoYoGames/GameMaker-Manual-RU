ds_priority_size

Эта функция возвращает "размер" очереди приоритетов, т.е. количество элементов, которые были в ней приоритезированы.

Синтаксис:

ds_priority_size(id);

АргументТип Описание
id The id of the data structure to check.

Возвращается:

Пример:

if !ds_priority_empty(control_priority)
{
    num = ds_priority_size(control_priority);
}

Приведенный выше код проверяет очередь приоритета DS на то, пуста она или нет, и если нет, то получает количество элементов, которые она содержит, и сохраняет это значение в переменной.
