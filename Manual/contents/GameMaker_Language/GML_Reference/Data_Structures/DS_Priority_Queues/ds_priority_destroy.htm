ds_priority_destroy

Эта функция удаляет из памяти заданную структуру данных очереди приоритетов, освобождая ресурсы, которые она использовала, и удаляя все содержащиеся в ней значения. Эту функцию всегда следует использовать, когда вы закончили использовать очередь приоритетов DS, чтобы предотвратить утечку памяти, которая может замедлить работу и привести к краху вашей игры.

ВАЖНО! Когда вы создаете структуру данных, значение индекса для ее идентификации является целочисленным значением, начинающимся с 0. Это означает, что структуры данных разных типов могут иметь одинаковое значение индекса, поэтому, если вы сомневаетесь, следует использовать функцию ds_exists() функцию перед обращением к ним. Также обратите внимание, что индексы используются повторно, поэтому значение индекса уничтоженной структуры данных может быть использовано вновь созданной, поэтому мы рекомендуем после уничтожения всегда устанавливать переменную, в которой хранился индекс DS, в -1.

Синтаксис:

ds_priority_destroy(id);

АргументТип Описание
id The id of the data structure to remove.

Возвращается:

Пример:

if (lives == 0)
{
    ds_priority_destroy(AI_queue);
    AI_queue = -1;
    room_goto(rm_Menu);
}

Приведенный выше код проверит значение встроенной глобальной переменной "lives", и если оно равно 0, то уничтожит приоритетную очередь DS, индексированную в переменной "AI_queue", а затем сменит комнаты.
