ds_priority_delete_value

Эта функция просто удалит заданное значение вместе с его приоритетом из индексированной очереди приоритетов.

Синтаксис:

ds_priority_delete_value(id,val);

АргументТип Описание
id The id of the priority queue to use.
val The value to delete from the priority queue.

Возвращается:

Пример:

if (ai_move == false)
{
    ds_priority_delete_value(ai_priority, AI_Move);
}

Приведенный выше код проверяет переменную экземпляра и, если она возвращает false, то удаляет индексированную функцию сценария из очереди приоритетов.
