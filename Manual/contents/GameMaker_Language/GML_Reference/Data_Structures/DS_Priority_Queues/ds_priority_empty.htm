ds_priority_empty

С помощью этой функции вы можете проверить заданную очередь приоритетов DS на предмет того, пуста она (возвращается true) или нет (возвращается false).

Синтаксис:

ds_priority_empty(id);

АргументТип Описание
id The id of the data structure to check.

Возвращается:

Пример:

if (count == 15) && (!ds_priority_empty(command_queue))
{
    ds_priority_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

Приведенный выше код проверяет переменную, достигла ли она определенного значения, и если достигла, то очищает очередь приоритетов DS, указанную в переменной "command_queue", устанавливает сигнал тревоги и сбрасывает переменную в 0.
