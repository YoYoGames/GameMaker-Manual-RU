ds_priority_find_priority

С помощью этой функции вы можете получить приоритет любого заданного значения. Если значение не существует в очереди приоритетов, то будет возвращено undefined.

Синтаксис:

ds_priority_find_priority(id, val);

АргументТип Описание
id The id of the priority queue to use.
val The value to find the priority of.

Возвращается:

Real or undefined

Пример:

p = ds_priority_find_priority(ai_priority, "intelligence");

Приведенный выше код будет хранить возвращенный приоритет для заданного значения в переменной экземпляра "p".
