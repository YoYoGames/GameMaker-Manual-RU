ds_priority_copy

Эта функция может быть использована для копирования содержимого одной очереди приоритетов в другую. Обратите внимание, что при этом НЕ удаляется содержимое исходной очереди приоритетов и не уничтожается исходная очередь приоритетов. При использовании этой функции очередь приоритетов, в которую производится копирование, должна быть создана ранее, и если она содержала какие-либо элементы до копирования, то они будут очищены первыми (то есть информация будет потеряна).

Синтаксис:

ds_priority_copy(id, source);

АргументТип Описание
id The id of the priority queue to copy to.
source The id of the priority queue to copy from.

Возвращается:

Пример:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    p_queue = ds_priority_create();
    ds_priority_copy(p_queue, other.p_queue);
}

Приведенная выше функция создает новый экземпляр, а затем в этом экземпляре создает новую очередь приоритетов DS и копирует содержимое очереди приоритетов в экземпляре, выполняющем блок кода, во вновь созданную очередь приоритетов экземпляра.
