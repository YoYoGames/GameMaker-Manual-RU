ds_queue_enqueue

Эта функция добавляет значение (вещественное или строковое) в хвост очереди DS. Функция может принимать еще 14 необязательных аргументов (всего 15 возможных добавлений), что позволяет добавлять несколько значений последовательно в хвост очереди за один вызов.

Синтаксис:

ds_queue_enqueue(id, val [, val2, ... val15]);

АргументТип Описание
id The id of the queue to add to.
val The value to add to the queue.
[val2, ... val15] Optional values to be added to the queue.

Возвращается:

Пример:

move_queue = ds_queue_create();
ds_queue_enqueue(move_queue, x + 200);
ds_queue_enqueue(move_queue, y);
ds_queue_enqueue(move_queue, x + 200);
ds_queue_enqueue(move_queue, y + 200);
ds_queue_enqueue(move_queue, x);
ds_queue_enqueue(move_queue, y + 200);
ds_queue_enqueue(move_queue, x);
ds_queue_enqueue(move_queue, y);

Приведенный выше код создает новую очередь DS и сохраняет ее индекс в переменной "move_queue". Затем он помещает несколько значений в очередь для дальнейшего использования.
