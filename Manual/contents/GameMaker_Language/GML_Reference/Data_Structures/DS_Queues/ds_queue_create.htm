ds_queue_create

Эта функция создает новую структуру данных очереди и возвращает значение индекса. Это значение должно храниться в переменной и использоваться во всех дальнейших вызовах функций, связанных с очередью.

ВАЖНО! Когда вы создаете структуру данных, значение индекса для ее идентификации является целым числом, начинающимся с 0. Это означает, что структуры данных разных типов могут иметь одинаковое значение индекса, поэтому, если вы сомневаетесь, следует использовать функцию ds_exists() функцию перед обращением к ним. Также обратите внимание, что индексы используются повторно, поэтому значение индекса уничтоженной структуры данных может быть использовано впоследствии вновь созданной.

Синтаксис:

ds_queue_create();

Возвращается:

DS Queue ID

Пример:

queue = ds_queue_create();

Это создаст новую очередь и присвоит ее индекс id переменной экземпляра "queue".
