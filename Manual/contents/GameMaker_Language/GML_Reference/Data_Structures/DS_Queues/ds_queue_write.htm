ds_queue_write

Эта функция возвращает строку, которая затем может быть сохранена или передана в другую структуру данных с помощью функции ds_queue_read() функцию.

ПРИМЕЧАНИЕ: Возвращаемая строка не является человекочитаемой строкой, а скорее дампом содержимого структуры данных.

Синтаксис:

ds_queue_write(id);

АргументТип Описание
id The id of the data structure to write.

Возвращается:

Пример:

ini_open("save.ini");
var str =ds_queue_write(queue);
ini_write_string("Queues", "0", str);
ds_queue_clear(queue);
ini_close();

Приведенный выше код открывает ini-файл, а затем записывает строку, содержащую информацию, хранящуюся в очереди DS, проиндексированной в переменной "queue". Затем очередь очищается, а ini-файл закрывается.
