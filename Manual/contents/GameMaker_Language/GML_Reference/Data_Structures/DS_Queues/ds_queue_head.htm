ds_queue_head

Эта функция считывает только первое значение очереди (то, которое находится "в голове"). Она не снимает очередь с этого значения, что означает, что оно все еще может быть прочитано в будущем этой функцией или функцией ds_queue_dequeue(). Если очередь пуста, то функция вернет константу undefined, в противном случае она вернет вещественное или строковое значение, содержащееся в очереди.

Синтаксис:

ds_queue_head(id);

АргументТип Описание
id The id of the data structure to read from.

Возвращается:

(Data type value stored in the queue)

Пример:

num = ds_queue_head(control_queue);

Приведенный выше код прочитает значение из очереди, проиндексированной в переменной "control_queue", и сохранит возвращаемое значение в переменной "num".
