ds_queue_dequeue

Эта функция удаляет головное значение из очереди DS, удаляет его из очереди и возвращает значение для хранения в переменной. Если очередь пуста, то функция возвращает константу undefined, в противном случае возвращается значение, содержащееся в очереди.

Синтаксис:

ds_queue_dequeue(id);

АргументТип Описание
id The id of the queue to dequeue from.

Возвращается:

Variable (Data type value stored in the queue) or undefined

Пример:

if !ds_queue_empty(move_queue)
{
    var xx = ds_queue_dequeue(move_queue);
    var yy = ds_queue_dequeue(move_queue);
    move_towards_point(xx, yy, 4);
}

Приведенный выше код проверяет очередь DS, указанную в переменной "move_queue", на то, пуста ли она, и если нет, то декеузирует два значения из головы очереди и использует их для задания направления движения.
