ds_queue_copy

Эта функция может быть использована для копирования содержимого одной очереди в другую. Обратите внимание, что при этом НЕ удаляется содержимое исходной очереди и не уничтожается исходная очередь. При использовании этой функции очередь, в которую копируется содержимое, должна быть создана ранее, и если она содержала какие-либо элементы до копирования, то они будут очищены первыми (то есть информация будет потеряна).

Синтаксис:

ds_queue_copy(id, source);

Аргумент Тип Описание
id Идентификатор очереди DS The id of the new queue.
source Идентификатор очереди DS The original queue to copy from.

Возвращается:

N/A

Пример:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    queue = ds_queue_create();
    ds_queue_copy(queue, other.queue);
}

Приведенная выше функция создает новый экземпляр, а затем в этом экземпляре создает новую очередь DS и копирует содержимое очереди в экземпляре, выполняющем блок кода, во вновь созданную очередь экземпляра.
