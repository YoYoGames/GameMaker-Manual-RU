ds_exists

С помощью этой функции вы можете проверить, существует ли структура данных заданного типа. Вы передаете значение "index" (хранящееся в переменной) и "тип" DS, который может быть любой из перечисленных ниже констант, и функция возвращает true, если структура данных существует, и false в противном случае.

Тип DS Постоянная
Постоянная Описание
ds_type_map Структура данных карты
ds_type_list Списочная структура данных
ds_type_stack Структура данных стека
ds_type_grid Структура данных сетки
ds_type_queue Структура данных очереди
ds_type_priority Структура данных о приоритетах

ПРИМЕЧАНИЕ Вы не можете использовать эту функцию для проверки типа структуры данных, так как один и тот же номер индекса может использоваться несколькими структурами данных разных типов.

Синтаксис:

ds_exists(ind, type);

Аргумент Тип Описание
ind Любой идентификатор DS The variable index to check for the data structure
type Тип DS Постоянная The type of data structure to check for (see the list of constants above)

Возвращается:

Boolean

Пример:

if !ds_exists(ai_grid, ds_type_grid)
{
    ai_grid = ds_grid_create(room_width / 32, room_height / 32);
}

Приведенный выше код проверяет (предварительно инициализированную) переменную "ai_grid" на предмет индексации структуры данных типа DS grid, и если таковой нет, то создает ее и сохраняет ее индекс в переменной.
