ds_map_is_map

С помощью этой функции вы можете проверить, хранится ли карта DS в заданном ключе map. Если заданный ключ содержит идентификатор карты DS, то функция вернет true, в противном случае она вернет false.

Обратите внимание, что это обнаружит только те карты, которые были добавлены с помощью функции ds_map_add_map() .

Синтаксис:

ds_map_is_map(id, key)

АргументТип Описание
id The id of the map to use.
key The key to replace.

Возвращается:

Пример:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++)
{
    if (ds_map_is_map(inventory, key))
    {
        ds_map_destroy(inventory[? key]);
    }
    key = ds_map_find_next(inventory);
}
ds_map_destroy(inventory);

Приведенный выше код выполняет цикл обработки карты DS и проверяет, не являются ли какие-либо ключи в ней ключами для других карт DS. Если да, то сохраненная DS-карта уничтожается, а в конце цикла уничтожается и основная DS-карта.
