ds_map_is_list

С помощью этой функции вы можете проверить, хранится ли список DS в заданном ключе карты. Если заданный ключ содержит идентификатор списка DS, то функция вернет true, в противном случае она вернет false.

Обратите внимание, что это обнаружит только списки, которые были добавлены с помощью функции ds_map_add_list() .

Синтаксис:

ds_map_is_list(id, key)

АргументТип Описание
id The id of the map to use.
key The key to check.

Возвращается:

Пример:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++)
{
    if (ds_map_is_list(inventory, key))
    {
        ds_list_destroy(inventory[? key]);
    }
    key = ds_map_find_next(inventory);
}
ds_map_destroy(inventory);

Приведенный выше код выполняет цикл по карте DS и проверяет, не являются ли какие-либо ключи в ней ключами для списка DS. Если да, то список DS уничтожается, а в конце цикла уничтожается и карта DS.
