ds_map_secure_save_buffer

Эта функция сохраняет ранее созданную карту DS в буфер. Вы передаете значение ID карты DS (возвращаемое функцией ds_map_create()) и идентификатор буфера для записи (возвращаемый функцией buffer_create()). Обратите внимание, что если сохраняемая DS-карта содержит массив, то при сохранении он будет преобразован в DS-список.

ВАЖНО! Защищенный сохраненный файл карты DS можно загрузить только на то устройство, которое его создало, и если вы попытаетесь загрузить файл, сохраненный на другом устройстве, то он не будет работать.

ПРИМЕЧАНИЕ: Эта функция не поддерживается на HTML5 и UWP.

Синтаксис:

ds_map_secure_save_buffer(filename);

=======

ds_map_secure_save_buffer

This function will save a previously created DS map to a buffer. You supply the DS map ID value (as returned by the function ds_map_create) and the buffer to write to (as returned by the function buffer_create). Note that if the DS map being saved contains an array, this will be converted into a DS list instead when saved.

 This function is not supported on HTML5.

Usage Notes

  • The secure saved DS map file can only be loaded on the device that created it, and if you try to load a file that was saved on a different device, then it will not work.
  • This function writes the DS map to the buffer at the current seek position (see buffer_tell) and increments it by the number of bytes written.

 

Syntax:

ds_map_secure_save_buffer(filename);
