ds_map_find_value

С помощью этой функции можно получить значение по заданному ключу. Входными значениями функции являются (ранее созданная) карта DS для использования и ключ для проверки.

ПРИМЕЧАНИЕ: Если такого ключа не существует, то функция вернет undefined. Вы всегда должны проверять это, используя is_undefined() функция

Синтаксис:

ds_map_find_value(id, key)

АргументТип Описание
id The id of the map to use.
key The key to find.

Возвращается:

Variable or undefined

Пример:

amount = ds_map_find_value(inventory, "food");

Или, используя аксессор карты "?":

amount = inventory[? "food"];

Приведенный выше код получит значение ключа "food" и сохранит его в переменной "amount".
