ds_map_clear

Эта функция очистит (ранее созданную) карту DS от всех пар ключ/значение, но не удалит саму карту DS из памяти. Для этого следует использовать функцию ds_map_destroy(). Обратите внимание, что при очистке карты все структуры данных, хранящиеся в ней, будут удалены, что приведет к утечке памяти, поэтому перед очисткой карты необходимо пройтись по ней и уничтожить все элементы структуры данных вручную, чтобы избежать этого. Единственный случай, когда это не требуется, это когда вы отметили какие-либо элементы в карте как список DS или как другую карту DS, в этом случае эти элементы будут уничтожены (не очищены!) и их память очистится автоматически при очистке родительской карты.

Синтаксис:

ds_map_clear(id);

АргументТип Описание
id The id of the data structure to clear.

Возвращается:

Пример:

if global.new_game
{
    ds_map_clear(inventory);
}

Приведенная выше функция проверит глобальную переменную на истинность и, если это так, то карта DS, индексируемая в переменной экземпляра "inventory", будет очищена от всех пар ключ/значение (но не уничтожена).
