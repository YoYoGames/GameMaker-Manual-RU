ds_map_find_previous

Эта функция возвращает предыдущий ключ, хранящийся в карте DS перед указанным в функции. Это может быть полезно, если вам приходится итерационно просматривать карту DS в поисках чего-либо, но по возможности этого следует избегать, так как это может быть медленно. Если такого ключа не существует, то функция вернет undefined. Вы всегда должны проверять это с помощью is_undefined() функция.

Синтаксис:

ds_map_find_previous(id, key);

АргументТип Описание
id The id of the map to use.
key The key to find the previous one to.

Возвращается:

Variable or undefined

Пример:

var size = ds_map_size(inventory) - 1;
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
{
    if (key != "gold")
    {
        key = ds_map_find_previous(inventory, key);
    }
    else break;
}

Приведенный выше код создает некоторые временные переменные, затем получает размер ds_map и находит последний ключ, сохраненный компьютером в карте. Затем он использует цикл for для обратного итерационного просмотра карты ds_map в поисках значения ключа "gold". Если оно найдено, то цикл завершается.
