ds_map_secure_load_buffer

Эта функция загружает сохраненную карту DS из буфера. Вы должны предварительно загрузить буфер в память (используя buffer_load()), а затем передать его в эту функцию, которая вернет карту DS, заполненную содержимым буфера. Обратите внимание, что буфер должен быть создан с помощью функции ds_map_secure_save_buffer(), чтобы функция работала правильно, а также обратите внимание, что если загружаемая DS-карта содержит массив, то при загрузке он будет преобразован в DS-список.

ВАЖНО! Одной из особенностей безопасного сохраненного файла является то, что он блокируется на устройстве, на котором был создан, поэтому вы не можете загрузить файл, сохраненный на одном устройстве, в проект, запущенный на другом устройстве.

ПРИМЕЧАНИЕ: Эта функция не поддерживается на HTML5 и UWP.

Синтаксис:

ds_map_secure_load_buffer(buffer);

Аргумент Тип Описание
buffer Идентификатор буфера The buffer ID of the buffer to load the map data from.

Возвращается:

DS Map ID

Пример:

var buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(buff);
buffer_delete(buff);

Приведенный выше код загрузит сохраненную карту DS из буфера и сохранит значение ее индекса в переменной для дальнейшего использования.

======= buffer Buffer The buffer to load the map data from.

 

Returns:

DS Map or -1 in case the data couldn't be read correctly from the buffer

 

Example:

var _buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(_buff);
buffer_delete(_buff);

The above code will load a securely saved DS map from a buffer and store its index value in a variable for future use.
