ds_map_find_first

Эта функция возвращает первый ключ, хранящийся в карте DS. Это не первый ключ в том порядке, в котором вы их добавили! Карты DS не хранятся в линейной форме, для этого используется список DS, поэтому все, что делает эта функция, это находит первый ключ, сохраненный компьютером. Это может быть полезно, если вам нужно итеративно просматривать карту DS в поисках чего-либо, но по возможности этого следует избегать, так как это может быть медленным.

Обратите внимание, что эта функция вернет undefined, если заданная карта DS пуста.

Синтаксис:

ds_map_find_first(id);

АргументТип Описание
id The id of the map to use.

Возвращается:

Variable or undefined

Пример:

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
    if (key != "gold")
    {
        key = ds_map_find_next(inventory, key);
    }
    else break;
}

Приведенный выше код создает некоторые временные переменные, затем получает размер карты DS и находит первый ключ, сохраненный компьютером в карте. Затем с помощью цикла for выполняется итерация по карте DS в поисках значения ключа "gold". Если оно найдено, то цикл завершается.
