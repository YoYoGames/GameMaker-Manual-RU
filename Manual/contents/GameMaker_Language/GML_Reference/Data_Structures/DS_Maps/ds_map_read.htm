ds_map_read

Эта функция принимает строку, которая ранее была создана функцией ds_map_write() и затем считывает ее в ранее созданную карту DS. Если карта, в которую считывается строка, содержит какие-либо пары ключ/значение, они будут сначала очищены перед повторным созданием сохраненной карты.

Обратите внимание, что если указанная строка DS была написана в среде GameMaker: Studio 1.2.x (или старше), необходимо указать дополнительный аргумент "legacy", установив его в значение true, поскольку после этого формат строки изменился.

Синтаксис:

ds_map_read(id, str, [legacy]);

Аргумент Тип Описание
id Идентификатор карты DS The id of the data structure to read the string into
str Строка The string to read
legacy Булево OPTIONAL Может быть либо true, либо false, либо полностью опущен.

Возвращается:

N/A

Пример:

inventory = ds_map_create();
ini_open("map.ini");
var t_string = ini_read_string("Saved", "0", "");
if (t_string != "")
{
    ds_map_read(inventory, t_string);
}
ini_close();

Приведенный выше код создает новую карту DS и сохраняет ее индекс id в переменной "inventory". Затем он открывает ini-файл и считывает строку из этого файла во временную переменную "t_string". Наконец, он проверяет правильность строки (не значение по умолчанию в ini - "") и, если это так, считывает строку во вновь созданную DS-карту, после чего снова закрывает ini.
