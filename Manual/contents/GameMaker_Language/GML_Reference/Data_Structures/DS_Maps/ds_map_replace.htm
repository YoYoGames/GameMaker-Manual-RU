ds_map_replace

С помощью этой функции вы можете изменить значение для заданного ключа в карте DS. Вы передаете индекс карты (возвращаемый функцией ds_map_create()) , затем ключ для замены - либо строку, либо целое число - и значение, на которое нужно заменить значение ключа. Если заданный ключ не существует, то он будет создан, а если существует, то текущее значение будет заменено на новое значением. Функция вернет true, если ключ существует и значение заменено, и false, если ключ не существует и был создан новый ключ с заданным значением.

Синтаксис:

ds_map_replace( id, key, val );

АргументТип Описание
id The id of the map to change.
key The key with the value that should be replaced by the new one
val The new value to replace the given value with

Возвращается:

Пример:

ds_map_replace(inventory, "torso", 55);

Приведенный выше код ищет в карте DS ключ "torso", и когда он его находит (или он создается, если его нет), текущее значение заменяется на указанное.
