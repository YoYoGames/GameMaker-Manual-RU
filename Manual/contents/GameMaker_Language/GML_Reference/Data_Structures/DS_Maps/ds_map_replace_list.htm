ds_map_replace_list

С помощью этой функции вы можете заменить список DS, который был сохранен в заданном "ключе", на другой список, созданный ранее. Эта функция предназначена для создания карт, совместимых с JSON, которые затем кодируются с помощью функции json_encode() и должна использоваться только в сочетании с этой функцией.

Синтаксис:

ds_map_replace_list(id, key, value)

Аргумент Тип Описание
id Идентификатор карты DS The id of the map to use.
key Строка The key to replace.
value Идентификатор списка DS The id of the ds_list to use to replace the one previously stored in the given key.

Возвращается:

N/A

Пример:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
ds_map_replace_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_list_destroy(j_list);

Приведенный выше код создаст DS List и заполнит его значениями различных глобальных переменных, а затем заменит ранее сохраненный список в DS Map "j_map".
