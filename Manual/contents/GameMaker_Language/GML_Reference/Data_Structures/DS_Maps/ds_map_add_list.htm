ds_map_add_list

С помощью этой функции вы можете назначить (ранее созданный) список DS ключу в заданной карте DS. Эта функция предназначена для создания карт, совместимых с JSON, которые затем кодируются с помощью функции json_encode() и должна использоваться только в сочетании с этой функцией. Если DS-карта содержит список, добавленный таким образом, уничтожение родительской карты также уничтожит содержащиеся в ней списки и освободит их память, а вызов функции ds_map_clear() на родительской карте также уничтожит и очистит все помеченные списки.

Синтаксис:

ds_map_add_list(id, key, value)

Аргумент Тип Описание
id Идентификатор карты DS The id of the map to use.
key Строка The key for the added list.
value Идентификатор списка DS The id of the list to add.

Возвращается:

N/A

Пример:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
var j_map = ds_map_create();
ds_map_add_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_map_destroy(j_map);

Приведенный выше код создает список и заполняет его различными значениями глобальных переменных. Этот список затем "вложен" в карту DS, а карта затем кодируется в строку JSON, после чего карта уничтожается, удаляя ее и все содержащиеся в ней списки из памяти.
