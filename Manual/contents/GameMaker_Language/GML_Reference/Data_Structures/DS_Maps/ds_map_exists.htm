ds_map_exists

Эта функция вернет true, если указанный ключ существует в (ранее созданной) карте DS, и false, если нет.

Синтаксис:

ds_map_exists(id, key);

Аргумент Тип Описание
id Идентификатор карты DS The id of the data structure to check
key Строка The key to check for

Возвращается:

Boolean

Пример:

if !ds_map_exists(inventory, "potions")
{
    ds_map_add(inventory, "potions", 1);
}

Приведенный выше код проверит карту DS, проиндексированную в переменной "inventory", на наличие ключа "potions" и, если его нет, добавит его на карту.
