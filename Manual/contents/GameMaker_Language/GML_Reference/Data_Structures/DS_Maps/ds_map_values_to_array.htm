ds_map_values_to_array

С помощью этой функции вы можете получить все значения, которые содержит карта DS для каждого ключа в карте. Вы указываете ID карты DS, из которой нужно получить ключи (как возвращается функцией ds_map_create()), и функция вернет массив, где каждая запись в массиве - это значение из карты DS (например, если карта имеет 10 ключей, то будет возвращен массив из 10 записей, каждая из которых соответствует значению ключа). Функция имеет необязательный второй аргумент, в котором вы можете указать созданный вами массив, в этом случае данные о значении ключа карты будут добавлены ко всем существующим данным в массиве. Обратите внимание, что функция изменяет непосредственно массив, но также возвращает ссылку на него (или ссылку на новый массив, если он не указан).

Синтаксис:

ds_map_values_to_array(id, [array])

Аргумент Тип Описание
id Идентификатор карты DS The id of the map to use.
[array] Массив OPTIONAL Массив для добавления значений карты DS.

Возвращается:

Array

Пример:

var _values = ds_map_values_to_array(lvl_score);
var _total = 0;
var _length = array_length(_values);

for (var i = 0; i < _length; i ++)
{
    _total += _values[i];
}

draw_text(32, 32, "Total score for all levels: " + string(_total));

Приведенная выше команда извлекает все значения из карты DS для каждого ключа в карте и сохраняет их в локальной переменной в виде массива. Затем выполняется циклический просмотр массива и добавление каждого значения в локальную переменную, которая затем используется для отображения общей суммы всех значений.
