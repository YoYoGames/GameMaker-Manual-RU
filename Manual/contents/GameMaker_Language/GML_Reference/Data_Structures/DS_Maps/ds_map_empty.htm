ds_map_empty

Эта функция просто вернет false, если указанная (ранее созданная) карта DS содержит какие-либо пары ключ/значение, или true, если нет.

Синтаксис:

ds_map_empty(id);

АргументТип Описание
id The id of the data structure to check.

Возвращается:

Пример:

if ds_map_empty(inventory)
{
    weight = 0;
}

Приведенный выше код проверяет, есть ли в карте DS, индексируемой в переменной "inventory", пары ключ/значение, и если нет, то устанавливает переменную "weight" в 0.
