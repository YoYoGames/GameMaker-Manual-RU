ds_map_add

Эта функция должна использоваться для добавления наборов пар ключ/значение в указанную карту DS. Вы можете проверить эту функцию, чтобы узнать, была ли она успешной или нет (она вернет true в случае успеха или false в противном случае), так как она может потерпеть неудачу, если в карте DS уже существует такой же ключ или вы указали несуществующую карту DS в качестве ID карты для добавления. Ключи и значения, которые вы предоставляете, могут состоять из любой комбинации типов данных, так что все нижеперечисленные - и больше - допустимы (хотя на практике чаще всего для ключа используется строка):

ds_map_add(map, 5, undefined);
ds_map_add(map, "level", 1);
ds_map_add(map, 89.6, "hello world");
ds_map_add(map, 5, buffer_get_address(buff));

Вы также можете добавить на карту, используя аксессор "?", как показано ниже:

map[? 5] = undefined;
map[? "level"] = 1;
map[? 89.6] = "hello world";
map[? 5] = buffer_get_address(buff);

ПРИМЕЧАНИЕ: В отличие от других структур данных в GameMaker, этот ключ не будет находиться в начале (или в конце) карты DS, скорее он просто будет находиться где-то в карте DS.

Синтаксис:

ds_map_add(id, key, val);

АргументТип Описание
id The id of the map to add to.
key The key of the value to add.
val The value to add to the map.

Возвращается:

Пример:

inventory = ds_map_create();
ds_map_add(inventory, "hp potion", 1);
ds_map_add(inventory, "gold", 100);

Это создаст новую карту, присвоив ее id переменной "inventory". Затем она добавит два новых ключа к карте, "hp potion" и "gold", и установит их начальные значения 1 и 100.
