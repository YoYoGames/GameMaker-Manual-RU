ds_map_keys_to_array

С помощью этой функции вы можете получить все ключи, которые содержит карта DS. Вы указываете ID карты DS, из которой нужно получить ключи (как возвращается функцией ds_map_create()), и функция вернет массив, каждая запись в котором будет ключом из карты DS. Функция имеет необязательный второй аргумент, в котором вы можете указать созданный вами массив, в этом случае данные ключей карты будут добавлены к существующим данным в массиве. Обратите внимание, что функция изменяет непосредственно массив, но также возвращает ссылку на него (или ссылку на новый массив, если он не указан).

Синтаксис:

ds_map_keys_to_array(id, [array])

Аргумент Тип Описание
id Идентификатор карты DS The id of the map to use.
[array] Массив OPTIONAL Массив для добавления ключей карты DS.

Возвращается:

Array

Пример:

map_keys = ds_map_keys_to_array(inventory);

Приведенный выше код извлекает ключи для карты DS, а затем сохраняет их в виде массива для дальнейшего использования.
