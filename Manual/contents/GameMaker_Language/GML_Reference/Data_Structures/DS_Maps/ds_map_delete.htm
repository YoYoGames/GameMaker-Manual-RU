ds_map_delete

С помощью этой функции вы можете удалить любой заданный ключ (и соответствующее ему значение) из заданной, ранее созданной, карты DS.

Синтаксис:

ds_map_delete(id, key);

Аргумент Тип Описание
id Идентификатор карты DS The id of the map to change.
key Строка The key to delete (along with its associated value).

Возвращается:

N/A

Пример:

ds_map_delete(inventory, "shield");

Приведенный выше код удалит ключ "shield" (и значение, с которым он сопряжен) из карты DS (инвентаря).
