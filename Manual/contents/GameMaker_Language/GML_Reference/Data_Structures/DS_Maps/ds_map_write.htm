ds_map_write

Эта функция преобразует данные карты DS указанной карты в строковый формат, который затем может быть записан в файл *.ini или *.txt для удобства хранения. Эта строка может быть позже считана обратно в новую ds_map с помощью функции ds_map_read().

ПРИМЕЧАНИЕ: Возвращаемая строка не является человекочитаемой строкой, а скорее дампом содержимого структуры данных.

Синтаксис:

ds_map_write(id);

АргументТип Описание
id The id of the data structure to use

Возвращается:

Пример:

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

Приведенный выше код открывает ini-файл, готовый к записи. Затем он использует ds_map_write() для генерации строки, которая сохраняется во временной переменной "t_string". Наконец, он записывает эту строку в ini файл, после чего закрывает его.
