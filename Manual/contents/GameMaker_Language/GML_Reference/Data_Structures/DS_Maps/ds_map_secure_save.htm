ds_map_secure_save

Эта функция сохраняет содержимое заданной карты DS в файл, который привязан к устройству, на котором он был создан (это означает, что он не может быть прочитан при передаче на любое другое устройство). Сам файл может иметь практически любое расширение (например, *.dat, *.json, *.bin и т.д.) и будет обфусцирован и сохранен в локальном хранилище на целевой платформе. Затем вы можете повторно загрузить ds_map с помощью функции ds_map_secure_load(). Обратите внимание, что если сохраняемая карта DS содержит массив, то при сохранении этот массив будет преобразован в список DS.

ВАЖНО! Одной из особенностей безопасного сохраненного файла является то, что он блокируется на устройстве, на котором был создан, поэтому вы не можете загрузить файл, сохраненный на одном устройстве, в проект, запущенный на другом устройстве.

Синтаксис:

ds_map_secure_save(map, filename);

АргументТип Описание
map The id of the data structure to use
filename The name of the file to save the map to

Возвращается:

Пример:

ds_map_secure_save(purchase_map, "p_data.dat");

Приведенный выше код сохранит карту DS, индексированную в переменной "p_data", в заданный файл для последующего извлечения.
