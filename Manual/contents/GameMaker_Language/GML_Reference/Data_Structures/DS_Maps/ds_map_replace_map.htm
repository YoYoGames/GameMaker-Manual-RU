ds_map_replace_map

С помощью этой функции вы можете заменить карту DS Map, которая была сохранена в заданном "ключе", на другую карту, которая была создана ранее. Эта функция предназначена для создания карт, совместимых с JSON, которые затем кодируются с помощью функции json_encode() и должна использоваться только в сочетании с этой функцией.

Синтаксис:

ds_map_replace_map(id, key, value)

Аргумент Тип Описание
id Идентификатор карты DS The id of the ds_map to use.
key Строка The key to replace.
value Идентификатор карты DS The id of the ds_map to use to replace the one previously stored in the given key.

Возвращается:

N/A

Пример:

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array);
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

Приведенный выше код создаст DS Map и заполнит его массивом и DS List, а затем заменит ранее сохраненную карту в DS Map "j_map".
