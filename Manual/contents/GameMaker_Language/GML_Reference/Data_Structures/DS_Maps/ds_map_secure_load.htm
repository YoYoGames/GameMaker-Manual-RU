ds_map_secure_load

Эта функция загружает ранее сохраненную карту DS из указанного файла. Этот файл должен быть предварительно создан с помощью функции ds_map_secure_save() и после загрузки функция вернет индекс карты DS, которая была создана на основе загруженных данных. Этот индекс карты DS должен быть сохранен в переменной и использоваться для всех последующих вызовов функции к этой карте. Обратите внимание, что если загружаемая карта DS была сохранена с массивом в качестве одного из ключевых значений, то при загрузке этот массив будет преобразован в список DS.

ВАЖНО! Одной из особенностей безопасного сохраненного файла является то, что он блокируется на устройстве, на котором был создан, поэтому вы не можете загрузить файл, сохраненный на одном устройстве, в проект, запущенный на другом устройстве.

Синтаксис:

ds_map_secure_load(filename);

АргументТип Описание
filename The name of the file to load the map data from.

Возвращается:

DS Map ID

Пример:

p_map = ds_map_secure_load("p_data.dat");

Приведенный выше код загрузит сохраненную карту DS и сохранит значение ее индекса в переменной для дальнейшего использования.
