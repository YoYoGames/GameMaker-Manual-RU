расстояние между жестом и двойным касанием

Эта функция используется для установки расстояния, в пределах которого необходимо повторно коснуться/нажать на экран после одного касания, чтобы сработал жест двойного касания. Расстояние измеряется в дюймах и по умолчанию имеет значение 0,1.

Синтаксис:

gesture_double_tap_distance(distance);

АргументТип Описание
distance The distance (in inches) to set for double tap gesture event detection.

Возвращается:

Пример:

if gesture_get_double_tap_distance() != 0.1
{
    gesture_get_double_tap_distance(0.1);
}

Приведенный выше код проверяет, установлено ли расстояние двойного касания для жестов в 0,1 дюйма, и если нет, то устанавливает это значение.
