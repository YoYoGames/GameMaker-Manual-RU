gesture_get_tap_count

Эта функция используется для проверки того, включен или отключен подсчет касаний. Функция возвращает true, если она включена, и false в противном случае, и вы можете включить или выключить отсчет касаний с помощью функции gesture_tap_count(). Если функция включена, каждое событие переключения будет иметь дополнительный элемент карты DS "tapcount", в котором будет зарегистрировано количество переключений, и если функция включена, это означает, что все события переключения будут запущены, т.е.: два переключения вызовут как событие одинарного переключения, так и событие двойного переключения, при этом количество переключений для события одинарного переключения будет равно 1, а для события двойного переключения - 2. Значение счета переключений будет сброшено на 0 по истечении времени, установленного для обнаружения двойного переключения (см. функцию gesture_double_tap_time()). Если счетчик касаний отключен, то первое касание не будет зарегистрировано, пока не пройдет время двойного касания и не будет обнаружено второе касание. Обратите внимание, что по умолчанию эта функция включена.

Синтаксис:

gesture_get_tap_count();

Возвращается:

Пример:

if !gesture_get_tap_count()
{
    getsure_tap_count(true);
}

Приведенный выше код проверяет, включен ли подсчет касаний, и если нет, то он включается.
