Ввод с помощью жестов

Функции ввода устройства хорошо подходят для работы с простыми касаниями экрана или ориентацией устройства, но иногда вам нужно, чтобы ваши проекты определяли более сложные вещи, например, возможность перетаскивать или "листать" экземпляр по комнате. Это можно сделать с помощью функций Device, но для облегчения работы GameMaker имеет полный набор функций Gesture Events для обнаружения двойных касаний, перетаскивания, щелчков, щипков и поворотов, как на уровне экземпляра, так и на глобальном уровне, а также для возврата информации о событии. Все перечисленные здесь функции предназначены для обеспечения большего контроля над тем, как и когда эти события будут вызываться.

ВАЖНО Эти функции не будут работать на целевой платформе HTML5. Они не выдадут ошибку при использовании, а просто будут проигнорированы.

Доступны следующие различные функции жестов:

  • gesture_get_drag_time
  • gesture_get_drag_distance
  • gesture_get_flick_speed
  • gesture_get_double_tap_time
  • gesture_get_double_tap_distance
  • gesture_get_tap_count
  • gesture_get_pinch_distance
  • gesture_get_pinch_angle_towards
  • gesture_get_pinch_angle_away
  • gesture_get_rotate_time
  • gesture_get_rotate_angle
  • gesture_drag_time
  • gesture_drag_distance
  • gesture_flick_speed
  • gesture_double_tap_time
  • gesture_double_tap_distance
  • gesture_tap_count
  • gesture_pinch_distance
  • gesture_pinch_angle_towards
  • gesture_pinch_angle_away
  • gesture_rotate_time
  • gesture_rotate_angle
