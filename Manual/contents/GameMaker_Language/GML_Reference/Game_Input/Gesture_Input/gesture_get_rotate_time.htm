gesture_get_rotate_time

Эта функция используется для получения времени, в течение которого пара касаний должна вращаться в одном направлении, чтобы сработал жест Rotate Start. Время измеряется в секундах и имеет значение по умолчанию 0,16 с.

ВАЖНО! В настоящее время для игры с частотой 60 кадров в секунду вы можете установить максимальное значение в одну секунду, иначе вращение не будет обнаружено. При более низкой частоте кадров это значение увеличится, например, для игры с частотой 30 кадров в секунду максимальное время может составлять 2 секунды.

Синтаксис:

gesture_get_rotate_time();

Возвращается:

(inches)

Пример:

if gesture_get_rotate_time() != 0.1
{
    gesture_rotate_time(0.1);
}

Приведенный выше код проверяет, установлено ли время поворота для жестов на 0,1 секунды, и если нет, то устанавливает это значение.
