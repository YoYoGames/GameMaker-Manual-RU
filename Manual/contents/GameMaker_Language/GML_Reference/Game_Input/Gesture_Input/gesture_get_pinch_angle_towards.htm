gesture_get_pinch_angle_towards

Эта функция используется для получения угла, в пределах которого прикосновение должно двигаться к другому прикосновению, прежде чем потенциально может быть запущен жест Pinch In. Угол измеряется в градусах и имеет значение по умолчанию 45°.

Синтаксис:

gesture_get_pinch_angle_towards();

Возвращается:

(inches)

Пример:

if gesture_get_pinch_angle_towards() != 45
{
    gesture_pinch_angle_towards(45);
}

Приведенный выше код проверяет, установлен ли угол зажатия для жестов в 45°, и если нет, то устанавливает это значение.
