жест_прижатия_угла_удаления

Эта функция используется для установки угла, в пределах которого прикосновение должно удаляться от другого прикосновения, прежде чем может быть запущен жест Pinch Out. Угол измеряется в градусах и имеет значение по умолчанию 45°.

Синтаксис:

gesture_pinch_angle_away();

Возвращается:

(inches)

Пример:

if gesture_get_pinch_angle_away() != 45
{
    gesture_pinch_angle_away(45);
}

Приведенный выше код проверяет, установлен ли угол отведения для жестов в 45°, и если нет, то устанавливает это значение.
