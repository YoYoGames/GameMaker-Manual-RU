жест_двойной_тап_время

Эта функция используется для установки времени, которое должно пройти между двумя касаниями/щелчками, чтобы вызвать событие жеста двойного касания. Время измеряется в секундах и по умолчанию имеет значение 0,16.

Синтаксис:

gesture_double_tap_time(time);

Аргумент Тип Описание
time Реально The time (in seconds) to set for double tap gesture event detection.

Возвращается:

N/A

Пример:

if gesture_get_double_tap_time() != 0.16
{
    gesture_get_double_tap_time(0.16);
}

Приведенный выше код проверяет, установлено ли время двойного нажатия для жестов на 0,16 секунды, и если нет, то устанавливает это значение.
