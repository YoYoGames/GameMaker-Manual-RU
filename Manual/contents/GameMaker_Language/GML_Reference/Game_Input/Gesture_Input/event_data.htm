данные события

Эта переменная является глобальной и используется для хранения карты DS, когда она используется в соответствующем событии (например, события жестов и широковещательные сообщения), и -1 во всех остальных случаях. Фактическое содержимое карты DS зависит от типа события, которое ее вызвало, поэтому обратитесь к отдельным разделам, посвященным этим событиям.

Синтаксис:

event_data;

Возвращается:

DS Map ID

Пример:

isFlick = event_data[?"isflick"];

if (isFlick)
{
    flickVelX = event_data[?"diffX"];
    flickVelY = event_data[?"diffY"];
}
else
{
    flickVelX = 0;
    flickVelY = 0;
}

Приведенный выше код берется из события жеста Drag End и проверяет, является ли это событие "flick", и если да, то извлекает необходимые данные из карты event_data DS и использует их для установки некоторых переменных. Если событие "flick" не обнаружено, то те же переменные устанавливаются в 0.
