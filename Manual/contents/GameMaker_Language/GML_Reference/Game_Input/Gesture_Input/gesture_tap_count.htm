счетчик жестов

Эта функция используется для установки того, включен или отключен счетчик касаний. При установке значения true отсчет включается, а при установке false - отключается, хотя вы можете проверить, в каком состоянии он находится, используя функцию gesture_get_tap_count(). Если функция включена, каждое событие переключения будет иметь дополнительный элемент карты DS "tapcount", в котором будет зарегистрировано количество переключений, а если включена, это означает, что все события переключения будут запущены, т.е.: два переключения вызовут и событие одинарного переключения, и событие двойного переключения, при этом количество переключений для события одинарного переключения будет равно 1, а для события двойного переключения - 2. Значение счета переключений будет сброшено на 0 по истечении времени, установленного для обнаружения двойного переключения (см. функцию gesture_double_tap_time()). Если счетчик касаний отключен, то первое касание не будет зарегистрировано, пока не пройдет время двойного касания и не будет обнаружено второе касание. Обратите внимание, что по умолчанию эта функция включена.

Синтаксис:

gesture_tap_count(enable);

АргументТип Описание
enable Enable (set to true) или отключить (установить на false) подсчет касаний.

Возвращается:

Пример:

if !gesture_get_tap_count()
{
    getsure_tap_count(true);
}

Приведенный выше код проверяет, включен ли подсчет касаний, и если нет, то он включается.
