gesture_get_double_tap_time

Эта функция используется для получения времени, которое требуется между двумя касаниями/щелчками для запуска события жеста двойного касания. Время измеряется в секундах и имеет значение по умолчанию 0,10.

Синтаксис:

gesture_get_double_tap_time();

Возвращается:

Real (seconds)

Пример:

if gesture_get_double_tap_time() != 0.16
{
    gesture_double_tap_time(0.16);
}

Приведенный выше код проверяет, установлено ли время двойного касания для жестов на 0,16 секунды, и если нет, то устанавливает это значение.
