keyboard_check

С помощью этой функции можно проверить, удерживается ли клавиша нажатой или нет. В отличие от keyboard_check_pressed() или keyboard_check_released() которая срабатывает только один раз при нажатии или отпускании клавиши, эта функция срабатывает на каждом шаге, в течение которого клавиша удерживается нажатой. Функция принимает значение кода клавиши, возвращаемое функцией ord() (только заглавные буквы от A-Z или цифры от 0-9), или любую из констант vk_*, перечисленных на главной странице ввода с клавиатуры.

Синтаксис:

keyboard_check(key);

Аргумент Тип Описание
key Константа виртуального ключа (vk_*) The key to check the down state of.

Возвращается:

Boolean

Пример:

if keyboard_check(vk_left)
{
    x -= 5;
}

Приведенный выше код будет проверять, нажата ли клавиша со стрелкой, и перемещать экземпляр на 5 пикселей влево каждый шаг, когда возвращается true.
