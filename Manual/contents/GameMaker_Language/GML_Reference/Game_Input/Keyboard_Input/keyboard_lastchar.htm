keyboard_lastchar

В этой переменной хранится строка последней нажатой клавиши. Эта переменная не является только для чтения, и вы можете изменить ее, например, установить в "" (пустую строку), если вы уже работали с ней.

Синтаксис:

keyboard_lastchar;

Возвращается:

Пример:

if keyboard_lastkey != -1
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

Приведенный выше код проверяет, не превышает ли переменная lastkey значение -1, и если да, то добавляет в переменную "str" все, что было последней клавишей в виде строки.
