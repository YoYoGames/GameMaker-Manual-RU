keyboard_lastkey

Эта переменная ссылается на значение, которое keyboard_key было в предыдущем кадре, возвращая ключевой код этой клавиши (возвращаются все стандартные константы ключевого кода). Эта переменная не является только для чтения, и вы можете изменить ее, например, установить значение -1, если вы его уже обработали.

Синтаксис:

keyboard_lastkey;

Возвращается:

Virtual Key Constant

Пример:

if (keyboard_lastkey != -1)
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

Приведенный выше код проверяет, не равна ли переменная lastkey значению -1, и если равна, то добавляет в переменную "str" строку, какой была последняя клавиша, после чего сбрасывает переменную keyboard_lastkey для дальнейшего ввода.
