очистка_клавиатуры

С помощью этой функции вы можете очистить текущее состояние клавиатуры, что означает, что если клавиша удерживается нажатой, она больше не будет распознаваться до тех пор, пока ее снова не отпустят (что не вызовет событие Keyboard Key Released и в этом случае). ) и нажата снова. Функция принимает значение кода клавиши, возвращаемое функцией ord() (только заглавные буквы от A-Z или цифры от 0-9), или любую из констант vk_*, перечисленных на главной странице " Ввод с клавиатуры ".

Синтаксис:

keyboard_clear(key);

АргументТип Описание
key The key to clear.

Возвращается:

Пример:

keyboard_clear(vk_space);

Приведенный выше код очищает состояние пробела.
