keyboard_key

С помощью этой переменной вы можете получить код клавиши, которая в данный момент нажата, и она вернет 0, если в момент проверки ни одна клавиша не нажата.

Синтаксис:

keyboard_key;

Возвращается:

Virtual Key Constant

Пример:

switch (keyboard_key)
{
    case vk_numpad1: gun = weapon[0][0]; break;
    case vk_numpad2: gun = weapon[1][0]; break;
    case vk_numpad3: gun = weapon[2][0]; break;
}

Приведенный выше код использует значение переменной keyboard_key, чтобы установить переменную в то же значение, что и массив.
