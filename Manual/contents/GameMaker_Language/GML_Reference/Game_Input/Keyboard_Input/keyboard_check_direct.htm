keyboard_check_direct

Эта функция возвращает true, если клавиша с определенным кодом нажата, или false, если нет, проверяя аппаратное обеспечение напрямую. Она позволяет выполнить несколько дополнительных проверок, в частности, вы можете использовать коды vk_lshift, vk_lcontrol, vk_lalt, vk_rshift, vk_rcontrol и vk_ralt для проверки того, нажата ли левая или правая клавиша shift, control или alt. Функция принимает значение кода клавиши, возвращаемое функцией ord() (только заглавные буквы от A-Z или цифры от 0-9), или любую из констант vk_*, перечисленных на главной странице ввода с клавиатуры.

ПРИМЕЧАНИЕ Эта функция доступна только для стандартной цели Windows, и результат не зависит от того, какое приложение находится в фокусе.

Синтаксис:

keyboard_check_direct(key);

=======

keyboard_check_direct

This function will return true if the key with the particular keycode is pressed, or false if it is not, by checking the hardware directly.

It allows for a few more checks, in particular you can use keycodes vk_lshift, vk_lcontrol, vk_lalt, vk_rshift, vk_rcontrol and vk_ralt to check whether the left or right shift, control or alt key is pressed. The function will take a keycode value as returned by the function ord (only capital letters from A-Z or numbers from 0-9), or any of the vk_* constants listed on the main Keyboard Input page.

 This function is only available for the standard Windows target and the result is independent of which application has focus. On all other platforms it behaves the same as keyboard_check.

 

Syntax:

keyboard_check_direct(key);
