keyboard_check_released

С помощью этой функции можно проверить, был ли ключ отпущен или нет. В отличие от keyboard_check() эта функция будет выполняться только один раз при каждом поднятии клавиши, поэтому, чтобы она сработала снова, клавиша должна быть сначала нажата, а затем снова отпущена. Функция принимает значение кода клавиши, возвращаемое функцией ord() (только заглавные буквы от A-Z или цифры от 0-9), или любую из констант vk_*, перечисленных на главной странице ввода с клавиатуры.

Синтаксис:

keyboard_check_released(key);

АргументТип Описание
key The key to check the released state of.

Возвращается:

Пример:

if keyboard_check_released(ord("P"))
{
    instance_create_layer(0, 0, "Controllers", obj_Pause);
}

Приведенный выше код проверит, была ли отпущена клавиша "P", и если да, то создаст экземпляр "obj_Pause".
