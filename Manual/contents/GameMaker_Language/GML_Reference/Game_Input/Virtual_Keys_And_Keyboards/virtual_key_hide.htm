virtual_key_hide

После того, как вы создали виртуальный ключ для ваших устройств, вам может понадобиться отладка и проверка того, что он правильно расположен и работает должным образом, и для этого вы должны вызвать функцию virtual_key_show(). Однако вам может понадобиться включать и выключать это представление, поэтому вы также можете скрыть виртуальный ключ с помощью этой функции, используя индекс виртуального ключа, который вы хотите скрыть (ранее созданный и сохраненный с помощью функции virtual_key_add()). После того как вы будете довольны, рекомендуется создать и разместить собственную графику в ее области.

Синтаксис

virtual_key_hide(index);

Аргумент Тип Описание
index Идентификатор виртуального ключа The index of the virtual key to hide.

Возвращается:

N/A

Пример:

if global.Debug
{
    virtual_key_show(global.left);
}
else
{
    virtual_key_hide(global.left);
}

Приведенный выше код проверяет глобальную переменную "global.Debug" и если она истинна, то виртуальная клавиша, индексированная в переменной "global.Left", будет нарисована на экране, а если ложна, то клавиша будет скрыта.
