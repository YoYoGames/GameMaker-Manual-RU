virtual_key_show

После того как вы создали виртуальный ключ для своих устройств, вам может понадобиться отладка и проверка правильности его расположения и работы. Для этого нужно вызвать эту функцию, используя индекс виртуальной клавиши, которую вы хотите увидеть (предварительно созданной и сохраненной с помощью функции virtual_key_add()), которая нарисует цветной прямоугольник для отображения клавиши на экране. Как только вы будете довольны, рекомендуется создать и разместить в его области свою собственную графику.

ПРИМЕЧАНИЕ На рисуемый прямоугольник будут влиять параметры смешивания, цвета и альфа, установленные соответствующими функциями рисования.

Синтаксис

virtual_key_show(index);

Аргумент Тип Описание
index Идентификатор виртуального ключа The index of the virtual key to show.

Возвращается:

N/A

Пример:

if global.Debug
{
    virtual_key_show(global.left);
}
else
{
    virtual_key_hide(global.left);
}

Приведенный выше код проверяет глобальную переменную "global.Debug" и если она истинна, то виртуальная клавиша, индексированная в переменной "global.Left", будет нарисована на экране, а если ложна, то клавиша будет скрыта.
