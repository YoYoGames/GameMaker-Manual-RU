device_mouse_x_to_gui

Эта функция возвращает x-позицию касания на устройстве по отношению к слою GUI. Поскольку слой GUI может иметь различные размеры независимо от разрешения экрана и размера комнаты, иногда бывает сложно определить точную точку на слое GUI, в которой пользователь коснулся экрана. С помощью этой функции вы можете получить эту позицию на основе фактического размера GUI, что упрощает создание и взаимодействие с кнопками GUI и элементами HUD. Если вы используете эту функцию на модулях HTML5 или PC и Mac, то это значение обновляется постоянно, пока устройство (обычно мышь) подключено, однако для мобильных устройств это значение будет обновляться только во время касания экрана, и обратите внимание, что максимальное количество касаний, которое может быть обнаружено, будет сильно зависеть от используемого устройства и ОС, на которой оно работает.

Синтаксис:

device_mouse_x_to_gui(device);

АргументТип Описание
device The device (from 0 - n) that is being checked.

Возвращается:

Пример:

if device_mouse_check_button(0, mb_left)
{
    if device_mouse_x_to_gui(0) > 0 && device_mouse_x_to_gui(0) < 32
    {
        pressed = true;
    }
    else
    {
        pressed = false;
    }
}

Приведенный выше код проверяет, нажимается ли мышь устройства, и если да, то опрашивает позицию x устройства на слое GUI, чтобы проверить, находится ли она в пределах параметров. Если да, то переменная "pressed" устанавливается в true, в противном случае - в false.
