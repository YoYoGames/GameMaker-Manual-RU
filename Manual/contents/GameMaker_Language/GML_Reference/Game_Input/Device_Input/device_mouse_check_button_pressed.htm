device_mouse_check_button_pressed

Эта функция возвращает true или false в зависимости от того, было ли указанное вами устройство "затронуто" (нажато) или нет. Аргумент device указывает на номер касания, который может быть от 0 до n, а максимальное количество касаний, которое может быть обнаружено, будет сильно зависеть от используемого устройства и ОС, под управлением которой оно работает (большинство устройств поддерживают не менее 4 одновременных касаний). Эта функция срабатывает только один раз при фактическом нажатии, и для проверки кнопок мыши можно использовать константы, перечисленные на этой странице. Обратите внимание, что mb_right будет обнаружен только в случае двойного касания и удержания при втором касании (это поведение можно отключить с помощью функции device_mouse_dbclick_enable()).

Синтаксис:

device_mouse_check_button_pressed(device, button);

АргументТип Описание
device The device (from 0 - n) that is being checked.
button The button of the device that is being checked.

Возвращается:

Пример:

if device_mouse_check_button_pressed(0, mb_left)
{
    press = true;
}

Приведенный выше код проверяет, было ли нажато устройство, и устанавливает переменную в true, если да.
