device_mouse_y

Эта функция возвращает положение y касания на устройстве. Если вы работаете на модулях HTML5 или PC и Mac, то это значение обновляется постоянно, пока устройство (обычно мышь) подключено, однако для мобильных устройств это значение будет обновляться только во время касания экрана. Обратите внимание, что максимальное количество касаний, которое может быть обнаружено, будет сильно зависеть от используемого устройства и ОС, на которой оно работает.

Синтаксис:

device_mouse_y(device);

АргументТип Описание
device The device (from 0 - n) that is being checked.

Возвращается:

Пример:

if device_mouse_check_button(0, mb_left)
{
    if device_mouse_y(0) > y-32 && device_mouse_y(0) < y+32
    {
        pressed = true;
    }
    else
    {
        pressed = false;
    }
}

Приведенный выше код проверяет, нажимается ли устройство, и если да, то опрашивает положение y устройства, чтобы проверить, находится ли оно в пределах параметров. Если да, то переменная "pressed" устанавливается в true, в противном случае - в false.
