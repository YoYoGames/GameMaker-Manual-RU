device_is_keypad_open

Эта функция проверяет устройство на наличие клавиатуры, и если таковая имеется, возвращает true, в противном случае возвращает false. Обратите внимание, что эта функция предназначена в основном для использования с устройствами Android.

Для пользователей Sony Xperia Play в GameMaker есть настройка карты кнопок и клавиш, поэтому вы можете использовать клавиатурные константы vk_up, vk_down, vk_left, vk_right для клавиш джойпада, а треугольник - ord("T"), квадрат - ord("S"), круг - vk_alt + vk_backspace, а кнопка назад - просто vk_backspace, крестик - vk_space, Select - vk_return и Start - vk_rshift, триггер L - ord("L") и триггер R - ord("R").

Синтаксис:

device_is_keypad_open

Возвращается:

Пример:

if device_is_keypad_open()
{
     global.Setting = 2;
}
else
{
     global.Setting = 1;
}

Приведенный выше код проверяет наличие клавиатуры, затем изменяет глобальную переменную в зависимости от возвращаемого значения.
