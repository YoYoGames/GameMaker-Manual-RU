device_mouse_dbclick_enable

С помощью этой функции можно настроить устройство на распознавание двойного нажатия mb_left (левой кнопки мыши) как ввода mb_right (правой кнопки мыши).

По умолчанию это значение установлено на true, что означает, что каждый раз, когда пользователь дважды быстро и последовательно касается экрана устройства, возвращаемое значение будет таким же, как если бы была нажата правая кнопка мыши. Если этот параметр включен, первое касание будет определено как mb_left, а второе - как mb_right, поэтому убедитесь, что используемый вами код учитывает это.

Синтаксис:

device_mouse_dbclick_enable(bool);

Аргумент Тип Описание
bool Булево Set double-click detection on (true) или выключить (false).

Возвращается:

Boolean

Пример:

if os_type == os_windows || os_type == os_mac
{
    device_mouse_dbclick_enable(false);
}

Приведенный выше код проверяет, является ли устройство, на котором запущена игра, ПК Windows или Mac, и если это одно из них, то отключает функцию двойного нажатия.
