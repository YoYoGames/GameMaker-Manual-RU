device_mouse_check_button

Эта функция возвращает true или false в зависимости от того, удерживается ли заданная кнопка мыши на данном устройстве. Для кнопки мыши вы можете задать одну из констант, перечисленных на этой странице.

Эта функция может использоваться для сенсорных экранов, а аргумент устройства может быть от 0 до n, где n - максимальное количество "пальцев", которые могут касаться экрана одновременно (максимальное количество касаний, которые могут быть обнаружены, будет сильно зависеть от используемого устройства, но большинство устройств должны определять как минимум до 4. будет обнаружено, во многом зависит от используемого устройства и ОС, под управлением которой оно работает, но большинство устройств должны обнаруживать не менее 4). Обратите внимание, что mb_right будет обнаружен только при двойном касании (это поведение можно отключить с помощью функции функция device_mouse_dbclick_enable()).

Синтаксис:

device_mouse_check_button(device, button);

АргументТип Описание
device The device (from 0 - n) that is being checked
button The button of the device that is being checked.

Возвращается:

Пример:

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
    room_goto(rm_Menu);
}

Приведенный выше код проверяет, удерживаются ли два касания одновременно, и если да, то переходит в другую комнату.
