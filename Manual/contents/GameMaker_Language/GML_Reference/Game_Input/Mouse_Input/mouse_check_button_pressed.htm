mouse_check_button_pressed

Эта функция возвращает true, если проверяемая кнопка мыши была нажата, или false, если нет. Эта функция срабатывает только один раз для любой кнопки мыши, когда она впервые нажата, и для ее повторного срабатывания кнопку нужно отпустить и нажать снова. Обратите внимание, что она будет считаться сработавшей в течение всего шага и для всех экземпляров, в которых есть какие-либо события мыши или которые используют эту же функцию.

Вы задаете кнопку мыши для проверки одной из следующих констант:

Синтаксис:

mouse_check_button_pressed(numb);

Аргумент Тип Описание
numb Постоянная кнопка мыши Which mouse button constant to check for.

Возвращается:

Boolean

Пример:

if mouse_check_button_pressed(mb_left)
{
    score += 50;
}

Приведенный выше код проверит, была ли нажата левая кнопка мыши, и если да, то добавит 50 к счету.
