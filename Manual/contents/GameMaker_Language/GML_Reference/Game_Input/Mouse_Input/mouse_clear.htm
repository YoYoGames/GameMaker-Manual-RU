очистка_мыши

Эта функция очищает текущее состояние заданной кнопки мыши. Это означает, что проверка на то, что она удерживается нажатой, не вернет true, пока игрок не отпустит кнопку и не нажмет ее снова (но состояние отпускания все равно будет обнаружено, если очистка выполняется, пока кнопка мыши удерживается нажатой).

Вы можете ввести одну из констант кнопок мыши, перечисленных на этой странице.

Синтаксис:

mouse_clear(button);

Аргумент Тип Описание
button Постоянная кнопка мыши Which mouse button constant to check for.

Возвращается:

Boolean

Пример:

mouse_clear(mb_any);

Приведенный выше код очистит состояние "вниз" всех кнопок мыши.
