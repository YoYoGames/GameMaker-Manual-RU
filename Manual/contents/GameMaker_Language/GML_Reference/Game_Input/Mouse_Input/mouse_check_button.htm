кнопка_проверки_мыши

Эта функция возвращает true, если проверяемая кнопка мыши удерживается нажатой, или false, если нет. Вы задаете кнопку мыши для проверки из одной из следующих констант:

Синтаксис:

mouse_check_button(numb);

Аргумент Тип Описание
numb Постоянная кнопка мыши Which mouse button constant to check for.

Возвращается:

Boolean

Пример:

if mouse_check_button(mb_left)
{
    instance_create_layer(mouse_x, mouse_y, "Effects", obj_Star);
}

Приведенный выше код будет проверять наличие левой кнопки мыши и при каждом ее удержании будет создавать экземпляр объекта, индексированного в "obj_Star".
