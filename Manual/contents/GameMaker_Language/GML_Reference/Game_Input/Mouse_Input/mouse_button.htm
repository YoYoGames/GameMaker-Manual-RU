кнопка_мыши

Эта переменная, доступная только для чтения, возвращает кнопку мыши, которая в данный момент нажата (в данный момент, как на этом шаге), и может возвращать любую из специальных констант мыши, кроме mb_any.

Синтаксис:

mouse_button;

Возвращается:

Mouse Button Constant (except mb_any)

Пример:

if mouse_button = mb_left
{
    x -= 1;
}

Это перемещает текущий экземпляр влево, если левая кнопка мыши нажата.
