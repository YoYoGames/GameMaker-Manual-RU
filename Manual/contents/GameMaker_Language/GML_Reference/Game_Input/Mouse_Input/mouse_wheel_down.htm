колесо_мыши_вниз

Эта функция возвращает true, если колесо мыши вращается вниз, и false в противном случае.

Синтаксис:

mouse_wheel_down();

Возвращается:

Пример:

if mouse_wheel_down()
{
    y += 10;
}

Это перемещает текущий экземпляр вниз по экрану, если колесико мыши повернуть вверх.
