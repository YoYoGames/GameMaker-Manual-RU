gamepad_get_button_threshold

Эта функция может использоваться для определения текущей пороговой настройки аналоговых кнопок для данного устройства. По умолчанию порог для всех аналоговых кнопок составляет 0,5, а диапазон - от 0 до 1. Порог определяет, в какой момент кнопка считается "нажатой" для игр, которые требуют, чтобы она действовала как цифровая кнопка.

Синтаксис:

gamepad_get_button_threshold(device);

АргументТип Описание
device Which gamepad device "slot" to check.

Возвращается:

Пример:

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

Приведенный выше код проверяет пороговое значение аналоговой кнопки геймпада, подключенного к устройству "slot" 0, и если оно не равно значению по умолчанию 0,5, то оно устанавливается на это значение.

Назад: Вход геймпада
