gamepad_hat_value

С помощью этой функции вы можете получить текущее значение "шляпы" геймпада. Каждое значение "шляпы" представляет собой битовую маску для различных направлений, где:

  • Вверх = 1
  • Право = 2
  • Вниз = 4
  • Левая = 8

Обратите внимание, что их можно комбинировать (например, при индексе шляпы, равном 3, проверка будет происходить вверх и вправо, или вы можете использовать оператор | для объединения различных значений), но одновременно могут быть нажаты не более 2 битов. Функция возвращает вещественное значение от 0 до 1, где 0 - не нажата, а 1 - полностью нажата (могут быть и промежуточные значения в зависимости от того, поддерживает ли геймпад аналоговый ввод для шляп).

Синтаксис:

gamepad_hat_value(device, hatindex);

Аргумент Тип Описание
device Реально Which gamepad device "slot" to check.
hatindex Реально Which gamepad hat (or hats) to check for.

Возвращается:

Real

Пример:

var _dir = gamepad_hat_value(global.PadIndex, 0);
switch (_dir)
{
    case 1: y -= 3; break;
    case 2: x += 3; break;
    case 3: y -= 3; x += 3; break;
    case 4: x -= 3; break;
    case 6: y += 3; x += 3; break;
    case 8: y += 3; break;
    case 9: y -= 3; x -= 3; break;
    case 12: y += 3; x -= 3; break;
}

Приведенный выше код сохраняет состояние шляпы "0" в локальной переменной, затем проверяет возвращаемое значение и действует соответствующим образом.
