gamepad_get_device_count

Эта функция показывает либо количество подключенных игровых планшетов, либо количество доступных "слотов" для подключения игровых планшетов.

Фактическое возвращаемое значение будет зависеть от платформы и внутренней конфигурации этой платформы, поэтому данную функцию следует использовать вместе с функцией gamepad_is_connected() чтобы быть уверенным в точном количестве подключенных планшетов в любой момент времени.

Синтаксис:

gamepad_get_device_count();

Возвращается:

Real

Пример:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++)
{
    if (gamepad_is_connected(i))
    {
        global.gp[i] = true;
    }
    else
    {
        global.gp[i] = false;
    }
}

Приведенный выше код перебирает доступные геймпады (или слоты для геймпадов) и затем проверяет каждый из них на наличие подключенного геймпада. возвращаемое значение затем используется для установки глобального массива на true или false для использования в будущих проверках.
