gamepad_is_supported

С помощью этой функции можно узнать, поддерживает ли целевая платформа геймпады (возвращается true) или нет (возвращается false).

Синтаксис:

gamepad_is_supported();

Возвращается:

Пример:

global.GP = gamepad_is_supported();

Приведенный выше код проверяет, поддерживается ли геймпад, и сохраняет возвращаемое значение в глобальной переменной для будущих проверок.
