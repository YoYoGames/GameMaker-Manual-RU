счётчик_кнопок_геймпада

Эта функция вернет общее количество кнопок, доступных для геймпада, подключенного к заданному "слоту" устройства.

Синтаксис:

gamepad_button_count(device);

АргументТип Описание
device Which gamepad device "slot" to check.

Возвращается:

Пример:

b_num = gamepad_button_count(0);

Приведенный выше код вернет количество кнопок, доступных на геймпаде, подключенном к устройству "slot" 0, и сохранит значение в переменной "b_num".
