gamepad_get_option

Эта функция может быть использована для получения заданного значения опции. Вы указываете "слот" геймпада для проверки, а также строку опции, и функция возвращает соответствующее значение. Доступная строка опций зависит от платформы, на которой выполняется проект, как указано ниже:

Вариант Платформа Описание Возвращает
"allow_rotation" tvOS Возвращает, будет ли пульт Siri Remote заблокирован на вращение (т.е. может использоваться только вертикально) или нет. При разблокировке вращение пульта изменит ввод, например, поворот на 90º влево заставит "правую" сторону сенсорной панели вернуться "вверх" и т.д... таким образом, весь ввод будет относиться к ориентации устройства. Булево
"dpad_absolute" tvOS Возвращает, принимает ли сенсорная панель Siri Remote абсолютный ввод или относительный. В абсолютном режиме центром сенсорной панели считается позиция (0,0) - т.е. место, где аналоговый стик находится в состоянии покоя или где был бы центр дпада - и движение вокруг этой позиции будет генерировать ввод. Если установлено относительное значение, то позиция (0,0) будет рассматриваться как место, где было сделано первоначальное касание тачпада, и весь ввод будет генерироваться из этой позиции, а не из центра тачпада. Булево

Синтаксис:

gamepad_get_option(device, option_name);

АргументТип Описание
device Which gamepad device "slot" to check.
option_name The name of the option to get the value from (a string, see the table above)

Возвращается:

Boolean, Real, String, or N/A (depends on platform and option)

Пример:

for(var i = 0; i < 12; ++i;)
{
    if (gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote")
    {
        if (gamepad_get_option(i, "allow_rotation") == false)
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

Приведенный выше код перебирает все слоты геймпада и проверяет наличие "Siri Remote" на платформе tvOS. Если таковой обнаружен, он устанавливает пульт так, чтобы разрешить вращение ввода.
