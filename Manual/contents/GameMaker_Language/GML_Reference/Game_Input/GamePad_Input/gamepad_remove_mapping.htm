gamepad_remove_mapping

Эта функция используется для удаления текущего отображения устройства из заданного индекса слота геймпада. После ее вызова индекс слота необходимо изменить с помощью функции gamepad_test_mapping() если вы хотите иметь возможность использовать константы геймпада для корректного определения ввода (прямой ввод всегда можно получить, используя функции gamepad_axis/button/hat_count() и gamepad_axis/button/hat_value() вместе).

Синтаксис:

gamepad_remove_mapping(index);

АргументТип Описание
index Which gamepad index "slot" to remove the mapping from.

Возвращается:

Пример:

if remap == true
{
    gamepad_remove_mapping(global.PadIndex);
}

Приведенный выше код удалит отображение из заданного слота индекса геймпада на основе значения переменной.
