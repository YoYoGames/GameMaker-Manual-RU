gamepad_set_option

Эта функция может быть использована для установки любой из доступных опций геймпада. Вы указываете "слот" геймпада для установки опции, а также строку опции, которую нужно установить, и значение, которое нужно использовать. Доступная строка опций зависит от платформы, на которой выполняется проект, как указано ниже:

Вариант Платформа Описание Значение
"allow_rotation" tvOS Установите, будет ли пульт Siri Remote заблокирован на вращение (т.е. может использоваться только в вертикальном положении) или нет. При разблокировке (значение true) вращение пульта изменит ввод, например, поворот на 90º влево заставит "правую" сторону сенсорной панели вернуться "вверх" и т.д... таким образом, весь ввод будет осуществляться относительно ориентации устройства. При блокировке (значение false) ввод не будет изменяться относительно положения контроллера и всегда будет рассматриваться так, как будто контроллер находится в вертикальном положении. Булево
"dpad_absolute" tvOS Устанавливает, принимает ли сенсорная панель Siri Remote абсолютный или относительный ввод. В абсолютном режиме (значение true) центром сенсорной панели считается положение (0,0) - т.е. место, где аналоговый стик находится в состоянии покоя, или место, где был бы центр dpad - и движение вокруг этого положения будет генерировать ввод. Если установлено относительное значение (значение false ), то позиция (0,0) будет рассматриваться как место, где было сделано первоначальное прикосновение к тачпаду, и весь ввод будет генерироваться из этой позиции, а не из центра тачпада. Булево

Синтаксис:

gamepad_set_option(device, option_name, value);

Аргумент Тип Описание
device Реально Which gamepad device "slot" to set.
option_name Строка The name of the option to set (a string, see the table above)
value Вещественный, булевский или строковый The value to set the option to (can be boolean, real or string - see the table above)

Возвращается:

N/A

Пример:

for(var i = 0; i < 12; ++i;)
{
    if gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote"
    {
        if gamepad_get_option(i, "allow_rotation") == false
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

Приведенный выше код перебирает все слоты геймпада и проверяет наличие "Siri Remote" на платформе tvOS. Если таковой обнаружен, он устанавливает пульт так, чтобы разрешить вращение ввода.
