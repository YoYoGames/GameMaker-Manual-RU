gamepad_set_button_threshold

С помощью этой функции можно установить текущее пороговое значение аналоговых кнопок для данного устройства. По умолчанию порог для всех аналоговых кнопок составляет 0,5, а диапазон - от 0 до 1.

Порог определяет, в какой момент кнопка считается "нажатой" для игр, в которых они должны действовать как цифровая кнопка.

Эта функция влияет на состояния проверки, нажатия и отпускания для аналоговых кнопок, но не влияет на значение, возвращаемое функцией gamepad_button_value()которая всегда будет возвращать исходное значение для кнопки.

Синтаксис:

gamepad_set_button_threshold(device, threshold);

Аргумент Тип Описание
device Реально Which gamepad device "slot" to check.
threshold Реально The new threshold value (from 0 - 1, default 0.5).

Возвращается:

N/A

Пример:

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

Приведенный выше код проверяет пороговое значение аналоговой кнопки геймпада, подключенного к устройству "slot" 0, и если оно не равно значению по умолчанию 0,5, то оно устанавливается на это значение.
