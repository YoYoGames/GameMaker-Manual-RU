gamepad_axis_count

Эта функция возвращает количество элементов управления "axis" на проверяемом устройстве. Эти элементы управления представляют собой аналоговые "палочки" направления на большинстве контроллеров.

Синтаксис:

gamepad_axis_count(device);

АргументТип Описание
device Which gamepad device "slot" to check.

Возвращается:

Пример:

axis = gamepad_axis_count(0))

Приведенный выше код сохраняет в переменной "axis" количество осей, доступных для геймпада, подключенного к устройству "slot" 0.
