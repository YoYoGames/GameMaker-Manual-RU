gamepad_button_check_pressed

Эта функция возвращает значение true или false в зависимости от того, была ли обнаружена нажатой данная кнопка геймпада или нет. Обратите внимание, что эта функция сработает только один раз при первом нажатии кнопки. Чтобы она сработала повторно, кнопку нужно сначала отпустить, а затем нажать снова. Если вам нужно проверить непрерывное нажатие кнопки, следует использовать функцию gamepad_button_check().

Если вы проверяете аналоговую кнопку, то проверка будет учитывать установку порога и вернет значение true только тогда, когда необработанное значение входа превысит заданный порог (вы можете получить это необработанное значение с помощью функции gamepad_button_value()).

Синтаксис:

gamepad_button_check_pressed(device, button);

Аргумент Тип Описание
device Реально Which gamepad device "slot" to check.
button Постоянная кнопка геймпада Which gamepad button constant to check for.

Возвращается:

Boolean

Пример:

if gamepad_button_check_pressed(0, gp_start)
{
    audio_play_sound(snd_Start, 0, false);
    room_goto(rm_Level_1);
}

Приведенный выше код определит, была ли нажата кнопка "start" геймпада, подключенного к устройству "slot" 0, или нет, и изменит комнату, если это произошло.
