gamepad_set_axis_deadzone

Эта функция может использоваться для установки "мертвой зоны" оси джойстика. Вы указываете слот устройства для настройки, а затем задаете значение от 0 до 1, и если входная величина меньше заданного значения, то считается, что ось джойстика находится в 0. Обратите внимание, что это глобальная настройка, которая будет влиять на все оси всех джойстиков, подключенных к указанному слоту устройства.

Использование этой функции не означает, что значение оси будет начинаться со значения мертвой зоны, так как функция gamepad_axis_value() всегда будет возвращать нормализованное значение между -1 Например, установка мертвой зоны на 0,2 означает, что нажатие стика вправо начнет возвращать значение от 0 до 1 только тогда, когда необработанное значение оси превысит 0,2 - таким образом, когда необработанное значение равно 0,2, возвращаемое значение будет равно 0, когда необработанное значение равно 0.5, возвращаемое значение будет 0.375, или если необработанное значение равно 0.9, то возвращаемое значение будет 0.875.

Синтаксис:

gamepad_set_axis_deadzone(device, deadzone);

АргументТип Описание
device Which gamepad device "slot" to check.
deadzone The dead zone value from 0 to 1.

Возвращается:

Пример:

if gamepad_is_connected(0) gamepad_set_axis_deadzone(0, 0.05);
if gamepad_is_connected(1) gamepad_set_axis_deadzone(1, 0.05);
if gamepad_is_connected(2) gamepad_set_axis_deadzone(2, 0.05);
if gamepad_is_connected(3) gamepad_set_axis_deadzone(3, 0.05);

Приведенный выше код установит мертвую зону оси джойстика на любом из 4 подключенных устройств на 0,05.
