gamepad_set_vibration

С помощью этой функции вы можете настроить вибрацию моторов геймпада, причем для каждого из моторов используется значение от 0 (без вибрации) до 1 (полная вибрация). Обратите внимание, что в этой функции нет ограничения по времени, поэтому вам нужно будет использовать переменную или сигнал тревоги, чтобы отключить вибрацию (установить моторы в 0) по истечении заданного времени, иначе геймпад будет продолжать вибрировать бесконечно.

ПРИМЕЧАНИЕ: В настоящее время эта функция доступна только для стандартных целевых модулей Windows, PS4 и Xbox One.

Синтаксис:

gamepad_set_vibration(device, left_motor, right_motor);

АргументТип Описание
device Which gamepad device "slot" to check.
left_motor The amount of vibration from the left motor from 0 to 1.
right_motor The amount of vibration from the right motor from 0 to 1.

Возвращается:

Пример:

if lives = 0
{
    gamepad_set_vibration(0, 1, 1);
    alarm[0] = room_speed / 2;
}

Приведенный выше код можно использовать (например) в событии столкновения, чтобы заставить геймпад, подключенный к "гнезду" 0, вибрировать в течение половины секунды, а установленный сигнал тревоги будет использоваться для его повторного отключения по истечении этого времени.
