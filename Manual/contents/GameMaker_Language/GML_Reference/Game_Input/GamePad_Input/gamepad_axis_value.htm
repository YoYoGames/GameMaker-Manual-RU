значение_оси_геймпада

Вы можете использовать эту функцию для получения значений различных осей от заданного слота геймпада. Вы указываете индекс слота геймпада для проверки, а также либо константу индекса оси (как указано здесь), либо целое значение между 0 и gamepad_axis_count()-1. Возвращаемое значение будет находиться между -1 и 1 для каждой из доступных горизонтальных и вертикальных осей.

Обратите внимание, что возвращаемое значение нормализуется с учетом значения, установленного для мертвой зоны оси (с помощью функции gamepad_set_axis_deadzone()), и поэтому всегда будет находиться между -1 и 0 и 0 и 1. Например, если установить мертвую зону на 0.2 означает, что нажатие стика вправо начнет возвращать значение от 0 до 1 только тогда, когда необработанное значение оси превысит 0,2 - например, когда необработанное значение равно 0,2, возвращаемое значение будет равно 0, когда необработанное значение равно 0,5, возвращаемое значение будет равно 0,375, или если необработанное значение равно 0,9, то возвращаемое значение будет равно 0,875.

Синтаксис:

gamepad_axis_value(device, axisIndex);

Аргумент Тип Описание
device Реально Which gamepad device "slot" to check.
axisIndex Постоянная оси геймпада The axis index to check (see the constants list).

Возвращается:

Real

Пример:

var haxis = gamepad_axis_value(0, gp_axislh);
var vaxis = gamepad_axis_value(0, gp_axislv);
direction = point_direction(0, 0, haxis, vaxis);
speed = point_distance(0 ,0, haxis, vaxis) * 5;

Приведенный выше код устанавливает скорость и направление движения экземпляра на основе возвращаемых значений левого аналогового регулятора геймпада, подключенного к устройству "слот" 0.
