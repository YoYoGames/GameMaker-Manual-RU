gamepad_button_check_released

Эта функция возвращает значение true или false в зависимости от того, была ли обнаружена отпущенной данная кнопка геймпада или нет. Обратите внимание, что эта функция сработает только один раз для кнопки в тот момент, когда она была отпущена. Чтобы она сработала снова, кнопка должна быть сначала нажата, а затем отпущена еще раз. Если вы проверяете аналоговую кнопку, то при проверке будет учитываться настройка порога срабатывания и функция вернет значение true только в том случае, если необработанное входное значение окажется ниже заданного порога (необработанное значение можно получить с помощью функции gamepad_button_value()).

Синтаксис:

gamepad_button_check_released(device, button);

Аргумент Тип Описание
device Реально Which gamepad device "slot" to check.
button Постоянная кнопка геймпада Which gamepad button constant to check for.

Возвращается:

Boolean

Пример:

if gamepad_button_check_released(0, gp_select)
{
    audio_play_sound(snd_Button, 0, false);
    global.Pause = !global.Pause;
}

Приведенный выше код определит, была ли нажата кнопка "select" геймпада, подключенного к устройству "slot" 0, и переключит глобальную переменную "Pause".
