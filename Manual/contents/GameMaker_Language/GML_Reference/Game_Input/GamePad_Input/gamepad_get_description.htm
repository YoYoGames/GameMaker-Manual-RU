gamepad_get_description

Эта функция возвращает строку с описанием данного геймпада, например, контроллер PS3 может вернуть строку, подобную "PLAYSTATION(R)3 Controller", или контроллер XBox360 может вернуть "Xbox 360 Controller (XInput STANDARD GAMEPAD)". Эта строка зависит от аппаратного обеспечения, и возвращаемое значение будет зависеть от геймпада, подключенного к проверяемому "слоту" устройства.

Синтаксис:

gamepad_get_description(device);

АргументТип Описание
device Which gamepad "slot" to get the name of.

Возвращается:

Пример:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    var _gamepad = gamepads[i];
    if (gamepad_is_connected(_gamepad))
    {
        draw_text(32, 32 + (i * 32), gamepad_get_description(_gamepad));
    }
    else
    {
        draw_text(32, 32 + (i * 32), "Gamepad not found");
    }
}

Приведенный выше код пройдется по всем слотам геймпада для проверки наличия подключенных устройств, а затем нарисует текст на экране в зависимости от того, подключен ли геймпад к слоту или нет.
