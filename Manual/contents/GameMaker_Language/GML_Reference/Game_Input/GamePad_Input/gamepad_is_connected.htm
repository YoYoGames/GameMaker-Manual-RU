gamepad_is_connected

Эта функция возвращает, подключен ли геймпад к данному "слоту" (возвращает true) или нет (возвращает false). Обычно эта функция используется вместе с функцией gamepad_get_device_count() для получения правильного количества доступных геймпадов и/или "слотов" для геймпадов. Обратите внимание, что может возникнуть небольшая задержка между подключением геймпада пользователем и обнаружением GameMaker его подключения (это особенно актуально при работе с контроллерами, подключенными по bluetooth).

ПРИМЕЧАНИЕ: На некоторых платформах - особенно консолях - эта функция может вернуть false сразу после подключения/выбора геймпада и может потребовать проверки в тревоге несколькими кадрами (шагами) позже, чтобы она правильно определила геймпад.

Синтаксис:

gamepad_is_connected(numb);

АргументТип Описание
device Which gamepad "slot" to check.

Возвращается:

Пример:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false;
}

Приведенный выше код перебирает доступные геймпады (или слоты для геймпадов) и затем проверяет каждый из них на наличие подключенного геймпада. возвращаемое значение затем используется для установки глобального массива на true или false для использования в будущих проверках.
