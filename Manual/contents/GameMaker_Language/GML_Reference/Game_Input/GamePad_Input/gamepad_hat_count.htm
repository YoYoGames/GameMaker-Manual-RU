gamepad_hat_count

Эта функция возвращает общее количество шапок, доступных для геймпада, подключенного к заданному "слоту" устройства. Под шляпами обычно подразумеваются кнопки вверх/вниз/влево/вправо, и обратите внимание, что на сайте Windows шляпы доступны только для контроллеров с прямым вводом. контроллеров (то есть, начиная со слота 4 и выше).

Синтаксис:

gamepad_hat_count(device);

АргументТип Описание
device Which gamepad device "slot" to check.

Возвращается:

Пример:

h_num = gamepad_hat_count(4);

Приведенный выше код вернет количество шапок, доступных на геймпаде, подключенном к устройству "slot" 4, и сохранит значение в переменной "h_num".
