значение_кнопки_геймпада

С помощью этой функции вы можете получить текущее значение аналоговой кнопки, от 0 до 1, где 0 - отсутствие нажатия, а 1 - полное нажатие. Вы указываете индекс слота геймпада для проверки, а также либо константу кнопки (как указано здесь), либо целочисленное значение от 0 до gamepad_button_count() - 1.

Обратите внимание, что это возвращает необработанное значение для кнопки и не учитывает настройки порога (более подробную информацию о порогах см. здесь ).

Синтаксис:

gamepad_button_value(device, button);

Аргумент Тип Описание
device Реально Which gamepad device "slot" to check.
button Постоянная кнопка геймпада Which gamepad button constant to check for.

Возвращается:

Real

Пример:

speed = gamepad_button_value(0, gp_shoulderrb) * 4;

Приведенный выше код использует значение аналогового триггера от геймпада, подключенного к устройству "слот" 0, для установки скорости экземпляра.
