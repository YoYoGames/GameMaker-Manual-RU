steam_ugc_create_query_user_ex

Эта функция может быть использована для запроса базы данных пользовательского контента. Функция требует значение ID для пользователя и ID игры, которая будет потреблять предмет, и/или ID игры, которая создала предмет. Также необходимо использовать следующие константы для типа данных для запроса, типа элемента для запроса и порядка, в котором будут отсортированы возвращаемые элементы, а также номер страницы - обратите внимание, что запрос вернет максимум 50 элементов.

Функция возвращает уникальное значение хэндла запроса, которое должно быть сохранено в переменной для использования в других функциях запроса. Обратите внимание, что эта функция только подготавливает запрос, но не отправляет его - для этого вы должны вызвать функцию steam_ugc_send_query() - и вы можете использовать другие функции steam_ugc_query_*() для уточнения поискового запроса перед его отправкой.

Синтаксис:

steam_ugc_create_query_user_ex(list_type, match_type, sort_order, page, account_id, creator_app_id, consumer_app_id);

Аргумент Описание
list_type Тип создаваемого списка данных (см. константы, перечисленные ниже).
match_type Тип элементов пользовательского контента для запроса (см. константы, перечисленные ниже).
sort_order Способ, которым следует упорядочить данные (см. константы, перечисленные ниже).
page Номер страницы для запроса.
account_id Идентификатор счета Steam.
creator_app_id Идентификатор приложения создателя элемента.
consumer_app_id Идентификатор потребительского приложения.

Возвращается:

Query Handle

Пример:

query_handle = steam_ugc_create_query_user_ex(ugc_list_Published, ugc_match_Items, ugc_sortorder_TitleAsc, page, global.AccountID, 0, global.GameID);

Приведенный выше код создает запрос запроса и сохраняет его обработку в переменной для дальнейшего использования.
