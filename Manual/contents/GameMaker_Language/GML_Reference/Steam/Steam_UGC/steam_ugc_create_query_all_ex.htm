steam_ugc_create_query_all_ex

Эта функция может быть использована для запроса базы данных UGC. Функция требует ID игры, которая будет потреблять элемент, и/или ID игры, которая создала элемент, а также необходимо использовать следующие константы для типа создаваемого запроса, тип элемента для запроса и номер страницы для запроса. Обратите внимание, что запрос возвращает максимум 50 элементов.

Тип запроса Постоянный Описание
ugc_query_RankedByVote См. документацию Steam.
ugc_query_RankedByPublicationDate -

ugc_query_RankedByTrend -
ugc_query_FavoritedByFriendsRankedByPublicationDate -
ugc_query_CreatedByFriendsRankedByPublicationDate -

ugc_query_CreatedByFollowedUsersRankedByPublicationDate -

ugc_query_RankedByVotesUp -
ugc_query_RankedByTextSearch -

Функция возвращает уникальное значение хэндла запроса, которое должно быть сохранено в переменной для использования в других функциях запроса. Обратите внимание, что эта функция только подготавливает запрос, но не отправляет его - для этого вы должны вызвать функцию steam_ugc_send_query() - и вы можете использовать другие функции steam_ugc_query_*() для уточнения поискового запроса перед его отправкой.

Синтаксис:

steam_ugc_create_query_all_ex(query_type, match_type, page, creator_app_id, consumer_app_id);

Аргумент Описание
query_type Тип запроса данных для создания (см. константы, перечисленные ниже).
match_type Тип элементов пользовательского контента для запроса (см. константы, перечисленные ниже).
page Номер страницы для запроса.
creator_app_id Идентификатор приложения создателя элемента.
consumer_app_id Идентификатор потребительского приложения.

Возвращается:

Query Handle

Пример:

query_handle = steam_ugc_create_query_all_ex(ugc_query_RankedByVote, page, global.AccountID, 0, global.GameID);

Приведенный выше код создает запрос запроса и сохраняет его обработку в переменной для дальнейшего использования.
