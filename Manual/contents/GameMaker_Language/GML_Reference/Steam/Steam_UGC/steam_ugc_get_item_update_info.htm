steam_ugc_get_item_update_info

Эта функция может быть использована для получения информации о текущем состоянии загрузки для заданного ID файла. Вы задаете ID элемента и указываете индекс пустой карты DS, которая затем будет заполнена следующими парами ключ/значение, если элемент существует:

  • "needs_update" - Возвращает true или false в зависимости от того, нуждается ли элемент в обновлении или нет.
  • "is_downloading" - Возвращает true или false в зависимости от того, загружается ли элемент в данный момент или нет.
  • "bytes_downloaded" - Количество байт, которое было загружено.
  • "bytes_total" - Общий размер (количество байт), требуемый для элемента на диске.

Если элемент существует, то функция вернет true и заполнит карту, в противном случае она вернет false и карта останется пустой.

Синтаксис:

steam_ugc_get_item_update_info(published_file_id, info_map);

Аргумент Описание
published_file_id Уникальный идентификатор файла для проверяемого UGC.
info_map (ранее созданный) индекс карты DS.

Возвращается:

Boolean

Пример:

var info_map = ds_map_create();
var info = steam_ugc_get_item_update_info(global.fileID, info_map);
if info
{
    draw_text(32, 15, "needs_update: " + string(info_map[? "needs_update"]));
    draw_text(32, 30, "is_downloading: " + string(info_map[? "is_downloading"]));
    draw_text(32, 45, "bytes_downloaded: " + string(info_map[? "bytes_downloaded"]));
    draw_text(32, 60, "bytes_total: " + string(info_map[? "bytes_total"]));
}

Приведенный выше код запросит статус загрузки элемента, проиндексированного в глобальной переменной "fileID", используя DS Map для хранения информации.
