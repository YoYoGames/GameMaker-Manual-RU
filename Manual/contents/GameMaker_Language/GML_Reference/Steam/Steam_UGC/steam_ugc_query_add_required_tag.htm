steam_ugc_query_add_required_tag

Эта функция может быть использована для дополнительной фильтрации любого заданного запроса пользовательского контента, в частности, для поиска только тех элементов пользовательского контента с заданным тегом. Ручка запроса - это значение, возвращенное при создании запроса (например, с помощью, steam_ugc_create_query_user()) а второй аргумент - строка, которую вы хотите использовать в качестве тега для включения. Функция вернет true, если фильтр запроса был установлен правильно, и false в противном случае.

Синтаксис:

steam_ugc_query_add_required_tag(ugc_query_handle , tag_name);

Аргумент Описание
ugc_query_handle Ручка запроса, которую следует использовать.
tag_name Имя тега для включения (строка).

Возвращается:

Boolean

Пример:

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByTrend, ugc_match_Items, 1);
steam_ugc_query_add_required_tag(query_handle, "RPG");
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

Приведенный выше код создает запрос запроса и сохраняет его хэндл в локальной переменной для дальнейшего использования в остальных функциях, которые определяют запрос перед отправкой запроса.
