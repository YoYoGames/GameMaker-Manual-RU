steam_send_screenshot

С помощью этой функции вы можете загрузить снимок экрана на страницу профиля Steam Community текущего зарегистрированного пользователя. Имя файла, которое вы указываете, является именем локального файла, который был создан при создании скриншота с помощью функций GameMaker Studio 2. экран_сохранить() или screen_save_part(). Ширина и высота определяют размер изображения, а функция вернет значение 0, если по какой-либо причине она не выполнится, и значение больше 0, если она выполнится успешно.

Синтаксис:

steam_send_screenshot(filename, width, height);

Аргумент Описание
filename Имя файла изображения для загрузки (строка).
width Ширина изображения (целое число).
height Высота изображения (целое число).

Возвращается:

Real

Пример:

if steam_is_screenshot_requested()
{
    var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
    screen_save(file)
    steam_send_screenshot(file, window_get_width(), window_get_height());
    global.scrn_num += 1;
}

Приведенный выше код будет опрашивать API Steam на предмет запроса скриншота, и если он был, то будет сгенерировано уникальное имя для файла изображения, сделан скриншот, и файл будет отправлен на страницу сообщества Steam для пользователя.
