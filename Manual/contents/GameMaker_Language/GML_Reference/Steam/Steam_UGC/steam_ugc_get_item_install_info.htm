steam_ugc_get_item_install_info

Эта функция может использоваться для получения информации о любом опубликованном файле, который был подписан и загружен в локальную область хранения Steam вашей игры. Вы задаете ID элемента и указываете индекс пустой карты DS, которая затем будет заполнена следующими парами ключ/значение, если элемент существует:

  • "size_on_disk" - Размер файла на диске (в байтах)
  • "legacy_item" - Будет true или false в зависимости от того, является ли этот файл устаревшим или нет.
  • "folder" - Полный путь к установленному содержимому (пожалуйста, обратитесь к разделу "Установка элементов" в документации Steam SDK, поскольку "старые" элементы, загруженные старым методом, обрабатываются по-другому).

Если элемент существует, то функция вернет true и заполнит карту, в противном случае она вернет false и карта останется пустой.

Синтаксис:

steam_ugc_get_item_install_info(published_file_id, info_map);

Аргумент Описание
published_file_id Уникальный идентификатор для обновляемого пользовательского центра.
info_map (ранее созданный) индекс карты DS.

Возвращается:

Boolean

Пример:

var item_map = ds_map_create();
steam_ugc_get_item_install_info(global.fileID, item_map);

Приведенный выше код запросит статус установки элемента, проиндексированного в глобальной переменной "fileID", используя DS Map для хранения информации.
