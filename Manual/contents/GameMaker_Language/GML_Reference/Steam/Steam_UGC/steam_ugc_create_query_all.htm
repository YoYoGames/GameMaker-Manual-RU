steam_ugc_create_query_all

Эта функция может быть использована для запроса базы данных UGC с использованием некоторых предварительно определенных типов запросов. Функция требует следующие константы для типа создаваемого запроса, типа элемента для запроса и номера страницы для запроса - обратите внимание, что запрос возвращает максимум 50 элементов.

Тип запроса Постоянный Описание
ugc_query_RankedByVote См. документацию Steam.
ugc_query_RankedByPublicationDate -
ugc_query_AcceptedForGameRankedByAcceptanceDate -
ugc_query_RankedByTrend -
ugc_query_FavoritedByFriendsRankedByPublicationDate -
ugc_query_CreatedByFriendsRankedByPublicationDate -
ugc_query_RankedByNumTimesReported -
ugc_query_CreatedByFollowedUsersRankedByPublicationDate -
ugc_query_NotYetRated -
ugc_query_RankedByTotalVotesAsc -
ugc_query_RankedByVotesUp -
ugc_query_RankedByTextSearch -

Тип соответствия Постоянная Описание
ugc_match_Items См. документацию Steam.
ugc_match_Items_Mtx -
ugc_match_Items_ReadyToUse -
ugc_match_Collections -
ugc_match_Artwork -
ugc_match_Videos -
ugc_match_Screenshots -
ugc_match_AllGuides -
ugc_match_WebGuides -
ugc_match_IntegratedGuides -
ugc_match_UsableInGame -
ugc_match_ControllerBindings -



Функция возвращает уникальное значение хэндла запроса, которое должно быть сохранено в переменной для использования в других функциях запроса. Обратите внимание, что эта функция только подготавливает запрос, но не отправляет его - для этого вы должны вызвать функцию steam_ugc_send_query() - и вы можете использовать другие функции steam_ugc_query_*() для уточнения поискового запроса перед его отправкой.

Синтаксис:

steam_ugc_create_query_all(query_type, match_type, page);

Аргумент Описание
query_type Тип создаваемого запроса (см. константы, перечисленные ниже).
match_type Тип элементов пользовательского контента для запроса (см. константы, перечисленные ниже).
page Номер страницы для запроса.

Возвращается:

Query Handle

Пример:

query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);

Приведенный выше код создает запрос общего запроса и сохраняет его обработку в переменной для дальнейшего использования.
