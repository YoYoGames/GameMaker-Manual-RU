steam_activate_overlay_store

С помощью этой функции вы можете открыть оверлей Steam на странице магазина для игры, чтобы пользователи могли купить или загрузить DLC (например). Вам необходимо предоставить уникальный App ID для игры или DLC, который вы получите из панели управления Steam при настройке.

Синтаксис:

steam_activate_overlay_store(app_id);

Аргумент Описание
app_id Уникальный идентификатор приложения для вашей игры.

Возвращается:

N/A

Пример:

if keyboard_check_pressed(vk_f1)
{
    steam_activate_overlay_store(global.DLC_id);
}

Приведенный выше код опрашивает клавиатуру на предмет нажатия клавиши F1, и если она нажата, то на странице открывается оверлей Steam для игрового контента, используя идентификатор приложения, сохраненный в глобальной переменной.
