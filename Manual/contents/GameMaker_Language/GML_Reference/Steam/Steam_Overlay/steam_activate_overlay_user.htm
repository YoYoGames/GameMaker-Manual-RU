steam_activate_overlay_user

Эта функция откроет оверлей Steam на один из выбранных диалогов, относящихся к указанному идентификатору пользователя. Доступны следующие возможные диалоговые строки:

  • "steamid" - открывает веб-браузер Steam Community на страницу пользователя или группы
  • "chat" - открыть окно чата для указанного пользователя

Обратите внимание, что идентификаторы Steam могут быть большими числами, поэтому вам может понадобиться привести значение вашего идентификатора к виду int64() перед тем, как передать его в функцию.

Синтаксис:

steam_activate_overlay_user(dialog_name, steamid);

Аргумент Описание
dialog_name Диалог, на котором нужно открыть наложение (см. ниже).
steamid Идентификатор пользователя Steam или идентификатор группы для использования.

Возвращается:

N/A

Пример:

var key = keyboard_lastkey;
switch (key)
{
    case vk_f1: steam_activate_overlay_user("steamid", global.GameGroupID); break;
    case vk_f2: steam_activate_overlay_user("chat", global.FriendID); break;
}

Приведенный выше код опрашивает последнюю нажатую клавишу клавиатуры, и если это функциональная клавиша 1 или функциональная клавиша 2, он откроет оверлей Steam для просмотра группы Steam, хранящейся в глобальной переменной "GamegroupID", или откроет окно чата для общения с пользователем, хранящимся в глобальной переменной "FriendID".
