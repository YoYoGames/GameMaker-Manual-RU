steam_is_overlay_activated

Эта функция позволяет узнать, активен ли у пользователя оверлей Steam или нет. Если оверлей активен и виден пользователю, функция вернет true, а если нет, то false. Примером использования этой функции может быть опрос API Steam на предмет наличия оверлея, чтобы можно было приостановить игру на время показа оверлея.

Синтаксис:

steam_is_overlay_activated();

Возвращается:

Boolean

Пример:

if steam_is_overlay_activated()
{
    global.Pause = true;
}

Приведенный выше код проверит, активен ли оверлей Steam, и если да, то установит глобальную переменную "Pause" в true.
