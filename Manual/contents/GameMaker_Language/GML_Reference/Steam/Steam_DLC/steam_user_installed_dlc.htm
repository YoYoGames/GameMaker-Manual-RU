steam_user_installed_dlc

Если в вашей игре есть DLC, созданные для нее, вы можете использовать эту функцию, чтобы проверить, установил ли пользователь это дополнение, прежде чем получить доступ к любым файлам, связанным с ним. Функция возвращает true, если у игрока установлен контент, и false, если нет, но обратите внимание, что пользователь также должен владеть DLC, поэтому вам следует использовать дополнительную функцию steam_user_owns_dlc() чтобы проверить, что он также является владельцем, прежде чем использовать эту функцию. Обратите внимание, что идентификаторы Steam могут быть большими числами, поэтому вам может понадобиться привести значение идентификатора к виду int64() перед передачей в функцию.

Синтаксис:

steam_user_installed_dlc(dlc_id);

Аргумент Описание
dlc_id Уникальный идентификатор для проверяемого DLC (целое число).

Возвращается:

Boolean

Пример:

global.Level_Max = 100;
if steam_user_owns_dlc(10354)
{
    if steam_user_installed_dlc(10354)
    {
        global.Level_max = 200;
    }
}

Приведенный выше код проверит, купил ли пользователь и установил DLC с идентификатором 10354, и если да, то установит глобальную переменную на другое значение.
