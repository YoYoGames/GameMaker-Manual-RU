steam_user_owns_dlc

Если в вашей игре есть DLC, созданные для нее, вы можете использовать эту функцию для проверки того, купил ли пользователь это содержимое, прежде чем получить доступ к любым связанным с ним файлам. Функция вернет true (1), если игрок владеет содержимым, false (0), если он не владеет им или заданный идентификатор DLC недействителен, или -1, если он не вошел в систему Steam.

Обратите внимание, что даже если пользователь владеет DLC, он может быть не установлен, поэтому следует дополнительно использовать функцию steam_user_installed_dlc() чтобы убедиться, что он установлен, прежде чем использовать его. Обратите внимание, что идентификаторы Steam могут быть большими числами, поэтому вам может потребоваться привести значение идентификатора к виду int64() перед передачей в функцию.

Синтаксис:

steam_user_owns_dlc(dlc_id);

Аргумент Описание
dlc_id Уникальный идентификатор для проверяемого DLC (целое число int64).

Возвращается:

Integer

Пример:

global.Level_Max = 100;
if steam_user_owns_dlc(10354)
{
    if steam_user_installed_dlc(10354)
    {
        global.Level_max = 200;
    }
}

Приведенный выше код проверит, купил ли пользователь и установил DLC с идентификатором 10354, и если да, то установит глобальную переменную на другое значение.
