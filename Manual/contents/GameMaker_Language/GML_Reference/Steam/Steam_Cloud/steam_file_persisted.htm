steam_file_persisted

С помощью этой функции вы можете проверить заданный файл на предмет его синхронизации с облаком Steam Cloud. Возвращаемое значение true означает, что да, а false - что нет.

Синтаксис:

steam_file_persisted(filename);

Аргумент Описание
filename Имя файла для проверки.

Возвращается:

Boolean

Пример:

if !steam_file_persisted("Save.txt")
{
    steam_file_share("Save.txt");
}

Приведенный выше код проверит, был ли файл сохранен на Steam Cloud, и если нет, то синхронизирует его.
