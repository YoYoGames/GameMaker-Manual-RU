steam_get_quota_total

При использовании облака Steam для хранения и синхронизации файлов вы должны установить квоту пространства, которое потребуется вашей игре. Эта квота устанавливается для каждой игры, поддерживаемой Cloud, по принципу "пользователь - игра". Например, если квота для игры X составляет 1 мегабайт, то каждая учетная запись Steam, владеющая игрой X, может хранить в Cloud не более 1 мегабайта данных, связанных с этой игрой. Любые другие игры с поддержкой Cloud, которыми владеет пользователь (например, игра Y), не будут затронуты данными, хранящимися в Game X. По умолчанию квота для новых игр Steamworks составляет один гигабайт, но вы можете изменить это значение в панели управления Steamworks для вашей игры.

ПРИМЕЧАНИЕ: После исчерпания квоты запись файлов будет прекращена. Если вы думаете, что квота может быть исчерпана для пользователя вашей игры, вы должны создать код для обработки этого, так как, ничего не делая, вы оставляете пользователей в ситуации, когда они не могут исправить ситуацию, что приведет к плохому восприятию вашей игры.

Синтаксис:

steam_get_quota_total();

Возвращается:

Real

Пример:

if steam_is_cloud_enabled_for_app()
{
    quota = steam_get_quota_total();
}

Приведенный выше код проверяет, включено ли облако steam для игры, и если да, то получает размер квоты хранения и сохраняет его в переменной.
