steam_file_share

С помощью этой функции вы можете заставить свою игру синхронизировать заданный файл с облаком Steam Cloud. Обычно в этом нет необходимости, так как игра автоматически синхронизируется в конце сеанса игрока, и это не рекомендуется Steam, но это может быть полезно для обеспечения немедленной синхронизации конфиденциальной информации. Функция возвращает значение 0 в случае неудачи по какой-либо причине и значение больше 0 в случае успеха.

Синтаксис:

steam_file_share(filename);

Аргумент Описание
filename Имя синхронизируемого файла.

Возвращается:

Real

Пример:

if !steam_file_persisted("Save.txt")
{
    steam_file_share("Save.txt");
}

Приведенный выше код проверит, был ли файл сохранен на Steam Cloud, и если нет, то синхронизирует его.
