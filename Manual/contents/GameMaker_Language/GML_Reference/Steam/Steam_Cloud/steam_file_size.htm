размер_файла

С помощью этой функции вы можете проверить размер файла, хранящегося на Steam Cloud. Возвращаемое вещественное число - это размер файла в байтах.

Синтаксис:

steam_file_size(filename);

Аргумент Описание
filename Имя файла, размер которого необходимо проверить.

Возвращается:

Real

Пример:

file_bytes = steam_file_size("Save.txt");

Приведенный выше код хранит размер файла из облака Steam Cloud в переменной "file_bytes".
