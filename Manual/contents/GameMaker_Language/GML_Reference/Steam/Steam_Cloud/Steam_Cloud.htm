Облако пара

Облако Steam предоставляет простую и прозрачную систему удаленного хранения файлов для вашей игры. Все файлы, записанные на диск с помощью функций облака GameMaker Studio 2 (перечисленных ниже), будут реплицированы на серверы Steam после выхода из игры. Если пользователь затем сменит компьютер, файлы будут загружены на новый компьютер до запуска игры, что означает, что игра сможет получить доступ к файлам, прочитав их с помощью соответствующих функций Steam. Клиент Steam выполняет работу по обеспечению синхронизации файлов на всех компьютерах, к которым обращается пользователь.

ПРИМЕЧАНИЕ По умолчанию Облако не включено для игры на Steamworks. Его необходимо предварительно включить на вкладке "Облако" в админке игры Steamworks, где необходимо установить квоту байтов и файлов. При следующей публикации конфигурации игры в Steamworks облачное хранилище будет готово к использованию.

Для доступа к облаку Steam Cloud из GameMaker Studio 2 можно использовать следующие функции

  • steam_is_cloud_enabled_for_app
  • steam_is_cloud_enabled_for_account
  • steam_get_quota_total
  • steam_get_quota_free
  • steam_file_exists
  • steam_file_size
  • steam_file_persisted
  • steam_file_write
  • steam_file_write_file
  • steam_file_read
  • steam_file_share
  • steam_file_delete
