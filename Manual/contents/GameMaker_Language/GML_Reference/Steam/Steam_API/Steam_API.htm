Общий Steam API

Следующий набор функций предназначен для проверки доступности определенных аспектов клиентского или серверного API Steam и, соответственно, возвращает либо true, либо false в зависимости от таких факторов, как занятость сервера, вход пользователя в систему и т.д.. Это означает, что эти функции следует использовать до вызова любой другой функции API Steam, чтобы убедиться, что клиент/сервер настроен правильно и взаимодействует с вашей игрой:

  • steam_initialised
  • steam_stats_ready
  • steam_get_app_id
  • steam_get_user_account_id
  • steam_get_user_steam_id
  • steam_get_persona_name
  • steam_get_user_persona_name
  • steam_is_user_logged_on
  • steam_current_game_language
  • steam_available_languages
