steam_get_persona_name

Вы можете использовать эту функцию, чтобы вернуть имя пользователя, который в настоящее время вошел в клиент Steam. Это видимое экранное имя, а не уникальный идентификатор пользователя (его можно найти с помощью функции steam_get_user_steam_id()).

Синтаксис:

steam_get_persona_name();

Возвращается:

String

Пример:

if steam_initialised()
{
    global.p_name = steam_get_persona_name();
}

Приведенный выше код установит глобальную переменную в текущее экранное имя пользователя, если клиентский API Steam правильно инициализирован.
