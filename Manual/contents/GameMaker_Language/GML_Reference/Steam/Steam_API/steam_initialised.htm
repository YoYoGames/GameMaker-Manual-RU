steam_initialised

При использовании API Steam эта функция может быть вызвана для проверки правильности инициализации клиентского API Steam перед выполнением дальнейших вызовов специфических функций Steam в вашей игре.

Синтаксис:

steam_initialised();

Возвращается:

Boolean

Пример:

global.steam_api = false;
if steam_initialised()
{
    if steam_stats_ready() && steam_is_overlay_enabled()
    {
        global.steam_api = true;
    }
}

Приведенный выше код установит глобальную переменную в true, если клиентский API Steam правильно инициализирован, вместе с функциями статистики и наложения Steam, или установит переменную в false в противном случае.
