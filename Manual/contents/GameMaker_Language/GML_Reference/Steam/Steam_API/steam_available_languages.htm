steam_available_languages

Эта функция может использоваться для получения списка всех доступных языков для Steam. Возвращаемое значение - это просто список языков, разделенных запятыми.

Синтаксис:

steam_available_languages();

Возвращается:

String

Пример:

language = steam_available_languages();

Вышеприведенная команда получает доступные языки для Steam в виде строки и сохраняет ее в переменной.
