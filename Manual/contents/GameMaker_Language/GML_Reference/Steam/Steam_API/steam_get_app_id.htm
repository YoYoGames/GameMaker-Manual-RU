steam_get_app_id

Эта функция используется для получения уникального идентификатора приложения, который Steam присваивает вашей игре, что необходимо для использования некоторых функций User Generated Content.

Синтаксис:

steam_get_app_id();

Возвращается:

Real

Пример:

global.app_id = steam_get_app_id();

Приведенный выше код получает уникальный идентификатор приложения для вашей игры на сайте Steam и сохраняет его в глобальной переменной.
