API Steam поддерживает постоянные таблицы лидеров с автоматически упорядоченными записями. Эти таблицы лидеров можно использовать для отображения глобальных и дружеских таблиц лидеров в вашей игре и на веб-странице сообщества вашей игры. Каждая игра может иметь до 10 000 таблиц лидеров, и каждая таблица лидеров может быть получена сразу после того, как в нее будет вставлен результат игрока, но обратите внимание, что в каждой таблице лидеров у игрока может быть только одна запись, хотя количество игроков в таблице лидеров не ограничено.

Каждая запись в таблице лидеров содержит имя, оценку и ранг лидера, и эти данные будут заменяться при создании новой записи в таблице лидеров для пользователя, а следующие функции можно использовать для добавления и извлечения этих данных из таблицы лидеров для вашей игры:

  • steam_create_leaderboard
  • steam_upload_score
  • steam_upload_score_buffer
  • steam_download_scores
  • steam_download_scores_around_user
  • steam_download_friends_scores
