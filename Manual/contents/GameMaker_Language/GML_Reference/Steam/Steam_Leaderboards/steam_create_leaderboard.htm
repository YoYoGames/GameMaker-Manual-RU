steam_create_leaderboard

С помощью этой функции вы можете создать новую таблицу лидеров для вашей игры. Первым аргументом является строка, определяющая имя вашей таблицы лидеров, и это имя должно использоваться во всех дальнейших вызовах функции, связанных с создаваемой таблицей лидеров. Вы можете затем определить метод сортировки (см. константы, перечисленные ниже), а также способ отображения информации (опять же, одна из констант, перечисленных ниже). Функция вернет значение -1, если ей не удалось создать таблицу лидеров, или значение больше -1, если это удалось.

ПРИМЕЧАНИЕ: Если вы ранее создали таблицу лидеров с таким же названием (либо через код, либо через вашу страницу Steam для игры), эта функция будет проигнорирована!

Сортировка Постоянная Описание
lb_sort_none Никакой сортировки. Информация будет отображаться "как есть".
lb_sort_ascending Сортировка таблицы лидеров в порядке возрастания.
lb_sort_descending Сортировка таблицы лидеров в порядке убывания.

Отображать постоянную Описание
lb_disp_none Показать таблицу лидеров "как есть".
lb_disp_numeric Показать таблицу лидеров в виде числового дисплея.
lb_disp_time_sec Показывать значения в таблице лидеров в виде времени, базовым значением которого являются секунды.
lb_disp_time_ms Показывать значения в таблице лидеров в виде времени, базовым значением которого являются миллисекунды.

Синтаксис:

steam_create_leaderboard(lb_name, sort_method, display_type);

Аргумент Описание
lb_name Название создаваемой доски лидеров (строка).
sort_method Метод сортировки записей в таблице лидеров (см. константы, перечисленные ниже).
display_type Способ отображения таблицы лидеров для пользователя (см. константы, перечисленные ниже).

Возвращается:

Real

Пример:

steam_create_leaderboard("Game Times", lb_sort_ascending, lb_disp_time_sec);

Приведенный выше код создаст таблицу лидеров под названием "Game Times" и настроит ее на отображение результатов в порядке возрастания и с отображением в секундах.
