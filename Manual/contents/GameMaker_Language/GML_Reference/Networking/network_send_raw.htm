network_send_raw

С помощью этой функции вы можете отправить "сырой" пакет данных по сети. Функция принимает идентификатор сокета для подключения, затем вы должны указать идентификатор буфера, содержащего данные для отправки (более подробную информацию о буферах см. в разделе Reference - Buffers), и, наконец, размер (в байтах) пакета данных. Отправляемые данные никак не форматируются GameMaker, и принимающие устройства получат их в виде потока, что означает, что вам придется обрабатывать их самостоятельно. Функция возвращает количество байт отправленных данных или число меньше 0, если отправка не удалась.

Аргумент опций

Последний аргумент является необязательным и используется только в WebSockets. Он позволяет выбрать между отправкой двоичных или текстовых данных. В этом аргументе может быть указана любая из этих констант:

=======

network_send_raw

This function sends a "raw" data packet through the network.

The function takes the Network Socket ID to connect through and then you must supply the Buffer which contains the data to be sent (for more information on buffers see Reference - Buffers) and finally the size (in bytes) of the data packet.

The data sent is not formatted by GameMaker in any way and the receiving devices will get the data as a stream which means you will have to handle it yourself. The function will return the number of bytes of data sent, or a number less than 0 if the send has failed.

 

Options Argument

The last argument is optional, and is only used with WebSockets. It allows you to choose between sending binary or text data. Either of these constants can be specified in this argument:
