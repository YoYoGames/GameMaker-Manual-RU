network_send_raw

С помощью этой функции вы можете отправить "сырой" пакет данных по сети. Функция принимает идентификатор сокета для подключения, затем вы должны указать идентификатор буфера, содержащего данные для отправки (более подробную информацию о буферах см. в разделе Reference - Buffers), и, наконец, размер (в байтах) пакета данных. Отправляемые данные никак не форматируются GameMaker, и принимающие устройства получат их в виде потока, что означает, что вам придется обрабатывать их самостоятельно. Функция возвращает количество байт отправленных данных или число меньше 0, если отправка не удалась.

Аргумент опций

Последний аргумент является необязательным и используется только в WebSockets. Он позволяет выбрать между отправкой двоичных или текстовых данных. В этом аргументе может быть указана любая из этих констант:

Тип сетевой отправки Постоянная
Постоянная Описание
network_send_binary Отправить двоичное сообщение
network_send_text Отправить текстовое сообщение

API некоторых платформ принимают только текстовые сообщения при использовании WebSockets (например, Twitch), поэтому в таких случаях можно использовать константу network_send_text. Если этот аргумент не указан, по умолчанию отправляются двоичные данные.

Синтаксис:

network_send_raw(socket, bufferid, size, [options]);

Аргумент Тип Описание
socket Идентификатор сетевого сокета The id of the socket to use.
bufferid Идентификатор буфера The id of the buffer to get the data from.
size Реально The size (in bytes) of the data.
options Тип сетевой отправки Константа OPTIONAL Используется для WebSockets для выбора между текстовыми и двоичными данными; если не указано, отправляются двоичные данные.

Возвращается:

Real

Пример:

buff = buffer_load("player_save.dat");
network_send_raw(sock, buff, buffer_get_size(buff));

Приведенная выше информация загружает ранее сохраненные данные буфера в память и возвращает идентификатор буфера для хранения в переменной "buff". Затем этот полный буфер отправляется в виде пакета необработанных данных по сети через сокет, идентифицируемый переменной "sock".
