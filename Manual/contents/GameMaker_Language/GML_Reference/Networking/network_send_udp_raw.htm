network_send_udp_raw

С помощью этой функции вы можете отправлять данные по сети с помощью UDP на сервер. Функция принимает идентификатор сокета для подключения, URL для подключения и порт для использования. Затем вы должны указать идентификатор буфера, который содержит данные для отправки (более подробную информацию о буферах см. в разделе Справочник - Буферы), и, наконец, размер (в байтах) данных. UDP является "бесконнектным" в том смысле, что вы фактически не выполняете соединение, вы просто посылаете пакет непосредственно на IP-адрес, а сервер получает входящие данные с IP-адреса и должен работать с ними "как есть". Отправляемые данные никак не форматируются GameMaker, и принимающие устройства будут получать данные в виде потока, что означает, что вам придется обрабатывать их самостоятельно. Функция возвращает количество байт отправленных данных или число меньше 0, если отправка не удалась.

ПРИМЕЧАНИЕ Эта функция не будет работать при использовании в проекте, работающем на HTML5-цели, и проекты HTML5 также не смогут принимать UDP.

Синтаксис:

network_send_udp_raw(socket, url, port, bufferid, size);

АргументТип Описание
socket The id of the socket to use.
url The url or IP to connect to (a string).
port The port to connect to.
bufferid The id of the buffer to get the data from.
size The size (in bytes) of the data.

Возвращается:

Real

Пример:

network_send_udp_raw(sock, "www.macsweeneygames.com", 6510, buff, buffer_tell(buff));

Приведенный выше код отправит необработанный UDP-пакет на сервер, заданный URL, по порту 6510. Данные берутся из буфера, индексированного в переменной "buff".
