network_connect_raw

С помощью этой функции вы можете отправить запрос на подключение к серверу. Функция принимает идентификатор сокета для подключения (см. network_create_socket()) и требует, чтобы вы указали IP-адрес для подключения (строка), а также порт для подключения, и в случае неудачи соединения будет возвращено значение меньше 0. Разница между этой функцией и network_connect() заключается в том, что эта функция может подключаться к любому серверу и ничего не делает с исходными данными, то есть вам придется самостоятельно реализовывать протоколы на стороне сервера. Обратите внимание, что по умолчанию функция работает синхронно, а это значит, что ваша игра может "зависнуть" при установлении соединения. Вы можете установить значение тайм-аута для соединения или сделать создание асинхронным, используя функцию network_set_config()или альтернативно использовать функцию network_connect_async_raw() вместо этого.

ПРИМЕЧАНИЕ Вы не можете использовать эту функцию в HTML5. Для WebSockets используйте функцию Async.

Синтаксис:

network_connect_raw(socket, url, port);

Аргумент Тип Описание
socket Идентификатор сетевого сокета The id of the socket to use.
url Строка The URL or IP to connect to (a string).
port Реально The port to connect to.

Возвращается:

Real or Network Socket ID

Пример:

client = network_create_socket(network_socket_tcp);
network_connect_raw(client, "www.macsweeneygames.com", 6510);

Приведенный выше код создаст новый TCP сокет и попытается создать через него "сырое" соединение с заданным URL на порту 6510.
