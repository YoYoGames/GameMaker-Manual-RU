network_connect_async

С помощью этой функции вы можете отправить запрос на подключение к серверу. Функция принимает идентификатор сокета для подключения (см. network_create_socket()) и требует, чтобы вы указали IP-адрес для подключения (строка), а также порт для подключения, и в случае неудачи соединения будет возвращено значение меньше 0. Соединение использует специальный протокол, который обеспечивает соединение только игр GameMaker, однако если вам необходимо подключиться к серверу, который не является игрой GameMaker, вы можете воспользоваться командой network_connect_async_raw(). Обратите внимание, что эта функция является асинхронной, генерируя событие Asynchronous Networking типа network_type_non_blocking_connect.

Синтаксис:

network_connect_async(socket, url, port);

АргументТип Описание
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

Возвращается:

Пример:

client = network_create_socket(network_socket_tcp);
network_connect_async(client, "192.134.0.1", 6510);

Приведенный выше код создаст новый TCP сокет и попытается установить через него асинхронное соединение с заданным IP адресом на порту 6510.
