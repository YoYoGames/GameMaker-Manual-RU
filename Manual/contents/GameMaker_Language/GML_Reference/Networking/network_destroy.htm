уничтожение сети

С помощью этой функции вы можете удалить подключение сетевого сокета из вашей игры.

ПРИМЕЧАНИЕ: Эта функция не будет работать при использовании в проекте, работающем на HTML5-цели.

Синтаксис:

network_destroy(socket);

АргументТип Описание
socket The id of the socket to remove.

Возвращается:

Пример:

if !os_is_network_connected()
{
    network_destroy(sock);
}

Приведенный выше код проверит, есть ли соединение для передачи данных, и если оно не найдено, уничтожит сокет с идентификатором "sock".
