network_set_timeout

С помощью этой функции вы можете установить тайм-аут для чтения и записи данных на сервер или с сервера через заданный сокет. Обратите внимание, что тайм-аут не генерирует никаких событий, поэтому вам придется самостоятельно работать с тайм-аутами, используя, например, сигналы тревоги. Обратите внимание, что это значение влияет только на отправку и получение данных, и если вы хотите изменить значение таймаута соединения, то вам следует использовать функцию network_set_config().

Синтаксис:

network_set_timeout(socket, read_timeout, write_timeout);

АргументТип Описание
socket The id of the socket to use.
read_timeout The milliseconds in which a transfer from a server will timeout.
write_timeout The milliseconds in which a transfer to a server will timeout.

Возвращается:

Пример:

network_set_timeout(sock, 3000, 3000);

Приведенный выше код установит тайм-аут для чтения и записи данных через сокет, индексированный в переменной "sock", равным 3 секундам.
