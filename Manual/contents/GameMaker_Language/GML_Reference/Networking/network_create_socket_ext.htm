network_create_socket_ext

Эта функция используется для создания нового клиентского сокета для вашей игры для общения по сети. Вы должны определить тип сокета (см. список констант ниже) и указать порт для использования, и функция вернет уникальный идентификатор, который должен использоваться во всех дальнейших вызовах функций для этого сокета, или значение меньше 0, если соединение не удалось.

Синтаксис:

network_create_socket_ext(protocol, port);

Аргумент Тип Описание
protocol Тип розетки Постоянная The network protocol to use
port Реально The port to use

Возвращается:

Network Socket ID

Пример:

client = network_create_socket_ext(network_socket_udp, 6510);

Приведенный выше код создаст новый UDP сокет на порту 6510.
