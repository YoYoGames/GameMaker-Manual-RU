network_create_server_raw

Эта функция используется для создания нового сетевого сервера для вашей игры, используя один из разрешенных протоколов подключения (см. константы, перечисленные ниже). Вы задаете тип сервера, затем указываете порт, который он будет использовать, и, наконец, количество максимальных соединений, которые должны быть разрешены в любой момент времени к серверу (обратите внимание, что это число зависит от вас, но слишком большое количество подключенных клиентов насытит сеть или процессор устройства не сможет справиться с обработкой такого количества игроков, поэтому используйте функцию с осторожностью). Функция возвращает уникальный id, который должен быть сохранен в переменной и использоваться для идентификации сервера во всех дальнейших сетевых функциях, или значение меньше 0, если соединение не удалось.

Поскольку это создает "сырой" сервер, он не будет принимать или использовать какие-либо встроенные заголовки данных GameMaker для связи, поэтому вы должны использовать функции network_send_raw() и network_send_udp_raw() для отправки неформатированных данных на созданный сервер.

Синтаксис:

network_create_server_raw(type, port, max_client);

АргументТип Описание
type The type of server to create (see the constants listed below).
port The port that the server will use.
max_client The maximum number of clients that can connect at once.

Возвращается:

Network Socket ID

Пример:

var port = 6510;
server = network_create_server_raw(network_socket_tcp, port, 32);
while (server < 0 && port < 65535)
{
    port++
    server = network_create_server(network_socket_tcp, port, 32);
}

Приведенный выше код попытается создать сервер, используя TCP через порт 6510. Если этот порт недоступен, он будет перебирать порты, чтобы найти тот, который доступен.
