network_create_socket

Эта функция используется для создания нового клиентского сокета для вашей игры для связи по сети. Вы должны определить тип сокета (см. список констант ниже), и функция вернет уникальный идентификатор для этого сокета, который должен использоваться во всех дальнейших вызовах функций для этого сокета, или значение меньше 0, если соединение не удалось.

СОВЕТ Вы можете использовать IP-адрес "127.0.0.1" для обратного подключения к тому же устройству, на котором запущена игра.

Синтаксис:

network_create_socket(type);

Аргумент Тип Описание
type Тип розетки Постоянная The type of socket connection to create (see the constants listed above).

Возвращается:

Network Socket ID

Пример:

if os_browser == browser_not_a_browser
{
    client = network_create_socket(network_socket_tcp);
    network_connect( client, "192.134.0.1", 6510 );
}
else
{
    client = network_create_socket(network_socket_ws);
    network_connect_raw_async( client, "192.134.0.1", 6520 );
}

Приведенный выше код проверит, запущена ли игра в браузере или нет, и создаст новый TCP или Web-сокет, после чего попытается подключиться через него к заданному IP-адресу на заданном порту.
