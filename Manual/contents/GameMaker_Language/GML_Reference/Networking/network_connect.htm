сетевой_коннект

С помощью этой функции вы можете отправить запрос на подключение к серверу. Функция принимает идентификатор сокета для подключения (см. network_create_socket()) и требует, чтобы вы указали IP-адрес для подключения (строка), а также порт для подключения. В случае неудачи соединения будет возвращено значение меньше 0. Соединение использует специальный протокол, который обеспечивает соединение только игр GameMaker, однако если вам нужно подключиться к серверу, который не является игрой GameMaker, вы можете воспользоваться командой network_connect_raw(). Обратите внимание, что по умолчанию функция является синхронной, а это означает, что ваша игра может "зависнуть" в момент установления соединения. Вы можете установить значение тайм-аута для соединения или сделать создание асинхронным, используя функцию network_set_config(), или же вместо нее использовать функцию network_connect_async().

ПРИМЕЧАНИЕ Вы не можете использовать эту функцию в HTML5. Для WebSockets используйте функцию Async.

Синтаксис:

network_connect(socket, url, port);

Аргумент Тип Описание
socket Идентификатор сетевого сокета The id of the socket to use.
url Строка The URL or IP to connect to (a string).
port Реально The port to connect to.

Возвращается:

Real or Network Socket ID

Пример:

client = network_create_socket(network_socket_tcp);
network_connect(client, "192.134.0.1", 6510);

Приведенный выше код создаст новый TCP сокет и попытается подключиться через него к заданному IP-адресу на порт 6510.
