network_send_packet

С помощью этой функции вы можете отправить "пакет" данных по сети. Функция принимает идентификатор сокета для подключения, затем вы должны указать идентификатор буфера, который содержит данные для отправки (более подробную информацию о буферах см. в разделе " Справочник - Буферы"), и, наконец, размер (в байтах) пакета данных. Пакеты, отправленные с помощью этой функции, будут отформатированы таким образом, чтобы игра GameMaker, получающая данные, могла правильно "разделить" пакеты, а функция вернет количество байт отправленных данных или число меньше 0, если отправка не удалась. Стоит отметить, что окончательный размер отправляемых данных, возвращаемый этой функцией, будет также включать информацию заголовка GameMaker, что составляет дополнительные 12 байт.

Синтаксис:

network_send_packet(socket, bufferid, size);

АргументТип Описание
socket The id of the socket to use.
bufferid The id of the buffer to get the data from.
size The size (in bytes) of the data.

Возвращается:

Real

Пример:

buff = buffer_load("player_save.dat");
network_send_packet(sock, buff, buffer_get_size(buff));

Приведенная выше информация загружает ранее сохраненные данные буфера в память и возвращает идентификатор буфера для хранения в переменной "buff". Затем этот полный буфер отправляется в виде пакета по сети через сокет, идентифицируемый переменной "sock".
