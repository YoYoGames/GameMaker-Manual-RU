GM_project_filename

Эта константа содержит полный путь и имя файла проекта YYP.

Синтаксис:

GM_project_filename;

Возвращается:

String

Пример:

draw_text(32, 32, GM_project_filename);

Приведенный выше код рисует значение константы GM project filename.
