os_is_network_connected

С помощью этой функции вы можете проверить, есть ли на вашем устройстве подключение к Интернету, и она вернет true, если есть, или false, если нет, и, в зависимости от ОС, может попытаться установить соединение, прежде чем вернуть значение.

Функция имеет дополнительный аргумент для включения/выключения попытки установить соединение при вызове, и эта опция в настоящее время используется только для цели Nintendo Switch. При установке значения true функция будет пытаться установить соединение, а при установке значения false она будет ограничиваться проверкой соединения. Обратите внимание, что попытка установить соединение может привести к зависанию ОС Switch на несколько секунд.

ПРИМЕЧАНИЕ Эта функция проверяет внутренний API устройства, который управляет соединениями, и поэтому может вернуть true, если есть соединение bluetooth, Wi-Fi или даже обычное сетевое соединение, позволяющее доступ в интернет.

Синтаксис:

os_is_network_connected([attempt_connection]);

Аргумент Тип Описание
attempt_connection Булево OPTIONAL Только для цели Nintendo Switch. Установите значение true для попытки подключения на уровне ОС при вызове.

Возвращается:

Boolean

Пример:

if os_is_network_connected()
{
    global.connected = true;
}

Приведенный выше код проверяет, есть ли у устройства подключение к Интернету, и если да, то устанавливает глобальную переменную.
