os_powersave_enable

С помощью этой функции можно включить или выключить функции энергосбережения устройства. Это важно, так как некоторые игры (например, использующие функции наклона) могут не генерировать события, которые ОС может интерпретировать как пользовательский ввод и таким образом выключить экран или выйти из игры. Установив эту функцию на false, вы можете отключить функции энергосбережения и обеспечить постоянную работу экрана (и игры).

ПРИМЕЧАНИЕ: Это ограничено целями iOS и Android.

Синтаксис:

os_powersave_enable(flag)

АргументТип Описание
flag Set to true или false, чтобы включить или отключить функции энергосбережения.

Возвращается:

Пример:

if (os_type == os_android) || (os_type == os_ios)
{
    os_powersave_enable(false);
}

Приведенный выше код проверяет тип ОС, и если это либо Android, либо iOS, то функции энергосбережения деактивируются.
