os_browser

Эта переменная, доступная только для чтения, содержит одну из различных констант (перечисленных ниже), которые нужны GameMaker для того, чтобы сообщить вам, в каком браузере вы в данный момент запускаете игру (если таковой имеется).

Синтаксис:

os_browser

Возвращается:

Browser Type Constant

Тип браузера Постоянная
Постоянная Описание
browser_not_a_browser Игра не воспроизводится в браузере
browser_unknown Неизвестный браузер
browser_ie Internet Explorer
browser_ie_mobile Internet Explorer на мобильном устройстве
browser_firefox Mozilla Firefox
browser_chrome Google Chrome
browser_safari Сафари
browser_safari_mobile Safari на мобильном устройстве
browser_opera Опера
browser_tizen Браузер для мобильных устройств Tizen
browser_windows_store Приложение для Windows

Пример:

if (os_browser == browser_not_a_browser)
{
    global.Config = 0;
}
else
{
    global.Config = 1;
}

Приведенный выше код проверяет, запущена ли игра в браузере или нет, и устанавливает значение глобальной переменной в зависимости от результата проверки.
