GM_runtime_version

Эта константа содержит номер версии среды выполнения, определенной в Preferences как среда выполнения, используемая для сборки проекта. Значение хранится в виде строки.

Синтаксис:

GM_runtime_version;

Возвращается:

String

Пример:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);
draw_text(32, 96, "Runtime " + GM_runtime_version);

Приведенный выше код принимает константы GM date, version и runtime и выводит их на экран.
