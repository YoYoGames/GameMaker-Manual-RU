os_lock_orientation

С помощью этой функции вы можете "заблокировать" ваше устройство в текущей ориентации до тех пор, пока вы не "освободите" его, чтобы снова разрешить все включенные в Game Options ориентации для данной целевой платформы. Обратите внимание, что перед блокировкой вы, скорее всего, захотите подтвердить желаемую ориентацию.

Синтаксис:

os_lock_orientation(flag)

АргументТип Описание
flag Set to true или false, чтобы включить или выключить блокировку ориентации.

Возвращается:

Пример:

if (os_type == os_android) || (os_type == os_ios)
{
    os_lock_orientation(true);
}

Приведенный выше код проверяет тип ОС, и если это либо Android, либо iOS, то блокировка ориентации отмечается как true.
