GM_build_date

Эта константа содержит дату и время, когда запускаемый исполняемый файл был создан GameMaker. Значение может быть разобрано с помощью функций Date и Time и берется из системного значения UTC во время компиляции.

Синтаксис:

GM_build_date;

Возвращается:

Datetime

Пример:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);

Приведенный выше код выведет номер версии игры, а также дату и время, когда она была скомпилирована.
