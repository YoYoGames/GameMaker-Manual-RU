environment_get_variable

Эта функция возвращает значение (строка) переменной окружения с заданным именем (также строка). Вы можете получить доступные переменные окружения на сайтах macOS и Ubuntu (Linux), набрав "env" в терминальном приложении, а для получения информацию о переменных окружения Windows, если вы используете командную строку, введите "echo %PATH%", а с помощью PowerShell - "ls env". Обратите внимание, что и на macOS, и на Ubuntu (Linux) переменная окружения "HOME" вернет путь "~/", который отображается на "/Users/<username>" на macOS и "/home/<username>" на Ubuntu (Linux).

ПРИМЕЧАНИЕ: Эта функция предназначена только для Windows, macOS и Ubuntu (Linux).

Синтаксис:

environment_get_variable(name);

АргументТип Описание
name The name (a string) of the environment variable to check.

Возвращается:

Пример:

e_str = environment_get_variable("APPDATA");

Приведенный выше код вернет полный путь для переменной окружения "%appdata%", которая обычно равна "C:\Users\{username}\AppData\Roaming".
