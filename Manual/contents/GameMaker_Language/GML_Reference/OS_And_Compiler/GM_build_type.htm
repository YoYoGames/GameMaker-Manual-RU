GM_build_type

Эта строковая константа хранит тип сборки: "exe" для созданного исполняемого файла или "run" для тестового запуска, запущенного с сайта IDE.

Синтаксис:

GM_build_type;

Возвращается:

String

Пример:

draw_text(32, 32, GM_build_type);

Приведенный выше код выведет тип сборки текущего запущенного исполняемого файла.
