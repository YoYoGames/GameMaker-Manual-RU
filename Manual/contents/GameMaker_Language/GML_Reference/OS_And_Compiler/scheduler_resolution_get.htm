scheduler_resolution_get

Эта функция используется для получения разрешения планировщика потоков Windows в миллисекундах. Если разрешение планировщика установлено в значение по умолчанию (как установлено Windows), функция вернет -1.

Информацию о планировщике потоков см. на странице scheduler_resolution_set().

ПРИМЕЧАНИЕ: Эта функция предназначена только для Windows .

Синтаксис:

scheduler_resolution_get();

Возвращается:

(or -1 for default)

Пример:

scheduler_resolution = scheduler_resolution_get();

Этот пример извлекает разрешение планировщика потоков Windows и сохраняет его в переменной "scheduler_resolution".
