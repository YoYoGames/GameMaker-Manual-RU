os_get_language

Эта функция возвращает строку с двухбуквенным кодом языка для ОС, на которой запущена игра, как установлено стандартом ISO639. Если эта информация недоступна, будет возвращена просто пустая строка "", или "en" для "английского" языка. Обратите внимание, что некоторые языки также имеют соответствующий региональный код, поэтому для различения разных регионов одной страны используйте функцию os_get_region().

В следующей таблице приведены примеры некоторых основных двухбуквенных кодов языков, определенных стандартом ISO 639:

ПРИМЕЧАНИЕ: Возвращается не код страны местоположения, а код языка ОС.

Синтаксис:

os_get_language()

Возвращается:

Пример:

switch (os_get_language())
{
    case "es": ini_open("spanish.ini"); break;
    case "fr": ini_open("french.ini"); break;
    case "it": ini_open("italian.ini"); break;
    default: ini_open("english.ini"); break;
}

Приведенный выше код проверяет язык ОС и открывает другой файл *.ini в зависимости от возвращаемого значения.
