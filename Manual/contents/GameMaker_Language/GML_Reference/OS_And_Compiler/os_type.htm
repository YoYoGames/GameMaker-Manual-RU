os_type

Эта переменная, доступная только для чтения, содержит одну из различных констант GameMaker, позволяющих определить, для какой операционной системы была создана игра. Обратите внимание, что это не обязательно совпадает с ОС устройства, на котором она запущена, поскольку, например, ваша игра может работать на ОС Amazon Fire, но быть создана для платформы Android (в этом случае os_type будет os_android).

Могут быть возвращены следующие константы:

Тип ОС Постоянная
Постоянная Описание
os_windows ОС Windows
os_uwp Платформа Windows 10 Universal Windows
os_operagx Opera GX
os_linux Linux
os_macosx macOS X
os_ios iOS (iPhone, iPad, iPod Touch)
os_tvos Apple tvOS
os_android Android
os_ps4 Sony PlayStation 4
os_ps5 Sony PlayStation 5
os_xboxone Microsoft Xbox One
os_xboxseriesxs Microsoft Xbox Series X/S
os_switch Nintendo Switch
os_unknown Неизвестная ОС

Синтаксис:

os_type

Возвращается:

OS Type Constant

Пример:

switch (os_type)
{
    case os_windows: global.Config = 0; break;
    case os_android: global.Config = 1; break;
    case os_linux: global.Config = 2; break;
    case os_macosx: global.Config = 3; break;
    case os_ios: global.Config = 4; break;
    case os_winphone: global.Config = 5; break;
}

Приведенный выше код проверяет ОС, на которой запущена игра, и соответствующим образом устанавливает глобальную переменную.
