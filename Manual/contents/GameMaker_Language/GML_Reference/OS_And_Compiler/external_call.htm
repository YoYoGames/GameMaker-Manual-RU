внешний_вызов

Если вы создали вызов внешней функции dll или dylib с помощью функции external_define()то вы можете использовать эту функцию для последующего вызова. Вы передаете имя ранее определенной функции, а также каждый из аргументов, которые она требует (каждый аргумент должен быть правильного типа, либо вещественным, либо строковым), и функция возвращает результат внешнего вызова.

Синтаксис:

external_call(id, args[0...15]);

Аргумент Тип Описание
id Внешняя функция The name of the function that you want to call
args[0...10] Вещественный или строка The different arguments that you want to pass through to the external function

Возвращается:

Variable (the type of value returned will depend on the defined function)

Пример:

my_function = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(my_function, x, y);

Приведенный выше пример кода вызывает ранее определенную внешнюю функцию и сохраняет возвращаемое значение в локальной переменной.
