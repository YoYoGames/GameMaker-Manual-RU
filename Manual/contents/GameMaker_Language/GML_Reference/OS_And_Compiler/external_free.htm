внешний_свободный

Эта функция освобождает память, связанную с файлом dll или dylib с заданным именем. Это следует делать всякий раз, когда данный файл больше не нужен в игре, обычно (например, в событии Game End).

Синтаксис:

external_free(id);

Аргумент Тип Описание
id Внешняя функция The name of the dll или dylib, которые вы хотите освободить

Возвращается:

N/A

Пример:

external_free("MyDLL.dll");

Приведенный выше код примера освободит память, связанную с заданной dll.
