gc_get_target_frame_time

С помощью этой функции вы можете получить текущее значение целевого кадра для сборщика мусора. Возвращаемое значение выражается в микросекундах (где 1 000 000 микросекунд равны одной секунде), а время целевого кадра по умолчанию составляет 100 микросекунд. Если вы хотите изменить это значение, то вам следует воспользоваться функцией gc_target_frame_time().

Синтаксис:

gc_get_target_frame_time(time);

Возвращается:

Пример:

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

Приведенный выше код проверяет текущее значение времени кадра для сборщика мусора, и если оно не равно 50 микросекундам, то оно устанавливается на это значение.
