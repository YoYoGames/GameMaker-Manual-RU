gc_enable

С помощью этой функции можно включить или выключить сборщик мусора. Вызов функции с аргументом true включает его, а с помощью false отключает (не рекомендуется). По умолчанию он включен.

Синтаксис:

gc_enable(enable);

АргументТип Описание
enable enable (true) или отключить (false) сборщик мусора.

Возвращается:

Пример:

if (global.debug == true)
{
    gc_enable(false);
}

Приведенный выше код отключает сборку мусора, если данная глобальная переменная имеет значение true.
