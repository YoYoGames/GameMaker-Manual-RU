gc_collect

С помощью этой функции вы можете запустить сборщик мусора, заставив его работать в конце текущего кадра (шага). Стоит отметить, что для этого сборщик мусора не должен быть активным. Вызов этой функции после отключения сборщика мусора (с помощью функции gc_enable()) включит сборщик мусора на один кадр, в котором все объекты, помеченные для сбора, будут удалены из памяти, прежде чем сборщик мусора будет снова отключен.

Синтаксис:

gc_collect();

Возвращается:

Пример:

if (global.debug == true && keyboard_check_pressed(vk_f1))
{
    gc_collect();
}

Приведенный выше код проверяет глобальную переменную и нажатую клавишу, и если это true, то в конце кадра (шага) запускается сборка мусора.
