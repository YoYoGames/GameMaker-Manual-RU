weak_ref_create

С помощью этой функции вы можете создать слабую ссылку на структуру, которая затем может быть использована для проверки того, жива ли эта структура (ссылка на нее) или нет в игре. Вы передаете ссылку на структуру, которую хотите отследить, а функция возвращает другую структуру, которая является слабой ссылкой на эту структуру. Обратите внимание, что вы можете проверить, является ли ссылка "сильной" или "слабой", используя функцию instanceof(), так как сильная ссылка вернет либо "struct" или имя функции-конструктора, создавшей структуру, либо "weakref", если это слабая ссылка. Также обратите внимание, что структура со слабой ссылкой будет иметь переменную "ref", к которой можно обратиться, чтобы получить сильную ссылку на данную структуру, если только она не была собрана в мусор, в этом случае она вернет undefined.

Синтаксис:

weak_ref_create(struct_to_track);

Аргумент Тип Описание
struct_to_track Структура The struct that you want to create a weak reference for

Возвращается:

Struct Weak Reference

Пример:

inventory_ref = weak_ref_create(inventory);

Приведенный выше код создает слабую ссылку на struct и сохраняет ее в переменной экземпляра для последующего использования.
