weak_ref_any_alive

С помощью этой функции вы можете проверить слабые ссылки на различные структуры, чтобы узнать, живы они еще или нет. Вы передаете массив слабых ссылок для проверки (каждая слабая ссылка должна быть создана с помощью функции weak_ref_create()), и функция вернет true, если любой если на эти структуры все еще где-то ссылаются, или false, если это не так и они были собраны в мусор. Обратите внимание, что если вы предоставите массив, в котором любое из значений не является слабой ссылкой, функция вернет undefined.

Обратите внимание, что функция также принимает два необязательных аргумента, первый из которых позволяет указать начальный индекс массива для проверки, а также аргумент, задающий количество позиций (длину) от этого индекса для проверки. Если они указаны, то будут проверяться только индексы массива в пределах этих параметров.

Синтаксис:

weak_ref_any_alive(array, [index], [length]);

Аргумент Тип Описание
array Массив слабых ссылок структур Array containing weak references to the structs that you want to check.
[index] Реально OPTIONAL Индекс в массиве, с которого следует начать проверку
[length] Реально OPTIONAL Количество позиций, начиная с заданного индекса, для проверки на наличие

Возвращается:

Boolean (or undefined)

Пример:

if weak_ref_any_alive(inventory_ref_array)
{
    instance_destroy(obj_Inventory_Control);
}

Приведенный выше код проверяет массив слабых ссылок, и если какая-либо из них еще жива, то экземпляр уничтожается.
