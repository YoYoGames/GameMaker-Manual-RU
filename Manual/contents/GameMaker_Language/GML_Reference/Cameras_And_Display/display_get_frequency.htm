display_get_frequency

Эта функция возвращает частоту (или частоту обновления) дисплея, на котором идет игра. Она возвращает реальное значение в виде кадров в секунду, поэтому, например, если ваш монитор работает на частоте 60 Гц, вы получите 60, если на частоте 144 Гц - 144 и так далее.

ПРИМЕЧАНИЕ Эта функция не поддерживается в HTML5.

Синтаксис:

display_get_frequency();

Возвращается:

Пример:

var _freq = display_get_frequency();

if (_freq >= 240)
{
    game_set_speed(240, gamespeed_fps);
}
else if (_freq >= 120)
{
    game_set_speed(120, gamespeed_fps);
}
else if (_freq >= 60)
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

Приведенный выше код получает частоту дисплея и выполняет некоторые условия, чтобы установить игру на 240, 120, 60 или 30 кадров в секунду. Это означает, что если частота дисплея 90 Гц, то игра будет работать со скоростью 60 кадров в секунду, если 144 Гц, то со скоростью 120 кадров в секунду и т. д. Конечно, вы можете передать частоту дисплея непосредственно в game_set_speed(), чтобы она использовалась в качестве частоты кадров игры.
