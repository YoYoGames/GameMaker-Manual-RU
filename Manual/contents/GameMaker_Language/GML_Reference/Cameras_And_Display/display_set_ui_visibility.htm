display_set_ui_visibility

Эта функция может использоваться для отображения или скрытия системного пользовательского интерфейса только на Android. Функция требует, чтобы вы предоставили один или несколько системных флагов в виде целочисленного значения. Если используется более одного флага, их необходимо объединить с помощью побитового "или", как показано в примере ниже. Список системных флагов Android можно найти здесь.

Синтаксис:

display_set_ui_visibility(flags);

Аргумент Тип флаги
flags Реально The Android SYSTEM flags to use (as an integer value).

Возвращается:

N/A

Пример:

var flags = 1024|4096;
display_set_ui_visibility(flags);

Приведенный выше код будет использовать системные флаги Android "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" и "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN" для установки видимости дисплея.
