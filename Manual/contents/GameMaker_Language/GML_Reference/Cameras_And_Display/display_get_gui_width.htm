display_get_gui_width

С помощью этой функции вы можете получить ширину (в пикселях) графического интерфейса, используемую в событии Draw GUI Event.

Синтаксис:

display_get_gui_width();

Возвращается:

Пример:

ads_move(display_get_gui_width() - ads_get_display_width(0), 0, 0);

Приведенный выше код установит рекламу для показа в правом верхнем углу дисплея.
