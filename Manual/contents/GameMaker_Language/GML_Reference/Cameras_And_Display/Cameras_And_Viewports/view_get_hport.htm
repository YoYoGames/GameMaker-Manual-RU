view_get_hport

Эта функция может быть использована для получения высоты заданного порта просмотра.

Синтаксис:

view_get_hport(view_port)

АргументТип Описание
view_port The view port to target (0 - 7)

Возвращается:

Real

Пример:

if view_get_hport(0) != (display_get_height () / 2)
{
    view_set_hport(0, display_get_height() / 2);
}

Приведенный выше код проверит высоту порта просмотра[0], и если она не равна половине высоты дисплея, то она будет установлена на это значение.
