camera_set_end_script

Эта функция может быть использована для задания функции сценария, которая будет вызываться в конце каждого игрового кадра, когда камера назначена на видимый и активный порт представления, после того, как все для этой камеры представления будет отрисовано. Вы задаете уникальное значение ID камеры (возвращаемое различными camera_create() функции) и ID функции, которая должна быть вызвана. Порядок вызова функций, прикрепленных к камерам, и фактического рендеринга вида камеры следующий:

  • Камеры для всех видимых и активных портов просмотра имеют функцию обновления , вызываемую как
  • Затем для каждого вида:
    • Вызывается функция begin для камеры для этого представления
    • Для этого вида выполняются события рисования
    • Вызывается функция end для камеры
  • Перейдите к следующему активному видимому виду и повторите...

Синтаксис:

camera_set_end_script(camera_id, script)

Аргумент Тип Описание
camera_id Идентификатор камеры The unique camera ID value returned when you created the camera.
script Функция сценария The script to run each game frame

Возвращается:

N/A

Пример:

camera_set_end_script(view_camera[0], endCamera);

Приведенный выше код устанавливает конечный сценарий для камеры, назначенной на порт просмотра [0].
