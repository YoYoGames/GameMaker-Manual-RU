camera_set_view_mat

Эта функция устанавливает матрицу просмотра для данной камеры. Вы передаете уникальное значение ID камеры (возвращаемое различными camera_create() функции) и матрицу вида, которую необходимо применить. Подробнее о создании матриц вида вы можете узнать из раздела Функции матрицы, а именно matrix_build_lookat().

Обратите внимание, что если ваша камера выполняет автоматическое отслеживание объектов - т.е. она была создана с помощью camera_create_view() с индексом объекта / ID экземпляра, который не равен -1, или вы устанавливаете камеру, определенную в редакторе комнат, или вы устанавливаете камеру по умолчанию - тогда эта матрица будет перезаписана на следующем игровом кадре.

Синтаксис:

camera_set_view_mat(camera_id, matrix)

АргументТип Описание
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

Возвращается:

Пример:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Приведенный выше код создает новую камеру и назначает ее порту вида[0]. Затем у этой камеры устанавливаются матрицы вида и проекции.
