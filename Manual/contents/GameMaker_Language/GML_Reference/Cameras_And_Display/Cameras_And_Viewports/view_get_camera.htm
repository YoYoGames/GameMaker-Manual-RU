view_get_camera

Эта функция может использоваться для получения уникального значения ID камеры для камеры, назначенной на данный порт просмотра (от 0 до 7). Если камера не назначена, функция возвращает -1.

Синтаксис:

view_get_camera(view_port)

АргументТип Описание
view_port The view port to target (0 - 7)

Возвращается:

Camera ID

Пример:

var cam = view_get_camera(0);
var cw = camera_get_view_width(cam);
var ch = camera_get_view_height(cam);
camera_set_view_pos(cam, mouse_x - (cw / 2), mouse_y - (ch / 2));

Приведенный выше код получит идентификатор камеры для камеры, назначенной на порт просмотра[0], а затем изменит ее положение.
