camera_get_view_height

Эта функция может использоваться для получения высоты (в пикселях) заданной камеры. Обратите внимание, что эта функция действительна только для камер, созданных с помощью функции camera_create_view() или для камер, добавленных в редакторе редактора.

Синтаксис:

camera_get_view_height(camera_id)

АргументТип Описание
camera_id The unique camera ID value returned when you created the camera

Возвращается:

Пример:

var vw = camera_get_view_width(view_camera[0]) / 2;
var vh = camera_get_view_height(view_camera[0]) / 2;
camera_set_view_pos(view_camera[0], x - vw, y - vh);

Приведенный выше код извлекает ширину и высоту камеры, назначенной на view port[0], а затем устанавливает ее положение относительно центра.
