view_set_yport

Эта функция используется для установки позиции y заданного порта просмотра. Вы указываете индекс порта просмотра (от 0 до 7) и новое положение, в которое его нужно поместить.

Синтаксис:

view_set_yport(view_port, y)

АргументТип Описание
view_port The view port to target (0 - 7)
y The new y position

Возвращается:

N/A

Пример:

if view_get_yport(0) != (display_get_height() / 2) - (view_hport[0] / 2)
{
    view_set_yport(0, (display_get_height() / 2) - (view_hport[0] / 2));
}

Приведенный выше код проверит положение y порта просмотра[0], и если оно не там, где нужно, оно будет установлено в это положение.
