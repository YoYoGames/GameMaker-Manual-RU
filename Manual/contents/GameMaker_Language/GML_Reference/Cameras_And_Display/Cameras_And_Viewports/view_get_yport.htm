view_get_yport

Эта функция может быть использована для получения позиции y заданного порта просмотра.

Синтаксис:

view_get_yport(view_port)

АргументТип Описание
view_port The view port to target (0 - 7)

Возвращается:

Real

Пример:

if view_get_yport(0) != (display_get_height() / 2) - (view_hport[0] / 2)
{
    view_set_yport(0, (display_get_height() / 2) - (view_hport[0] / 2));
}

Приведенный выше код проверит положение y порта просмотра[0], и если оно не там, где нужно, то оно будет установлено в это положение.
