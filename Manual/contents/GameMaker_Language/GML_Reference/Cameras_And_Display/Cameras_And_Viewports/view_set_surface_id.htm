view_set_surface_id

С помощью этой переменной вы можете задать содержимое порта просмотра для рисования на поверхности. При работе с поверхностями часто требуется захватить всю видимую область экрана на поверхность, и поэтому вы назначаете ее порту просмотра с помощью этой функции. Это означает, что все, что отображается в выбранном порте просмотра, теперь будет рисоваться на назначенной поверхности, а содержимое этого порта просмотра больше не будет отображаться, что означает, что вам нужно будет либо:

  • Включите новый вид и рисуйте поверхность только в этом виде (используя view_current для проверки того, какой вид рисуется)
  • Нарисуйте поверхность в событии Draw GUI или Post Draw экземпляра, поскольку эти события не зависят от представлений.

При использовании этой функции вы указываете индекс порта вида (от 0 до 7) и индекс поверхности (либо индекс поверхности application_surface либо уникальное значение индекса, возвращаемое функцией surface_create()) или, если поверхность была назначена ранее и вы хотите ее удалить, значение -1. Для получения дополнительных примеров установки порта просмотра на поверхность смотрите переменную view_surface_id.

Синтаксис:

view_set_surface_id(view_port, surf)

Аргумент Тип Описание
view_port Реально The view port to target (0 - 7)
surf Идентификатор поверхности The surface to apply to the view

Возвращается:

Real

Пример:

if view_get_surface_id(0) == -1
{
    view_set_surface_id(0, global.vSurf);
}

Приведенный выше код проверит, была ли поверхность назначена порту вида[0], и если нет, то она будет назначена.
