camera_set_view_target

Вы можете использовать эту функцию для установки цели следования камеры обзора в помещении. Вы указываете уникальное значение ID камеры (возвращаемое различными camera_create() а затем укажите экземпляр цели или ID объекта, на который вы хотите установить вид камеры. Обратите внимание, что если вы выбираете ID объекта, а в комнате находится более одного экземпляра этого объекта, то GameMaker никак не может узнать, за каким экземпляром вы хотите следить, поэтому это может быть любой из них.

Синтаксис:

camera_set_view_target(camera_id, instance_id/object_id)

Аргумент Тип Описание
camera_id Идентификатор камеры The unique camera ID value returned when you created the camera
instance_id/object_id Идентификаторактива или экземпляраобъекта Instance or object to have the camera target for following

Возвращается:

N/A

Пример:

camera_set_view_target(view_camera[0], id);

Приведенный выше код установит цель камеры просмотра для камеры, назначенной порту просмотра[0], на цель экземпляра, выполняющего этот код.
