camera_get_view_y

Эта функция может быть использована для получения положения вида по оси y для данной камеры. Обратите внимание, что эта функция действительна только для камер, созданных с помощью функции camera_create_view() или для камер, добавленных в редакторе помещений.

Синтаксис:

camera_get_view_y(camera_id)

АргументТип Описание
camera_id The unique camera ID value returned when you created the camera

Возвращается:

Пример:

var vx = camera_get_view_x(view_camera[0]);
var vy = camera_get_view_y(view_camera[0]);
draw_text(vx + 5, vy + 5, "SCORE" + string(score));

Приведенный выше код извлекает позицию камеры, назначенной порту view port[0], а затем рисует текст относительно этой позиции.
