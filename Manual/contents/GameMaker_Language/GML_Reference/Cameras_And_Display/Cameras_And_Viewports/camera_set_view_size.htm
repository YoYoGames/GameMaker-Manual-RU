camera_set_view_size

Вы можете использовать эту функцию для обновления размера камеры обзора в помещении. Вы указываете уникальное значение ID камеры (как возвращается различными camera_create() функциями), а затем указываете ширину и высоту (в пикселях) для установки камеры.

Синтаксис:

camera_set_view_size(camera_id, width, height)

АргументТип Описание
camera_id The unique camera ID value returned when you created the camera.
width The width of the camera view in pixels
height The height of the camera view in pixels

Возвращается:

Пример:

camera_set_view_size(view_camera[0], view_wport[0], view_hport[0]);

Приведенный выше код установит размер камеры просмотра для камеры, назначенной на порт просмотра[0].
