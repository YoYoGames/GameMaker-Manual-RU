view_set_visible

Эта функция может использоваться для установки видимости заданного порта представления. Функция принимает индекс порта представления (от 0 до 7) и булево значение true, если он виден, и false, если нет.

Синтаксис:

view_set_visible(view_port, visible)

АргументТип Описание
view_port The view port to target (0 - 7)
visible Visibility toggle (true является видимым, а false - невидимым)

Возвращается:

N/A

Пример:

if !view_get_visible(0)
{
    view_set_visible(0, true);
}

Приведенный выше код проверит, является ли порт представления[0] видимым, и если нет, то он будет установлен в видимый.
