camera_get_view_speed_y

Эта функция может использоваться для получения скорости движения заданной камеры вдоль оси y (вертикальное движение). Возвращаемое значение будет в пикселях на игровой кадр.

Синтаксис:

camera_get_view_speed_y(camera_id)

АргументТип Описание
camera_id The unique camera ID value returned when you created the camera

Возвращается:

Пример:

var xs = camera_get_view_speed_x(view_camera[0]);
var ys = camera_get_view_speed_y(view_camera[0]);
if xs != 5 || ys != 5
{
    camera_set_view_speed(view_camera[0], 5, 5);
}

Приведенный выше код извлекает xspeed и yspeed камеры, назначенной порту view[0], а затем проверяет, совпадают ли они с заданным значением. Если нет, то камера просмотра устанавливается на эту скорость.
