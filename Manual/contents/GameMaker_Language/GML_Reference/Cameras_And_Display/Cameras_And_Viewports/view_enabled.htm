view_enabled

Эта переменная контролирует, включены или нет порты обзора, видимые в комнате. Если для портов просмотра установлено значение видимый, а затем отключить этот параметр, вся комната будет отрисована на экране с масштабированием по размеру окна вместо того, чтобы различные камеры были отрисованы через порты просмотра.

Синтаксис:

view_enabled;

Возвращается:

(true: enabled, false: disabled)

Пример:

if !view_enabled
{
    view_visible[0] = true;
    view_enabled = true;
}

Приведенный выше код проверяет, включены ли представления для комнаты, и если нет, то убеждается, что порт представления[0] виден, а затем включает представления для комнаты.
