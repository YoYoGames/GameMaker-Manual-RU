camera_get_view_mat

Эта функция может быть использована для получения матрицы вида. Функция возвращает матричный массив, который затем может быть использован в других матричных функциях или для установки матрицы вида другой камеры (с помощью функции camera_set_view_mat()). Обратите внимание, что эта функция обычно используется для камер, созданных с помощью функции camera_create(), но она также может быть использована для камер, созданных в редакторе помещений (или с помощью функции camera_create_view()) - в этом случае возвращаемая матрица будет действительна только после первого использования данной камеры в порте просмотра.

Синтаксис:

camera_get_view_mat(camera_id)

АргументТип Описание
camera_id The unique camera ID value returned when you created the camera

Возвращается:

Matrix Array

Пример:

mat = camera_get_view_mat(camera_view[0]);

Приведенный выше код получает матрицу вида для камеры, назначенной на порт вида[0].
