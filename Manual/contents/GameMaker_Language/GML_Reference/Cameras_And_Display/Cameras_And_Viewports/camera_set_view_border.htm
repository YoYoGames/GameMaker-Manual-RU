camera_set_view_border

Вы можете использовать эту функцию для установки границы следования камеры обзора в помещении. Вы задаете уникальное значение ID камеры (возвращаемое различными функциями camera_create ), а затем задаете размер границы x и размер границы y (горизонтальный и вертикальный). Эти значения будут использоваться только в том случае, если камере просмотра назначен объект для следования (либо в редакторе комнат, либо с помощью функции camera_set_view_target()) и относятся к тому, насколько далеко от границы вида должен находиться объект, прежде чем вид обновит свою позицию для следования.

Синтаксис:

camera_set_view_border(camera_id, x_border, y_border)

АргументТип Описание
camera_id The unique camera ID value returned when you created the camera.
x_border The border (in pixels) for the view camera along the horizontal axis
y_border The border (in pixels) for the view camera along the vertical axis

Возвращается:

Пример:

camera_set_view_border(view_camera[0], 64, 64);

Приведенный выше код установит границу камеры просмотра для камеры, назначенной порту просмотра[0].
