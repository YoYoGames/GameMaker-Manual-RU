камера_применить

Эта функция немедленно применит заданные настройки камеры к текущему виду камеры, который рендерится. Хотя вы можете установить любой вид камеры с помощью специальных функций camera_set_*, новые настройки не будут использоваться для рендеринга до следующего кадра игры, но с помощью этой функции вы можете сделать эти изменения немедленно. Если вы используете несколько видов, то функция должна быть использована в событии Draw Event, и вы можете использовать переменную view_current для выборочного применения заданной камеры только к тому виду камеры, который рендерится в данный момент. Обратите внимание, что при этом НЕ изменяется значение ID камеры, а только настройки для этой камеры, и что если вы используете эту функцию для камеры по умолчанию (т.е. для внутренней камеры, которую GameMaker использует, когда ни одна камера вида не активна), то изменения будут перезаписаны на следующем игровом кадре внутренними настройками для рисования комнаты.

Синтаксис:

camera_apply(camera_id)

АргументТип Описание
camera_id The unique camera ID value returned when you created the camera.

Возвращается:

Пример:

if view_current == 0
{
    camera_apply(cutscene_cam);
}

Приведенный выше код (в событии Draw Event) проверяет, какой вид камеры в данный момент отображается, и если это вид камеры[0], то к нему применяются настройки камеры, на которую ссылается переменная "cutscene_cam".
