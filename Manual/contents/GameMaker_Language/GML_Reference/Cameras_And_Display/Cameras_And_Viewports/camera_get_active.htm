camera_get_active

Эта функция может быть использована для получения уникального значения ID камеры для активной в данный момент камеры.

Синтаксис:

camera_get_active()

Возвращается:

Camera ID

Пример:

var active = camera_get_active();
if active != view_camera[0]
{
    view_camera[0] = active;
}

Приведенный выше код получает идентификатор камеры для активной камеры и устанавливает камеру просмотра для порта[0], чтобы использовать ее.
