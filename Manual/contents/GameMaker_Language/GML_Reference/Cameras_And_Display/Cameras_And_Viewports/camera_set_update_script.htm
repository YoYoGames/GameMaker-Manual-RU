camera_set_update_script

Эта функция может быть использована для задания функции сценария, которая будет вызываться каждый игровой кадр, когда камера назначена на видимый и активный порт просмотра. Вы указываете уникальное значение ID камеры (возвращаемое различными функциями camera_create ) и ID функции, которая будет вызываться. Порядок вызова функций, прикрепленных к камерам, и фактического рендеринга вида камеры следующий:

  • Камеры для всех видимых и активных портов просмотра имеют функцию обновления , вызываемую как
  • Затем для каждого вида:
    • Вызывается функция begin для камеры для этого вида
    • Для этого вида выполняются события рисования
    • Вызывается функция end для камеры
  • Перейдите к следующему активному видимому виду и повторите...

Применение пользовательской функции обновления к камере переопределяет ее поведение обновления по умолчанию. Например, если вы настроили камеру на следование за объектом (например, obj_player) при вызове camera_create_view(), или задали это в свойствах комнаты, она перестанет следовать за этим объектом после установки пользовательского сценария обновления.

Синтаксис:

camera_set_update_script(camera_id, script)

Аргумент Тип Описание
camera_id Идентификатор камеры The unique camera ID value returned when you created the camera.
script Функция сценария The script function to run each game frame

Возвращается:

N/A

Пример:

camera_set_update_script(view_camera[0], updateCamera);

Приведенный выше код устанавливает функцию обновления для камеры, назначенной на порт просмотра [0].
