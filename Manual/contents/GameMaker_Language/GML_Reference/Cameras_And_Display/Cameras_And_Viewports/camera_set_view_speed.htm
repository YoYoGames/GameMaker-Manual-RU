camera_set_view_speed

Вы можете использовать эту функцию для обновления скорости камеры обзора в помещении. Вы задаете уникальное значение ID камеры (как возвращается различными camera_create() функциями), а затем задаете x и y (горизонтальную и вертикальную) скорость, с которой она должна двигаться, если установлена для следования за данным экземпляром. Скорость рассчитывается как пиксели за шаг и может быть установлена на "-1", чтобы камера двигалась мгновенно, но если камера не настроена следовать за каким-либо объектом, то заданные здесь значения не будут иметь видимого эффекта. установленные здесь значения не будут иметь видимого эффекта.

Синтаксис:

camera_set_view_speed(camera_id, xspeed, yspeed)

АргументТип Описание
camera_id The unique camera ID value returned when you created the camera.
xspeed The speed (number of pixels per game frame) that the view should move on the horizontal (x) axis
yspeed The speed (number of pixels per game frame) that the view should move on the vertical (y) axis

Возвращается:

Пример:

camera_set_view_speed(view_camera[0], 5, 5);

Приведенный выше код установит скорость камеры просмотра для камеры, назначенной на порт просмотра[0].
