camera_set_proj_mat

Эта функция устанавливает матрицу проекции для данной камеры. Вы передаете уникальное значение ID камеры (возвращаемое различными camera_create() функции) и матрицу проекции, которую необходимо применить. Подробнее о создании проекционных матриц вы можете узнать из раздела Матричные функции, а именно matrix_build_projection_perspective() и matrix_build_projection_ortho().

При использовании этой функции необходимо помнить о некоторых моментах:

  • Если ваша камера выполняет автоматическое отслеживание объектов - т.е. она была создана с использованием camera_create_view() с индексом объекта / ID экземпляра, который не равен -1, или вы устанавливаете камеру, определенную в Room Editor, или вы устанавливаете камеру по умолчанию - тогда эта матрица будет перезаписана на следующем игровом кадре.
  • Если вы используете эту функцию для переключения на перспективную проекцию (в отличие от ортографической проекции по умолчанию), автоматическое рисование плиток sprites перестанет работать и будет выполняться вручную. Это относится к любым фоновым слоям, использующим горизонтальную или вертикальную черепицу, а также к функциям draw_sprite_tiled() и draw_sprite_tiled_ext(), которые при использовании перспективной проекции будут рисовать sprite только один раз вместо черепицы.

Синтаксис:

camera_set_proj_mat(camera_id, matrix)

АргументТип Описание
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

Возвращается:

Пример:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Приведенный выше код создает новую камеру и назначает ее порту вида[0]. Затем у этой камеры устанавливаются матрицы вида и проекции.
