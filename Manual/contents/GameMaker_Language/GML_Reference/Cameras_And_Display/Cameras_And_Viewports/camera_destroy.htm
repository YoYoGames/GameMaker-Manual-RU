уничтожение камеры

С помощью этой функции вы можете уничтожить любую камеру. При вызове функции вы передаете уникальное значение ID камеры, которое вы получаете из функций camera_create_* или из view_camera массива при использовании редактора комнат для настройки порта просмотра и вида. Вы никогда не должны уничтожать камеру, которая в данный момент назначена на видимый вид, если только вы не назначаете новую камеру на этот вид в том же шаге, и вы всегда должны уничтожать любые камеры, которые вы создали с помощью кода, когда они больше не нужны вашей игре, чтобы предотвратить утечку памяти, и вы также можете уничтожить камеры по умолчанию, если вы назначили их в редакторе комнаты, но вы должны назначить новую камеру на порт вида (или отключить ее), иначе вы получите странные результаты.

Синтаксис:

camera_destroy(camera_id)

АргументТип Описание
camera_id The unique camera ID value returned when you created the camera.

Возвращается:

N/A

Пример:

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

Приведенный выше код уничтожает камеру, назначенную в данный момент порту просмотра [0], затем создает новую камеру и назначает ее ID этому порту.
