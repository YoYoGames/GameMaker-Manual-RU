camera_set_view_pos

Вы можете использовать эту функцию для обновления положения вида камеры в помещении. Вы указываете уникальное значение ID камеры (возвращаемое различными функциями camera_create() ), а затем указываете позиции x и y. для установки камеры.

Синтаксис:

camera_set_view_pos(camera_id, x, y)

АргументТип Описание
camera_id The unique camera ID value returned when you created the camera.
x The x position to place the view at (in the room)
y The y position to place the view at (in the room)

Возвращается:

Пример:

camera_set_view_pos(view_camera[0], x - (view_wport[0] / 2), y - (view_hport[0] / 2));

Приведенный выше код установит положение камеры просмотра для камеры, назначенной на порт просмотра[0].
