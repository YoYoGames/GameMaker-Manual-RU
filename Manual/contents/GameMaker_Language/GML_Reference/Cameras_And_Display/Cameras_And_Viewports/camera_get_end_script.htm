camera_get_end_script

Эта функция может быть использована для получения идентификатора функции сценария, назначенной в качестве конечной функции для данной камеры. Если функция сценария не назначена, то функция вернет -1.

Синтаксис:

camera_get_end_script(camera_id)

АргументТип Описание
camera_id The unique camera ID value returned when you created the camera

Возвращается:

Script Function or -1 if no function is assigned

Пример:

var scr = camera_get_end_script(camera_view[0]);
if (scr != cutScene)
{
    camera_set_update_script(camera_view[0], cutScene);
}

Приведенный выше код проверяет функцию сценария, назначенную в качестве начального сценария для камеры, назначенной на порт просмотра[0], и если это не "cutScene", то она устанавливается на эту функцию.
