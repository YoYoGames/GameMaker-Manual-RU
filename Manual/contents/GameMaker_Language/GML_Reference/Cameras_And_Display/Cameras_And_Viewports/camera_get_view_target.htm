camera_get_view_target

Эта функция может быть использована для получения цели следования для заданной камеры. Возвращаемое значение может быть индексом объекта, идентификатором экземпляра или -1, если цель следования не была установлена.

Синтаксис:

camera_get_view_target(camera_id)

АргументТип Описание
camera_id The unique camera ID value returned when you created the camera

Возвращается:

Object Asset, Instance ID or -1

Пример:

var target = camera_get_view_target(view_camera[0]);
if (target != obj_Player)
{
    camera_set_view_target(view_camera[0], obj_Player);
}

Приведенный выше код извлекает цель следования камеры, назначенной порту вида[0], а затем проверяет, совпадает ли она с индексом объекта "obj_Player". Если нет, то камера вида устанавливается на следование за экземпляром этого объекта.
