display_set_gui_maximise

Эта функция может быть использована для максимизации слоя GUI и установки его масштабирования и смещения относительно размеров экрана, а не положения и размеров поверхности приложения по умолчанию. По умолчанию слой GUI имеет разрешение 1:1 с разрешением поверхности приложения и рисуется в позиции (0,0) поверхности. Однако это не всегда то, что вы хотите, поэтому вы можете использовать эту функцию, чтобы установить его рисование относительно абсолютной (0,0) позиции дисплея или игрового окна. Какие аргументы вы передадите этой функции, зависит от того, какой эффект вы хотите получить от слоя GUI. При простом вызове функции без аргументов слой GUI будет отрисован в позиции (0,0) экрана или игрового окна, при этом ширина и высота будут масштабированы, чтобы вместить всю область:

display_set_gui_maximise();

Однако вы можете задать коэффициент масштабирования для слоя GUI, и ширина и высота будут масштабироваться на эту величину. Помните, что слой GUI всегда подгоняется под размер дисплея или игрового окна, или поверхности приложения, поэтому установка этого значения в любое значение, отличное от 1, в основном масштабирует количество пикселей по ширине и высоте. Если ваш дисплей имеет размер 1024x768, а вы установили масштабный коэффициент на 0,5, то ваш слой GUI будет иметь размер, равный половине размера дисплея, что фактически удваивает размер пикселей:

display_set_gui_maximise(0.5, 0.5);

Установка значений таким образом также установит позицию рисования в (0,0) окна дисплея или игры, поэтому вы можете предоставить значения смещения для "перемещения" позиции (0,0):

var pos = application_get_position();
display_set_gui_maximise(0.5, 0.5, pos[0], pos[1]);

Наконец, вы можете сбросить слой GUI с помощью этой функции, установив коэффициенты масштабирования на -1. В результате слой GUI снова будет иметь масштаб 1:1, а позиция рисования будет установлена в положение (0,0) поверхности приложения, а не дисплея или окна.

display_set_gui_maximise(-1, -1);

Синтаксис:

display_set_gui_maximise(xscale, yscale, xoffset, yoffset);

Аргумент Тип Описание
xscale Реально OPTIONAL Коэффициент горизонтального масштабирования (используйте -1, чтобы вернуть значение по умолчанию).
yscale Реально OPTIONAL Коэффициент вертикального масштабирования (используйте -1, чтобы вернуть значение по умолчанию).
xoffset Реально OPTIONAL Позиция смещения x для рисования.
yoffset Реально OPTIONAL Позиция смещения по оси y для рисования.

Возвращается:

N/A

Пример:

display_set_gui_size(display_get_width() / 2, display_get_height() / 2);
display_set_gui_maximise(2, 2, 0, 0);

Приведенный выше код зафиксирует событие draw GUI на заданной ширине и высоте, масштабируя все компоненты для размещения на дисплее, используя эту пропорцию.
