display_get_height

Эта функция возвращает высоту дисплея (в пикселях). Обратите внимание, что для HTML5 это значение является высотой окна браузера, а не физическим размером дисплея.

Синтаксис:

display_get_height();

Возвращается:

Пример:

myheight = display_get_height();

Это установит переменную "myheight" на высоту дисплея.
