display_get_width

Эта функция возвращает ширину дисплея (в пикселях). Обратите внимание, что для HTML5 это значение является шириной окна браузера, а не физическим размером дисплея.

Синтаксис:

display_get_width();

Возвращается:

Пример:

mywidth = display_get_width();

Это установит переменную "mywidth" на ширину дисплея.
