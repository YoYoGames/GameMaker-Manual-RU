window_set_rectangle

С помощью этой функции вы можете задать положение игрового окна в браузере (HTML5) или на экране (Windows, Ubuntu (Linux) или macOS) , а также задать масштаб окна. Для получения дополнительной информации о положении окна и его размере см. window_set_position() и window_set_size().

Синтаксис:

window_set_rectangle(x, y, w, h);

АргументТип Описание
x The new x coordinate of the window.
y The new y coordinate of the window.
w The new width of the window.
h The new height of the window.

Возвращается:

Пример:

window_set_rectangle(0, 0, display_get_width(), display_get_height());

Приведенный выше код установит окно игры так, чтобы оно занимало всю область дисплея (либо браузер, либо экран, в зависимости от целевой платформы).
