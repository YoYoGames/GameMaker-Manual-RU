ширина_окна

С помощью этой функции вы можете получить текущую ширину (в пикселях) окна игры.

Синтаксис:

window_get_width();

Возвращается:

Пример:

ww = window_get_width();

Приведенный выше код будет хранить текущую ширину окна в переменной "ww".
