рукоятка_окна

С помощью этой функции вы можете получить внутреннее значение Windows id (HWND, указатель). Эта функция действительно полезна только для авторов расширений, которым нужен window handle для вызова Windows API в коде DLL (возвращаемый указатель следует привести к строке, а затем в C++ просто привести его к HWND). В таблице ниже показаны поддерживаемые платформы и то, что они возвращают:

Платформа Возвращает
Windows Окно HWND
macOS Класс NSWindow
Ubuntu (Linux) XWindow handle
HTML5 Идентификатор холста

Синтаксис:

window_handle();

Возвращается:

Pointer

Пример:

win_id = windows_handle();

Приведенный выше код будет хранить идентификатор окна игры в переменной.
