window_set_position

С помощью этой функции вы можете установить окно игры в определенную позицию на дисплее (на macOS, Linux(Ubuntu) и Windows) или в браузере (HTML5).

ПРИМЕЧАНИЕ: Если в вашей HTML5-игре используется пользовательский index.html, который устанавливает холст в фиксированное положение, то эта функция не будет влиять на положение окна.

Синтаксис:

window_set_position(x, y);

АргументТип Описание
x The x coordinate of where to position the window.
y The y coordinate of where to position the window.

Возвращается:

Пример:

window_set_position(0, 0);

Приведенный выше код позиционирует окно игры в левом верхнем углу браузера или дисплея (в зависимости от используемого целевого модуля).
