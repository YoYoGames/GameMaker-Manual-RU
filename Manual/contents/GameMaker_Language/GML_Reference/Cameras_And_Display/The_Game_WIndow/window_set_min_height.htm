window_set_min_height

Эта функция может быть использована для установки минимальной высоты окна вашей игры. Если вы включите опцию изменения размера окна в параметрах игры для целевой платформы, то игрок сможет изменить размер игрового окна до любого желаемого размера, однако с помощью этой функции вы можете ограничить минимальную высоту указанным вами размером. Если вы хотите вернуться к поведению по умолчанию (т.е. без минимума), то используйте значение -1.

ПРИМЕЧАНИЕ: Эта функция доступна только на платформах Windows, macOS и Linux.

Синтаксис:

window_set_min_height(height);

АргументТип Описание
height The minimum height in pixels for the game window.

Возвращается:

Пример:

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

Приведенный выше код установит минимальную и максимальную ширину и высоту для игрового окна.
