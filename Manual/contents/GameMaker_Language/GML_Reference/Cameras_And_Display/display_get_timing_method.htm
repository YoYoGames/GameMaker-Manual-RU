display_get_timing_method

Эта функция может использоваться для получения метода синхронизации, который будет использоваться для рендеринга вашей игры. Метод может быть одной из констант, перечисленных ниже.

Для получения дополнительной информации о различных методах определения времени см. display_set_timing_method().

Синтаксис:

display_get_timing_method();

Возвращается:

Display Timing Method Constant

Метод синхронизации дисплея Постоянный
Постоянная Описание
tm_sleep Значение запаса сна является основным методом хронометража
tm_countvsyncs Тайминг Vsync - основной метод тайминга (по умолчанию для всех поддерживаемых платформ)

Пример:

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if (display_get_sleep_margin() != 20)
    {
        display_set_sleep_margin(20);
    }
}

Приведенный выше код проверит метод синхронизации, и если он не установлен на tm_sleep, то он будет установлен на это значение, а запас сна будет установлен на 20.
