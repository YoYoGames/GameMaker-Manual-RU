window_mouse_get_y

С помощью этой функции вы можете получить позицию y курсора мыши (в пикселях) в пределах браузера, если это игра HTML5, или в пределах дисплея, если это игра Windows, Ubuntu (Linux) или macOS.

ПРИМЕЧАНИЕ: Об обычных функциях мыши см. раздел " Ввод данных с помощью мыши".

Синтаксис:

window_mouse_get_y();

Возвращается:

Пример:

wy = window_mouse_get_y();

Приведенный выше код сохраняет текущее положение окна мыши по оси y в переменной "wy".
