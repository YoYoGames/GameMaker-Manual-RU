window_mouse_set

С помощью этой функции вы можете изменить или установить положение мыши в игровом окне, что может быть полезно, например, в играх FPS. Функция будет работать только тогда, когда игра находится в фокусе, а использование alt + tab разблокирует мышь.

ПРИМЕЧАНИЕ Об обычных функциях мыши см. раздел " Ввод мыши".

Синтаксис:

window_mouse_set(x, y);

АргументТип Описание
x The x coordinate in the window.
y The y coordinate in the window.

Возвращается:

Пример:

window_mouse_set(window_get_width() / 2, window_get_height() / 2);

Приведенный выше код центрирует мышь в окне игры.
