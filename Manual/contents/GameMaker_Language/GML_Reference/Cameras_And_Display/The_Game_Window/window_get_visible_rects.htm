window_get_visible_rects

С помощью этой функции вы можете найти область перекрытия прямоугольника, определенного от (x1,y1) до (x2,y2) на каждом из подключенных дисплеев. Функция вернет массив с 8 значениями для каждого дисплея (т.е. если у вас два дисплея, массив будет иметь 16 индексов), где значения [0 ... 3] соответствуют значениям overlapx1, overlapy1, overlapx2, overlapy2 - определяют область перекрытия на данном дисплее и будут установлены в 0,0,0,0,0, если перекрытия нет - а значения [4 ... 7] соответствуют значениям monitorx1, monitory1, monitorx2, monitory2 - координаты дисплея в виртуальном пространстве дисплея. Это можно использовать для проверки того, будет ли сохраненная позиция окна видна или нет (пользователь мог отключить внешний монитор или переместить окно за пределы экрана, в результате чего сохраненная позиция окна оказалась недействительной), например.

Синтаксис:

window_get_visible_rects(x1, y1, x2, y2);

АргументТип Описание
x1 The left edge of the rectangle to check
y1 The top edge of the rectangle to check.
x2 The right edge of the rectangle to check
y2 The bottom edge of the rectangle to check.

Возвращается:

Пример:

var wx = window_get_x();
var wy = window_get_y();
var ww = window_get_width();
var wh = window_get_height();
display_data = window_get_visible_rects(wx, wy, wx + ww, wy + wh);
display_num = array_length(display_data) / 8;

Приведенный выше код создаст одномерный массив, хранящийся в переменной display_data, содержащий информацию о дисплеях, а также создаст переменную display_num для хранения количества найденных активных дисплеев.
