window_has_focus

С помощью этой функции вы можете опросить состояние окна (или вкладки), и если оно потеряло фокус, функция вернет false, в противном случае она вернет true. В большинстве случаев вы можете просто использовать os_is_paused() для проверки этого, но в некоторых специфических случаях (например, игры в Chrome Apps) эта функция не сработает, и в этом случае вместо нее следует использовать эту функцию.

ПРИМЕЧАНИЕ: Эта функция действительна только для HTML5, Windows, и MacOS.

Синтаксис:

window_has_focus();

Возвращается:

Пример:

if !window_has_focus()
{
    pause_game();
}

Приведенный выше код проверит, находится ли окно игры в фокусе или нет, и если функция вернет false, то будет вызвана функция.
