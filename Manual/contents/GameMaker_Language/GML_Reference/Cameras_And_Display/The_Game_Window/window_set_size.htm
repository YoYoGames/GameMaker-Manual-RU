размер_окна

С помощью этой функции вы можете изменить размер игрового окна. На настольных платформах это обычно относится к клиентской области окна, исключая строку заголовка и границы:

Если вы используете эту функцию для изменения размера игрового окна, содержимое окна будет масштабироваться в соотношении 1:1 с фоновым холстом, что сохранит четкость и ясность изображения (хотя оно будет масштабироваться, если размер окна отличается от размера комнаты или вида). Например, если у вас есть окно размером 200 x 150 пикселей, и игрок растягивает окно, перетаскивая границы, конечный размер окна может быть 400 x 300 пикселей, но холст рисунка все еще 200 x 150, что дает размытое изображение, поскольку соотношение пикселей теперь 1:2. Однако если вы используете эту функцию для изменения размера окна, соотношение 1:1 сохраняется, хотя изображение масштабируется в соответствии с новым размером окна.

Синтаксис:

window_set_size( w, h );

АргументТип Описание
w The new width of the window.
h The new height of the window.

Возвращается:

N/A

Пример:

if mouse_check_button_pressed(mb_left)
{
    global.scale += 1;
    window_set_size(200 * global.scale, 150 * global.scale);
}

Приведенный выше код будет изменять размер окна в зависимости от значения переменной "global.scale" при нажатии кнопки мыши.
