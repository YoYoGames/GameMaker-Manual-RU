Окно игры

Игра, которую вы создаете, происходит в окне (даже в полноэкранном режиме), и это окно имеет ряд свойств, таких как положение, размер, является ли оно полноэкранным и т.д.. Эти параметры обычно устанавливаются автоматически на основе размера комнаты и включенных портов просмотра в сочетании с настройками из Game Options для целевой платформы, но вы можете изменить их во время игры с помощью функций, перечисленных на этой странице.

ПРИМЕЧАНИЕ Эти функции предназначены только для Windows, Ubuntu (Linux), macOS и целевых модулей HTML5 и могут не работать на любом другом устройстве.

На следующем рисунке показано, как некоторые общие функции окна соотносятся и взаимодействуют друг с другом:

Обратите внимание, что на изображении выше показано игровое окно, нарисованное в браузере, но для других целей браузер можно заменить дисплеем, поэтому функция типа window_get_y() вернет положение верхней части игрового окна относительно дисплея.
