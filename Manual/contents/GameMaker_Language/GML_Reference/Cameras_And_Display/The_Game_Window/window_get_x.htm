window_get_x

С помощью этой функции вы можете получить позицию x (в пикселях) в браузере, если это игра HTML5, или в дисплее, если это игра Windows, Ubuntu (Linux) или macOS.

Синтаксис:

window_get_x();

Возвращается:

Пример:

wx = window_get_x();

Приведенный выше код сохраняет текущее положение окна по оси x в переменной "wx".
