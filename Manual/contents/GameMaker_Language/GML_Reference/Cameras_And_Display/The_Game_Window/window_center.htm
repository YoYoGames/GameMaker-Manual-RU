window_center

С помощью этой функции вы можете отцентрировать окно игры на дисплее, если целевой модуль Windows, Ubuntu (Linux) или macOS, или отцентрировать его в браузере, если целевой модуль HTML5. Эта функция не имеет эффекта на любом другом устройстве.

ПРИМЕЧАНИЕ Если вы изменили размер окна игры на текущем шаге (например, переключившись с полноэкранного режима на оконный или используя window_set_size()), и хотите отцентрировать новое окно, эту функцию следует вызвать по крайней мере на один шаг позже (например, используя alarm), иначе она будет работать некорректно.

Синтаксис:

window_center();

Возвращается:

Пример:

if !window_get_fullscreen()
{
    window_center();
}

Приведенный выше код центрирует окно игры в пределах дисплея, если оно не находится в полноэкранном режиме.
