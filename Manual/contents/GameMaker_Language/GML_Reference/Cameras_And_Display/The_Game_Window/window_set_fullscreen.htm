window_set_fullscreen

С помощью этой функции вы можете установить полноэкранный режим игрового окна (true) или нет (false). Обратите внимание, что для macOS вы должны снять флажок "Start In Fullscreen" и установить флажок "Allow the player to resize the game window" в Game Options, иначе эта функция не будет работать. Также обратите внимание, что эта функция не будет работать на HTML5, если она не добавлена в качестве "кликабельного" обратного вызова (подробнее см. здесь ).

Синтаксис:

window_set_fullscreen(full);

АргументТип Описание
full Whether to set the screen to fullscreen (true) or not (false).

Возвращается:

Пример:

if mouse_check_button_pressed(mb_left)
{
    if window_get_fullscreen()
    {
        window_set_fullscreen(false);
    }
    else
    {
        window_set_fullscreen(true);
    }
}

Приведенный выше код проверяет нажатие кнопки мыши и затем переводит окно в полноэкранный режим, если оно еще не переведено, или переводит его в оконный режим, если оно уже переведено.
