экран_сохранения

С помощью этой функции вы можете сохранить скриншот игры в том виде, в котором он сейчас отображается в игровом окне.

Файл будет сохранен в рабочую директорию игры, которая в windows находится по адресу <drive>:/Users/YOURUSERNAME/AppData/Local/[Game Name]/, а в Mac - ~/Library/Application Support/[Game Name]/. Захваченное изображение будет окончательным рендером поверхности приложения игры, или (если поверхность приложения была отключена) это будет изображение всего дисплея или окна.

Важно отметить, что функция предназначена для вызова в событии Draw GUI End Event, которое должно давать согласованные результаты на всех платформах. Использование ее в любом другом событии может не сработать так, как ожидается, и дать разные результаты на разных платформах и даже устройствах.

Если вам нужно больше контроля над тем, что сохраняется, используйте функцию screen_save_part().

ПРИМЕЧАНИЕ Эта функция не работает на цели HTML5.

Синтаксис:

screen_save(fname);

Аргумент Тип Описание
fname Строка The filename for the screenshot to save

Возвращается:

N/A

Пример:

if keyboard_check_pressed(ord("S"))
{
    screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png")
}

Приведенный выше код делает снимок экрана игры, когда пользователь нажимает клавишу "S". Этот снимок экрана сохраняется в определенном каталоге и нумеруется с помощью переменной, которая затем увеличивается.
