gif_open

С помощью этой функции вы можете создать пустое изображение в формате GIF, готовое к добавлению данных. При вызове функции необходимо указать ширину и высоту GIF (в пикселях, причем верхнего предела размера нет, кроме доступной памяти), и функция вернет уникальное значение ID, используемое для идентификации подарка в последующих функциях, или вернет -1, если gif не удалось инициализировать (например, если ширина/высота слишком велики для доступной памяти). Вы также можете указать необязательный аргумент для задания "прозрачного цвета" для GIF. Это значение RGB (без альфа-компонента), и оно очистит GIF до этого цвета, прежде чем в него что-либо будет добавлено. Если вы не укажете аргумент "чистый цвет", по умолчанию будет использоваться черный цвет.

Обратите внимание, что при использовании этой функции вы должны вызвать gif_save() для завершения создания GIF перед открытием другого файла для записи (поэтому для каждого gif_open() должен быть сопроводительный gif_save()).

Синтаксис:

gif_open(width, height, [clear_colour]);

Аргумент Тип Описание
width Реально The width of the gif to create
height Реально The height of the gif to create
[clear_colour] Цвет OPTIONAL Цвет (RGB), к которому нужно очистить gif

Возвращается:

GIF ID

Пример:

if save_gif == true
{
    if (count == 0)
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if (count < 30)
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

Приведенный выше код создаст файл изображения GIF с 30 кадрами, взятыми с поверхности приложения, а затем сохранит его.
