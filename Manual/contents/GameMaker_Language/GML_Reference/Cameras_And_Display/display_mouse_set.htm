display_mouse_set

С помощью этой функции вы можете изменить или установить положение мыши на игровом дисплее, что может быть полезно, например, в играх FPS. Функция работает, только когда игра находится в фокусе, а использование ALT + Tab разблокирует мышь.

Синтаксис:

display_mouse_set(x, y);

АргументТип Описание
x The x coordinate in the display.
y The y coordinate in the display.

Возвращается:

Пример:

display_mouse_set(display_get_width() / 2, display_get_height() / 2);

Приведенный выше код центрирует мышь на игровом дисплее.
