display_reset

Эта функция сбрасывает настройки дисплея на те, которые были установлены при запуске игры, а также позволяет изменить текущий уровень используемого полноэкранного сглаживания и использовать ли вертикальную синхронизацию. Доступные уровни сглаживания - 0, 2, 4 и 8, при этом при запуске по умолчанию установлено значение 0, а для v-sync по умолчанию установлено значение false (выключено). Включение v-sync может обеспечить более плавный игровой процесс, но при этом потребуется больше вычислительной мощности, поэтому перед использованием следует внимательно изучить его влияние. То же самое касается и сглаживания, где чем выше число, тем больше требуется обработки.

Поскольку не все целевые устройства одинаковы, некоторые могут не поддерживать, например, сглаживание 8x или 4x, поэтому существует переменная, доступная только для чтения, для получения различных уровней AA, которые может отображать устройство, на котором запущена игра:

display_aa

Эта переменная возвращает целочисленное значение, основанное на установке битов для различных уровней. Так, только для 2xAA она сообщит 2, для 2x и 4x доступности она сообщит 6. Для 8 и 4 будет сообщено 12. Для всех трех уровней (2, 4 и 8) она сообщит 14.

Синтаксис:

display_reset(aa, vsync);

Аргумент Тип Описание
aa Реально The level of anti-aliasing filtering (0, 2, 4 or 8).
vsync Булево Toggle vertical synchronisation to on (true) or off (false).

Возвращается:

Real

Пример:

if (display_aa > 12)
{
    display_reset(8, true);
}

Приведенный выше код установит уровень сглаживания на 8, если поддерживается, и включит v-sync.
