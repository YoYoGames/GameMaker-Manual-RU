экран_сохранить_часть

С помощью этой функции можно сохранить скриншот части игры в том виде, в котором она отображается на экране. Координаты выделения основаны на абсолютном размере дисплея (или окна, если оно не полноэкранное), поэтому (0,0) - это всегда левый верхний угол экрана (или окна). Ниже приведено изображение, показывающее, как работает эта функция:

Файл будет сохранен в рабочем каталоге игры. Важно отметить, что функция предназначена для вызова в событии Draw GUI End Event, что должно дать согласованные результаты на всех платформах. Использование ее в любом другом событии может не сработать, как ожидалось, и дать разные результаты на разных платформах и даже устройствах.

ПРИМЕЧАНИЕ Эта функция не работает на цели HTML5.

Синтаксис:

screen_save_part(fname,x,y,w,h);

АргументТип Описание
fname The filename for the screenshot to save
x The x coordinate of the top left corner of the area of the screen to capture
y The y coordinate of the top left corner of the area of the screen to capture
w The width of the area of the screen to capture
h The height of the area of the screen to capture

Возвращается:

Пример:

if keyboard_check_pressed(ord("S"))
{
    screen_save_part("Screens\Screen_" + string(num++) + ".png", 0, 0, 100, 100);
}

Приведенный выше код делает скриншот части игрового экрана размером 100 пикселей в квадрате, когда пользователь нажимает клавишу "S". Этот снимок экрана сохраняется в определенной папке и нумеруется с помощью переменной, которая затем увеличивается.
