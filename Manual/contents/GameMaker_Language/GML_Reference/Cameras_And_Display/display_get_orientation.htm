display_get_orientation

Эта функция возвращает одну из четырех констант GameMaker, позволяющих определить, в каком режиме находится устройство, на котором запущена игра, - альбомном или портретном (см. таблицу ниже).

Обратите внимание, что эта функция может неправильно определять ориентацию устройства при использовании в целевом модуле HTML5. Однако это легко имитируется с помощью следующего скрипта:

return (browser_width < browser_height);

Такая функция будет возвращать true для портрета и false для пейзажа.

Синтаксис:

display_get_orientation()

Возвращается:

Display Orientation Constant

Ориентация дисплея Постоянная
Постоянная Описание
display_landscape Устройство удерживается горизонтально, т.е. самый длинный край расположен слева направо, а кнопка меню находится справа.
display_landscape_flipped Как и выше, только теперь кнопка меню находится слева.
display_portrait Устройство удерживается вертикально, т.е. самый длинный край находится сверху вниз, а кнопка меню - внизу.
display_portrait_flipped Как и выше, только теперь кнопка меню находится вверху.

Пример:

if display_get_orientation() == display_landscape
{
    global.Config = 0;
}
else
{
    global.Config = 1;
}

Приведенный выше код проверяет ориентацию устройства и устанавливает глобальную переменную в зависимости от возвращенного значения.
