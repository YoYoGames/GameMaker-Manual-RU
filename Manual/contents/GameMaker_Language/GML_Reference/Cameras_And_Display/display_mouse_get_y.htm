display_mouse_get_y

Эта функция возвращает положение мыши по оси y на экране. Следует отметить, что эта функция работает правильно только при использовании на объектах Windows и macOS.

Его можно использовать и для HTML5, но он будет возвращать значение только относительно 0,0 самого холста, и не будет возвращать никакого значения, пока мышь находится за пределами холста.

Для других устройств он вернет 0, и вам следует использовать функцию device_mouse_raw_x() и device_mouse_raw_y() вместо этого.

Синтаксис:

display_mouse_get_y();

Возвращается:

Real

Пример:

my_y = display_mouse_get_y();

Это установит my_y в координату y мыши относительно дисплея.
