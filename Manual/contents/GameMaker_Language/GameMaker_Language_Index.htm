Код GML

GameMaker Language (также называемый просто GML) является собственным языком сценариевGameMaker. Этот язык построен таким образом, чтобы позволить пользователям создавать свои игры интуитивно понятным и гибким способом, обладая при этом всеми возможностями любого другого крупного языка программирования. Он также является основой для GML Visual и при необходимости может быть использован совместно с ним.
