Ярлыки клавиатуры

В GameMaker существует огромное количество сочетаний клавиш, и хотя большинство из них отмечены на сайте IDE рядом с пунктами меню, к которым они относятся, не все из них очевидны для пользователя в самом начале. Здесь вы можете найти полный список всех сочетаний клавиш для удобства пользования. В целом, сочетания клавиш делятся на следующие три категории:

  • Глобальные - Глобальные ярлыки - это те, которые работают независимо от того, где вы находитесь в GameMaker IDE, если только ваш фокус не находится на определенном окне или гаджете, который его отменяет.
  • Окно - ярлыки окна - это ярлыки, которые предназначены для одного окна или вкладки, если только ярлык гаджета не отменяет его.
  • Гаджет - ярлыки гаджетов очень специфичны и работают только при использовании одного гаджета (например, подборщика цветов).

Ниже приведен полный список ярлыков, разбитый на разделы по редакторам:
