Меню инструментов

Это меню позволяет вам открыть различные отдельные инструменты, доступные вам в GameMaker, со следующими опциями:

  • Группы текстур - просмотр окна группы текстур.
  • Аудиогруппы - Просмотр окна аудиогрупп.
  • Sound Mixer - просмотр окна Sound Mixer.
  • Generate Project Images - Открывает окно Generate Project Images.
  • Создать локальный пакет - открывает окно Asset Packager, позволяющее создавать и сохранять локальные пакеты активов для дальнейшего использования.
  • Импорт локального пакета - открывает окно проводника файлов, в котором можно выбрать локальный пакет активов и импортировать его, либо добавив его в текущий проект, либо создав новый проект. для него.
