Меню Marketplace

Помимо создания собственных активов и кода для своих проектов, у вас также есть доступ к сайту GameMaker Marketplace, где вы можете загрузить как бесплатные, так и платные активы для добавления в свои проекты. Подробнее о сайте Marketplace вы можете узнать из разделе, посвященном GameMaker Marketplace, а опции из этого меню приведены ниже:

  • Перейдите в Marketplace - откройте окно Marketplace, где вы можете просматривать активы.
  • Моя библиотека - просмотр текущей библиотеки материалов Marketplace.
  • Моя история покупок - Эта опция откроет браузер и переведет вас на страницу учетной записи пользователя, вошедшего в систему, чтобы вы могли просмотреть историю покупок для этой учетной записи.
  • Обновить существующий пакет - Эта опция откроет мастер обновления пакета, чтобы вы могли обновить существующий пакет Marketplace, который вы ранее создали.
  • Создать пакет - С помощью этой опции можно открыть мастер создания пакета, чтобы создать новый пакет активов или учебник для сайта Marketplace.
