Меню Windows

Это меню предназначено для открытия общих окон доков IDE. Поскольку доки могут быть переупорядочены, а их содержимое может быть удалено как отдельные окна или даже полностью закрыто, это окно позволяет вам снова открыть их, если вы обнаружите, что они вам для чего-то нужны. Окна, которые можно открыть отсюда, - это окно браузера активов и все различные окна вывода (Output, Search, Source Control, Точки останова и Ошибки компиляции/синтаксиса).

Далее вы можете открыть окно Инспектор. Это окно используется для проверки свойств выбранного в данный момент элемента IDE, и эти свойства можно редактировать по мере необходимости.

После опции Инспектор у вас есть список Недавние окна, а также возможность открыть окно Переменные среды, где вы можете увидеть различные пути, на которые указывают различные переменные среды, используемые в Предпочтениях. на которые указывают различные переменные окружения, используемые в Предпочтениях. Эти переменные нельзя редактировать (ни их названия, ни пути/настройки), но вы можете выбрать и скопировать/вставить любую из них для использования в другом месте. в другом месте.

Наконец, у вас есть возможность открыть учебные окна. Это позволит вновь открыть все окна, связанные с учебным проектом. но только если этот проект был создан в рамках учебного курса (в противном случае эта опция будет недоступна). Информацию о создании собственных обучающих проектов можно найти в разделе о Marketplace (поскольку все обучающие проекты создаются, размещаются и загружаются оттуда).
