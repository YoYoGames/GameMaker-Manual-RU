Меню сборки

В меню сборки вы можете выбрать сборку проекта для тестирования, отладки или в качестве окончательного исполняемого файла. Доступны следующие опции:

  • Запустить - Запустить текущий проект на выбранной целевой платформе для тестирования. О различных опциях компиляции, доступных при компиляции, см. раздел Компиляция. Обратите внимание, что эта команда аналогична нажатию кнопки Run на сайте IDE.
  • Отладка - Запустите текущий проект для тестирования с помощью отладчика, чтобы вы могли увидеть подробную информацию о производительности и отладить любые ошибки или проблемы, которые могут возникнуть. Для получения дополнительной информации об отладке см. раздел Отладка. Обратите внимание, что эта команда аналогична нажатию кнопки Отладка на странице IDE.
  • Очистить - GameMaker сохраняет кэш файлов, чтобы ускорить время компиляции при тестировании. Однако иногда "несвежий" кэш может привести к странным и неожиданным ошибкам в вашей игре, и в этом случае первым делом следует очистить кэш и провести повторное тестирование. Также неплохо очистить кэш перед созданием окончательного исполняемого файла для целевой платформы. Обратите внимание, что эта команда аналогична нажатию кнопки Clean Cache на странице IDE.
  • Create Executable - компилирует текущий проект и создает исполняемый пакет (или пакеты) для целевой платформы. Сначала откроется проводник файлов, чтобы вы могли выбрать место сохранения проекта, а затем скомпилировать игру. Дополнительную информацию о настройке целей компиляции см. в разделе Компиляция. Обратите внимание, что эта команда аналогична нажатию кнопки Create Executable на странице IDE.
