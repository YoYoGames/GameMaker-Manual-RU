Меню "Файл

В меню "Файл" вы можете изменить, сохранить или импортировать проекты, а также установить предпочтения GameMaker IDE. Доступны следующие опции:

  • Новый проект - предназначен для создания нового проекта с нуля в GameMaker. Откроется диалог "Сохранить", и вам будет предложено выбрать место для сохранения нового проекта. После того, как вы дадите проекту имя и укажете место для сохранения, вы перейдете в новое, пустое рабочее пространство, чтобы начать работу над проектом. Файлы проектов сохраняются с расширением *.yyp.
  • Открыть проект - откроется проводник файлов, в котором можно выбрать ранее созданный файл проекта *.yyz для загрузки. Обратите внимание, что при импорте, если проект был создан с использованием версии ниже 2.3.0, то вы получите следующее предупреждение: Если вы решите продолжить, то откроется проводник файлов, и вы сможете дать проекту обновленное имя и место для сохранения, так что оригинальный проект не будет перезаписан. После преобразования проекта в новый формат 2.3.0+ он откроется в IDE, и вы получите сообщение о том, что преобразование прошло успешно. Вам следует проверить отчет о совместимости, который был создан в процессе преобразования, и посмотреть, есть ли что-то, что необходимо исправить, прежде чем пытаться запустить или скомпилировать проект. Обратите внимание, что если вы получаете повторяющиеся ошибки при попытке открыть проект, созданный в версии, предшествующей 2.3.0, то вам следует обратиться по адресу YoYo Games (используя опцию "Сообщить об ошибке" из меню справки), предоставив исходный файл *.yyp, который вы пытаетесь конвертировать.
  • Импорт проекта - Здесь вы можете импортировать проекты, которые ранее были экспортированы в архив *.yyz (см. раздел "Экспорт проекта", ниже), или - если у вас есть проект, созданный с помощью устаревшей версии GameMaker: Studio 1.4 - вы можете выбрать эту опцию для импорта файла *.gmx или *.gmz из этой линейки продуктов. Как и при открытии проекта, если файл *.yyz был создан в версии GameMaker до обновления 2.3.0, то его нужно будет конвертировать, так как формат файла проекта изменился с этим обновлением.

ПРИМЕЧАНИЕ: Чтобы импортировать предыдущий проект GameMaker в GameMaker, он должен быть создан с помощью версии 1.4, и даже в этом случае он может не работать непосредственно при импорте. Между GameMaker и GameMaker: Studio 1.4, поэтому ваши старые проекты могут потребовать доработки, чтобы они заработали. Пожалуйста, обратитесь к статье Перенос игры GM:S 1.4 на GameMaker в базе знаний GameMaker и к разделу руководства, посвященному устаревшим функциям и скриптам совместимости.

ВАЖНО! При импорте проекта *.yyz, содержащего шрифтовые ресурсы, может потребоваться "регенерация" шрифтов путем их удаления/добавления или изменения шрифта в редакторе шрифтов, а затем его повторной установки. Это происходит потому, что существующая информация о шрифтах сохраняется на сайте *.yyz при экспорте, а при последующем импорте или на другой машине в проекте могут быть не установлены необходимые шрифты (или может быть установлен другой шрифт с тем же именем). Это следует делать только в том случае, если у вас возникли проблемы с отображением шрифта после импорта.

  • Последние проекты - Вы можете использовать эту опцию для просмотра недавно использованных проектов. При выборе этой опции вновь откроется Стартовая страница (если вы ее закрыли), и вы сможете выбрать недавно использованный проект из списка.
  • Сохранить проект - сохранит вашу игру в том же месте, которое вы указали при создании проекта (или в том месте, откуда вы его открыли).
  • Сохранить проект как - То же самое, что и выше, только вы указываете местоположение и новое имя проекта.
  • Экспорт проекта - позволяет экспортировать проект либо в виде файла, либо в виде шаблона:
    • YYZ - экспортирует проект в один файл *.yyz, что очень удобно для отправки копий людям или для сохранения резервных копий.
    • Как шаблон - Это позволит вам экспортировать ваш проект как шаблон, который вы сможете использовать в качестве основы для других проектов на The Start Page. При выборе этой опции появится следующее окно:
      Сначала нужно нажать на "Редактировать информацию о шаблоне", чтобы убедиться, что информация о шаблоне верна, а затем выбрать "Экспорт", когда вы откроете это приглашение во второй раз, чтобы сохранить проект как шаблон. Затем вы увидите его в меню "Шаблон" на начальном экране.
  • Параметры - выбрав этот пункт, вы перейдете в окно параметров, где можно настроить внешний вид и работу GameMaker.
  • Новая IDE - При выборе этой опции будет открыт новый экземпляр IDE для использования (он откроется на стартовой странице).
  • Моя учетная запись - Эта опция откроет ваш веб-браузер по умолчанию и приведет вас на страницу вашей учетной записи YoYo.
  • Выход - при выборе этого пункта вы выйдете из GameMaker. Вам нужно будет снова войти в систему с той же лицензией (или любой другой), чтобы иметь возможность использовать продукт в дальнейшем.

ВАЖНО! Если вы пользовались бесплатной лицензией GameMaker, а затем приобрели одну из других лицензий, вам, возможно, придется выйти из системы IDE, затем перезапустить GameMaker и снова войти в систему, чтобы убедиться, что новая лицензия применена правильно (хотя это должно произойти автоматически).

  • Exit - выход из GameMaker на рабочий стол.
