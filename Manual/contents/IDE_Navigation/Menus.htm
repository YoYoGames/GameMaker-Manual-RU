Меню

Верхнюю часть сайта GameMaker IDE занимает панель меню. Здесь вы найдете ряд выпадающих меню, которые можно использовать для навигации по сайту IDE , выполнять задания или открывать дополнительные окна с инструментами, которые помогут вам в создании игр. Обратите внимание, что здесь перечислены меню, доступные по умолчанию, но некоторые редакторы активов добавляют дополнительные меню в зависимости от контекста. Эти дополнительные объясняются на страницах, относящихся к редактору, который их создал.

По умолчанию доступны следующие меню:

  • Меню "Файл
  • Меню редактирования
  • Меню сборки
  • Меню Windows
  • Меню инструментов
  • Меню Marketplace
  • Меню макетов
  • Меню справки
