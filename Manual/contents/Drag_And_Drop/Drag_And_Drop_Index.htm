GML Visual

GML Visual - это метод визуального создания сценариев для программирования на языке GameMaker Language (GML). Он заключается в использовании блоков действий для построения игровой логики визуальным и интуитивно понятным способом, и является идеальным инструментом для людей, которые изучают программирование, или для людей с более визуальными наклонностями.
