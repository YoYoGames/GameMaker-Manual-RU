Если клавиша нажата

С помощью этого действия вы можете опросить клавиатуру, чтобы узнать, удерживается ли в данный момент какая-либо клавиша. Действие будет возвращать true на каждом шаге, когда клавиша удерживается нажатой, или false в противном случае, хотя если вы щелкните модификатор not, то сможете проверить, не удерживается ли клавиша, т.е. действие будет возвращать true, если клавиша не нажата, и false, если нажата. Если вам нужно проверить только одно нажатие клавиши, используйте действие If Key Pressed.

Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже:

Любые действия, отброшенные в другом месте, будут выполняться после блока "if" блока.

Если клавиша не удерживается, проверка выполняется выполняется при отпускании клавиши для сброса цвета смешивания на белый.
