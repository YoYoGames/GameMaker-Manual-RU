Если нажата клавиша

Это действие проверяет первоначальное нажатие на клавишу клавиатуры. Оно возвращает true на одном игровом шаге (кадре), когда клавиша нажата изначально, и false во всех остальных случаях. Если вы включите модификатор not, это действие будет обратным и будет проверять, не была ли клавиша нажата, возвращая false на игровом шаге (кадре), когда она была нажата, и true во всех остальных случаях. Если вам нужно проверить, удерживается ли клавиша нажатой, а не только начальное событие нажатия, то используйте действие If Key Down.

Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже:

Любые действия, отброшенные в других местах, будут выполняться после блока "if".

Если проверка true на первоначальное нажатие, то воспроизводится звук, затем, пока клавиша удерживается, смешивание sprite изменяется на красное. Если клавиша не удерживается, проверка выполняется при отпускании клавиши, чтобы сбросить цвет смешивания на белый.
