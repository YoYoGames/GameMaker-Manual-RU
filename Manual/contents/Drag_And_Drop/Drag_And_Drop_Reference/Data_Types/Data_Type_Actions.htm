Типы данных Библиотека действий

ВGameMaker существует множество различных типов данных, таких как целые числа, строки, указатели и т.д.... и некоторые из них могут быть преобразованы в другие. В этом разделе руководства показаны различные действия, доступные для работы с различными типами данных и их преобразования.

ПРИМЕЧАНИЕ: вы можете узнать больше о различных типах данных здесь.

В библиотеке " Типы" предусмотрены следующие действия:

Десятичное в целое
Строка в число
Число в строку

Назад: Указатель
Следующее: Случайные действия
