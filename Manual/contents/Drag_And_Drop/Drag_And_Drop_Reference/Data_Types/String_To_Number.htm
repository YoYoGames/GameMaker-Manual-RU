Строка в число

Это действие может преобразовать строку в число. Вы предоставляете строку для изменения, а затем указываете целевую переменную для возврата реального числа (эта переменная может быть помечена как временная локальная переменная). Обратите внимание, что если в строке есть символы кроме цифр и десятичной точки, то она не будет преобразована, хотя если строка начинается с цифры, то она будет преобразована, а остальная часть строки исключена. Важно отметить, что при передаче десятичных дробей в это действие действие, возвращаемая строка будет округлена только до двух знаков после запятой. Ниже приведена таблица вводимых значений и ожидаемого результата

Строка Возвращаемое значение
"123" 123
"123 Hello World" 123
"Hello World" 0 (по умолчанию, если строка не может быть преобразована, возвращается 0)
"Hello World 123" 0
"123.45678" 123.46

Аргументы:

Аргумент Описание

Пример:

Приведенный выше код блока действий инициализирует глобальную переменную, а затем загружает файл с буфером, сохраненным в нем. Две строки считываются из буфера и сохраняются во временных локальных переменных, после чего одна из них преобразуется в вещественное число и сохраняется в другой временной локальной переменной. Затем они присваиваются глобальным переменным, и буфер удаляется.
