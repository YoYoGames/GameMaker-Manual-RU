Число в строку

Это действие преобразует вещественное число в строку. Если число не является целым и имеет десятичные знаки, они будут округлены до ближайших двух знаков, поэтому, например, число 123.45678 вернет строку "123.46".

Аргументы:

Аргумент Описание
Value Вещественное число для преобразования
Target Целевая переменная для хранения возвращаемого значения

Приведенный выше код блока действий принимает десятичное значение и преобразует его в целое число, сохраняя возвращаемое значение во временной локальной переменной. Затем это значение превращается в строку и выводится на экран.
