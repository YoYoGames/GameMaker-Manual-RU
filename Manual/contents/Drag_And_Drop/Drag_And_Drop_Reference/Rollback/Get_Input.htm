Получить вход (откат)

Это действие возвращает структуру, содержащую входные значения для игрока. Оно принимает необязательный аргумент, указывающий ID игрока, для которого будут возвращены входные данные.

Рекомендуется использовать If Key Down (Rollback), If Key Pressed (Rollback) и If Key Up (Rollback) для проверки вводов в многопользовательской игре.

При использовании в экземпляре игрока, который был создан с помощью Define Player (Rollback), вам не нужно указывать ID игрока, поэтому вы можете просто указать player_id.

При использовании в любых других экземплярах или экземплярах игроков, созданных вручную, необходимо указать ID игрока.

Информацию о том, какие входы возвращаются и как их можно изменить, см. в разделе Определение входов.
