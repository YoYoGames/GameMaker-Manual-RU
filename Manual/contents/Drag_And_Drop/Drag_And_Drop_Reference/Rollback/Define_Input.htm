Определить вход (откат)

Эта функция используется для определения пользовательских входов для всех игроков. Она принимает пару Input/Value, где Input - это имя входа, а Value - константа входа, назначенная ему.

Чтобы узнать, как назначить пользовательские входы, см. раздел Пользовательские элементы управления.

Если используется, эта функция должна выполняться до действия присоединения/создания.

Синтаксис действия:

Аргументы:

Аргумент Описание
Input Имя входа для определения
Value Входная константа для назначения (см. раздел " Пользовательские элементы управления")

Пример:

Приведенное выше действие определяет входы " огонь", " влево", " вправо", " вверх" и " вниз " для всех игроков.
