Начать игру на откат

Это действие заставляет текущую многопользовательскую игру начаться, пока все игроки не присоединились к ней. Сначала необходимо создать многопользовательскую игру с помощью Create Rollback Game.

Обычно система ждет, пока все игроки присоединятся, прежде чем автоматически начать игру. Это действие можно использовать, чтобы начать игру раньше, пока все игроки не присоединились.

Вы можете вызвать команду Use Manual Start, чтобы заставить Rollback ждать этого вызова даже после того, как все игроки присоединились.

Выполнение этого действия вызовет событие Rollback Start.
