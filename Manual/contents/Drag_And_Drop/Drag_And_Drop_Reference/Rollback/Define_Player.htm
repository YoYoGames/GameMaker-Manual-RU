Определить игрока (откат)

Это действие указывает системе Rollback объект, который должен использоваться для игроков. Это необязательное действие, и при его использовании система будет автоматически создавать экземпляры для игроков, которые подключаются, и уничтожать экземпляры для игроков, которые отсоединяются.

Вы можете указать слой, на котором будут созданы экземпляры игроков. Если слой не указан, игроки будут созданы на слое "Instances", который должен существовать в комнате.

Информацию о создании игры с этой функцией или без нее см. в разделе Определение объекта игрока.

Если используется, эта функция должна выполняться до действия присоединения/запуска.

Каждый экземпляр, созданный с помощью этой функции, получает переменные экземпляра, перечисленные на этой странице.

Синтаксис действия:

Аргументы:

Аргумент Описание
Object Объект, используемый для экземпляров игроков
Layer OPTIONAL Имя слоя, в котором будут созданы экземпляры игроков. "Instances" по умолчанию (если значение не указано).
