Определить задержку входного кадра (откат)

Это действие определяет фиксированную задержку кадров для локальных входов плеера.

По умолчанию ваша игра будет использовать адаптивную задержку ввода, которая основана на задержке между игроками. Это подходит практически для всех игр, однако некоторые специфические типы игр (например, файтинги) могут предпочесть использовать фиксированную задержку ввода вместо адаптивной, которая может меняться в любое время.

Не рекомендуется изменять этот параметр, если вы не знаете, что именно вы делаете.

Указанное значение задержки должно быть в количестве кадров. Если указать -1, игра вернется к использованию адаптивной задержки. Максимальная задержка, которую можно задать, составляет 10 кадров.
