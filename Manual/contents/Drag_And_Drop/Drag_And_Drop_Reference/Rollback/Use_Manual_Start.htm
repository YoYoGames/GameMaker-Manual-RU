Использовать ручной запуск (откат)

По умолчанию многопользовательская игра автоматически начинается, когда все игроки присоединяются к ней.

Вызов этого действия отключает автозапуск, поэтому игра будет ждать, пока вы запустите ее вручную.

Если используется, это действие должно выполняться до действия присоединения/создания.
