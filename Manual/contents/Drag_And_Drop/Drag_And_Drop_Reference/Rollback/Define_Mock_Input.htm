Определите имитационный вход (откат)

Это действие можно использовать во время Sync Test для назначения временного "имитационного" ввода для указанного игрока. Это отменяет элементы управления, определенные для игрока, и любой случайный ввод из Sync Test.

Синтаксис действия:

Аргументы:

Аргумент Описание
Player ID Номер игрока, который получает этот набор входных данных для подражания
Input Имя входа для определения
Value Входная константа для назначения (см. раздел " Пользовательские элементы управления")

Пример:

Приведенное выше действие определяет шуточные входы fire, left, right, up и down для игрока 1.
