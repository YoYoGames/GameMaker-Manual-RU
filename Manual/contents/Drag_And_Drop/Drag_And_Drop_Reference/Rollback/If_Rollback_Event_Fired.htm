Если произошло событие отката

Это действие используется в событии Rollback Event, чтобы проверить, какой тип события был запущен.

Он позволяет прикреплять действия справа, которые запускаются только при наступлении заданного события.

Аргументы:

Аргумент Описание
Event Событие отката для проверки
Not Включите эту функцию, чтобы проверить, не было ли запущено данное событие (это означает, что условие будет истинным для всех остальных событий).
