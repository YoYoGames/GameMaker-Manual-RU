Получить информацию об откате

Эта функция возвращает структуру, содержащую информацию об игроке. Ее следует использовать только при ручном управлении экземплярами игроков. Если вы используете Define Player, то каждый экземпляр игрока будет автоматически получать эту информацию в виде переменных.

Информацию о содержимом возвращаемой структуры см. в разделе: rollback_get_info

Синтаксис действия:

Аргументы:

Аргумент Описание
Player ID Идентификатор игрока, для которого необходимо получить информацию
Target Переменная, в которой должна храниться возвращаемая структура
