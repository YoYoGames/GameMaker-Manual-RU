Откат многопользовательских действий

Действия отката используются для создания многопользовательских игр с помощью системы отката GameMaker.

Для ознакомления см: Rollback Multiplayer

Список переменных, используемых в системе Rollback, см: Переменные Rollback

Для многопользовательской игры Rollback предусмотрены следующие действия:

  • Создать игру
  • Присоединиться к игре
  • Начать игру
  • Оставить игру
  • Определить вход
  • Определить задержку входного кадра
  • Определите имитационный вход
  • Определить игрока
  • Отображение событий
  • Получить информацию
  • Получить вход
  • Если клавиша нажата (откат)
  • При нажатии клавиши (откат)
  • Если клавиша вверх (откат)
  • Если произошло событие отката
  • Синхронизация по кадру
  • Используйте случайный ввод
  • Используйте ручной запуск
  • Отправить сообщение
