Создать игру с откатом

Это действие запускает новую многопользовательскую игру. Оно принимает количество игроков, которые должны быть подключены для начала игры, и по желанию позволяет отключить проверку синхронизации.

При вызове с отключенным Sync Test система ждет, пока заданное количество игроков присоединится к игре, прежде чем начать игру. Убедитесь, что не создаете и не изменяете управляемые экземпляры до начала игры.

СОВЕТ Вы можете по желанию заставить игру начаться до того, как все игроки присоединятся к ней, выполнив команду Начать игру.

В настоящее время в одной игре могут участвовать не более 4 игроков.

ПРЕДУПРЕЖДЕНИЕ Не рекомендуется устанавливать регион по умолчанию в этом действии, если только вы не позволяете игроку выбрать его через меню.

Синтаксис действия:

Аргументы:

Аргумент Описание
Count of players Количество игроков, которые должны быть подключены для начала игры. Если используется Define Player, это количество экземпляров, которые создаются автоматически.
Sync test Определяет, должен ли быть включен Sync Test; по умолчанию он включен. При отключении функции онлайн функциональность включена.
Region Указывает регион, в котором будет создана виртуальная комната. Для получения информации о получении регионов см. rollback_create_game().
