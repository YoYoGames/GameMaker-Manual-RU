Установить цвет частиц

Это действие задает цвет или цвета, которые будут использоваться при создании частицы sprite и в течение ее жизни. Сначала вы указываете значение ID типа частицы (оно должно совпадать со значением, полученным при создании частицы с помощью действия Create Particle Type ), а затем вы можете задать цвета, которые будут использоваться.

Вы можете выбрать три цвета, и sprite будет смешивать их между собой в течение времени жизни частицы. Например, если частица имеет красный, зеленый и желтый цвета, а время ее жизни составляет 60 игровых кадров, то при создании (кадр 0) она будет красный цвет, затем он будет выцветать в сторону зеленого на половине своего времени жизни (кадр 30), а затем перейдет в желтый в конце своего времени жизни (кадр 60). Если вы используете пользовательский sprite вместо формы частицы, любой цвет, кроме белого, будет смешиваться с используемым sprite.

ПРИМЕЧАНИЕ: Альфа-компонент цвета не используется, только RGB-компоненты из палитры цветов. Для установки альфа-компонента используйте действие Set Particle Alpha.
