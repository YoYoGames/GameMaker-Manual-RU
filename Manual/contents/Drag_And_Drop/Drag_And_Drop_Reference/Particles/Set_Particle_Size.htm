Установить размер частиц

Это действие используется для установки размера частиц, который может быть статичным или установлен на увеличение/уменьшение с течением времени. Вы предоставляете уникальное значение ID типа для установки размера (полученное при создании типа с помощью действия Create Particle Type) и затем задаете диапазон начальных размеров, затем величину увеличения и, наконец, "вихляние". Размер создаваемой частицы - это скалярное значение, где 1 - это тот же размер, что и базовое изображение частицы, 0,5 - половина размера 0,5 - половина размера базового изображения, а значение 2 - удвоенный размер. Если минимальный и максимальный размеры не совпадают, то начальный размер любой создаваемой частицы будет иметь случайное значение между ними.

Опция увеличения размера частицы позволяет ввести значение, которое будет добавляться к базовому скалярному значению каждый игровой кадр времени жизни частицы. Таким образом, если задать это значение, например, 0,1, то каждый игровой кадр к значению масштаба будет добавляться 0,1. Обратите внимание, что здесь можно использовать и отрицательные значения, чтобы изображение частицы также уменьшалось. Обратите внимание, что минимальное уменьшение масштаба, которое принимается, составляет -0,01, а минимальное увеличение - 0,01.

Наконец, вы можете установить параметр "wiggle". Это значение, которое будет генерироваться случайным образом в диапазоне от 0 до заданного вами значения и затем добавляться или вычитаться к размеру на каждом шаге жизни частиц, заставляя нарисованное изображение колебаться по размер. Очевидно, что большие значения более заметны, чем меньшие, и это значение может быть даже отрицательным с максимальным диапазоном от -20 до 20.

Синтаксис действия:

Аргументы:

Аргумент Описание
Type Уникальное значение ID для типа частиц, для которых необходимо установить размер
Min Size Минимальный масштаб для типа частиц (по умолчанию 0,1)
Max Size Максимальный масштаб для типа частиц (по умолчанию 1)
Increase Значение, добавляемое к размеру типа частицы каждый игровой кадр (может быть отрицательным, минимум +/-0.01).
Wiggle Максимальное значение, которое можно произвольно прибавить или отнять от размера частиц за шаг (+/-20).
