Установить спрайт частиц

Это действие можно использовать для установки sprite, который частица может использовать при рисовании. Вы предоставляете уникальное значение ID типа, для которого нужно установить sprite (полученное при создании типа с помощью действия Create Particle Type) и ресурс sprite для использования. Если sprite имеет вложенные изображения, то частица будет анимировать кадры со скоростью, установленной в редакторе изображений для sprite. Также обратите внимание, что если вы установите значение цвета для частицы (с помощью действия Set Particle Colour ), то этот цвет будет смешиваться с используемым sprite, подобно тому, как установка смешивания изображения для экземпляра приводит к смешиванию цвета с экземпляром sprite (для отсутствия смешивания используйте белый цвет).

ПРИМЕЧАНИЕ: Если вы используете это действие, то вы не должны использовать действие Set Particle Shape, так как вы можете иметь только sprite или форму, но не то и другое.
