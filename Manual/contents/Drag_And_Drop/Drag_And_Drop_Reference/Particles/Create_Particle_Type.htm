Создать тип частицы

В ваших проектах может быть несколько различных типов частиц, каждый из которых имеет свои свойства, такие как направление, цвет и т.д., которые задаются с помощью действий для различных типов частиц. Однако сначала вам нужно создать тип частиц с помощью этого действия, которое, по сути, выделяет небольшой объем памяти для хранения всех деталей частиц, чтобы вы могли использовать их в любое время в будущем. Вы задаете переменную для хранения уникального значения ID для типа частицы, а затем во всех будущих действиях, связанных с этой частицей, вы можете обращаться к этой переменной. Вы можете пометить эту переменную как временную локальную переменную, но это будет означать, что она будет удалена в конце сценария действия или события, то есть тип частицы будет по-прежнему существовать, но вы останетесь без средств доступа к ней, поэтому при ее использовании следует соблюдать осторожность (в целом типы частиц никогда не нужно хранить во временной локальной переменной, но для некоторых вещей, таких как притягивание частиц к поверхности, это может потребоваться).

ПРИМЕЧАНИЕ: Созданный тип частиц остается в системной памяти на все время игры, поэтому, если вам понадобится создать его заново в любой момент (например, если вы вызовете действие Restart Game ), то вы должны уничтожить текущий тип частиц, иначе произойдет утечка памяти. Это происходит, когда ресурс добавляется в память, а затем "отсылается", то есть GameMaker больше не имеет возможности взаимодействовать с этим ресурсом. больше. Это приводит к засорению памяти недоступными ресурсами, что, в свою очередь, может вызвать проблемы в вашей игре. Чтобы уничтожить тип частицы, если он больше не нужен, используйте действие Destroy Particle Type.

Вы также можете установить режим наложения, который будет использоваться для частиц. По умолчанию они будут рисоваться в "обычном" режиме наложения, поэтому цвета не будут изменяться в зависимости от того, что было нарисовано под частицей, однако, если опция "Наложение" включена, они будут нарисованы с использованием аддитивного смешивания, что означает, что частицы будут нарисованы с изменением цветов в зависимости от того, что было нарисовано под частицей. Аддитивное смешивание дает очень яркие и "светящиеся" эффекты, и является идеально подходит для взрывов, лазеров, огня и т.д., в то время как обычное смешивание лучше подходит для таких эффектов, как дождь, дым и обломки.
