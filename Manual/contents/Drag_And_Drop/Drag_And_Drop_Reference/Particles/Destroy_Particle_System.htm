Уничтожить систему частиц

Это действие "уничтожит" данную систему частиц (т.е. освободит память, используемую системой частиц). Вы вводите уникальное значение ID системы для уничтожения, полученное при создании системы действием Create Particle System. Это действие следует вызывать всякий раз, когда вам больше не нужна система частиц в вашей игре, или когда вы хотите заново создать систему частиц (например, непосредственно перед вызовом Restart Game). Обратите внимание, что если система частиц не была помечена как постоянная, то вам не нужно вызывать это действие, так как система частиц (и все связанные с ней эмиттеры) будет автоматически очищена на стороне комнаты.

ВАЖНО! Эта функция также уничтожит и удалит все эмиттеры, которые могли быть созданы и связаны с уничтожаемой системой (см. раздел " Создать эмиттер частиц " для получения дополнительной информации).
