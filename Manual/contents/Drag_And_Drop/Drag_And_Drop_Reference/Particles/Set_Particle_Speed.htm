Установленная скорость частиц

Это действие используется для установки скорости движения частицы, которая может быть статичной или увеличиваться/уменьшаться с течением времени. Вы предоставляете уникальное значение ID типа, для которого нужно задать размер (оно было получено при создании типа действием Create Particle Type), затем задаете диапазон начальных скоростей, затем величину увеличения и, наконец, "вихляние". Скорость, с которой создается частица, рассчитывается как пиксели на игровой кадр, поэтому при минимальном значении 2 и максимальном значении 5 частицы будут двигаться со скоростью от 2 до 5 пикселей на игровой кадр.

Опция увеличения скорости частицы позволяет ввести значение, которое будет добавляться к базовому значению скорости каждый игровой кадр времени жизни частицы. Таким образом, если задать значение, например, 0,1, то каждый игровой кадр будет добавлять 0,1 к скорости. Обратите внимание, что здесь можно использовать и отрицательные значения, чтобы заставить частицу замедлиться, хотя скорость никогда не опустится ниже 0. Обратите внимание, что минимальное уменьшение скорости, которое принимается, равно -0,01, а минимальное увеличение - 0,01.

Наконец, вы можете установить параметр "wiggle". Это значение, которое будет генерироваться случайным образом в диапазоне от 0 до заданного вами значения и затем прибавляться или вычитаться к скорости на каждом шаге жизни частиц, заставляя нарисованное изображение колебаться в скорости. Очевидно, что большие значения более заметны, чем меньшие, и это значение может быть даже отрицательным с максимальным диапазоном от -20 до 20.

Синтаксис действия:

Аргументы:

Аргумент Описание
Type Уникальное значение ID для типа частиц, для которых необходимо установить размер
Min Speed Минимальная скорость для данного типа частиц (по умолчанию 0,1)
Max Speed Максимальная скорость для данного типа частиц (по умолчанию 1)
Increase Значение, добавляемое к скорости типа частиц в каждом игровом кадре (может быть отрицательным, минимум +/-0.01).
Wiggle Максимальное значение, которое произвольно добавляется или вычитается из скорости частиц за шаг (+/-20).
