Взорвавшиеся частицы

Это действие позволяет вам в любой момент врывать частицы, не прибегая к помощи эмиттера. Вы предоставляете уникальное значение ID системы для создания частиц (полученное при создании системы действием Create Particle System) и уникальный ID типа частиц для создания (полученный при создании типа частиц действием Create Particle Type), а также позицию x/y для создания частиц и их количество. Позиция может быть абсолютной позицией в комнате, или вы можете отметить ее как относительную позицию по отношению к экземпляру, вызывающему действие.

Вы можете вызывать это действие каждый игровой кадр, чтобы воспроизвести эффект потока эмиттера, или вызвать его один раз в событии (например, в событии Destroy), чтобы просто разорвать частицы.
