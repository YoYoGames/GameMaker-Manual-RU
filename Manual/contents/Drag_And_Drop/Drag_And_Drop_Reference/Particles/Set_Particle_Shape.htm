Установить форму частиц

Это действие можно использовать для установки встроенной формы sprite, которую частица должна использовать при рисовании. Вы предоставляете уникальное значение ID типа, для которого нужно установить форму (полученное при создании типа с помощью действия Create Particle Type) и форму, которую нужно использовать.
