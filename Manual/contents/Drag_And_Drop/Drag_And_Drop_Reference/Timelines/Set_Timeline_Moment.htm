Установите временной момент

Это действие используется для перехода от текущего момента временной шкалы к новому. Временные линии движутся вперед через различные "моменты", и каждый момент обычно занимает один игровой кадр (как событие шага объекта). После запуска временной линии она будет двигаться вперед через моменты, пока не достигнет конца, в этот момент она либо вернется к первому моменту временной шкалы, либо остановится совсем. Однако этот линейный подход к временной шкале может быть изменен с помощью этого действия, чтобы установить другой момент в любое время. момент в любое время, что позволяет создавать собственные временные циклы, пропускать участки временной шкалы при определенных обстоятельствах и т.д.. С помощью этого действия вы просто задаете момент для перехода на временную шкалу, и следующий игровой кадр будет выполняться в этот момент. (обратите внимание, что вы должны назначить временную шкалу экземпляру с помощью действия Set Instance Timeline).
