Выберите

С помощью этого действия вы можете задать несколько различных вариантов (это может быть реальное число, строка, идентификатор ресурса или константа), из которых действие выберет один для возврата. Вы можете добавить дополнительные параметры, нажав на значок плюса рядом с и указать другое значение, которое может быть возвращено. Возвращенное значение будет сохранено в целевой переменной, которая может быть создана ранее или может быть новой временной локальной переменной (если вы установите флажок "Temp").
