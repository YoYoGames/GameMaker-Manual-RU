Создать последовательность

С помощью этого действия можно создать элемент последовательности любой последовательности из браузера активов. Вы задаете имя последовательности и позицию x/y в помещении, где она должна быть создана, а также слой, на котором она должна быть создана (этот слой должен быть слоем актива). Вы можете отметить флаги "relative", чтобы расположить последовательность относительно экземпляра, выполняющего блоки кода действий. Когда вы создаете последовательность в игре таким образом, ей присваивается уникальное значение ID элемента, с помощью которого вы можете идентифицировать последовательность в дальнейших действиях, поэтому данное действие позволяет вам определить переменную "target" для хранения этого значения ID (как показано в примере ниже). Обратите внимание, что цель может быть помечена как временная (локальная) переменная, которая будет создана для данного действия и может быть использована во всех последующих действиях для события, в котором находится действие.

Аргументы:

Аргумент Описание

x Положение x в комнате для создания последовательности (может быть относительно вызывающего экземпляра).
y Положение y в комнате для создания последовательности (может быть относительно вызывающего экземпляра).
layer Имя (строка) слоя актива для создания последовательности.
Target Переменная, на которую будет нацелено возвращаемое значение.
