Получить длину последовательности

С помощью этого действия вы можете получить длину (в кадрах) заданного элемента последовательности в комнате. Вы предоставляете идентификатор элемента последовательности (возвращенный действием Создать последовательность), и это действие вернет его длину в целевой переменной. Это будет количество кадров, в течение которых выполняется последовательность.

Обратите внимание, что целевая переменная может быть помечена как временная (локальная), которая будет создана для действия и может использоваться во всех последующих действиях для события, в котором происходит действие.

Target Целевая переменная для хранения длины.
