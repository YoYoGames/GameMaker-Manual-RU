Получить головку последовательности

С помощью этого действия вы можете получить текущее положение головки воспроизведения заданного элемента последовательности в комнате. Вы указываете ID элемента последовательности (возвращенный действием Create Sequence), и это действие вернет текущий кадр, на котором находится головка воспроизведения последовательности, в переменную target. Информацию о головке воспроизведения см. в разделе Использование листа допинга.

Обратите внимание, что целевая переменная может быть помечена как временная (локальная), которая будет создана для данного действия и может использоваться во всех последующих действиях для события, в котором происходит действие.

Аргумент Описание
Variable Идентификатор элемента последовательности для получения позиции головки.
Target Целевая переменная для фиксации положения головы.
