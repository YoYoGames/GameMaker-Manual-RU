Получить позицию последовательности

С помощью этого действия вы можете получить текущую позицию для данного элемента последовательности в комнате. Вы предоставляете ID элемента последовательности (возвращенный действием Создать последовательность), и это действие возвращает его положение по X и Y в целевых переменных. Обратите внимание, что целевые переменные могут быть помечены как временные (локальные) переменные, которые будут созданы для данного действия и могут быть использованы во всех последующих действиях для события, в котором выполняется действие.
