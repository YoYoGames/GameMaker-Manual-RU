Выходная игра

С помощью этого действия вы можете завершить игру, вызвав событие Other - Game End Event. Обратите внимание, что это происходит не в момент вызова действия, а в конце текущего поэтому любые действия, которые вы выполняете после вызова этого действия, будут выполняться до тех пор, пока не будет достигнут конец события или сценария.

Обратите внимание, что это действие имеет следующие ограничения:

  • Он сообщит об ошибке на цели iOS, поскольку это противоречит условиям их торговой площадки.
  • Он не будет иметь никакого эффекта при вызове для целей HTML5 и Windows UWP.
  • Он будет работать на Windows (включая игры, созданные для Steam), Android, Ubuntu (Linux) и Mac.
