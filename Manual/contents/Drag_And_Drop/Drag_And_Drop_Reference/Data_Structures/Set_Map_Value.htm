Установить значение карты

С помощью этого действия вы можете добавить пару ключ/значение в структуру данных карты. Вы передаете переменную, которая содержит индекс карты (возвращается действием Create Map), затем задаете "ключ" (который является идентификатором в карте для значения), а затем "значение", которое должно быть связано с этим ключом. Оба ключа могут быть либо целочисленными значениями (например, 0, 200, 36712 и т.д.), либо строками (например, "Name", "Shield" и т.д.), а значение может быть любым допустимым типом данных.

Аргумент Описание

Key Ключ для установки (целое число или строка)
Value Значение, которое будет храниться с заданным ключом
