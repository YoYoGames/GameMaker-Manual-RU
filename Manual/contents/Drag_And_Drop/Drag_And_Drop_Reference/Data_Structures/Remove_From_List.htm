Удалить из списка

Это действие можно использовать для удаления элемента с индексом из заданного списка. Вы передаете переменную, которая хранит индекс списка (возвращенный действием Создать список) и позицию индекса в списке для удаления, где индексная позиция находится в диапазоне от 1 до (длина списка -1). Это действие ничего не возвращает, поэтому если вам нужно значение в позиции индекса, то перед удалением индекса следует использовать Get List Item At.

Index Индекс в списке для удаления

Если оно существует, то элемент удаляется из списка, а экземпляр уничтожается.
