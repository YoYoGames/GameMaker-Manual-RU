Получить значение сетки

С помощью этого действия вы можете установить значение для определенной ячейки в структуре данных сетки. Вы передаете переменную, которая содержит индекс сетки (возвращается действием Создать сетку), затем указываете позицию x и y (столбец и строка) для ячейки, которую нужно установить, и, наконец, значение для этой ячейки. Вы также должны задать целевую переменную (которая может быть помечена как временная локальная переменная) для хранения возвращаемого значения ячейки, причем значение может быть любым допустимым типом данных. Обратите внимание, что позиции x и y для ячейки должны быть целыми целыми числами в пределах сетки.

Аргументы:

Аргумент Описание
Grid Индекс (хранится в переменной) сетки, которую нужно получить
x Положение ячейки вдоль оси x для получения (столбец)
y Положение ячейки вдоль оси y для получения (строка)
Target Целевая переменная, которая будет использоваться для хранения возвращаемого значения

Затем это значение проверяется, и если оно является значением ID экземпляра, то этот экземпляр уничтожается, а ячейка сетки устанавливается на ключевое слово noone.
