Перемещение в стек

С помощью этого действия вы можете поместить значение в структуру данных стека. Вы предоставляете переменную, которая содержит индекс стека (возвращается действием Create Stack), а затем указываете значение, которое нужно протолкнуть на вершину. При использовании стековых значения всегда добавляются на "вершину" стека, и никогда в конец или в середину (если вам нужна такая функциональность, используйте структуру данных list).
