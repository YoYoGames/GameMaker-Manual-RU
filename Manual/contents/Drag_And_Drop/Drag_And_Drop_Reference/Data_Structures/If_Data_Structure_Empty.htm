Если структура данных пуста

С помощью этого действия вы можете проверить, пуста ли заданная структура данных или нет. Сначала вам нужно выбрать тип структуры данных для проверки (либо список, либо карта, либо стек - никаких сеток, поскольку они не могут быть пустыми), а затем предоставить переменную переменную, которая содержит индекс структуры данных (как возвращается действиями Create List, Create Map и Create Stack). Действие вернет true, если если структура данных пуста, и false, если нет. Обратите внимание, что вы можете установить флаг "not" в действии, чтобы проверить, содержит ли структура данных данные, проверив, не пуста ли она.
