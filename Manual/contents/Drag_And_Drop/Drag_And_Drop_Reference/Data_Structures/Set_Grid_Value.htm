Установить значение сетки

С помощью этого действия вы можете установить значение для определенной ячейки в структуре данных сетки. Вы передаете переменную, которая содержит индекс сетки (возвращается действием Создать сетку), затем указываете позицию x и y (столбец и строка) для ячейки, которую нужно установить, и, наконец, значение для этой ячейки. Значение может быть любым допустимым типом данных, но позиции x и y должны быть целыми целыми числами в пределах сетки.

Аргументы:

Аргумент Описание
Grid Индекс (хранится в переменной) сетки для установки
x Позиция ячейки вдоль оси x для установки (столбец)
y Положение ячейки вдоль оси y для установки (строка)
Value Значение, на которое нужно установить ячейку сетки
