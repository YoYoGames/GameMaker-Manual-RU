Четкая сетка

С помощью этого действия вы можете очистить структуру данных сетки до одного значения (каждая ячейка в сетке будет установлена на заданное значение). Вы передаете переменную, содержащую индекс сетки (как возвращается действием Создать сетку) и затем задаете значение, до которого нужно очистить сетку.

Grid Индекс (хранится в переменной) сетки, которую нужно очистить
Value Значение, до которого нужно очистить сетку
