Вставка в список

С помощью этого действия вы можете восстановить индексную позицию элемента в заданном списке. Вы предоставляете переменную, которая хранит индекс списка (возвращенный действием Создать список) и значение, которое вы хотите найти позицию индекса в списке, а также целевую переменную для хранения возвращенной индексной позиции элемента (которая может быть помечена как временная локальная переменная, которая будет использоваться до конца сценария или события). Когда вы вставляете значение в список, все индексы выше заданного индекса для вставки будут сдвинуты на один вверх, поэтому если у вас есть список с 4 элементами (индексные позиции от 0 до 3) и вы помещаете новый элемент в индексную позицию 2, предыдущий элемент 2 станет элементом 3, а предыдущий элемент 3 станет элементом 4. Обратите внимание, что если искомое значение не существует в списке, то не будет возвращаться индексная позиция списка, и возвращаемое значение будет равно -1.
