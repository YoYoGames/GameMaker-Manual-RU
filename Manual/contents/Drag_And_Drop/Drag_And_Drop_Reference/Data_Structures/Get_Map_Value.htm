Получить значение карты

С помощью этого действия вы можете получить значение, связанное с заданным ключом в структуре данных карты. Вы предоставляете переменную, которая содержит индекс карты (как возвращается действием Create Map), а затем задаете "ключ" (который это идентификатор значения в карте), а также предоставляете целевую переменную для хранения возвращаемого значения (которая может быть помечена как временная локальная переменная).

ПРИМЕЧАНИЕ: Если вы зададите ключ, который не существует в карте, действие вернет undefined. Проверить это можно с помощью действия If Undefined.

Аргументы:

Аргумент Описание
Map Индекс (хранится в переменной) карты, из которой нужно получить значение
Key Ключ, который нужно получить (вещественный или строковый)
Target Целевая переменная для хранения возвращаемого значения карты
