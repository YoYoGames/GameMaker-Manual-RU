Получить количество элементов списка

Вы можете использовать это действие, чтобы узнать, сколько записей содержится в списке. Вы предоставляете переменную, которая содержит индекс списка (возвращенный действием Создать список), и целевую переменную для хранения возвращаемого числа элементов в списке (которая может быть помечена как временная локальная переменная, которая будет использоваться до конца сценария или события).
