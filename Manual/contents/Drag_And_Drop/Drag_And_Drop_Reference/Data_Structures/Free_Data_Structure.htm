Свободная структура данных

Это действие освободит структуру данных из памяти и должно вызываться всякий раз, когда структура данных больше не нужна, например, в событии Destroy даже экземпляра или в событии Room End и т.д.. Освобождение структуры данных означает, что никакие дальнейшие действия не могут быть использованы для ссылки на нее, поскольку она больше не существует, хотя вы также можете проверить это с помощью действия If Data Structure Exists.

Обратите внимание, что если структура данных не имеет индекса, назначенного переменной глобальной области видимости, она не будет переноситься из одной комнаты в другую. Например, если вы создадите экземпляр игрока и структура данных list будет присвоена переменной этого экземпляра, список нужно будет освободить, если игрок будет уничтожен или если изменится комната с экземпляром игрока в ней (если только экземпляр игрока не помечен как постоянный), поэтому экземпляр игрока будет иметь два действия Free Data Structure для обоих этих событий. Важно, чтобы все созданные структуры данных уничтожались таким образом, когда они больше не нужны, чтобы предотвратить утечку памяти, которая может вызвать проблемы с производительностью или даже крах вашей игры.
