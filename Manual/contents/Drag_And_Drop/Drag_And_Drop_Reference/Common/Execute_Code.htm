Выполнить код

Это действие просто добавляет блок редактора кода в список GML Visual, куда вы можете добавить любой действительный код GML, который вам нужен для объекта. Обратите внимание, что весь код в действии будет выполнен до того, как будет выполнено следующее до того, как будет выполнено следующее действие в списке. Чтобы было легче отследить, что делают эти действия, вы можете назвать их, используя идентификатор JSDoc @description, как показано на рисунке ниже. на рисунке ниже:
