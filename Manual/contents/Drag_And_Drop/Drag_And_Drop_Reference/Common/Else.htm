Else

Действие "else" используется только в сочетании с If Variable (или любым другим действием "if") и представляет собой способ определения набора действий, которые должны быть выполнены, если оценка "if" возвращает false. Например, у вас может быть действие "если", которое проверяет, если переменная здоровья игроков меньше или равна 0, и если это значение равно true, вы уничтожаете экземпляр. Однако если вы хотите выполнить другой набор действий кода, если здоровье игрока не меньше 0, то вы можете добавить "else" и затем указать действия, которые должны быть выполнены, пока if оценивается как false.
