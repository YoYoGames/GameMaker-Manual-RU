Выполнить сценарий

Это действие можно использовать для вызова функции сценария или функции, определяемой пользователем. В целом, это действие может использоваться только для поддержки наследия и не должно применяться в текущих или новых проектах. Чтобы использовать вы передаете актив скрипта для вызова - или имя переменной пользовательской функции для вызова - и затем предоставляете любые аргументы, которые он требует, т.е. любые значения (строки, целые числа, индексы ресурсов и т.д.), которые вы хотите ввести в скрипт. или функцию, чтобы она могла их использовать. При необходимости вы можете добавить дополнительные аргументы, нажав на значок плюса рядом с действием.

Некоторые сценарии и пользовательские функции могут также возвращать значение для последующего использования в списке блоков действий, поэтому вы можете указать целевую переменную, которая должна хранить это возвращаемое значение. Она может быть создана ранее или может быть новой временной локальной (если вы установите флажок "Temp"). Поля "цель" и "аргумент" можно оставить пустыми, если они не требуются.

Обратите внимание, что обычно лучше использовать действие Function Call для вызова пользовательских функций, и обычно не нужно использовать это действие или вызывать актив сценария непосредственно в коде.

Аргументы:

Аргумент Описание
Script Имя сценария или определяемой пользователем функции для вызова.
Argument0 ... Argument3 Различные аргументы (значения), которые должны быть переданы скрипту или функции (неиспользуемые аргументы можно оставить пустыми)
Target Имя переменной, которая будет использоваться для возвращаемых значений (можно оставить пустым).
