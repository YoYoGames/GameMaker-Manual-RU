Вызов функции

Это действие можно использовать для вызова любой функции GML или любой функции, определенной пользователем. Вы задаете функцию для вызова, а затем предоставляете любые дополнительные аргументы, которые она требует, т.е. любые значения (строки, целые числа, индексы ресурсов и т.д...), которые вы хотите ввести в функцию, чтобы она могла их использовать. Вы можете добавить дополнительные аргументы, нажав на значок плюса рядом с и убедитесь, что количество аргументов соответствует количеству аргументов используемой функции (т.е. если функция требует 3 аргумента, вы должны ввести 3 аргумента). Некоторые функции могут также возвращать значение для последующего использования в блоке действия список, и поэтому вы также можете указать целевую переменную, которая должна хранить это возвращаемое значение, которая может быть создана ранее или может быть новой временной локальной переменной (если вы установите флажок "Temp"). И цель и поле аргумента можно оставить пустыми, если они не требуются.

Аргументы:

Аргумент Описание
Function Функция GML для вызова.
Argument0 ... ArgumentX Различные аргументы (значения), которые должны быть переданы в функцию.
Target Имя переменной, на которую будут нацелены возвращаемые значения (можно оставить пустым).
