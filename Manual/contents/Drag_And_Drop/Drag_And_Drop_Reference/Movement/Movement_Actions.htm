Библиотека действий движения

В любой игре перемещение и положение имеют первостепенное значение, поэтому в GameMaker имеется полный набор действий для любой ситуации. Два основных способа перемещения экземпляра - это либо задать фактическую позицию (x/y) в пределах комнаты, либо задать вектор скорости/направления. комнаты, либо задать вектор скорости/направления. Показанные здесь действия охватывают оба способа, и вы можете использовать переменную Common Action Assign Variable для непосредственной установки положения x и y самостоятельно.
