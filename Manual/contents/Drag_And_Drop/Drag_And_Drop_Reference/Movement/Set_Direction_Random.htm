Установить направление Случайно

Это действие используется для установки направления движения экземпляра на одно или несколько случайных направлений, выбранных из 8 кардинальных точек. Вы можете щелкнуть одну или несколько из 8 стрелок направления внутри действия, и действие выберет одно из них случайным образом для использования в качестве направления для экземпляра. Если вы не выберете ни одну из стрелок, действие не повлияет на текущее направление экземпляра.
