Направление заданного значения

Это действие используется для задания направления из вектора, образованного текущей позицией экземпляра и заданной позицией. Вы указываете позиции x и y, на которые нужно "указать" - они могут быть помечены как относительные по отношению к позиции экземпляра - и направление от экземпляра к этой точке будет использоваться для экземпляра.
