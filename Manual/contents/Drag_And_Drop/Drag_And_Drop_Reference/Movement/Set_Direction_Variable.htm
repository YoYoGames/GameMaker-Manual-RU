Установить переменную направления

Это действие используется для задания направления для экземпляра, которое не фиксировано с шагом 45°. Вы задаете значение направления от 0° до 360° - где вправо - 0°, вверх - 90°, влево - 180° и вниз - 270° - а также можете установить флаг relative, чтобы добавить/вычесть значение, которое вы задаете, к текущему направлению.

Аргументы:

Аргумент Описание
Direction Направление от 0° до 360° (значения вне диапазона будут обернуты).
