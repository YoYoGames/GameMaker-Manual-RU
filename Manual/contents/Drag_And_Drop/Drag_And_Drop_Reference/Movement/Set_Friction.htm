Установить трение

Это действие используется для установки "трения" для экземпляра. Трение означает, что определенная величина будет вычитаться из вектора скорости каждый игровой кадр, где вы указываете величину вычитания. Обычно лучше всего работают меньшие значения, например 0.2, но фактическое значение будет зависеть от настроек вашего проекта. Если вы установите флаг relative, то указанная сумма будет добавлена к текущему значению трения вместо того, чтобы установить его.
