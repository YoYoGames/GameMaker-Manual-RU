Перейти к точке

Это действие используется для перемещения экземпляра из одного места в другое. Экземпляр не будет плавно перемещаться в новое место, а будет "прыгать", чтобы быть помещенным туда, так что если значения достаточно велики, то он "проскочит" все другие экземпляры по пути. Можно задать абсолютные значения для перемещения экземпляра, а можно установить флаг "relative", чтобы экземпляр перемещался относительно текущей позиции. Например, если вы хотите, чтобы экземпляр переместился только на 10 пикселей влево, установите значение x равным -10, а значение y равным 0 и отметьте флаг "относительно" для обоих значений. Можно использовать отрицательные значения для абсолютных координат, и тогда экземпляр будет перемещен за пределы области базовой комнаты.
