Удалить буфер

Это действие можно использовать для удаления буфера из памяти, освобождая ресурсы, использованные для его создания, и удаляя любые данные, которые он может содержать в данный момент. Вы предоставляете уникальное значение ID буфера, возвращаемое действием Создать буфер. Обратите внимание, что вы можете выбрать несколько буферов для удаления, нажав на значок плюса рядом с действием и добавив еще один идентификатор буфера.

ПРИМЕЧАНИЕ: Важно всегда удалять любые динамически создаваемые ресурсы из памяти, когда они больше не нужны, чтобы предотвратить утечку памяти.
