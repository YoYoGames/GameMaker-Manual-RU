Получить размер буфера

Это действие можно использовать для получения размера буфера в байтах. Вы предоставляете уникальное значение ID буфера, возвращаемое действием Create Buffer, а затем указываете целевую переменную для возврата значения размера. Цель может быть помечена как временная локальная переменная, в этом случае она будет создана для вас и удалена из памяти в конце текущего сценария действия или события.
