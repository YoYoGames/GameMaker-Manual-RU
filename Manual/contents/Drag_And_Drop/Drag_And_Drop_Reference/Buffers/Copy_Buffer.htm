Буфер копирования

Это действие можно использовать для копирования всех данных, хранящихся в одном буфере, в другой. При использовании двух буферов и копировании из одного в другой оба буфера должны быть предварительно созданы с помощью действия Create Buffer, а в качестве аргументов "от" и "до" указываются уникальные идентификаторы буферов.

ПРИМЕЧАНИЕ: Копирование в один и тот же буфер невозможно.
