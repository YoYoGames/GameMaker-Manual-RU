If End Of Buffer

Это действие можно использовать для проверки того, достигла ли позиция чтения/записи конца буфера. Вы указываете уникальное значение ID буфера, возвращаемое действием Create Buffer, а затем действие возвращает true, если позиция чтения/записи достигла конца всех доступных данных в буфере, или false, если нет. Вы можете установить флаг "not", чтобы действие проверяло, не находится ли позиция чтения/записи в конце буфера.

Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже:
