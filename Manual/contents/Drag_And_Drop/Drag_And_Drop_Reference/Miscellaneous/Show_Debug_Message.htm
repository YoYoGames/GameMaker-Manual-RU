Показать отладочное сообщение

С помощью этого действия вы можете отправить отладочное сообщение на выход компилятора. Это сообщение может быть либо строкой, либо вещественным числом, но если вы хотите использовать оба варианта, то число должно быть преобразовано в строку (см. действие Number To String). Обратите внимание, что вы можете вывести несколько сообщений с помощью этого действия, щелкнув по значку , чтобы развернуть действие с дополнительными полями вывода.
