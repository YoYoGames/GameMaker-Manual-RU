Установить скорость анимации

Этот блок действий устанавливает image_speed (скорость анимации) для sprite, назначенного данному экземпляру. Вы устанавливаете идеальное количество кадров в секунду в редакторе спрайтов, а затем можете использовать это значение чтобы изменить его. По умолчанию скорость анимации установлена на 1, что означает, что игра будет пытаться поддерживать количество кадров в секунду, которое вы определили для sprite, но вы можете использовать значения больше 1 для более быстрой анимации (кадры могут быть пропущены), или десятичные значения для более медленной анимации (кадры будут показываться с разным шагом), или даже отрицательные значения для изменения скорости анимации.
