Мероприятие для родителей по вызову

Этот блок действий выполнит событие родительского объекта, дочерним объектом которого является данный экземпляр, прежде чем продолжить выполнение действий в реальном экземпляре. Когда вы назначаете родительский объект экземпляру, все события этого объекта "наследуются" дочерним объектом. Однако если вы затем добавите событие в дочерний объект, оно отменит родительское событие. Используя этот блок действий, вы можете заставить дочерний объект выполнять как родительское событие, так и событие в самом экземпляре. Дополнительные сведения о родителях см. в разделе Редактор объектов.
