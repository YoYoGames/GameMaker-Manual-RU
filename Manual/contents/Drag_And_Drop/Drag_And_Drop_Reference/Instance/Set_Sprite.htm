Установить спрайт

Этот блок действий устанавливает sprite_index экземпляра на другой sprite из браузера активов. Вы выбираете sprite, на который нужно перейти, а затем указываете изображение (кадр анимации), которое нужно показать. Обратите внимание, что если вы не хотите, чтобы sprite анимировался после установки кадра, вам нужно использовать действие Set Animation Speed и установить его на 0, или если вы хотите, чтобы sprite продолжал анимировать, то установите значение для использования встроенной переменной image_index. вместо этого.

Изменение sprite также изменит маску столкновений экземпляра, если только вы не указали отдельный mask_index в редакторе объектов. Обратите внимание, что для того, чтобы изменения в этом действии чтобы изменения в этом действии были видимыми, экземпляр должен либо не иметь события рисования (и тогда GameMaker по умолчанию нарисует sprite), либо быть нарисованным с помощью действия Draw Self.
