Установить вращение экземпляра

Этот блок действий задает угол image_angle (поворот) экземпляра. Угол измеряется в градусах: вправо - 0°, вверх - 90°, влево - 180° и вниз - 270°. Если установить эту переменную в 0, то экземпляр будет сброшен, то есть он будет рисовать любой sprite, назначенный в том виде, в котором он был определен в редакторе sprite. Обратите внимание, что для того, чтобы изменения в этом действии были заметны, экземпляр должен либо не иметь события рисования (и тогда GameMaker по умолчанию нарисует sprite), либо быть нарисованным с помощью действия Draw Self.
