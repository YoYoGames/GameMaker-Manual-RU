Установить цвет экземпляра

Этот блок действий устанавливает цвет image_blend для "смешивания" с экземпляром sprite. Значение по умолчанию - шестнадцатеричный код $FFFFFFFF (который представляет белый цвет в формате ARGB ), и это нарисует sprite без смешивания и с полной альфой. Любое другое значение (включая внутренние константы цвета, такие как c_red, или c_aqua) приведет к смешиванию указанного цвета с исходным sprite, а также установит компонент image_alpha компонент для данного экземпляра, но только если установлен флажок Использовать альфу. При этом будет перезаписано любое значение альфа, установленное ранее с помощью действия Set Instance Alpha.

Обратите внимание, что для того, чтобы изменения в этом действии были видны, экземпляр должен либо не иметь события рисования (и тогда GameMaker по умолчанию нарисует sprite), либо быть нарисованным с помощью действия Draw Self. Важно важно отметить, что на платформе HTML5 следует ограничить количество смешиваний (если только не используется WebGL), так как каждое смешивание sprite должно кэшироваться отдельно, поэтому наличие большого количества смешиваний sprites может негативно сказаться на производительности.
