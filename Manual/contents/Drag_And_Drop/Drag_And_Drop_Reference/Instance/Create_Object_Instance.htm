Создать экземпляр объекта

С помощью этого действия можно создать экземпляр любого объекта из браузера активов. Вы указываете имя объекта и позицию x/y в помещении, где он должен быть создан, а также имя слоя, на котором будет создан экземпляр (он должен быть Instance Layer). Вы можете установить флаги "relative", чтобы расположить экземпляр относительно экземпляра, выполняющего блоки кода действия. При создании экземпляра в игре ему присваивается уникальное значение ID, с помощью которого вы можете Поэтому данное действие позволяет вам определить переменную "target", которая будет хранить это значение ID (как показано в примере ниже). Обратите внимание, что цель может быть помечена как временная (локальная) переменная, которая будет будет создана для данного действия и может быть использована во всех последующих действиях.

Target Переменная, на которую будет нацелено возвращаемое значение.
