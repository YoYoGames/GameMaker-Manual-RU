Установить альфу экземпляра

Этот блок действий устанавливает значение image_alpha для экземпляра. Альфа изображения - это значение "прозрачности" для рисования любого sprite, назначенного данному экземпляру, и имеет значение от 0 до 1, где 0 будет полностью прозрачным, а 1 - полностью непрозрачным. полностью прозрачным, а 1 - полностью непрозрачным. Обратите внимание, что если на sprite есть прозрачные или полупрозрачные пиксели, то их прозрачность будет масштабироваться на эту величину, поэтому если на sprite есть несколько 50% прозрачных пикселей, а альфа экземпляра установлена на 0,5, то итоговая альфа для прозрачных пикселей будет равна 25%. для прозрачных пикселей будет равна 25%. Вы можете установить это значение как относительную величину

Обратите внимание, что для того, чтобы изменения в этой переменной были видны, экземпляр должен либо не иметь события рисования (и тогда GameMaker по умолчанию нарисует sprite), либо быть нарисованным с помощью действия Draw Self .
