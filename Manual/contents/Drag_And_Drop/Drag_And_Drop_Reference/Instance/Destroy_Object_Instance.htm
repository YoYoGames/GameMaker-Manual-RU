Уничтожить экземпляр объекта

Вы можете вызывать это действие всякий раз, когда хотите "уничтожить" экземпляр, что приведет к его удалению из комнаты. Обратите внимание, что при уничтожении экземпляра его событие Destroy вызывается сразу после действия, вызывающего уничтожение. Также обратите внимание, что хотя событие Destroy Event выполняется в тот же момент, когда вызывается действие, экземпляр не удаляется из игры немедленно, а продолжает выполнять код, содержащийся в текущем событии, до конца, и только когда текущее событие закончится, экземпляр будет удален из игры, и тогда будет вызвано событие Clean Up Event.
