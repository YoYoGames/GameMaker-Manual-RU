Получить количество экземпляров

С помощью этого действия вы можете узнать количество экземпляров любого объекта, активного в данный момент в комнате. Вы можете выбрать любой объект из списка ресурсов или использовать специальное ключевое слово all для получить количество всех активных экземпляров. Вам также необходимо указать целевую переменную для возврата значения, чтобы его можно было использовать в дальнейших действиях. Если обозначить целевую переменную как "Temp", то будет создана новая временная (локальную) переменную для хранения возвращаемого значения, как показано в примере ниже.
