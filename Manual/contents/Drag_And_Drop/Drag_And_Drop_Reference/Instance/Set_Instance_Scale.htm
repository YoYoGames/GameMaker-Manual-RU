Установить масштаб экземпляра

Этот блок действий устанавливает значения image_xscale и image_yscale для экземпляра. Входные значения здесь являются модификаторами, которые будут применены к sprite, назначенному экземпляру, где масштаб 1 (значение по умолчанию) означает отсутствие масштабирования (1:1), меньшие значения будут масштабироваться в меньшую сторону (0,5, например, уменьшит ширину вдвое), а большие значения будут масштабироваться в меньшую сторону (1:1). масштабирование (1:1), меньшие значения будут масштабироваться вниз (0,5, например, уменьшит ширину sprite в два раза), а большие значения будут масштабироваться вверх. Вы можете использовать отрицательные значения, чтобы перевернуть sprite и масштабировать его, если только используемое значение не равно -1 (в этом случае картинка sprite просто переворачивается или зеркально отражается относительно своего начала координат без масштабирования). Вы можете установить горизонтальный или вертикальный масштаб относительно текущих значений, в этом случае вы будете добавлять или вычитать новое значение из текущих значений масштаба.

Обратите внимание, что изменение этих значений также изменит способ обнаружения столкновений экземпляром, если только вы не задали отдельную mask_index (маску столкновений) в редакторе объектов. Обратите внимание, что чтобы изменения в этой переменной были видны, экземпляр должен либо не иметь события рисования (и тогда GameMaker по умолчанию нарисует sprite), либо быть нарисованным с помощью действия Draw Self.
