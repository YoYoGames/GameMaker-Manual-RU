Получить скорость следования по траектории

Это действие получает скорость, в пикселях на игровой кадр, с которой экземпляр настроен следовать по пути (обратите внимание, что вы должны вызвать действие Start Following Path для назначения пути экземпляру, чтобы это действие вернуть значение, отличное от 0). Возвращаемое значение будет сохранено в указанной вами переменной target, или вы можете отметить переменную как временную локальную переменную, чтобы создать ее и присвоить значение.
