Установите скорость следования по траектории

Это действие устанавливает скорость, в пикселях на игровой кадр, с которой экземпляр должен следовать по пути (обратите внимание, что для того, чтобы это действие имело эффект, необходимо вызвать действие Start Following Path, чтобы назначить путь экземпляру. какой-либо эффект). Можно использовать отрицательные значения, чтобы обозначить, что экземпляр должен следовать по пути в обратном направлении, и стоит отметить, что скорость, заданная здесь, может быть изменена модификатором скорости, установленным в редакторе пути для каждой точки пути, поэтому если вы задана скорость пути 2, а в пути одна из точек имеет модификатор скорости 200%, то фактическая скорость пикселей на игровой кадр в этой точке будет равна 4.
