Установить положение вдоль траектории

Это действие устанавливает положение экземпляра вдоль пути. Значение нормализуется от 0 до 1, поэтому если вы установите его, например, на 0,5, экземпляр будет перемещен точно в середину пути. Обратите внимание, что для того, чтобы действие имело эффект, необходимо вызвать действие Start Following Path для назначения пути экземпляру.
