Получить главный объем

Это действие возвращает значение основной громкости для звуковой системы. Вы задаете целевую переменную для возврата значения громкости (вы можете отметить ее как временную локальную переменную). Возвращаемое значение будет находиться в диапазоне от 0 до 1, где 0 - тишина, а 1 - полная громкость. громкость.

Пример:

Приведенный выше код блока действий проверяет глобальную переменную, чтобы узнать, является ли она true или false. Если она равна true, то извлекается значение основной громкости звуковой системы и сохраняется во временной локальной переменной. Затем проверяется, не равно ли это значение 0,5, и если это не так, то основная громкость звуковой системы устанавливается на 0,5. Если значение глобальной переменной равно false, то основная громкость устанавливается равной 1.
