Set Audio Loop State

This action is used to set the loop state of an already playing sound, i.e. one played with the Play Audio action.

 

Action Syntax:

Arguments:

Argument Description
Sound The sound, played with Play Audio, to set the loop state of
Loop Whether to loop the sound between its loop start and end or not (true or false)
