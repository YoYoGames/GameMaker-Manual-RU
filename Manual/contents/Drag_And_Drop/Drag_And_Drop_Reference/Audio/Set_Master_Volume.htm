Установите мастер-громкость

С помощью этого действия вы можете установить громкость для всей звуковой системы. Значение громкости может быть от 0 (тихо) до 1 (полная громкость), шкала линейная, так что значение 0,5 означает половину громкости. Обратите внимание, что это действие повлияет на все экземпляры всех звуков, воспроизводимых в данный момент в комнате, и повлияет на все будущие экземпляры всех воспроизводимых звуков. Это действие также не зависит от громкости, установленной для каждого отдельного звука, поэтому если вы настроили звук на воспроизведение с громкостью 0,5 и затем установить главную громкость на 0,5, звук будет воспроизводиться с громкостью 0,25.

Пример:

Приведенный выше код блока действий проверяет глобальную переменную, чтобы узнать, является ли она true или false. Если она равна true, то извлекается значение основной громкости звуковой системы и сохраняется во временной локальной переменной. Затем проверяется, не равно ли это значение 0,5, и если это не так, то основная громкость звуковой системы устанавливается на 0,5. Если значение глобальной переменной равно false, то основная громкость устанавливается равной 1.
