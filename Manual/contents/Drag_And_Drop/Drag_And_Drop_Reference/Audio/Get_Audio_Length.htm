Получить длину звука

Это действие можно использовать для получения длины звука в вашей игре. Вы указываете ресурс звука для получения длины и переменную target для возврата значения (или вы можете отметить target как временную переменную, чтобы действие создало переменную для использования до конца события или сценария). Длительность будет возвращена в целевую переменную (которую можно сделать временной) в секундах и может быть десятичным значением.

Аргументы:

Аргумент Описание
Sound Звуковой ресурс для получения длины
Target Целевая переменная для хранения возвращаемого значения

Пример:

Приведенный выше код блока действий получает длительность воспроизведения заданного звукового ресурса и сохраняет ее ее во временную локальную переменную. Затем она используется для генерации случайного значения от 0 до длины звука, которое затем сохраняется в другой временной локальной переменной. Это новое случайное значение затем используется для установки начальной позиции для и звук воспроизводится.
