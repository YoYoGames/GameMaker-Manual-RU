Получение звукового сигнала

Это действие извлекает высоту тона предоставленного вами звукового ресурса. Возвращаемое значение будет множителем для фактического записанного тона образца, где значение 1 означает отсутствие изменений. Более подробную информацию о высоте тона см. в действии Set Audio Pitch.

Аргументы:

Аргумент Описание
Sound Звуковой ресурс для получения высоты тона
Target Целевая переменная для хранения возвращаемого шага

Пример:

Приведенный выше код блока действий сначала получает высоту тона для заданного звука и сохраняет ее во временной локальную переменную. Затем он проверяет нажатие клавиши, и если таковое обнаружено, значение высоты тона проверяется на то, меньше ли оно 2. Если да, то звук останавливается, к переменной высоты тона добавляется 0,1, и затем эта переменная используется для установки высоты тона звука снова перед воспроизведением. высоты тона звука перед его воспроизведением.
