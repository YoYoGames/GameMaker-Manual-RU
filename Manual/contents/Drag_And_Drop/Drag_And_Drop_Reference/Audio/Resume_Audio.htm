Резюме Аудио

Это действие можно использовать для возобновления ранее приостановленного звука. Это действие можно использовать для любого звука, который был приостановлен с помощью Pause Audio или Pause All Audio, и оно не влияет на те звуки, которые которые не были приостановлены ранее.

Аргументы:

Аргумент Описание
Sound Возобновление воспроизведения звукового ресурса

Пример:

Приведенный выше код блока действий проверяет глобальную переменную, и если ее значение равно true, то данный звук будет приостановлен, в противном случае он будет возобновлен.
