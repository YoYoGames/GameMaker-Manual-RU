Настройка громкости звука

С помощью этого действия можно установить громкость заданного звука. Вы выбираете звук в проводнике активов, а затем устанавливаете значение громкости. Громкость может быть от 0 (тихо) до 1 (полная громкость), шкала линейная, так что значение 0,5 означает половину громкости. Обратите внимание, что это действие повлияет на все экземпляры звука, которые воспроизводятся в данный момент в комнате, и повлияет на все будущие экземпляры воспроизводимого звука.

Поле "Время" определяет, через какое время изменение громкости вступит в силу. Значение 0 приводит к мгновенному изменению, в то время как большее значение будет постепенно изменять громкость в течение заданного количества миллисекунд.

Аргументы:

Аргумент Описание
Sound Звуковой ресурс для установки громкости
Volume Новая громкость для звука (от 0 до 1, по умолчанию 1)
Time Количество времени, в течение которого будет уменьшаться громкость (в миллисекундах).

Приведенный выше код блока действий проверяет глобальную переменную на предмет того, имеет ли она значение true или false. Если она равна true, то громкость данного звука извлекается и сохраняется во временной локальной переменной. Затем проверяется, не равно ли это значение 0,5, и если нет, то громкость звука уменьшается до 0,5 за 1000 миллисекунд (1 секунда). Если значение глобальной переменной равно false, то громкость звука устанавливается равной 1. Поскольку значение времени здесь установлено в 0, изменение громкости происходит немедленно.
