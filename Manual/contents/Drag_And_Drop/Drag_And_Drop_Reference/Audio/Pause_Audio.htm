Приостановить аудио

Это действие можно использовать для приостановки выбранного звука, если он воспроизводится в данный момент. Обратите внимание, если воспроизводится несколько экземпляров одного и того же звука, все они будут поставлены на паузу. вызвано. Запустить воспроизведение приостановленного звука можно с помощью действия Resume Audio.
