Получить индекс плитки в пикселе

Это действие используется для получения индекса плитки на заданном слое. Слой задается из имени слоя (строка, используемая для названия слоя в редакторе комнат), а затем задается позиция x и y для поиска плитки, индекс которой нужно получить. Плитка будет та, в которую попадает заданная позиция в комнате, а индекс будет возвращен в переменную target, где 0 - нет плитки, а целое положительное число - индекс плитки. Позиции x и y могут быть помечены как относительные к позиции вызывающего экземпляра, а переменная target может быть помечена как временная (локальная) переменная, которая может использоваться до конца события.

y Позиция y для проверки
Target Целевая переменная для хранения индекса плитки
