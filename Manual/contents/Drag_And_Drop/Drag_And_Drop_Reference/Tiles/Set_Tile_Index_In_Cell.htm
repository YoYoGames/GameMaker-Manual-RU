Установить индекс плитки в ячейке

С помощью этого действия можно установить плитку на заданном слое на новый индекс. Слой задается из имени слоя (строка, как используется для названия слоя в редакторе помещений), затем указывается строка и столбец ячейки в слое, чтобы получить плитку для установки. Наконец, задается индекс плитки, на которую нужно установить плитку (плитки индексируются слева направо сверху вниз, с 0 в левом верхнем углу). Обратите внимание, что вы всегда можете установить индекс плитки на 0, чтобы очистить плитку.

Аргументы:

Аргумент Описание
Layer Слой, на котором будет установлена плитка
Column Столбец (слева направо) ячейки
Row Строка (сверху вниз) ячейки
Tile Индекс плитки для установки ячейки

Приведенный выше код блока действий создает цикл для прохождения по верхней строке заданного слоя помещения, и по мере его выполнения проверяется, установлена ли плитка, индексированная в ячейке столбца, на 0. Если нет, то индекс плитки для этой ячейки устанавливается на 0, и временная (локальная) переменная увеличивается, прежде чем цикл выполнит еще одну проверку набор проверок.
