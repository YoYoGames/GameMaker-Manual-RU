Получение данных о плитке на пикселе

С помощью этого действия можно получить данные плитки для ячейки на слое карты плитки из позиции. Данные плитки - это просто значение, отражающее индекс плитки вместе со значениями флипа, зеркала и поворота, а также любыми пользовательскими масками плитки, которые которые вы использовали. В этом действии сначала указывается слой, на который нужно нацелиться, используя имя слоя (строку), определенное в редакторе помещения, а затем оси x и y в помещении, для которых нужно получить данные плитки. Возвращаемое значение данных тайла будет совпадать с любой ячейкой карты тайлов, находящейся в данной точке. Например, если ваш набор плиток имеет размер 16x16, то слой карты плиток будет содержать плитки в ячейках размером 16x16 пикселей, поэтому, если вы используете это действие для нацеливания на позицию комнаты (8, 8), вы получите данные плитки для ячейки (0, 0) карты плитки.

ПРИМЕЧАНИЕ: Выбранный слой должен быть определен как слой карты плиток в редакторе помещений, иначе могут возникнуть ошибки.

Возвращенное значение будет сохранено в целевой переменной, которая может быть создана ранее или может быть новой временной переменной (если вы установите флажок "Temp"). Более подробную информацию о данных тайлов можно найти в разделе GML, посвященном функциям карты тайлов.
