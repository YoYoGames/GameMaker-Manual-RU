Установить индекс плитки в пикселе

С помощью этого действия вы можете установить плитку на заданном слое на новый индекс. Слой задается из имени слоя (строка, используемая для названия слоя в редакторе комнат), а затем задается позиция x и y, чтобы получить плитку для установки. Позиции x и y могут быть помечены как относительные к позиции вызывающего экземпляра, а устанавливаемая плитка будет той, в которую попадает данная позиция в помещении, а используемый набор плиток будет тем, который назначен слою в редакторе помещения. редактор. Наконец, вы указываете индекс плитки, на которую нужно установить плитку (плитки индексируются слева направо сверху вниз, с 0 в левом верхнем углу). Обратите внимание, что вы всегда можете установить индекс плитки на 0, чтобы очистить плитку.
