Установить данные плитки в ячейку

С помощью этого действия можно установить данные плитки для ячейки плитки на слое карты плитки. Данные плитки - это просто значение, отражающее индекс плитки, а также значения флипа, зеркала и поворота, а также любые пользовательские маски плитки, которые вы использовали. В этом действии сначала указывается целевой слой с помощью имени слоя (строки), определенного в редакторе помещений, а затем столбец и строка слоя карты плитки, для которых нужно установить данные плитки. для. Ячейка" - это область на сетке карты плиток, которая содержит плитку, для которой вы хотите установить данные, поэтому, например, если ваши плитки 16x16, а комната 1024x768, карта плиток будет иметь 64 столбца и 48 строк.

ПРИМЕЧАНИЕ: Выбранный слой должен быть определен как слой карты плиток в редакторе помещений, иначе могут возникнуть ошибки.

Последним аргументом действия являются сами данные плитки. Обычно вы получаете данные тайла для ячейки карты тайла с помощью действия Получить данные тайла в ячейке, а затем манипулируете ими с помощью действия Установить преобразование данных тайла, прежде чем снова установить их с помощью этого действия. Более подробную информацию о данных тайла см. в разделе GML " Функции карты тайла".
