Комплект плитки

С помощью этого действия вы можете задать слою использование наборов плиток, отличных от тех, что созданы в браузере активов. Вы задаете имя слоя (строка, как определено в редакторе помещений), а затем ресурс набора плиток для использования, и все плитки в помещении на этом слое будут нарисованы с новым набором плиток.

Аргументы:

Аргумент Описание
Layer Слой, на котором будет установлен новый набор плиток.
Tileset Набор плиток, используемый для слоя.
