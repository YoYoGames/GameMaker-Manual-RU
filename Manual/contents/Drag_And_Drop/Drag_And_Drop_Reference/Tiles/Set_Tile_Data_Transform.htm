Установить преобразование данных плитки

С помощью этого действия вы можете изменить одно из трех преобразований плитки в данных плитки. Вы указываете переменную, в которой хранятся данные плитки, а затем выбираете преобразование, которое нужно изменить. Нажав на флажок "Применить", вы примените это преобразование и снятие флажка приведет к его удалению. Данные плитки - это просто значение, которое отражает индекс плитки вместе с примененными преобразованиями, а также любые пользовательские маски плитки, которые вы использовали, и это действие позволяет вам изменить три из этих значений: Перевернуть, отразить и повернуть. Обратите внимание, что обычно сначала необходимо получить данные плитки с помощью Get Tile Data In Cell или Get Tile Data At Pixel, затем применить требуемые преобразования, а затем снова установить плитку в карту плиток с помощью Set Tile Data In Cell или Set Tile Data At Pixel, поскольку это действие только устанавливает значение, но не применяет его к плитке, из которой оно было взято. Для получения дополнительной информации о данных тайла см. раздел GML, посвященный функциям карты тайла.
