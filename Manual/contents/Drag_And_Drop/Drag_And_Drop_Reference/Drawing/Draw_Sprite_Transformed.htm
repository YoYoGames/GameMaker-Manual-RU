Нарисовать Sprite Трансформированный

Эта функция рисует заданный sprite аналогично действию Draw Sprite, но с дополнительными опциями для изменения масштаба, смешивания, вращения и рамки рисуемого sprite. Изменение этих значений не изменяет ресурс (только то, как он рисуется), и вы можете использовать любую из доступных переменных свойств спрайта вместо прямых значений для всех аргументов в функции. Изображение ниже иллюстрирует как различные значения влияют на отрисовку спрайта:

ПРИМЕЧАНИЕ Смешивание цветов рекомендуется для цели HTML5 только при включенном WebGL, хотя вы все равно можете задать цвет смешивания, если он не включен, и он будет смешивать sprite как обычно. Однако при таком смешивании создается дубликат. sprite который затем сохраняется в кэше и используется при необходимости. Это далеко не оптимально, и если вы используете многократное изменение цвета, это замедлит производительность вашей игры, если вы не активируете WebGL. если вы не хотите использовать WebGL, то вы можете установить размер кэша шрифтов, чтобы попытаться ограничить это, если потребуется, используя функцию sprite_set_cache_size().

ПРИМЕЧАНИЕ: Это действие предназначено только для использования в различных событиях рисования и не будет рисовать ничего, если использовать его в других местах.

Аргументы:

Аргумент Описание
Sprite sprite для рисования
Frame Кадр sprite для рисования (используйте встроенную переменную image_index для текущего кадра).
X Позиция x для рисования в пределах комнаты
Y Позиция y для рисования в пределах комнаты
Scale X Масштабный коэффициент для оси X (используйте встроенную переменную image_xscale для текущего масштаба X).
Scale Y Масштабный коэффициент для оси Y (используйте встроенную переменную image_yscale для текущего масштаба Y).
Rotation Угол рисования для sprite (от 0 до 360, вращение против часовой стрелки, где 0 - вправо - используйте встроенную переменную image_angle для текущего вращения).
Colour Цвет для смешивания с sprite (по умолчанию белый, а для текущего цвета можно использовать встроенную переменную image_blend ).
