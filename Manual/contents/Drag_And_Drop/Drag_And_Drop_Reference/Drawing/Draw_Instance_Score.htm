Балл за участие в розыгрыше

С помощью этого действия вы можете нарисовать значение счета экземпляра. При использовании любого действия с оценкой в GameMaker у экземпляра, который его вызывает, будет добавлена новая переменная области видимости экземпляра под названием score. Эту переменную можно проверять и устанавливать, как и любую другую переменную. В случае с этим действием переменная будет выведена на экран в заданной позиции вместе с любым текстом подписи, который вы хотите предоставить. Текст подписи будет нарисован перед значением оценки, и если текст должен быть обернут символом "", и оба текста будут отформатированы с использованием текущего шрифта, цвета и выравнивания.

ПРИМЕЧАНИЕ: Это действие предназначено только для использования в различных событиях рисования и не будет рисовать ничего, если использовать его в других местах.
