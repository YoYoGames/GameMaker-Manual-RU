Нарисовать преобразованное значение

С помощью этого действия вы можете вывести на экран строковую надпись и/или переменную или значение. Аргумент caption представляет собой строку, которую вы хотите добавить перед значением, и при необходимости может быть оставлен пустым, а значение может быть переменной, фиксированным значением или другой строкой - в этом случае она должна быть заключена в кавычки и также может быть оставлена пустой, если это необходимо. Затем вы указываете позицию, в которой должна быть нарисована строка, либо как абсолютную позицию, либо как позицию относительно экземпляра, рисующего текста. Дополнительно можно задать масштаб текста, растягивая его по горизонтальной или вертикальной оси, а также поворот (углы поворота текста задаются против часовой стрелки, поэтому при установке значения 90° текст будет повернут влево, а -90° - вправо). вправо). Масштабирование осуществляется по заданному коэффициенту, поэтому при масштабе 0,5 ширина текста уменьшится вдвое, при масштабе 2 - вдвое, а значение 1 по умолчанию не изменяет масштаба. Обратите внимание, что внешний вид текста при рисовании зависит от используемого шрифта, цвета и выравнивания. Все эти параметры можно изменить перед рисованием, как показано в примере ниже.

ПРИМЕЧАНИЕ: Это действие предназначено только для использования в различных событиях рисования и не будет рисовать ничего, если использовать его в других местах.

Аргументы:

Аргумент Описание
Caption Надпись для рисования (строка), можно оставить пустой

x Позиция x для рисования

xscale Коэффициент масштабирования x (горизонтальный), по умолчанию 1
yscale Коэффициент масштабирования y (вертикальный), по умолчанию 1
Rotation Угол поворота для рисования текста
