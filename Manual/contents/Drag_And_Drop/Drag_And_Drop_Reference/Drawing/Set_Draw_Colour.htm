Установите цвет рисунка

С помощью этого действия вы можете установить цвет, который будет использоваться для любых фигур (прямоугольник/эллипс/линия) или для любого текста, который будет нарисован впоследствии. Обратите внимание, что это действие не обязательно вызывать перед каждым действием рисования, его можно задать только один раз в начале игры. Например, если весь ваш текст будет желтым, вы можете вызвать это действие в событии Create Event первой комнаты игры, и с этого момента весь текст (и фигуры) будет рисоваться желтым. Однако если вам нужно использовать несколько разных цветов, то вам нужно будет вызывать это действие перед каждым текстом или фигурой, для которых требуется другой цвет.

В нижней части действия есть флажок Use alpha. Это позволяет использовать альфа-компонент цвета для установки альфы рисунка (так же, как и при использовании Set Draw Alpha. Если вы не хотите включать альфа-компонент и просто иметь сплошной цвет, этот флажок можно отключить (по умолчанию он включен).
