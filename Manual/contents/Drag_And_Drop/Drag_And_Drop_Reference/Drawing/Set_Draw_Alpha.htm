Установить альфа рисунка

С помощью этого действия вы можете установить альфа-коэффициент, который будет использоваться для любых фигур (прямоугольник/эллипс/линия) или для любого текста, который будет нарисован впоследствии. Обратите внимание, что это действие не обязательно вызывать перед каждым действием рисования, оно может быть установлено только один раз в начале игры. Alpha - это параметр, который контролирует, насколько "прозрачными" будут нарисованные объекты, и основан на шкале от 0 до 1, где 0 - полностью прозрачный, а 1 - полностью непрозрачный.
