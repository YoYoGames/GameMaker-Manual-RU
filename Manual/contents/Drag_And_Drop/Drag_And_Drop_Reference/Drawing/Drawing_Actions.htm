Библиотека действий при рисовании

В библиотеке действий рисования можно найти действия, необходимые для рисования sprites, текста или фигур, а также для установки определенных свойств рисования. фигуры, а также установить определенные свойства рисования. Большинство из этих действий предназначены только для использования в различных событиях рисования объекта и могут не работать, если используются вне События рисования. Исключением являются действия Set, которые можно добавить в любое событие, и они будут влиять на все рисование для всех последующих экземпляров.

Важно отметить, что если вы добавите какие-либо действия в основное событие Draw Event объекта, то он не будет рисовать sprite, который был назначен данному экземпляру, пока вы явно не прикажете GameMaker нарисовать его, используя действие например, Draw Self. В принципе, GameMaker будет по умолчанию рисовать любой sprite, назначенный экземпляру, только если в событии Draw Event нет ничего другого.
