Нарисовать градиентный прямоугольник

Это действие нарисует прямоугольник в заданной позиции в комнате, используя набор цветов смешивания для создания градиента. Вы указываете верхнюю левую позицию и нижнюю правую позицию, и прямоугольник будет нарисован между ними, а позиция может быть либо абсолютной позицией в комнате, либо позицией относительно экземпляра, вызывающего действие. Затем вы задаете цвета, которые будут смешиваться с каждым из углов, а также можете установить, может ли прямоугольник быть заполнен или очерчен, установив флажок в нижней части.

ПРИМЕЧАНИЕ: Это действие предназначено только для использования в различных событиях рисования и не будет рисовать ничего, если использовать его в других местах.

Аргументы:

Аргумент Описание
Left Левая позиция, с которой начинается рисование
Top Верхняя позиция, с которой следует начинать рисовать
Right Правильная позиция для завершения рисования
Bottom Нижняя позиция для завершения рисования
Top Left Цвет для смешивания из левого верхнего угла
Top Right Цвет для смешивания в правом верхнем углу
Bottom Left Цвет для смешивания в левом нижнем углу
Bottom Right Цвет для смешивания в правом нижнем углу
