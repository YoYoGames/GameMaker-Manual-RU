Нарисовать панель здоровья

С помощью этого действия можно нарисовать цветную полосу для графического представления значения. Отображаемая величина представляет собой процентное значение и должна находиться в диапазоне от 0 до 100, однако вы можете использовать и большие значения с помощью формулы (CURRENT_Value / MAXIMUM_value) * 100, Так, например, вы можете использовать это для отображения силы, здоровья, маны, времени или чего-либо еще, что может быть полезно для отображения в виде столбика. Помимо значения для рисования, вы можете задать направление для рисования панели здоровья (слева направо или справа налево и сверху-вниз или снизу-вверх), а также позицию, либо в пределах комнаты, либо относительно экземпляра, вызывающего действие. Помимо позиции, вы также можете задать цвет, который будет смешиваться с каждым из четырех углов панели здоровья, создавая градиенты от красного к зеленому (например, но вы можете использовать любые цвета). Аргументы:

Аргумент Описание
Direction Направление рисования содержимого панели здоровья
Value Значение (от 0 до 100), которое будет использоваться для панели здоровья
Left Положение левой части панели здоровья
Top Положение верхней части панели здоровья
Right Положение правой части панели здоровья
Bottom Положение нижней части панели здоровья
Background Цвет, который будет использоваться в качестве фона на панели здоровья
Outline Цвет, используемый для обводки панели здоровья
Min Colour Цвет для смешивания с минимальным значением
Max Colour Цвет для смешивания с максимальным значением
