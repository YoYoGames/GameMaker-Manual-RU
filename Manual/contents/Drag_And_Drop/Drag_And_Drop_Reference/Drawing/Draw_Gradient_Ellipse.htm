Нарисовать градиентный эллипс

Это действие нарисует эллипс в заданном месте комнаты, используя набор цветов смешивания для создания градиента. Вы задаете левое верхнее и правое нижнее положение области, в которую должен "вписаться" эллипс, и эллипс будет нарисован между ними. Положение может быть либо абсолютным положением в комнате, либо положением относительно экземпляра, вызывающего действие, и вы можете задать цвета для смешивания от края эллипса и от его центра. Эллипс может быть нарисован заполненным или очерченным, установив флажок заполнения в нижней части.

ПРИМЕЧАНИЕ: Это действие предназначено только для использования в различных событиях рисования и не будет рисовать ничего, если использовать его в других местах.

Аргументы:

Аргумент Описание
Left Левая позиция, с которой начинается рисование
Top Верхняя позиция, с которой следует начинать рисовать
Right Правильная позиция для завершения рисования
Bottom Нижняя позиция для завершения рисования
Center Цвет смешивается от центра
Edge Цвет для смешивания с краем
