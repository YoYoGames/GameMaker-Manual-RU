Копирование файла

С помощью этого действия вы можете скопировать любой файл, сохраненный вашей игрой или входящий в состав Included Files. Вы указываете имя файла для копирования (в виде строки), а затем имя файла, который нужно создать как копию исходного файла (также в виде строки и включая расширение файла). Указанный файл будет продублирован и получит новое имя.
