Если файл существует

С помощью этого действия вы можете проверить, существует ли файл с заданным именем. Вы выбираете файл для проверки (в виде жала и включая расширение), и функция возвращает true, если он существует, и false, если нет. Обратите внимание, что функция вернет true, если файл существует в пакете игры как Included File или если файл был сохранен во время работы игры (более подробную информацию см. в разделе " Файловая система ").
