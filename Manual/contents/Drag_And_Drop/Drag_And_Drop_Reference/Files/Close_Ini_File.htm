Закрыть файл Ini

Это действие закроет текущий открытый файл Ini. Это действие следует вызывать в тот момент, когда вы закончили чтение или запись в любой открытый Ini файл (Ini файлы открываются с помощью действия Open Ini File). Если вы не используете это действие после использования действия Write To Ini File, то на диск ничего не будет записано, так как информация о файле хранится в памяти до вызова этого действия, после чего она извлекается из памяти и записывается в файл на диске. Также обратите внимание, что если вы попытаетесь открыть Ini файл, не закрыв предварительно другой (или тот же самый), вы получите ошибку.
