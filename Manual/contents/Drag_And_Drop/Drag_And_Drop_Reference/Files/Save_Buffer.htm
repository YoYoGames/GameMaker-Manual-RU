Сохранить буфер

С помощью этого действия вы можете сохранить файл, содержащий данные из буфера. Вы указываете уникальный идентификатор буфера (полученный при создании буфера с помощью Create Buffer) и задаете имя файла (в виде строки, с расширением по вашему выбору), и буфер будет записан в этот файл, готовый к повторной загрузке с помощью Load Buffer.

Аргументы:

Аргумент Описание
Buffer Уникальное значение идентификатора буфера для сохранения
Filename Имя (в виде строки) файла для сохранения

Приведенный выше код блока действий создаст переменную для хранения данных буфера, а затем проверит, не был ли ранее создан файл сохранения буфера был создан ранее. Если файл существует, то он загружается в созданную нами переменную буфера, если же файла не существует, то он создается, записывается и затем сохраняется, готовый к загрузке в следующий раз.
