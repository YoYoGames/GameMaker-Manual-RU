Чтение Ini файла

С помощью этого действия вы можете прочитать часть данных из Ini файла. Вы должны предварительно открыть Ini файл для чтения/записи с помощью действия Открыть Ini файл, а затем с помощью этого действия указать тип данных для чтения (либо вещественное число, либо строка, заключенная в ""), а также заголовок секции и ключ в этой секции, из которого нужно получить данные. Вы также можете указать значение по умолчанию, которое может быть как вещественным числом, так и строкой, а тип должен совпадать с "Type", установленным в действии. Это значение по умолчанию будет возвращено действием, если произойдет одно из следующих событий:

  • Открытого вами файла Ini не существует
  • Заголовок раздела, на который вы ссылаетесь, не существует
  • Ключ, из которого вы пытаетесь получить данные, не существует

При любом из вышеперечисленных обстоятельств вы получите возвращенное значение по умолчанию, а не значение из файла Ini. Возвращенное значение будет сохранено в указанной вами целевой переменной, которая также может быть создана в этот момент, если отмечена как как временная локальная переменная.

ВАЖНО! Попытка прочитать данные из Ini файла, который не был открыт ранее с помощью действия Open Ini File, приведет к серьезной ошибке и краху вашей игры.

Синтаксис действия:

Аргументы:

Аргумент Описание
Type Тип данных для чтения, либо string (заключенный в кавычки ""), либо real number
Section Раздел файла Ini для чтения (в виде строки)
Key Ключ в разделе, из которого нужно читать (в виде строки)
Default Значение по умолчанию, которое возвращается, если файл, раздел или ключ не существует.
Target Целевая переменная для хранения возвращаемого значения из действия

Пример:

Приведенный выше код блока действий открывает файл Ini для чтения, затем использует данные, считанные из него, для установки 3 различных глобальных переменных, после чего снова закроет файл.
