Открыть файл Ini

С помощью этого действия вы можете открыть файл Ini для чтения и/или записи. Если файл Ini не существует в проверяемом месте, GameMaker может создать его, но только если вы запишете в него данные. Если вы только считываете информацию из Ini файла, то будут возвращены значения по умолчанию для функции чтения, но сам Ini файл создан не будет.

Обратите внимание, что одновременно у вас может быть открыт только один Ini файл, и не забудьте использовать Close Ini File, как только вы закончите чтение/запись из Ini файла, так как информация не сохраняется на диске до этого момента (она хранится в памяти, пока файл не будет закрыт). до этого момента (она хранится в памяти до тех пор, пока файл не будет закрыт).
