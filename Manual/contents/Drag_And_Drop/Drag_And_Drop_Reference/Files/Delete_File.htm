Удалить файл

С помощью этого действия вы можете удалить файл, который был создан ранее. Вы указываете имя файла (в виде строки), который нужно удалить. Обратите внимание, что вы не можете удалить файлы, включенные в пакет игры, а только те, которые были созданы с помощью действий Close Ini или Save Buffer (подробнее см. раздел " Файловая система ").
