Переименовать файл

С помощью этого действия вы можете переименовать любой файл, сохраненный вашей игрой. Вы указываете имя файла, которое нужно изменить (в виде строки), а затем новое имя (также в виде строки и включая расширение файла), и файл будет переименован. Обратите внимание, что вы не можете переименовать ни один файл, который был включен в игру в качестве включенного файла (более подробную информацию см. в разделе " Файловая система ").
