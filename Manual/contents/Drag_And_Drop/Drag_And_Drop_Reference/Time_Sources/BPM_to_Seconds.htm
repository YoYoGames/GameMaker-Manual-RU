Скорость в минуту в секунды

Это действие принимает значение "ударов в минуту" и возвращает длительность каждого удара в секундах.

Это можно использовать при создании источника времени, чтобы использовать значение BPM для источника времени. Важно, чтобы такой источник времени использовал секунды в качестве единицы измерения.

Аргументы:

Аргумент Описание
Value Значение частоты ударов в минуту
Target Переменная для хранения возвращаемого значения секунды на удар
Temp Должна ли переменная создаваться как временная переменная
