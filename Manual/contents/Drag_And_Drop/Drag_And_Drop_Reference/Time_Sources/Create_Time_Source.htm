Создать источник времени

Это действие создает новый источник времени и сохраняет его в переменной Target. Для получения более подробной информации читайте раздел Источники времени.

Это основано на time_source_create() функция. Прочитайте ее страницу для получения подробной информации об аргументах.

Чтобы активировать источник времени после его создания, необходимо использовать Start Time Source.

Вы должны уничтожить источник времени с помощью Destroy Time Source, когда он вам больше не нужен.

Синтаксис действия:

Аргументы:

Аргумент Описание
Target Переменная, в которой будет храниться новый источник времени
Parent Родительский источник времени: либо встроенный источник времени, либо пользовательский.
Period Длительность периода источника времени, через какое время он истечет
Units Единицы, в которых выражается период (секунды или кадры)
Callback Адрес method или функция сценария для вызова по истечении срока действия источника времени
Arguments OPTIONAL Массив, содержащий аргументы для передачи в функцию обратного вызова
Repetitions OPTIONAL Сколько раз источник времени должен быть запущен в общей сложности, или -1 для неограниченного повторения.
Expiry Type ОПЦИОНАЛЬНЫЙ Тип истечения срока действия для источника времени

Пример:

Этот блок кода выполняет следующие действия:

  • Он создает новую функцию с включенной опцией "Temp". При этом создается локальная переменная метода.
    • Этот метод принимает один аргумент, msg, который он печатает в выходной журнал.
  • Затем создается новый источник времени с длительностью периода 1 секунда.
    • Он указывает переменную callback_method в качестве "Обратного вызова" и массив с одним аргументом для функции.
    • Этот источник времени настроен на неограниченное повторение, так как -1 указано в "Повторениях".
  • После этого запускается источник времени.

Этот источник времени будет выполнять свой обратный вызов каждую секунду, печатая "Прошла секунда!" в выходной журнал.
