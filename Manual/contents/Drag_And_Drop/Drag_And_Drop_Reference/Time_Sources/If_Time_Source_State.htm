Если [Источник времени] Государство

Это действие позволяет управлять другими действиями на основе состояния данного источника времени.

Проверяет, является ли Константа состояния источника времени указанное в "State", является (или не является) состоянием Источника времени, указанного в "Time Source".

Аргументы:

Аргумент Описание
Time Source Сайт Идентификатор источника времени для проверки состояния
Not Включите эту функцию, чтобы инвертировать условие
State Сайт Постоянная состояния источника времени для проверки

Пример:


В идеале этот блок кода должен запускаться в событии нажатия клавиши.

Он проверяет, является ли состояние источника времени "активным", и в этом случае он приостанавливает работу этого источника времени.

Однако, если состояние "приостановлено", оно снова запускает Источник времени.
