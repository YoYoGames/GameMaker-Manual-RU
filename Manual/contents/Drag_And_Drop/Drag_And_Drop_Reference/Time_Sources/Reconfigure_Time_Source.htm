Перенастройка источника времени

Это действие позволяет изменить настройки для существующего источника времени, указанного в поле "Источник времени".

Здесь задаются те же параметры, что и при создании источника времени, за исключением родителя.

Это приведет к сбросу и деактивации источника времени, поэтому вам придется запустить его снова.

Аргументы:

Аргумент Описание
Time Source Сайт Идентификатор источника времени для изменения конфигурации
Period Длительность периода источника времени, через какое время он истечет
Units Единицы, в которых выражается период (секунды или кадры)
Callback Адрес method для вызова, когда истекает срок действия источника времени
Arguments OPTIONAL Массив, содержащий аргументы для передачи в функцию обратного вызова
Repetitions OPTIONAL Сколько раз источник времени должен быть запущен в общей сложности, или -1 для неограниченного повторения.
Expiry Type ОПЦИОНАЛЬНЫЙ Тип истечения срока действия для источника времени
