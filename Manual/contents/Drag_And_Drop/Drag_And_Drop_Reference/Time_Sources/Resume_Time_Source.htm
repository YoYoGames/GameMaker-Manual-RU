Источник времени резюме

Это действие возобновляет работу данного источника времени и его дочерних элементов. Он должен быть ранее приостановлен с помощью Pause Time Source.

Time Source Сайт Идентификатор источника времени приостановить

Пример:


В идеале этот блок кода должен запускаться в событии нажатия клавиши.

Он проверяет, является ли состояние источника времени "активным", и в этом случае он приостанавливает работу этого источника времени.

Однако, если состояние "приостановлено", оно возобновляет Источник времени.
