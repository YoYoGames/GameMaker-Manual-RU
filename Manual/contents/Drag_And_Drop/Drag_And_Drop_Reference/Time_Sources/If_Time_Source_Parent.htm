Если [Источник времени] Родитель

Это действие позволяет управлять другими действиями на основе родителя данного источника времени.

Он проверяет, является ли источник времени, указанный в "Parent", родителем (или нет) источника времени, указанного в "Time Source".

Аргументы:

Аргумент Описание
Time Source Сайт Идентификатор источника времени для проверки родителя
Not Включите эту функцию, чтобы инвертировать условие
Parent Адрес Идентификатор источника времени для сравнения с родителем ранее указанного источника времени
