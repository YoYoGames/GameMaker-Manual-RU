Получить оценку

Вы можете использовать это действие для получения значения переменной экземпляра, созданной для "score". Вы указываете целевую переменную для хранения возвращаемого значения "score" (которая может быть помечена как временная локальная переменная), но учтите, что если переменная не была установлена ранее (с помощью Set Score), то она будет создана за вас, и действие вернет 0 (в отличие от обычных переменных экземпляра, которые выдадут ошибку, если вы попытаетесь получить к ним доступ не задав их предварительно).
