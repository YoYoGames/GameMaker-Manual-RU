Получить здоровье

Вы можете использовать это действие для получения значения переменной экземпляра, созданной для "здоровья" объекта. Вы указываете целевую переменную для хранения возвращаемого значения "здоровья" (которая может быть помечена как временная локальная переменная), но обратите внимание, что если переменная не была установлена ранее (с помощью Set Health), то переменная будет создана за вас, и действие вернет 0 (в отличие от обычных переменных экземпляра, которые выдадут ошибку, если вы попытаетесь обратиться к ним, не установив их сначала). получить к ним доступ, не установив их предварительно).
