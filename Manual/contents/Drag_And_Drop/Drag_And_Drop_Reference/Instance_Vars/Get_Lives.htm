Получить жизнь

Вы можете использовать это действие для получения значения переменной экземпляра, созданной для "lives". Вы указываете целевую переменную для хранения возвращаемого значения "lives" (которая может быть помечена как временная локальная переменная), но учтите, что если переменная не была установлена ранее (с помощью Set Lives), то переменная будет создана за вас, и действие вернет 0 (в отличие от обычных переменных экземпляра, которые выдадут ошибку, если вы попытаетесь получить к ним доступ не установив их предварительно).
