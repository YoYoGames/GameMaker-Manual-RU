Переменная экземпляра Библиотека действий

Действия с переменными экземпляра предназначены для начала создания игры и используются для создания и поддержания определенных переменных области действия экземпляра score, lives и health (подробнее об области действия переменных можно узнать здесь). здесь). Эти действия будут создавать эти переменные для вас в любое время, когда они используются, и предназначены скорее для удобства или для новичков, а если вам нужно более сложный контроль над игровыми значениями, вы должны создавать свои собственные переменные, используя соответствующие действия (см. здесь).

Для использования этих переменных экземпляра существуют следующие действия:

Установить оценку
Получить оценку
Установленные жизни
Получить жизнь
Установить здоровье
Получить здоровье
Если оценка
Если жизни
Если здоровье
