Установить здоровье

С помощью этого действия вы можете создать или установить переменную экземпляра для здоровья. Вы предоставляете значение, которое нужно либо установить в переменную "здоровье", либо добавить к переменной "здоровье" (если вы отметите флажок " относительно ", то значение будет добавлено к текущему значению "здоровье"). Обратите внимание, что если вы хотите использовать действие Draw Instance Healthbar, то это значение должно быть от 0 до 100, в противном случае оно может быть любым. Также обратите внимание, что в отличие от обычных переменных экземпляра имя переменной "здоровье" хранится внутри, поэтому для получения значения показателя необходимо использовать действие Get Health.
