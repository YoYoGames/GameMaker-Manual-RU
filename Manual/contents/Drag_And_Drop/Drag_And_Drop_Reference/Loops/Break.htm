Перерыв

Это действие можно использовать только внутри действия цикла (например, Repeat или While), и его вызов приведет к выходу из цикла в этот момент. Обычно цикл выполняется либо фиксированное количество раз, либо до тех пор, пока не будет выполнено определенное условие. но вы можете использовать действие Break для преждевременного завершения цикла в любой момент (действия в цикле после Break не будут выполнены).
