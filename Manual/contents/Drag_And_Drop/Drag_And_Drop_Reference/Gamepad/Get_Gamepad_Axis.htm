Получить ось геймпада

Вы можете использовать это действие для получения значений различных осей с данного геймпада. Вы указываете индекс геймпада (это номер "слота", который занимает геймпад) и выбираете ось для проверки: левый или правый стик, горизонтальную или вертикальной. Возвращаемое значение будет от -1 до 1 для каждой из доступных горизонтальных и вертикальных осей: -1 - влево и 1 - вправо для горизонтальной оси, и -1 - вверх и 1 - вниз для вертикальной оси. Это возвращаемое значение будет сохранено в целевой переменной, которую вы предоставите для дальнейшего использования. Вы можете отметить ее как переменную "Temp", что означает, что действие создаст новую временную (локальную) переменную для хранения возвращаемого значения, как показано в примере ниже.

Стоит отметить, что установка мертвой зоны для оси стика не влияет на возвращаемое значение, которое всегда будет между -1 и 1. Например, установка мертвой зоны на 0.2 означает, что нажатие стика вправо начнет возвращать значение от 0 до 1, только когда необработанное значение оси превысит 0,2 - таким образом, когда необработанное значение равно 0,2, возвращаемое значение будет 0, когда необработанное значение равно 0,5, возвращаемое значение будет 0,375, или если необработанное значение 0,9, то возвращаемое значение будет 0,875.

Аргументы:
