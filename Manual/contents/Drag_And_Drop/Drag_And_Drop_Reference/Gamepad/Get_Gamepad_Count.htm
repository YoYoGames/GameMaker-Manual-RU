Получить счетчик геймпада

Вы можете использовать это действие, чтобы получить количество геймпадов, подключенных к устройству, на котором запущена ваша игра. Возвращаемое число будет от 0 до 12.

Пример:

Приведенный выше код блока действий получает количество подключенных геймпадов и присваивает это значение временной локальной переменной. Затем проверяется, больше ли она 0, и соответствующим образом устанавливается глобальная переменная.
