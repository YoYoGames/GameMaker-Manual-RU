Подключите геймпад

Вы можете использовать это действие, чтобы проверить, подключен ли к данному "слоту" геймпада геймпад. Вы передаете индекс геймпада (это номер "слота", который занимает геймпад), а возвращаемое значение будет сохранено в целевой переменной которую вы предоставите для дальнейшего использования (она может быть помечена как временная локальная переменная, которую вы создаете для этого события или сценария).

Target Целевая переменная для хранения возвращаемого значения.
