Предпочтения HTML5

Параметры HTML5 имеют следующие опции:

  • Тип упаковки по умолчанию: При создании конечного исполняемого пакета HTML5 вы можете выбрать между Package as loose files - который просто создаст все необходимые файлы в папке - и Package as Zip - который создаст сжатый файл ZIP , содержащий игру и все дополнительные файлы. По умолчанию вам будет предоставлен выбор, какой из этих двух вариантов вы хотите выбрать при компиляции, но вы можете установить это предпочтение, чтобы всегда использовать один или другой. Обратите внимание, что когда эта опция установлена на Show Dialogue, в реальном диалоговом окне, появляющемся при компиляции, выбор "Remember Packaging Option" установит это предпочтение на выбранный вами вариант.
  • Порт веб-сервера по умолчанию: позволяет изменить порт по умолчанию для веб-сервера HTML5. Значение по умолчанию - 51264.
  • Адрес веб-сервера по умолчанию: Здесь вы можете изменить адрес веб-сервера по умолчанию для веб-сервера HTML5. Значение по умолчанию - 127.0.0.1.
  • Разрешенные IP-адреса веб-сервера по умолчанию: Здесь вы можете добавить любые разрешенные IP-адреса для веб-сервера, и они будут установлены по умолчанию для всех устройств, добавленных через Target Manager.

ВАЖНО При использовании конкретного целевого устройства (т.е. браузера) вместо этих настроек используются значения IP и порта, установленные для этого конкретного устройства. Проверьте Диспетчер устройств, чтобы увидеть список ваших устройств с их подробной информацией.
