Предпочтения tvOS

Настройки tvOS имеют следующие параметры:

  • Автоматический ответ на предупреждение об ограничении постоянного хранилища tvOS : игры tvOS имеют ограниченное локальное хранилище для файлов, и при компиляции для этой цели по умолчанию будет показано предупреждение, напоминающее об этом факте. Вы можете изменить настройку здесь, чтобы отключить это предупреждение, выбрав "OK". По умолчанию установлено значение "Показать сообщение".
  • Подавить сборку и запуск: По умолчанию, когда вы создаете проект tvOS, а затем собираете окончательный исполняемый пакет, GameMaker собирает пакет, а затем пытается запустить его на всех подключенных устройствах. Установка этой опции подавить сборку приложения и его запуск на устройстве и просто создать проект XCode на главном компьютере Mac.
  • Путь установки macOS: Путь установки файлов проекта на компьютере Mac для сборки.
  • Идентификатор команды по умолчанию: Здесь вы можете добавить свой идентификатор команды по умолчанию, присвоенный вам компанией Apple. Этот идентификатор команды будет использоваться при отправке файлов вашей игры на сайт Xcode для сборки приложения и позволит Xcode генерировать необходимые сертификаты подписи. Обратите внимание, что эта настройка будет применяться по умолчанию ко всем играм, созданным для tvOS, но ее можно изменить для каждого проекта в разделе Общие параметры tvOS. Параметры игры.
