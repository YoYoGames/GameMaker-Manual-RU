Предпочтения Opera GX

В настройках Opera GX есть следующие опции:

  • Расположение Emscripten SDK: Это местоположение Emscripten SDK, необходимого для сборок YYC, созданных для Opera GX. Информацию о настройке SDK читайте в разделе Настройка для Opera GX.
  • Порт веб-сервера по умолчанию: Позволяет изменить порт по умолчанию для веб-сервера Opera GX. Значение по умолчанию - 51264.
  • Адрес веб-сервера по умолчанию: Здесь вы можете изменить адрес веб-сервера по умолчанию для веб-сервера Opera GX. Значение по умолчанию - 127.0.0.1.
  • Разрешенные IP-адреса веб-сервера по умолчанию: Здесь вы можете добавить любые разрешенные IP-адреса для веб-сервера, и они будут установлены по умолчанию для всех устройств, добавленных через Target Manager.
