Предпочтения Windows UWP

Настройки Windows UWP имеют следующие параметры:

  • Расположение Visual Studio: Эта опция предназначена для указания GameMaker на установку Microsoft Visual Studio на компьютере сборки.
  • Win UWP Tools location: Эта опция предназначена для указания GameMaker на путь установки инструментов UWP на компьютере сборки.
  • Архитектура пакета по умолчанию: По умолчанию при компиляции исполняемого файла UWP отображается диалоговое окно для выбора между архитектурами x86, x64 и ARM . Вы можете установить этот параметр на одну из этих трех архитектур, которая будет выбираться по умолчанию при компиляции исполняемого файла.
  • Автоматическая реакция на создание пакета магазина: В диалоговом окне, отображаемом при компиляции, вы можете выбрать создание пакета магазина. Вы можете установить для этой опции значение Да или Нет , которое будет использоваться по умолчанию всякий раз, когда вы выбираете компиляцию исполняемого файла.
