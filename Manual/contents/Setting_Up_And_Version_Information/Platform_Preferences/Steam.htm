Предпочтения Steam

SDK Steam используется для сборок Windows, macOS и Ubuntu (Linux), которые будут распространяться через платформу Steam. В этом разделе приведены параметры настройки, необходимые для обеспечения его работы в ваших проектах:

  • Путь к Steam SDK: Системный путь к месту установки Steam SDK. После установки этого пути к SDK вам нужно будет активировать Steam в Опциях игры, а затем для каждой платформы добавить App ID.
