Предпочтения macOS

Настройки macOS имеют следующие параметры:

  • Идентификатор команды по умолчанию: Здесь вы можете добавить свой идентификатор команды по умолчанию, присвоенный вам компанией Apple. Этот идентификатор команды будет использоваться при отправке файлов вашей игры на Xcode для сборки приложения и позволит Xcode генерировать необходимые сертификаты подписи. Обратите внимание, что эта настройка будет применяться по умолчанию ко всем играм, созданным для macOS, но ее можно изменить для каждого проекта в разделе Общие параметры игры macOS.
  • Автоматический выбор упаковки: Здесь вы можете установить предпочтение по умолчанию для создания исполняемого файла для цели macOS. Опция "Показать диалог" всегда будет показывать диалог перед созданием исполняемого файла, предлагая вам выбрать между DMG и Zip, но вы можете установить любой из этих вариантов в Предпочтениях, чтобы пропустить этот диалог. Обратите внимание, что включение опции "Запомнить вариант пакета" в диалоге также установит этот параметр на выбранный тип пакета.
