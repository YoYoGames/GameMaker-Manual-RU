Переопределение предпочтений клавиш

Раздел "Переопределить клавиши " раздела "Параметры" позволяет вам изменить сопоставления для различных сочетаний клавиш, предусмотренных в IDE GameMaker.

Поиск

В верхней части находится строка поиска, которую можно использовать для поиска определенных команд, введя название (например, "Копировать"), комбинацию клавиш (например, "CTRL+C") или местоположение (например, "Текстовый редактор").

Включение кнопки рядом с полем поиска позволяет осуществлять поиск с помощью ввода сочетания клавиш. Любые клавиши, на которые вы нажимаете, пока эта кнопка включена, будут вводиться как ярлык, а не как обычный текст.

Рядом со строкой поиска находится меню Фильтр, которое описано в отдельном разделе далее на этой странице.

Список команд

В "Списке команд" отображаются все доступные ярлыки, которые можно переназначить, разделенные на различные категории в зависимости от расположения этих ярлыков. При нажатии на любое место в этом списке отображаются команды, которые можно использовать в этом месте:

Информация по каждой команде разделена на две колонки:

  • Команда: Это название команды/короткого пути и описание действия, которое она выполняет.
  • Клавиши: Здесь отображается текущее отображение клавиатуры для команды. Вы можете дважды щелкнуть на любом элементе в этой колонке, чтобы переназначить команду, просто нажав соответствующую комбинацию клавиш на клавиатуре (они должны удерживаться одновременно, аналогично тому, как будет использоваться ярлык). Как только вы отпустите новую комбинацию клавиш, она будет зарегистрирована и применена к команде. Нажатие кнопки во время редактирования команды выведет вас из режима редактирования и не приведет к изменению выбранной команды.

Вы можете изменить сортировку списка команд, нажав на заголовок столбца (либо "Команда", либо "Ключи"), чтобы отсортировать список, используя значения из этого столбца, чередуя возрастание и убывание при каждом нажатии.

Окно столкновения

Если вы попытаетесь установить комбинацию клавиш, которая уже используется другими командами, вы увидите специальное окно "Clash":
В этом окне будет показано местоположение и имя команды, которая уже использует комбинацию клавиш, которую вы пытаетесь использовать. В случае если комбинация клавиш используется несколькими командами, в списке будут показаны все из них.

Здесь у вас есть две опции: "Отменить" и "Переопределить (ВСЕ) и переназначить":

  • Нажатие кнопки "Отмена" приведет к закрытию окна Clash и не внесет изменений ни в одну команду.
  • Нажатие кнопки "Override (ALL) and Reassign" назначит новую комбинацию клавиш выбранной команде и очистит привязки всех других команд, которые ранее использовали ту же комбинацию.

После использования функции "Переопределить и переназначить" GameMaker выделит переопределенную команду в списке команд, чтобы вы могли назначить ей новую комбинацию клавиш. Если было отменено несколько команд, фильтр будет изменен, чтобы отобразить только эти команды, и вы сможете применить к ним новые сочетания клавиш. Этот фильтр можно очистить, нажав на кнопку "Очистить".

Если вы хотите использовать другую привязку для выбранной команды, вы можете развернуть раздел "Предлагаемые свободные клавиши", чтобы увидеть список комбинаций клавиш, которые не используются нигде на сайте IDE. Нажав на "Использовать" рядом с любой из этих комбинаций, вы назначите ее выбранной команде и решите проблему.

Меню фильтра

Меню Фильтр позволяет указать, какие команды должны появляться в списке команд. В верхней части меню есть две опции:

  • "Показывать несвязанные команды": Включение этого параметра отобразит все команды, которые были развязаны, т.е. их привязки были удалены. Команды, которые остаются несвязанными, не могут быть использованы, так как для них не назначена комбинация клавиш.
  • "Показывать только переопределенные команды": Включение этого параметра отобразит только те команды, которые были изменены вручную, т.е. их привязки отличаются от привязок по умолчанию.

Ниже есть множество опций, позволяющих указать места, которые будут отображаться в списке команд. По умолчанию установлен флажок "All Locations", что означает, что вы увидите команды из всех мест на сайте IDE. Снятие флажка "All Locations" снимет флажки со всех остальных пунктов, расположенных ниже, что позволит вам вручную выбрать те, которые вы хотите видеть; например, снятие флажка со всех пунктов и установка флажка "Image Editor" покажет вам только команды, которые можно использовать в редакторе изображений.

Очистка команд

Двойной щелчок на существующей комбинации клавиш позволит вам изменить ее, однако во время этого вы можете нажать на кнопку "Очистить" под списком команд, чтобы очистить привязку для этой команды.

Если вы случайно переопределили некоторые сочетания клавиш и хотите вернуть их к настройкам по умолчанию, вы можете нажать кнопку "Восстановить настройки по умолчанию" в верхней части окна "Параметры", чтобы вернуть все команды к их отображениям по умолчанию, или посетить страницу " Сочетания клавиш", чтобы проверить привязки по умолчанию и вернуть команды вручную.
