Предпочтения редактора спрайтов

Параметры редактора спрайтов используются для определения внешнего вида и функций редактора спрайтов. Существуют следующие параметры:

  • Начало по умолчанию sprite: Эта опция позволяет установить положение начала по умолчанию для всех новых ресурсов sprite, которые вы создаете. Значение по умолчанию - верхний левый угол.
  • Тип маски столкновений по умолчанию sprite: Эта опция позволяет установить вид маски столкновений по умолчанию, который будет использоваться для новых sprites. Можно выбрать следующие варианты: прямоугольник, повернутый прямоугольник, эллипс, ромб или точная / точная на кадр. По умолчанию используется прямоугольник.
  • Режим анимации по умолчанию sprite: Эта опция позволяет установить режим анимации по умолчанию для sprites в редакторе sprite. Вы можете выбрать либо " Кадры в секунду", либо " Кадры на игровой кадр", при этом по умолчанию используется значение " Кадры в секунду".
  • Скорость анимации по умолчанию sprite : Эта опция позволяет установить значение скорости анимации по умолчанию при создании нового актива sprite. Обратите внимание, что это значение связано с режимом анимации, и, например, 3 кадра в секунду это не то же самое, что 30 кадров на игровой кадр, поэтому убедитесь, что вы правильно установили это значение в зависимости от предпочитаемого режима. Значение по умолчанию - 30.
  • Ширина по умолчанию sprite : Эта опция позволяет установить ширину по умолчанию для всех новых создаваемых sprites. Значение по умолчанию - 64.
  • Высота по умолчанию sprite: Эта опция позволяет установить высоту по умолчанию для всех новых sprites. Значение по умолчанию - 64.
  • Применить изменение размера и масштаба к началу координат sprite: Если эта опция отмечена, то при изменении размера или масштаба sprite будет изменено начало координат sprite, при этом новое начало координат будет перемещено/масштабировано в соответствии с новыми значениями для sprite. Обратите внимание, что это применимо только для пользовательских sprite истоков, которые не заблокированы в редакторе sprite. Если начало заблокировано или sprite использует одно из положений по умолчанию - например, "Центр" или "Сверху слева" и т.д., - то эта опция будет будет выделена серым цветом и недоступна для изменения. По умолчанию включена.

Диалоги подтверждения

Доступные здесь параметры управляют сообщениями, появляющимися при необходимости подтверждения какого-либо действия. Вы можете изменить эти настройки, чтобы показывать сообщение всегда или отклонять его при определенном действии (Да или Нет):

  • Автоматическая реакция на импорт файлов: Когда вы импортируете новое изображение, а одно уже существует, по умолчанию вам будет предложено продолжить импорт и перезаписать существующее изображение или отменить операцию. Этот параметр позволяет подавить это сообщение и всегда либо отменять импорт, выбрав "Нет", либо перезаписывать текущий sprite, выбрав "Да". Значение по умолчанию - "Показывать сообщение".
  • Автоматическая реакция на сбой загрузки изображения: Когда вы импортируете новый образ и импорт не удается, по умолчанию вам будет показано предупреждающее сообщение, информирующее вас о том, что произошел сбой. Эта опция позволяет отключить это сообщение, выбрав "Ок". Значение по умолчанию - "Показать сообщение".
