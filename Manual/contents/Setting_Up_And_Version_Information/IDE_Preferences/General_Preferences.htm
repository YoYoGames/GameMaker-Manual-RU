Общие предпочтения

Общие параметры используются для определения общего поведения среды GameMaker IDE. Самые общие из них перечислены на главной странице, в то время как многие другие разделены на несколько подкатегорий, которые описаны далее на этой странице.

  • Язык IDE: Эта опция позволяет выбрать язык для сайта IDE из прилагаемого списка. По умолчанию это английский язык.
  • Шрифт IDE: Эта опция позволяет вам изменить шрифт, используемый на сайте IDE. По умолчанию это Open Sans.
  • Максимальная частота кадров редактора GameMaker: Этот параметр позволяет установить максимальную частоту кадров, с которой будет обновляться GameMaker IDE, что позволяет установить частоту кадров в соответствии со скоростью обновления вашего монитора. Значение по умолчанию - 60.
  • Скин IDE: Здесь вы можете выбрать один из различных скинов, которые может использовать IDE. По умолчанию GameMaker поставляется с двумя скинами, светлым и темным, и при начальной установке используется темный скин.
  • Аудиоустройство по умолчанию: Здесь вы можете выбрать аудиоустройство по умолчанию, которое будет использовать GameMaker IDE. Вы можете обновить список, нажав кнопку Refresh Button .
  • Автоматическая реакция на подтверждение выхода: Эта опция позволяет установить, будет ли показано подтверждающее сообщение при выходе из GameMaker. По умолчанию установлено значение "показать сообщение", но его можно установить так, чтобы пропустить сообщение, как если бы вы нажали кнопку OK.
  • Automatic Response to The System Font Changed Dialog: Этот параметр позволяет установить, следует ли показывать подтверждающее сообщение, когда системный шрифт был добавлен, удален или отредактирован. По умолчанию установлено значение "показать сообщение", но его можно установить, чтобы пропустить сообщение, как если бы вы нажали кнопку OK.
  • Включить уведомления: Установка этого флажка позволит IDE показывать уведомления о новых исполнениях и т.д.. По умолчанию эта опция включена.
  • Тайм-аут уведомления (мс): Устанавливает продолжительность времени (в миллисекундах), в течение которого будет отображаться уведомление. По умолчанию - 6000.
  • Выход при выходе: Если эта опция отмечена, то IDE автоматически завершит сеанс пользователя при закрытии GameMaker. Эта опция предназначена в первую очередь для пользователей системы образования, поскольку, если она отмечена, это означает, что при каждом завершении работы программы пользователь будет выходить из системы, освобождая место в сессии для других пользователей. По умолчанию эта функция выключена.
  • Показывать экран приветствия при запуске: Этот параметр включен по умолчанию и просто указывает GameMaker показывать короткую заставку при запуске программы. Если снять флажок, то заставка не будет отображаться.
  • Разрешить закрытие пустых вкладок с помощью сочетания клавиш: Вы можете закрыть все открытые вкладки с помощью сочетания клавиш / + W, но если вы отключите эту опцию, сочетание клавиш перестанет работать. По умолчанию эта опция включена.
  • Открытие новых ресурсов для редактирования при перетаскивании файлов в IDE: Некоторые ресурсы (например, sprites или звуки) можно перетащить на основную рабочую область в GameMaker IDE, и они будут обнаружены и добавлены как новый ресурс в браузере ресурсов. По умолчанию соответствующий редактор ресурсов также будет открыт для редактирования, но если снять флажок с этой опции, то редакторы не будут открыты, и вам придется открывать их вручную.
  • Открывать новые ресурсы для редактирования при создании: Если этот флажок установлен, то каждый раз, когда вы создаете новый ресурс в браузере активов, соответствующий редактор ресурсов будет открываться автоматически. Этот параметр установлен по умолчанию, и снятие флажка означает, что вам придется вручную открывать все вновь созданные ресурсы.
  • Автоматически перезагружать измененные файлы: Если этот флажок установлен, то IDE будет автоматически сканировать и проверять файлы на предмет того, не были ли они изменены вне GameMaker, и перезагружать их, если это так.
  • Отключите анимацию переходов IDE: GameMaker IDE имеет ряд анимированных переходов для таких вещей, как открытие окон и т.д... однако они не всем по вкусу, и вы можете ощутить прирост производительности, если отключите их на старых или маломощных машинах. Вы можете отключить их здесь, по умолчанию они включены.
  • Opt-in To Feedback Surveys: Когда эта опция отмечена, вы сообщаете GameMaker, что желаете участвовать в опросах обратной связи, то есть время от времени во время работы будет появляться сообщение об участке, над которым вы работаете, с просьбой поставить оценку и оставить отзыв. По умолчанию эта опция включена.
  • Включить переопределение DPI: Эта опция позволяет переопределить настройки DPI по умолчанию на свои собственные. Вы должны сначала включить опцию, а затем установить либо значение Raw DPi Override (значение по умолчанию 96), либо процентное соотношение DPi (значение по умолчанию 100%). После изменения вам будет предложено перезапустить IDE, что может потребовать сохранения любой работы, выполняемой в текущем открытом проекте. Обратите внимание, что минимальный процент DPi составляет 50%, а максимальный - 500%, и если вы наведете курсор мыши на значок информации, то увидите рекомендуемое максимальное значение для используемого дисплея:

    Обратите внимание, что если вы превысите рекомендуемое значение DPi для монитора, появится уведомление, позволяющее сбросить DPi.
