Предпочтения начального экрана

Параметры начального экрана позволяют изменить поведение начальной страницы. Основными опциями являются:

  • Показать экскурсию: Позволяет выбрать, будет ли показан вводный тур при запуске IDE. Эта функция автоматически отключается при первом пропуске тура.
  • Показывать мастер настройки: Этот параметр позволяет выбрать, будет ли мастер настройки отображаться при запуске IDE. Мастер установки, по сути, проводит вас через шаги, необходимые для создания нового проекта.
  • Стиль списка последних проектов: Это позволяет изменить отображение списка последних проектов на стартовой странице между "Списком" и "Сеткой".
