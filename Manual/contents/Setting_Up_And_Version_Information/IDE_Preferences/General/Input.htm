Предпочтения ввода

Параметрыввода управляют определенными аспектами ввода с помощью мыши и клавиатуры в IDE. Доступны следующие параметры:

  • Медленное время двойного щелчка (мс): Некоторые области IDE требуют более медленного двойного щелчка для корректной работы, например, при переименовании ресурса (быстрый двойной щелчок откроет редактор ресурсов, а медленный двойной щелчок позволит вам переименовать его). Эта опция позволяет установить скорость обнаружения, по умолчанию она равна 500 мс.
  • Время блокировки прокрутки (мс): Этот параметр определяет время, которое GameMaker должен ждать между прокручиваемыми окнами, которые перекрываются. По сути, когда вы прокручиваете окно "A", курсор может навестись на другую прокручиваемую область в окне "B", поэтому этот параметр определяет, как долго вы должны ждать после остановки прокрутки "A", прежде чем сможете прокрутить "B", чтобы случайно не начать прокручивать разные панели. Значение по умолчанию равно 500 мс.
  • Обернуть курсор вокруг экрана при панорамировании: Если эта функция включена, курсор будет огибать границы экрана при панорамировании рабочего пространства. По умолчанию включено.
  • Скорость прокрутки: Этот параметр управляет скоростью прокрутки в различных рабочих пространствах. Значение по умолчанию равно 1.
  • Расстояние перетаскивания вкладки до размаксимизации рабочего стола (px): Когда вы перетаскиваете вкладку рабочего пространства из ее положения по умолчанию в пределах IDE, если в это время активно только одно рабочее пространство, то окно GameMaker разворачивается и перемещается с помощью мыши (если активно более одного рабочего пространства, то создается новый экземпляр IDE для размещения рабочего пространства). Этот параметр определяет, насколько далеко нужно перетащить вкладку, чтобы рабочая область распознала ее; значение по умолчанию - 50px.
  • Минимальный уровень масштабирования для взаимодействия с содержимым окна (%): Этот параметр управляет точкой отсечения масштаба, при которой мышь больше не сможет взаимодействовать с окнами рабочей области (т.е. добавлять узлы, перетаскивать окна и т.д.). Значение по умолчанию равно 40.
  • Для масштабирования колесиком мыши требуется клавиша Ctrl: Если флажок установлен, во всех редакторах (и в основной рабочей области) будет включен зум с помощью / + колесика мыши , если флажок снят, зум будет включаться только с помощью колесика мыши MMB Icon.
  • Ctrl + MMB для открытия унаследованного события вместо руководства: Когда эта опция отмечена, удерживая / + среднюю кнопку мыши в объекте, вы откроете наследуемое событие объекта (если оно есть), а не руководство.
  • Добавляет смещение при выделении текста для предотвращения перескакивания строк: Когда этот параметр отмечен, высота задуманной строки при выделении одной строки в любом текстовом окне будет смещена, что делает ее менее подверженной вертикальной неточности. Эта опция включена по умолчанию.
  • Включить режим ноутбука на всех устройствах: Установка этого флажка отобразит кнопку Laptop Mode на сайте IDE, что позволит вам включать или выключать режим по мере необходимости, независимо от того, является ли устройство, на котором запущен GameMaker, ноутбуком или нет. По умолчанию флажок не установлен.
  • Принудительный режим ноутбука: Этот параметр изменяет, будет ли использоваться режим ноутбука или нет, и по умолчанию он включен. Вы можете отключить этот параметр, чтобы режим ноутбука не включался автоматически при запуске IDE на ноутбуке.
