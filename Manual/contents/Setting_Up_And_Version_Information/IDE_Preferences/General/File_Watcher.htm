Предпочтения наблюдателя за файлами

File Watcher - это название части IDE, которая следит за файлами, связанными с проектом, на предмет изменений, вызванных внешними программами (например, редактированием изображения во внешнем редакторе) или внутренними изменениями, например, редактированием сценария. актив. В целом, эти настройки не нужно редактировать, но на некоторых системах - например, тех, которые используют SSD в качестве диска для сохранения файлов проекта - наблюдатель за файлами может сообщать вам об изменениях, когда вы делаете такие вещи, как простое сохранение проекта или редактирование любого актива сценария. или при редактировании любого актива проекта, и в этом случае вам может понадобиться изменить эти настройки.

Доступны следующие настройки:

  • Включить File Watcher: Здесь можно включить или отключить систему File Watcher. Если эта функция отключена, IDE больше не будет обнаруживать файлы, измененные внешними источниками, поэтому при отключении этой функции следует соблюдать осторожность. этой функции. По умолчанию функция File Watcher включена.
  • Установите допуск File Watcher в MS после сохранения В IDE: Это настраивает время между сохранением файла и проверкой изменений наблюдателем за файлами. Вы можете изменить это значение, если вы сохраняете проект, а File Watcher ошибочно появляется сообщение о том, что что-то изменилось, в этом случае вы увеличиваете задержку до тех пор, пока сообщение не перестанет появляться. Значение по умолчанию составляет 2000 мс, но может варьироваться от 1 мс до 10 000 мс.
