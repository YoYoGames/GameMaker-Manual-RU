Предпочтения пути

Параметры Path управляют различными путями к файлам, которые требуются элементам в GameMaker IDE, а также для любых внешних редакторов, которые вам необходимы. Для IDE существуют следующие опции:

  • Временный каталог: Место для сохранения всех временных папок. По умолчанию на Windows это

    \Users\<Username>\AppData\Local\Temp\GameMaker

    А на macOS это:

    /var/folders/<Path_Hash>/GameMaker

  • Каталог кэша IDE: Место для сохранения кэша IDE. По умолчанию на Windows это

    \Users\<Username>\AppData\Roaming\GameMaker\Cache

    А на macOS это:

    /Users/<Username>/.config/GameMaker/Cache

  • Каталог кэша активов: Место для сохранения кэша активов для каждого проекта. По умолчанию на Windows это

    \Users\<Username>\AppData\Roaming\GameMaker\Cache

    А на macOS это:

    /Users/<Username>/.config/GameMaker/Cache

  • Расположение "Мои проекты": Место, где GameMaker будет первоначально создавать новые проекты. По умолчанию на Windows это

    \Users\<Username>\Documents\GameMaker

    А на macOS это

    /Users/<Username>/GameMaker

  • Automatically delete temp directory on close: Установка этого флажка заставит GameMaker автоматически удалять папку Temp, которую он создает в проекте для компиляции и т.д.. Этот параметр включен по умолчанию, а снятие флажка отключает его (это означает, что вам придется вручную удалять все временные файлы).
  • Автоматически удалять кэш активов при закрытии: Если этот флажок установлен, то папка кэша компилятора активов будет удаляться при выходе из GameMaker. По умолчанию эта опция выключена, и ее включение будет означать, что при каждой загрузке и запуске любого проекта кэш нужно будет создавать заново (что может занять время, зависящее от размера игры).
  • Автоматически удалять кэш IDE при закрытии: Если этот флажок установлен, то кэш-папка компилятора IDE будет удаляться при выходе из GameMaker. По умолчанию эта опция выключена, и ее включение будет означать, что при каждом запуске GameMaker кэш придется создавать заново.
  • Удалить временную папку: Нажатие этой кнопки приведет к удалению временной папки проекта.
  • Удалить кэш активов: Нажатие этой кнопки приведет к удалению кэша активов компилятора для проекта.
  • Удалить кэш IDE: Нажав на эту кнопку, вы удалите кэш IDE.

Для задания путей к внешним редакторам существуют следующие варианты:

  • Путь к внешнему редактору/просмотрщику для SWF-файлов: Если вы работаете с форматом SWF sprites, вы можете установить это значение на путь к предпочитаемой программе просмотра/редактору, и когда вы нажмете кнопку Редактировать изображение в редакторе Sprite, откроется данная программа. По умолчанию путь не задан.
  • Путь к внешнему редактору/просмотрщику для файлов Spine: Если вы работаете с форматом Spine sprites , вы можете установить это значение на путь к предпочитаемой программе просмотра/редактору, и когда вы нажмете кнопку Редактировать изображение в редакторе Sprite, откроется данная программа. По умолчанию путь не задан.
