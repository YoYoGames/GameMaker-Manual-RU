Перейти к настройкам

Здесь задаются настройки для окна Go To, которое можно открыть с помощью сочетания клавиш / + T или щелкнув правой кнопкой мыши на пустой области в рабочей области .

  • Максимальное количество отображаемых результатов поиска: Этот параметр управляет максимальным количеством возвращаемых результатов при использовании окна поиска Go To.
  • Включить стилизацию результатов поиска (выключить для лучшей производительности): Этот параметр включает или отключает стилизацию текста для активов, найденных в окне Go To, и включен по умолчанию. Отключение этого параметра должно повысить производительность, если вы считаете, что окно поиска Go To работает медленно на вашей системе.
  • Поиск в активах: Эта опция позволяет окну Go To искать в браузере активов любые активы, соответствующие поисковому запросу, и включена по умолчанию.
  • Поиск в предпочтениях: Эта опция позволяет окну Go To искать в Preferences любые опции, соответствующие поисковому запросу, и включена по умолчанию.
  • Поиск в Redefine Keys: Эта опция позволяет окну Go To искать в меню Redefine Keys любые опции, соответствующие поисковому запросу, и включена по умолчанию.
  • Поиск в параметрах игры: Эта опция позволяет окну Go To искать в Game Options любые опции, соответствующие поисковому запросу, и включена по умолчанию.
  • Поиск в пользовательских функциях: Эта опция позволяет окну Go To искать в ваших сценариях и объектах любые функции, соответствующие поисковому запросу, и включена по умолчанию. Обратите внимание, что эта опция работает только с синтаксисом функции сценария (function name()), но не с синтаксисом переменной метода (name = function()).
