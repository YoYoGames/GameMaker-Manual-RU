Предпочтения ресурсов

Настройки здесь содержат параметры того, как определенные ресурсы будут рисоваться в ваших проектах:

  • Начальный цвет рисования по умолчанию для новых проектов: Когда вы используете шрифт или рисуете фигуру с помощью функций GameMaker или действий GML Visual без указания цвета, будет использоваться цвет, заданный здесь. Цвет по умолчанию является белый ( ARGB: $FFFFFFFF).
  • Начальный цвет окна по умолчанию для новых проектов: Эта опция используется для установки цвета фона HTML-страницы для цели HTML5, а также для установки цвета, отображаемого вне комнат на всех целях (включая HTML5). Цвет по умолчанию черный (ARGB: $000000FF).
