Предпочтения справки

Справочное руководство разработано для работы как веб-страница, и хотя файлы могут храниться локально (в зависимости от ваших настроек), для их просмотра требуется использование внешнего браузера. Здесь можно изменить следующие настройки, связанные со справочным руководством:

  • Язык руководства: Здесь задается язык руководства. По умолчанию это "English".
  • Использовать онлайн-руководство: Включите этот параметр, чтобы использовать веб-сайт интерактивного руководства при каждом открытии руководства, или отключите его, чтобы запускать руководство локально. По умолчанию эта функция отключена.
  • Порт справочного руководства: Устанавливает номер порта для использования веб-сервера микросправки, если отключен параметр "Использовать интерактивную справку". Значение по умолчанию - 51290.
