Предпочтения рабочего пространства

Настройки здесь влияют на то, как вы будете перемещаться в различных рабочих пространствах GameMaker рабочие пространства. Опции следующие:

  • Клавиатурная навигация по рабочему пространству Диапазон степеней: В рабочей области может быть открыто несколько окон для различных объектов и ресурсов, поэтому для быстрого перемещения между ними можно использовать комбинацию клавиш / + + <Arrow Keys>. Учитывая, что окна в рабочей области не выровнены по сетке, IDE должен выбрать, какое окно вы имеете в виду, когда нажимаете любое из 8 возможных направлений, и эта опция позволяет задать "угол обзора", который используется. Значение по умолчанию составляет 40°.
  • Отступы столбцов цепочки рабочего пространства (px): Элементы рабочего пространства часто "связаны" друг с другом (например, редактор кода связан с редактором событий, а редактор событий связан с редактором объектов), и этот параметр позволяет установить расстояние по горизонтали между любыми двумя связанными окнами. Значение по умолчанию - 30px.
  • Отступ строки цепочки рабочего пространства (px): Как упоминалось выше, элементы рабочего пространства часто "цепляются" друг к другу, и этот параметр позволяет установить вертикальное расстояние между любыми двумя цепочками. Значение по умолчанию - 20px.
  • Количество сегментов цепочки рендеринга рабочего пространства: Здесь вы можете установить максимальное количество сегментов для цепочек рендеринга. Уменьшение этого значения может повысить производительность на низкопроизводительных машинах. Значение по умолчанию равно 20.
  • Цвет звеньев цепи рабочего пространства: Это цвет, который будет использоваться для отображения цепей между элементами рабочего пространства. Цвет по умолчанию - ARGB $99DD00.
  • Навигация по рабочему пространству с помощью клавиатуры сбрасывает масштаб: Как упоминалось выше, вы можете переходить от окна к окну в рабочей области с помощью сочетаний клавиш. Вы также можете увеличивать и уменьшать масштаб рабочей области с помощью кнопок / + колесико мыши . Установка этой опции означает, что если вы увеличили масштаб рабочей области и используете горячие клавиши для перехода к другому окну, то рабочая область будет возвращена к виду 1:1 и центрирована на выбранном окне. По умолчанию эта опция выключена.
  • Цепочки рабочих пространств могут пересекаться: При включении этого параметра цепочки рабочих пространств могут пересекаться и накладываться друг на друга. По умолчанию этот параметр выключен.
