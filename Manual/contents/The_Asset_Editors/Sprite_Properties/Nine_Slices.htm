Девять ломтиков

Девятикратная нарезка (также известная как 9-кратная нарезка) - это техника, используемая для масштабирования прямоугольных изображений с целью сохранения деталей, благодаря чему они сохраняют свою первоначальную форму после масштабирования.

Это достигается путем разделения изображения на девять отдельных частей или "кусочков". Угловые фрагменты перемещаются во время масштабирования и рисуются в том же масштабе, затем краевые и центральные фрагменты растягиваются или повторяются (в зависимости от настроек), чтобы заполнить пространство.

Четыре направляющие размещаются на sprite для разделения изображения на девять фрагментов. На следующем рисунке показано, как направляющие остаются на месте после масштабирования sprite с помощью Nine Slice:

Режим плитки среза определяет, как этот срез будет отображаться при масштабировании sprite. Его можно установить только для краевых и центральных срезов (угловые срезы не масштабируются).

Следующий пример демонстрирует каждый режим плитки при применении к центральному срезу:Если для sprite включена функция Nine Slicing, она будет использоваться для изменения размера sprite везде, где он появляется, и будет работать в любых существующих местах, где использовался sprite, без необходимости внесения каких-либо других изменений.

ПРИМЕЧАНИЕ: Девятикратная нарезка не будет работать с функциями, которые рисуют только часть sprite или искажают ее, например, draw_sprite_part() или draw_sprite_pos().

В сочетании с последовательностями эта функция может использоваться для построения пользовательских интерфейсов, а также, например, для создания прямоугольных стен в игровом уровне. Память текстур также можно экономить с помощью Девять нарезок, так как меньшее изображение может быть использовано для покрытия большей области на экране.

Девятикратное нарезание может быть включено на Sprites в IDE. Для получения дополнительной информации см. раздел Редактор Sprite .
