Сниппеты кода

Очень удобным инструментом, который вы имеете в своем распоряжении при редактировании кода, является использование сниппетов кода. При использовании редактора сценариев вы можете нажать F4, чтобы открыть всплывающее окно фрагментов кода, которое позволяет выбрать один из часто используемых методов кода из списка:

В этом окне вы можете выбрать мышью фрагмент, который хотите использовать, или нажать соответствующую горячую клавишу, перечисленную справа. Это добавит сниппет в ваш код для редактирования.

При желании вы можете определить собственные фрагменты кода. Перед этим вам необходимо создать файл с именем "snippets.txt" в одном из следующих каталогов:

  • Окна: %ProgramData%\GameMaker\User\
  • macOS: /Users/Shared/GameMaker/User/

Эта папка не изменяется при любом обновлении GameMaker, поэтому отредактированный вами файл останется нетронутым (но любые правки файла базовых фрагментов в каталоге установки будут отменены).

После создания файла в каталоге User вы можете редактировать его с помощью любого текстового редактора, следуя следующим правилам:

  • Каждый сниппет должен быть на отдельной строке (не должно быть пустых строк)
  • Каждый фрагмент должен начинаться с горячей клавиши, за которой следует "-", затем название фрагмента (то, что отображается в меню), за которым следует двоеточие ":", например:

I - Создание экземпляра:

После двоеточия вы добавляете фрагмент кода

I - Создание экземпляра:instance_create_layer(x, y, |layer|, object);

Код, который вы добавляете, также должен соответствовать определенному формату:

  • Весь код находится в одной строке
  • Вы используете знак "$" для новой строки
    • ПРИМЕЧАНИЕ При создании нескольких новых строк между каждым знаком "$" должен быть пробел. Например, чтобы добавить три новые строки, нужно написать $ $ $.
  • Вы помещаете текст, который необходимо выделить, между символами "|". Редактор кода автоматически выделит этот текст для редактирования при добавлении фрагмента.

Смотрите примеры, которые уже есть в базовом файле фрагментов, чтобы увидеть, как все устроено в соответствии с правилами, описанными выше. Вы можете найти базовый файл в каталоге установки GameMaker:

  • Windows: GameMaker\TextEditor\snippets.txt
  • macOS: /Applications/GameMaker.app/Contents/MonoBundle/TextEditor/snippets.txt
