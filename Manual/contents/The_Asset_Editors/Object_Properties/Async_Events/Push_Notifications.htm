Push-уведомления

Событие Push-уведомления запускается обратным вызовом из push-уведомления на ОС устройства, либо из локального источника, либо из удаленного источника. Оно генерирует DS Map, которая является эксклюзивной для этого события и хранится в специальной переменной async_load.

Это событие используется исключительно расширениями (например, расширением Firebase Cloud Messaging ), поэтому инструкции по использованию этого события см. в документации к вашему расширению.
