Параметры репозитория

Установив исполняемый файл Git и включив Source Control для вашего проекта, вам теперь необходимо связать GameMaker IDE с репозиторием, что делается одним из двух способов:

  • Создание репозитория проекта
  • Клонирование существующего хранилища

В первом случае, при создании репозитория проекта, будет создан репозиторий Git непосредственно там, где находится ваш проект. На языке Git это "непустой репозиторий", поэтому Push/Pull в данном случае не имеют смысла, но вы все равно сможете фиксировать и отслеживать изменения проекта, а также при необходимости возвращать изменения и ревизии. Чтобы настроить его, вам просто нужно выбрать Source Control > Create Project Repository в главном меню.

Для удобства вы также можете клонировать внешний репозиторий через IDE. Опять же, вам понадобится имя пользователя и пароль для репозитория, к которому вы собираетесь подключиться, установленные в ваших настройках идентификации, а затем просто нажмите Source Control > Clone Repository. Заполните URL (используя вариант HTTPS, а не SSH) и укажите, куда вы хотите его отправить:

После завершения клонирования он автоматически откроет браузер файлов, чтобы вы могли открыть проект, если захотите.
