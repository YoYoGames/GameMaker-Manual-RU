Настройка плагина Git

Прежде чем мы сможем использовать плагин Git, нам необходимо настроить исполняемый файл Git для использования, зайдя в раздел Plugins - Source Control (Git) в Preferences:

Путь к исполняемому файлу Git должен быть автоматически заполнен, если он найден с помощью GameMaker, однако если поле пустое, вы можете нажать на кнопку и вручную выбрать файл git.exe, включенный в папку cmd вашей установки Git, которая обычно находится внутри C:\Program Files\Git\.

Остальные параметры Git см. на странице Параметры плагина.
