Micro Web Server

Основной функцией этого небольшого инструмента является подключение и передача файлов на некоторые из доступных целевых платформ для GameMaker (например, на HTML5). Он также обслуживает файлы для руководства GameMaker при автономном использовании.

Когда вы запускаете игру на объекте, для которого требуется Micro Web Server, вы увидите, как открывается новое окно, когда игра передается на выбранное устройство:

ПРИМЕЧАНИЕ При первом запуске игры на цели, для которой требуется Micro Web Server, может появиться предупреждение брандмауэра. В этом случае следует разрешить веб-серверу взаимодействовать через брандмауэр.

Когда Micro Web Server запущен, вы можете получить доступ к своей игре с любого локального устройства (и даже удаленно, если правильно настроено перенаправление портов). Просто зайдите на локальный IP-адрес вашего компьютера через порт, указанный в этом окне, и вы сможете играть в свою игру с другого устройства, например, мобильного телефона или другого компьютера.

Допустим, локальный IP-адрес вашего компьютера - 192.168.0.110( как его найти,читайте здесь ), а порт, указанный в окне Micro Web Server, - 51264. Запустив игру на компьютере, вы подключитесь к 192.168.0.110:51264 через браузер на мобильном телефоне и будете играть в свою игру.

Веб-сервер покажет определенное количество отладочной информации, связанной с файлами, которые были обслужены, и имеет кнопку для ее очистки в нижней части. Рядом с кнопкой "Очистить" указан путь к корневому JS-файлу проекта, а под ней находится опция Скорость сети. По умолчанию она установлена на "Без ограничений", но нажатие на нее открывает список различных сетевых опций:

Изменение этой настройки также изменит параметр Размер пакета вправо. Вы также можете изменить этот параметр вручную, в этом случае для параметра Скорость сети будет установлено значение "Пользовательская". Установка значения, отличного от максимального, эффективно снизит скорость передачи данных по сети и позволит вам тестировать игры в среде с низкой пропускной способностью.

Вы также можете увидеть список разрешенных IP-адресов вместе с номером используемого порта , но их нельзя редактировать в Micro Web Server. Вместо этого их следует установить в настройках диспетчера устройств для данного целевого устройства или в настройках HTML5 Preferences при использовании устройства "По умолчанию".
