Язык GameMaker

Вы можете использовать GameMaker Language для программирования игр двумя различными способами: размещая блоки для визуального "написания" кода или написания кода вручную.

Первый вариант, GML Visual, лучше всего подходит для новичков и любителей, которые хотят быстро приступить к созданию собственных игр.

Второй вариант, GML Code, позволяет вам использовать всю мощь GameMaker Language, написав код вручную.

Хотя с GML Visual легче начинать, GML Code по-прежнему является простым в использовании языком программирования с мощными возможностями(функции, методы, структуры и конструкторы и т.д.).

Создание сценариев Visual/Code

Вам предлагается выбрать между GML Code и GML Visual при создании нового события Object, нового актива Script или чего-либо другого, что приводит к созданию сценария:

Помимо выбора между GML Code и GML Visual, это окно содержит два флажка:

  • Не спрашивать снова для этого проекта: Когда эта опция включена, она устанавливает параметр Game Option "Default scripting language" на ваш новый выбор, поэтому вас не попросят выбрать его снова для текущего проекта.
  • Не спрашивать больше никогда: Когда эта опция включена, она устанавливает предпочтение "Язык сценариев по умолчанию" на ваш новый выбор, так что вас никогда не попросят выбрать его снова, ни для каких проектов.
