YoYo Compiler

YoYo Compiler (YYC ) - это специальный компилятор для создания исполняемых пакетов, использующих машинный код вместо интерпретируемого кода и бегущей строки (именно это используется при компиляции GameMaker VM). Скомпилированный код быстрее выполняется, но на компиляцию исполняемого файла уходит больше времени.

Лицензия

Ваша лицензия GameMaker может включать или не включать компилятор YoYo. Чтобы узнать это, см. страницу Справочного центра " Какая подписка мне подходит?".

Настройка SDK

Для того чтобы YoYo Compiler мог компилировать для целевой платформы, вам необходимо установить правильные SDK.

Для общего обзора по этому вопросу см. следующую страницу Справочного центра: GameMaker Required SDKs.

Информацию о SDK, необходимых для каждой платформы, можно также найти в Справочном центре:

  • Opera GX
  • Windows
  • macOS
  • Ubuntu
  • Android
  • iOS (включая iPadOS)
  • tvOS
