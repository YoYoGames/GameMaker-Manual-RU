Опции игры в Ubuntu

В этом разделе описаны различные доступные вам опции, которые определяют, как будут компилироваться ваши игровые проекты Ubuntu (Linux). Различными разделами являются:

Общие сведения

На этой вкладке вы задаете название вашей игры, которое будет отображаться во время работы, а также можете указать контактный email и описание (краткое и длинное), а также URL для домашней страницы игры или разработчика.

Здесь также есть возможность включить Steam в вашей игре. Если вы включите эту опцию, убедитесь, что у вас установлен Steamworks SDK (см. здесь), и что вы добавили идентификатор приложения Steam в Общие параметры игры.

Наконец, у вас есть возможность отключить песочницу файловой системы. Отметив эту опцию, вы отключите "песочницу" GameMaker для сохранения и загрузки файлов, что позволит вашим играм получать доступ к файлам из любой точки системы, на которой запущена игра. По умолчанию эта опция не отмечена, так как мы рекомендуем поддерживать песочницу GameMaker для файлов на всех системах, и если она отмечена, некоторые области сохранения/загрузки могут быть недоступны в зависимости от разрешений на уровне ОС. Более подробную информацию о песочнице можно найти на страницах, посвященных файловой системе GameMaker.

Графика

Графические параметры - это параметры, которые вы должны настроить, чтобы определить, как ваша игра будет использовать видеокарту вашей целевой машины Ubuntu. Следующие параметры включены для изменения:

  • Start Fullscreen: Если флажок установлен, игра будет запускаться в полноэкранном режиме, в противном случае она будет запускаться в оконном режиме. По умолчанию эта опция выключена.
  • Разрешить полноэкранное переключение: Если этот параметр установлен, пользователь может переключаться из полноэкранного режима в оконный и обратно, используя стандартные сочетания клавиш Ubuntu OS. По умолчанию эта функция выключена.
  • Интерполировать цвета между пикселями: Включает линейную интерполяцию, которая, по сути, "сглаживает" пиксели. Для четкой пиксельной графики ее следует выключить, но если вы хотите получить красивые альфа-смешения и сглаженные края графики, ее лучше оставить включенной. По умолчанию она выключена.
  • Отображать курсор: Когда эта опция включена, в вашей игре будет отображаться курсор ОС по умолчанию. Если эта опция отключена, то при наведении мыши на окно игры курсор не будет виден, если только вы не создали его в рамках проекта игры (по умолчанию он включен).
  • Использовать синхронизацию для предотвращения разрывов: Включает или выключает v-синхронизацию. Обратите внимание, что если у вас есть игра с комнатной скоростью 120, а у игрока монитор с частотой обновления 60, включение этой опции зафиксирует скорость игры на 60. По умолчанию эта опция выключена.
  • Разрешить изменение размера окна: Установка этого флажка позволяет пользователю изменять размер игрового окна (опция "Окно без границ" должна быть выключена, чтобы это работало). По умолчанию эта опция выключена.
  • Масштабирование: Здесь вы можете выбрать сохранение соотношения сторон (таким образом, комната 4:3 будет иметь формат буквы на экране 16:9) или полное масштабирование (растягивание изображения на весь экран).

    ВНИМАНИЕ! Выключение поверхности приложения отключит все опции масштабирования, установленные в разделе Windows Game Options, пока она не будет снова включена. Дополнительные сведения см. в разделе Поверхность приложения.

Наконец, есть возможность задать размер страницы текстуры. По умолчанию (и наиболее совместимый) размер - 2048x2048, но вы можете выбрать любой размер от 256x256 до 8192x8192. Также есть кнопка Предварительный просмотр которая сгенерирует страницы текстур для этой платформы и затем откроет окно, чтобы вы могли посмотреть, как они выглядят. Это может быть очень полезно, если вы хотите увидеть, как структурированы текстурные страницы, и предотвратить создание текстурных страниц большего (или меньшего) размера, чем необходимо. Более подробную информацию о текстурных страницах можно найти здесь.

ПРИМЕЧАНИЕ: Имейте в виду, что чем больше размер страницы текстур, тем менее совместима ваша игра на ПК с более низкими техническими характеристиками.

Изображения

Здесь вы можете добавить заставку для вашей игры и установить иконку игры, которая будет использоваться. Иконка должна быть изображением размером 64x64 пикселя в формате .png, а заставка должна быть как минимум такого же размера, как первая комната в вашей игре (или ее порт просмотра), и тоже в формате .png. Заставка будет отображаться во время загрузки игры, и ее можно включить или выключить, отметив опцию Use Splash Screen (по умолчанию она выключена).

Стоит отметить, что в GameMaker есть инструмент Project Image Generator, который можно использовать для автоматического создания всех изображений, необходимых для различных целевых платформ, для которых компилируется ваша игра. Если вы используете этот инструмент, вам следует пересмотреть созданные изображения, чтобы убедиться, что они соответствуют вашим требованиям.
