Опции игры Opera GX

Параметры игры Opera GX позволяют изменять свойства вашей игры. Здесь представлены следующие разделы:

Общий

В этом окне отображается статус входа в Opera GX. Вы можете нажать на значок Refresh, чтобы войти в систему, используя учетную запись Opera.

Затем у вас есть следующие варианты:

  • Выберите игру: В этом списке можно выбрать существующую игру для обновления или загрузить новую игру.
  • Название игры: Это имя (заголовок) вашей игры на GX.games.

    ВАЖНО Для того чтобы загрузка на GX.games прошла успешно, важно, чтобы вы использовали только разрешенные символы в имени игры (которое по умолчанию установлено на имя вашего проекта GameMaker, поэтому убедитесь, что вы выбрали правильное имя проекта или изменили имя игры вручную).

    Если вы загрузите игру с названием, содержащим запрещенные символы, вы получите эту ошибку: Неверные данные. Проверьте вывод

    Разрешенными символами являются:
    • a-z, A-Z (с поддержкой локальных схем, таких как å)
    • 0-9
    • пробельные символы
    • : - ! ? . , ' & ( )
  • Версия: Это версия игры, которая была загружена последней. Вы можете изменить это поле только до того, как ваша игра будет загружена на GX.games.
  • Следующая версия: Это номер версии, используемый для следующей загрузки в GX.games. Вы можете изменить его только после того, как ваша игра будет загружена на GX.games. Вам не нужно изменять этот параметр вручную для каждой загрузки, так как GameMaker автоматически увеличит последнюю цифру номера версии за вас.
  • Зарегистрироваться на GX.games: Нажмите на эту кнопку, чтобы зарегистрировать новую запись на GX.games. Это возможно только в том случае, если для параметра "Выбрать игру" установлено значение "Новая игра".
  • Выберите группу: Перед загрузкой новой игры вы можете выбрать студию, к которой будет принадлежать игра.
  • Редактировать игру в Opera: Откройте GX.games DevCloud и перейдите на страницу "Подробности игры". Эта страница доступна только тогда, когда вы вошли в систему и ваша игра загружена.
  • Внутренний URL Share: Открывает приватную версию вашей игры, только если она была включена на GX.games DevCloud.
  • Public Share URL: Открывает публичную версию вашей игры, только если она была включена на GX.games DevCloud.

    Оба этих URL можно открыть для других пользователей, чтобы они могли играть в вашу игру.

Графика

Здесь вы можете изменить следующие детали, связанные с тем, как будет отображаться ваша игра:

  • Отображать курсор: Управляет тем, будет ли виден курсор мыши при запуске игры. Полезно для игр, которые не требуют ввода мыши или используют пользовательский спрайт мыши.
  • Интерполировать цвета между пикселями: Включает линейную интерполяцию, которая, по сути, "сглаживает" пиксели. Для четкой пиксельной графики ее следует выключить, но если у вас есть красивые альфа-смешения и сглаженные края графики, ее лучше оставить включенной. По умолчанию эта функция включена.
  • Масштабирование: Ваша игра может быть настроена на автоматическое масштабирование рисуемого холста для поддержания соотношения сторон в браузере, или вы можете выбрать полномасштабный режим. При полномасштабном варианте игра не будет отображаться на весь экран в браузере, а будет растягиваться, чтобы нарисованное соответствовало размеру холста, определенному первой комнатой игры. По умолчанию это соотношение сторон сохраняется.

Наконец, есть возможность задать размер страницы текстуры. По умолчанию (и наиболее совместимый) размер - 2048x2048, но вы можете выбрать любой размер от 256x256 до 8192x8192. Также есть кнопка Предварительный просмотр которая сгенерирует страницы текстур для этой платформы и затем откроет окно, чтобы вы могли посмотреть, как они выглядят. Это может быть очень полезно, если вы хотите увидеть, как структурированы текстурные страницы, и предотвратить создание текстурных страниц большего (или меньшего) размера, чем необходимо. Более подробную информацию о текстурных страницах можно найти здесь.
